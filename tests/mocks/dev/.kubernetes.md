This file provides a mock implementation of a Kubernetes client specifically designed for testing UI components and development workflows in the OpenFrame CLI.

## Key Components

- **MockKubernetesClient**: Main mock struct that simulates Kubernetes API operations
- **NewMockKubernetesClient()**: Factory function that creates a pre-populated mock with sample namespaces and services
- **Namespace Operations**: `GetNamespaces()`, `ValidateNamespace()`, `AddNamespace()`, `ClearNamespaces()`
- **Service Operations**: `GetServices()`, `GetService()`, `ValidateService()`, `GetServicesByType()`
- **Test Utilities**: Methods to simulate failures (`SetShouldFailNamespaces()`, `SetShouldFailServices()`) and manage state (`Reset()`, `AddService()`)

## Usage Example

```go
// Create mock client with sample data
mockClient := NewMockKubernetesClient()

// Get namespaces
namespaces, err := mockClient.GetNamespaces(context.Background())
// Returns: ["default", "production", "staging", "development", "kube-system"]

// Get services in a namespace
services, err := mockClient.GetServices(context.Background(), "default")
// Returns services like "my-api" and "my-frontend"

// Test error scenarios
mockClient.SetShouldFailNamespaces(true)
_, err = mockClient.GetNamespaces(context.Background())
// Returns error for testing failure paths

// Add custom test data
mockClient.AddNamespace("test-env")
mockClient.AddService("test-env", intercept.ServiceInfo{
    Name: "test-service",
    Namespace: "test-env",
    Type: "ClusterIP",
})
```
