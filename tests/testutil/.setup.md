Test utilities for OpenFrame CLI components, providing mock dependencies and test environment configuration.

## Key Components

- **InitializeTestMode()** - Enables test mode for UI components to suppress interactive elements
- **NewTestMockExecutor()** - Creates a mock command executor for isolated unit testing
- **CreateStandardTestFlags()** - Sets up a complete test environment with mocked dependencies (k3d commands, cluster manager)
- **CreateIntegrationTestFlags()** - Creates flag container for integration tests using real dependencies
- **SetVerboseMode()** - Helper to configure verbose logging during tests

## Usage Example

```go
package mytest

import (
    "testing"
    "github.com/flamingo-stack/openframe-cli/internal/testutil"
)

func TestClusterOperation(t *testing.T) {
    // Initialize test environment
    testutil.InitializeTestMode()
    
    // Create test flags with mocked dependencies
    flags := testutil.CreateStandardTestFlags()
    testutil.SetVerboseMode(flags, true)
    
    // Test will use mocked k3d responses
    // "k3d cluster list" returns empty array
    // "k3d cluster get" returns "cluster not found"
    
    // Your test logic here...
}

func TestIntegration(t *testing.T) {
    // For integration tests with real dependencies
    flags := testutil.CreateIntegrationTestFlags()
    // This will use actual k3d commands
}
```
