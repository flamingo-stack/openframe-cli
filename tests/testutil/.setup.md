Test utility helper functions for setting up test environments and mock dependencies for the OpenFrame CLI cluster management functionality.

## Key Components

- **`InitializeTestMode()`** - Configures UI components for test mode to suppress interactive elements
- **`NewTestMockExecutor()`** - Creates a mock command executor for unit testing
- **`CreateStandardTestFlags()`** - Sets up a complete test environment with mocked k3d responses for unit tests
- **`CreateIntegrationTestFlags()`** - Creates a flag container using real dependencies for integration testing
- **`SetVerboseMode()`** - Helper to configure verbose logging for test scenarios

## Usage Example

```go
package main_test

import (
    "testing"
    "github.com/flamingo-stack/openframe-cli/internal/testutil"
)

func TestClusterOperations(t *testing.T) {
    // Initialize test environment
    testutil.InitializeTestMode()
    
    // Create mock test environment
    flags := testutil.CreateStandardTestFlags()
    testutil.SetVerboseMode(flags, true)
    
    // Test will use mocked k3d responses
    // No actual k3d commands will be executed
}

func TestIntegration(t *testing.T) {
    // Use real dependencies for integration testing
    flags := testutil.CreateIntegrationTestFlags()
    
    // This will execute actual k3d commands
    // Requires k3d to be installed
}
```

The package provides two testing modes: unit tests with complete mocking and integration tests with real command execution, making it easy to test cluster management functionality at different levels.
