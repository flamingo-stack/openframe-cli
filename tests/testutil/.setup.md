This file provides test utilities and setup functions for configuring test environments in the OpenFrame CLI project.

## Key Components

- **InitializeTestMode()**: Enables test mode for UI components
- **NewTestMockExecutor()**: Creates a mock command executor for testing
- **CreateStandardTestFlags()**: Sets up a flag container with mock dependencies for unit tests
- **CreateIntegrationTestFlags()**: Creates a flag container with real dependencies for integration tests
- **SetVerboseMode()**: Configures verbose logging for test scenarios

## Usage Example

```go
import "github.com/flamingo-stack/openframe-cli/internal/testutil"

// Setup unit tests with mocked dependencies
func TestClusterOperations(t *testing.T) {
    testutil.InitializeTestMode()
    flags := testutil.CreateStandardTestFlags()
    testutil.SetVerboseMode(flags, true)
    
    // Use flags with pre-configured mock responses
    // Mock will return empty cluster list and "not found" errors
}

// Setup integration tests with real dependencies
func TestIntegration(t *testing.T) {
    flags := testutil.CreateIntegrationTestFlags()
    testutil.SetVerboseMode(flags, false)
    
    // Use flags with real k3d manager for actual testing
}

// Create custom mock executor
mockExecutor := testutil.NewTestMockExecutor()
mockExecutor.SetResponse("custom-command", &executor.CommandResult{
    ExitCode: 0,
    Stdout:   "success",
})
```
