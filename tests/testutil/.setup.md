Test utility functions for setting up OpenFrame CLI test environments with mock dependencies and configuration helpers.

## Key Components

- **InitializeTestMode()** - Enables UI test mode to disable interactive elements
- **NewTestMockExecutor()** - Creates a mock command executor for testing
- **CreateStandardTestFlags()** - Sets up unit test environment with mocked k3d responses
- **CreateIntegrationTestFlags()** - Creates real dependency container for integration tests
- **SetVerboseMode()** - Helper to configure verbose logging for tests

## Usage Example

```go
import "github.com/flamingo-stack/openframe-cli/internal/testutil"

func TestClusterOperations(t *testing.T) {
    // Initialize test environment
    testutil.InitializeTestMode()
    
    // Create test flags with mocked dependencies
    flags := testutil.CreateStandardTestFlags()
    testutil.SetVerboseMode(flags, true)
    
    // Use flags for testing cluster operations
    // Mock executor will return predefined responses
    result := flags.TestManager.ListClusters()
    
    // For integration tests
    integrationFlags := testutil.CreateIntegrationTestFlags()
    // This will use real k3d commands
}

func TestWithMockExecutor(t *testing.T) {
    executor := testutil.NewTestMockExecutor()
    executor.SetResponse("custom command", &executor.CommandResult{
        ExitCode: 0,
        Stdout:   "success",
    })
}
```

The package provides two testing approaches: unit tests with mocked k3d commands that return empty cluster lists, and integration tests that execute real k3d operations against the system.
