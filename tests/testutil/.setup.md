A utility package providing test setup functions and mock configurations for cluster operations and command execution testing.

## Key Components

- **InitializeTestMode()** - Enables test mode for UI components by setting `ui.TestMode = true`
- **NewTestMockExecutor()** - Creates a new mock command executor for testing without external dependencies
- **CreateStandardTestFlags()** - Sets up a complete test flag container with mocked k3d cluster responses
- **CreateIntegrationTestFlags()** - Creates flag container for integration tests using real dependencies
- **SetVerboseMode()** - Helper to configure verbose logging for test scenarios

## Usage Example

```go
// Unit test setup with mocked dependencies
func TestClusterOperations(t *testing.T) {
    testutil.InitializeTestMode()
    flags := testutil.CreateStandardTestFlags()
    testutil.SetVerboseMode(flags, true)
    
    // flags.TestManager will use mocked k3d responses
    // flags.Executor will return predefined command results
}

// Integration test setup with real dependencies
func TestRealCluster(t *testing.T) {
    flags := testutil.CreateIntegrationTestFlags()
    // Uses actual k3d commands and cluster managers
}

// Using mock executor directly
func TestCommandExecution(t *testing.T) {
    mockExec := testutil.NewTestMockExecutor()
    mockExec.SetResponse("custom command", &executor.CommandResult{
        ExitCode: 0,
        Stdout:   "success",
    })
}
```
