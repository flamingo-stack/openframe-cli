Provides a standardized testing framework for cobra CLI commands with comprehensive test coverage across structure, flags, arguments, and execution scenarios.

## Key Components

- **TestClusterCommand** - Main test orchestrator that runs all standard test suites for cluster commands
- **testCommandStructure** - Validates command metadata (name, descriptions, RunE function presence)
- **testCommandFlags** - Tests flag handling including help and invalid flag scenarios  
- **testCommandCLI** - Validates command line argument parsing and validation
- **testCommandExecution** - Tests basic command execution with dry-run and help flag support

## Usage Example

```go
func TestCreateCommand(t *testing.T) {
    setupFunc := func() {
        // Initialize mocks, set environment variables
        mockClient = &MockClusterClient{}
        os.Setenv("OPENFRAME_CONFIG", "/tmp/test-config")
    }
    
    teardownFunc := func() {
        // Clean up resources
        mockClient = nil
        os.Unsetenv("OPENFRAME_CONFIG")
    }
    
    cmdFunc := func() *cobra.Command {
        return cluster.NewCreateCommand(mockClient)
    }
    
    testutil.TestClusterCommand(t, "create", cmdFunc, setupFunc, teardownFunc)
}
```

The framework automatically tests command structure validation, flag parsing, argument handling, and execution scenarios including dry-run mode when available. Each test function accepts setup/teardown callbacks for test isolation and resource management.
