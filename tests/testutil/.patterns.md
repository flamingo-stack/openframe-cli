Provides a standardized testing pattern for Cobra CLI cluster commands with comprehensive validation across structure, flags, arguments, and execution scenarios.

## Key Components

- **`TestClusterCommand`** - Main test orchestrator that runs all standard tests for cluster commands
- **`testCommandStructure`** - Validates basic command properties (name, descriptions, RunE function)
- **`testCommandFlags`** - Tests flag handling including help and invalid flag scenarios  
- **`testCommandCLI`** - Validates command-line argument parsing and validation
- **`testCommandExecution`** - Tests actual command execution with dry-run and help flags

## Usage Example

```go
func TestCreateCommand(t *testing.T) {
    setupFunc := func() {
        // Initialize mocks, set environment variables
        mockClient := &MockClient{}
        os.Setenv("CLUSTER_CONFIG", "/tmp/test-config")
    }
    
    teardownFunc := func() {
        // Clean up resources
        os.Unsetenv("CLUSTER_CONFIG")
    }
    
    cmdFunc := func() *cobra.Command {
        return NewCreateCommand()
    }
    
    testutil.TestClusterCommand(t, "create", cmdFunc, setupFunc, teardownFunc)
}
```

This pattern ensures consistent testing coverage across all cluster commands by validating command structure, flag behavior, argument handling, and basic execution scenarios in a reusable way.
