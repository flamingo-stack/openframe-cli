A fluent assertion library for testing command line tool outputs, providing chainable methods to validate stdout, stderr, and error conditions.

## Key Components

- **AssertCommandOutput**: Main struct containing fluent assertion methods for command output validation
- **NewAssertCommandOutput()**: Constructor for creating assertion instances
- **Error assertions**: `NoError()`, `HasError()`, `ErrorContains()`
- **Output assertions**: `StdoutContains()`, `StderrContains()`, `OutputContains()`, `ContainsAll()`
- **Empty checks**: `StdoutEmpty()`, `StderrEmpty()`
- **Helper functions**: `AssertCommandSuccess()`, `AssertCommandFailure()`

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    // Test successful command
    stdout, stderr, err := executeCommand("my-tool", "--help")
    AssertCommandSuccess(t, stdout, stderr, err).
        StdoutContains("Usage:").
        StdoutContains("Options:")

    // Test command failure
    stdout, stderr, err = executeCommand("my-tool", "--invalid")
    AssertCommandFailure(t, stdout, stderr, err).
        ErrorContains("unknown flag").
        StderrContainsAny("invalid", "unknown")

    // Chain multiple assertions
    NewAssertCommandOutput(t, stdout, stderr, err).
        NoError().
        StderrEmpty().
        ContainsAll("version", "build")
}
```
