A fluent assertion library for testing command-line output in Go applications, providing chainable methods to verify stdout, stderr, and error conditions.

## Key Components

- **AssertCommandOutput**: Main assertion struct with fluent interface for command output testing
- **NewAssertCommandOutput**: Factory function to create new asserter instances
- **Error Assertions**: `NoError()`, `HasError()`, `ErrorContains()` for error condition testing
- **Output Assertions**: `StdoutContains()`, `StderrContains()`, `OutputContains()` for content verification
- **Empty Checks**: `StdoutEmpty()`, `StderrEmpty()` for validating no output
- **Batch Operations**: `ContainsAll()`, `StderrContainsAny()` for multiple condition testing
- **Helper Functions**: `AssertCommandSuccess()`, `AssertCommandFailure()` for common patterns

## Usage Example

```go
func TestCommandExecution(t *testing.T) {
    // Test successful command
    stdout, stderr, err := runCommand("ls", "-la")
    testutil.AssertCommandSuccess(t, stdout, stderr, err).
        StdoutContains("total").
        StdoutContains(".git")

    // Test command failure
    stdout, stderr, err = runCommand("invalid-command")
    testutil.AssertCommandFailure(t, stdout, stderr, err).
        ErrorContains("command not found").
        StderrContainsAny("not found", "unknown command")

    // Custom assertion chain
    testutil.NewAssertCommandOutput(t, stdout, stderr, err).
        NoError().
        ContainsAll("file1.txt", "file2.txt").
        OutputContains("success")
}
```
