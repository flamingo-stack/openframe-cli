Utility functions for creating test cluster configurations in the OpenFrame CLI test suite.

## Key Components

- **TestClusterConfig**: Factory function that creates a pre-configured cluster configuration for testing purposes

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/flamingo-stack/openframe-cli/internal/testutil"
)

func TestClusterCreation(t *testing.T) {
    // Create a test cluster configuration
    config := testutil.TestClusterConfig("test-cluster")
    
    fmt.Printf("Cluster: %s\n", config.Name)
    fmt.Printf("Type: %s\n", config.Type)
    fmt.Printf("Nodes: %d\n", config.NodeCount)
    fmt.Printf("K8s Version: %s\n", config.K8sVersion)
    
    // Use in your tests
    assert.Equal(t, "test-cluster", config.Name)
    assert.Equal(t, models.ClusterTypeK3d, config.Type)
    assert.Equal(t, 3, config.NodeCount)
}
```

The `TestClusterConfig` function provides sensible defaults for testing cluster operations, using K3d as the cluster type with 3 nodes running Kubernetes v1.28.0. This standardizes test configurations and reduces boilerplate code in test files.
