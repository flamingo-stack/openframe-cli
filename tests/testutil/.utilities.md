This file provides utility functions for creating test fixtures and mock data structures used in testing scenarios for cluster configurations.

## Key Components

- `TestClusterConfig(name string) *models.ClusterConfig` - Factory function that creates a standardized test cluster configuration with predefined defaults

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/testutil"
)

func TestClusterCreation(t *testing.T) {
    // Create a test cluster configuration
    config := testutil.TestClusterConfig("test-cluster")
    
    // Use the configuration in your tests
    assert.Equal(t, "test-cluster", config.Name)
    assert.Equal(t, models.ClusterTypeK3d, config.Type)
    assert.Equal(t, 3, config.NodeCount)
    assert.Equal(t, "v1.28.0", config.K8sVersion)
}
```

The function returns a cluster configuration with sensible defaults: K3d cluster type, 3 nodes, and Kubernetes version v1.28.0, making it easy to create consistent test data across different test suites.
