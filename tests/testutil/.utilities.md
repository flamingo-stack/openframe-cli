Utility functions for creating test data structures in OpenFrame CLI testing scenarios.

## Key Components

- **TestClusterConfig**: Factory function that creates a standardized cluster configuration for testing purposes

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/testutil"
)

func TestClusterCreation(t *testing.T) {
    // Create a test cluster configuration
    config := testutil.TestClusterConfig("test-cluster")
    
    // The config will have default test values:
    // - Name: "test-cluster"
    // - Type: K3d cluster type
    // - NodeCount: 3 nodes
    // - K8sVersion: "v1.28.0"
    
    // Use in your tests
    assert.Equal(t, "test-cluster", config.Name)
    assert.Equal(t, 3, config.NodeCount)
}
```

The `TestClusterConfig` function provides a consistent way to generate cluster configurations for unit tests, ensuring all tests use the same baseline configuration while allowing customization of the cluster name.
