Test utility functions for creating mock cluster configurations in the OpenFrame CLI test suite.

## Key Components

- **TestClusterConfig** - Factory function that creates standardized test cluster configurations with default values for testing purposes

## Usage Example

```go
package testutil_test

import (
    "testing"
    "github.com/flamingo-stack/openframe-cli/internal/testutil"
)

func TestClusterCreation(t *testing.T) {
    // Create a test cluster configuration
    config := testutil.TestClusterConfig("test-cluster")
    
    // Verify default values
    if config.Name != "test-cluster" {
        t.Errorf("Expected name 'test-cluster', got %s", config.Name)
    }
    
    if config.Type != models.ClusterTypeK3d {
        t.Errorf("Expected K3d cluster type, got %s", config.Type)
    }
    
    // Use in cluster operations
    err := clusterManager.Create(config)
    if err != nil {
        t.Fatalf("Failed to create test cluster: %v", err)
    }
}
```

The utility provides consistent test data with sensible defaults: 3-node K3d cluster running Kubernetes v1.28.0, making it easy to write reliable unit tests without duplicating configuration setup code.
