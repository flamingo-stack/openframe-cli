A utility package providing test helper functions for creating standardized test configurations in the OpenFrame CLI test suite.

## Key Components

- **TestClusterConfig()** - Creates a standardized cluster configuration for testing purposes

## Usage Example

```go
package mypackage_test

import (
    "testing"
    "github.com/flamingo-stack/openframe-cli/internal/testutil"
)

func TestClusterOperations(t *testing.T) {
    // Create a test cluster configuration
    config := testutil.TestClusterConfig("test-cluster")
    
    // Use the config in your tests
    assert.Equal(t, "test-cluster", config.Name)
    assert.Equal(t, models.ClusterTypeK3d, config.Type)
    assert.Equal(t, 3, config.NodeCount)
    assert.Equal(t, "v1.28.0", config.K8sVersion)
}

func TestMultipleClusterConfigs(t *testing.T) {
    // Create multiple test configurations
    devConfig := testutil.TestClusterConfig("dev-cluster")
    prodConfig := testutil.TestClusterConfig("prod-cluster")
    
    // Both will have the same defaults but different names
    assert.NotEqual(t, devConfig.Name, prodConfig.Name)
    assert.Equal(t, devConfig.Type, prodConfig.Type)
}
```

This utility ensures consistent test data across the test suite by providing pre-configured cluster settings with sensible defaults (K3d type, 3 nodes, Kubernetes v1.28.0).
