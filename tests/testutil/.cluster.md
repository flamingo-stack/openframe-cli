A testing utility package providing helper functions for validating Cobra CLI commands and creating standardized test configurations for cluster operations.

## Key Components

### Command Testing Functions
- **`TestCommandStructure`** - Validates basic Cobra command properties (Use, Short, RunE, Long descriptions)
- **`TestCLICommand`** - Executes commands with arguments and validates output/error conditions

### Flag Container Functions
- **`TestFlags`** - Creates standard test flag configurations via `CreateStandardTestFlags()`
- **`IntegrationFlags`** - Creates integration test flag configurations via `CreateIntegrationTestFlags()`

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    // Validate command structure
    cmd := myapp.NewListCommand()
    testutil.TestCommandStructure(t, cmd, "list", "List resources")
    
    // Test command execution
    testutil.TestCLICommand(t, cmd, 
        []string{"--format", "json"}, 
        false, // expect no error
        "resource1", "resource2") // expect these strings in output
    
    // Use test flags
    flags := testutil.TestFlags()
    // Configure your test with flags...
    
    // For integration tests
    integrationFlags := testutil.IntegrationFlags()
    // Use for end-to-end testing...
}
```

This utility simplifies testing CLI commands by providing reusable validation patterns and standardized test configurations for both unit and integration tests.
