A utility package providing helper functions for testing CLI commands and cluster operations in the OpenFrame CLI project.

## Key Components

- **TestCommandStructure()** - Validates basic cobra.Command properties including Use, Short description, RunE function, and Long description
- **TestCLICommand()** - Executes a command with specified arguments and validates both error conditions and output content
- **TestFlags()** - Creates a standard flag container for unit testing
- **IntegrationFlags()** - Creates a flag container specifically for integration tests

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    cmd := &cobra.Command{
        Use:   "mycommand",
        Short: "My test command",
        Long:  "A longer description",
        RunE:  func(cmd *cobra.Command, args []string) error { return nil },
    }

    // Test command structure
    testutil.TestCommandStructure(t, cmd, "mycommand", "My test command")

    // Test command execution
    testutil.TestCLICommand(t, cmd, []string{"arg1"}, false, "expected output")

    // Use test flags
    flags := testutil.TestFlags()
    integrationFlags := testutil.IntegrationFlags()
}
```

The package relies on external functions `CreateStandardTestFlags()` and `CreateIntegrationTestFlags()` for flag container creation.
