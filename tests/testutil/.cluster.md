This file provides testing utilities for validating Cobra CLI commands and their behavior in the OpenFrame CLI application.

## Key Components

- **TestCommandStructure()** - Validates basic command properties like usage strings, descriptions, and RunE function presence
- **TestCLICommand()** - Executes commands with arguments and validates output/error behavior
- **TestFlags()** - Creates standardized flag containers for unit testing
- **IntegrationFlags()** - Creates flag containers specifically for integration testing

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    cmd := &cobra.Command{
        Use:   "deploy",
        Short: "Deploy application",
        Long:  "Deploy your application to the cluster",
        RunE:  func(cmd *cobra.Command, args []string) error { return nil },
    }
    
    // Test command structure
    testutil.TestCommandStructure(t, cmd, "deploy", "Deploy application")
    
    // Test command execution
    testutil.TestCLICommand(t, cmd, []string{"--help"}, false, "Deploy your application")
    
    // Use test flags
    flags := testutil.TestFlags()
    // Configure and use flags...
}
```

The helper functions use `t.Helper()` to ensure proper test line reporting and provide consistent validation patterns for CLI command testing across the codebase.
