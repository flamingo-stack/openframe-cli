This file provides testing utilities for validating Cobra CLI commands and creating standardized flag configurations for OpenFrame CLI tests.

## Key Components

- **TestCommandStructure** - Validates basic command properties like usage, description, and RunE function
- **TestCLICommand** - Executes commands with arguments and validates output/error expectations
- **TestFlags** - Creates standard flag containers for unit testing
- **IntegrationFlags** - Creates flag containers specifically for integration tests

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    // Test command structure
    cmd := &cobra.Command{
        Use:   "test",
        Short: "Test command",
        Long:  "A test command for validation",
        RunE:  func(cmd *cobra.Command, args []string) error {
            return nil
        },
    }
    
    testutil.TestCommandStructure(t, cmd, "test", "Test command")
    
    // Test command execution
    testutil.TestCLICommand(t, cmd, []string{}, false, "expected output")
    
    // Use test flags
    flags := testutil.TestFlags()
    // Configure command with test flags
}
```

The utilities follow testing best practices by using `t.Helper()` to provide clear test failure locations and standardized assertion patterns for CLI command validation.
