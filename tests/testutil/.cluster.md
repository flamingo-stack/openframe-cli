A utility package providing helper functions for testing CLI commands and cluster operations within the OpenFrame testing framework.

## Key Components

- **TestCommandStructure** - Validates basic properties of Cobra commands including Use, Short description, RunE function, and Long description
- **TestCLICommand** - Executes CLI commands with specified arguments and validates output/error behavior
- **TestFlags** - Creates standard test flag containers using `CreateStandardTestFlags()`
- **IntegrationFlags** - Creates integration test flag containers using `CreateIntegrationTestFlags()`

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    // Validate command structure
    testutil.TestCommandStructure(t, myCmd, "my-command", "Does something")
    
    // Test successful command execution
    testutil.TestCLICommand(t, myCmd, []string{"--flag", "value"}, false, "Expected output")
    
    // Test command failure
    testutil.TestCLICommand(t, myCmd, []string{"--invalid"}, true, "error message")
    
    // Get test flags
    flags := testutil.TestFlags()
    integrationFlags := testutil.IntegrationFlags()
}
```

These helpers standardize testing patterns across the OpenFrame CLI, ensuring consistent validation of command structure, execution behavior, and flag handling. The functions use testify assertions and follow Go testing best practices with `t.Helper()` to provide accurate test failure locations.
