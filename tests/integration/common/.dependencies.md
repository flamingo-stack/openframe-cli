This file provides a dependency checking system for integration tests, allowing tests to gracefully skip when required external tools are not available.

## Key Components

- **`Dependency`** struct - Represents an external dependency with check command and install instructions
- **`IsAvailable()`** method - Executes check command to verify dependency availability
- **`RequireOrSkip()`** method - Skips test with warning message if dependency is missing
- **Predefined dependencies** - Docker, K3d, Kubectl, Helm with their respective check commands
- **Convenience functions** - `RequireClusterDependencies()`, `RequireK8sDependencies()`, `RequireAllDependencies()`

## Usage Example

```go
func TestK8sIntegration(t *testing.T) {
    // Check all dependencies before running test
    common.RequireAllDependencies(t)
    
    // Test will only run if Docker, k3d, kubectl, and helm are available
    // ... integration test code ...
}

func TestCustomDependency(t *testing.T) {
    customDep := &common.Dependency{
        Name:       "custom-tool not available",
        CheckCmd:   []string{"custom-tool", "--version"},
        InstallMsg: "Install custom-tool from https://example.com",
    }
    
    customDep.RequireOrSkip(t, "custom-tool missing, skipping test")
    // ... test code ...
}
```
