A comprehensive test utility for building and executing OpenFrame CLI commands with intelligent caching and result analysis.

## Key Components

### Global Variables
- `cliBinary` - Path to the built CLI binary
- `projectRoot` - Cached project root directory path

### Core Functions
- **`InitializeCLI()`** - Builds the CLI binary with timestamp-based caching
- **`CleanupCLI()`** - Removes the test CLI binary
- **`GetProjectRoot()`** - Locates the project root by finding `go.mod`
- **`RunCLI(args...)`** - Executes CLI commands and captures output

### Result Type
- **`CLIResult`** - Contains stdout, stderr, exit code, and error information with helper methods

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func TestCLICommands() {
    // Initialize CLI binary (builds if needed)
    if err := InitializeCLI(); err != nil {
        log.Fatal("Failed to initialize CLI:", err)
    }
    defer CleanupCLI()

    // Run a CLI command
    result := RunCLI("version")
    
    if result.Success() {
        fmt.Printf("CLI output: %s\n", result.Stdout)
    } else {
        fmt.Printf("Command failed: %s\n", result.ErrorMessage())
        fmt.Printf("Exit code: %d\n", result.ExitCode)
    }

    // Check for specific error conditions
    configResult := RunCLI("config", "invalid-command")
    if configResult.Failed() {
        fmt.Printf("Expected failure: %s\n", configResult.ErrorMessage())
    }
}
```

The utility automatically handles binary building, caching based on source file timestamps, and provides rich result objects for test assertions and debugging.
