Provides utilities for building, running, and managing a CLI binary during testing with intelligent caching and project root detection.

## Key Components

**Functions:**
- `InitializeCLI()` - Builds the CLI binary with caching (rebuilds only if source is newer)
- `CleanupCLI()` - Removes the test CLI binary and cleans up resources
- `GetProjectRoot()` - Finds and caches the project root directory by locating go.mod
- `RunCLI(args...)` - Executes the CLI binary with given arguments and captures output

**Types:**
- `CLIResult` - Contains command execution results with stdout, stderr, exit code, and error
- Methods: `Success()`, `Failed()`, `Output()`, `ErrorMessage()`

## Usage Example

```go
// Initialize and build the CLI binary
err := InitializeCLI()
if err != nil {
    log.Fatal("Failed to initialize CLI:", err)
}
defer CleanupCLI()

// Run a CLI command
result := RunCLI("status", "--format", "json")

if result.Success() {
    fmt.Println("Command output:", result.Stdout)
} else {
    fmt.Printf("Command failed (exit %d): %s\n", 
        result.ExitCode, result.ErrorMessage())
}

// Check project structure
root := GetProjectRoot()
fmt.Println("Project root:", root)
```
