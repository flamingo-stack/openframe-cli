Provides utilities for building and running the OpenFrame CLI binary in test environments with intelligent caching and cross-platform path resolution.

## Key Components

- **InitializeCLI()**: Builds the CLI binary with timestamp-based caching to avoid unnecessary rebuilds
- **RunCLI(args...)**: Executes CLI commands and captures stdout/stderr with proper error handling
- **GetProjectRoot()**: Locates the project root directory by searching for go.mod files
- **CLIResult**: Struct containing command execution results with convenience methods
- **CleanupCLI()**: Removes the test binary to clean up resources

## Usage Example

```go
// Initialize the CLI binary (builds if needed)
err := InitializeCLI()
if err != nil {
    log.Fatal("Failed to build CLI:", err)
}

// Run a CLI command
result := RunCLI("cluster", "create", "--name", "test")
if result.Success() {
    fmt.Println("Command output:", result.Stdout)
} else {
    fmt.Printf("Command failed with exit code %d: %s\n", 
        result.ExitCode, result.ErrorMessage())
}

// Clean up when done
defer CleanupCLI()

// Check project structure
root := GetProjectRoot()
fmt.Println("Project root:", root)
```

The module automatically handles binary caching, path resolution across different test directories, and provides detailed execution results for comprehensive CLI testing scenarios.
