A utility package for building, managing, and executing CLI binaries during testing with intelligent caching and project root discovery.

## Key Components

- **`InitializeCLI()`** - Builds the CLI binary with smart caching that only rebuilds when source files are newer
- **`CleanupCLI()`** - Removes the test CLI binary and resets state
- **`GetProjectRoot()`** - Discovers project root by searching for `go.mod` with fallback strategies
- **`RunCLI(args...)`** - Executes the CLI binary with arguments and captures output/errors
- **`CLIResult`** - Struct containing execution results with helper methods for success/failure checking

## Usage Example

```go
// Initialize CLI binary (builds only if needed)
err := InitializeCLI()
if err != nil {
    log.Fatal("Failed to initialize CLI:", err)
}
defer CleanupCLI()

// Execute CLI commands
result := RunCLI("cluster", "list")
if result.Success() {
    fmt.Println("Output:", result.Stdout)
} else {
    fmt.Printf("Command failed with exit code %d\n", result.ExitCode)
    fmt.Println("Error:", result.ErrorMessage())
}

// Check for specific error conditions
if result.Failed() && strings.Contains(result.Stderr, "connection refused") {
    // Handle connection errors
}
```

The package provides intelligent binary caching, robust error handling, and convenient methods for testing CLI applications across different working directories.
