Provides utilities for managing k3d Kubernetes test clusters with optimized Docker resource cleanup. This module handles the lifecycle of test clusters including creation, deletion, and comprehensive cleanup of associated Docker resources.

## Key Components

- **GenerateTestClusterName()** - Creates timestamped unique cluster names for testing
- **CreateTestCluster(name)** - Creates a k3d cluster with 1 agent node and 60s timeout
- **DeleteTestCluster(name)** - Removes a specific test cluster
- **ClusterExists(name)** - Checks cluster existence using k3d list command
- **StopTestCluster(name)** - Stops a running test cluster
- **CleanupTestCluster(name)** - Comprehensive cleanup with Docker resource removal
- **CleanupAllTestClusters()** - Bulk cleanup of all test-related clusters
- **cleanupDockerResources()** - Removes orphaned k3d networks, containers, and registries
- **cleanupClusterSpecificResources(name)** - Targeted cleanup for specific cluster resources

## Usage Example

```go
// Create and manage a test cluster
clusterName := GenerateTestClusterName()
err := CreateTestCluster(clusterName)
if err != nil {
    log.Fatal("Failed to create cluster:", err)
}

// Check if cluster exists
exists, err := ClusterExists(clusterName)
if err != nil || !exists {
    log.Fatal("Cluster verification failed")
}

// Cleanup when done
defer CleanupTestCluster(clusterName)

// Bulk cleanup all test clusters
CleanupAllTestClusters()
```

The module includes performance optimizations like reduced timeouts, direct k3d command usage, and comprehensive Docker resource cleanup to prevent conflicts in test environments.
