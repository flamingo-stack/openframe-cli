This package provides utilities for managing k3d Kubernetes test clusters, handling cluster lifecycle operations and Docker resource cleanup to prevent conflicts during testing.

## Key Components

- **GenerateTestClusterName()** - Creates unique cluster names with Unix timestamps
- **CreateTestCluster(name)** - Creates k3d clusters with 1 agent node and 60s timeout
- **DeleteTestCluster(name)** - Removes specified test clusters
- **ClusterExists(name)** - Checks cluster existence using k3d list command
- **StopTestCluster(name)** - Stops running test clusters
- **CleanupTestCluster(name)** - Comprehensive cleanup including Docker resources
- **CleanupAllTestClusters()** - Bulk cleanup of test clusters matching naming patterns
- **cleanupDockerResources()** - Removes leftover k3d networks, containers, and registries
- **cleanupClusterSpecificResources(name)** - Targeted cleanup for specific cluster resources

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "github.com/openframe-org/openframe/pkg/common"
)

func main() {
    // Generate a unique test cluster name
    clusterName := common.GenerateTestClusterName()
    fmt.Printf("Creating cluster: %s\n", clusterName)
    
    // Create the test cluster
    if err := common.CreateTestCluster(clusterName); err != nil {
        log.Fatalf("Failed to create cluster: %v", err)
    }
    
    // Check if cluster exists
    exists, err := common.ClusterExists(clusterName)
    if err != nil || !exists {
        log.Printf("Cluster verification failed")
    }
    
    // Cleanup when done
    defer common.CleanupTestCluster(clusterName)
    
    // Your test logic here...
    
    // Clean up all test clusters if needed
    common.CleanupAllTestClusters()
}
```

The package includes performance optimizations like Docker environment variables, strategic sleep delays, and pattern-based cleanup to handle various test cluster naming conventions efficiently.
