This file provides utilities for managing k3d (lightweight Kubernetes) test clusters, including creation, deletion, and cleanup operations for integration testing environments.

## Key Components

- **GenerateTestClusterName()** - Creates unique cluster names with Unix timestamps
- **CreateTestCluster(name)** - Creates a k3d cluster with 1 agent node and 60s timeout
- **DeleteTestCluster(name)** - Removes a specific test cluster
- **ClusterExists(name)** - Checks if a cluster exists using k3d list command
- **StopTestCluster(name)** - Stops a running test cluster
- **CleanupTestCluster(name)** - Comprehensive cleanup of cluster and Docker resources
- **CleanupAllTestClusters()** - Removes all test-related clusters based on naming patterns
- **cleanupDockerResources()** - Internal function to remove leftover Docker networks and containers
- **cleanupClusterSpecificResources(name)** - Internal function for targeted cluster resource cleanup

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "time"
)

func runIntegrationTest() {
    // Generate unique cluster name
    clusterName := GenerateTestClusterName()
    fmt.Printf("Creating cluster: %s\n", clusterName)
    
    // Create test cluster
    if err := CreateTestCluster(clusterName); err != nil {
        log.Fatalf("Failed to create cluster: %v", err)
    }
    
    // Ensure cleanup happens
    defer CleanupTestCluster(clusterName)
    
    // Check if cluster exists
    exists, err := ClusterExists(clusterName)
    if err != nil || !exists {
        log.Fatal("Cluster verification failed")
    }
    
    // Run your tests here...
    time.Sleep(2 * time.Second)
    
    // Cleanup all test clusters if needed
    CleanupAllTestClusters()
}
```
