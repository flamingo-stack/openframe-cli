Provides utilities for managing k3d Kubernetes test clusters, including creation, deletion, and cleanup operations with Docker resource management.

## Key Components

- **GenerateTestClusterName()** - Creates unique timestamped cluster names for testing
- **CreateTestCluster(name)** - Creates a k3d cluster with 1 agent node and 60s timeout
- **DeleteTestCluster(name)** - Removes a specific test cluster
- **ClusterExists(name)** - Checks if a cluster exists using k3d list command
- **StopTestCluster(name)** - Stops a running test cluster
- **CleanupTestCluster(name)** - Comprehensive cleanup including Docker resources
- **CleanupAllTestClusters()** - Removes all test-related clusters and Docker artifacts

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    // Generate unique cluster name
    clusterName := GenerateTestClusterName()
    fmt.Printf("Creating cluster: %s\n", clusterName)
    
    // Create test cluster
    if err := CreateTestCluster(clusterName); err != nil {
        log.Fatal(err)
    }
    
    // Check if cluster exists
    exists, err := ClusterExists(clusterName)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Cluster exists: %v\n", exists)
    
    // Cleanup when done
    defer CleanupTestCluster(clusterName)
    
    // Or cleanup all test clusters
    CleanupAllTestClusters()
}
```

The module handles Docker network and container cleanup automatically to prevent resource conflicts between test runs.
