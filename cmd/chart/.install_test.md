This test file validates the behavior and configuration of the Helm chart installation command for ArgoCD deployment.

## Key Components

- **TestInstallCommand**: Validates basic command structure (name, descriptions, RunE function)
- **TestInstallCommandFlags**: Tests flag existence, shorthand notation, and default values
- **TestInstallCommandHelp**: Verifies help text content and formatting
- **TestInstallCommandUsage**: Checks command usage string format
- **TestInstallCommandWithDryRun**: Tests dry-run flag parsing and extraction
- **TestInstallCommandFlagHandling**: Comprehensive flag combination testing
- **MockExecutor**: Mock implementation for testing command execution without side effects
- **TestRunInstallCommand**: Validates command execution structure and flag extraction

## Usage Example

```go
// Run specific test for command flags
func TestCommandFlags(t *testing.T) {
    cmd := getInstallCmd()
    
    // Test flag defaults
    forceDefault, _ := cmd.Flags().GetBool("force")
    assert.False(t, forceDefault)
    
    // Test flag extraction
    cmd.Flags().Set("dry-run", "true")
    flags, err := extractInstallFlags(cmd)
    assert.NoError(t, err)
    assert.True(t, flags.DryRun)
}
```
