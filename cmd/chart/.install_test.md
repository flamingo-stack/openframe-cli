This file contains comprehensive unit tests for the Helm chart installation command functionality in the openframe CLI tool.

## Key Components

- **TestInstallCommand**: Validates the basic structure and properties of the install command
- **TestInstallCommandFlags**: Tests command-line flag definitions, shortcuts, and default values
- **TestInstallCommandHelp**: Verifies help text content includes expected ArgoCD installation information
- **TestInstallCommandUsage**: Confirms command usage syntax
- **TestInstallCommandWithDryRun**: Tests dry-run flag parsing and extraction
- **TestInstallCommandFlagHandling**: Comprehensive table-driven tests for various flag combinations
- **MockExecutor**: Test double for command execution with configurable responses
- **TestRunInstallCommand**: Validates the main command execution function structure

## Usage Example

```go
// Running the flag handling test
func TestExample(t *testing.T) {
    cmd := getInstallCmd()
    
    // Set flags for testing
    cmd.Flags().Set("dry-run", "true")
    cmd.Flags().Set("force", "true")
    
    // Extract and validate flags
    flags, err := extractInstallFlags(cmd)
    assert.NoError(t, err)
    assert.True(t, flags.DryRun)
    assert.True(t, flags.Force)
}

// Using the mock executor
mockExec := NewMockExecutor()
mockExec.results["kubectl version"] = &executor.CommandResult{
    ExitCode: 0,
    Stdout:   "v1.28.0",
}
```

The tests ensure proper command configuration, flag handling, and help documentation for the ArgoCD installation workflow.
