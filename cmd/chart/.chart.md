Provides the main chart command for managing Helm charts and ArgoCD lifecycle operations in OpenFrame CLI.

## Key Components

- **GetChartCmd()**: Returns the root chart command with aliases and subcommands
- **PersistentPreRunE**: Validates prerequisites and displays UI context for subcommands
- **RunE**: Handles the case when no subcommand is provided, showing help output

## Usage Example

```go
// Add the chart command to the root CLI
rootCmd := &cobra.Command{Use: "openframe"}
rootCmd.AddCommand(chart.GetChartCmd())

// Users can then run:
// openframe chart install
// openframe c install  (using alias)
// openframe chart      (shows help)
```

The command provides ArgoCD installation and management capabilities, requiring an existing cluster created with `openframe cluster create`. It includes prerequisite checking through the installer and context-aware logo display for better user experience.
