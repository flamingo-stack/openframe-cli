Provides the main entry point for Helm chart management commands in the OpenFrame CLI. This module sets up the chart command group with ArgoCD installation capabilities and automatic prerequisite checking.

## Key Components

- **`GetChartCmd()`** - Returns the root chart command with subcommands and configuration
- **Command Structure** - Defines "chart" command with alias "c" for managing Helm charts
- **Prerequisites Integration** - Automatically checks and installs required dependencies before execution
- **UI Integration** - Displays OpenFrame logo and context-aware help

## Usage Example

```go
// Add chart command to CLI root
rootCmd := &cobra.Command{Use: "openframe"}
rootCmd.AddCommand(chart.GetChartCmd())

// Command usage from CLI:
// openframe chart install
// openframe c install my-cluster
```

The command implements a persistent pre-run hook that validates prerequisites and shows the logo for subcommands, while the main run function displays help when no subcommand is specified. The module expects an `getInstallCmd()` function to provide the install subcommand functionality.
