Test file for the chart command functionality, validating the structure and behavior of the chart CLI command.

## Key Components

- **`TestChartRootCommand`** - Main test function that validates the chart command structure
- **`init()`** - Initializes test mode using testutil
- **`GetChartCmd()`** - Function being tested (imported from chart package)

## Usage Example

```go
// Run the test
go test ./chart_test.go

// The test validates that the chart command:
// - Has correct name "chart"
// - Contains proper short/long descriptions
// - Has a RunE function for execution
// - Includes expected help text about Helm chart management

func TestChartRootCommand(t *testing.T) {
    cmd := GetChartCmd()
    assert.Equal(t, "chart", cmd.Name())
    assert.Contains(t, cmd.Short, "Manage Helm charts")
}
```

This test ensures the chart command is properly configured with appropriate metadata and functionality for Helm chart lifecycle management operations.
