This file implements the `install` subcommand for the chart package, providing functionality to install ArgoCD and app-of-apps configurations on Kubernetes clusters with various deployment modes and customization options.

## Key Components

- **`getInstallCmd()`** - Creates and configures the Cobra command for chart installation
- **`runInstallCommand()`** - Main command execution handler that processes flags and delegates to services
- **`InstallFlags`** - Struct containing all configuration flags for the installation process
- **`extractInstallFlags()`** - Extracts and validates command line flags with deployment mode validation
- **`getVerboseFlag()`** - Retrieves verbose flag with fallback logic from root or current command
- **`addInstallFlags()`** - Defines all available command line flags and their default values

## Usage Example

```go
// Get the install command and add to parent command
installCmd := getInstallCmd()
parentCmd.AddCommand(installCmd)

// The command supports various installation modes:
// openframe chart install my-cluster --deployment-mode=oss-tenant
// openframe chart install --dry-run --verbose
// openframe chart install --github-branch develop --force

// Flag extraction example
flags, err := extractInstallFlags(cmd)
if err != nil {
    return fmt.Errorf("failed to extract flags: %w", err)
}

// Create installation request
req := types.InstallationRequest{
    Args:           []string{"cluster-name"},
    Force:          flags.Force,
    DryRun:         flags.DryRun,
    DeploymentMode: flags.DeploymentMode,
    // ... other fields
}
```

The command supports three deployment modes (oss-tenant, saas-tenant, saas-shared) and includes validation to ensure non-interactive mode requires a deployment mode specification.
