Implements the `chart install` subcommand for the OpenFrame CLI, responsible for installing ArgoCD and app-of-apps on Kubernetes clusters with configurable deployment modes.

## Key Components

- **`getInstallCmd()`** - Returns the install subcommand with usage examples and flag definitions
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub repo/branch, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command flags with business rule validation
- **`addInstallFlags()`** - Defines all available command flags and their defaults

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example flag extraction
flags, err := extractInstallFlags(cmd)
if err != nil {
    return err
}

// Create installation request
req := types.InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          flags.Force,
    DryRun:         flags.DryRun,
    DeploymentMode: flags.DeploymentMode,
    GitHubRepo:     flags.GitHubRepo,
    GitHubBranch:   flags.GitHubBranch,
}

// Execute installation
err = services.InstallChartsWithConfig(req)
```

The command supports interactive and non-interactive modes, with validation ensuring non-interactive mode requires a deployment mode selection.
