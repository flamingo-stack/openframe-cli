A command-line interface handler for installing ArgoCD and app-of-apps charts on Kubernetes clusters. This file implements the `install` subcommand with comprehensive flag handling, validation, and support for multiple deployment modes.

## Key Components

- **`getInstallCmd()`** - Creates and configures the install cobra command with usage examples
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub settings, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command-line flags with deployment mode validation
- **`addInstallFlags()`** - Defines all available command flags with defaults
- **`getVerboseFlag()`** - Utility to retrieve verbose flag with fallback logic

## Usage Example

```go
// Create install command
installCmd := getInstallCmd()

// Example flag extraction
flags, err := extractInstallFlags(cmd)
if err != nil {
    return err
}

// Installation request setup
req := types.InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          flags.Force,
    DryRun:         flags.DryRun,
    DeploymentMode: "oss-tenant",
    NonInteractive: true,
}

// Execute installation
err = services.InstallChartsWithConfig(req)
```

The command supports interactive and non-interactive modes, with validation ensuring deployment mode is specified for automated installations.
