This file implements the chart install command for the OpenFrame CLI, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters with configurable deployment modes.

## Key Components

- **`getInstallCmd()`** - Creates the cobra command for chart installation with comprehensive help text and examples
- **`runInstallCommand()`** - Main command handler that processes arguments and delegates to the installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub repo/branch, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command flags with deployment mode validation
- **`addInstallFlags()`** - Defines all available command line flags
- **`getVerboseFlag()`** - Utility to retrieve verbose flag with fallback logic

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example command usage:
// openframe chart install my-cluster --force --deployment-mode=oss-tenant
// openframe chart install --dry-run --github-branch=develop
// openframe chart install --non-interactive --deployment-mode=saas-shared

// The command validates deployment modes and handles both interactive 
// and non-interactive installation flows
```

The command supports three deployment modes (oss-tenant, saas-tenant, saas-shared) and includes validation to ensure non-interactive mode requires a specified deployment mode.
