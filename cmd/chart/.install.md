Implements the CLI command for installing ArgoCD and app-of-apps charts on Kubernetes clusters with support for multiple deployment modes and interactive/non-interactive execution.

## Key Components

- **getInstallCmd()** - Creates and configures the Cobra command with usage examples and flags
- **runInstallCommand()** - Main execution handler that processes flags and delegates to installation service
- **InstallFlags** - Struct containing all configuration options for chart installation
- **extractInstallFlags()** - Validates and extracts command line flags with deployment mode validation
- **addInstallFlags()** - Defines all available command line flags and their defaults
- **getVerboseFlag()** - Retrieves verbose flag with fallback logic from root or current command

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example command executions:
// openframe chart install                                    # Interactive mode
// openframe chart install my-cluster                        # Install on specific cluster  
// openframe chart install --deployment-mode=oss-tenant     # Skip deployment selection
// openframe chart install --force --dry-run                 # Force with preview mode

// The command supports flags for:
// - GitHub repository and branch configuration
// - Certificate directory specification
// - Deployment modes (oss-tenant, saas-tenant, saas-shared)
// - Non-interactive execution for CI/CD pipelines
```

The command installs ArgoCD version 8.2.7 with custom values and configurable app-of-apps from GitHub repositories, with automatic certificate regeneration and comprehensive validation.
