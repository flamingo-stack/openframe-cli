Implements the Helm chart installation command for the OpenFrame CLI, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters with various deployment modes.

## Key Components

- **getInstallCmd()**: Creates the main install subcommand with usage examples and help text
- **runInstallCommand()**: Executes the installation process by extracting flags and calling the installation service
- **InstallFlags**: Struct containing all command-line flags (force, dry-run, GitHub repo/branch, deployment mode, etc.)
- **extractInstallFlags()**: Validates and extracts command flags with deployment mode validation
- **addInstallFlags()**: Defines all available command-line flags for the install command

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// The command supports various execution modes:
// openframe chart install                                    # Interactive mode
// openframe chart install my-cluster                        # Install on specific cluster  
// openframe chart install --deployment-mode=oss-tenant     # Skip deployment selection
// openframe chart install --dry-run                        # Preview installation
// openframe chart install --force --non-interactive        # CI/CD mode

// Add to parent command
parentCmd.AddCommand(installCmd)
```

The command validates deployment modes (oss-tenant, saas-tenant, saas-shared) and requires deployment mode when using non-interactive mode. It integrates with the services layer for actual chart installation and uses shared error handling for consistent error display.
