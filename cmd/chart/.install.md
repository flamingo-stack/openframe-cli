This file implements the Helm chart installation command for the OpenFrame CLI, specifically handling ArgoCD and app-of-apps deployment to Kubernetes clusters.

## Key Components

- **`getInstallCmd()`** - Creates the cobra command with usage patterns and help text
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options
- **`extractInstallFlags()`** - Validates and extracts command-line flags with business rule validation
- **`addInstallFlags()`** - Defines all available command-line flags and their defaults
- **`getVerboseFlag()`** - Utility for retrieving verbose flag from command hierarchy

## Usage Example

```go
// Command creation
cmd := getInstallCmd()

// Example flag extraction
flags, err := extractInstallFlags(cmd)
if err != nil {
    return fmt.Errorf("flag validation failed: %w", err)
}

// Installation request setup
req := types.InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          flags.Force,
    DryRun:         flags.DryRun,
    DeploymentMode: flags.DeploymentMode,
    GitHubRepo:     flags.GitHubRepo,
    GitHubBranch:   flags.GitHubBranch,
}

// Execute installation
err = services.InstallChartsWithConfig(req)
```

The command supports both interactive and non-interactive modes, with validation ensuring deployment mode is specified for CI/CD scenarios.
