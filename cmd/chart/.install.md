This file implements the `chart install` command for the openframe CLI, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters with customizable deployment modes and configuration options.

## Key Components

- **`getInstallCmd()`** - Creates the cobra command for chart installation with usage examples and help text
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub settings, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command flags with deployment mode validation
- **`addInstallFlags()`** - Configures all available command-line flags for the install command
- **`getVerboseFlag()`** - Utility to retrieve verbose flag with fallback logic

## Usage Example

```go
// Command usage examples:
// Interactive mode (default)
openframe chart install

// Install on specific cluster
openframe chart install my-cluster

// Non-interactive with deployment mode
openframe chart install --deployment-mode=oss-tenant --non-interactive

// Dry run with custom GitHub branch
openframe chart install --dry-run --github-branch develop

// Force installation with custom repository
openframe chart install --force --github-repo https://github.com/custom/repo
```

The command supports three deployment modes (oss-tenant, saas-tenant, saas-shared) and includes validation to ensure proper flag combinations for automated deployments.
