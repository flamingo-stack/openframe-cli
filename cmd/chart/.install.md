Implements the chart installation command for the OpenFrame CLI, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters with various deployment modes and configuration options.

## Key Components

- **getInstallCmd()** - Returns the cobra command for chart installation with comprehensive flag support
- **runInstallCommand()** - Main execution handler that processes flags and delegates to installation service
- **InstallFlags** - Struct containing all installation configuration flags (force, dry-run, GitHub settings, etc.)
- **extractInstallFlags()** - Extracts and validates command flags with deployment mode validation
- **addInstallFlags()** - Configures all available command line flags
- **getVerboseFlag()** - Utility to extract verbose flag with fallback logic

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Execute with various options
// Interactive mode: openframe chart install
// Specific cluster: openframe chart install my-cluster
// Non-interactive: openframe chart install --deployment-mode=oss-tenant --non-interactive

// Extract flags for processing
flags, err := extractInstallFlags(cmd)
if err != nil {
    return err
}

// Create installation request
req := types.InstallationRequest{
    Args:           args,
    Force:          flags.Force,
    DeploymentMode: flags.DeploymentMode,
    NonInteractive: flags.NonInteractive,
}
```

The command supports multiple deployment modes (oss-tenant, saas-tenant, saas-shared) and validates that non-interactive mode requires a deployment mode to be specified.
