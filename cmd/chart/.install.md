Implements the `install` subcommand for the chart package, providing a Cobra CLI interface to install ArgoCD and app-of-apps on Kubernetes clusters.

## Key Components

- **`getInstallCmd()`** - Creates and configures the main install command with usage examples
- **`runInstallCommand()`** - Handles command execution, validates flags, and delegates to installation services
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub settings, deployment modes, etc.)
- **`extractInstallFlags()`** - Extracts and validates command flags with business logic validation
- **`addInstallFlags()`** - Defines all available command-line flags and their defaults
- **`getVerboseFlag()`** - Utility to extract verbose flag with fallback handling

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// The command supports various usage patterns:
// openframe chart install my-cluster
// openframe chart install --deployment-mode=oss-tenant
// openframe chart install --dry-run --verbose

// Flags are automatically validated, including:
// - Valid deployment modes (oss-tenant, saas-tenant, saas-shared)
// - Non-interactive mode requires deployment-mode
// - GitHub repository and branch configuration
```
