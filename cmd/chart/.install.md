This file implements the `install` command for the OpenFrame CLI chart package, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters.

## Key Components

- **`getInstallCmd()`** - Creates and configures the cobra command for chart installation
- **`runInstallCommand()`** - Main command execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub settings, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command-line flags with built-in validation for deployment modes
- **`getVerboseFlag()`** - Utility to retrieve verbose flag from command hierarchy with fallback
- **`addInstallFlags()`** - Configures all available command-line flags for the install command

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// The command supports various usage patterns:
// openframe chart install                                    # Interactive mode
// openframe chart install my-cluster                        # Specific cluster
// openframe chart install --deployment-mode=oss-tenant     # Skip deployment selection
// openframe chart install --force --dry-run                # Force with preview
// openframe chart install --github-branch=develop --non-interactive
```
