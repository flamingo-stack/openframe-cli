Implements the `chart install` subcommand for installing ArgoCD and app-of-apps on Kubernetes clusters. Provides both interactive and non-interactive modes with comprehensive flag validation.

## Key Components

- **`getInstallCmd()`** - Creates the cobra command with usage examples and flag definitions
- **`runInstallCommand()`** - Main command handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration flags
- **`extractInstallFlags()`** - Extracts and validates command flags with deployment mode validation
- **`addInstallFlags()`** - Defines all available command-line flags
- **`getVerboseFlag()`** - Retrieves verbose flag with fallback logic

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example command usage:
// openframe chart install my-cluster --force --dry-run
// openframe chart install --deployment-mode=oss-tenant --non-interactive
// openframe chart install --github-branch develop --cert-dir /path/to/certs

// The command supports multiple deployment modes:
// - oss-tenant
// - saas-tenant  
// - saas-shared

// Non-interactive mode requires deployment-mode to be specified
```

The command installs ArgoCD (v8.2.7) and app-of-apps from a configurable GitHub repository, with automatic certificate regeneration and flexible deployment options.
