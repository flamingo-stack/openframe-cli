Implements the `chart install` command for deploying ArgoCD and app-of-apps to Kubernetes clusters. This command provides both interactive and non-interactive modes with configurable deployment options.

## Key Components

- **`getInstallCmd()`** - Creates the cobra command with usage examples and flag definitions
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options
- **`extractInstallFlags()`** - Extracts and validates command-line flags with deployment mode validation
- **`addInstallFlags()`** - Defines all available command-line flags for installation

## Usage Example

```go
// Command usage examples:
// openframe chart install                                    # Interactive mode
// openframe chart install my-cluster                        # Install on specific cluster  
// openframe chart install --deployment-mode=oss-tenant     # Skip deployment selection
// openframe chart install --deployment-mode=saas-shared --non-interactive  # Full CI/CD mode
// openframe chart install --github-branch develop --dry-run # Test with develop branch

// The command supports flags for:
cmd.Flags().BoolP("force", "f", false, "Force installation even if charts already exist")
cmd.Flags().Bool("dry-run", false, "Show what would be installed without executing")
cmd.Flags().String("deployment-mode", "", "oss-tenant, saas-tenant, or saas-shared")
cmd.Flags().Bool("non-interactive", false, "Skip all prompts")
```
