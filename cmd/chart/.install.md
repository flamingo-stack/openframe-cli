Implements the CLI install command for deploying ArgoCD and app-of-apps on Kubernetes clusters with configurable deployment modes and GitHub repository sources.

## Key Components

- **`getInstallCmd()`** - Returns the main install subcommand with usage documentation and examples
- **`runInstallCommand()`** - Executes the installation process by extracting flags and calling the installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub settings, etc.)
- **`extractInstallFlags()`** - Extracts and validates command flags with deployment mode validation
- **`getVerboseFlag()`** - Retrieves verbose flag with fallback logic from root or current command
- **`addInstallFlags()`** - Configures all available command-line flags for the install command

## Usage Example

```go
// Get the install command for CLI registration
installCmd := getInstallCmd()

// The command supports various execution modes:
// Interactive: openframe chart install
// Cluster-specific: openframe chart install my-cluster  
// CI/CD mode: openframe chart install --deployment-mode=oss-tenant --non-interactive
// Custom repo: openframe chart install --github-repo=https://github.com/custom/repo --github-branch=develop

// Flag validation ensures:
// - Valid deployment modes: oss-tenant, saas-tenant, saas-shared
// - Non-interactive mode requires deployment-mode
// - Proper GitHub repository URL format
```

The command installs ArgoCD (v8.2.7) with custom values and configurable app-of-apps from GitHub repositories, supporting both interactive and automated deployment workflows.
