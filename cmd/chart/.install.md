Implements the chart install command for the OpenFrame CLI, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters. The command supports both interactive and non-interactive modes with configurable deployment options.

## Key Components

- **getInstallCmd()** - Creates the install cobra command with usage examples and help text
- **runInstallCommand()** - Main command handler that processes flags and delegates to installation service
- **InstallFlags** - Struct containing all installation configuration options
- **extractInstallFlags()** - Extracts and validates command-line flags with deployment mode validation
- **getVerboseFlag()** - Retrieves verbose flag with fallback logic from root or current command
- **addInstallFlags()** - Defines all available command-line flags for the install command

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example flag extraction
flags, err := extractInstallFlags(cmd)
if err != nil {
    return err
}

// Create installation request
req := types.InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          flags.Force,
    DeploymentMode: flags.DeploymentMode,
    GitHubRepo:     flags.GitHubRepo,
    NonInteractive: flags.NonInteractive,
}

// Execute installation
err = services.InstallChartsWithConfig(req)
```

The command supports multiple deployment modes (oss-tenant, saas-tenant, saas-shared) and validates that non-interactive mode requires a deployment mode to be specified.
