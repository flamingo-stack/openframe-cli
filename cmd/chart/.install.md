A CLI subcommand that handles the installation of ArgoCD and app-of-apps on Kubernetes clusters with configurable deployment modes and GitHub repository integration.

## Key Components

- **`getInstallCmd()`** - Returns the install cobra command with comprehensive help text and usage examples
- **`runInstallCommand()`** - Main command execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration flags (force, dry-run, GitHub settings, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command flags with business rule validation
- **`getVerboseFlag()`** - Utility to extract verbose flag with fallback logic
- **`addInstallFlags()`** - Configures all available command flags for the install command

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example flag extraction and validation
flags, err := extractInstallFlags(cmd)
if err != nil {
    return err
}

// Build installation request
req := types.InstallationRequest{
    Args:           args,
    Force:          flags.Force,
    DryRun:         flags.DryRun,
    GitHubRepo:     flags.GitHubRepo,
    GitHubBranch:   flags.GitHubBranch,
    DeploymentMode: flags.DeploymentMode,
    NonInteractive: flags.NonInteractive,
}

// Execute installation
err = services.InstallChartsWithConfig(req)
```

The command supports three deployment modes (oss-tenant, saas-tenant, saas-shared), validates non-interactive mode requires deployment mode specification, and integrates with shared error handling for consistent CLI experience.
