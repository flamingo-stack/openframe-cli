Provides the `install` subcommand for the chart package, handling ArgoCD and app-of-apps installation on Kubernetes clusters with support for multiple deployment modes and configuration options.

## Key Components

- **getInstallCmd()** - Creates the install cobra command with usage examples and flag definitions
- **runInstallCommand()** - Main command handler that processes flags and delegates to installation service
- **InstallFlags** - Struct containing all installation configuration flags
- **extractInstallFlags()** - Extracts and validates command flags with deployment mode validation
- **getVerboseFlag()** - Helper to retrieve verbose flag from command hierarchy
- **addInstallFlags()** - Defines all available command-line flags for installation

## Usage Example

```go
// Get the install command for use in CLI
cmd := getInstallCmd()

// Example installation request structure
req := types.InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          true,
    DryRun:         false,
    Verbose:        true,
    GitHubRepo:     "https://github.com/flamingo-stack/openframe-oss-tenant",
    GitHubBranch:   "main",
    DeploymentMode: "oss-tenant",
    NonInteractive: false,
}

// The command supports various deployment modes
// Valid modes: "oss-tenant", "saas-tenant", "saas-shared"
```

The command installs ArgoCD (version 8.2.7) and configures app-of-apps patterns, with automatic certificate regeneration and support for both interactive and CI/CD automation modes.
