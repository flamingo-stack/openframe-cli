Implements the `chart install` subcommand for installing ArgoCD and app-of-apps on Kubernetes clusters. This command provides both interactive and non-interactive modes with comprehensive flag validation.

## Key Components

- **`getInstallCmd()`** - Creates the Cobra command with usage, examples, and flag definitions
- **`runInstallCommand()`** - Main command execution handler that processes flags and calls installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub settings, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command flags with business rule enforcement
- **`addInstallFlags()`** - Defines all available command flags with descriptions and defaults

## Usage Example

```go
// Basic installation
openframe chart install

// Install on specific cluster with custom branch
openframe chart install my-cluster --github-branch develop

// Non-interactive mode for CI/CD
openframe chart install --deployment-mode=saas-shared --non-interactive

// Dry run to preview installation
openframe chart install --dry-run --force
```

The command supports three deployment modes (`oss-tenant`, `saas-tenant`, `saas-shared`) and includes validation to ensure `--deployment-mode` is specified when using `--non-interactive` mode.
