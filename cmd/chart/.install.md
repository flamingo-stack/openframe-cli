Implements the chart install subcommand for the OpenFrame CLI, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters with various configuration options.

## Key Components

- **`getInstallCmd()`** - Returns configured Cobra command for chart installation
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to services
- **`InstallFlags`** - Struct containing all installation configuration flags
- **`extractInstallFlags()`** - Extracts and validates command-line flags with business logic validation
- **`addInstallFlags()`** - Configures available command-line flags for the install command
- **`getVerboseFlag()`** - Utility to retrieve verbose flag with fallback logic

## Usage Example

```go
// The command supports multiple execution modes:

// Interactive mode (default)
// openframe chart install

// Specific cluster installation
// openframe chart install my-cluster

// CI/CD mode with all options
// openframe chart install --deployment-mode=saas-shared --non-interactive --dry-run

// Custom repository and branch
// openframe chart install --github-repo=https://github.com/custom/repo --github-branch=develop

// The command validates deployment modes (oss-tenant, saas-tenant, saas-shared)
// and enforces that non-interactive mode requires a deployment mode
```
