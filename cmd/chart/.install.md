This file implements the `install` subcommand for the chart package, providing functionality to install ArgoCD and app-of-apps configurations on Kubernetes clusters.

## Key Components

- **`getInstallCmd()`** - Creates and configures the cobra command for chart installation
- **`runInstallCommand()`** - Main command execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration flags (force, dry-run, GitHub repo/branch, deployment mode, etc.)
- **`extractInstallFlags()`** - Flag parsing and validation logic with deployment mode validation
- **`getVerboseFlag()`** - Utility to extract verbose flag with fallback mechanisms
- **`addInstallFlags()`** - Defines all available command-line flags for the install command

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// The command supports various usage patterns:
// Interactive mode: openframe chart install
// Specific cluster: openframe chart install my-cluster  
// CI/CD mode: openframe chart install --deployment-mode=oss-tenant --non-interactive

// Flag validation ensures deployment modes are valid
// and non-interactive mode requires deployment-mode to be specified
```

The command installs ArgoCD (version 8.2.7) and configurable app-of-apps from GitHub repositories, with support for different deployment modes (oss-tenant, saas-tenant, saas-shared) and both interactive and non-interactive operation modes.
