This file implements the `install` subcommand for the chart module, which installs ArgoCD and app-of-apps on Kubernetes clusters. It provides comprehensive flag handling, validation, and both interactive and non-interactive modes for CI/CD environments.

## Key Components

- **`getInstallCmd()`** - Creates and configures the install cobra command with help text and examples
- **`runInstallCommand()`** - Main command execution handler that processes flags and calls installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub repo/branch, etc.)
- **`extractInstallFlags()`** - Extracts and validates command line flags with deployment mode validation
- **`getVerboseFlag()`** - Retrieves verbose flag from root or current command with fallback
- **`addInstallFlags()`** - Defines all available command line flags for the install command

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example flag extraction and validation
flags, err := extractInstallFlags(cmd)
if err != nil {
    return fmt.Errorf("flag validation failed: %w", err)
}

// Build installation request
req := types.InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          flags.Force,
    DryRun:         flags.DryRun,
    DeploymentMode: "oss-tenant",
    NonInteractive: true,
}

// Execute installation
err = services.InstallChartsWithConfig(req)
```

The command supports multiple deployment modes (oss-tenant, saas-tenant, saas-shared) and includes validation to ensure non-interactive mode has the required deployment mode specified.
