Implements the `install` subcommand for the OpenFrame CLI chart command, handling ArgoCD and app-of-apps installation on Kubernetes clusters with configurable deployment modes and GitHub integration.

## Key Components

- **getInstallCmd()** - Creates and configures the Cobra command for chart installation
- **runInstallCommand()** - Main command handler that processes flags and executes installation
- **InstallFlags** - Struct containing all configuration options for installation
- **extractInstallFlags()** - Extracts and validates command line flags with deployment mode validation
- **getVerboseFlag()** - Retrieves verbose flag from command hierarchy with fallback logic
- **addInstallFlags()** - Adds all supported flags to the command definition

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example flag configuration
cmd.Flags().String("deployment-mode", "", "oss-tenant, saas-tenant, or saas-shared")
cmd.Flags().Bool("non-interactive", false, "Skip all prompts")
cmd.Flags().String("github-repo", "https://github.com/flamingo-stack/openframe-oss-tenant", "Repository URL")

// Install request structure
req := types.InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          true,
    DryRun:         false,
    DeploymentMode: "oss-tenant",
    NonInteractive: true,
}
```

The command supports three deployment modes (oss-tenant, saas-tenant, saas-shared) and includes validation to ensure non-interactive mode requires a deployment mode specification.
