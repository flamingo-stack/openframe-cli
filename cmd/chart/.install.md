Implements the `install` subcommand for the OpenFrame CLI chart module, providing functionality to install ArgoCD and app-of-apps on Kubernetes clusters with various deployment modes and configuration options.

## Key Components

- **`getInstallCmd()`** - Creates and configures the Cobra command for chart installation
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options (force, dry-run, GitHub settings, deployment mode, etc.)
- **`extractInstallFlags()`** - Extracts and validates command-line flags with deployment mode validation
- **`addInstallFlags()`** - Defines all available command-line flags for the install command
- **`getVerboseFlag()`** - Utility to extract verbose flag with fallback logic

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// The command supports various usage patterns:
// Interactive mode: openframe chart install
// Specific cluster: openframe chart install my-cluster  
// CI/CD mode: openframe chart install --deployment-mode=saas-shared --non-interactive
// Custom branch: openframe chart install --github-branch develop --force

// Flags are automatically validated, including:
// - Valid deployment modes (oss-tenant, saas-tenant, saas-shared)
// - Required deployment mode when using non-interactive
// - GitHub repository and branch configuration
```
