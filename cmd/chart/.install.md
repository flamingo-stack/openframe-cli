Implements the `chart install` subcommand for deploying ArgoCD and app-of-apps patterns to Kubernetes clusters with configurable deployment modes and interactive/non-interactive installation options.

## Key Components

- **`getInstallCmd()`** - Creates the cobra command with usage examples and flag definitions
- **`runInstallCommand()`** - Main execution handler that processes flags and delegates to installation service
- **`InstallFlags`** - Struct containing all installation configuration options
- **`extractInstallFlags()`** - Parses and validates command flags with deployment mode validation
- **`getVerboseFlag()`** - Retrieves verbose flag with fallback logic from root or current command
- **`addInstallFlags()`** - Defines all available command flags and their default values

## Usage Example

```go
// Create the install command
installCmd := getInstallCmd()

// Example command execution scenarios:
// openframe chart install                                    # Interactive mode
// openframe chart install my-cluster                        # Target specific cluster  
// openframe chart install --deployment-mode=oss-tenant     # Skip deployment selection
// openframe chart install --dry-run --verbose              # Preview installation
// openframe chart install --non-interactive --deployment-mode=saas-shared  # CI/CD mode

// The command supports these flags:
// --force/-f: Force installation over existing charts
// --dry-run: Preview without executing
// --github-repo: Custom repository URL
// --github-branch: Repository branch (default: main)
// --cert-dir: Certificate directory path
// --deployment-mode: oss-tenant, saas-tenant, or saas-shared
// --non-interactive: Skip prompts, requires deployment-mode
```
