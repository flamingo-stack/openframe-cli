Implements the "install" subcommand for deploying ArgoCD and app-of-apps charts to Kubernetes clusters. This command provides both interactive and non-interactive modes with comprehensive flag validation.

## Key Components

- **getInstallCmd()** - Creates the cobra command with usage examples and flag definitions
- **runInstallCommand()** - Main command execution handler that processes flags and delegates to installation service
- **InstallFlags** - Struct containing all installation configuration options (force, dry-run, GitHub settings, deployment mode)
- **extractInstallFlags()** - Extracts and validates command flags with deployment mode validation
- **addInstallFlags()** - Defines all available command flags with defaults

## Usage Example

```go
// Create the install subcommand
installCmd := getInstallCmd()

// Example flag extraction in a command handler
flags, err := extractInstallFlags(cmd)
if err != nil {
    return err
}

// Create installation request
req := types.InstallationRequest{
    Args:           args,
    Force:          flags.Force,
    DryRun:         flags.DryRun,
    DeploymentMode: flags.DeploymentMode,
    NonInteractive: flags.NonInteractive,
}

// Execute installation
err = services.InstallChartsWithConfig(req)
```

The command supports various deployment modes (oss-tenant, saas-tenant, saas-shared) and validates that non-interactive mode requires a deployment mode to be specified.
