Test file for the scaffold command functionality that validates command configuration, flag handling, and argument parsing.

## Key Components

- **TestGetScaffoldCmd**: Validates basic command properties including usage string, description, and function bindings
- **TestScaffoldCmd_FlagBinding**: Ensures all required flags are present with correct names
- **TestScaffoldCmd_FlagTypes**: Tests flag parsing and value retrieval for all supported flag types
- **TestScaffoldCmd_FlagDefaults**: Verifies default values for all command flags
- **TestScaffoldCmd_FlagToModelMapping**: Tests mapping between command flags and the ScaffoldFlags model
- **TestScaffoldCmd_ArgumentHandling**: Validates command argument validation logic
- **TestScaffoldCmd_Examples**: Checks that usage examples are properly included in help text
- **TestScaffoldCmd_UsageText**: Validates comprehensive help text content

## Usage Example

```go
func TestMyScaffoldFeature(t *testing.T) {
    cmd := getScaffoldCmd()
    
    // Test flag parsing
    err := cmd.ParseFlags([]string{
        "--port", "9090",
        "--namespace", "test-ns",
        "--skip-bootstrap",
    })
    require.NoError(t, err)
    
    // Verify parsed values
    port, _ := cmd.Flags().GetInt("port")
    assert.Equal(t, 9090, port)
    
    skipBootstrap, _ := cmd.Flags().GetBool("skip-bootstrap")
    assert.True(t, skipBootstrap)
}
```
