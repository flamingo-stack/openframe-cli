Provides the main development command group for local Kubernetes workflows, including traffic interception and live development tools. This command serves as a parent for development-related subcommands with integrated prerequisites checking.

## Key Components

- **GetDevCmd()** - Returns the configured dev command with subcommands
- **PersistentPreRunE** - Validates prerequisites for intercept and skaffold operations
- **Subcommands** - Integrates `intercept` and `skaffold` subcommands
- **Global flags** - Adds shared development workflow flags

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/dev"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the dev command group
    devCmd := dev.GetDevCmd()
    rootCmd.AddCommand(devCmd)
    
    // The dev command provides:
    // openframe dev                    # Shows help
    // openframe dev intercept my-service
    // openframe dev skaffold my-service
    
    rootCmd.Execute()
}
```

The command automatically validates that required tools (Telepresence, Skaffold) are available before executing subcommands and displays the application logo for better user experience.
