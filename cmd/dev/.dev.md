Entry point for the development tools command group, providing local Kubernetes workflows with traffic interception and live reloading capabilities.

## Key Components

- **GetDevCmd()** - Main command factory function that returns the configured dev command
- **PersistentPreRunE** - Middleware that handles logo display and prerequisite checks
- **Subcommands** - Integrates `intercept` and `skaffold` subcommands for development workflows

## Usage Example

```go
// Register the dev command with the root CLI
rootCmd.AddCommand(dev.GetDevCmd())

// The command supports these workflows:
// openframe dev intercept my-service    // Traffic interception
// openframe dev skaffold my-service     // Live reloading deployment
// openframe dev                         // Shows help and logo
```

The command automatically validates prerequisites for Telepresence (intercept) and Skaffold tools, displays the application logo for user experience, and provides both short alias (`d`) and full command access to development workflows.
