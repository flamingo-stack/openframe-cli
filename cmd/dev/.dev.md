Provides the main development command group for local Kubernetes workflows, supporting traffic interception and live reloading capabilities through Telepresence and Skaffold integration.

## Key Components

- **GetDevCmd()** - Returns the root `dev` command with subcommands for local development workflows
- **PersistentPreRunE** - Validates prerequisites for intercept and skaffold operations
- **Subcommands** - Includes `intercept` and `skaffold` commands via `getInterceptCmd()` and `getScaffoldCmd()`
- **Global flags** - Adds shared development flags through `models.AddGlobalFlags()`

## Usage Example

```go
// Get the development command group
devCmd := GetDevCmd()

// The command supports these usage patterns:
// openframe dev intercept my-service
// openframe dev skaffold my-cluster
// openframe dev  # shows help with logo

// Add to root command
rootCmd.AddCommand(devCmd)
```

The command automatically checks prerequisites for Telepresence (intercept) and Skaffold operations, displays the UI logo for subcommands, and provides aliases (`d`) for shorter CLI usage.
