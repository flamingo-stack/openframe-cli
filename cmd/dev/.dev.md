Provides the main `dev` command and orchestrates development workflow subcommands for local Kubernetes development with Telepresence and Skaffold integration.

## Key Components

- **`GetDevCmd()`** - Returns the root dev command with aliases, help text, and prerequisite checking
- **Subcommands** - Integrates `intercept` and `skaffold` commands via `getInterceptCmd()` and `getScaffoldCmd()`
- **Prerequisites** - Automatically validates tool requirements based on the subcommand being executed
- **Global flags** - Inherits shared configuration flags through `models.AddGlobalFlags()`

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/dev"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the dev command to your CLI
    rootCmd.AddCommand(dev.GetDevCmd())
    
    rootCmd.Execute()
}
```

The command supports:
```bash
openframe dev                          # Shows help and logo
openframe dev intercept my-service     # Traffic interception  
openframe dev skaffold my-cluster      # Development deployment
```
