Provides the main entry point for development workflow commands, exposing traffic interception and scaffolding tools for local Kubernetes development.

## Key Components

- **GetDevCmd()**: Returns the root development command with subcommands for `intercept` and `skaffold`
- **PersistentPreRunE**: Handles prerequisite checking and UI display logic for subcommands
- **Subcommands**: Adds intercept and skaffold commands via `getInterceptCmd()` and `getScaffoldCmd()`
- **Global Flags**: Integrates shared flags through `models.AddGlobalFlags()`

## Usage Example

```go
// Get the development command group
devCmd := GetDevCmd()

// The command supports aliases and provides help
// Available subcommands:
// - intercept: Traffic interception with Telepresence
// - skaffold: Development deployments with live reloading

// Example CLI usage:
// openframe dev intercept my-service
// openframe dev skaffold my-cluster
// openframe d intercept my-service  # using alias
```

The command group provides a unified interface for local development workflows, automatically checking prerequisites for Telepresence (intercept) and Skaffold operations, and displaying the OpenFrame logo for better user experience.
