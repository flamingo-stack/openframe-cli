Provides the main development tools command group for the OpenFrame CLI, organizing local Kubernetes development workflows including traffic interception and live reloading capabilities.

## Key Components

- **GetDevCmd()**: Returns the root `dev` command with `intercept` and `skaffold` subcommands
- **PersistentPreRunE**: Handles prerequisite validation and UI logo display for subcommands
- **Subcommands**: `intercept` for traffic interception and `skaffold` for development deployments
- **Global flags**: Inherits shared configuration flags via `models.AddGlobalFlags()`

## Usage Example

```go
// Get the dev command with all subcommands
devCmd := GetDevCmd()

// Add to root CLI command
rootCmd.AddCommand(devCmd)

// Example CLI usage:
// openframe dev intercept my-service
// openframe dev skaffold my-cluster
// openframe dev --help
```

The command automatically validates prerequisites (Telepresence for intercept, Skaffold tooling for skaffold) and provides a consistent UI experience with logo display. When called without subcommands, it displays help information.
