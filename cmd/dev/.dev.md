This file provides the main dev command for local Kubernetes development workflows, integrating Telepresence traffic interception and Skaffold deployment capabilities.

## Key Components

- **GetDevCmd()** - Returns the root `dev` command with subcommands for local development
- **PersistentPreRunE** - Handles prerequisite checks for intercept and skaffold operations
- **Subcommands** - Integrates intercept and skaffold commands via `getInterceptCmd()` and `getScaffoldCmd()`
- **Global Flags** - Adds consistent flags across dev subcommands using `models.AddGlobalFlags()`

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/dev"
    "github.com/spf13/cobra"
)

// Add dev command to root CLI
rootCmd := &cobra.Command{Use: "openframe"}
rootCmd.AddCommand(dev.GetDevCmd())

// Available subcommands:
// openframe dev intercept my-service
// openframe dev skaffold my-service
// openframe d intercept my-service  (using alias)
```

The command automatically checks prerequisites for Telepresence and Skaffold tools before executing subcommands, and displays help when called without arguments.
