This file provides the main entry point for development workflow commands, offering tools for local Kubernetes development with Telepresence and Skaffold integration.

## Key Components

- **`GetDevCmd()`** - Returns the root dev command with subcommands for intercept and skaffold operations
- **Subcommands** - Integrates intercept and skaffold commands for traffic interception and live reloading
- **Prerequisites checking** - Validates required tools are installed before executing subcommands
- **Global flags** - Adds shared configuration flags from the models package

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/dev"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the dev command group
    devCmd := dev.GetDevCmd()
    rootCmd.AddCommand(devCmd)
    
    rootCmd.Execute()
}
```

The command supports aliases and provides help text with examples:
```bash
openframe dev intercept my-service
openframe d skaffold my-service
```
