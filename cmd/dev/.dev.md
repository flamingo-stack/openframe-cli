This file defines the main `dev` command for the OpenFrame CLI, providing local Kubernetes development tools including traffic interception and live reloading capabilities.

## Key Components

- **`GetDevCmd()`** - Returns the root `dev` command with aliases and subcommands
- **Subcommands Integration** - Incorporates `intercept` and `skaffold` subcommands
- **Prerequisites Validation** - Automatically checks tool requirements before command execution
- **Global Flags Support** - Adds shared flags via `models.AddGlobalFlags()`

## Usage Example

```go
// Get the dev command with all subcommands
devCmd := GetDevCmd()

// The command supports aliases and includes help text
// Command structure:
//   openframe dev intercept my-service
//   openframe dev skaffold my-service
//   openframe d intercept my-service  // using alias

// Prerequisites are automatically checked:
// - Telepresence for intercept commands  
// - Skaffold tools for skaffold commands

// When run without subcommands, displays help and logo
```

The command implements persistent pre-run hooks that validate prerequisites based on the specific subcommand being executed, ensuring users have the necessary tools (Telepresence, Skaffold) before attempting development workflows.
