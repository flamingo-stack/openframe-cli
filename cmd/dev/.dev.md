This file provides the main entry point for development workflow commands in the OpenFrame CLI, offering local Kubernetes development tools including traffic interception and live reloading capabilities.

## Key Components

- **`GetDevCmd()`**: Returns the root development command with subcommands for intercept and skaffold operations
- **Subcommands**: Integrates `intercept` (traffic interception) and `skaffold` (live reloading) commands
- **Prerequisites checking**: Validates required tools are installed before command execution
- **Global flags**: Adds shared configuration flags using the models package

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/dev"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the dev command group
    devCmd := dev.GetDevCmd()
    rootCmd.AddCommand(devCmd)
    
    // Available commands:
    // openframe dev intercept my-service
    // openframe dev skaffold my-cluster
    // openframe dev (shows help)
    
    rootCmd.Execute()
}
```

The command automatically checks prerequisites for Telepresence (intercept) and Skaffold tools, displays the OpenFrame logo for context, and provides helpful usage examples in its long description.
