A command factory that creates the main `dev` command with development workflow subcommands for Kubernetes local development using Telepresence and Skaffold.

## Key Components

- **GetDevCmd()** - Returns the configured `dev` command with intercept and skaffold subcommands
- **PersistentPreRunE** - Validates prerequisites for intercept (Telepresence) and skaffold operations
- **Subcommands** - Adds intercept and skaffold development workflow commands
- **Global flags** - Inherits shared configuration flags from the models package

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/dev"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the dev command to your CLI
    devCmd := dev.GetDevCmd()
    rootCmd.AddCommand(devCmd)
    
    // Users can now run:
    // openframe dev                    # Shows help
    // openframe dev intercept my-service
    // openframe dev skaffold my-cluster
    rootCmd.Execute()
}
```

The command provides a development-focused interface for local Kubernetes workflows, automatically checking that required tools like Telepresence and Skaffold are available before executing subcommands.
