Core command factory that creates the `dev` command for local Kubernetes development workflows with Telepresence traffic interception and Skaffold deployment capabilities.

## Key Components

- **`GetDevCmd()`** - Main factory function that returns configured cobra command
- **PersistentPreRunE hook** - Handles logo display and prerequisite validation
- **Subcommands** - `intercept` for traffic interception and `skaffold` for live reloading
- **Global flags integration** - Uses shared models for consistent flag handling

## Usage Example

```go
// Add dev command to root CLI
rootCmd := &cobra.Command{Use: "openframe"}
devCmd := dev.GetDevCmd()
rootCmd.AddCommand(devCmd)

// Command structure created:
// openframe dev                    # Shows help
// openframe dev intercept service  # Traffic interception  
// openframe dev skaffold cluster   # Live deployment
```

The command automatically validates Telepresence prerequisites for intercept operations and Skaffold prerequisites for deployment workflows, ensuring developers have the required tools installed before execution.
