A command builder for the development tools module that creates a CLI command group for local Kubernetes development workflows. Provides the root `dev` command with traffic interception and live reloading capabilities.

## Key Components

- **GetDevCmd()**: Main function that returns a configured Cobra command for development tools
- **PersistentPreRunE**: Hook that checks prerequisites and displays UI logos for subcommands
- **RunE**: Handler that displays help when no subcommand is provided
- **Subcommands**: Intercept and Skaffold commands for development workflows

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/dev"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the dev command group
    devCmd := dev.GetDevCmd()
    rootCmd.AddCommand(devCmd)
    
    // This enables commands like:
    // openframe dev intercept my-service
    // openframe dev skaffold my-service
    // openframe d intercept my-service (using alias)
    
    rootCmd.Execute()
}
```

The command supports both Telepresence traffic interception and Skaffold live reloading workflows, with prerequisite validation and context-aware UI display.
