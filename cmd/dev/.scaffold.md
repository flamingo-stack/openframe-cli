Provides the `skaffold` subcommand for the development CLI, enabling live-reloading development environments with Kubernetes integration. This command orchestrates the setup of development infrastructure including cluster bootstrapping and Skaffold deployment configuration.

## Key Components

- **getScaffoldCmd()** - Creates and configures the `skaffold` cobra command with flags and help text
- **runScaffold()** - Executes the scaffold workflow by coordinating with the scaffold service
- **ScaffoldFlags** - Configuration structure for scaffold-specific options like port, namespace, and sync directories
- **Command Integration** - Inherits global flags (verbose, dry-run) from parent dev command

## Usage Example

```go
// Get the scaffold command for registration
scaffoldCmd := getScaffoldCmd()

// The command supports various deployment scenarios:
// openframe dev skaffold                    # Interactive setup
// openframe dev skaffold my-cluster         # Named cluster
// openframe dev skaffold --port 8080        # Custom port
// openframe dev skaffold --namespace dev    # Specific namespace
// openframe dev skaffold --skip-bootstrap   # Skip cluster setup

// Command execution flow:
// 1. Parse arguments and flags
// 2. Extract global flags (verbose, dry-run)
// 3. Create executor and scaffold service
// 4. Run complete scaffold workflow
```

The command provides comprehensive development environment setup with features like prerequisites validation, cluster bootstrapping with development settings, live code reloading, and integration with existing OpenFrame infrastructure.
