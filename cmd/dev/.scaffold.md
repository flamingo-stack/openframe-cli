This file defines the Skaffold development command for the OpenFrame CLI, providing hot reloading and live development capabilities for Kubernetes services.

## Key Components

- **`getScaffoldCmd()`** - Returns a configured Cobra command for the scaffold functionality
- **`runScaffold()`** - Executes the scaffold workflow with proper context and service initialization
- **`ScaffoldFlags`** - Configuration structure containing development-specific options like port, namespace, image settings, and sync directories

## Usage Example

```go
// The command is typically used through the CLI:
// openframe dev skaffold my-dev-cluster --port 8080 --namespace dev

// Programmatic usage within the dev package:
func setupDevCommands() {
    scaffoldCmd := getScaffoldCmd()
    
    // Execute with custom flags
    flags := &models.ScaffoldFlags{
        Port:            8080,
        Namespace:       "development",
        SkipBootstrap:   false,
        HelmValuesFile:  "custom-values.yaml",
    }
    
    err := runScaffold(cmd, []string{"dev-cluster"}, flags)
    if err != nil {
        log.Fatal(err)
    }
}
```

The scaffold command manages the complete development lifecycle from prerequisites validation through cluster bootstrapping to live reloading setup, making it ideal for rapid development workflows.
