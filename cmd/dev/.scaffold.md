Defines the CLI command for scaffolding development environments with live reloading capabilities using Skaffold integration.

## Key Components

- **`getScaffoldCmd()`** - Creates and configures the `skaffold` cobra command with comprehensive flags
- **`runScaffold()`** - Executes the scaffold workflow by initializing services and delegating to scaffold service
- **Command flags** - Supports port configuration, namespace targeting, image specification, file syncing, and bootstrap control
- **Integration points** - Connects CLI layer with scaffold service and command executor

## Usage Example

```go
// Get the scaffold command for CLI registration
scaffoldCmd := getScaffoldCmd()

// The command supports various usage patterns:
// openframe dev skaffold                    # Interactive cluster creation
// openframe dev skaffold my-dev-cluster    # Named cluster
// openframe dev skaffold --port 8080       # Custom port
// openframe dev skaffold --skip-bootstrap  # Skip cluster setup

// Command execution flow:
// 1. Parse flags and arguments
// 2. Create command executor with dry-run/verbose settings
// 3. Initialize scaffold service
// 4. Run scaffold workflow with provided configuration
```

The command provides a complete development environment setup including prerequisite validation, cluster bootstrapping with development settings, and live reloading capabilities through Skaffold integration.
