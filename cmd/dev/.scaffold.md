This file provides a Cobra CLI command for setting up a development environment with live reloading capabilities using Skaffold.

## Key Components

- **getScaffoldCmd()**: Creates and configures the `skaffold` command with comprehensive flags for development workflow
- **runScaffold()**: Executes the scaffold command by initializing the scaffold service and running the workflow
- **ScaffoldFlags**: Configuration struct containing development-specific options like port, namespace, image settings, and sync directories

## Usage Example

```go
// Basic scaffold command execution
cmd := getScaffoldCmd()
err := cmd.Execute()

// The command supports various development scenarios:
// Interactive cluster creation
openframe dev skaffold

// Named cluster with custom port
openframe dev skaffold my-dev-cluster --port 3000

// Skip bootstrap with custom Helm values
openframe dev skaffold --skip-bootstrap --helm-values ./custom-values.yaml

// Full development setup with file sync
openframe dev skaffold --sync-local ./src --sync-remote /app --namespace dev
```

The command integrates with the OpenFrame CLI ecosystem, providing a streamlined development experience with hot reloading, cluster management, and flexible deployment options for rapid iteration.
