This file defines the Cobra CLI command for scaffolding development environments with live reloading capabilities using Skaffold.

## Key Components

- **`getScaffoldCmd()`** - Creates and configures the Cobra command with comprehensive flags for development workflows
- **`runScaffold()`** - Executes the scaffold workflow by initializing services and delegating to the scaffold service
- **`models.ScaffoldFlags`** - Configuration struct containing development-specific options like ports, namespaces, and sync directories

## Usage Example

```go
// Initialize the scaffold command
scaffoldCmd := getScaffoldCmd()

// The command supports various usage patterns:
// Basic interactive scaffolding
// openframe dev skaffold

// With specific cluster name
// openframe dev skaffold my-dev-cluster

// With custom configuration
// openframe dev skaffold --port 8080 --namespace dev --skip-bootstrap

// The command will:
// 1. Validate Skaffold prerequisites
// 2. Bootstrap cluster (unless skipped)
// 3. Set up live reloading environment
// 4. Enable hot deployment capabilities
```

The command integrates with OpenFrame's infrastructure to provide a complete development environment with features like cluster bootstrapping, live code reloading, and customizable deployment configurations through various flags for ports, namespaces, Docker images, and file synchronization.
