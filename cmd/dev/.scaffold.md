A command handler that orchestrates the development scaffolding workflow for deploying services with live reloading capabilities using Skaffold and Kubernetes.

## Key Components

- **`getScaffoldCmd()`** - Returns a Cobra command configured for scaffold operations
- **`runScaffold()`** - Executes the scaffold workflow by delegating to the scaffold service
- **`models.ScaffoldFlags`** - Configuration structure for scaffold-specific options
- **Command flags** - Port, namespace, image, sync directories, bootstrap options, and Helm values

## Usage Example

```go
// Create the scaffold command
scaffoldCmd := getScaffoldCmd()

// Add to parent command
devCmd.AddCommand(scaffoldCmd)

// Example CLI usage:
// openframe dev skaffold my-cluster --port 3000 --namespace dev
// openframe dev skaffold --sync-local ./src --sync-remote /app/src
// openframe dev skaffold --skip-bootstrap --helm-values custom-values.yaml
```

The command supports interactive cluster creation, custom port configuration, file synchronization for hot reloading, and optional cluster bootstrapping with development-optimized settings.
