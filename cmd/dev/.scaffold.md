This file defines a Cobra CLI command for scaffolding development environments with Skaffold integration, enabling live code reloading and hot deployment for development workflows.

## Key Components

- **`getScaffoldCmd()`** - Creates and configures the "skaffold" command with flags and documentation
- **`runScaffold()`** - Handles command execution by initializing services and running the scaffold workflow
- **`ScaffoldFlags`** - Configuration structure containing port, namespace, image, sync paths, and bootstrap options
- **Command flags** - Port configuration, Kubernetes namespace, Docker image, file sync paths, and bootstrap controls

## Usage Example

```go
// The command can be used in several ways:

// Basic scaffolding with interactive cluster creation
// openframe dev skaffold

// Scaffold with specific cluster name
// openframe dev skaffold my-dev-cluster

// Custom configuration with flags
// openframe dev skaffold --port 9000 --namespace dev-env --skip-bootstrap

// Development with file synchronization
// openframe dev skaffold --sync-local ./src --sync-remote /app/src --image my-app:dev
```

The command integrates with Skaffold to provide a complete development environment setup, including prerequisite validation, cluster bootstrapping with development-friendly settings, and live reloading capabilities for rapid development cycles.
