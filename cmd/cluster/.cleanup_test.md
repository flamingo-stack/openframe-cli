Test file for the cluster cleanup command functionality, providing automated testing with mocked command execution.

## Key Components

- **`TestCleanupCommand`** - Main test function that validates the cleanup command behavior
- **`setupFunc`** - Configures test environment with mock executor for command simulation
- **`teardownFunc`** - Resets global state after test completion
- **Test initialization** - Sets up test mode using `testutil.InitializeTestMode()`

## Usage Example

```go
// Run the cleanup command test
func TestCleanupCommand(t *testing.T) {
    // Setup mock executor for testing
    setupFunc := func() {
        utils.SetTestExecutor(testutil.NewTestMockExecutor())
    }
    
    // Clean up after test
    teardownFunc := func() {
        utils.ResetGlobalFlags()
    }

    // Execute the test with proper setup/teardown
    testutil.TestClusterCommand(t, "cleanup", getCleanupCmd, setupFunc, teardownFunc)
}
```

The test uses the common cluster command testing pattern with mock executors to validate cleanup functionality without executing actual system commands.
