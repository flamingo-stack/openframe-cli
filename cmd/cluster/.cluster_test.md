This file contains unit tests for the cluster command functionality, ensuring the root cluster command works correctly in the OpenFrame CLI.

## Key Components

- **`init()`** - Initializes test mode using the testutil package
- **`TestClusterRootCommand()`** - Tests the root cluster command execution without requiring additional setup

```go
// Run the cluster command test
func TestClusterRootCommand(t *testing.T) {
    // Tests the basic cluster command functionality
    testutil.TestClusterCommand(t, "cluster", GetClusterCmd, nil, nil)
}
```

The test leverages the `testutil.TestClusterCommand` helper function to validate that the cluster command can be executed successfully. The `nil` parameters indicate no additional setup or teardown is required for testing the root command.
