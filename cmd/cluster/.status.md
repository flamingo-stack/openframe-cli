Implements the `cluster status` command for the OpenFrame CLI, providing detailed cluster health and information display functionality.

## Key Components

- **`getStatusCmd()`** - Creates and configures the Cobra command for cluster status operations
- **`runClusterStatus()`** - Executes the status operation with interactive cluster selection and service integration
- **Interactive Selection** - Uses UI components for user-friendly cluster selection when no name is provided
- **Flag Integration** - Supports detailed output, app filtering, and verbose modes through global and status-specific flags

## Usage Example

```go
// Register the status command with the cluster command group
clusterCmd.AddCommand(getStatusCmd())

// Command usage examples:
// openframe cluster status my-cluster
// openframe cluster status --detailed
// openframe cluster status my-cluster --no-apps --verbose
```

The command supports optional cluster name arguments and falls back to interactive selection. It validates flags during pre-execution and wraps the main operation with common setup utilities. Status output includes cluster health, node information, installed applications, and resource usage based on the specified flags.
