Implements the cluster status command for the OpenFrame CLI, providing detailed information about Kubernetes cluster health and configuration.

## Key Components

- **getStatusCmd()** - Creates and configures the `cluster status` Cobra command with flags and validation
- **runClusterStatus()** - Executes the status operation, handling cluster selection and displaying status information
- **Command flags** - Supports `--detailed` and `--no-apps` flags for customizing output verbosity
- **Interactive selection** - Provides user-friendly cluster selection when no cluster name is specified

## Usage Example

```go
// Register the status command with the cluster command group
clusterCmd.AddCommand(getStatusCmd())

// Command usage examples:
// openframe cluster status my-cluster
// openframe cluster status --detailed
// openframe cluster status my-cluster --no-apps
```

The command integrates with the service layer to retrieve cluster information and uses the operations UI for interactive cluster selection. It validates flags before execution and wraps the command with common setup utilities for consistent error handling and logging.
