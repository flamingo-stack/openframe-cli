Provides a CLI command for displaying detailed Kubernetes cluster status information including health, nodes, applications, and resource usage.

## Key Components

- **getStatusCmd()** - Creates and configures the `cluster status` cobra command with flags and validation
- **runClusterStatus()** - Executes the status operation, handles cluster selection, and displays status information
- **Command flags** - Supports `--detailed` and `--no-apps` options for customizing output verbosity
- **Interactive selection** - Allows users to select from available clusters when name not provided

## Usage Example

```go
// Create the status command
statusCmd := getStatusCmd()

// The command supports multiple usage patterns:
// openframe cluster status my-cluster
// openframe cluster status --detailed
// openframe cluster status my-cluster --no-apps

// Example of running status check programmatically
func checkClusterStatus() error {
    service := utils.GetCommandService()
    operationsUI := ui.NewOperationsUI()
    
    clusters, err := service.ListClusters()
    if err != nil {
        return err
    }
    
    clusterName, err := operationsUI.SelectClusterForOperation(clusters, []string{}, "check status")
    if err != nil {
        return err
    }
    
    return service.ShowClusterStatus(clusterName, true, false, true)
}
```
