Test file for the cluster status command functionality, providing unit test coverage for status-related operations in the cluster package.

## Key Components

- **TestStatusCommand** - Main test function that validates the cluster status command behavior
- **setupFunc** - Test setup function that configures a mock executor for isolated testing
- **teardownFunc** - Test cleanup function that resets global flags after test execution
- **init()** - Package initialization that enables test mode for the entire test suite

## Usage Example

```go
// Run the status command test
func TestStatusCommand(t *testing.T) {
    setupFunc := func() {
        utils.SetTestExecutor(testutil.NewTestMockExecutor())
    }
    teardownFunc := func() {
        utils.ResetGlobalFlags()
    }
    
    testutil.TestClusterCommand(t, "status", getStatusCmd, setupFunc, teardownFunc)
}
```
