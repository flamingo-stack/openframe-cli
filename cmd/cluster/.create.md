Implements the CLI command for creating Kubernetes clusters with interactive and non-interactive modes, supporting various cluster types and configuration options.

## Key Components

- **`getCreateCmd()`** - Returns a configured cobra.Command for cluster creation with flags and validation
- **`runCreateCluster()`** - Main execution logic handling both interactive wizard and direct creation modes
- **Interactive Mode** - Uses UI configuration handler for step-by-step cluster setup
- **Non-Interactive Mode** - Creates clusters directly from command flags with validation
- **Configuration Validation** - Validates cluster names, node counts, and other parameters

## Usage Example

```go
// Interactive mode with default cluster name
cmd := getCreateCmd()
cmd.Execute() // Shows creation mode selection

// Direct creation with custom configuration
args := []string{"my-cluster"}
flags := &models.CreateFlags{
    NodeCount: 3,
    ClusterType: "k3d",
    SkipWizard: true,
}
runCreateCluster(cmd, args)

// Dry-run mode to preview configuration
flags.DryRun = true
runCreateCluster(cmd, args) // Shows config summary without creation
```

The command supports both `openframe cluster create` (interactive) and `openframe cluster create --skip-wizard --nodes 3` (direct) usage patterns, with comprehensive validation and error handling.
