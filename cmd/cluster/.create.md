This file implements the cluster creation command for the OpenFrame CLI tool, providing both interactive and non-interactive modes for Kubernetes cluster creation.

## Key Components

- **`getCreateCmd()`** - Creates and configures the cobra command for cluster creation with flags and validation
- **`runCreateCluster()`** - Main execution function that handles configuration gathering and cluster creation
- **Interactive Mode** - Uses UI components to guide users through cluster configuration
- **Non-Interactive Mode** - Creates clusters directly from command-line flags with sensible defaults
- **Configuration Validation** - Validates cluster names, node counts, and other parameters

## Usage Example

```go
// Interactive cluster creation with selection menu
cmd := getCreateCmd()
cmd.Execute() // Prompts user for configuration options

// Non-interactive creation with defaults
args := []string{"my-cluster", "--skip-wizard"}
runCreateCluster(cmd, args)

// Dry run to preview configuration
globalFlags.Create.DryRun = true
runCreateCluster(cmd, []string{"test-cluster"})

// Custom configuration via flags
args := []string{"dev-cluster", "--nodes", "5", "--type", "k3d", "--skip-wizard"}
runCreateCluster(cmd, args)
```

The command supports flexible cluster creation workflows, from quick defaults to detailed interactive configuration, with built-in validation and preview capabilities for development environments.
