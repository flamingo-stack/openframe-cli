This file implements the core cluster creation command for the OpenFrame CLI tool, handling both interactive and non-interactive cluster configuration modes.

## Key Components

- **`getCreateCmd()`** - Creates and configures the Cobra command for cluster creation with flags and validation
- **`runCreateCluster()`** - Main execution logic that handles interactive UI flow or direct flag-based configuration
- **Interactive Mode** - Uses UI configuration handler for step-by-step cluster setup
- **Non-Interactive Mode** - Builds cluster configuration directly from command flags and arguments
- **Configuration Validation** - Validates cluster names, node counts, and other parameters before creation

## Usage Example

```go
// Interactive mode with selection menu
cmd := getCreateCmd()
cmd.Execute() // Shows creation mode selection

// Direct creation with defaults
globalFlags := utils.GetGlobalFlags()
globalFlags.Create.SkipWizard = true
runCreateCluster(cmd, []string{"my-cluster"})

// Custom configuration
globalFlags.Create.NodeCount = 5
globalFlags.Create.ClusterType = "k3d"
globalFlags.Create.DryRun = true
runCreateCluster(cmd, []string{"prod-cluster"})
```

The command supports flexible cluster creation workflows - from quick defaults to comprehensive interactive configuration, with built-in validation and dry-run capabilities for safe cluster provisioning.
