Implements the `cluster create` command for creating new Kubernetes clusters with both interactive and non-interactive modes. Supports multiple cluster types and provides configuration validation and dry-run capabilities.

## Key Components

- **`getCreateCmd()`**: Returns the configured cobra.Command for cluster creation with flags and validation
- **`runCreateCluster()`**: Main execution function that handles both interactive wizard and direct creation modes
- **Configuration Flow**: Integrates with UI layer for interactive configuration or uses command-line flags for direct creation
- **Validation**: Performs cluster name validation and flag validation before execution

## Usage Example

```go
// Interactive mode (default)
openframe cluster create

// Direct creation with defaults
openframe cluster create my-cluster --skip-wizard

// Custom configuration
openframe cluster create dev-cluster --nodes 3 --type k3d --skip-wizard

// Dry run to see configuration
openframe cluster create test-cluster --dry-run

// With specific Kubernetes version
openframe cluster create --k8s-version v1.28.0 --skip-wizard
```

The command supports both quick creation with sensible defaults and step-by-step interactive configuration through a wizard interface.
