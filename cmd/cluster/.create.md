Creates a new Kubernetes cluster with interactive configuration options or direct creation using command-line flags. This command supports both a wizard-based setup flow and quick creation with defaults.

## Key Components

- **getCreateCmd()** - Initializes the Cobra command with flags, validation, and help text
- **runCreateCluster()** - Main execution logic that handles interactive vs non-interactive modes
- **Configuration Flow** - Routes between UI-driven configuration and flag-based setup
- **Validation** - Validates cluster names and node counts before creation

## Usage Example

```go
// Interactive mode (default)
openframe cluster create

// Quick creation with defaults
openframe cluster create --skip-wizard

// Custom configuration with flags
openframe cluster create my-cluster --nodes 3 --type k3d --skip-wizard

// Dry run to preview configuration
openframe cluster create --dry-run
```

The command integrates with the UI layer for interactive configuration and the service layer for actual cluster creation. It supports validation of cluster names and node counts, with automatic fallbacks to sensible defaults when needed.
