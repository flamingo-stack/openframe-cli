This file implements the cluster creation command for the OpenFrame CLI, providing both interactive and non-interactive modes for creating Kubernetes clusters.

## Key Components

- **`getCreateCmd()`** - Returns the Cobra command definition with flags, validation, and help text
- **`runCreateCluster()`** - Main execution function that handles cluster creation workflow
- **Interactive Mode** - Uses UI components to guide users through cluster configuration
- **Non-interactive Mode** - Creates clusters directly from command-line flags with sensible defaults
- **Configuration Validation** - Validates cluster names, node counts, and other parameters

## Usage Example

```go
// Interactive cluster creation (default behavior)
openframe cluster create

// Create with custom name, showing selection menu
openframe cluster create my-dev-cluster

// Direct creation with defaults (skips wizard)
openframe cluster create --skip-wizard

// Customized non-interactive creation
openframe cluster create production --nodes 5 --type k3d --skip-wizard

// Dry run to preview configuration
openframe cluster create --dry-run --nodes 3
```

The command supports both quick creation with defaults and step-by-step interactive configuration. It validates all inputs, shows configuration summaries, and integrates with the service layer to perform actual cluster operations.
