This file implements the `create` command for the OpenFrame CLI cluster management functionality, allowing users to create Kubernetes clusters with either interactive configuration or command-line flags.

## Key Components

- **`getCreateCmd()`** - Creates and configures the cobra command with flags, validation, and help text
- **`runCreateCluster()`** - Main execution logic that handles both interactive and non-interactive cluster creation modes
- **Interactive Mode** - Uses UI configuration handler for step-by-step cluster setup
- **Non-Interactive Mode** - Builds cluster configuration directly from command flags and arguments
- **Validation** - Validates cluster names, node counts, and other configuration parameters

## Usage Example

```go
// Create the create command
createCmd := getCreateCmd()

// Example cluster configurations that would be handled:
config := models.ClusterConfig{
    Name:       "my-cluster",
    Type:       models.ClusterTypeK3d,
    K8sVersion: "v1.28.0",
    NodeCount:  3,
}

// The command supports various usage patterns:
// openframe cluster create                    # Interactive mode
// openframe cluster create my-cluster        # Interactive with custom name  
// openframe cluster create --skip-wizard     # Direct creation with defaults
// openframe cluster create --nodes 3 --type k3d --skip-wizard
```

The command integrates with the broader CLI architecture through the utils package for flag management and service layer for actual cluster operations. It provides flexibility for both quick cluster creation and detailed configuration workflows.
