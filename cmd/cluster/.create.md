This Go file implements the cluster creation command for the OpenFrame CLI tool, providing both interactive and non-interactive modes for creating Kubernetes clusters.

## Key Components

- **`getCreateCmd()`** - Constructs and configures the cobra command for cluster creation with flags and validation
- **`runCreateCluster()`** - Main execution logic that handles interactive/non-interactive flows and delegates to service layer
- **Interactive Mode** - Uses UI layer for step-by-step cluster configuration when `--skip-wizard` is not set
- **Non-Interactive Mode** - Builds cluster configuration directly from command flags and arguments
- **Validation** - Performs cluster name validation and flag validation before execution

## Usage Example

```go
// Create the create subcommand
createCmd := getCreateCmd()

// Example command usage scenarios:
// openframe cluster create                    # Interactive mode with selection menu
// openframe cluster create my-cluster        # Interactive mode with custom name  
// openframe cluster create --skip-wizard     # Direct creation with defaults
// openframe cluster create --nodes 3 --type k3d --skip-wizard # Custom config

// The command supports dry-run mode for configuration preview
// openframe cluster create --dry-run

// Configuration is handled through models.ClusterConfig:
config := models.ClusterConfig{
    Name:       "my-cluster",
    Type:       models.ClusterTypeK3d,
    K8sVersion: "1.28.0",
    NodeCount:  3,
}
```
