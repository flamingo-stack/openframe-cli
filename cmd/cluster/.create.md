Creates a new Kubernetes cluster through either an interactive wizard or command-line flags, with support for different cluster types and configurations.

## Key Components

- **`getCreateCmd()`**: Returns the main create command with full CLI setup including flags, validation, and help documentation
- **`runCreateCluster()`**: Executes cluster creation logic, handling both interactive and non-interactive modes
- **Interactive Mode**: Uses UI layer for step-by-step cluster configuration when `--skip-wizard` is not set
- **Non-Interactive Mode**: Builds cluster config directly from command flags and arguments
- **Configuration Summary**: Shows cluster settings before creation in dry-run or verbose modes

## Usage Example

```go
// Basic cluster creation with defaults
cmd := getCreateCmd()
err := cmd.Execute() // Creates "openframe-dev" cluster

// Create named cluster interactively
args := []string{"my-cluster"}
err := runCreateCluster(cmd, args)

// Non-interactive creation with custom settings
// Command: openframe cluster create --nodes 5 --type k3d --skip-wizard
globalFlags := utils.GetGlobalFlags()
globalFlags.Create.NodeCount = 5
globalFlags.Create.ClusterType = "k3d"
globalFlags.Create.SkipWizard = true

config := models.ClusterConfig{
    Name:       "my-cluster",
    Type:       models.ClusterTypeK3d,
    NodeCount:  5,
    K8sVersion: "v1.28.0",
}
```

The command supports dry-run mode for configuration preview and integrates with the service layer for actual cluster provisioning.
