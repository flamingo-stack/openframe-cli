This file implements the cluster creation command for the OpenFrame CLI, providing both interactive and non-interactive modes for creating Kubernetes clusters.

## Key Components

- **`getCreateCmd()`** - Returns a configured cobra command for cluster creation with proper flags and validation
- **`runCreateCluster()`** - Main execution logic that handles both interactive wizard and direct creation modes
- **Flag validation** - Pre-run validation for cluster creation parameters including name validation and node count checks
- **Configuration flow** - Integrates with UI components for interactive configuration or builds config from command flags

## Usage Example

```go
// Interactive mode with cluster selection menu
openframe cluster create

// Direct creation with defaults, skipping wizard
openframe cluster create my-cluster --skip-wizard

// Custom configuration with specific parameters
openframe cluster create dev-cluster --nodes 3 --type k3d --skip-wizard

// Dry run to preview configuration
openframe cluster create test-cluster --dry-run
```

The command supports flexible cluster creation workflows, automatically validating inputs and providing defaults when needed. It integrates with the UI layer for interactive configuration and the service layer for actual cluster provisioning.
