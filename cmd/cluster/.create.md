This file implements the Kubernetes cluster creation command for the OpenFrame CLI, providing both interactive and non-interactive cluster configuration modes.

## Key Components

- **`getCreateCmd()`** - Creates and configures the `cluster create` Cobra command with flags, validation, and help text
- **`runCreateCluster()`** - Main execution logic that handles both wizard-based interactive configuration and direct flag-based cluster creation
- **Interactive UI Integration** - Uses `ui.ConfigurationHandler` for guided cluster setup when `--skip-wizard` is not specified
- **Flag Validation** - Validates cluster names, node counts, and other configuration parameters before execution
- **Dry Run Support** - Shows configuration summary without actually creating clusters when `--dry-run` is used

## Usage Example

```go
// Interactive mode (default) - shows configuration wizard
cmd := getCreateCmd()
cmd.Execute() // openframe cluster create

// Direct creation with flags
cmd.SetArgs([]string{"my-cluster", "--nodes", "3", "--type", "k3d", "--skip-wizard"})
cmd.Execute()

// Dry run to preview configuration
cmd.SetArgs([]string{"--dry-run", "--nodes", "5"})
cmd.Execute()

// Use default cluster name with custom settings
cmd.SetArgs([]string{"--nodes", "1", "--skip-wizard"})
cmd.Execute() // Creates "openframe-dev" cluster
```
