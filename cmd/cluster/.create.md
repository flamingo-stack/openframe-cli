This file implements the cluster creation command for the OpenFrame CLI tool, providing both interactive and non-interactive modes for creating Kubernetes clusters.

## Key Components

- **`getCreateCmd()`** - Configures and returns the `cluster create` Cobra command with flags and validation
- **`runCreateCluster()`** - Main execution function that handles both interactive and non-interactive cluster creation workflows
- **Interactive Mode** - Uses UI components to guide users through cluster configuration
- **Non-interactive Mode** - Creates clusters directly from command flags and arguments
- **Configuration Validation** - Validates cluster names, node counts, and other parameters before creation

## Usage Example

```go
// Get the create command and add it to parent command
cmd := getCreateCmd()
parentCmd.AddCommand(cmd)

// Example command execution flows:
// Interactive mode (default):
// $ openframe cluster create
// $ openframe cluster create my-cluster

// Non-interactive mode with flags:
// $ openframe cluster create --skip-wizard
// $ openframe cluster create my-cluster --nodes 3 --type k3d --skip-wizard

// Dry run mode:
// $ openframe cluster create --dry-run
```

The command supports flexible cluster creation with sensible defaults (3 nodes, k3d type, "openframe-dev" name) while allowing full customization through either interactive wizards or command-line flags. It includes comprehensive validation and can show configuration summaries before execution.
