This file implements the CLI command for creating Kubernetes clusters in the OpenFrame CLI tool. It provides both interactive and non-interactive cluster creation modes with validation and configuration management.

## Key Components

- **`getCreateCmd()`** - Creates and configures the cobra command with flags, validation, and help text
- **`runCreateCluster()`** - Main command execution logic that handles both interactive and non-interactive modes
- **Interactive mode** - Uses UI configuration handler for step-by-step cluster setup
- **Non-interactive mode** - Uses command-line flags with validation and default values
- **Configuration validation** - Validates cluster names, node counts, and other parameters

## Usage Example

```go
// Create command with interactive configuration
cmd := getCreateCmd()

// The command supports various usage patterns:
// openframe cluster create                    # Interactive mode with selection menu
// openframe cluster create my-cluster        # Interactive mode with custom name
// openframe cluster create --skip-wizard     # Direct creation with defaults
// openframe cluster create --nodes 3 --type k3d --skip-wizard

// Non-interactive execution example
config := models.ClusterConfig{
    Name:       "my-cluster",
    Type:       models.ClusterTypeK3d,
    K8sVersion: "1.27",
    NodeCount:  3,
}
```

The command integrates with the service layer for actual cluster creation and provides comprehensive validation, dry-run capabilities, and flexible configuration options through both CLI flags and interactive wizards.
