This file implements the `create` command for creating Kubernetes clusters in the OpenFrame CLI tool. It provides both interactive wizard and command-line flag modes for cluster configuration.

## Key Components

- **`getCreateCmd()`** - Returns a configured cobra.Command for cluster creation with flags and validation
- **`runCreateCluster()`** - Main execution function that handles interactive/non-interactive modes and delegates to service layer
- **Interactive Mode** - Uses UI configuration handler for step-by-step cluster setup
- **Non-interactive Mode** - Creates clusters directly from command-line flags with defaults
- **Validation** - Validates cluster names, node counts, and global flags before execution

## Usage Example

```go
// Interactive mode (default)
cmd := getCreateCmd()
cmd.Execute() // Shows selection menu for configuration

// Non-interactive with defaults
args := []string{"my-cluster", "--skip-wizard"}
runCreateCluster(cmd, args)

// Custom configuration
args := []string{"prod-cluster", "--nodes", "5", "--type", "k3d", "--skip-wizard"}
runCreateCluster(cmd, args)

// Dry run mode
globalFlags.Create.DryRun = true
runCreateCluster(cmd, []string{"test-cluster"})
```

The command supports flexible cluster creation with validation, configuration summaries, and integration with the broader OpenFrame CLI infrastructure through the utils and service layers.
