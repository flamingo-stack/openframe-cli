Implements the `create` command for creating Kubernetes clusters with interactive or flag-based configuration. Provides wizard-based setup with defaults or direct creation via command-line flags.

## Key Components

- **getCreateCmd()** - Returns configured cobra command with flags and validation
- **runCreateCluster()** - Main execution logic handling interactive vs non-interactive modes
- **Configuration Flow** - Supports both UI wizard and direct flag-based cluster creation
- **Validation** - Pre-run validation for global flags and cluster names
- **Dry Run Support** - Configuration preview without actual cluster creation

## Usage Example

```go
// Interactive mode with wizard
openframe cluster create

// Direct creation with defaults
openframe cluster create my-cluster --skip-wizard

// Custom configuration with flags
openframe cluster create --nodes 3 --type k3d --skip-wizard

// Dry run to preview configuration
openframe cluster create --dry-run
```

The command integrates with the UI layer for interactive configuration and the service layer for actual cluster creation. Supports k3d cluster types with configurable node counts and Kubernetes versions.
