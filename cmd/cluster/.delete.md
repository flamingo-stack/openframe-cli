Implements the `delete` subcommand for removing Kubernetes clusters and cleaning up associated resources.

## Key Components

- **getDeleteCmd()**: Creates the Cobra command with usage patterns, validation, and flag configuration
- **runDeleteCluster()**: Main execution function that handles cluster selection, type detection, and deletion
- **Cluster Selection**: Interactive UI for selecting clusters when none specified
- **Resource Cleanup**: Stops intercepts, deletes cluster, and cleans up Docker resources
- **Force Mode**: Optional `--force` flag to skip confirmation prompts

## Usage Example

```go
// Command registration
rootCmd.AddCommand(getDeleteCmd())

// CLI usage examples:
// openframe cluster delete my-cluster
// openframe cluster delete my-cluster --force  
// openframe cluster delete  // interactive selection

// The delete process:
// 1. Lists available clusters
// 2. Shows selection UI if no cluster specified
// 3. Detects cluster type (kind, k3s, etc.)
// 4. Confirms deletion (unless --force used)
// 5. Executes cleanup through service layer
// 6. Shows progress and completion status
```

The command provides a safe deletion workflow with confirmation prompts and comprehensive cleanup of cluster resources, Docker containers, and configuration files.
