This file implements the cluster deletion command for the OpenFrame CLI, providing functionality to delete Kubernetes clusters with proper cleanup and user interaction.

## Key Components

- **`getDeleteCmd()`** - Creates and configures the `delete` cobra command with flags, validation, and help text
- **`runDeleteCluster()`** - Main execution function that handles cluster selection, type detection, and deletion process
- **Command validation** - Pre-run validation of global and delete-specific flags
- **Interactive UI** - User-friendly cluster selection and confirmation dialogs
- **Error handling** - Comprehensive error handling with verbose output options
