Test suite for the cluster create command functionality, ensuring proper command initialization and execution in a controlled testing environment.

## Key Components

- **TestCreateCommand**: Main test function that validates the cluster create command behavior
- **setupFunc**: Test setup function that configures a mock executor for isolated testing
- **teardownFunc**: Test cleanup function that resets global state between test runs
- **init()**: Package initialization that enables test mode for the entire test suite

## Usage Example

```go
// Run the create command test
func TestCreateCommand(t *testing.T) {
    // Setup mock executor and test environment
    setupFunc := func() {
        utils.SetTestExecutor(testutil.NewTestMockExecutor())
    }
    
    // Clean up after test completion
    teardownFunc := func() {
        utils.ResetGlobalFlags()
    }
    
    // Execute comprehensive command testing
    testutil.TestClusterCommand(t, "create", getCreateCmd, setupFunc, teardownFunc)
}
```

This test file follows Go testing conventions and uses dependency injection through mock executors to ensure commands can be tested without affecting the actual system state.
