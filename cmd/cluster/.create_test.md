Test file that validates the cluster creation command functionality using a mock executor to simulate cluster operations.

## Key Components

- **TestCreateCommand**: Main test function that validates the `create` command behavior
- **setupFunc**: Configures test environment with mock executor for simulating cluster operations
- **teardownFunc**: Cleans up test state by resetting global flags
- **testutil.TestClusterCommand**: Utility function that runs standardized cluster command tests

## Usage Example

```go
// Run the create command test
func TestCreateCommand(t *testing.T) {
    setupFunc := func() {
        utils.SetTestExecutor(testutil.NewTestMockExecutor())
    }
    teardownFunc := func() {
        utils.ResetGlobalFlags()
    }
    
    testutil.TestClusterCommand(t, "create", getCreateCmd, setupFunc, teardownFunc)
}
```

The test leverages the testutil framework to ensure consistent testing patterns across cluster commands, with proper setup and cleanup of mock executors and global state.
