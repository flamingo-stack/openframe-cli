This file implements a Cobra CLI command for listing Kubernetes clusters managed by the OpenFrame CLI tool.

## Key Components

- **getListCmd()** - Creates and configures the `list` command with flags, validation, and help text
- **runListClusters()** - Core function that retrieves and displays cluster information from all registered providers
- **Global flag integration** - Leverages utility functions for flag management, validation, and command setup

## Usage Example

```go
// Register the list command with the cluster parent command
clusterCmd.AddCommand(getListCmd())

// The command can be executed as:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet
```

The command displays cluster information in a formatted table including name, type, status, and node count. It supports standard output formatting options through global flags (verbose/quiet modes) and includes comprehensive validation for all flag combinations before execution.
