Defines the Cobra command for listing Kubernetes clusters managed by OpenFrame CLI. This file implements the `cluster list` subcommand functionality with flag validation and cluster display capabilities.

## Key Components

- **`getListCmd()`** - Creates and configures the Cobra command for cluster listing with flags and validation
- **`runListClusters()`** - Executes the cluster listing logic, retrieving and displaying cluster information
- **Command validation** - Pre-run validation of global and list-specific flags using the models package
- **Service integration** - Uses utils package for command service access and common setup patterns

## Usage Example

```go
// Create the list command
listCmd := getListCmd()

// The command supports various flags and can be executed as:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet

// Programmatically execute (typically handled by Cobra):
err := runListClusters(cmd, args)
if err != nil {
    log.Fatalf("Failed to list clusters: %v", err)
}
```

The command integrates with the OpenFrame CLI's global flag system and uses a service-based architecture to retrieve cluster information from multiple providers, displaying results in a formatted table with options for verbose or quiet output modes.
