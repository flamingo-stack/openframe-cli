A Cobra CLI command implementation for listing Kubernetes clusters managed by OpenFrame CLI, providing formatted output with cluster information including name, type, status, and node count.

## Key Components

- **`getListCmd()`** - Creates and configures the `list` command with global flag initialization, validation, and command setup
- **`runListClusters()`** - Executes the cluster listing logic, retrieves clusters from the service, and displays them in a formatted table
- **Global flag integration** - Supports `--verbose` and `--quiet` output options through the utils package
- **Command validation** - Pre-run validation ensures proper flag configuration before execution

## Usage Example

```go
// Register the list command with the cluster command group
clusterCmd.AddCommand(getListCmd())

// Command supports various output formats:
// openframe cluster list                    # Standard output
// openframe cluster list --verbose          # Detailed output
// openframe cluster list --quiet            # Minimal output
```

The command integrates with the OpenFrame service layer to retrieve cluster data from all registered providers and presents it in a user-friendly table format. Error handling ensures informative feedback when cluster retrieval fails, and the modular design allows for easy extension of list-specific functionality.
