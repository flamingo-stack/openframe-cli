This file implements the "list" command functionality for displaying Kubernetes clusters managed by the OpenFrame CLI.

## Key Components

- **getListCmd()** - Creates and configures the Cobra command for listing clusters with flags, validation, and help text
- **runListClusters()** - Executes the cluster listing logic by retrieving clusters from the service and displaying them
- **Command validation** - Pre-run validation of global and list-specific flags before execution
- **Flag integration** - Supports global flags like `--verbose` and `--quiet` for output formatting

## Usage Example

```go
// Create the list command
listCmd := getListCmd()

// The command can be executed with various flags:
// openframe cluster list
// openframe cluster list --verbose  
// openframe cluster list --quiet

// Programmatically run the list functionality
err := runListClusters(cmd, []string{})
if err != nil {
    log.Fatalf("Failed to list clusters: %v", err)
}
```

The command retrieves all registered clusters from various providers and displays them in a formatted table with information like cluster name, type, status, and node count. The output format can be controlled through the `--quiet` and `--verbose` flags.
