A Cobra command implementation that provides cluster listing functionality for the OpenFrame CLI, allowing users to view all registered Kubernetes clusters with configurable output formatting.

## Key Components

- **`getListCmd()`** - Creates and configures the "list" subcommand with flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic, retrieving clusters from the service and displaying them
- **Global flags integration** - Leverages shared utility functions for consistent flag handling across commands
- **Pre-run validation** - Ensures all flags are properly synchronized and validated before execution

## Usage Example

```go
// Create the list command
listCmd := getListCmd()

// The command can be executed with various flags:
// openframe cluster list                 (basic listing)
// openframe cluster list --verbose       (detailed output)
// openframe cluster list --quiet         (minimal output)

// Add to parent cluster command
clusterCmd.AddCommand(listCmd)

// Execute the listing logic directly
err := runListClusters(cmd, []string{})
if err != nil {
    return fmt.Errorf("cluster listing failed: %w", err)
}
```

The command integrates with the OpenFrame CLI's service layer to retrieve cluster information from all registered providers and displays it in a formatted table with configurable verbosity levels.
