Defines a Cobra command for listing Kubernetes clusters managed by OpenFrame CLI, providing formatted output with cluster information from all registered providers.

## Key Components

- **`getListCmd()`** - Creates and configures the `list` command with flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic, retrieving and displaying cluster information
- **Flag integration** - Integrates with global flags system for consistent CLI behavior
- **Validation pipeline** - Implements pre-run validation for command flags and configuration

## Usage Example

```go
// Register the list command as a subcommand
clusterCmd.AddCommand(getListCmd())

// Command usage from CLI:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet
```

The command displays cluster information in a formatted table including name, type, status, and node count. It supports standard CLI flags like `--verbose` for detailed output and `--quiet` for minimal output, with proper validation and error handling throughout the execution pipeline.
