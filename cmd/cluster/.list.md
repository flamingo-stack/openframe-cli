This file implements the `list` subcommand for the OpenFrame CLI cluster management functionality, providing users with the ability to view all managed Kubernetes clusters.

## Key Components

- **`getListCmd()`** - Creates and configures the Cobra command for listing clusters with flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic by retrieving clusters from the service and displaying them in a formatted table

## Usage Example

```go
// The command is typically used through the CLI:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet

// Internal usage within the application:
listCmd := getListCmd()

// The command integrates with the cluster service to fetch and display data
service := utils.GetCommandService()
clusters, err := service.ListClusters()
if err != nil {
    return fmt.Errorf("failed to list clusters: %w", err)
}

// Display results with formatting options
globalFlags := utils.GetGlobalFlags()
err = service.DisplayClusterList(clusters, globalFlags.List.Quiet, globalFlags.Global.Verbose)
```

The command includes comprehensive flag validation, global flag synchronization, and supports different output verbosity levels for flexible cluster information display.
