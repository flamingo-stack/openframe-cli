A Go file that implements the `list` subcommand for the OpenFrame CLI cluster management functionality, providing users with the ability to view all managed Kubernetes clusters.

## Key Components

- **getListCmd()** - Factory function that creates and configures the Cobra command for listing clusters
- **runListClusters()** - Command execution handler that retrieves and displays cluster information
- **Command Configuration** - Pre-run validation and flag setup for the list operation
- **Service Integration** - Uses cluster service to fetch and format cluster data

## Usage Example

```go
// Create the list command
listCmd := getListCmd()

// The command supports various output options
// Command line usage:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet

// Internal execution flow
func runListClusters(cmd *cobra.Command, args []string) error {
    service := utils.GetCommandService()
    
    // Retrieve all managed clusters
    clusters, err := service.ListClusters()
    if err != nil {
        return fmt.Errorf("failed to list clusters: %w", err)
    }
    
    // Display results with formatting options
    globalFlags := utils.GetGlobalFlags()
    return service.DisplayClusterList(clusters, 
        globalFlags.List.Quiet, 
        globalFlags.Global.Verbose)
}
```

The command integrates with the global flag system to support verbose and quiet output modes, validates flags during pre-execution, and delegates actual cluster retrieval and display formatting to the service layer.
