This file implements the `list` subcommand for the cluster management functionality, allowing users to list all Kubernetes clusters managed by OpenFrame CLI.

## Key Components

- **`getListCmd()`** - Creates and configures the cobra command for listing clusters with flags, validation, and help text
- **`runListClusters()`** - Core execution function that retrieves clusters from the service and displays them in a formatted table
- **Flag integration** - Supports global flags like `--verbose` and `--quiet` for output customization
- **Validation pipeline** - Includes pre-run validation for global and list-specific flags

## Usage Example

```go
// Command registration (typically in parent command)
clusterCmd.AddCommand(getListCmd())

// CLI usage examples:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet
```

The command integrates with the OpenFrame CLI's cluster service to fetch cluster information from all registered providers and displays details including cluster name, type, status, and node count in a user-friendly format.
