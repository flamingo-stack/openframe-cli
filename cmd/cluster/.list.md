This file defines the Cobra CLI command for listing all Kubernetes clusters managed by OpenFrame CLI.

## Key Components

- **`getListCmd()`** - Creates and configures the `cluster list` Cobra command with global flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic by retrieving clusters from the service and displaying them in a formatted table

## Usage Example

```go
// The command is typically registered in a parent cluster command
clusterCmd.AddCommand(getListCmd())

// CLI usage examples shown in the command help:
// openframe cluster list
// openframe cluster list --verbose  
// openframe cluster list --quiet
```

The command integrates with the OpenFrame CLI's global flag system, providing consistent validation and error handling. It retrieves cluster information from all registered providers and displays details like name, type, status, and node count in a user-friendly format with support for verbose and quiet output modes.
