This file implements the cluster list command functionality for the OpenFrame CLI tool, providing a Cobra command to display all managed Kubernetes clusters.

## Key Components

- **`getListCmd()`**: Creates and configures the Cobra command for listing clusters with proper flag initialization and validation
- **`runListClusters()`**: Executes the cluster listing logic by retrieving clusters from the service and displaying them in a formatted table

## Usage Example

```go
// Get the list command
listCmd := getListCmd()

// The command supports various flags and can be used as:
// openframe cluster list
// openframe cluster list --verbose  
// openframe cluster list --quiet

// Internal usage - the command execution flow:
service := utils.GetCommandService()
clusters, err := service.ListClusters()
if err != nil {
    return fmt.Errorf("failed to list clusters: %w", err)
}

globalFlags := utils.GetGlobalFlags()
return service.DisplayClusterList(clusters, globalFlags.List.Quiet, globalFlags.Global.Verbose)
```

The command integrates with the global flag system for consistent behavior across the CLI and includes proper validation through the PreRunE hook. It displays cluster information including name, type, status, and node count from all registered providers.
