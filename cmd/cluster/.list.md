This file implements the `list` command for the OpenFrame CLI cluster management functionality, allowing users to view all Kubernetes clusters across registered providers.

## Key Components

- **`getListCmd()`** - Creates and configures the `list` cobra command with flags, validation, and help documentation
- **`runListClusters()`** - Executes the cluster listing logic, retrieving and displaying cluster information in a formatted table

## Usage Example

```go
// Register the list command with the cluster command group
clusterCmd.AddCommand(getListCmd())

// Command execution flow
service := utils.GetCommandService()
clusters, err := service.ListClusters()
if err != nil {
    return fmt.Errorf("failed to list clusters: %w", err)
}

globalFlags := utils.GetGlobalFlags()
return service.DisplayClusterList(clusters, globalFlags.List.Quiet, globalFlags.Global.Verbose)
```

The command supports global flags like `--verbose` and `--quiet` for controlling output detail, validates all flags before execution, and uses the command service to retrieve and format cluster data from multiple providers into a user-friendly table display.
