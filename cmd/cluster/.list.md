A command implementation for listing Kubernetes clusters in the OpenFrame CLI, providing formatted display of cluster information from registered providers.

## Key Components

- **`getListCmd()`** - Creates and configures the `cluster list` Cobra command with flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic by calling the service layer and displaying results

## Usage Example

```go
// Get the list command for registration
listCmd := getListCmd()

// The command can be executed via CLI:
// openframe cluster list
// openframe cluster list --verbose 
// openframe cluster list --quiet

// Internal execution flow
func runListClusters(cmd *cobra.Command, args []string) error {
    service := utils.GetCommandService()
    
    clusters, err := service.ListClusters()
    if err != nil {
        return fmt.Errorf("failed to list clusters: %w", err)
    }
    
    globalFlags := utils.GetGlobalFlags()
    return service.DisplayClusterList(clusters, globalFlags.List.Quiet, globalFlags.Global.Verbose)
}
```

The command integrates with the global flag system for consistent behavior across commands, validates input flags during pre-run, and delegates cluster retrieval and display formatting to the service layer. It supports verbose and quiet output modes for different user preferences.
