A Cobra CLI command implementation for listing Kubernetes clusters managed by the OpenFrame CLI tool.

## Key Components

**Main Exports:**
- `getListCmd()` - Creates and configures the cluster list command with flags and validation
- `runListClusters()` - Executes the cluster listing logic and displays results

**Dependencies:**
- Uses Cobra framework for CLI command structure
- Integrates with cluster models, utils, and service layers
- Supports global flags for verbose/quiet output modes

## Usage Example

```go
// Register the list command with the cluster parent command
clusterCmd.AddCommand(getListCmd())

// The command supports various invocation patterns:
// openframe cluster list
// openframe cluster list --verbose  
// openframe cluster list --quiet
```

The command follows a standard pattern of initialization → validation → execution, with proper error handling and flag management. It retrieves cluster information from all registered providers and displays them in a formatted table through the service layer's display functionality.
