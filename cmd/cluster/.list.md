A Cobra CLI command implementation that provides cluster listing functionality for the OpenFrame CLI tool, allowing users to view all managed Kubernetes clusters in a formatted table.

## Key Components

- **`getListCmd()`** - Creates and configures the `cluster list` Cobra command with proper flag initialization and validation
- **`runListClusters()`** - Command execution handler that retrieves cluster data and displays it using the service layer
- **Global flag integration** - Leverages shared utilities for flag management, validation, and command setup
- **Service abstraction** - Uses the command service to handle cluster retrieval and display formatting

## Usage Example

```go
// Register the list command with the cluster command group
clusterCmd.AddCommand(getListCmd())

// The command supports various output options:
// openframe cluster list                    // Standard output
// openframe cluster list --verbose          // Detailed information
// openframe cluster list --quiet            // Minimal output
```

The command follows a standard pattern with pre-run validation, service-based execution, and error handling. It integrates with the global flag system to support common CLI options like verbose and quiet modes, and displays cluster information including name, type, status, and node count from all registered providers.
