This file implements the "list" subcommand for the cluster management functionality, allowing users to display all Kubernetes clusters managed by OpenFrame CLI.

## Key Components

- **`getListCmd()`** - Creates and configures the cobra command for listing clusters with global flag initialization and validation
- **`runListClusters()`** - Executes the cluster listing logic, retrieving clusters from the service and displaying them in a formatted table
- **Command validation** - Pre-run validation of global and list-specific flags through the models package
- **Service integration** - Uses the command service for both cluster retrieval and display formatting

## Usage Example

```go
// The command is typically registered in the cluster command group
clusterCmd := &cobra.Command{
    Use: "cluster",
    Short: "Manage Kubernetes clusters",
}

// Add the list subcommand
clusterCmd.AddCommand(getListCmd())

// Command usage from CLI:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet
```

The command supports standard global flags like `--verbose` and `--quiet` for controlling output detail, and integrates with the broader cluster management system through the utils package's command service pattern.
