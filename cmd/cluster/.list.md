Defines the `list` subcommand for the OpenFrame CLI cluster management, enabling users to view all registered Kubernetes clusters across different providers.

## Key Components

- **`getListCmd()`** - Creates and configures the cobra command with flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic by fetching clusters from the service and displaying them

## Usage Example

```go
// Command usage from CLI
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet

// Internal usage in command setup
listCmd := getListCmd()
rootCmd.AddCommand(listCmd)

// The command will:
// 1. Initialize and validate global flags
// 2. Call service.ListClusters() to fetch cluster data
// 3. Display results in a formatted table with cluster name, type, status, and node count
```

The command integrates with the global flag system for output formatting options (verbose/quiet modes) and uses the common command wrapper for consistent error handling and setup across all cluster commands.
