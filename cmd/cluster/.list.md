This file implements the `list` subcommand for the cluster management functionality in the OpenFrame CLI, providing a way to display all registered Kubernetes clusters in a formatted table.

## Key Components

- **`getListCmd()`** - Creates and configures the cobra command for listing clusters with validation and flag setup
- **`runListClusters()`** - Executes the cluster listing logic, retrieves clusters from the service and displays them

## Usage Example

```go
// Create the list command
listCmd := getListCmd()

// The command can be executed with various options:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet

// Example of how this integrates into the CLI:
rootCmd := &cobra.Command{Use: "openframe"}
clusterCmd := &cobra.Command{Use: "cluster"}
clusterCmd.AddCommand(getListCmd())
rootCmd.AddCommand(clusterCmd)
```

The command supports global flags like `--verbose` and `--quiet` for controlling output verbosity, and includes comprehensive validation through the PreRunE hook. It leverages the common command setup wrapper for consistent error handling and initialization across the CLI.
