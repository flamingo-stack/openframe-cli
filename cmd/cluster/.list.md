This file defines the "list" subcommand for the cluster module, enabling users to display all Kubernetes clusters managed by OpenFrame CLI.

## Key Components

- **`getListCmd()`** - Creates and configures the cobra command for listing clusters with flags initialization, validation, and command setup
- **`runListClusters()`** - Executes the cluster listing logic by retrieving clusters from the service and displaying them in a formatted table

## Usage Example

```go
// The command is typically registered in the parent cluster command
clusterCmd.AddCommand(getListCmd())

// Command line usage examples:
// openframe cluster list
// openframe cluster list --verbose  
// openframe cluster list --quiet
```

The command supports global flags like `--verbose` for detailed output and `--quiet` for minimal output. It validates flags during pre-execution, retrieves cluster information from all registered providers, and displays the results in a user-friendly table format showing cluster name, type, status, and node count.
