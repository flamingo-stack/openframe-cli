A Cobra CLI command implementation that provides functionality to list all Kubernetes clusters managed by the OpenFrame CLI, displaying cluster information in a formatted table.

## Key Components

- **`getListCmd()`** - Creates and configures the `list` command with flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic by calling the service layer and displaying results
- **Command validation** - Pre-run validation of global and list-specific flags
- **Service integration** - Uses the command service to retrieve and display cluster data

## Usage Example

```go
// Register the list command with the cluster command group
clusterCmd.AddCommand(getListCmd())

// The command can be invoked as:
// openframe cluster list
// openframe cluster list --verbose  
// openframe cluster list --quiet
```

The command integrates with the global flag system to support verbose and quiet output modes, validates input flags before execution, and delegates the actual cluster retrieval and display logic to the service layer. It follows the standard Cobra command pattern with proper error handling and flag management.
