This file implements a Cobra CLI command for listing Kubernetes clusters managed by the OpenFrame CLI, with support for various display options and global flag validation.

## Key Components

- **`getListCmd()`** - Creates and configures the `list` subcommand with flags, validation, and help text
- **`runListClusters()`** - Executes the cluster listing logic by fetching clusters from the service and displaying them in a formatted table
- **Global flags integration** - Utilizes shared flags like `--verbose` and `--quiet` for customizable output formatting

## Usage Example

```go
// Command usage examples:
// openframe cluster list
// openframe cluster list --verbose
// openframe cluster list --quiet

// The command will display cluster information in a table format:
// NAME        TYPE    STATUS    NODES
// dev-cluster k3s     running   3
// prod-cluster k8s    stopped   5
```

The command integrates with the cluster service to retrieve and display cluster data, supporting both verbose output for detailed information and quiet mode for minimal output. All global flags are validated before execution to ensure proper configuration.
