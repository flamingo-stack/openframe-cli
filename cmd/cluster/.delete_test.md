Test file that validates the cluster delete command functionality using a mock execution environment.

## Key Components

- **TestDeleteCommand**: Main test function that validates the delete command behavior
- **setupFunc**: Initializes a mock executor for testing command execution
- **teardownFunc**: Resets global flags after test completion
- **init()**: Ensures test mode is properly initialized

## Usage Example

```go
// Run the delete command test
func TestDeleteCommand(t *testing.T) {
    setupFunc := func() {
        utils.SetTestExecutor(testutil.NewTestMockExecutor())
    }
    teardownFunc := func() {
        utils.ResetGlobalFlags()
    }

    testutil.TestClusterCommand(t, "delete", getDeleteCmd, setupFunc, teardownFunc)
}
```

The test follows a standard setup-execute-teardown pattern, using a mock executor to simulate command execution without affecting real cluster resources. The `testutil.TestClusterCommand` helper handles the core test logic for cluster commands.
