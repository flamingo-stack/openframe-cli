Implements the cluster cleanup command for the OpenFrame CLI, providing functionality to remove unused Docker images and resources from Kubernetes cluster nodes to free up disk space.

## Key Components

- **`getCleanupCmd()`**: Creates and configures the cobra command with flags, validation, and aliases
- **`runCleanupCluster()`**: Main execution function that handles cluster selection, type detection, and cleanup operations
- **`GetCleanupCmdForTesting()`**: Test helper function that exposes the command for unit testing

## Usage Example

```go
// Create the cleanup command
cleanupCmd := getCleanupCmd()

// Add to parent cluster command
clusterCmd.AddCommand(cleanupCmd)

// Command supports various invocation patterns:
// openframe cluster cleanup                    # Interactive cluster selection
// openframe cluster cleanup my-cluster        # Target specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmations
```

The command integrates with the broader cluster management system, using shared utilities for flag management, cluster service operations, and UI interactions. It includes comprehensive error handling and user-friendly operation feedback through the operations UI component.
