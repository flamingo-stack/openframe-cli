Provides the `cleanup` command for removing unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **`getCleanupCmd()`** - Creates the cobra command with flags, validation, and help text
- **`runCleanupCluster()`** - Main execution logic that handles cluster selection, type detection, and cleanup operations
- **`GetCleanupCmdForTesting()`** - Test-friendly command constructor

The command supports optional cluster name argument, interactive cluster selection, and includes safety confirmations unless `--force` is used.

## Usage Example

```go
// Create cleanup command
cleanupCmd := getCleanupCmd()

// Command usage patterns:
// openframe cluster cleanup                    # Interactive selection
// openframe cluster cleanup my-cluster        # Specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmations

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command integrates with the cluster service layer to detect cluster types (Docker, Kind, etc.) and performs appropriate cleanup operations while providing user-friendly progress feedback through the operations UI.
