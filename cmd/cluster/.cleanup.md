Implements the cluster cleanup command for the OpenFrame CLI, removing unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cleanup command with flags, validation, and help text
- **`runCleanupCluster()`** - Main execution logic for cluster cleanup operations
- **`GetCleanupCmdForTesting()`** - Exported function for unit testing access to the command

## Usage Example

```go
// Create cleanup command instance
cleanupCmd := getCleanupCmd()

// Command usage examples:
// openframe cluster cleanup                    # Interactive cluster selection
// openframe cluster cleanup my-cluster        # Cleanup specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmation prompts

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command follows a typical flow: validates flags, lists available clusters, provides interactive selection UI, detects cluster type, executes cleanup through the service layer, and displays operation status. It integrates with the global flag system and includes proper error handling and user-friendly messaging through the operations UI component.
