Handles cluster resource cleanup operations by removing unused Docker images and resources to free disk space on cluster nodes.

## Key Components

- **getCleanupCmd()** - Creates the cobra command with cleanup-specific flags and validation
- **runCleanupCluster()** - Main execution logic that handles cluster selection, type detection, and cleanup operations
- **GetCleanupCmdForTesting()** - Test-accessible command constructor

## Usage Example

```go
// Create cleanup command
cleanupCmd := getCleanupCmd()

// Add to parent command
rootCmd.AddCommand(cleanupCmd)

// Command supports these usage patterns:
// openframe cluster cleanup
// openframe cluster cleanup my-cluster
// openframe cluster cleanup my-cluster --force
```

The command provides interactive cluster selection when no name is specified, validates cleanup flags before execution, and uses a friendly UI to show operation progress and results. It integrates with the service layer to detect cluster types and perform platform-specific cleanup operations.
