This file implements the cluster cleanup command functionality for the OpenFrame CLI, providing a way to remove unused Docker images and resources from cluster nodes to free up disk space.

## Key Components

- **getCleanupCmd()** - Creates and configures the cobra command for cluster cleanup operations
- **runCleanupCluster()** - Main execution function that handles cluster selection, type detection, and cleanup orchestration
- **GetCleanupCmdForTesting()** - Test-friendly function that exposes the cleanup command for unit testing

## Usage Example

```go
// Create cleanup command
cleanupCmd := getCleanupCmd()

// Execute cleanup on specific cluster
err := runCleanupCluster(cleanupCmd, []string{"my-cluster"})
if err != nil {
    log.Fatal(err)
}

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command supports interactive cluster selection when no name is provided, includes safety confirmations unless forced, and provides user-friendly progress indicators through the operations UI. It integrates with the global flag system for consistent behavior across the CLI and validates all flags before execution.
