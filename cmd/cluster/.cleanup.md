Provides a CLI command for cleaning up unused cluster resources like Docker images to free disk space on development clusters.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cleanup Cobra command with flags, validation, and aliases
- **`runCleanupCluster()`** - Core execution logic that handles cluster selection, type detection, and cleanup operations
- **`GetCleanupCmdForTesting()`** - Test-friendly wrapper function for accessing the cleanup command

## Usage Example

```go
// Create cleanup command
cleanupCmd := getCleanupCmd()

// Command supports various invocations:
// openframe cluster cleanup
// openframe cluster cleanup my-cluster
// openframe cluster cleanup my-cluster --force

// The command will:
// 1. List available clusters if none specified
// 2. Prompt for cluster selection with confirmation
// 3. Detect cluster type (Docker, k3d, etc.)
// 4. Execute cleanup operations
// 5. Display friendly progress and result messages

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command integrates with the cluster service layer to perform actual cleanup operations and uses a UI component to provide user-friendly prompts and status updates during the cleanup process.
