Implements the cluster cleanup command for the OpenFrame CLI, allowing users to remove unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the `cleanup` command with flags, validation, and aliases
- **`runCleanupCluster()`** - Executes the cleanup operation including cluster selection, type detection, and resource cleanup
- **`GetCleanupCmdForTesting()`** - Returns the cleanup command for testing purposes

## Usage Example

```go
// Register the cleanup command
cleanupCmd := getCleanupCmd()
rootCmd.AddCommand(cleanupCmd)

// Command usage examples:
// openframe cluster cleanup
// openframe cluster cleanup my-cluster
// openframe cluster cleanup my-cluster --force

// Testing usage
testCmd := GetCleanupCmdForTesting()
```

The command supports interactive cluster selection when no name is provided, includes force flag for non-interactive cleanup, and provides comprehensive error handling with friendly UI feedback. It integrates with the service layer to detect cluster types and perform the actual cleanup operations while displaying operation progress to users.
