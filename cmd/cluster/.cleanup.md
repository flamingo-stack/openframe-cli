Implements the `cleanup` subcommand for removing unused cluster resources like Docker images to free disk space on development clusters.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cleanup cobra command with flags, validation, and aliases
- **`runCleanupCluster()`** - Main execution logic that handles cluster selection, type detection, and cleanup operations
- **`GetCleanupCmdForTesting()`** - Exports the cleanup command for unit testing

## Usage Example

```go
// Register the cleanup command with the cluster command group
clusterCmd.AddCommand(getCleanupCmd())

// Command usage from CLI:
// openframe cluster cleanup                    # Interactive cluster selection
// openframe cluster cleanup my-cluster        # Clean specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmation prompts
```

The command integrates with the broader cluster management system, using the service layer for actual cleanup operations and the UI layer for user interactions and progress feedback. It supports both interactive and non-interactive modes through the `--force` flag.
