Provides a Cobra CLI command for cleaning up unused Docker resources and images from development clusters to free disk space.

## Key Components

- **getCleanupCmd()** - Creates and configures the main cleanup command with flags, validation, and aliases
- **runCleanupCluster()** - Executes the cleanup operation with cluster selection, type detection, and user-friendly UI
- **GetCleanupCmdForTesting()** - Test helper function that returns the cleanup command for unit testing

## Usage Example

```go
// Get the cleanup command and add it to the cluster command
cleanupCmd := getCleanupCmd()
clusterCmd.AddCommand(cleanupCmd)

// The command supports multiple usage patterns:
// openframe cluster cleanup                    # Interactive cluster selection
// openframe cluster cleanup my-cluster         # Target specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmations

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command integrates with the cluster service layer to list available clusters, detect cluster types (Docker Desktop, Kind, etc.), and perform cleanup operations. It includes comprehensive error handling and provides a user-friendly interface with operation status messages and confirmation prompts when not using the `--force` flag.
