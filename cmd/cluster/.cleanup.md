Implements the cluster cleanup command for removing unused resources from Kubernetes clusters. This command helps free disk space by cleaning up Docker images and other unused resources on cluster nodes.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cleanup cobra command with flags and validation
- **`runCleanupCluster()`** - Main execution function that handles cluster selection, type detection, and cleanup operations
- **`GetCleanupCmdForTesting()`** - Test helper function that exposes the cleanup command for testing

## Usage Example

```go
// Create the cleanup command
cleanupCmd := getCleanupCmd()

// Command supports various usage patterns:
// openframe cluster cleanup                    (interactive selection)
// openframe cluster cleanup my-cluster        (specific cluster)
// openframe cluster cleanup my-cluster --force (skip confirmations)

// The command handles:
// - Cluster listing and selection via UI
// - Cluster type detection
// - Resource cleanup execution
// - User-friendly progress and error messages
```

The command integrates with the cluster service layer for actual cleanup operations and provides an interactive UI for cluster selection when no specific cluster name is provided.
