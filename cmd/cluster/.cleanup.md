Implements the cluster cleanup command for the OpenFrame CLI, removing unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **getCleanupCmd()** - Creates and configures the cleanup cobra command with flags, validation, and aliases
- **runCleanupCluster()** - Main execution logic that handles cluster selection, type detection, and cleanup operations
- **GetCleanupCmdForTesting()** - Exported function that returns the cleanup command for testing purposes

The command supports optional cluster name argument, force flag for non-interactive execution, and provides user-friendly prompts for cluster selection.

## Usage Example

```go
// Register the cleanup command
cleanupCmd := getCleanupCmd()
rootCmd.AddCommand(cleanupCmd)

// Command line usage examples:
// openframe cluster cleanup                    # Interactive cluster selection
// openframe cluster cleanup my-cluster        # Cleanup specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmation prompts

// For testing
func TestCleanupCommand(t *testing.T) {
    cmd := GetCleanupCmdForTesting()
    // Test command configuration and execution
}
```

The cleanup process includes cluster listing, user-friendly selection UI with confirmation prompts, cluster type detection, and execution of the actual cleanup operations through the service layer.
