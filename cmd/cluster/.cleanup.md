Provides a Cobra CLI command for cleaning up unused resources from Kubernetes clusters, helping free disk space by removing unused Docker images and other resources.

## Key Components

- **getCleanupCmd()** - Creates and configures the `cluster cleanup` Cobra command with flags and validation
- **runCleanupCluster()** - Main execution function that handles cluster selection, type detection, and cleanup operations
- **GetCleanupCmdForTesting()** - Exports the command for unit testing purposes

The command supports an optional cluster name argument, includes a `--force` flag for automated cleanup, and provides interactive cluster selection when no name is specified.

## Usage Example

```go
// Get the cleanup command and add to parent command
cleanupCmd := getCleanupCmd()
clusterCmd.AddCommand(cleanupCmd)

// Command usage examples:
// openframe cluster cleanup                    # Interactive selection
// openframe cluster cleanup my-cluster        # Cleanup specific cluster  
// openframe cluster cleanup my-cluster --force # Skip confirmation prompts

// Testing usage
testCmd := GetCleanupCmdForTesting()
```

The command integrates with the cluster service layer to detect cluster types (kind, k3s, etc.) and perform appropriate cleanup operations, with user-friendly UI feedback throughout the process.
