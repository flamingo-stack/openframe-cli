Implements the `cleanup` command for the OpenFrame CLI, providing functionality to clean up unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cleanup command with flags, validation, and aliases
- **`runCleanupCluster()`** - Main execution logic that handles cluster selection, type detection, and cleanup operation
- **`GetCleanupCmdForTesting()`** - Test-friendly export of the cleanup command

## Usage Example

```go
// Create cleanup command
cleanupCmd := getCleanupCmd()

// The command supports various usage patterns:
// openframe cluster cleanup                    # Interactive selection
// openframe cluster cleanup my-cluster         # Target specific cluster  
// openframe cluster cleanup my-cluster --force # Skip confirmations

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command integrates with the cluster service layer to detect cluster types, provides interactive cluster selection through the operations UI, and includes proper error handling and user feedback throughout the cleanup process.
