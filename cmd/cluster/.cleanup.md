Handles the cleanup command implementation for removing unused cluster resources and freeing disk space from development clusters.

## Key Components

- **getCleanupCmd()**: Creates the cobra command definition with flags, aliases, validation, and help text
- **runCleanupCluster()**: Main execution logic that lists clusters, handles selection, detects cluster type, and performs cleanup
- **GetCleanupCmdForTesting()**: Test-friendly wrapper for accessing the cleanup command

## Usage Example

```go
// Get the cleanup command for the CLI
cleanupCmd := getCleanupCmd()

// Add to parent cluster command
clusterCmd.AddCommand(cleanupCmd)

// Command supports these usage patterns:
// openframe cluster cleanup
// openframe cluster cleanup my-cluster  
// openframe cluster cleanup my-cluster --force

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command includes comprehensive validation through `ValidateCleanupFlags()`, interactive cluster selection via `SelectClusterForCleanup()`, and user-friendly progress messaging. It automatically detects cluster types and delegates the actual cleanup work to the service layer while providing rich CLI feedback throughout the operation.
