Implements the cluster cleanup command for the OpenFrame CLI, removing unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cleanup cobra command with flags and validation
- **`runCleanupCluster()`** - Main execution logic for cluster cleanup operations
- **`GetCleanupCmdForTesting()`** - Test-friendly command factory function

The command supports optional cluster name arguments, force flags, and provides interactive cluster selection when no name is specified.

## Usage Example

```go
// Register the cleanup command
cleanupCmd := getCleanupCmd()
rootCmd.AddCommand(cleanupCmd)

// The command can be used as:
// openframe cluster cleanup                    // Interactive selection
// openframe cluster cleanup my-cluster        // Specific cluster
// openframe cluster cleanup my-cluster --force // Skip confirmation

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command integrates with the service layer to detect cluster types (kind, k3d, etc.) and performs cleanup operations with user-friendly progress indicators and error handling.
