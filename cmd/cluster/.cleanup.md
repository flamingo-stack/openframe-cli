Implements the `openframe cluster cleanup` command for removing unused Docker resources and freeing disk space on cluster nodes.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cobra command with flags, validation, and aliases
- **`runCleanupCluster()`** - Main execution logic that handles cluster selection, type detection, and cleanup operations
- **`GetCleanupCmdForTesting()`** - Exports the command for testing purposes

## Usage Example

```go
// Create cleanup command
cleanupCmd := getCleanupCmd()

// Command supports various invocation patterns:
// openframe cluster cleanup                    # Interactive cluster selection
// openframe cluster cleanup my-cluster         # Clean specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmation prompts

// Access command for testing
testCmd := GetCleanupCmdForTesting()
```

The command includes interactive cluster selection with confirmation prompts (unless `--force` is used), validates cleanup-specific flags, detects cluster types automatically, and provides user-friendly operation status messages through the UI layer. It integrates with the service layer for actual cleanup operations and handles error cases gracefully.
