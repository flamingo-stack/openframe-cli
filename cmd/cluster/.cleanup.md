Implements the cluster cleanup command for the OpenFrame CLI, removing unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **`getCleanupCmd()`** - Creates and configures the cobra command with flags, validation, and aliases
- **`runCleanupCluster()`** - Main execution logic that handles cluster selection, type detection, and cleanup operation
- **`GetCleanupCmdForTesting()`** - Test-friendly command accessor

## Usage Example

```go
// Get the cleanup command
cleanupCmd := getCleanupCmd()

// Add to parent command
rootCmd.AddCommand(cleanupCmd)

// Command supports various usage patterns:
// openframe cluster cleanup                    # Interactive selection
// openframe cluster cleanup my-cluster         # Specific cluster
// openframe cluster cleanup my-cluster --force # Skip confirmation
```

The command integrates with the cluster service layer for operations and provides a user-friendly interface with operation status messages, cluster selection prompts, and confirmation dialogs unless the `--force` flag is used.
