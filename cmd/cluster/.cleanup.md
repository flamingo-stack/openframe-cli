Provides CLI command functionality for cleaning up unused cluster resources, removing Docker images and other artifacts to free disk space on development clusters.

## Key Components

- **getCleanupCmd()**: Creates the main cleanup cobra command with flags, validation, and help text
- **runCleanupCluster()**: Executes the cleanup operation including cluster selection, type detection, and resource cleanup
- **GetCleanupCmdForTesting()**: Returns cleanup command instance for testing purposes

## Usage Example

```go
// Get the cleanup command for CLI registration
cleanupCmd := getCleanupCmd()

// The command supports various usage patterns:
// openframe cluster cleanup                    // Interactive cluster selection
// openframe cluster cleanup my-cluster        // Cleanup specific cluster
// openframe cluster cleanup my-cluster --force // Skip confirmation prompts

// For testing
testCmd := GetCleanupCmdForTesting()
```

The command integrates with the cluster service layer to detect cluster types (k3d, kind, etc.) and performs cleanup operations with user-friendly prompts and progress feedback through the operations UI.
