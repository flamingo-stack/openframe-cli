Implements the cluster cleanup command functionality for the OpenFrame CLI, providing Docker image and resource cleanup for development clusters.

## Key Components

- **getCleanupCmd()** - Creates and configures the cobra command for cluster cleanup with flags and validation
- **runCleanupCluster()** - Core execution logic that handles cluster selection, type detection, and cleanup operations
- **GetCleanupCmdForTesting()** - Testing utility that exposes the cleanup command for unit tests

## Usage Example

```go
// Get the cleanup command for CLI registration
cleanupCmd := getCleanupCmd()

// Execute cleanup with cluster name argument
err := runCleanupCluster(cmd, []string{"my-cluster"})
if err != nil {
    log.Fatal(err)
}

// Use in testing
testCmd := GetCleanupCmdForTesting()
```

The command supports optional cluster name arguments, force flags for non-interactive cleanup, and integrates with the UI system for user-friendly cluster selection and operation feedback. It validates flags before execution and provides detailed error handling throughout the cleanup process.
