Implements the cluster cleanup command for the OpenFrame CLI, removing unused Docker images and resources from cluster nodes to free disk space.

## Key Components

- **getCleanupCmd()** - Creates and configures the Cobra command with flags, validation, and aliases
- **runCleanupCluster()** - Main execution logic that handles cluster selection, type detection, and cleanup operations
- **GetCleanupCmdForTesting()** - Test-friendly command constructor for unit testing

## Usage Example

```go
// Get the cleanup command
cleanupCmd := getCleanupCmd()

// Execute cleanup with specific cluster name
err := cleanupCmd.Execute([]string{"my-cluster"})

// Execute with force flag (no confirmation prompts)
cleanupCmd.Flags().Set("force", "true")
err = cleanupCmd.Execute([]string{"my-cluster", "--force"})

// Interactive cleanup (prompts for cluster selection)
err = cleanupCmd.Execute([]string{})
```

The command supports optional cluster name argument, interactive cluster selection when no name is provided, and includes validation for cleanup-specific flags. It integrates with the UI layer to provide user-friendly operation feedback and confirmation prompts unless the `--force` flag is used.
