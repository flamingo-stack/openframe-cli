This file defines the main cluster command and its subcommand structure for the OpenFrame CLI tool, providing Kubernetes cluster lifecycle management functionality.

## Key Components

- **GetClusterCmd()** - Returns the root cluster command with all subcommands attached
- **clusterCmd** - Main cobra command with aliases, help text, and prerequisite checking
- **Subcommands** - create, delete, list, status, cleanup operations
- **Global flags** - Shared configuration options across all cluster commands
- **Prerequisites checking** - Validates required tools before command execution

## Usage Example

```go
// Get the cluster command with all subcommands
clusterCmd := GetClusterCmd()

// Add to root CLI command
rootCmd.AddCommand(clusterCmd)

// The command supports these operations:
// openframe cluster create    - Interactive cluster creation
// openframe cluster delete    - Remove clusters  
// openframe cluster list      - Show managed clusters
// openframe cluster status    - Display cluster details
// openframe cluster cleanup   - Clean unused resources
// openframe k create          - Using alias 'k'
```

The command includes persistent pre-run hooks that display the OpenFrame logo and verify prerequisites like K3d installation before executing any cluster operations.
