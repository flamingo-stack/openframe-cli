This file provides the main entry point for the cluster command group in the OpenFrame CLI, handling Kubernetes cluster management operations.

## Key Components

- **GetClusterCmd()** - Main function that returns the root cluster command with all subcommands
- **Cluster Command** - Root cobra command with alias "k" for managing Kubernetes clusters
- **Subcommands** - Integrates create, delete, list, status, and cleanup operations
- **Global Flags** - Initializes and applies global configuration flags
- **Prerequisites Check** - Validates system requirements before command execution

## Usage Example

```go
// Get the cluster command group
clusterCmd := GetClusterCmd()

// The command supports these operations:
// openframe cluster create    - Create new K3d cluster
// openframe cluster delete    - Remove cluster
// openframe cluster list      - Show all clusters  
// openframe cluster status    - Display cluster info
// openframe cluster cleanup   - Clean unused resources

// Can also use the alias:
// openframe k create
// openframe k list
```

The command includes a persistent pre-run hook that checks prerequisites and displays the OpenFrame logo for subcommands. When called without arguments, it shows help information. All subcommands inherit global flags and the prerequisites validation.
