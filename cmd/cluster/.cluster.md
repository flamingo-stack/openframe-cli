Main entry point for cluster management commands, providing a Cobra command structure for Kubernetes cluster lifecycle operations.

## Key Components

- **GetClusterCmd()** - Returns the root cluster command with subcommands
- **PersistentPreRunE** - Validates prerequisites and displays UI logo for subcommands
- **RunE** - Shows help when no subcommand is specified
- **Subcommands** - Integrates create, delete, list, status, and cleanup operations

## Usage Example

```go
// Get the cluster command for CLI integration
clusterCmd := GetClusterCmd()

// Add to root command
rootCmd.AddCommand(clusterCmd)

// The command supports various operations:
// openframe cluster create    - Interactive cluster creation
// openframe cluster delete    - Remove clusters
// openframe cluster list      - Show managed clusters
// openframe cluster status    - Display cluster details
// openframe cluster cleanup   - Clean unused resources
```

The command includes global flag initialization, prerequisite checking, and a consistent UI experience across all cluster operations. It uses the alias "k" for shorter command invocation.
