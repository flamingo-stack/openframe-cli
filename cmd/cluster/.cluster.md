Main entry point for the cluster command group that provides Kubernetes cluster lifecycle management functionality through the OpenFrame CLI.

## Key Components

- **GetClusterCmd()** - Returns the root cluster command with all subcommands and global configuration
- **Subcommands** - create, delete, list, status, and cleanup operations for K3d clusters
- **Global Flags** - Shared configuration options across all cluster subcommands
- **Prerequisites Check** - Validates required dependencies before command execution

## Usage Example

```go
// Get the complete cluster command with all subcommands
clusterCmd := GetClusterCmd()

// The command supports these operations:
// openframe cluster create    - Interactive cluster creation
// openframe cluster delete    - Remove cluster and resources
// openframe cluster list      - Show managed clusters  
// openframe cluster status    - Display cluster details
// openframe cluster cleanup   - Clean unused resources

// Add to root CLI command
rootCmd.AddCommand(clusterCmd)
```

The command includes automatic logo display, prerequisite validation, and supports the alias `k` for shorter usage (`openframe k create`). Each subcommand inherits global flags and runs prerequisite checks before execution.
