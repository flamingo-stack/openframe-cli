This file defines the main cluster command and its structure for managing Kubernetes clusters in the OpenFrame CLI.

## Key Components

- **GetClusterCmd()** - Returns the root cluster command with all subcommands attached
- **Subcommands** - Integrates create, delete, list, status, and cleanup cluster operations
- **Global flags** - Initializes and adds global configuration flags via models package
- **Prerequisites check** - Validates system requirements before executing subcommands
- **UI integration** - Shows logo and provides consistent user experience

## Usage Example

```go
// Get the cluster command for CLI registration
clusterCmd := GetClusterCmd()

// The command supports these operations:
// openframe cluster create    - Interactive cluster creation
// openframe cluster delete    - Remove clusters
// openframe cluster list      - Show all clusters
// openframe cluster status    - Display cluster details
// openframe cluster cleanup   - Remove unused resources

// Can also use alias:
// openframe k create

// Running without subcommand shows help
// openframe cluster
```

The command includes persistent pre-run hooks that check prerequisites and display the logo for a polished CLI experience.
