Provides the main cluster command entry point for the Flamingo Stack OpenFrame CLI, managing Kubernetes cluster operations through Cobra CLI framework.

## Key Components

### Functions
- **`GetClusterCmd()`** - Returns the root cluster command with all subcommands and configuration

### Command Structure
- **Root command**: `cluster` (alias: `k`) - Main cluster management command
- **Subcommands**: create, delete, list, status, cleanup
- **Global flags**: Initialized through utils package
- **Prerequisites**: Docker and other dependencies checked before execution

## Usage Example

```go
// Get the cluster command for CLI registration
clusterCmd := cluster.GetClusterCmd()

// The command provides these operations:
// openframe cluster create    - Interactive cluster creation
// openframe cluster delete    - Remove clusters with cleanup
// openframe cluster list      - Show managed clusters
// openframe cluster status    - Display cluster details
// openframe cluster cleanup   - Remove unused resources

// Register with root command
rootCmd.AddCommand(clusterCmd)
```

The command includes persistent pre-run checks for prerequisites (Docker, K3d), shows the Flamingo logo for subcommands, and supports K3d clusters for local Kubernetes development. When called without subcommands, it displays help information.
