Main command handler for the OpenFrame cluster management module that provides Kubernetes cluster lifecycle operations.

## Key Components

**Main Export:**
- `GetClusterCmd()` - Returns the root cluster command with all subcommands attached

**Command Structure:**
- Root command: `cluster` (alias: `k`) - Manages Kubernetes clusters
- Subcommands: create, delete, list, status, cleanup
- Global flags initialization and attachment
- Prerequisites checking via `PersistentPreRunE`

**Key Features:**
- Interactive K3d cluster management for local development
- Logo display handling for CLI branding
- Comprehensive help text with examples
- Error handling for prerequisite validation

## Usage Example

```go
// Get the cluster command for CLI integration
clusterCmd := GetClusterCmd()

// The command supports various operations:
// openframe cluster create    - Interactive cluster creation
// openframe cluster delete    - Remove clusters
// openframe cluster list      - Show all clusters
// openframe cluster status    - Detailed cluster info
// openframe cluster cleanup   - Remove unused resources

// Command aliases are also supported:
// openframe k create          - Short form using 'k' alias
```

The command provides a complete cluster management interface with prerequisite checking, global flag support, and consistent UI branding across all subcommands.
