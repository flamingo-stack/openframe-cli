This file defines the main cluster command and orchestrates all cluster management functionality for the OpenFrame CLI tool.

## Key Components

- **`GetClusterCmd()`** - Primary function that creates and returns the root cluster command with all subcommands
- **Subcommands** - Integrates create, delete, list, status, and cleanup operations
- **Global flags initialization** - Sets up shared configuration options across all cluster commands
- **Prerequisites checking** - Validates system requirements before executing cluster operations
- **UI integration** - Handles logo display and user interface elements

## Usage Example

```go
// Get the complete cluster command tree
clusterCmd := GetClusterCmd()

// Add to your root command
rootCmd.AddCommand(clusterCmd)

// The command supports aliases and provides comprehensive help
// CLI usage examples:
// openframe cluster create
// openframe k list
// openframe cluster status my-cluster
// openframe cluster cleanup
```

The command includes persistent pre-run hooks that check prerequisites and display the UI logo, while the main run function shows help when no subcommand is specified. It supports the alias "k" for shorter command invocation.
