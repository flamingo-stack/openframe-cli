Main entry point for the cluster command group in the OpenFrame CLI, providing Kubernetes cluster lifecycle management with K3d support.

## Key Components

- **GetClusterCmd()** - Returns the root cluster command with all subcommands configured
- **clusterCmd** - Main cobra command with aliases, help text, and subcommand registration
- **PersistentPreRunE** - Validates prerequisites and displays UI elements for subcommands
- **Subcommands** - create, delete, list, status, and cleanup operations

## Usage Example

```go
// Register the cluster command with the CLI
rootCmd := &cobra.Command{Use: "openframe"}
rootCmd.AddCommand(cluster.GetClusterCmd())

// The command supports these operations:
// openframe cluster create    - Interactive cluster creation
// openframe k delete          - Delete cluster (using alias)
// openframe cluster list      - Show managed clusters
// openframe cluster status    - Display cluster details
// openframe cluster cleanup   - Remove unused resources
```

The command uses a persistent pre-run hook to check system prerequisites before executing any cluster operations and displays the OpenFrame logo for better user experience.
