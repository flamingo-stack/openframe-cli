A comprehensive test suite for the root command functionality of the OpenFrame CLI, verifying command structure, version handling, help output, and system initialization.

## Key Components

- **TestRootCommand**: Validates basic command properties (Use, Short, Long descriptions)
- **TestRootCommandHelp**: Tests help flag output using testutil helpers
- **TestRootCommandVersion**: Verifies version flag functionality
- **TestGetRootCmd**: Tests root command creation with custom version info
- **TestSystemService**: Validates system service initialization and log directory creation
- **TestVersionInfo**: Ensures default version information is properly initialized
- **TestExecuteWithVersion**: Panic-safety test for the execute function

## Usage Example

```go
// Run specific test
go test -run TestRootCommand

// Test version command functionality
cmd := GetRootCmd(VersionInfo{
    Version: "1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
})

// Verify command structure
if cmd.Use != "openframe" {
    t.Errorf("unexpected command name: %s", cmd.Use)
}
```

The tests use `testutil.TestCLICommand` for consistent CLI testing and enable test mode to suppress UI output during execution.
