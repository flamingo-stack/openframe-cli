Test file for the root command functionality of the OpenFrame CLI. This file validates the core command structure, version handling, and system service initialization.

## Key Components

- **TestRootCommand**: Tests the basic command structure and metadata
- **TestRootCommandHelp/Version**: Tests command-line flag behavior using testutil
- **TestGetRootCmd**: Validates root command creation with custom version info
- **TestSystemService**: Tests system service initialization and log directory creation
- **TestVersionInfo**: Validates default version information structure
- **TestExecuteWithVersion**: Verifies the execute function exists without panicking

## Usage Example

```go
// Test custom version information
func TestCustomVersion(t *testing.T) {
    versionInfo := VersionInfo{
        Version: "1.0.0",
        Commit:  "abc123",
        Date:    "2024-01-01",
    }
    
    cmd := GetRootCmd(versionInfo)
    expected := "1.0.0 (abc123) built on 2024-01-01"
    
    if cmd.Version != expected {
        t.Errorf("expected version %q, got %q", expected, cmd.Version)
    }
}

// Test system service functionality
func TestServiceSetup(t *testing.T) {
    service := config.NewSystemService()
    err := service.Initialize()
    
    if err != nil {
        t.Fatalf("service initialization failed: %v", err)
    }
}
```
