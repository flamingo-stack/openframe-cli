This file serves as the main entry point for the OpenFrame CLI, defining the root Cobra command and orchestrating all subcommands for Kubernetes cluster management and development tools.

## Key Components

- **VersionInfo struct**: Contains version, commit, and build date information
- **GetRootCmd()**: Returns the configured root command with version info
- **Execute()/ExecuteWithVersion()**: Main execution functions for the CLI
- **buildRootCommand()**: Constructs the root command with subcommands and configuration
- Subcommand getters: `getClusterCmd()`, `getChartCmd()`, `getBootstrapCmd()`, `getDevCmd()`

## Usage Example

```go
// Basic execution with default version
func main() {
    if err := cmd.Execute(); err != nil {
        os.Exit(1)
    }
}

// Execution with custom version info
versionInfo := cmd.VersionInfo{
    Version: "1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}

if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    fmt.Fprintf(os.Stderr, "Error: %v\n", err)
    os.Exit(1)
}

// Get root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
```

The CLI provides interactive wizards for cluster setup, Helm chart management, bootstrapping, and development workflows with both interactive and flag-based operation modes.
