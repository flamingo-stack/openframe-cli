Main entry point for the OpenFrame CLI that defines the root command structure and version management. This file establishes the command hierarchy and provides interactive wizard functionality for Kubernetes cluster management.

## Key Components

- **VersionInfo**: Struct containing version, commit, and build date information
- **GetRootCmd()**: Creates the root cobra command with version info and subcommands
- **Execute()**: Entry point functions for running the CLI with default or custom version info
- **buildRootCommand()**: Internal function that constructs the complete command structure
- **Subcommand getters**: Functions that retrieve commands for cluster, chart, bootstrap, and dev operations

## Usage Example

```go
// Run CLI with default version
func main() {
    if err := cmd.Execute(); err != nil {
        os.Exit(1)
    }
}

// Run CLI with custom version info
func main() {
    version := cmd.VersionInfo{
        Version: "1.0.0",
        Commit:  "abc123",
        Date:    "2024-01-15",
    }
    
    if err := cmd.ExecuteWithVersion(version); err != nil {
        os.Exit(1)
    }
}

// Get root command for testing
func TestRootCommand() {
    rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
    // Test command structure...
}
```

The CLI provides interactive wizards for new users and supports flag-based operation for automation, with global verbose and silent flags available across all subcommands.
