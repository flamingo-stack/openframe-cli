This file implements the root command and entry point for the OpenFrame CLI, a Kubernetes platform management tool that provides interactive wizards for cluster bootstrapping and development operations.

## Key Components

- **VersionInfo struct**: Holds version metadata (version, commit, date) for the CLI
- **GetRootCmd()**: Creates the root cobra command with version information
- **Execute()/ExecuteWithVersion()**: Entry points for running the CLI with default or custom version info
- **Subcommand getters**: Functions that return cluster, chart, bootstrap, and dev subcommands

## Usage Example

```go
// Basic execution with default version
if err := cmd.Execute(); err != nil {
    log.Fatal(err)
}

// Execute with custom version info
versionInfo := cmd.VersionInfo{
    Version: "1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}
if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    log.Fatal(err)
}

// Get root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
```

The CLI features global flags for verbose/silent output, custom usage templates, and automatic configuration initialization through a system service layer. When run without subcommands, it displays the OpenFrame logo and help information.
