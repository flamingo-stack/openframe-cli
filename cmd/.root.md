This file defines the root command structure for the OpenFrame CLI, a Kubernetes cluster bootstrapping and development tool built with Cobra. It serves as the main entry point and orchestrates all subcommands.

## Key Components

- **VersionInfo**: Struct for storing CLI version metadata (version, commit, date)
- **GetRootCmd()**: Factory function that builds the main command with version info
- **Execute()**: Default entry point using standard version info
- **ExecuteWithVersion()**: Entry point with custom version information
- **Subcommand factories**: Functions that return cluster, chart, bootstrap, and dev commands

## Usage Example

```go
// Use default version info
if err := Execute(); err != nil {
    fmt.Fprintf(os.Stderr, "Error: %v\n", err)
    os.Exit(1)
}

// Use custom version info (typically set during build)
versionInfo := VersionInfo{
    Version: "v1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}

rootCmd := GetRootCmd(versionInfo)
if err := rootCmd.Execute(); err != nil {
    os.Exit(1)
}
```

The CLI provides interactive wizard-style prompts for new users, supports multiple Kubernetes cluster types (K3d, Kind, cloud providers), and includes developer tools like Telepresence integration. Global flags include `--verbose` and `--silent` for output control.
