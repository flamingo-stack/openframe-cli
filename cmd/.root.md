A root command implementation for the OpenFrame CLI that manages Kubernetes cluster bootstrapping and development operations. It sets up the main command structure with subcommands and provides version management capabilities.

## Key Components

- **VersionInfo**: Struct holding version, commit hash, and build date information
- **GetRootCmd()**: Returns the configured root cobra command with version info
- **buildRootCommand()**: Constructs the root command with interactive wizard features
- **Execute/ExecuteWithVersion()**: Entry points for running the CLI with optional version info
- **Command getters**: Functions that return subcommands for cluster, chart, bootstrap, and dev operations

## Usage Example

```go
// Basic execution with default version
func main() {
    if err := cmd.Execute(); err != nil {
        os.Exit(1)
    }
}

// Execute with custom version info
versionInfo := cmd.VersionInfo{
    Version: "v1.2.3",
    Commit:  "abc123",
    Date:    "2024-01-01",
}

if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    os.Exit(1)
}

// Get root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
rootCmd.SetArgs([]string{"cluster", "create"})
err := rootCmd.Execute()
```

The root command provides a wizard-style interface for OpenFrame operations, with global flags for verbose/silent output and custom usage templates for better user experience.
