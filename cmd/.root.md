The root command entry point for the OpenFrame CLI, providing version management, global configuration, and subcommand orchestration with an interactive terminal UI.

## Key Components

- **VersionInfo**: Struct holding CLI version metadata (version, commit, date)
- **GetRootCmd()**: Main factory function that builds the root cobra command
- **Execute()/ExecuteWithVersion()**: Entry points for CLI execution with version handling
- **buildRootCommand()**: Internal command builder with custom templates and configuration
- **Subcommand getters**: Factory functions for cluster, chart, bootstrap, and dev commands

## Usage Example

```go
// Basic execution with default version
func main() {
    if err := cmd.Execute(); err != nil {
        os.Exit(1)
    }
}

// Execution with custom version info
versionInfo := cmd.VersionInfo{
    Version: "1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}

if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    log.Fatal(err)
}

// Getting root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
rootCmd.SetArgs([]string{"cluster", "create"})
err := rootCmd.Execute()
```

The command includes global flags for verbose/silent modes, custom usage templates, and automatic logo display when run without subcommands. It initializes system configuration on startup and follows the cobra CLI pattern for Kubernetes tooling.
