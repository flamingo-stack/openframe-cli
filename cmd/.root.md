This file defines the root command and entry point for the OpenFrame CLI, a Kubernetes platform bootstrapping tool with interactive wizard-style interfaces.

## Key Components

- **`VersionInfo`**: Struct holding version, commit, and build date information
- **`GetRootCmd()`**: Constructs the main cobra command with subcommands and global flags
- **`Execute()` / `ExecuteWithVersion()`**: Entry points for running the CLI with default or custom version info
- **Subcommand getters**: `getClusterCmd()`, `getChartCmd()`, `getBootstrapCmd()`, `getDevCmd()`
- **Global flags**: `--verbose`, `--silent` for output control

## Usage Example

```go
// Run with default version info
func main() {
    if err := cmd.Execute(); err != nil {
        os.Exit(1)
    }
}

// Run with custom version info
versionInfo := cmd.VersionInfo{
    Version: "v1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}
if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    os.Exit(1)
}

// Get root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
rootCmd.SetArgs([]string{"--help"})
rootCmd.Execute()
```

The CLI provides interactive wizards for Kubernetes cluster management, Helm chart operations, bootstrapping, and development workflows with both guided and flag-based operation modes.
