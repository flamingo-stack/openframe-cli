Defines the root command and entry point for the OpenFrame CLI, a Kubernetes cluster bootstrapping and development tool built with Cobra.

## Key Components

- **VersionInfo**: Struct holding version metadata (version, commit, date)
- **GetRootCmd()**: Creates the root cobra command with version information
- **Execute()/ExecuteWithVersion()**: Entry points to run the CLI with optional version info
- **Subcommand getters**: Functions returning cluster, chart, bootstrap, and dev commands
- **DefaultVersionInfo**: Default version information for development builds

## Usage Example

```go
// Basic execution with default version
func main() {
    if err := cmd.Execute(); err != nil {
        os.Exit(1)
    }
}

// Execute with custom version info
versionInfo := cmd.VersionInfo{
    Version: "1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}
if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    os.Exit(1)
}

// Get root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
```

The CLI provides interactive wizards for Kubernetes management, Helm integration, and developer tools with both interactive and flag-based operation modes.
