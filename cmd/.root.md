This file serves as the main entry point for the OpenFrame CLI, defining the root Cobra command structure and orchestrating all subcommands for Kubernetes cluster management and development tools.

## Key Components

- **VersionInfo**: Struct holding version metadata (version, commit, date)
- **GetRootCmd()**: Returns configured root command with version info
- **Execute()/ExecuteWithVersion()**: Main execution functions for the CLI
- **buildRootCommand()**: Constructs the root command with subcommands and flags
- Subcommand getters: `getClusterCmd()`, `getChartCmd()`, `getBootstrapCmd()`, `getDevCmd()`

## Usage Example

```go
// Execute CLI with default version
func main() {
    if err := cmd.Execute(); err != nil {
        os.Exit(1)
    }
}

// Execute with custom version info
versionInfo := cmd.VersionInfo{
    Version: "v1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}

if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    log.Fatal(err)
}

// Get root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
```

The root command provides an interactive wizard interface for Kubernetes platform bootstrapping, with support for cluster management, Helm integration, and developer tools like Telepresence intercepts.
