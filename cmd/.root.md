The root command module for the OpenFrame CLI, defining the main entry point and command structure for a Kubernetes platform bootstrapping tool.

## Key Components

- **VersionInfo**: Struct containing version metadata (version, commit, date)
- **DefaultVersionInfo**: Default version information for development builds
- **GetRootCmd()**: Returns the configured root cobra command with version info
- **Execute()**: Main execution function using default version info
- **ExecuteWithVersion()**: Execution function with custom version information

## Usage Example

```go
// Execute with default version
if err := Execute(); err != nil {
    log.Fatal(err)
}

// Execute with custom version info
versionInfo := VersionInfo{
    Version: "1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}
if err := ExecuteWithVersion(versionInfo); err != nil {
    log.Fatal(err)
}

// Get root command for testing
rootCmd := GetRootCmd(DefaultVersionInfo)
rootCmd.SetArgs([]string{"--help"})
rootCmd.Execute()
```

The root command integrates four main subcommands (cluster, chart, bootstrap, dev) and provides global flags for verbose and silent modes. It includes custom usage templates and initializes system configuration on startup.
