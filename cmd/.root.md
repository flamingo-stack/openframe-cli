Defines the root command structure for the OpenFrame CLI, a Kubernetes platform bootstrapping tool with interactive wizard-style prompts and developer tools.

## Key Components

- **`VersionInfo`** - Struct holding version, commit, and build date information
- **`GetRootCmd()`** - Returns configured root cobra command with version info
- **`Execute()`/`ExecuteWithVersion()`** - Entry points for running the CLI
- **Subcommand getters** - `getClusterCmd()`, `getChartCmd()`, `getBootstrapCmd()`, `getDevCmd()`

## Usage Example

```go
// Basic execution with default version
err := cmd.Execute()
if err != nil {
    log.Fatal(err)
}

// Execute with custom version info
versionInfo := cmd.VersionInfo{
    Version: "1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}
err := cmd.ExecuteWithVersion(versionInfo)

// Get root command for testing
rootCmd := cmd.GetRootCmd(versionInfo)
rootCmd.SetArgs([]string{"cluster", "create"})
err := rootCmd.Execute()
```

The CLI provides interactive Kubernetes cluster management with features like K3d/Kind support, Helm integration, ArgoCD app-of-apps patterns, and developer tools including Telepresence intercepts.
