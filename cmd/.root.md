This file defines the root command and entry point for the OpenFrame CLI, a Kubernetes cluster bootstrapping and development tool built with Cobra.

## Key Components

- **VersionInfo** - Struct holding version, commit, and build date information
- **GetRootCmd()** - Returns the configured root Cobra command with subcommands
- **Execute()** - Main entry point using default version info
- **ExecuteWithVersion()** - Entry point with custom version info
- Subcommand getters for cluster, chart, bootstrap, and dev commands

## Usage Example

```go
// Execute CLI with default version
if err := cmd.Execute(); err != nil {
    log.Fatal(err)
}

// Execute with custom version info
versionInfo := cmd.VersionInfo{
    Version: "v1.0.0",
    Commit:  "abc123",
    Date:    "2024-01-01",
}
if err := cmd.ExecuteWithVersion(versionInfo); err != nil {
    log.Fatal(err)
}

// Get root command for testing
rootCmd := cmd.GetRootCmd(cmd.DefaultVersionInfo)
rootCmd.SetArgs([]string{"cluster", "create"})
```

The root command provides an interactive wizard-style interface with verbose/silent flags, custom usage templates, and integrates with the config service for initialization. When run without subcommands, it displays the OpenFrame logo and help text.
