This file defines the Cobra CLI command for the OpenFrame bootstrap operation, which provides a streamlined setup process for complete OpenFrame environments.

## Key Components

- **GetBootstrapCmd()**: Returns a configured Cobra command that orchestrates cluster creation and chart installation in a single operation
- **Command Flags**: 
  - `deployment-mode`: Specifies deployment type (oss-tenant, saas-tenant, saas-shared)
  - `non-interactive`: Enables CI/CD-friendly execution
  - `verbose`: Shows detailed logging including ArgoCD sync progress

## Usage Example

```go
// Get the bootstrap command and add to root command
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// The command supports various execution modes:
// Interactive: openframe bootstrap
// With cluster name: openframe bootstrap my-cluster  
// CI/CD mode: openframe bootstrap --deployment-mode=oss-tenant --non-interactive
// Verbose: openframe bootstrap --verbose --deployment-mode=saas-shared
```

The bootstrap command internally calls `bootstrap.NewService().Execute()` to handle the actual orchestration of cluster creation and chart installation, providing a unified entry point for OpenFrame environment setup.
