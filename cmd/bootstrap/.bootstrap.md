Provides a streamlined command for complete OpenFrame environment setup, combining cluster creation and chart installation in a single operation.

## Key Components

- **GetBootstrapCmd()** - Returns a cobra command that bootstraps a complete OpenFrame environment by sequentially running cluster creation and ArgoCD/OpenFrame chart installation

## Usage Example

```go
package main

import (
    "github.com/your-org/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add bootstrap command to CLI
    bootstrapCmd := bootstrap.GetBootstrapCmd()
    rootCmd.AddCommand(bootstrapCmd)
    
    rootCmd.Execute()
}
```

The bootstrap command supports multiple execution modes:

```bash
# Interactive mode (default)
openframe bootstrap

# With custom cluster name  
openframe bootstrap my-cluster

# CI/CD mode with pre-configured settings
openframe bootstrap --deployment-mode=saas-shared --non-interactive

# Verbose logging for troubleshooting
openframe bootstrap -v --deployment-mode=oss-tenant
```

The command accepts deployment mode flags (`oss-tenant`, `saas-tenant`, `saas-shared`), non-interactive mode for automation, and verbose logging options. It delegates the actual bootstrap logic to the internal bootstrap service while providing a clean CLI interface.
