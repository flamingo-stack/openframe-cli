Provides a streamlined OpenFrame environment setup command that combines cluster creation and chart installation into a single bootstrap operation.

## Key Components

- **GetBootstrapCmd()** - Returns a configured Cobra command for the bootstrap operation
- **Command Configuration** - Sets up usage patterns, help text, and command-line flags
- **Flag Definitions** - Supports deployment mode selection, non-interactive mode, and verbose logging

## Usage Example

```go
import (
    "github.com/your-org/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

// Add bootstrap command to root command
rootCmd := &cobra.Command{Use: "openframe"}
rootCmd.AddCommand(bootstrap.GetBootstrapCmd())

// Command supports various execution modes:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # Custom cluster name
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive --verbose      # CI/CD mode with detailed logs
```

The bootstrap command orchestrates the complete OpenFrame setup process by sequentially executing cluster creation and chart installation, with support for different deployment modes (oss-tenant, saas-tenant, saas-shared) and operational modes (interactive, non-interactive, verbose).
