Provides the cobra command configuration for bootstrapping a complete OpenFrame environment with cluster creation and chart installation in a single streamlined operation.

## Key Components

- **GetBootstrapCmd()**: Returns a configured cobra.Command that orchestrates the complete OpenFrame bootstrap process
- **bootstrap.NewService()**: Internal service that handles the actual bootstrap execution logic

## Usage Example

```go
// Get the bootstrap command for CLI registration
bootstrapCmd := GetBootstrapCmd()

// The command supports various execution modes:
// Interactive mode (default)
// openframe bootstrap

// With custom cluster name
// openframe bootstrap my-cluster

// Non-interactive with deployment mode
// openframe bootstrap --deployment-mode=oss-tenant --non-interactive

// Verbose logging
// openframe bootstrap -v --deployment-mode=saas-shared
```

The command combines cluster creation and chart installation (ArgoCD + OpenFrame) into a single operation, supporting deployment modes (oss-tenant, saas-tenant, saas-shared), interactive/non-interactive modes, and verbose logging options.
