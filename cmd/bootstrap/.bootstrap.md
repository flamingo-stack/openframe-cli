A command factory that creates the bootstrap CLI command for setting up a complete OpenFrame environment, combining cluster creation and chart installation into a single streamlined operation.

## Key Components

- **GetBootstrapCmd()** - Returns a configured cobra.Command for the bootstrap operation
- **Command Configuration** - Sets up usage patterns, help text, and argument validation
- **Flag Registration** - Adds deployment mode, interactive, and verbose flags
- **Service Delegation** - Routes execution to the internal bootstrap service

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the bootstrap command to the root command
    bootstrapCmd := bootstrap.GetBootstrapCmd()
    rootCmd.AddCommand(bootstrapCmd)
    
    // The command can now be used as:
    // openframe bootstrap
    // openframe bootstrap my-cluster --deployment-mode=oss-tenant
    // openframe bootstrap --verbose --non-interactive
    
    rootCmd.Execute()
}
```

The command supports multiple execution modes from interactive prompts to fully automated CI/CD scenarios, and provides detailed logging options for troubleshooting the bootstrap process.
