This file defines a Cobra CLI command for bootstrapping a complete OpenFrame environment, combining cluster creation and chart installation into a single streamlined command.

## Key Components

- **GetBootstrapCmd()** - Returns a configured Cobra command that orchestrates the complete OpenFrame setup process
- **Command Flags**:
  - `deployment-mode` - Pre-selects deployment type (oss-tenant, saas-tenant, saas-shared)
  - `non-interactive` - Enables CI/CD mode by skipping prompts
  - `verbose` - Shows detailed logging including ArgoCD sync progress

## Usage Example

```go
package main

import (
    "github.com/your-org/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add bootstrap command to root
    bootstrapCmd := bootstrap.GetBootstrapCmd()
    rootCmd.AddCommand(bootstrapCmd)
    
    rootCmd.Execute()
}
```

The command can be used in several ways:
- Interactive mode: `openframe bootstrap`
- With cluster name: `openframe bootstrap my-cluster`
- CI/CD mode: `openframe bootstrap --deployment-mode=oss-tenant --non-interactive`
- Verbose logging: `openframe bootstrap -v --deployment-mode=saas-shared`
