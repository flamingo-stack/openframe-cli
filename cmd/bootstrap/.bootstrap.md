This file defines the CLI command for bootstrapping a complete OpenFrame environment, providing a streamlined setup experience.

## Key Components

- **`GetBootstrapCmd()`** - Returns a configured Cobra command for the bootstrap operation
- **Command Configuration** - Sets up usage patterns, help text, and argument validation
- **Flag Definitions** - Configures deployment mode, interactive options, and verbosity settings
- **Execution Handler** - Delegates actual bootstrap logic to the internal bootstrap service

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{
        Use: "openframe",
    }
    
    // Add the bootstrap command to the root command
    rootCmd.AddCommand(bootstrap.GetBootstrapCmd())
    
    // Execute the CLI
    rootCmd.Execute()
}
```

The bootstrap command performs a complete OpenFrame setup by sequentially executing cluster creation and chart installation. It supports various deployment modes (oss-tenant, saas-tenant, saas-shared) and can run in both interactive and non-interactive modes. The verbose flag enables detailed logging including ArgoCD synchronization progress, making it suitable for both development and CI/CD environments.
