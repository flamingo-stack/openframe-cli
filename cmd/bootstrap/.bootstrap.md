Provides the CLI command interface for bootstrapping a complete OpenFrame environment in a single operation.

## Key Components

- **GetBootstrapCmd()** - Returns a configured Cobra command that orchestrates the complete OpenFrame setup process
- **Command flags**:
  - `deployment-mode` - Pre-selects deployment type (oss-tenant, saas-tenant, saas-shared)
  - `non-interactive` - Enables fully automated execution
  - `verbose` - Shows detailed progress logs including ArgoCD sync status

## Usage Example

```go
// Register the bootstrap command with the root CLI
rootCmd := &cobra.Command{Use: "openframe"}
rootCmd.AddCommand(GetBootstrapCmd())

// Command execution examples:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # Custom cluster name
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive --verbose      # Full automation with logs
```

The bootstrap command combines cluster creation and chart installation into a single streamlined workflow, making it the recommended entry point for new OpenFrame deployments. It delegates the actual implementation to the internal bootstrap service while providing a clean CLI interface with flexible configuration options.
