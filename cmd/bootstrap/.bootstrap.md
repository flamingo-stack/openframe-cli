This file defines the CLI command for bootstrapping a complete OpenFrame environment in a single operation.

## Key Components

- **GetBootstrapCmd()** - Returns a configured Cobra command that orchestrates the complete OpenFrame setup process
- **Command flags**:
  - `--deployment-mode` - Specifies deployment type (oss-tenant, saas-tenant, saas-shared)
  - `--non-interactive` - Enables headless operation using existing configuration
  - `--verbose` - Shows detailed logging including ArgoCD synchronization progress

## Usage Example

```go
// Get the bootstrap command and add it to root command
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// Command usage examples:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # Custom cluster name
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive --verbose      # CI/CD friendly mode
```

The bootstrap command combines cluster creation and chart installation into a single streamlined workflow, making it ideal for quick OpenFrame environment setup. It accepts an optional cluster name argument and supports both interactive and non-interactive modes for different use cases.
