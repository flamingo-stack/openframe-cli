A CLI command factory that creates the bootstrap command for setting up complete OpenFrame environments with Kubernetes cluster creation and chart installation.

## Key Components

- **GetBootstrapCmd()** - Factory function that returns a configured cobra.Command for bootstrapping OpenFrame environments
- **Command Configuration** - Sets up usage patterns, help text, and argument validation for the bootstrap command
- **Flag Definitions** - Configures deployment mode selection, interactive/non-interactive modes, and verbose logging options

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add bootstrap command to root
    bootstrapCmd := bootstrap.GetBootstrapCmd()
    rootCmd.AddCommand(bootstrapCmd)
    
    rootCmd.Execute()
}
```

The bootstrap command combines cluster creation and chart installation into a single operation. It supports interactive mode by default, with options for CI/CD automation through deployment mode preselection and non-interactive flags. The command delegates actual execution to the internal bootstrap service while providing comprehensive CLI interface configuration.
