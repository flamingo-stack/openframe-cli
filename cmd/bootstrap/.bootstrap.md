A factory function that creates the main bootstrap command for OpenFrame, providing a streamlined way to set up a complete OpenFrame environment by orchestrating cluster creation and chart installation.

## Key Components

- **GetBootstrapCmd()**: Factory function that returns a configured `*cobra.Command` for the bootstrap operation
- **bootstrap.NewService()**: Delegates actual bootstrap logic to the internal bootstrap service
- **Command Flags**: Supports deployment mode selection, non-interactive mode, and verbose logging

## Usage Example

```go
// Get the bootstrap command and add it to root command
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// The command supports multiple usage patterns:
// Interactive mode (default)
// openframe bootstrap

// With custom cluster name
// openframe bootstrap my-cluster

// CI/CD mode with specific deployment
// openframe bootstrap --deployment-mode=oss-tenant --non-interactive

// Verbose logging for debugging
// openframe bootstrap -v --deployment-mode=saas-shared
```

The bootstrap command combines cluster creation and chart installation into a single operation, accepting optional cluster names and configuration flags for deployment mode, interactivity, and logging verbosity. It serves as the main entry point for users getting started with OpenFrame.
