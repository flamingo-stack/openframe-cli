Provides the bootstrap command for setting up a complete OpenFrame environment with a single command. This command combines cluster creation and chart installation into a streamlined workflow.

## Key Components

- **GetBootstrapCmd()** - Returns a configured Cobra command for bootstrapping OpenFrame environments
- **Command configuration** - Sets up usage patterns, help text, and command-line flags
- **Flag definitions** - Supports deployment mode selection, non-interactive mode, and verbose logging

## Usage Example

```go
// Get the bootstrap command and add it to your CLI
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// The command supports various usage patterns:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # With custom cluster name  
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive                # CI/CD friendly mode
// openframe bootstrap --verbose                         # Detailed logging
```

The bootstrap command executes two main operations sequentially: creating a Kubernetes cluster and installing ArgoCD with OpenFrame charts. It accepts an optional cluster name argument and provides flags for deployment mode selection, non-interactive operation, and verbose output control.
