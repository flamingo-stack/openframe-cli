A Cobra command wrapper that provides a streamlined bootstrap experience for setting up a complete OpenFrame environment by combining cluster creation and chart installation into a single command.

## Key Components

- **GetBootstrapCmd()** - Returns a configured Cobra command for bootstrapping OpenFrame environments
- **bootstrap.NewService()** - Delegates execution to the internal bootstrap service
- **Command flags** - Supports deployment mode selection, non-interactive mode, and verbose logging

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add bootstrap command to CLI
    rootCmd.AddCommand(bootstrap.GetBootstrapCmd())
    
    rootCmd.Execute()
}
```

The bootstrap command combines cluster creation and chart installation:
- Interactive mode: `openframe bootstrap`
- With cluster name: `openframe bootstrap my-cluster`
- CI/CD mode: `openframe bootstrap --deployment-mode=oss-tenant --non-interactive`
- Verbose logging: `openframe bootstrap -v`
