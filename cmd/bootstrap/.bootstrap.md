This file defines a CLI command for bootstrapping a complete OpenFrame environment, combining cluster creation and chart installation into a single streamlined operation.

## Key Components

- **`GetBootstrapCmd()`**: Returns a configured Cobra command that orchestrates the complete OpenFrame setup process
- **Command flags**: 
  - `deployment-mode`: Specifies deployment type (oss-tenant, saas-tenant, saas-shared)
  - `non-interactive`: Enables CI/CD mode with no prompts
  - `verbose`: Shows detailed logging including ArgoCD sync progress

## Usage Example

```go
package main

import (
    "github.com/your-org/openframe-cli/cmd"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add the bootstrap command
    bootstrapCmd := cmd.GetBootstrapCmd()
    rootCmd.AddCommand(bootstrapCmd)
    
    rootCmd.Execute()
}
```

The command can be used in various ways:
```go
// Interactive mode
openframe bootstrap

// With custom cluster name
openframe bootstrap my-cluster

// Non-interactive CI/CD mode
openframe bootstrap --deployment-mode=oss-tenant --non-interactive

// Verbose logging
openframe bootstrap -v --deployment-mode=saas-shared
```
