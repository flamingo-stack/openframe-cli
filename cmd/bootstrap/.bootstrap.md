Provides a Cobra command for bootstrapping a complete OpenFrame environment. This command combines cluster creation and chart installation into a single streamlined workflow.

## Key Components

- **GetBootstrapCmd()**: Returns a configured Cobra command that bootstraps an OpenFrame environment
- **Command flags**: Supports deployment mode selection, non-interactive mode, and verbose logging
- **Service integration**: Delegates execution to the internal bootstrap service

## Usage Example

```go
// Get the bootstrap command for CLI registration
bootstrapCmd := GetBootstrapCmd()

// The command supports various execution modes:
// Interactive mode (default)
// openframe bootstrap

// With cluster name
// openframe bootstrap my-cluster

// Non-interactive with deployment mode
// openframe bootstrap --deployment-mode=oss-tenant --non-interactive

// Verbose mode with detailed ArgoCD sync progress
// openframe bootstrap -v --deployment-mode=saas-shared
```

The bootstrap command performs a two-step process: creating a Kubernetes cluster via `openframe cluster create` and installing ArgoCD/OpenFrame charts via `openframe chart install`. It accepts optional cluster names and supports three deployment modes (oss-tenant, saas-tenant, saas-shared) with configurable interactivity levels.
