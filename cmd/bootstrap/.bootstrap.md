This file provides a CLI command wrapper for bootstrapping a complete OpenFrame environment, combining cluster creation and chart installation into a single streamlined operation.

## Key Components

- **GetBootstrapCmd()** - Returns a configured cobra.Command for the bootstrap operation
- **Command Definition** - Configures the "bootstrap" subcommand with usage, help text, and argument validation
- **Flag Configuration** - Adds deployment mode, interactive mode, and verbosity flags
- **Service Delegation** - Delegates actual execution to the internal bootstrap service

## Usage Example

```go
// Get the bootstrap command for CLI registration
bootstrapCmd := GetBootstrapCmd()

// The command supports various usage patterns:
// Interactive mode (default)
// openframe bootstrap

// With custom cluster name
// openframe bootstrap my-cluster

// Non-interactive with deployment mode
// openframe bootstrap --deployment-mode=oss-tenant --non-interactive

// Verbose logging
// openframe bootstrap --verbose --deployment-mode=saas-shared

// Add to root command
rootCmd.AddCommand(bootstrapCmd)
```

The bootstrap command combines `openframe cluster create` and `openframe chart install` into a single operation, providing deployment mode selection, interactive/non-interactive modes, and detailed logging options for streamlined OpenFrame environment setup.
