Provides a cobra command for bootstrapping complete OpenFrame environments with automated cluster creation and chart installation.

## Key Components

- **GetBootstrapCmd()**: Returns a cobra.Command that orchestrates the full OpenFrame setup process
- **bootstrap.NewService().Execute()**: Delegates the actual bootstrap logic to the internal bootstrap service
- **Command flags**: Supports deployment-mode selection, non-interactive mode, and verbose logging

## Usage Example

```go
// Get the bootstrap command and add it to the root command
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// Example command line usage:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # With custom cluster name
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive --verbose      # Full automation with detailed logs
```

The bootstrap command streamlines OpenFrame deployment by combining cluster creation and chart installation into a single operation, supporting both interactive and CI/CD-friendly non-interactive modes with configurable deployment types (OSS tenant, SaaS tenant, or SaaS shared).
