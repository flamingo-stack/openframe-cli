This file defines the CLI command interface for bootstrapping a complete OpenFrame environment in a single operation.

## Key Components

- **GetBootstrapCmd()** - Returns a configured cobra.Command that orchestrates the complete OpenFrame setup process
- **Command flags**:
  - `--deployment-mode` - Pre-selects deployment type (oss-tenant, saas-tenant, saas-shared)
  - `--non-interactive` - Enables CI/CD mode by skipping all prompts
  - `--verbose` - Shows detailed logging including ArgoCD sync progress

## Usage Example

```go
// Get the bootstrap command and add it to your CLI
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// The command can be used in various ways:
// Interactive mode (prompts for options)
// $ openframe bootstrap

// With custom cluster name
// $ openframe bootstrap my-cluster

// CI/CD mode with pre-configured options
// $ openframe bootstrap --deployment-mode=oss-tenant --non-interactive

// Verbose mode for detailed output
// $ openframe bootstrap --verbose --deployment-mode=saas-shared
```

The bootstrap command combines cluster creation and chart installation into a single streamlined operation, making it easier for users to get started with OpenFrame by automatically running both `openframe cluster create` and `openframe chart install` in sequence.
