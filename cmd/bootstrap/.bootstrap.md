This file defines the CLI command interface for bootstrapping a complete OpenFrame environment, providing a streamlined way to create clusters and install necessary components.

## Key Components

- **GetBootstrapCmd()**: Returns a configured Cobra command that handles the bootstrap operation
- **Command flags**: 
  - `deployment-mode`: Specifies deployment type (oss-tenant, saas-tenant, saas-shared)
  - `non-interactive`: Enables CI/CD-friendly mode without prompts
  - `verbose`: Shows detailed logging including ArgoCD sync progress

## Usage Example

```go
package main

import (
    "github.com/your-org/openframe-cli/cmd/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add bootstrap command
    rootCmd.AddCommand(bootstrap.GetBootstrapCmd())
    
    rootCmd.Execute()
}
```

The command can be used in several ways:
```bash
# Interactive mode
openframe bootstrap

# With custom cluster name
openframe bootstrap my-cluster

# Non-interactive with deployment mode
openframe bootstrap --deployment-mode=oss-tenant --non-interactive

# Verbose logging
openframe bootstrap --verbose
```

The bootstrap command combines cluster creation and chart installation into a single operation, making it ideal for quick OpenFrame environment setup.
