This file provides the CLI command interface for OpenFrame's bootstrap functionality, which streamlines the complete environment setup process.

## Key Components

- **GetBootstrapCmd()**: Returns a configured Cobra command that orchestrates cluster creation and chart installation in a single operation

## Usage Example

```go
// Get the bootstrap command for CLI registration
bootstrapCmd := GetBootstrapCmd()

// The command supports various execution modes:
// Interactive mode (prompts user for inputs)
// openframe bootstrap

// Non-interactive with specific deployment mode
// openframe bootstrap my-cluster --deployment-mode=oss-tenant --non-interactive

// Verbose mode for detailed logging
// openframe bootstrap --verbose --deployment-mode=saas-shared
```

The bootstrap command accepts an optional cluster name argument and supports flags for deployment mode selection (`oss-tenant`, `saas-tenant`, `saas-shared`), non-interactive execution, and verbose logging. When executed, it delegates to the internal bootstrap service to perform the actual cluster creation and ArgoCD/OpenFrame chart installation workflow.
