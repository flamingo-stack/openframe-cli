This file defines the CLI command interface for bootstrapping a complete OpenFrame environment setup.

## Key Components

- **`GetBootstrapCmd()`** - Returns a configured Cobra command that orchestrates the complete OpenFrame bootstrap process
- **Command flags**:
  - `deployment-mode` - Specifies deployment type (oss-tenant, saas-tenant, saas-shared)
  - `non-interactive` - Enables CI/CD mode by skipping prompts
  - `verbose` - Shows detailed logging including ArgoCD sync progress

## Usage Example

```go
// Get the bootstrap command and add it to root command
rootCmd := &cobra.Command{Use: "openframe"}
rootCmd.AddCommand(GetBootstrapCmd())

// Command usage examples:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # Custom cluster name
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive --verbose      # CI/CD mode with detailed logs
```

The command combines cluster creation and chart installation into a single streamlined operation, making it ideal for quick OpenFrame environment setup. It delegates the actual bootstrap logic to the internal `bootstrap.NewService().Execute()` method while providing a user-friendly CLI interface with comprehensive help text and examples.
