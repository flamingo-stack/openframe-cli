This file defines a CLI command for bootstrapping a complete OpenFrame environment by orchestrating cluster creation and chart installation.

## Key Components

- **GetBootstrapCmd()** - Returns a Cobra command that handles the bootstrap workflow
- **Command configuration** - Sets up usage patterns, help text, and argument validation
- **Flag definitions** - Configures deployment mode, interactive behavior, and verbose logging options

## Usage Example

```go
// Get the bootstrap command and add it to your CLI
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// The command supports various usage patterns:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # Custom cluster name  
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive --verbose      # CI/CD friendly mode
```

The bootstrap command streamlines OpenFrame setup by combining cluster creation and chart installation into a single operation, with support for both interactive and automated deployment scenarios.
