This file defines the CLI command for bootstrapping a complete OpenFrame environment, providing a streamlined setup experience that combines cluster creation and chart installation.

## Key Components

- **GetBootstrapCmd()**: Returns a configured Cobra command for the bootstrap operation
- **Command Configuration**: Sets up the "bootstrap" command with usage patterns, descriptions, and argument validation
- **Flag Definitions**: Configures deployment mode, interactive mode, and verbosity options
- **Command Execution**: Delegates to the bootstrap service for actual implementation

## Usage Example

```go
// Get the bootstrap command and add it to your CLI
bootstrapCmd := GetBootstrapCmd()
rootCmd.AddCommand(bootstrapCmd)

// The command supports various usage patterns:
// openframe bootstrap                                    # Interactive mode
// openframe bootstrap my-cluster                        # Custom cluster name
// openframe bootstrap --deployment-mode=oss-tenant     # Skip deployment selection
// openframe bootstrap --non-interactive -v             # CI/CD mode with verbose output
```

The command accepts an optional cluster name argument and provides flags for deployment mode selection (`oss-tenant`, `saas-tenant`, `saas-shared`), non-interactive operation, and verbose logging. It performs both cluster creation and ArgoCD/OpenFrame chart installation in a single operation.
