This file defines a CLI command wrapper for bootstrapping a complete OpenFrame environment setup.

## Key Components

- **GetBootstrapCmd()**: Returns a configured Cobra command that orchestrates the complete OpenFrame bootstrap process
- **Command flags**: 
  - `deployment-mode`: Specifies deployment type (oss-tenant, saas-tenant, saas-shared)
  - `non-interactive`: Enables CI/CD-friendly mode without prompts
  - `verbose`: Shows detailed logging including ArgoCD sync progress

## Usage Example

```go
package main

import (
    "github.com/your-project/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    rootCmd := &cobra.Command{Use: "openframe"}
    
    // Add bootstrap command to root
    bootstrapCmd := bootstrap.GetBootstrapCmd()
    rootCmd.AddCommand(bootstrapCmd)
    
    rootCmd.Execute()
}
```

The command combines cluster creation and chart installation into a single operation, supporting both interactive and automated deployment scenarios. It accepts an optional cluster name argument and delegates execution to the internal bootstrap service.
