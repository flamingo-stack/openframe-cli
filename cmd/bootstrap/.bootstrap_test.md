Test file for validating the bootstrap command functionality and structure in the OpenFrame CLI. Contains comprehensive tests for command properties, argument validation, and integration with the CLI framework.

## Key Components

- **`TestBootstrapCommand`** - Validates basic bootstrap command structure including name, descriptions, and RunE function
- **`TestBootstrapCommandStructure`** - Tests detailed command properties like usage pattern, aliases, subcommands, and help text content
- **`TestBootstrapArgumentValidation`** - Verifies argument validation logic accepts 0-1 arguments but rejects multiple arguments
- **`init()`** - Initializes test mode using testutil package
- **Test dependencies** - Uses testify/assert for assertions and custom testutil package

## Usage Example

```go
// Run specific test
go test -run TestBootstrapCommand

// Run all bootstrap tests
go test ./bootstrap

// Test with verbose output
go test -v ./bootstrap -run TestBootstrapCommandStructure

// Example of what the tests validate
cmd := GetBootstrapCmd()
assert.Equal(t, "bootstrap", cmd.Name())
assert.Contains(t, cmd.Short, "Bootstrap complete OpenFrame environment")
```

The tests ensure the bootstrap command maintains correct CLI structure, proper help documentation, and validates user input according to the expected argument pattern of `bootstrap [cluster-name]`.
