Test file that validates the bootstrap command's structure, configuration, and argument handling for the OpenFrame CLI tool.

## Key Components

- **`TestBootstrapCommand`** - Tests basic command properties like name, descriptions, and RunE function presence
- **`TestBootstrapCommandStructure`** - Validates command usage pattern, aliases, subcommands, and content verification
- **`TestBootstrapArgumentValidation`** - Tests argument validation logic for 0-1 arguments acceptance

## Usage Example

```go
// Run specific test function
func TestBootstrapCommand(t *testing.T) {
    cmd := GetBootstrapCmd()
    assert.Equal(t, "bootstrap", cmd.Name())
    assert.NotEmpty(t, cmd.Short)
    assert.NotNil(t, cmd.RunE)
}

// Test argument validation
func TestBootstrapArgumentValidation(t *testing.T) {
    cmd := GetBootstrapCmd()
    if cmd.Args != nil {
        // Valid: no arguments
        err := cmd.Args(cmd, []string{})
        assert.NoError(t, err)
        
        // Valid: one argument
        err = cmd.Args(cmd, []string{"cluster-name"})
        assert.NoError(t, err)
    }
}
```

The tests ensure the bootstrap command follows expected CLI patterns, has proper documentation, and correctly validates input arguments for cluster bootstrapping operations.
