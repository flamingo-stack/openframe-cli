# Architecture Diagrams

This directory contains Mermaid diagrams generated from architecture analysis.

## Diagrams

- **[High-Level System Architecture](./high-level-system-architecture.mmd)** - `.mmd` file
- **[Module Dependencies and Data Flow](./module-dependencies-and-data-flow.mmd)** - `.mmd` file
- **[Bootstrap Workflow Sequence](./bootstrap-workflow-sequence.mmd)** - `.mmd` file

## Viewing Diagrams

These `.mmd` files can be viewed:
- In VS Code with Mermaid extension
- On GitHub (renders inline)
- Via [Mermaid Live Editor](https://mermaid.live)
- In documentation tools that support Mermaid
