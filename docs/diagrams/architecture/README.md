# Architecture Diagrams

This directory contains Mermaid diagrams generated from architecture analysis.

## Diagrams

- **[High-Level Architecture Diagram](./high-level-architecture-diagram.mmd)** - `.mmd` file
- **[Module Dependency Diagram](./module-dependency-diagram.mmd)** - `.mmd` file
- **[Bootstrap Command Execution Flow](./bootstrap-command-execution-flow.mmd)** - `.mmd` file

## Viewing Diagrams

These `.mmd` files can be viewed:
- In VS Code with Mermaid extension
- On GitHub (renders inline)
- Via [Mermaid Live Editor](https://mermaid.live)
- In documentation tools that support Mermaid
