This is the main entry point for the OpenFrame CLI application that initializes the command-line interface and handles execution errors.

## Key Components

- **main()** - Application entry point that executes the CLI commands and handles errors
- **cmd.Execute()** - Delegates to the cmd package to run the actual CLI logic
- Error handling with stderr output and proper exit codes

## Usage Example

```go
// This file is typically not imported directly, but run as the main application
// Build and run the CLI:
// go build -o openframe-cli main.go
// ./openframe-cli [command] [flags]

// The main function handles all CLI execution:
func main() {
    if err := cmd.Execute(); err != nil {
        fmt.Fprintf(os.Stderr, "Error: %v\n", err)
        os.Exit(1)
    }
}
```

The application follows the standard Go CLI pattern where the main function is minimal and delegates the actual command logic to a dedicated cmd package, ensuring clean separation of concerns and proper error handling with appropriate exit codes.
