Test file for the cluster package that validates type definitions, flag containers, and domain models.

## Key Components

- **NewFlagContainer()** - Tests creation of flag containers with default values
- **FlagContainer methods** - Tests `SyncGlobalFlags()` and `Reset()` functionality
- **Domain model tests** - Validates `ClusterConfig`, `ClusterInfo`, `NodeInfo`, and `ProviderOptions` structs
- **Error type tests** - Tests custom error types like `ErrClusterNotFound`, `ErrProviderNotFound`
- **Interface compliance** - Verifies that implementations satisfy `ClusterService` and `ClusterManager` interfaces
- **Flag type tests** - Tests various flag structs (`GlobalFlags`, `CreateFlags`, etc.)

## Usage Example

```go
// Test flag container creation and synchronization
func TestFlagContainerUsage() {
    container := NewFlagContainer()
    
    // Set global flags
    container.Global.Verbose = true
    container.Global.DryRun = true
    
    // Sync to all command flags
    container.SyncGlobalFlags()
    
    // Verify sync worked
    assert.Equal(t, container.Global.Verbose, container.Create.GlobalFlags.Verbose)
}

// Test domain model creation
func TestDomainModels() {
    config := models.ClusterConfig{
        Name:       "test-cluster",
        Type:       models.ClusterTypeK3d,
        NodeCount:  3,
        K8sVersion: "v1.25.0-k3s1",
    }
    
    assert.Equal(t, "test-cluster", config.Name)
}
```
