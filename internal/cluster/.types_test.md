This file contains comprehensive unit tests for the cluster management type system, validating flag containers, domain models, error types, and interface implementations.

## Key Components

- **FlagContainer Tests**: Tests for `NewFlagContainer()`, `SyncGlobalFlags()`, and `Reset()` methods
- **Domain Model Tests**: Validation of cluster types, configurations, and node information structures
- **Error Type Tests**: Custom error types including `ClusterNotFoundError`, `ProviderNotFoundError`, etc.
- **Interface Compliance Tests**: Verification that concrete types implement required interfaces
- **Flag Structure Tests**: Tests for global, create, delete, and list flag structures

## Usage Example

```go
// Test flag container creation and synchronization
func TestFlagOperations() {
    container := NewFlagContainer()
    container.Global.Verbose = true
    container.SyncGlobalFlags()
    
    // All command flags inherit global settings
    assert.Equal(t, true, container.Create.GlobalFlags.Verbose)
}

// Test domain model usage
func TestClusterConfig() {
    config := models.ClusterConfig{
        Name:       "test-cluster",
        Type:       models.ClusterTypeK3d,
        NodeCount:  3,
        K8sVersion: "v1.25.0-k3s1",
    }
    
    assert.Equal(t, "k3d", string(config.Type))
}

// Test error handling
func TestCustomErrors() {
    err := models.NewClusterNotFoundError("missing-cluster")
    var clusterErr models.ErrClusterNotFound
    assert.True(t, errors.As(err, &clusterErr))
}
```
