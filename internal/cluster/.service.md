This service provides cluster lifecycle management for OpenFrame's Kubernetes development environments, handling creation, deletion, status checking, and cleanup operations.

## Key Components

**Main Service:**
- `ClusterService` - Core service managing cluster operations through k3d
- Constructor functions for different configurations (standard, UI-suppressed, custom options)

**Primary Operations:**
- `CreateCluster()` - Creates new clusters with duplicate detection
- `DeleteCluster()` - Handles cluster deletion with progress feedback
- `ListClusters()` - Lists all available clusters
- `GetClusterStatus()` - Retrieves cluster information and status
- `GetRestConfig()` - Returns Kubernetes REST client configuration
- `CleanupCluster()` - Comprehensive cleanup of cluster resources

**Cleanup Functions:**
- `cleanupHelmReleases()` - Removes all Helm deployments
- `cleanupKubernetesResources()` - Cleans up common namespaces
- `cleanupDockerResources()` - Removes Docker containers and images

## Usage Example

```go
// Create a cluster service
executor := executor.NewCommandExecutor()
service := NewClusterService(executor)

// Create a new cluster
config := models.ClusterConfig{
    Name: "dev-cluster",
    Type: models.ClusterTypeK3d,
}

restConfig, err := service.CreateCluster(config)
if err != nil {
    log.Fatal(err)
}

// Check cluster status
status, err := service.GetClusterStatus("dev-cluster")
if err != nil {
    log.Fatal(err)
}

// Cleanup and delete cluster
err = service.CleanupCluster("dev-cluster", models.ClusterTypeK3d, true, false)
if err != nil {
    log.Fatal(err)
}

err = service.DeleteCluster("dev-cluster", models.ClusterTypeK3d, false)
if err != nil {
    log.Fatal(err)
}
```

The service includes intelligent duplicate detection for cluster creation, comprehensive cleanup procedures for Helm releases and Kubernetes resources, and UI suppression options for automated environments.
