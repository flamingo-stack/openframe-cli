This service provides a high-level abstraction for Kubernetes cluster management operations, particularly focused on k3d clusters. It handles cluster creation, deletion, status checking, and comprehensive cleanup with support for both interactive and automated environments.

## Key Components

- **ClusterService**: Main service struct that orchestrates cluster operations
- **NewClusterService**: Creates a standard cluster service instance
- **NewClusterServiceSuppressed**: Creates a service with UI suppression for automation
- **CreateCluster**: Handles cluster creation with existence checking and status display
- **DeleteCluster**: Manages cluster deletion with progress indicators
- **ListClusters**: Returns information about all available clusters
- **GetClusterStatus**: Retrieves detailed status for a specific cluster
- **CleanupCluster**: Performs comprehensive cleanup of cluster resources including Helm releases and Docker images

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/cluster"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
)

func main() {
    // Create executor and service
    exec := executor.NewRealExecutor()
    service := cluster.NewClusterService(exec)
    
    // Create a new cluster
    config := models.ClusterConfig{
        Name: "my-cluster",
        Type: models.ClusterTypeK3d,
    }
    
    err := service.CreateCluster(config)
    if err != nil {
        log.Fatal(err)
    }
    
    // Check cluster status
    status, err := service.GetClusterStatus("my-cluster")
    if err != nil {
        log.Fatal(err)
    }
    
    // Cleanup when done
    err = service.CleanupCluster("my-cluster", models.ClusterTypeK3d, false, false)
    if err != nil {
        log.Fatal(err)
    }
}
```
