This file provides a high-level service layer for cluster lifecycle management, abstracting cluster operations like creation, deletion, and cleanup across different cluster types (primarily K3d).

## Key Components

- **ClusterService**: Main service struct that orchestrates cluster operations through a K3d manager
- **Constructor Functions**: Multiple factory methods for creating service instances with different configurations
- **Core Operations**: 
  - `CreateCluster()` - Creates new clusters with existence checks and user-friendly messaging
  - `DeleteCluster()` - Handles cluster deletion with progress indication
  - `ListClusters()` - Retrieves all available clusters
  - `GetClusterStatus()` - Gets detailed cluster information
  - `CleanupCluster()` - Comprehensive cleanup of cluster resources
- **Cleanup Methods**: Specialized cleanup for Helm releases, Kubernetes resources, and Docker artifacts
- **UI Integration**: Conditional UI suppression for automation scenarios

## Usage Example

```go
// Create a cluster service
executor := executor.NewCommandExecutor()
service := NewClusterService(executor)

// Create a new cluster
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
}
err := service.CreateCluster(config)

// List all clusters
clusters, err := service.ListClusters()

// Clean up cluster resources before deletion
err = service.CleanupCluster("my-cluster", models.ClusterTypeK3d, true, false)

// Delete the cluster
err = service.DeleteCluster("my-cluster", models.ClusterTypeK3d, false)
```

The service provides both interactive and automated modes, with comprehensive error handling and progress feedback for cluster management operations.
