This file provides a comprehensive service layer for managing Kubernetes cluster operations, particularly focused on k3d clusters. It handles cluster lifecycle management including creation, deletion, listing, and cleanup operations.

## Key Components

**ClusterService**: Main service struct that orchestrates cluster operations with support for UI suppression and terminal detection.

**Constructor Functions**:
- `NewClusterService()` - Creates service with default settings
- `NewClusterServiceSuppressed()` - Creates service with UI elements disabled for automation
- `NewClusterServiceWithOptions()` - Creates service with custom k3d manager

**Core Operations**:
- `CreateCluster()` - Creates new clusters with existence checking and progress feedback
- `DeleteCluster()` - Handles cluster deletion with progress tracking
- `ListClusters()` - Lists all available clusters
- `GetClusterStatus()` - Retrieves cluster status information
- `CleanupCluster()` - Performs comprehensive cluster cleanup including Helm releases, Kubernetes resources, and Docker artifacts

**Utility Functions**:
- `isTerminalEnvironment()` - Detects if running in a proper terminal
- `DetectClusterType()` - Identifies cluster type for operations

## Usage Example

```go
// Create service for interactive use
executor := executor.NewCommandExecutor()
service := NewClusterService(executor)

// Create a new cluster
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
}
err := service.CreateCluster(config)

// List all clusters
clusters, err := service.ListClusters()

// Cleanup cluster resources
err = service.CleanupCluster("my-cluster", models.ClusterTypeK3d, true, false)

// Delete cluster
err = service.DeleteCluster("my-cluster", models.ClusterTypeK3d, false)
```
