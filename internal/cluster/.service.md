This Go file implements a comprehensive cluster management service for OpenFrame's Kubernetes cluster operations, providing high-level business logic for cluster lifecycle management.

## Key Components

- **ClusterService** - Main service struct managing cluster operations with k3d backend
- **Constructor functions** - `NewClusterService`, `NewClusterServiceSuppressed`, `NewClusterServiceWithOptions` for different use cases
- **Cluster lifecycle methods** - `CreateCluster`, `DeleteCluster`, `ListClusters`, `GetClusterStatus`
- **Configuration management** - `GetRestConfig`, `DetectClusterType` for cluster interaction
- **Cleanup operations** - `CleanupCluster` with specialized cleanup for Helm releases, Kubernetes resources, and Docker resources
- **UI handling** - Terminal detection and UI suppression for automation scenarios

## Usage Example

```go
// Create a new cluster service with default settings
exec := executor.NewRealCommandExecutor()
service := NewClusterService(exec)

// Create a new cluster
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
}

restConfig, err := service.CreateCluster(config)
if err != nil {
    log.Fatal(err)
}

// List all clusters
clusters, err := service.ListClusters()
if err != nil {
    log.Fatal(err)
}

// Delete a cluster
err = service.DeleteCluster("my-cluster", models.ClusterTypeK3d, false)
if err != nil {
    log.Fatal(err)
}

// For automation scenarios, suppress interactive UI
automatedService := NewClusterServiceSuppressed(exec)
```

The service handles terminal detection for appropriate UI rendering and provides comprehensive cleanup capabilities including Helm releases, Kubernetes namespaces, and Docker resources. It gracefully handles existing clusters and provides detailed feedback through spinners and status messages.
