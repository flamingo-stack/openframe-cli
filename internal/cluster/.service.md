This file implements a comprehensive cluster management service that provides high-level operations for Kubernetes cluster lifecycle management, primarily focused on K3d clusters.

## Key Components

- **ClusterService**: Main service struct providing cluster operations with configurable UI suppression
- **NewClusterService/NewClusterServiceSuppressed/NewClusterServiceWithOptions**: Factory functions for creating service instances
- **CreateCluster**: Creates new clusters with existence checking and user-friendly messaging
- **DeleteCluster**: Handles cluster deletion with progress feedback
- **ListClusters/GetClusterStatus**: Provides cluster discovery and status operations
- **CleanupCluster**: Comprehensive cleanup including Helm releases, Kubernetes resources, and Docker artifacts
- **DetectClusterType**: Identifies cluster provider type

## Usage Example

```go
// Create a cluster service
executor := executor.NewCommandExecutor()
service := cluster.NewClusterService(executor)

// Create a new cluster
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
}
err := service.CreateCluster(config)

// List all clusters
clusters, err := service.ListClusters()

// Clean up cluster resources before deletion
err = service.CleanupCluster("my-cluster", models.ClusterTypeK3d, true, false)

// Delete the cluster
err = service.DeleteCluster("my-cluster", models.ClusterTypeK3d, false)
```
