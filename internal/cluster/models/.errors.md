Defines domain-specific error types for cluster operations, providing structured error handling with detailed context and meaningful error messages.

## Key Components

- **ErrClusterNotFound** - Error when a cluster cannot be found by name
- **ErrProviderNotFound** - Error when no provider exists for a cluster type
- **ErrInvalidClusterConfig** - Error for invalid cluster configuration with field details
- **ErrClusterAlreadyExists** - Error when attempting to create a duplicate cluster
- **ErrClusterOperation** - Generic cluster operation error with wrapped cause
- **Constructor functions** - `NewXxxError()` functions for creating typed errors

## Usage Example

```go
// Creating and handling domain errors
err := NewClusterNotFoundError("my-cluster")
if err != nil {
    fmt.Println(err) // "cluster 'my-cluster' not found"
}

// Configuration error with details
configErr := NewInvalidConfigError("replicas", -1, "must be positive")
fmt.Println(configErr) // "invalid cluster config - replicas: -1 (must be positive)"

// Operation error with wrapped cause
opErr := NewClusterOperationError("create", "test-cluster", fmt.Errorf("timeout"))
fmt.Println(opErr) // "cluster create operation failed for 'test-cluster': timeout"

// Error type checking
var clusterErr ErrClusterNotFound
if errors.As(err, &clusterErr) {
    fmt.Printf("Missing cluster: %s", clusterErr.Name)
}
```
