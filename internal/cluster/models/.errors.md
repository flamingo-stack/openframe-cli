Defines domain-specific errors for cluster operations, providing structured error types that represent business-level failures with contextual information.

## Key Components

**Error Types:**
- `ErrClusterNotFound` - Cluster lookup failures
- `ErrProviderNotFound` - Missing provider for cluster type
- `ErrInvalidClusterConfig` - Configuration validation errors
- `ErrClusterAlreadyExists` - Duplicate cluster creation attempts
- `ErrClusterOperation` - General operation failures with error wrapping

**Constructor Functions:**
- `NewClusterNotFoundError()` - Creates cluster not found errors
- `NewProviderNotFoundError()` - Creates provider not found errors
- `NewInvalidConfigError()` - Creates configuration validation errors
- `NewClusterAlreadyExistsError()` - Creates duplicate cluster errors
- `NewClusterOperationError()` - Creates operation errors with cause wrapping

## Usage Example

```go
// Handle cluster lookup failures
if cluster == nil {
    return NewClusterNotFoundError("my-cluster")
}

// Validate configuration
if config.NodeCount < 1 {
    return NewInvalidConfigError("NodeCount", config.NodeCount, "must be at least 1")
}

// Wrap underlying errors
if err := provider.CreateCluster(config); err != nil {
    return NewClusterOperationError("create", config.Name, err)
}

// Error checking with type assertions
if clusterErr, ok := err.(ErrClusterNotFound); ok {
    log.Printf("Cluster %s not found", clusterErr.Name)
}
```
