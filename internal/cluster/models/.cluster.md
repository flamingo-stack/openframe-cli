Defines data models for Kubernetes cluster management, supporting multiple cluster types (K3d and GKE) with configuration and status tracking capabilities.

## Key Components

- **ClusterType**: Enum for supported cluster types (`k3d`, `gke`)
- **ClusterConfig**: Configuration structure for creating new clusters
- **ClusterInfo**: Complete cluster information including status and nodes
- **NodeInfo**: Individual node details within a cluster
- **ProviderOptions**: Provider-specific configuration options
- **K3dOptions**: K3d-specific settings like port mappings
- **GKEOptions**: GKE-specific settings for zone and project

## Usage Example

```go
// Create a new K3d cluster configuration
config := ClusterConfig{
    Name:       "my-dev-cluster",
    Type:       ClusterTypeK3d,
    NodeCount:  3,
    K8sVersion: "1.28.0",
}

// Set up K3d-specific options
options := ProviderOptions{
    K3d: &K3dOptions{
        PortMappings: []string{"8080:80@loadbalancer"},
    },
    Verbose: true,
}

// Track cluster information
info := ClusterInfo{
    Name:       config.Name,
    Type:       config.Type,
    Status:     "running",
    NodeCount:  config.NodeCount,
    K8sVersion: config.K8sVersion,
    CreatedAt:  time.Now(),
    Nodes: []NodeInfo{
        {Name: "worker-1", Status: "ready", Role: "worker"},
        {Name: "control-plane", Status: "ready", Role: "master"},
    },
}
```
