This file defines data structures for managing Kubernetes cluster configurations and metadata across different cluster providers like k3d and GKE.

## Key Components

- **ClusterType**: Enumeration for supported cluster types (k3d, GKE)
- **ClusterConfig**: Configuration structure for creating new clusters with name, type, node count, and Kubernetes version
- **ClusterInfo**: Runtime information about existing clusters including status, nodes, and creation time
- **NodeInfo**: Individual node details within a cluster (name, status, role)
- **ProviderOptions**: Provider-specific configuration options container
- **K3dOptions/GKEOptions**: Provider-specific settings for k3d and GKE clusters

## Usage Example

```go
// Create a k3d cluster configuration
config := ClusterConfig{
    Name:       "dev-cluster",
    Type:       ClusterTypeK3d,
    NodeCount:  3,
    K8sVersion: "v1.27.0",
}

// Define k3d-specific options
options := ProviderOptions{
    K3d: &K3dOptions{
        PortMappings: []string{"8080:80@loadbalancer"},
    },
    Verbose: true,
}

// Create cluster info for tracking
info := ClusterInfo{
    Name:      config.Name,
    Type:      config.Type,
    Status:    "running",
    NodeCount: config.NodeCount,
    CreatedAt: time.Now(),
}
```
