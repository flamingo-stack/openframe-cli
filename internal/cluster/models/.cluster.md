Defines Go data structures and types for managing Kubernetes cluster configurations and metadata across different cluster providers like k3d and GKE.

## Key Components

- **ClusterType**: Enum constants for supported cluster types (`k3d`, `gke`)
- **ClusterConfig**: Configuration structure for creating new clusters
- **ClusterInfo**: Runtime information about existing clusters including status and nodes
- **NodeInfo**: Individual node metadata within a cluster
- **ProviderOptions**: Provider-specific configuration options
- **K3dOptions/GKEOptions**: Provider-specific settings for k3d and GKE clusters

## Usage Example

```go
// Create a cluster configuration
config := &ClusterConfig{
    Name:       "dev-cluster",
    Type:       ClusterTypeK3d,
    NodeCount:  3,
    K8sVersion: "v1.28.0",
}

// Define provider options
options := &ProviderOptions{
    K3d: &K3dOptions{
        PortMappings: []string{"8080:80", "8443:443"},
    },
    Verbose: true,
}

// Check cluster information
info := &ClusterInfo{
    Name:      "dev-cluster",
    Type:      ClusterTypeK3d,
    Status:    "running",
    NodeCount: 3,
    Nodes: []NodeInfo{
        {Name: "node-1", Status: "ready", Role: "control-plane"},
        {Name: "node-2", Status: "ready", Role: "worker"},
    },
}
```
