This file defines data models for managing Kubernetes clusters, supporting both local K3d and Google Kubernetes Engine (GKE) cluster types with their respective configurations and metadata.

## Key Components

- **ClusterType**: Enumerated type for supported cluster providers (K3d, GKE)
- **ClusterConfig**: Configuration structure for creating new clusters with name, type, node count, and Kubernetes version
- **ClusterInfo**: Complete cluster information including status, creation time, and node details
- **NodeInfo**: Individual node metadata with name, status, and role information
- **ProviderOptions**: Container for provider-specific configuration options
- **K3dOptions**: K3d-specific settings including port mappings
- **GKEOptions**: GKE-specific settings for zone and project configuration

## Usage Example

```go
// Create a new K3d cluster configuration
config := &ClusterConfig{
    Name:       "my-dev-cluster",
    Type:       ClusterTypeK3d,
    NodeCount:  3,
    K8sVersion: "v1.28.0",
}

// Define K3d-specific options
options := &ProviderOptions{
    K3d: &K3dOptions{
        PortMappings: []string{"8080:80@loadbalancer", "8443:443@loadbalancer"},
    },
    Verbose: true,
}

// Represent cluster information after creation
info := &ClusterInfo{
    Name:       config.Name,
    Type:       config.Type,
    Status:     "running",
    NodeCount:  config.NodeCount,
    K8sVersion: config.K8sVersion,
    CreatedAt:  time.Now(),
    Nodes: []NodeInfo{
        {Name: "k3d-my-dev-cluster-server-0", Status: "Ready", Role: "control-plane"},
        {Name: "k3d-my-dev-cluster-agent-0", Status: "Ready", Role: "worker"},
    },
}
```
