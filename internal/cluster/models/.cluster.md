This file defines data models for Kubernetes cluster management, providing structures for cluster configuration, information, and provider-specific options across different cluster types like k3d and GKE.

## Key Components

- **ClusterType**: Enum-style constants defining supported cluster types (`k3d`, `gke`)
- **ClusterConfig**: Configuration structure for creating new clusters with basic settings
- **ClusterInfo**: Comprehensive cluster information including status, nodes, and metadata
- **NodeInfo**: Individual node details within a cluster
- **ProviderOptions**: Container for provider-specific configuration options
- **K3dOptions**: Local k3d cluster configuration (port mappings)
- **GKEOptions**: Google Kubernetes Engine specific settings (zone, project)

## Usage Example

```go
// Create a new k3d cluster configuration
config := ClusterConfig{
    Name:       "dev-cluster",
    Type:       ClusterTypeK3d,
    NodeCount:  3,
    K8sVersion: "1.28.0",
}

// Set up k3d-specific options
options := ProviderOptions{
    K3d: &K3dOptions{
        PortMappings: []string{"8080:80@loadbalancer"},
    },
    Verbose: true,
}

// Create cluster info structure
info := ClusterInfo{
    Name:       config.Name,
    Type:       config.Type,
    Status:     "running",
    NodeCount:  config.NodeCount,
    K8sVersion: config.K8sVersion,
    CreatedAt:  time.Now(),
    Nodes: []NodeInfo{
        {Name: "worker-1", Status: "ready", Role: "worker"},
        {Name: "control-plane", Status: "ready", Role: "master"},
    },
}
```
