Defines the core interfaces for the cluster management domain, establishing contracts between cluster providers, business logic services, and provider registry components.

## Key Components

- **ClusterProvider**: Core interface for cluster provider implementations with CRUD operations (Create, Delete, Start, List, Status, DetectType, GetKubeconfig)
- **ClusterService**: Business logic interface that orchestrates cluster operations using providers  
- **ProviderRegistry**: Registry pattern interface for managing multiple cluster provider implementations

## Usage Example

```go
// Implementing a cluster provider
type MyProvider struct{}

func (p *MyProvider) Create(ctx context.Context, config ClusterConfig) error {
    // Implementation for creating clusters
    return nil
}

func (p *MyProvider) List(ctx context.Context) ([]ClusterInfo, error) {
    // Implementation for listing clusters
    return []ClusterInfo{}, nil
}

// Using the service
var service ClusterService
err := service.CreateCluster(ctx, ClusterConfig{
    Name: "my-cluster",
    Type: KindCluster,
})

// Registry usage
var registry ProviderRegistry
registry.RegisterProvider(KindCluster, &MyProvider{})
provider, err := registry.GetProvider(KindCluster)
```
