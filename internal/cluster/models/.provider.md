Defines core interfaces for cluster provider abstraction layer in the OpenFrame platform. This file establishes the contract between cluster operations and their implementations, enabling support for multiple cluster types (kind, minikube, k3d) through a common interface.

## Key Components

### ClusterProvider Interface
- **Create()** - Creates a new cluster with specified configuration
- **Delete()** - Removes clusters with optional force flag
- **Start()** - Starts stopped clusters
- **List()** - Retrieves all managed clusters
- **Status()** - Gets detailed cluster status information
- **DetectType()** - Identifies cluster type for management
- **GetKubeconfig()** - Returns kubeconfig for cluster access

### ClusterService Interface
- **CreateCluster()** - Business logic for cluster creation
- **DeleteCluster()** - Handles cluster removal operations
- **StartCluster()** - Manages cluster startup processes
- **ListClusters()** - Aggregates clusters from all providers
- **GetClusterStatus()** - Retrieves comprehensive cluster information
- **DetectClusterType()** - Determines cluster provider type

### ProviderRegistry Interface
- **RegisterProvider()** - Registers provider implementations
- **GetProvider()** - Retrieves provider by cluster type
- **GetAllProviders()** - Returns complete provider mapping

## Usage Example

```go
// Register a kind provider
registry.RegisterProvider(KindClusterType, kindProvider)

// Create a new cluster
config := ClusterConfig{
    Name:    "dev-cluster",
    Type:    KindClusterType,
    Version: "1.28.0",
}

provider, _ := registry.GetProvider(KindClusterType)
err := provider.Create(ctx, config)

// List all clusters across providers
clusters, err := service.ListClusters(ctx)
for _, cluster := range clusters {
    fmt.Printf("Cluster: %s, Status: %s\n", cluster.Name, cluster.Status)
}
```
