Defines the core interfaces for cluster management providers, establishing contracts for cluster lifecycle operations and provider registration.

## Key Components

**ClusterProvider Interface**
- Core contract for cluster provider implementations
- Methods: `Create`, `Delete`, `Start`, `List`, `Status`, `DetectType`, `GetKubeconfig`

**ClusterService Interface** 
- Business logic layer for cluster operations
- Higher-level methods: `CreateCluster`, `DeleteCluster`, `StartCluster`, `ListClusters`, `GetClusterStatus`, `DetectClusterType`

**ProviderRegistry Interface**
- Manages multiple cluster providers
- Methods: `RegisterProvider`, `GetProvider`, `GetAllProviders`
