This file contains comprehensive unit tests for CLI flag structures and their associated functions in a Kubernetes cluster management tool.

## Key Components

- **Flag Structure Tests**: Tests for `GlobalFlags`, `CreateFlags`, `ListFlags`, `StatusFlags`, `DeleteFlags`, and `CleanupFlags` structs
- **Flag Addition Tests**: Tests for `AddGlobalFlags`, `AddCreateFlags`, `AddListFlags`, `AddStatusFlags`, `AddDeleteFlags`, and `AddCleanupFlags` functions
- **Flag Validation Tests**: Tests for validation functions including `ValidateCreateFlags`, `ValidateGlobalFlags`, etc.
- **Inheritance Testing**: Verifies that command-specific flags properly inherit global flags

## Usage Example

```go
func TestCreateFlags(t *testing.T) {
    // Test default values
    flags := &CreateFlags{}
    assert.Empty(t, flags.ClusterType)
    assert.Equal(t, 0, flags.NodeCount)
    
    // Test with custom values
    flags = &CreateFlags{
        ClusterType: "k3d",
        NodeCount:   5,
        K8sVersion:  "v1.25.0-k3s1",
        SkipWizard:  true,
    }
    
    // Validate flag addition to cobra command
    cmd := &cobra.Command{}
    AddCreateFlags(cmd, flags)
    typeFlag := cmd.Flags().Lookup("type")
    assert.NotNil(t, typeFlag)
}
```

The tests ensure proper flag initialization, cobra command integration, and validation logic for CLI operations like cluster creation, listing, and deletion.
