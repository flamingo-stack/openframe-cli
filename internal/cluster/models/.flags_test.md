This file contains comprehensive unit tests for command-line flags structures and their associated helper functions used in a Kubernetes cluster management CLI application.

## Key Components

**Test Suites:**
- `TestGlobalFlags` - Tests for common flags like verbose, dry-run, and force
- `TestCreateFlags` - Tests for cluster creation flags including type, node count, and K8s version
- `TestListFlags` - Tests for cluster listing flags with quiet mode
- `TestStatusFlags` - Tests for status command flags with detailed and no-apps options
- `TestDeleteFlags` - Tests for deletion command flags
- `TestCleanupFlags` - Tests for cleanup command flags
- `TestAdd*Flags` functions - Tests flag binding to Cobra commands
- `TestFlagValidation` - Tests validation logic for all flag types

## Usage Example

```go
func TestCreateFlags(t *testing.T) {
    // Test default values
    flags := &CreateFlags{}
    assert.Empty(t, flags.ClusterType)
    assert.Equal(t, 0, flags.NodeCount)
    
    // Test with custom values
    flags = &CreateFlags{
        ClusterType: "k3d",
        NodeCount:   5,
        K8sVersion:  "v1.25.0-k3s1",
        SkipWizard:  true,
    }
    
    // Validate flags
    err := ValidateCreateFlags(flags)
    assert.NoError(t, err)
}
```

The tests verify flag initialization, inheritance of global flags, Cobra command integration, and validation rules including node count constraints.
