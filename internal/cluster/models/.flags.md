This file defines command-line flag structures and validation logic for cluster management commands in the OpenFrame CLI tool.

## Key Components

**Flag Structures:**
- `GlobalFlags` - Alias for common flags shared across commands
- `CreateFlags` - Flags for cluster creation (type, node count, K8s version, skip wizard)
- `ListFlags` - Flags for listing clusters (quiet mode)
- `StatusFlags` - Flags for cluster status (detailed view, skip apps)
- `DeleteFlags` - Flags for cluster deletion (force confirmation skip)
- `CleanupFlags` - Flags for cleanup operations (aggressive cleanup mode)

**Setup Functions:**
- `AddGlobalFlags()` - Adds common flags to commands
- `Add*Flags()` - Command-specific flag setup functions

**Validation Functions:**
- `ValidateClusterName()` - Validates cluster names against Kubernetes DNS-1123 rules
- `Validate*Flags()` - Validates flag combinations for each command

## Usage Example

```go
// Setting up create command flags
var createFlags CreateFlags
cmd := &cobra.Command{Use: "create"}

// Add global and create-specific flags
AddGlobalFlags(cmd, &createFlags.GlobalFlags)
AddCreateFlags(cmd, &createFlags)

// Validate flags before execution
if err := ValidateCreateFlags(&createFlags); err != nil {
    return err
}

// Validate cluster name
if err := ValidateClusterName("my-cluster"); err != nil {
    return err
}
```
