This file defines command-line flags and validation logic for the OpenFrame CLI cluster management commands.

## Key Components

### Flag Structures
- **GlobalFlags** - Alias for `flags.CommonFlags` providing shared configuration options
- **CreateFlags** - Flags for cluster creation (type, node count, K8s version, wizard skip)
- **ListFlags** - Flags for listing clusters (quiet mode)
- **StatusFlags** - Flags for cluster status (detailed output, skip apps)
- **DeleteFlags** - Flags for cluster deletion (force confirmation skip)
- **CleanupFlags** - Flags for cluster cleanup (aggressive cleanup mode)

### Flag Setup Functions
- **AddGlobalFlags()** - Adds common flags using the shared flag manager
- **AddCreateFlags()**, **AddListFlags()**, etc. - Command-specific flag registration
- **ValidateClusterName()** - Validates cluster names against Kubernetes DNS-1123 subdomain rules
- **ValidateCreateFlags()**, **ValidateListFlags()**, etc. - Command-specific flag validation

## Usage Example

```go
// Setting up create command flags
var createFlags CreateFlags
createCmd := &cobra.Command{
    Use: "create",
    Run: func(cmd *cobra.Command, args []string) {
        // Validate flags before execution
        if err := ValidateCreateFlags(&createFlags); err != nil {
            return err
        }
        // Use validated flags
        fmt.Printf("Creating cluster with %d nodes\n", createFlags.NodeCount)
    },
}

// Add all relevant flags
AddGlobalFlags(createCmd, &createFlags.GlobalFlags)
AddCreateFlags(createCmd, &createFlags)

// Validate cluster name
if err := ValidateClusterName("my-cluster-01"); err != nil {
    log.Fatal(err)
}
```

The validation ensures cluster names follow Kubernetes naming conventions (1-63 characters, alphanumeric with hyphens, proper start/end characters) and validates flag combinations for each command type.
