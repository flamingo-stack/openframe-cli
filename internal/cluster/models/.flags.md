This Go file defines command-line flag structures and validation functions for various cluster management commands in the OpenFrame CLI, providing type-safe flag handling and Kubernetes-compliant cluster name validation.

## Key Components

**Flag Structures:**
- `GlobalFlags` - Type alias for common flags shared across commands
- `CreateFlags` - Cluster creation flags (type, node count, K8s version, wizard skip)
- `ListFlags` - Listing flags with quiet mode option
- `StatusFlags` - Status command flags (detailed view, skip apps)
- `DeleteFlags` - Deletion flags with force confirmation skip
- `CleanupFlags` - Cleanup flags with aggressive cleanup option

**Flag Setup Functions:**
- `AddGlobalFlags()` - Adds common flags to any command
- `AddCreateFlags()`, `AddListFlags()`, etc. - Add command-specific flags
- Each function binds flags to their respective struct fields

**Validation Functions:**
- `ValidateClusterName()` - Enforces Kubernetes DNS-1123 subdomain naming rules
- `ValidateCreateFlags()`, `ValidateListFlags()`, etc. - Validate flag combinations
- All validation functions check global flags first, then command-specific rules

## Usage Example

```go
// Setting up create command flags
createFlags := &models.CreateFlags{}
models.AddGlobalFlags(createCmd, &createFlags.GlobalFlags)
models.AddCreateFlags(createCmd, createFlags)

// Validating flags before execution
if err := models.ValidateCreateFlags(createFlags); err != nil {
    return fmt.Errorf("invalid flags: %w", err)
}

// Validating cluster name
if err := models.ValidateClusterName("my-cluster-01"); err != nil {
    return fmt.Errorf("invalid cluster name: %w", err)
}
```
