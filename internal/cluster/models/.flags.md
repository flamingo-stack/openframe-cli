This file defines command-line flag structures and validation logic for the OpenFrame CLI application, providing type-safe flag handling for different cluster management commands.

## Key Components

### Flag Structures
- **`GlobalFlags`** - Type alias for common flags shared across all commands
- **`CreateFlags`** - Flags for cluster creation (type, node count, K8s version, wizard options)
- **`ListFlags`** - Flags for listing clusters (quiet mode)
- **`StatusFlags`** - Flags for cluster status (detailed view, app checking)
- **`DeleteFlags`** - Flags for cluster deletion (force option)
- **`CleanupFlags`** - Flags for cleanup operations (aggressive cleanup)

### Flag Setup Functions
- **`AddGlobalFlags()`** - Adds common flags to any command
- **`Add[Command]Flags()`** - Command-specific flag registration functions
- **`ValidateClusterName()`** - Validates cluster names against Kubernetes DNS-1123 rules
- **`Validate[Command]Flags()`** - Validation functions for each flag structure

## Usage Example

```go
// Setting up create command flags
var createFlags CreateFlags
createCmd := &cobra.Command{
    Use: "create",
    RunE: func(cmd *cobra.Command, args []string) error {
        // Validate flags before execution
        if err := ValidateCreateFlags(&createFlags); err != nil {
            return err
        }
        
        // Validate cluster name if provided
        if len(args) > 0 {
            if err := ValidateClusterName(args[0]); err != nil {
                return err
            }
        }
        
        // Use flags in command logic
        fmt.Printf("Creating cluster with %d nodes\n", createFlags.NodeCount)
        return nil
    },
}

// Add flags to command
AddGlobalFlags(createCmd, &createFlags.GlobalFlags)
AddCreateFlags(createCmd, &createFlags)
```
