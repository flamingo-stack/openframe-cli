This file defines command-specific flag structures and validation functions for the OpenFrame CLI, providing a centralized way to manage flags across different cluster management commands.

## Key Components

### Flag Structures
- **`GlobalFlags`** - Type alias for shared `flags.CommonFlags`
- **`CreateFlags`** - Flags for cluster creation (type, node count, K8s version, wizard skip)
- **`ListFlags`** - Flags for listing clusters (quiet mode)
- **`StatusFlags`** - Flags for cluster status (detailed view, skip apps)
- **`DeleteFlags`** - Flags for cluster deletion (force option)
- **`CleanupFlags`** - Flags for cleanup operations (aggressive cleanup)

### Flag Setup Functions
- **`AddGlobalFlags()`** - Adds common flags to commands
- **`Add*Flags()`** - Command-specific flag registration functions

### Validation Functions
- **`ValidateClusterName()`** - Validates cluster names per Kubernetes DNS-1123 rules
- **`Validate*Flags()`** - Validates flag combinations for each command

## Usage Example

```go
// Setting up create command flags
createFlags := &CreateFlags{}
AddGlobalFlags(createCmd, &createFlags.GlobalFlags)
AddCreateFlags(createCmd, createFlags)

// Validating flags before execution
if err := ValidateCreateFlags(createFlags); err != nil {
    return fmt.Errorf("invalid flags: %w", err)
}

// Validating cluster name
if err := ValidateClusterName("my-cluster"); err != nil {
    return fmt.Errorf("invalid cluster name: %w", err)
}
```

The validation ensures cluster names follow Kubernetes naming conventions (alphanumeric + hyphens, max 63 chars, start/end with alphanumeric) and validates flag combinations for consistency.
