Defines flag structures and validation for OpenFrame CLI cluster management commands, providing type-safe command-line argument handling with Kubernetes naming conventions.

## Key Components

- **GlobalFlags** - Type alias for `flags.CommonFlags` providing shared configuration options
- **Command-specific flag structs** - `CreateFlags`, `ListFlags`, `StatusFlags`, `DeleteFlags`, `CleanupFlags` embedding global flags with command-specific options
- **Flag setup functions** - `AddGlobalFlags`, `AddCreateFlags`, `AddListFlags`, etc. for registering flags with Cobra commands
- **ValidateClusterName** - Validates cluster names against DNS-1123 subdomain rules with k3d-specific constraints
- **Validation functions** - Type-safe validation for each flag structure

## Usage Example

```go
// Setting up create command flags
createFlags := &CreateFlags{}
AddGlobalFlags(createCmd, &createFlags.GlobalFlags)
AddCreateFlags(createCmd, createFlags)

// Validating cluster name
if err := ValidateClusterName("my-cluster-01"); err != nil {
    return fmt.Errorf("invalid cluster name: %w", err)
}

// Validating create flags
if err := ValidateCreateFlags(createFlags); err != nil {
    return fmt.Errorf("invalid flags: %w", err)
}

// Accessing flags after parsing
clusterType := createFlags.ClusterType
nodeCount := createFlags.NodeCount
verbose := createFlags.Verbose // from GlobalFlags
```
