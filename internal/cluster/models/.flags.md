Provides command-line flag structures and validation for cluster management operations. Centralizes flag definitions for create, list, status, delete, and cleanup commands with embedded global flags.

## Key Components

**Flag Structures:**
- `CreateFlags` - Cluster creation options (type, node count, K8s version, wizard skip)
- `ListFlags` - List display options (quiet mode)
- `StatusFlags` - Status display options (detailed view, skip apps)
- `DeleteFlags` - Deletion options (force confirmation skip)
- `CleanupFlags` - Cleanup options (aggressive cleanup mode)

**Helper Functions:**
- `AddGlobalFlags()` - Adds common flags to commands
- `Add{Command}Flags()` - Command-specific flag registration
- `ValidateClusterName()` - Kubernetes naming convention validation
- `Validate{Command}Flags()` - Flag combination validation

## Usage Example

```go
// Setting up create command flags
var createFlags CreateFlags
cmd := &cobra.Command{
    Use: "create",
    Run: func(cmd *cobra.Command, args []string) {
        if err := ValidateCreateFlags(&createFlags); err != nil {
            return err
        }
        // Use flags...
    },
}

AddGlobalFlags(cmd, &createFlags.GlobalFlags)
AddCreateFlags(cmd, &createFlags)

// Validate cluster name
if err := ValidateClusterName("my-cluster-1"); err != nil {
    log.Fatal(err)
}
```
