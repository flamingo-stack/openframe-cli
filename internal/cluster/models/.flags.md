This file defines command-line flag structures and validation functions for cluster management operations in the OpenFrame CLI tool.

## Key Components

**Flag Structures:**
- `GlobalFlags` - Alias for common flags used across all commands
- `CreateFlags` - Flags for cluster creation (type, node count, K8s version, wizard skip)
- `ListFlags` - Flags for listing clusters (quiet mode)
- `StatusFlags` - Flags for cluster status (detailed view, skip apps)
- `DeleteFlags` - Flags for cluster deletion (force confirmation skip)
- `CleanupFlags` - Flags for cleanup operations (aggressive cleanup mode)

**Setup Functions:**
- `AddGlobalFlags()` - Adds common flags to any command
- `AddCreateFlags()`, `AddListFlags()`, etc. - Add command-specific flags

**Validation Functions:**
- `ValidateClusterName()` - Validates cluster names per Kubernetes DNS-1123 rules
- `ValidateCreateFlags()`, `ValidateListFlags()`, etc. - Validate flag combinations

## Usage Example

```go
// Setting up create command with flags
var createFlags CreateFlags
cmd := &cobra.Command{
    Use: "create",
    Run: func(cmd *cobra.Command, args []string) {
        // Command logic here
    },
}

AddGlobalFlags(cmd, &createFlags.GlobalFlags)
AddCreateFlags(cmd, &createFlags)

// Validate flags before execution
if err := ValidateCreateFlags(&createFlags); err != nil {
    return err
}

// Validate cluster name
if err := ValidateClusterName(clusterName); err != nil {
    return err
}
```
