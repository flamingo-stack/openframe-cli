This file defines the core interfaces for implementing commands in the OpenFrame CLI, providing a standardized contract for command structure, execution, and flag management.

## Key Components

**CommandFlags Interface**
- `GetGlobal()` - Returns global flags accessible across all commands

**CommandExecutor Interface**
- `GetExecutor()` - Provides access to the command execution engine

**ClusterCommand Interface**
- `GetCommand()` - Returns the cobra.Command instance for CLI integration
- `Execute()` - Handles the main command logic with context and arguments
- `ValidateFlags()` - Validates command-specific flag values
- `SetupFlags()` - Configures flags specific to the command

## Usage Example

```go
type MyClusterCommand struct{}

func (c *MyClusterCommand) GetCommand(flags CommandFlags) *cobra.Command {
    cmd := &cobra.Command{
        Use:   "deploy",
        Short: "Deploy cluster resources",
    }
    c.SetupFlags(cmd, flags)
    return cmd
}

func (c *MyClusterCommand) Execute(cmd *cobra.Command, args []string, 
    flags CommandFlags, executor CommandExecutor) error {
    // Implement command logic
    return executor.GetExecutor().Run("kubectl", "apply", "-f", args[0])
}

func (c *MyClusterCommand) ValidateFlags(flags CommandFlags) error {
    // Validate required flags
    return nil
}

func (c *MyClusterCommand) SetupFlags(cmd *cobra.Command, flags CommandFlags) {
    cmd.Flags().String("namespace", "default", "Target namespace")
}
```
