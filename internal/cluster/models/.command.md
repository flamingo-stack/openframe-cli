Defines the core interfaces for implementing CLI commands in the OpenFrame cluster management system, establishing a contract for command structure, execution, and flag handling.

## Key Components

- **CommandFlags**: Interface for accessing global command flags
- **CommandExecutor**: Interface providing access to command execution context
- **ClusterCommand**: Main interface defining the complete command lifecycle including:
  - `GetCommand()`: Returns the cobra.Command instance
  - `Execute()`: Contains the command's business logic
  - `ValidateFlags()`: Validates command-specific flags
  - `SetupFlags()`: Configures command flags
