Test file that validates custom error types for cluster management operations. Provides comprehensive testing for error creation, formatting, type assertion, and error chaining behavior.

## Key Components

**Test Functions:**
- `TestErrClusterNotFound` - Tests cluster not found error creation and unwrapping
- `TestErrProviderNotFound` - Tests provider not found error with cluster types
- `TestErrInvalidClusterConfig` - Tests configuration validation error handling
- `TestErrClusterAlreadyExists` - Tests duplicate cluster detection errors
- `TestErrClusterOperation` - Tests operation-specific error wrapping
- `TestErrorFormatting` - Validates error message formatting
- `TestErrorChaining` - Tests error unwrapping and chaining behavior

## Usage Example

```go
// Run specific error tests
func TestMyErrorHandling(t *testing.T) {
    // Test cluster not found error
    err := NewClusterNotFoundError("test-cluster")
    var clusterErr ErrClusterNotFound
    assert.True(t, errors.As(err, &clusterErr))
    assert.Equal(t, "test-cluster", clusterErr.Name)
    
    // Test error chaining
    configErr := NewInvalidConfigError("nodeCount", -1, "must be positive")
    opErr := NewClusterOperationError("create", "test", configErr)
    
    // Verify both errors are in the chain
    var invalidErr ErrInvalidClusterConfig
    assert.True(t, errors.As(opErr, &invalidErr))
}
```
