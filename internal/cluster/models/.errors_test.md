Test suite for custom cluster management error types, providing comprehensive validation of error creation, formatting, and chaining functionality.

## Key Components

- **TestErrClusterNotFound** - Tests cluster not found error creation and type assertions
- **TestErrProviderNotFound** - Tests provider not found error with cluster types
- **TestErrInvalidClusterConfig** - Tests configuration validation errors with field details
- **TestErrClusterAlreadyExists** - Tests duplicate cluster detection errors
- **TestErrClusterOperation** - Tests operational errors with cause wrapping
- **TestErrorFormatting** - Validates error message formatting consistency
- **TestErrorChaining** - Tests error wrapping and unwrapping behavior

## Usage Example

```go
// Run all error tests
go test ./models -v -run TestErr

// Run specific error type tests
go test ./models -v -run TestErrClusterNotFound

// Test error formatting
go test ./models -v -run TestErrorFormatting

// Test error chaining behavior
go test ./models -v -run TestErrorChaining
```

The tests ensure proper error type assertions using `errors.As()`, validate error message formatting, and verify that wrapped errors can be correctly unwrapped and identified in error chains.
