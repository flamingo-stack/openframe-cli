Test suite for cluster-related model structures and data types. Provides comprehensive testing coverage for Kubernetes cluster configuration, information, and provider-specific options.

## Key Components

- **ClusterType Tests**: Validates cluster type constants (K3d, GKE) and string conversions
- **ClusterConfig Tests**: Tests cluster configuration structure with name, type, node count, and Kubernetes version
- **ClusterInfo Tests**: Validates cluster runtime information including status, nodes, and creation time
- **NodeInfo Tests**: Tests individual node information with name, status, and role validation
- **ProviderOptions Tests**: Tests provider-specific configuration options for K3d and GKE
- **Serialization Tests**: Basic validation of JSON struct tag correctness

## Usage Example

```go
// Test cluster configuration creation
config := ClusterConfig{
    Name:       "test-cluster",
    Type:       ClusterTypeK3d,
    NodeCount:  3,
    K8sVersion: "v1.25.0-k3s1",
}

// Test cluster information with nodes
info := ClusterInfo{
    Name:      "production-cluster",
    Type:      ClusterTypeGKE,
    Status:    "running",
    NodeCount: 5,
    Nodes: []NodeInfo{
        {Name: "master-1", Status: "ready", Role: "control-plane"},
        {Name: "worker-1", Status: "ready", Role: "worker"},
    },
}

// Test provider-specific options
options := ProviderOptions{
    K3d: &K3dOptions{
        PortMappings: []string{"8080:80@loadbalancer"},
    },
    Verbose: true,
}
```
