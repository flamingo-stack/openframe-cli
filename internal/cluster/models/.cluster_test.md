Test suite for cluster management models providing comprehensive validation of cluster types, configurations, and provider-specific options. Validates struct initialization, field assignments, and serialization behavior for Kubernetes cluster management components.

## Key Components

- **TestClusterType** - Validates `ClusterType` constants and string conversions for k3d and GKE cluster types
- **TestClusterConfig** - Tests cluster configuration struct with name, type, node count, and Kubernetes version fields
- **TestClusterInfo** - Validates cluster information struct including status, creation time, and node details
- **TestNodeInfo** - Tests node information with name, status, and role validation across different node types
- **TestProviderOptions** - Validates provider-specific configuration options for K3d and GKE
- **TestK3dOptions/TestGKEOptions** - Tests provider-specific option structs for port mappings and cloud configurations
- **TestJSONSerialization** - Validates struct field serialization behavior

## Usage Example

```go
func TestMyClusterLogic(t *testing.T) {
    // Create test cluster configuration
    config := ClusterConfig{
        Name:       "test-cluster",
        Type:       ClusterTypeK3d,
        NodeCount:  3,
        K8sVersion: "v1.25.0-k3s1",
    }
    
    // Test cluster info with nodes
    nodes := []NodeInfo{
        {Name: "node1", Status: "ready", Role: "control-plane"},
        {Name: "node2", Status: "ready", Role: "worker"},
    }
    
    info := ClusterInfo{
        Name:      config.Name,
        Type:      config.Type,
        Status:    "running",
        NodeCount: len(nodes),
        Nodes:     nodes,
        CreatedAt: time.Now(),
    }
    
    // Test with provider options
    options := ProviderOptions{
        K3d: &K3dOptions{
            PortMappings: []string{"8080:80@loadbalancer"},
        },
        Verbose: true,
    }
    
    assert.Equal(t, "test-cluster", info.Name)
    assert.Equal(t, ClusterTypeK3d, info.Type)
}
```
