Test suite for the cluster service package that validates cluster management functionality through comprehensive unit tests.

## Key Components

**Test Functions:**
- `TestNewClusterService` - Validates service constructor initialization
- `TestNewClusterServiceWithOptions` - Tests service creation with custom options
- `TestClusterService_CreateCluster` - Tests cluster creation functionality
- `TestClusterService_DeleteCluster` - Tests cluster deletion operations
- `TestClusterService_ListClusters` - Tests cluster listing functionality
- `TestClusterService_GetClusterStatus` - Tests status retrieval
- `TestClusterService_DetectClusterType` - Tests cluster type detection
- `TestClusterService_CleanupCluster` - Tests cluster cleanup operations
- `TestClusterService_ShowClusterStatus` - Tests status display
- `TestClusterService_DisplayClusterList` - Tests list display formatting
- `TestClusterService_WithRealExecutor` - Tests with real command executor

**Helper Functions:**
- `createTestExecutor()` - Creates mock executor with predefined k3d responses

## Usage Example

```go
// Run all tests
go test ./cluster -v

// Run specific test
go test ./cluster -run TestNewClusterService

// Create mock executor for testing
func TestCustomClusterOperation(t *testing.T) {
    exec := createTestExecutor()
    service := NewClusterService(exec)
    
    config := models.ClusterConfig{
        Name:      "test-cluster",
        Type:      models.ClusterTypeK3d,
        NodeCount: 1,
    }
    
    err := service.CreateCluster(config)
    if err != nil {
        t.Errorf("unexpected error: %v", err)
    }
}
```
