This file contains comprehensive unit tests for command helper utilities in the OpenFrame CLI, focusing on global flag management, command service creation, and testing support functionality.

## Key Components

- **Flag Lifecycle Tests**: `TestInitGlobalFlags`, `TestGetGlobalFlags` - Test initialization and retrieval of global flags
- **Service Creation Tests**: `TestGetCommandService` - Validates command service creation with various flag states  
- **Command Wrapper Tests**: `TestWrapCommandWithCommonSetup` - Tests command function wrapping with error handling
- **Flag Operations Tests**: `TestSyncGlobalFlags`, `TestValidateGlobalFlags` - Test flag synchronization and validation
- **Testing Support Tests**: `TestTestingSupport` - Test helper functions for setting mock executors and resetting flags
- **Integration Test Support**: `TestIntegrationTestSupport` - Tests integration testing flag management
- **Edge Case Tests**: `TestCmdHelpersEdgeCases`, `TestConcurrency` - Comprehensive testing of edge cases and concurrent access

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    // Reset flags for clean test state
    ResetGlobalFlags()
    
    // Initialize with test executor
    mockExecutor := testutil.NewTestMockExecutor()
    SetTestExecutor(mockExecutor)
    
    // Test command functionality
    service := GetCommandService()
    assert.NotNil(t, service)
    
    // Test command wrapping
    wrappedFunc := WrapCommandWithCommonSetup(func(cmd *cobra.Command, args []string) error {
        return nil
    })
    
    cmd := &cobra.Command{}
    err := wrappedFunc(cmd, []string{})
    assert.NoError(t, err)
}
```
