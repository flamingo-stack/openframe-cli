Utility functions for managing global command flags and creating cluster services across all CLI commands, providing centralized configuration management and error handling for the OpenFrame CLI.

## Key Components

- **Global Flag Management**: Thread-safe initialization and access to global flags via `InitGlobalFlags()`, `GetGlobalFlags()`, and `SyncGlobalFlags()`
- **Service Factory Functions**: `GetCommandService()` and `GetSuppressedCommandService()` for creating cluster service instances
- **Command Wrapper**: `WrapCommandWithCommonSetup()` provides consistent error handling and setup for all CLI commands
- **Test Support**: Functions like `SetTestExecutor()` and `ResetGlobalFlags()` for test isolation and mocking
- **Integration Test Helpers**: Compatibility functions for integration testing with verbose mode control

## Usage Example

```go
// Initialize global flags at startup
utils.InitGlobalFlags()

// Create a command with common setup
cmd := &cobra.Command{
    Use: "deploy",
    RunE: utils.WrapCommandWithCommonSetup(func(cmd *cobra.Command, args []string) error {
        service := utils.GetCommandService()
        return service.Deploy(args[0])
    }),
}

// For testing, inject a mock executor
mockExec := &MockExecutor{}
utils.SetTestExecutor(mockExec)
defer utils.ResetGlobalFlags()
```
