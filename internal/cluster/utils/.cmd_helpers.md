Provides utility functions for managing global flags, command services, and common CLI setup patterns across cluster commands. This file serves as a centralized helper for command initialization, service creation, and error handling workflows.

## Key Components

- **Global Flag Management**: `InitGlobalFlags()`, `GetGlobalFlags()`, `SyncGlobalFlags()` - Thread-safe initialization and access to global command flags
- **Command Service Factory**: `GetCommandService()`, `GetSuppressedCommandService()` - Creates cluster services with appropriate executors for normal and automated operations  
- **Command Wrapper**: `WrapCommandWithCommonSetup()` - Wraps cobra commands with standardized error handling and setup logic
- **Testing Support**: `SetTestExecutor()`, `ResetGlobalFlags()` - Utilities for injecting mock executors and resetting state in tests
- **Integration Test Helpers**: `SetVerboseForIntegrationTesting()`, `ResetTestFlags()` - Compatibility functions for integration testing scenarios

## Usage Example

```go
// Initialize global flags at application start
InitGlobalFlags()

// Create a command service for business operations
service := GetCommandService()

// Wrap a cobra command with common setup
cmd := &cobra.Command{
    Use: "deploy",
    RunE: WrapCommandWithCommonSetup(func(cmd *cobra.Command, args []string) error {
        return service.Deploy(args[0])
    }),
}

// In tests, inject a mock executor
SetTestExecutor(mockExecutor)
defer ResetGlobalFlags()
```
