Helper utilities for command setup, flag management, and service creation in the OpenFrame CLI. Provides centralized flag container management and standardized command wrapping with error handling.

## Key Components

- **Global Flag Management**
  - `InitGlobalFlags()` - Thread-safe initialization of global flag container
  - `GetGlobalFlags()` - Returns current global flags instance
  - `SyncGlobalFlags()` - Synchronizes flags across commands
  - `ValidateGlobalFlags()` - Validates global flag values

- **Service Creation**
  - `GetCommandService()` - Creates cluster service with current flags
  - `GetSuppressedCommandService()` - Creates service with UI suppression for automation

- **Command Wrapping**
  - `WrapCommandWithCommonSetup()` - Standardizes command execution with error handling

- **Testing Utilities**
  - `SetTestExecutor()` - Injects mock executor for testing
  - `ResetGlobalFlags()` - Cleans up global state
  - `SetVerboseForIntegrationTesting()` - Configures verbose mode for tests

## Usage Example

```go
// Initialize flags once at startup
utils.InitGlobalFlags()

// Create command with standardized setup
var myCmd = &cobra.Command{
    Use: "example",
    RunE: utils.WrapCommandWithCommonSetup(func(cmd *cobra.Command, args []string) error {
        // Get service with current flags
        service := utils.GetCommandService()
        return service.DoSomething()
    }),
}

// For testing - inject mock executor
mockExec := &executor.MockCommandExecutor{}
utils.SetTestExecutor(mockExec)
defer utils.ResetGlobalFlags()
```
