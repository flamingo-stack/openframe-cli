A shared utilities module that provides global flag management and command setup helpers for the OpenFrame CLI, centralizing common operations like error handling, service creation, and test configuration.

## Key Components

- **`globalFlags`** - Thread-safe global flag container shared across all cluster commands
- **`InitGlobalFlags()`** - Initializes the global flag container with mutex protection
- **`GetCommandService()`** - Creates cluster service instances with proper executor configuration
- **`GetSuppressedCommandService()`** - Creates cluster service with UI suppression for automation
- **`WrapCommandWithCommonSetup()`** - Command wrapper that provides standardized error handling and setup
- **`SyncGlobalFlags()`/`ValidateGlobalFlags()`** - Flag synchronization and validation utilities
- **Test helpers** - Functions for setting test executors and managing integration test flags

## Usage Example

```go
// Initialize global flags at application startup
utils.InitGlobalFlags()

// Wrap command functions with common setup
var myCmd = &cobra.Command{
    Use: "deploy",
    RunE: utils.WrapCommandWithCommonSetup(func(cmd *cobra.Command, args []string) error {
        // Get service with current flag configuration
        service := utils.GetCommandService()
        return service.Deploy(args[0])
    }),
}

// In tests, inject mock executor
mockExec := &testutil.MockExecutor{}
utils.SetTestExecutor(mockExec)

// Reset state after tests
defer utils.ResetGlobalFlags()
```

The module handles dry-run mode, verbose output, error classification, and provides consistent behavior across all CLI commands while supporting both real execution and testing scenarios.
