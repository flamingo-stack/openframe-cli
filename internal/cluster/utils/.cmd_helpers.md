This file provides utility functions for managing global CLI command state, flag synchronization, and command service creation across the OpenFrame CLI application.

## Key Components

- **GlobalFlags Management**: Thread-safe global flag container initialization and access via `InitGlobalFlags()`, `GetGlobalFlags()`, and `ResetGlobalFlags()`
- **Command Services**: Factory functions `GetCommandService()` and `GetSuppressedCommandService()` for creating cluster services with appropriate executors
- **Command Wrapper**: `WrapCommandWithCommonSetup()` provides standardized error handling and setup for CLI commands
- **Flag Operations**: `SyncGlobalFlags()` and `ValidateGlobalFlags()` for flag management across commands
- **Testing Support**: Functions like `SetTestExecutor()` and `ResetTestFlags()` for test environment setup

## Usage Example

```go
// Initialize and use global flags
utils.InitGlobalFlags()
flags := utils.GetGlobalFlags()

// Create a command service
service := utils.GetCommandService()

// Wrap a command with common setup
cmd.RunE = utils.WrapCommandWithCommonSetup(func(cmd *cobra.Command, args []string) error {
    return service.CreateCluster(clusterConfig)
})

// For testing
utils.SetTestExecutor(mockExecutor)
defer utils.ResetTestFlags()
```

The file serves as a central coordination point for CLI command infrastructure, ensuring consistent flag handling, service creation, and error management across all commands.
