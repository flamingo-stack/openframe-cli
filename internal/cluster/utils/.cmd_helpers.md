A utility module that provides command helpers for managing global flags, service creation, and command execution in the OpenFrame CLI application.

## Key Components

### Global Flag Management
- **`InitGlobalFlags()`** - Thread-safe initialization of global flag container
- **`GetGlobalFlags()`** - Retrieves current global flags instance
- **`SyncGlobalFlags()`** - Synchronizes global flags to all command flags
- **`ValidateGlobalFlags()`** - Validates global flag values

### Service Creation
- **`GetCommandService()`** - Creates standard cluster service with current flag configuration
- **`GetSuppressedCommandService()`** - Creates cluster service with UI suppression for automation

### Command Wrapper
- **`WrapCommandWithCommonSetup()`** - Wraps command functions with error handling and setup logic

### Testing Support
- **`SetTestExecutor()`** - Injects test executor for unit testing
- **`ResetGlobalFlags()`** - Resets global state for test isolation
- **`SetVerboseForIntegrationTesting()`** - Configures verbose mode for integration tests

## Usage Example

```go
// Initialize global flags before command execution
utils.InitGlobalFlags()

// Create a command with common setup
var myCmd = &cobra.Command{
    Use: "example",
    RunE: utils.WrapCommandWithCommonSetup(func(cmd *cobra.Command, args []string) error {
        // Get service with current flag configuration
        service := utils.GetCommandService()
        
        // Execute business logic
        return service.SomeOperation()
    }),
}

// For testing, inject a mock executor
utils.SetTestExecutor(mockExecutor)
defer utils.ResetGlobalFlags()
```

The module ensures thread-safe access to global flags and provides consistent service creation patterns across all CLI commands, with special handling for testing scenarios and error management.
