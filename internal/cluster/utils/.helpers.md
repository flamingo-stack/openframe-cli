Provides utility functions for cluster validation, type parsing, and error handling in the OpenFrame CLI tool.

## Key Components

- **ClusterSelectionResult**: Deprecated struct for cluster selection results (use UI types instead)
- **ValidateClusterName()**: Validates cluster name format using domain validation
- **ParseClusterType()**: Converts string to ClusterType enum with K3d default
- **GetNodeCount()**: Returns validated node count with default of 3 nodes
- **CreateClusterError()**: Creates standardized cluster operation errors

## Usage Example

```go
import "github.com/flamingo-stack/openframe-cli/internal/utils"

// Validate cluster name
err := utils.ValidateClusterName("my-cluster")
if err != nil {
    log.Fatal(err)
}

// Parse cluster type
clusterType := utils.ParseClusterType("gke") // Returns ClusterTypeGKE
defaultType := utils.ParseClusterType("invalid") // Returns ClusterTypeK3d

// Get validated node count
nodeCount := utils.GetNodeCount(0) // Returns 3 (default)
nodeCount = utils.GetNodeCount(5)  // Returns 5

// Create standardized error
err = utils.CreateClusterError("create", "my-cluster", models.ClusterTypeK3d, originalErr)
```
