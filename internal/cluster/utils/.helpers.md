This file provides utility functions for cluster operations and validation, serving as a bridge between cluster models and CLI commands with many functions migrated to specialized packages for better code organization.

## Key Components

- **ClusterSelectionResult** - Deprecated struct for cluster selection results (use UI types instead)
- **ValidateClusterName()** - Validates cluster name format using domain validation
- **ParseClusterType()** - Converts string input to ClusterType enum with k3d as default
- **GetNodeCount()** - Returns validated node count with default fallback to 3 nodes
- **CreateClusterError()** - Creates standardized error messages for cluster operations

## Usage Example

```go
// Validate cluster name
err := ValidateClusterName("my-cluster-123")
if err != nil {
    return fmt.Errorf("invalid cluster name: %w", err)
}

// Parse cluster type from user input
clusterType := ParseClusterType("gke") // Returns models.ClusterTypeGKE

// Get validated node count
nodeCount := GetNodeCount(-1) // Returns 3 (default)
nodeCount = GetNodeCount(5)   // Returns 5

// Create standardized error
if err := someClusterOperation(); err != nil {
    return CreateClusterError("create", "my-cluster", models.ClusterTypeK3d, err)
}
```

**Note:** Many functions have been moved to the `ui` package for better separation of concerns. Use `ui.ConfirmClusterDeletion()`, `ui.ShowClusterOperationCancelled()`, and similar functions instead of the deprecated helpers.
