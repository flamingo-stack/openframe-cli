This file contains cluster validation utilities and helper functions for the OpenFrame CLI cluster management system. It provides standardized validation, parsing, and error handling for cluster operations.

## Key Components

- **`ValidateClusterName(name string) error`** - Validates cluster name format using domain validation rules
- **`ParseClusterType(typeStr string) models.ClusterType`** - Converts string input to ClusterType enum (k3d, gke)
- **`GetNodeCount(nodeCount int) int`** - Returns validated node count with default fallback to 3 nodes
- **`CreateClusterError(operation, clusterName string, clusterType models.ClusterType, err error) error`** - Creates standardized cluster operation errors
- **`ClusterSelectionResult`** - Deprecated struct for cluster selection results (use UI types instead)

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/flamingo-stack/openframe-cli/internal/utils"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
)

func createCluster(name, typeStr string, nodes int) error {
    // Validate inputs
    if err := utils.ValidateClusterName(name); err != nil {
        return err
    }
    
    clusterType := utils.ParseClusterType(typeStr)
    nodeCount := utils.GetNodeCount(nodes)
    
    // Simulate cluster creation
    if err := performClusterCreation(name, clusterType, nodeCount); err != nil {
        return utils.CreateClusterError("create", name, clusterType, err)
    }
    
    return nil
}
```
