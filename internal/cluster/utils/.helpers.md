This file provides utility functions for cluster operations in the OpenFrame CLI tool, including validation, type parsing, and error handling helpers.

## Key Components

**Validation Functions:**
- `ValidateClusterName(name string) error` - Validates cluster name format using domain validation rules
- `ParseClusterType(typeStr string) models.ClusterType` - Converts string input to ClusterType enum (k3d, gke)
- `GetNodeCount(nodeCount int) int` - Returns validated node count with default of 3

**Error Handling:**
- `CreateClusterError(operation, clusterName string, clusterType models.ClusterType, err error) error` - Creates standardized cluster operation errors

**Deprecated Types:**
- `ClusterSelectionResult` - Legacy struct for cluster selection results (replaced by UI types)

## Usage Example

```go
// Validate cluster name
if err := utils.ValidateClusterName("my-cluster"); err != nil {
    return fmt.Errorf("invalid cluster name: %w", err)
}

// Parse cluster type from user input
clusterType := utils.ParseClusterType("k3d")
fmt.Printf("Selected type: %s\n", clusterType)

// Get validated node count
nodeCount := utils.GetNodeCount(0) // Returns 3 (default)
nodeCount = utils.GetNodeCount(5)  // Returns 5

// Create standardized error
err := utils.CreateClusterError("create", "my-cluster", models.ClusterTypeK3d, 
    fmt.Errorf("connection failed"))
```

**Note:** Many functions have been moved to the `ui` package for better separation of concerns. This file now focuses primarily on validation and error handling utilities.
