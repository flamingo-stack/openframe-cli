This file provides utility functions for cluster operations, validation, and error handling in the OpenFrame CLI tool.

## Key Components

- **ClusterSelectionResult** - Deprecated struct for cluster selection results (use UI types instead)
- **ValidateClusterName()** - Validates cluster name format using domain validation
- **ParseClusterType()** - Converts string to ClusterType enum with k3d as default
- **GetNodeCount()** - Returns validated node count with default of 3 nodes
- **CreateClusterError()** - Creates standardized cluster operation errors

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/flamingo-stack/openframe-cli/internal/utils"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
)

func main() {
    // Validate cluster name
    if err := utils.ValidateClusterName("my-cluster"); err != nil {
        fmt.Printf("Invalid cluster name: %v\n", err)
        return
    }
    
    // Parse cluster type
    clusterType := utils.ParseClusterType("gke")
    fmt.Printf("Cluster type: %s\n", clusterType)
    
    // Get validated node count
    nodeCount := utils.GetNodeCount(0) // Returns 3 (default)
    fmt.Printf("Node count: %d\n", nodeCount)
    
    // Create standardized error
    err := utils.CreateClusterError("create", "my-cluster", models.ClusterTypeK3d, 
        fmt.Errorf("connection failed"))
    fmt.Printf("Error: %v\n", err)
}
```

Note: Many functions have been moved to the UI package for better separation of concerns. This file now focuses primarily on validation and error handling utilities.
