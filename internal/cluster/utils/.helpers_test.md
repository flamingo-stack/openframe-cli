This file contains comprehensive unit tests for cluster utility helper functions, validating cluster name validation, type parsing, node count handling, and error creation functionality.

## Key Components

- **TestValidateClusterName**: Tests cluster name validation including valid names, invalid characters, empty/whitespace-only names
- **TestParseClusterType**: Tests parsing of cluster types (k3d, GKE) with case-insensitive handling and fallback behavior
- **TestGetNodeCount**: Tests node count validation with boundary conditions and default fallback to 3
- **TestClusterSelectionResult**: Tests the cluster selection result struct creation
- **TestCreateClusterError**: Tests cluster error creation and wrapping functionality
- **TestTypeAliases**: Tests model type aliases work correctly
- **TestConstants**: Validates cluster type constants
- **TestEdgeCases**: Comprehensive edge case testing for all functions

## Usage Example

```go
// Run specific test function
go test -run TestValidateClusterName

// Run all tests in the file
go test ./utils -v

// Run tests with coverage
go test -cover ./utils

// Example test execution for cluster name validation
func TestValidateClusterName(t *testing.T) {
    err := ValidateClusterName("test-cluster")
    assert.NoError(t, err)
    
    err = ValidateClusterName("invalid_name")
    assert.Error(t, err)
}
```
