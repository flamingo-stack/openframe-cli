Test suite for cluster utility helper functions, providing comprehensive coverage of validation, parsing, and error handling functionality.

## Key Components

- **TestValidateClusterName**: Tests cluster name validation including valid names, invalid characters, empty/whitespace-only inputs
- **TestParseClusterType**: Tests cluster type parsing with case-insensitive handling for k3d/GKE types and default fallback
- **TestGetNodeCount**: Tests node count validation with boundary cases and default value handling
- **TestClusterSelectionResult**: Tests the cluster selection result struct creation
- **TestCreateClusterError**: Tests cluster operation error creation and wrapping
- **TestTypeAliases**: Tests cluster type and info model aliases
- **TestEdgeCases**: Comprehensive edge case testing for all utility functions

## Usage Example

```go
// Run specific test function
go test -run TestValidateClusterName

// Run all helper tests
go test ./utils -v

// Test with coverage
go test -cover ./utils

// Run edge case tests only
go test -run TestEdgeCases
```

The test suite validates cluster name formatting rules, ensures proper cluster type parsing with defaults, handles node count boundaries, and tests error creation/wrapping functionality. It includes extensive edge case coverage for whitespace handling, case sensitivity, and boundary values.
