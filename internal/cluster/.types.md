Defines the core type system for cluster command flag management, providing a centralized container for all cluster-related command flags and their synchronization.

## Key Components

- **`FlagContainer`** - Main struct that holds all cluster command flags and dependencies
  - Contains flag instances for all cluster commands (create, list, status, delete, cleanup)
  - Includes executor and test manager dependencies
  - Implements `models.CommandFlags` interface
- **`NewFlagContainer()`** - Constructor that creates a new flag container with sensible defaults
- **`SyncGlobalFlags()`** - Synchronizes global flags across all command-specific flag structures
- **`Reset()`** - Resets all flags to zero values for testing purposes

## Usage Example

```go
// Create a new flag container with defaults
flags := NewFlagContainer()

// Access specific command flags
flags.Create.ClusterType = "k3d"
flags.Create.NodeCount = 5

// Synchronize global settings across all commands
flags.Global.Verbose = true
flags.SyncGlobalFlags()

// Get global flags (interface implementation)
globalFlags := flags.GetGlobal()

// Reset for testing
flags.Reset()
```

The container provides a clean abstraction for managing the complex flag inheritance patterns needed across different cluster management commands.
