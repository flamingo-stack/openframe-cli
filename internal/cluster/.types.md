Defines a unified flag container structure that manages command-line flags and dependencies for cluster operations. This centralizes flag management across all cluster commands (create, list, status, delete, cleanup) and provides interfaces for testing and command execution.

## Key Components

- **`FlagContainer`** - Main struct that holds all flag structures and dependencies for cluster commands
- **`NewFlagContainer()`** - Constructor that initializes the container with default values (k3d cluster, 3 nodes, K8s v1.31.5)
- **`SyncGlobalFlags()`** - Synchronizes global flags across all command-specific flag structures
- **`Reset()`** - Resets all flags to zero values for testing purposes
- **Interface implementations** - Implements `models.CommandFlags` and provides executor access

## Usage Example

```go
// Create a new flag container with defaults
flags := NewFlagContainer()

// Set executor for command execution
flags.Executor = executor.NewRealExecutor()

// Configure global flags
flags.Global.Verbose = true
flags.Global.DryRun = true

// Sync global settings to all commands
flags.SyncGlobalFlags()

// Access command-specific flags
flags.Create.NodeCount = 5
flags.Create.ClusterType = "k3d"

// Reset for testing
flags.Reset()
```

The structure supports dependency injection for testing by allowing custom executors and K3D managers to be set, while maintaining clean separation between flag data and execution logic.
