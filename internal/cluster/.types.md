Central type definitions for the cluster command system, providing a unified container for all flag structures and their dependencies.

## Key Components

- **FlagContainer**: Main struct that holds all cluster command flag instances and their dependencies
- **GetGlobal()**: Interface method returning global flags for command execution
- **GetExecutor()**: Interface method providing access to the command executor
- **NewFlagContainer()**: Constructor that initializes a new flag container with default values
- **SyncGlobalFlags()**: Method to propagate global flags across all command-specific flag structures
- **Reset()**: Utility method for resetting all flags to zero values (primarily for testing)

The container includes flags for all cluster operations (Create, List, Status, Delete, Cleanup) plus global configuration flags, along with executor and test manager dependencies.

## Usage Example

```go
// Create a new flag container with defaults
flags := NewFlagContainer()

// Access specific command flags
flags.Create.ClusterType = "k3d"
flags.Create.NodeCount = 5

// Set global configuration
flags.Global.Verbose = true
flags.Global.ConfigPath = "/path/to/config"

// Sync global flags to all commands
flags.SyncGlobalFlags()

// Use in command execution
executor := &executor.RealExecutor{}
flags.Executor = executor

// Reset for testing
flags.Reset()
```
