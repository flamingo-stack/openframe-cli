This file defines the central flag management structure for cluster commands, providing a unified container for all command-specific flags and their synchronization.

## Key Components

**Types:**
- `FlagContainer` - Main structure holding all flag types and dependencies for cluster commands

**Methods:**
- `NewFlagContainer()` - Creates initialized flag container with default values
- `GetGlobal()` - Returns global flags (implements `models.CommandFlags` interface)
- `GetExecutor()` - Returns command executor (implements `models.CommandExecutor` interface)
- `SyncGlobalFlags()` - Propagates global flag values to all command-specific flags
- `Reset()` - Resets all flags to zero values for testing

## Usage Example

```go
// Create a new flag container with defaults
flags := NewFlagContainer()

// Set global configuration
flags.Global.Verbose = true
flags.Global.DryRun = false

// Sync global flags to all commands
flags.SyncGlobalFlags()

// Access command-specific flags
flags.Create.ClusterType = "k3d"
flags.Create.NodeCount = 5

// Set executor for command execution
flags.Executor = &realExecutor{}

// Use in cluster operations
clusterManager := NewManager(flags)
```

The container includes default values for create operations (k3d cluster with 3 nodes running Kubernetes v1.31.5-k3s1) and supports dependency injection for testing through the `TestManager` field.
