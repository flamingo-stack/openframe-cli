Defines core types and flag management structures for the OpenFrame cluster command system, providing a centralized container for all cluster-related command flags and their dependencies.

## Key Components

- **`FlagContainer`** - Central structure holding all cluster command flags and dependencies
- **`NewFlagContainer()`** - Factory function creating initialized flag container with defaults
- **`SyncGlobalFlags()`** - Synchronizes global settings across all command-specific flags
- **`Reset()`** - Resets all flags to zero values for testing purposes
- **Interface implementations** - Provides `GetGlobal()` and `GetExecutor()` methods for command interfaces

## Usage Example

```go
// Create a new flag container with defaults
flags := NewFlagContainer()

// Access specific command flags
flags.Create.ClusterType = "k3d"
flags.Create.NodeCount = 5
flags.Create.K8sVersion = "v1.31.5-k3s1"

// Set global flags
flags.Global.Verbose = true
flags.Global.ConfigPath = "/path/to/config"

// Sync global settings to all commands
flags.SyncGlobalFlags()

// Use in cluster operations
executor := &executor.RealExecutor{}
flags.Executor = executor

// Reset for testing
flags.Reset()
```

The container provides a clean separation between flag definitions and their usage, with built-in dependency injection for command execution and testing scenarios.
