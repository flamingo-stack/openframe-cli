This file defines the central type system for managing command flags and dependencies across cluster operations in the OpenFrame CLI tool.

## Key Components

**FlagContainer**: Main structure that aggregates all command-specific flag types and provides dependency injection for command execution and testing.

**Core Methods**:
- `NewFlagContainer()` - Creates initialized container with default values (k3d cluster, 3 nodes, k8s v1.31.5-k3s1)
- `SyncGlobalFlags()` - Propagates global flags to all command-specific flag structures
- `Reset()` - Clears all flags to zero values for testing
- `GetGlobal()` - Interface method returning global flags
- `GetExecutor()` - Interface method returning command executor

## Usage Example

```go
// Create a new flag container with defaults
flags := NewFlagContainer()

// Set global configuration
flags.Global.Verbose = true
flags.Global.ConfigPath = "/custom/path"

// Configure create-specific flags
flags.Create.ClusterName = "my-cluster"
flags.Create.NodeCount = 5

// Sync global flags to all commands
flags.SyncGlobalFlags()

// Use with dependency injection
flags.Executor = &executor.RealCommandExecutor{}
flags.TestManager = k3d.NewK3dManager()

// Reset for testing
flags.Reset()
```

The structure supports both production execution and unit testing through configurable executor and manager dependencies.
