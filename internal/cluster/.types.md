Defines the central type system for cluster command management, providing a unified container for command flags and their dependencies.

## Key Components

- **FlagContainer**: Primary struct that aggregates all cluster command flags (Global, Create, List, Status, Delete, Cleanup) along with execution dependencies
- **GetGlobal()**: Interface method returning global flags for command execution
- **GetExecutor()**: Interface method providing access to the command executor
- **NewFlagContainer()**: Factory function creating initialized flag container with k3d defaults (3 nodes, Kubernetes v1.31.5-k3s1)
- **SyncGlobalFlags()**: Synchronizes global configuration across all command-specific flag structures
- **Reset()**: Testing utility that resets all flags to zero values

## Usage Example

```go
// Create a new flag container with defaults
container := NewFlagContainer()

// Access specific command flags
container.Create.NodeCount = 5
container.Create.ClusterType = "k3d"

// Sync global settings across all commands
container.Global.Verbose = true
container.SyncGlobalFlags()

// Use in command execution
executor := container.GetExecutor()
globalFlags := container.GetGlobal()

// Reset for testing
container.Reset()
```

The FlagContainer serves as the central configuration hub for all cluster operations, ensuring consistent flag management and providing a clean interface for command execution dependencies.
