Test file for wizard step components that validates the creation and basic functionality of interactive UI prompts for cluster configuration.

## Key Components

- **TestWizardSteps()** - Tests successful instantiation of WizardSteps
- **TestWizardSteps_PromptClusterType()** - Validates cluster type prompt method exists
- **TestWizardSteps_PromptK8sVersion()** - Validates Kubernetes version prompt method exists  
- **TestWizardSteps_ConfirmConfiguration()** - Tests configuration confirmation with valid cluster config

## Usage Example

```go
// Run specific test
go test -run TestWizardSteps_ConfirmConfiguration ./internal/ui

// Run all wizard step tests
go test ./internal/ui -v

// Test with coverage
go test -cover ./internal/ui
```

The tests focus on structural validation rather than interactive behavior, ensuring methods exist and don't panic with valid inputs. The `ConfirmConfiguration` test uses a sample `ClusterConfig` with K3d cluster type to verify the method handles valid configuration data without errors.
