Test file for the WizardSteps UI component, validating the initialization and method existence of cluster configuration wizard functionality.

## Key Components

- **TestWizardSteps**: Validates `NewWizardSteps()` constructor returns a non-nil instance
- **TestWizardSteps_PromptClusterType**: Ensures cluster type prompting method exists
- **TestWizardSteps_PromptK8sVersion**: Verifies Kubernetes version prompting method exists  
- **TestWizardSteps_ConfirmConfiguration**: Tests configuration confirmation method with valid cluster configuration

## Usage Example

```go
// Run the test suite
go test ./internal/ui/wizard_steps_test.go

// Test individual components
func TestExample(t *testing.T) {
    steps := NewWizardSteps()
    
    // Verify wizard initialization
    assert.NotNil(t, steps)
    
    // Test with valid cluster config
    config := models.ClusterConfig{
        Name:       "test-cluster",
        Type:       models.ClusterTypeK3d,
        NodeCount:  3,
        K8sVersion: "latest",
    }
    
    // Ensure no panics during confirmation
    assert.NotPanics(t, func() {
        steps.ConfirmConfiguration(config)
    })
}
```

The tests focus on structural validation rather than interactive behavior testing, ensuring the wizard methods exist and handle valid configurations without errors.
