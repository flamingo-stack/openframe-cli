A collection of interactive wizard step implementations for gathering cluster configuration input from users through terminal prompts.

## Key Components

- **WizardSteps**: Main struct containing all wizard step implementations
- **PromptClusterName**: Interactive cluster name input with validation
- **PromptClusterType**: Cluster type selection (k3d/GKE) with visual menu
- **PromptNodeCount**: Worker node count input with range validation
- **PromptK8sVersion**: Kubernetes version selection from predefined options
- **ConfirmConfiguration**: Configuration summary display and confirmation prompt
- **renderConfigurationTable**: Helper for displaying configuration in table format

## Usage Example

```go
wizard := NewWizardSteps()

// Gather cluster configuration
name, err := wizard.PromptClusterName("my-cluster")
if err != nil {
    return err
}

clusterType, err := wizard.PromptClusterType()
if err != nil {
    return err
}

nodeCount, err := wizard.PromptNodeCount(3)
if err != nil {
    return err
}

k8sVersion, err := wizard.PromptK8sVersion()
if err != nil {
    return err
}

// Build and confirm configuration
config := models.ClusterConfig{
    Name:       name,
    Type:       clusterType,
    NodeCount:  nodeCount,
    K8sVersion: k8sVersion,
}

confirmed, err := wizard.ConfirmConfiguration(config)
if err != nil || !confirmed {
    return err
}
```
