This file implements individual wizard step components for an interactive cluster creation wizard, providing user prompts for cluster configuration parameters.

## Key Components

- **WizardSteps**: Main struct containing wizard step methods
- **NewWizardSteps()**: Constructor function
- **PromptClusterName()**: Prompts for cluster name with validation
- **PromptClusterType()**: Interactive selection for cluster type (k3d/GKE)
- **PromptNodeCount()**: Prompts for number of worker nodes (1-10 range)
- **PromptK8sVersion()**: Selection prompt for Kubernetes versions
- **ConfirmConfiguration()**: Displays config summary and confirmation prompt
- **renderConfigurationTable()**: Helper to render configuration table with pterm

## Usage Example

```go
wizard := NewWizardSteps()

// Get cluster name
name, err := wizard.PromptClusterName("my-cluster")
if err != nil {
    return err
}

// Get cluster type
clusterType, err := wizard.PromptClusterType()
if err != nil {
    return err
}

// Get node count
nodeCount, err := wizard.PromptNodeCount(3)
if err != nil {
    return err
}

// Get Kubernetes version
k8sVersion, err := wizard.PromptK8sVersion()
if err != nil {
    return err
}

// Show configuration and confirm
config := models.ClusterConfig{
    Name: name,
    Type: clusterType,
    NodeCount: nodeCount,
    K8sVersion: k8sVersion,
}

confirmed, err := wizard.ConfirmConfiguration(config)
```
