Implements interactive wizard steps for cluster creation with user prompts and configuration validation.

## Key Components

### WizardSteps
Main struct containing wizard step implementations for cluster setup workflow.

### Core Methods
- `PromptClusterName()` - Prompts for cluster name with validation rules
- `PromptClusterType()` - Interactive selection between k3d and GKE cluster types  
- `PromptNodeCount()` - Prompts for number of worker nodes (1-10 range)
- `PromptK8sVersion()` - Selection menu for Kubernetes version options
- `ConfirmConfiguration()` - Shows configuration summary and confirmation prompt
- `renderConfigurationTable()` - Renders configuration table with pterm styling

### Features
- Input validation using shared UI validators
- Consistent prompt styling with promptui templates
- Fallback rendering for configuration display
- Integration with cluster models for type validation

## Usage Example

```go
steps := NewWizardSteps()

// Prompt for cluster configuration
name, err := steps.PromptClusterName("my-cluster")
if err != nil {
    return err
}

clusterType, err := steps.PromptClusterType()
if err != nil {
    return err
}

nodeCount, err := steps.PromptNodeCount(3)
if err != nil {
    return err
}

version, err := steps.PromptK8sVersion()
if err != nil {
    return err
}

// Create and confirm configuration
config := models.ClusterConfig{
    Name: name,
    Type: clusterType,
    NodeCount: nodeCount,
    K8sVersion: version,
}

confirmed, err := steps.ConfirmConfiguration(config)
if err != nil {
    return err
}
```
