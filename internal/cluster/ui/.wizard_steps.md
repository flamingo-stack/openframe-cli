This file provides wizard step implementations for interactive cluster configuration prompts in a CLI application.

## Key Components

- **WizardSteps**: Main struct containing wizard step methods
- **NewWizardSteps()**: Factory function to create a new wizard steps handler
- **PromptClusterName()**: Prompts for cluster name with validation
- **PromptClusterType()**: Interactive selection for cluster type (k3d/GKE)
- **PromptNodeCount()**: Prompts for number of worker nodes (1-10 range)
- **PromptK8sVersion()**: Selection prompt for Kubernetes version
- **ConfirmConfiguration()**: Displays configuration summary and confirmation prompt
- **renderConfigurationTable()**: Helper to render configuration tables with pterm

## Usage Example

```go
// Create wizard steps handler
steps := ui.NewWizardSteps()

// Prompt for cluster configuration
name, err := steps.PromptClusterName("my-cluster")
if err != nil {
    return err
}

clusterType, err := steps.PromptClusterType()
if err != nil {
    return err
}

nodeCount, err := steps.PromptNodeCount(3)
if err != nil {
    return err
}

// Create config and confirm
config := models.ClusterConfig{
    Name:       name,
    Type:       clusterType,
    NodeCount:  nodeCount,
    K8sVersion: "latest",
}

confirmed, err := steps.ConfirmConfiguration(config)
if err != nil || !confirmed {
    return err
}
```
