This file provides interactive wizard step implementations for cluster configuration setup, handling user prompts for cluster parameters like name, type, node count, and Kubernetes version.

## Key Components

- **WizardSteps**: Main struct containing wizard step methods
- **NewWizardSteps()**: Constructor function
- **PromptClusterName()**: Prompts for cluster name with validation
- **PromptClusterType()**: Presents cluster type selection (k3d/GKE)
- **PromptNodeCount()**: Prompts for worker node count (1-10)
- **PromptK8sVersion()**: Offers Kubernetes version selection
- **ConfirmConfiguration()**: Shows config summary and confirmation
- **renderConfigurationTable()**: Helper for styled table rendering

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/ui"
)

func main() {
    wizard := ui.NewWizardSteps()
    
    // Prompt for cluster configuration
    name, err := wizard.PromptClusterName("my-cluster")
    if err != nil {
        return
    }
    
    clusterType, err := wizard.PromptClusterType()
    if err != nil {
        return
    }
    
    nodeCount, err := wizard.PromptNodeCount(3)
    if err != nil {
        return
    }
    
    k8sVersion, err := wizard.PromptK8sVersion()
    if err != nil {
        return
    }
    
    config := models.ClusterConfig{
        Name:       name,
        Type:       clusterType,
        NodeCount:  nodeCount,
        K8sVersion: k8sVersion,
    }
    
    confirmed, err := wizard.ConfirmConfiguration(config)
    if err != nil {
        return
    }
    
    if confirmed {
        // Proceed with cluster creation
    }
}
```
