A wizard step implementation for interactive cluster configuration with validation and user prompts.

## Key Components

- **WizardSteps**: Main struct containing step-by-step wizard methods
- **PromptClusterName()**: Collects cluster name with validation
- **PromptClusterType()**: Presents cluster type selection (k3d/GKE)
- **PromptNodeCount()**: Validates worker node count (1-10 range)
- **PromptK8sVersion()**: Kubernetes version selection menu
- **ConfirmConfiguration()**: Displays summary table and confirmation prompt

## Usage Example

```go
// Initialize wizard steps
wizard := NewWizardSteps()

// Collect cluster configuration
name, err := wizard.PromptClusterName("my-cluster")
if err != nil {
    return err
}

clusterType, err := wizard.PromptClusterType()
if err != nil {
    return err
}

nodeCount, err := wizard.PromptNodeCount(3)
if err != nil {
    return err
}

// Create config and confirm
config := models.ClusterConfig{
    Name:       name,
    Type:       clusterType,
    NodeCount:  nodeCount,
}

confirmed, err := wizard.ConfirmConfiguration(config)
if err != nil || !confirmed {
    return err
}
```

The wizard provides a complete interactive flow with input validation, styled prompts using promptui, and a formatted configuration summary table.
