This file implements individual wizard step functions for interactive cluster configuration prompts in the OpenFrame CLI.

## Key Components

- **WizardSteps** - Main struct containing wizard step implementations
- **PromptClusterName()** - Prompts for cluster name with validation
- **PromptClusterType()** - Interactive selection for cluster type (k3d/GKE)
- **PromptNodeCount()** - Prompts for worker node count with range validation
- **PromptK8sVersion()** - Version selection from predefined Kubernetes versions
- **ConfirmConfiguration()** - Shows configuration summary and confirmation prompt
- **renderConfigurationTable()** - Helper for displaying configuration in table format

## Usage Example

```go
// Create wizard steps handler
steps := NewWizardSteps()

// Prompt for cluster configuration
name, err := steps.PromptClusterName("my-cluster")
if err != nil {
    return err
}

clusterType, err := steps.PromptClusterType()
if err != nil {
    return err
}

nodeCount, err := steps.PromptNodeCount(3)
if err != nil {
    return err
}

version, err := steps.PromptK8sVersion()
if err != nil {
    return err
}

// Create configuration and confirm
config := models.ClusterConfig{
    Name:       name,
    Type:       clusterType,
    NodeCount:  nodeCount,
    K8sVersion: version,
}

confirmed, err := steps.ConfirmConfiguration(config)
if err != nil {
    return err
}
```
