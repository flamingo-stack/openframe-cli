This file contains comprehensive unit tests for the UI prompt functions used in cluster management operations.

## Key Components

- **TestSelectClusterByName** - Tests the interactive cluster selection functionality, including empty cluster handling and name list creation
- **TestHandleClusterSelection** - Tests cluster selection logic with command-line arguments vs interactive prompts
- **TestConfirmClusterDeletion** - Tests deletion confirmation prompts with force flag handling
- **TestFormatClusterSuccessMessage** - Tests success message formatting for cluster operations
- **TestValidationLogic** - Tests input validation for cluster names
- **TestConstants** - Tests cluster type constants (K3d, GKE)

## Usage Example

```go
// Running the tests
func TestExample() {
    clusters := []ClusterInfo{
        {Name: "my-cluster", Status: "running"},
        {Name: "dev-cluster", Status: "stopped"},
    }
    
    // Test cluster selection with args
    result, err := HandleClusterSelection(clusters, []string{"my-cluster"}, "Select cluster")
    // Returns: "my-cluster", nil
    
    // Test confirmation with force flag
    confirmed, err := ConfirmClusterDeletion("my-cluster", true)
    // Returns: true, nil (bypasses interactive prompt)
    
    // Test message formatting
    message := FormatClusterSuccessMessage("my-cluster", "k3d", "running")
    // Returns formatted string with cluster details
}
```

The tests primarily focus on non-interactive logic and edge cases, as the interactive UI components require mocking to test effectively.
