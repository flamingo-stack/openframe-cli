This file contains comprehensive unit tests for cluster-related UI prompt functionality, covering interactive selection, validation, and formatting operations.

## Key Components

- **TestSelectClusterByName**: Tests cluster selection from a list, including edge cases like empty cluster arrays and invalid names
- **TestHandleClusterSelection**: Tests argument-based cluster selection with fallback to interactive prompts
- **TestConfirmClusterDeletion**: Tests deletion confirmation prompts with force flag handling
- **TestFormatClusterSuccessMessage**: Tests success message formatting for cluster operations
- **TestValidationLogic**: Tests input validation logic for cluster names
- **TestConstants**: Tests cluster type constant definitions

## Usage Example

```go
// Run all tests
go test -v ./ui

// Run specific test function
go test -v ./ui -run TestSelectClusterByName

// Test cluster selection handling
clusters := []ClusterInfo{
    {Name: "test-cluster", Status: "running"},
}
args := []string{"my-cluster"}
result, err := HandleClusterSelection(clusters, args, "Select cluster")

// Test message formatting
message := FormatClusterSuccessMessage("test-cluster", "k3d", "running")
```

The tests focus on validating non-interactive logic while acknowledging that actual UI prompts cannot be fully tested without mocking the promptui library.
