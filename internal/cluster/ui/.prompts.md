This file provides UI components and prompts specifically for cluster management operations, abstracting interactive user input handling from business logic.

## Key Components

**Type Aliases:**
- `ClusterType`, `ClusterInfo` - Re-exported from domain models for UI convenience
- `ClusterTypeK3d`, `ClusterTypeGKE` - Re-exported cluster type constants

**Functions:**
- `SelectClusterByName()` - Interactive cluster selection from a list
- `HandleClusterSelection()` - Handles cluster selection from args or interactive prompt
- `ConfirmClusterDeletion()` - Confirmation prompt for cluster deletion with force flag support
- `ShowClusterOperationCancelled()` - Displays cancellation message
- `FormatClusterSuccessMessage()` - Formats colored success messages with cluster details

**Internal Helpers:**
- `selectFromList()` - Generic list selection wrapper
- `confirmAction()` - Generic confirmation prompt wrapper

## Usage Example

```go
// Interactive cluster selection
clusters := []ClusterInfo{{Name: "dev-cluster"}, {Name: "prod-cluster"}}
selectedName, err := SelectClusterByName(clusters, "Select a cluster:")

// Handle selection from args or interactive prompt
clusterName, err := HandleClusterSelection(clusters, args, "Choose cluster:")

// Confirm deletion with optional force flag
confirmed, err := ConfirmClusterDeletion("my-cluster", false)

// Format success message
message := FormatClusterSuccessMessage("my-cluster", "k3d", "running")
pterm.Success.Println(message)
```
