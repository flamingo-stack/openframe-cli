This package provides user interface components and prompts specifically for cluster management operations in the CLI application.

## Key Components

**Type Aliases & Constants:**
- `ClusterType` and `ClusterInfo` - Re-exported from domain models for UI convenience
- `ClusterTypeK3d` and `ClusterTypeGKE` - Cluster type constants

**Interactive Selection Functions:**
- `SelectClusterByName()` - Interactive cluster selection from a list
- `HandleClusterSelection()` - Handles cluster selection from args or interactive prompt
- `ConfirmClusterDeletion()` - Confirmation prompt for cluster deletion with force flag support

**Display Functions:**
- `ShowClusterOperationCancelled()` - Standardized cancellation message
- `FormatClusterSuccessMessage()` - Formatted success message with cluster details

## Usage Example

```go
// Interactive cluster selection
clusters := []ClusterInfo{{Name: "dev-cluster"}, {Name: "prod-cluster"}}
selectedName, err := SelectClusterByName(clusters, "Select a cluster to manage:")
if err != nil {
    return err
}

// Handle selection from args or interactive prompt
clusterName, err := HandleClusterSelection(clusters, os.Args[1:], "Choose cluster:")

// Confirm deletion with force flag
confirmed, err := ConfirmClusterDeletion("my-cluster", false)
if confirmed {
    // Proceed with deletion
    fmt.Println(FormatClusterSuccessMessage("my-cluster", "k3d", "deleted"))
}
```
