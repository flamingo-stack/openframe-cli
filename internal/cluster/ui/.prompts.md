UI utility functions for cluster-related interactive prompts and user interactions. This file provides a clean separation between UI presentation logic and business logic by accepting pre-fetched data rather than depending on business interfaces directly.

## Key Components

**Type Aliases & Constants:**
- `ClusterType`, `ClusterInfo` - Re-exported from domain models for UI convenience
- `ClusterTypeK3d`, `ClusterTypeGKE` - Cluster type constants

**Selection Functions:**
- `SelectClusterByName()` - Interactive cluster selection from a list
- `HandleClusterSelection()` - Handles cluster selection from args or interactive prompt
- `selectFromList()` - Generic list selection utility

**Confirmation Functions:**
- `ConfirmClusterDeletion()` - Confirmation prompt for cluster deletion with force flag support
- `confirmAction()` - Generic confirmation prompt wrapper

**Display Functions:**
- `ShowClusterOperationCancelled()` - Standard cancellation message
- `FormatClusterSuccessMessage()` - Formatted success message with cluster details

## Usage Example

```go
// Interactive cluster selection
clusters := []ClusterInfo{{Name: "dev-cluster"}, {Name: "prod-cluster"}}
selected, err := SelectClusterByName(clusters, "Select cluster to delete:")
if err != nil {
    return err
}

// Handle selection from args or prompt
clusterName, err := HandleClusterSelection(clusters, os.Args[1:], "Choose cluster:")
if err != nil {
    return err
}

// Confirm deletion with force flag
confirmed, err := ConfirmClusterDeletion(clusterName, false)
if err != nil || !confirmed {
    ShowClusterOperationCancelled()
    return err
}

// Display success message
message := FormatClusterSuccessMessage("my-cluster", "k3d", "running")
pterm.Success.Println(message)
```
