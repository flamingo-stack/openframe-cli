Provides UI prompt functions and user interaction utilities specifically for cluster management operations in the OpenFrame CLI.

## Key Components

- **Type Aliases**: Re-exports `ClusterType` and `ClusterInfo` from domain models for UI convenience
- **Constants**: Re-exports cluster type constants (`ClusterTypeK3d`, `ClusterTypeGKE`) 
- **Selection Functions**:
  - `SelectClusterByName()` - Interactive cluster selection from a list
  - `HandleClusterSelection()` - Handles cluster selection from args or interactive prompt
- **Confirmation Functions**:
  - `ConfirmClusterDeletion()` - Confirms cluster deletion with force flag support
- **Display Functions**:
  - `ShowClusterOperationCancelled()` - Shows cancellation message
  - `FormatClusterSuccessMessage()` - Formats success messages with cluster info
- **Helper Functions**: Internal prompt utilities that delegate to shared UI components

## Usage Example

```go
// Interactive cluster selection
clusters := []ClusterInfo{{Name: "dev-cluster"}, {Name: "prod-cluster"}}
clusterName, err := SelectClusterByName(clusters, "Select cluster to deploy to:")

// Handle selection with args fallback
selectedCluster, err := HandleClusterSelection(clusters, os.Args[1:], "Choose cluster:")

// Confirm destructive operation
confirmed, err := ConfirmClusterDeletion("my-cluster", false)
if confirmed {
    // Proceed with deletion
    fmt.Println(FormatClusterSuccessMessage("my-cluster", "k3d", "deleted"))
}
```

The package follows dependency inversion principles by accepting pre-fetched data rather than business logic interfaces, keeping UI concerns separate from domain logic.
