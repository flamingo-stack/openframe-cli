Provides interactive user prompts and selection utilities specifically for cluster management operations, handling user input for cluster selection, confirmation dialogs, and status messaging.

## Key Components

**Type Aliases:**
- `ClusterType` - Re-exported from models package for UI consistency
- `ClusterInfo` - Re-exported cluster information structure

**Constants:**
- `ClusterTypeK3d`, `ClusterTypeGKE` - Re-exported cluster type constants

**Core Functions:**
- `SelectClusterByName()` - Interactive cluster selection from available clusters
- `HandleClusterSelection()` - Handles cluster selection from args or interactive prompt
- `ConfirmClusterDeletion()` - Confirmation dialog for cluster deletion operations
- `ShowClusterOperationCancelled()` - Displays cancellation message
- `FormatClusterSuccessMessage()` - Formats success messages with cluster details

**Helper Functions:**
- `selectFromList()` - Generic list selection wrapper
- `confirmAction()` - Confirmation prompt wrapper

## Usage Example

```go
// Interactive cluster selection
clusters := []ClusterInfo{
    {Name: "dev-cluster", Type: ClusterTypeK3d},
    {Name: "prod-cluster", Type: ClusterTypeGKE},
}

// Select cluster interactively
selectedName, err := SelectClusterByName(clusters, "Select cluster to manage:")
if err != nil {
    return err
}

// Handle selection from args or interactive
clusterName, err := HandleClusterSelection(clusters, args, "Choose cluster:")
if err != nil {
    return err
}

// Confirm deletion
confirmed, err := ConfirmClusterDeletion(clusterName, false)
if !confirmed {
    ShowClusterOperationCancelled()
    return nil
}
```
