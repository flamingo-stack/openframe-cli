Interactive user interface components for cluster management operations in the OpenFrame CLI.

## Key Components

**Type Aliases:**
- `ClusterType` - Re-exported from models for UI convenience
- `ClusterInfo` - Re-exported cluster information structure

**Selection Functions:**
- `SelectClusterByName()` - Interactive cluster selection from available clusters
- `HandleClusterSelection()` - Handles cluster selection from args or interactive prompt
- `selectFromList()` - Generic list selection wrapper

**Confirmation & Messaging:**
- `ConfirmClusterDeletion()` - Prompts for deletion confirmation with force flag support
- `ShowClusterOperationCancelled()` - Displays standardized cancellation message
- `FormatClusterSuccessMessage()` - Creates formatted success messages with cluster details

## Usage Example

```go
// Interactive cluster selection
clusters := []ClusterInfo{{Name: "dev-cluster"}, {Name: "prod-cluster"}}
selected, err := SelectClusterByName(clusters, "Select a cluster:")
if err != nil {
    return err
}

// Handle selection from args or prompt
clusterName, err := HandleClusterSelection(clusters, os.Args[1:], "Choose cluster:")
if err != nil {
    return err
}

// Confirm deletion
confirmed, err := ConfirmClusterDeletion("my-cluster", false)
if err != nil || !confirmed {
    ShowClusterOperationCancelled()
    return nil
}

// Display success message
message := FormatClusterSuccessMessage("my-cluster", "k3d", "running")
pterm.Success.Println(message)
```

The module follows a clean separation pattern where UI components receive pre-fetched data rather than depending on business logic interfaces, promoting better testability and maintainability.
