Provides interactive configuration wizards and handlers for Kubernetes cluster setup, offering both quick default and step-by-step interactive configuration modes.

## Key Components

- **`ClusterConfig`** - Configuration struct holding cluster parameters (name, type, node count, K8s version)
- **`ConfigWizard`** - Interactive step-by-step configuration wizard with defaults and validation
- **`ConfigurationHandler`** - Main handler orchestrating configuration flows and mode selection
- **`SelectCluster()`** - Interactive cluster selection from existing clusters
- **`GetClusterNameOrDefault()`** - Utility for resolving cluster names from arguments or defaults

## Usage Example

```go
// Create configuration handler
handler := ui.NewConfigurationHandler()

// Get cluster configuration (interactive or quick mode)
config, err := handler.GetClusterConfig("my-cluster")
if err != nil {
    log.Fatal(err)
}

// Or use the wizard directly
wizard := ui.NewConfigWizard()
wizard.SetDefaults("dev-cluster", models.ClusterTypeK3d, 5, "v1.28.0")
config, err := wizard.Run()

// Select from existing clusters
clusters := []models.ClusterInfo{...}
selected, err := ui.SelectCluster(clusters, "Choose a cluster")

// Get cluster name with fallback
name := ui.GetClusterNameOrDefault(os.Args[1:], "default-cluster")
```
