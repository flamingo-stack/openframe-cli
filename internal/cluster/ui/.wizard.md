Interactive configuration wizard for Kubernetes cluster creation with step-by-step prompting and validation. Provides both quick default setup and guided configuration flows.

## Key Components

- **ClusterConfig**: Configuration structure holding cluster parameters (name, type, node count, k8s version)
- **ConfigWizard**: Interactive wizard that guides users through cluster configuration steps
- **ConfigurationHandler**: High-level handler managing creation mode selection and configuration flows
- **SelectCluster**: Utility function for interactive cluster selection from available clusters
- **GetClusterNameOrDefault**: Helper function for resolving cluster names from command arguments

## Usage Example

```go
// Quick configuration with defaults
handler := NewConfigurationHandler()
config, err := handler.GetClusterConfig("my-cluster")
if err != nil {
    return err
}

// Interactive wizard with custom defaults
wizard := NewConfigWizard()
wizard.SetDefaults("custom-cluster", models.ClusterTypeK3d, 5, "v1.28.0")
clusterConfig, err := wizard.Run()
if err != nil {
    return err
}

// Cluster selection from existing clusters
selectedCluster, err := SelectCluster(availableClusters, "Select cluster to manage")
if err != nil {
    return err
}

// Get cluster name with fallback
clusterName := GetClusterNameOrDefault(os.Args[1:], "default-cluster")
```
