Provides interactive wizards and configuration handlers for Kubernetes cluster creation, offering both quick default setup and step-by-step guided configuration.

## Key Components

- **ClusterConfig**: Structure holding cluster configuration data (name, type, node count, K8s version)
- **ConfigWizard**: Interactive wizard for step-by-step cluster configuration
- **ConfigurationHandler**: Main handler managing configuration flows and creation modes
- **SelectCluster()**: Interactive cluster selection from available clusters
- **GetClusterNameOrDefault()**: Helper function for resolving cluster names from command arguments

## Usage Example

```go
// Create and run configuration wizard
wizard := NewConfigWizard()
wizard.SetDefaults("my-cluster", models.ClusterTypeK3d, 5, "v1.27.0")
config, err := wizard.Run()
if err != nil {
    log.Fatal(err)
}

// Use configuration handler for complete flow
handler := NewConfigurationHandler()
clusterConfig, err := handler.GetClusterConfig("production-cluster")
if err != nil {
    log.Fatal(err)
}

// Select from existing clusters
clusters := []models.ClusterInfo{{Name: "dev", Status: "running"}}
selected, err := SelectCluster(clusters, "Choose a cluster")
if err != nil {
    log.Fatal(err)
}

// Get cluster name with fallback
name := GetClusterNameOrDefault([]string{"custom"}, "default-cluster")
```

The wizard provides two modes: quick setup with defaults and interactive configuration with user prompts for each setting. All interactions use the promptui library for consistent terminal UI experience.
