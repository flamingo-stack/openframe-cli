Interactive configuration wizard for Kubernetes cluster creation with step-by-step prompts and mode selection. Provides both quick default setup and guided configuration flows.

## Key Components

### Core Types
- **`ClusterConfig`** - Holds cluster configuration parameters (name, type, node count, K8s version)
- **`ConfigWizard`** - Interactive wizard for step-by-step cluster configuration
- **`ConfigurationHandler`** - High-level handler for complete configuration flows

### Main Functions
- **`NewConfigWizard()`** - Creates wizard with default values (k3d, 3 nodes, latest K8s)
- **`Run()`** - Executes interactive wizard with 5 configuration steps
- **`SelectCluster()`** - Interactive cluster selection from available clusters
- **`GetClusterConfig()`** - Complete configuration flow with mode selection
- **`GetClusterNameOrDefault()`** - Helper for extracting cluster names from CLI args

### Configuration Modes
- **Quick Mode** - Uses sensible defaults for rapid cluster creation
- **Interactive Mode** - Step-by-step wizard for custom configuration

## Usage Example

```go
// Create and run interactive wizard
wizard := ui.NewConfigWizard()
wizard.SetDefaults("my-cluster", models.ClusterTypeK3d, 5, "v1.28.0")
config, err := wizard.Run()
if err != nil {
    log.Fatal(err)
}

// Use configuration handler for complete flow
handler := ui.NewConfigurationHandler()
clusterConfig, err := handler.GetClusterConfig("dev-cluster")
if err != nil {
    log.Fatal(err)
}

// Select from existing clusters
clusters := []models.ClusterInfo{{Name: "cluster-1", Status: "Running"}}
selected, err := ui.SelectCluster(clusters, "Choose cluster")
if err != nil {
    log.Fatal(err)
}

// Get cluster name with fallback
name := ui.GetClusterNameOrDefault(os.Args[1:], "default-cluster")
```
