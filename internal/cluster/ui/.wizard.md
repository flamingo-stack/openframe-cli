Provides interactive configuration wizards and handlers for Kubernetes cluster creation, offering both quick default setup and step-by-step guided configuration flows.

## Key Components

- **ClusterConfig**: Configuration struct holding cluster parameters (name, type, node count, K8s version)
- **ConfigWizard**: Interactive wizard for step-by-step cluster configuration
- **ConfigurationHandler**: High-level handler managing the complete configuration flow
- **SelectCluster()**: Interactive cluster selection from available clusters
- **GetClusterNameOrDefault()**: Utility function for resolving cluster names with fallback defaults

## Usage Example

```go
// Quick configuration flow
handler := NewConfigurationHandler()
config, err := handler.GetClusterConfig("my-cluster")
if err != nil {
    log.Fatal(err)
}

// Interactive wizard with custom defaults
wizard := NewConfigWizard()
wizard.SetDefaults("prod-cluster", models.ClusterTypeK3d, 5, "v1.28.0")
config, err := wizard.Run()

// Cluster selection from existing clusters
selectedCluster, err := SelectCluster(clusters, "Select cluster to modify")
if err != nil {
    log.Fatal(err)
}

// Name resolution with fallback
clusterName := GetClusterNameOrDefault(os.Args[1:], "default-cluster")
```

The wizard supports two modes: quick setup with sensible defaults (k3d cluster with 3 nodes) and interactive mode that prompts for each configuration option. All prompts use the promptui library for consistent CLI experience and include confirmation steps before proceeding.
