This file provides interactive UI components for configuring and selecting Kubernetes clusters through command-line wizards and prompts.

## Key Components

- **ClusterConfig**: Struct holding cluster configuration parameters (name, type, node count, K8s version)
- **ConfigWizard**: Interactive wizard for step-by-step cluster configuration with defaults
- **ConfigurationHandler**: Manages the complete configuration flow with quick vs. interactive modes
- **SelectCluster()**: Interactive cluster selection from a list of available clusters
- **GetClusterNameOrDefault()**: Helper function for extracting cluster names from command arguments

## Usage Example

```go
// Create and run interactive configuration wizard
wizard := NewConfigWizard()
wizard.SetDefaults("my-cluster", models.ClusterTypeK3d, 3, "v1.28.0")
config, err := wizard.Run()
if err != nil {
    log.Fatal(err)
}

// Handle complete configuration flow with mode selection
handler := NewConfigurationHandler()
clusterConfig, err := handler.GetClusterConfig("my-cluster")
if err != nil {
    log.Fatal(err)
}

// Interactive cluster selection
selectedCluster, err := SelectCluster(availableClusters, "Select a cluster")
if err != nil {
    log.Fatal(err)
}

// Extract cluster name from CLI args with fallback
clusterName := GetClusterNameOrDefault(os.Args[1:], "default-cluster")
```
