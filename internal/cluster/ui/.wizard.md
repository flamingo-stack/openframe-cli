Provides interactive wizards and configuration handlers for Kubernetes cluster creation and management. Includes both quick setup with defaults and step-by-step interactive configuration flows.

## Key Components

- **`ClusterConfig`** - Configuration struct holding cluster parameters (name, type, node count, k8s version)
- **`ConfigWizard`** - Interactive wizard for step-by-step cluster configuration
- **`ConfigurationHandler`** - Main handler that manages configuration flows (quick vs wizard modes)
- **`SelectCluster()`** - Interactive cluster selection from a list with formatted display
- **`GetClusterNameOrDefault()`** - Helper function for extracting cluster names from command arguments

## Usage Example

```go
// Quick configuration
handler := NewConfigurationHandler()
config, err := handler.GetClusterConfig("my-cluster")
if err != nil {
    log.Fatal(err)
}

// Interactive wizard
wizard := NewConfigWizard()
wizard.SetDefaults("dev-cluster", models.ClusterTypeK3d, 5, "v1.28.0")
config, err := wizard.Run()

// Cluster selection
clusters := []models.ClusterInfo{...}
selected, err := SelectCluster(clusters, "Choose a cluster")

// Extract cluster name from CLI args
clusterName := GetClusterNameOrDefault(os.Args[1:], "default-cluster")
```
