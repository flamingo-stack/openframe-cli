A utility package that provides consistent cluster selection logic for OpenFrame CLI commands, supporting both interactive and argument-based cluster selection with validation.

## Key Components

- **Selector**: Main struct that handles cluster selection operations with operation-specific messaging
- **NewSelector()**: Factory function to create a new selector for a given operation
- **SelectCluster()**: Handles single cluster selection with argument validation and interactive fallback
- **SelectMultipleClusters()**: Manages multi-cluster selection for batch operations
- **ValidateClusterExists()**: Checks if a named cluster exists in the available clusters
- **GetClusterByName()**: Retrieves cluster information by name with error handling
- **FilterClusters()**: Applies custom predicates to filter cluster lists

## Usage Example

```go
// Create a selector for a delete operation
selector := ui.NewSelector("delete")

// Get available clusters from your cluster service
clusters := []models.ClusterInfo{
    {Name: "prod-cluster"},
    {Name: "dev-cluster"},
}

// Single cluster selection (with args or interactive)
clusterName, err := selector.SelectCluster(clusters, args)
if err != nil {
    return fmt.Errorf("cluster selection failed: %w", err)
}

// Multiple cluster selection
selectedClusters, err := selector.SelectMultipleClusters(clusters, args)
if err != nil {
    return fmt.Errorf("multiple cluster selection failed: %w", err)
}

// Filter clusters by status
activeClusters := selector.FilterClusters(clusters, func(c models.ClusterInfo) bool {
    return c.Status == "active"
})

// Validate and get specific cluster
cluster, err := selector.GetClusterByName(clusters, "prod-cluster")
if err != nil {
    return fmt.Errorf("cluster lookup failed: %w", err)
}
```
