This file provides a centralized cluster selection utility that handles both interactive and argument-based cluster selection with consistent validation and error handling across CLI commands.

## Key Components

- **Selector**: Main struct that handles cluster selection logic for different operations
- **NewSelector**: Factory function to create a new selector for a specific operation
- **SelectCluster**: Selects a single cluster from available options (interactive or via argument)
- **SelectMultipleClusters**: Handles selection of multiple clusters with validation
- **ValidateClusterExists**: Checks if a cluster exists in the provided list
- **GetClusterByName**: Retrieves cluster information by name
- **FilterClusters**: Filters clusters based on a predicate function

## Usage Example

```go
// Create a selector for delete operation
selector := ui.NewSelector("delete")

// Single cluster selection
clusters := []models.ClusterInfo{
    {Name: "prod-cluster"},
    {Name: "staging-cluster"},
}

// Interactive selection (no args)
clusterName, err := selector.SelectCluster(clusters, []string{})

// Argument-based selection
clusterName, err := selector.SelectCluster(clusters, []string{"prod-cluster"})

// Multiple cluster selection
selectedClusters, err := selector.SelectMultipleClusters(clusters, []string{})

// Filter clusters by status
activeClusters := selector.FilterClusters(clusters, func(c models.ClusterInfo) bool {
    return c.Status == "active"
})
```
