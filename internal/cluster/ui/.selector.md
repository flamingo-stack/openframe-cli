A centralized UI component for handling cluster selection logic across different OpenFrame CLI commands, supporting both interactive and argument-based selection modes.

## Key Components

- **Selector**: Main struct that handles cluster selection with operation-specific context
- **SelectCluster**: Handles single cluster selection with argument validation and interactive fallback
- **SelectMultipleClusters**: Manages multi-cluster selection for batch operations
- **ValidateClusterExists**: Checks cluster existence in the provided list
- **GetClusterByName**: Retrieves specific cluster information by name
- **FilterClusters**: Applies custom filtering logic to cluster lists

## Usage Example

```go
// Single cluster selection
selector := NewSelector("deployment")
clusters := []models.ClusterInfo{
    {Name: "prod-cluster"},
    {Name: "dev-cluster"},
}

// With command line argument
clusterName, err := selector.SelectCluster(clusters, []string{"prod-cluster"})
if err != nil {
    return err
}

// Interactive selection (empty args)
clusterName, err = selector.SelectCluster(clusters, []string{})

// Multiple cluster selection
selector = NewSelector("scaling")
selectedClusters, err := selector.SelectMultipleClusters(clusters, []string{})

// Cluster validation and filtering
exists := selector.ValidateClusterExists(clusters, "prod-cluster")
prodClusters := selector.FilterClusters(clusters, func(c models.ClusterInfo) bool {
    return strings.Contains(c.Name, "prod")
})
```
