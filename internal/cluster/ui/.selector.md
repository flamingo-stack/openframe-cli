A utility for handling cluster selection logic with support for both command-line arguments and interactive user prompts across different OpenFrame CLI operations.

## Key Components

- **Selector**: Main struct that encapsulates cluster selection logic for a specific operation
- **NewSelector()**: Constructor that creates a selector for a given operation type
- **SelectCluster()**: Handles single cluster selection via arguments or interactive prompt
- **SelectMultipleClusters()**: Manages multi-cluster selection with validation
- **ValidateClusterExists()**: Checks if a cluster exists in the provided list
- **GetClusterByName()**: Retrieves cluster info by name
- **FilterClusters()**: Filters clusters based on a predicate function

## Usage Example

```go
// Create selector for a deploy operation
selector := ui.NewSelector("deploy")

// Single cluster selection
selectedCluster, err := selector.SelectCluster(clusters, args)
if err != nil {
    return fmt.Errorf("selection failed: %w", err)
}

// Multiple cluster selection
selectedClusters, err := selector.SelectMultipleClusters(clusters, args)
if err != nil {
    return fmt.Errorf("multi-selection failed: %w", err)
}

// Validate cluster exists
if !selector.ValidateClusterExists(clusters, "my-cluster") {
    return fmt.Errorf("cluster not found")
}

// Filter clusters by status
activeClusters := selector.FilterClusters(clusters, func(c models.ClusterInfo) bool {
    return c.Status == "active"
})
```
