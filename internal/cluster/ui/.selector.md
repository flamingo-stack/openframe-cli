This file provides a unified cluster selection interface that supports both command-line arguments and interactive prompts for various cluster operations.

## Key Components

- **Selector struct**: Main component that encapsulates cluster selection logic for a specific operation
- **NewSelector()**: Constructor that creates a new selector instance for a given operation
- **SelectCluster()**: Handles single cluster selection with argument validation and interactive fallback
- **SelectMultipleClusters()**: Manages multi-cluster selection with batch validation
- **ValidateClusterExists()**: Checks if a named cluster exists in the provided list
- **GetClusterByName()**: Retrieves cluster information by name
- **FilterClusters()**: Applies predicate-based filtering to cluster lists

## Usage Example

```go
// Create selector for deployment operation
selector := NewSelector("deployment")

// Single cluster selection
clusters := []models.ClusterInfo{{Name: "prod"}, {Name: "staging"}}
selectedCluster, err := selector.SelectCluster(clusters, []string{"prod"})
if err != nil {
    log.Fatal(err)
}

// Multiple cluster selection with interactive prompt
selectedClusters, err := selector.SelectMultipleClusters(clusters, nil)
if err != nil {
    log.Fatal(err)
}

// Filter active clusters
activeOnly := selector.FilterClusters(clusters, func(c models.ClusterInfo) bool {
    return c.Status == "active"
})
```
