This file provides a UI display service for cluster management operations, handling the presentation layer for cluster lifecycle and status information.

## Key Components

**Types:**
- `ClusterDisplayInfo` - Struct containing cluster information for display (name, type, status, nodes, etc.)
- `NodeDisplayInfo` - Node-specific display information (name, role, status)
- `ClusterConfigDisplay` - Configuration summary for cluster creation
- `DisplayService` - Main service handling all cluster UI display operations

**Key Methods:**
- `NewDisplayService()` - Creates a new display service instance
- `ShowClusterList()` - Renders a formatted table of clusters with status colors
- `ShowClusterStatus()` - Displays detailed cluster status including node information
- `ShowConfigurationSummary()` - Shows cluster configuration before creation
- Cluster lifecycle methods (`ShowClusterCreationStart/Success`, `ShowClusterDeletionStart/Success`, etc.)

## Usage Example

```go
// Create display service
displayService := NewDisplayService()

// Show cluster creation progress
displayService.ShowClusterCreationStart("my-cluster", "kind", os.Stdout)

// Display cluster list with formatted table
clusters := []ClusterDisplayInfo{
    {
        Name: "production", 
        Type: "eks", 
        Status: "running",
        NodeCount: 3,
        CreatedAt: time.Now(),
    },
}
displayService.ShowClusterList(clusters, os.Stdout)

// Show detailed cluster status
displayService.ShowClusterStatus(&clusters[0], os.Stdout)
```
