A service that handles cluster-related UI display operations, providing formatted output for cluster management commands like creation, deletion, status, and listing.

## Key Components

**Types:**
- `ClusterDisplayInfo` - Cluster information with name, type, status, nodes, and timestamps
- `NodeDisplayInfo` - Individual node details with name, role, and status
- `ClusterConfigDisplay` - Cluster configuration summary for display
- `DisplayService` - Main service for handling UI display operations

**Methods:**
- `ShowClusterCreationStart/Success` - Display cluster creation progress
- `ShowClusterDeletionStart/Success` - Display cluster deletion progress  
- `ShowClusterStartInProgress/Success` - Display cluster startup progress
- `ShowClusterList` - Render formatted table of clusters
- `ShowClusterStatus` - Display detailed cluster and node status
- `ShowConfigurationSummary` - Show cluster config with optional dry-run mode

## Usage Example

```go
displayService := NewDisplayService()

// Show cluster creation
displayService.ShowClusterCreationStart("my-cluster", "k3s", os.Stdout)

// Display cluster list
clusters := []ClusterDisplayInfo{
    {
        Name: "my-cluster",
        Type: "k3s", 
        Status: "running",
        NodeCount: 3,
        CreatedAt: time.Now(),
    },
}
displayService.ShowClusterList(clusters, os.Stdout)

// Show configuration summary
config := &ClusterConfigDisplay{
    Name: "my-cluster",
    Type: "k3s",
    K8sVersion: "v1.28.0",
    NodeCount: 3,
}
displayService.ShowConfigurationSummary(config, false, false, os.Stdout)
```
