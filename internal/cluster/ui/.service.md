This service provides UI display functionality for cluster management operations, handling formatted output and user interactions for cluster-related commands.

## Key Components

**Core Types:**
- `DisplayService` - Main service for handling cluster UI display operations
- `ClusterDisplayInfo` - Struct containing cluster information for display (name, type, status, nodes, etc.)
- `NodeDisplayInfo` - Struct for individual node display data
- `ClusterConfigDisplay` - Struct for cluster configuration summary display

**Display Methods:**
- `ShowClusterList()` - Renders cluster list in table format using pterm
- `ShowClusterStatus()` - Displays detailed cluster status with node information
- `ShowConfigurationSummary()` - Shows cluster configuration before creation
- Various lifecycle methods for creation, deletion, and start operations

## Usage Example

```go
package main

import (
    "os"
    "time"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/ui"
)

func main() {
    service := ui.NewDisplayService()
    
    // Display cluster creation start
    service.ShowClusterCreationStart("my-cluster", "k8s", os.Stdout)
    
    // Show cluster list
    clusters := []ui.ClusterDisplayInfo{{
        Name: "my-cluster",
        Type: "k8s", 
        Status: "running",
        NodeCount: 3,
        CreatedAt: time.Now(),
    }}
    service.ShowClusterList(clusters, os.Stdout)
    
    // Show configuration summary
    config := &ui.ClusterConfigDisplay{
        Name: "new-cluster",
        Type: "k8s",
        K8sVersion: "1.28.0",
        NodeCount: 2,
    }
    service.ShowConfigurationSummary(config, false, false, os.Stdout)
}
```
