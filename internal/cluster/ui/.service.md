A service that handles all cluster-related UI display operations, separating presentation concerns from business logic for the OpenFrame CLI's cluster management interface.

## Key Components

**Core Types:**
- `DisplayService` - Main service for UI display operations
- `ClusterDisplayInfo` - Represents cluster information with nodes for display
- `NodeDisplayInfo` - Represents individual node display data  
- `ClusterConfigDisplay` - Represents cluster configuration summary

**Key Methods:**
- `ShowClusterList()` - Displays clusters in a formatted table with status colors
- `ShowClusterStatus()` - Shows detailed cluster status including node information
- `ShowConfigurationSummary()` - Displays configuration before cluster creation
- `ShowClusterCreation/Deletion/StartXXX()` - Progress and success messages for operations

## Usage Example

```go
package main

import (
    "os"
    "time"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/ui"
)

func main() {
    service := ui.NewDisplayService()
    
    // Display cluster creation progress
    service.ShowClusterCreationStart("my-cluster", "kind", os.Stdout)
    
    // Show cluster list
    clusters := []ui.ClusterDisplayInfo{
        {
            Name: "production", 
            Type: "kind", 
            Status: "running",
            NodeCount: 3,
            CreatedAt: time.Now(),
            Nodes: []ui.NodeDisplayInfo{
                {Name: "control-plane", Role: "master", Status: "ready"},
            },
        },
    }
    service.ShowClusterList(clusters, os.Stdout)
    
    // Display configuration summary
    config := &ui.ClusterConfigDisplay{
        Name: "test-cluster",
        Type: "kind", 
        K8sVersion: "v1.28.0",
        NodeCount: 1,
    }
    service.ShowConfigurationSummary(config, false, false, os.Stdout)
}
```
