Provides a clean interface for displaying cluster management information and status updates in the OpenFrame CLI, handling formatted output for cluster operations like creation, deletion, and status display.

## Key Components

- **ClusterDisplayInfo** - Struct representing cluster information with name, type, status, node count, creation time, and associated nodes
- **NodeDisplayInfo** - Struct for individual node display data including name, role, and status
- **ClusterConfigDisplay** - Struct for cluster configuration summary display
- **DisplayService** - Main service struct that handles all cluster-related UI display operations

### Core Methods

- `ShowClusterList()` - Renders a formatted table of all clusters with colored status indicators
- `ShowClusterStatus()` - Displays detailed cluster information including node details
- `ShowConfigurationSummary()` - Shows cluster configuration before creation with dry-run support
- `ShowClusterCreationStart/Success()` - Progress messages for cluster creation workflow
- `ShowClusterDeletionStart/Success()` - Progress messages for cluster deletion workflow
- `ShowClusterStartInProgress/Success()` - Progress messages for cluster startup workflow

## Usage Example

```go
// Create display service
displaySvc := NewDisplayService()

// Show cluster list
clusters := []ClusterDisplayInfo{
    {
        Name: "dev-cluster",
        Type: "kind",
        Status: "running",
        NodeCount: 3,
        CreatedAt: time.Now(),
    },
}
displaySvc.ShowClusterList(clusters, os.Stdout)

// Show cluster creation progress
displaySvc.ShowClusterCreationStart("prod-cluster", "k3s", os.Stdout)
// ... perform creation ...
displaySvc.ShowClusterCreationSuccess("prod-cluster", os.Stdout)

// Display configuration summary
config := &ClusterConfigDisplay{
    Name: "test-cluster",
    Type: "kind",
    K8sVersion: "v1.28.0",
    NodeCount: 2,
}
displaySvc.ShowConfigurationSummary(config, false, false, os.Stdout)
```
