This service file provides UI display operations for cluster management, handling formatted output for cluster creation, status, and listing operations while maintaining separation between presentation and business logic.

## Key Components

- **DisplayService**: Main service struct that handles all cluster-related UI display operations
- **ClusterDisplayInfo**: Struct containing cluster information for display (name, type, status, nodes, creation time)
- **NodeDisplayInfo**: Struct representing individual node display data (name, role, status)
- **ClusterConfigDisplay**: Struct for cluster configuration display information
- **Display Methods**: Various methods for showing cluster operations like creation, deletion, status, and listing

## Usage Example

```go
// Create a new display service
displayService := NewDisplayService()

// Show cluster creation progress
displayService.ShowClusterCreationStart("my-cluster", "kind", os.Stdout)

// Display cluster list
clusters := []ClusterDisplayInfo{
    {
        Name:      "dev-cluster",
        Type:      "kind",
        Status:    "running",
        NodeCount: 3,
        CreatedAt: time.Now(),
        Nodes: []NodeDisplayInfo{
            {Name: "control-plane", Role: "master", Status: "ready"},
            {Name: "worker-1", Role: "worker", Status: "ready"},
        },
    },
}
displayService.ShowClusterList(clusters, os.Stdout)

// Show detailed cluster status
displayService.ShowClusterStatus(&clusters[0], os.Stdout)

// Display configuration summary
config := &ClusterConfigDisplay{
    Name:       "prod-cluster",
    Type:       "k3d",
    K8sVersion: "v1.28.0",
    NodeCount:  5,
}
displayService.ShowConfigurationSummary(config, false, false, os.Stdout)
```

The service uses pterm for enhanced table formatting with fallback to simple text formatting, and integrates with shared UI utilities for consistent status coloring across the application.
