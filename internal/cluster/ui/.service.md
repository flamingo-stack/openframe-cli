A UI service that provides display methods for cluster-related operations, handling formatted output for cluster creation, deletion, status display, and configuration summaries.

## Key Components

**Types:**
- `ClusterDisplayInfo` - Holds cluster information including name, type, status, node count, creation time, and node details
- `NodeDisplayInfo` - Represents individual node information with name, role, and status
- `ClusterConfigDisplay` - Contains cluster configuration data for display purposes
- `DisplayService` - Main service struct that provides all display methods

**Core Methods:**
- `NewDisplayService()` - Constructor for creating a new display service instance
- `ShowClusterCreationStart/Success()` - Display messages for cluster creation operations
- `ShowClusterDeletionStart/Success()` - Display messages for cluster deletion operations
- `ShowClusterStartInProgress/Success()` - Display messages for cluster start operations
- `ShowClusterList()` - Renders a formatted table of clusters with status colors
- `ShowClusterStatus()` - Shows detailed cluster information including nodes
- `ShowConfigurationSummary()` - Displays cluster configuration with optional dry-run messaging

## Usage Example

```go
package main

import (
    "os"
    "time"
)

func main() {
    displayService := NewDisplayService()
    
    // Show cluster creation
    displayService.ShowClusterCreationStart("my-cluster", "k3s", os.Stdout)
    
    // Display cluster list
    clusters := []ClusterDisplayInfo{
        {
            Name: "production",
            Type: "k3s", 
            Status: "running",
            NodeCount: 3,
            CreatedAt: time.Now(),
        },
    }
    displayService.ShowClusterList(clusters, os.Stdout)
    
    // Show configuration summary
    config := &ClusterConfigDisplay{
        Name: "new-cluster",
        Type: "k3s",
        K8sVersion: "v1.28.0",
        NodeCount: 3,
    }
    displayService.ShowConfigurationSummary(config, false, false, os.Stdout)
}
```
