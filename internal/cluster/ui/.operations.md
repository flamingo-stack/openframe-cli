This file provides a user-friendly interface layer for cluster operations, handling user interactions, confirmations, and status displays for cluster management commands.

## Key Components

- **OperationsUI**: Main struct providing cluster operation interfaces
- **SelectClusterForOperation**: Generic cluster selection with validation
- **SelectClusterForDelete**: Cluster selection with deletion confirmation
- **SelectClusterForCleanup**: Cluster selection with cleanup confirmation  
- **ShowOperationStart/Success/Error**: Status display methods
- **ShowConfigurationSummary**: Configuration preview display
- **ShowNoResourcesMessage**: Empty state handling

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/ui"
)

func main() {
    opsUI := ui.NewOperationsUI()
    
    clusters := []models.ClusterInfo{
        {Name: "dev-cluster"},
        {Name: "prod-cluster"},
    }
    
    // Select cluster for operation with validation
    clusterName, err := opsUI.SelectClusterForOperation(
        clusters, 
        []string{"dev-cluster"}, 
        "restart",
    )
    if err != nil {
        log.Fatal(err)
    }
    
    // Show operation progress
    opsUI.ShowOperationStart("restart", clusterName)
    
    // Perform operation...
    
    opsUI.ShowOperationSuccess("restart", clusterName)
}
```

The UI layer handles both command-line arguments and interactive selection, provides confirmation dialogs for destructive operations, and displays comprehensive status messages with troubleshooting tips.
