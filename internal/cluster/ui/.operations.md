This file provides a user interface layer for interactive cluster operations within the OpenFrame CLI tool.

## Key Components

### OperationsUI
Main UI service that provides user-friendly interfaces for cluster operations including interactive selection, confirmation prompts, and status messages.

### Core Methods
- `SelectClusterForOperation()` - Generic cluster selection for any operation
- `SelectClusterForDelete()` - Cluster selection with deletion confirmation 
- `SelectClusterForCleanup()` - Cluster selection with cleanup confirmation
- `ShowOperationStart/Success/Error()` - Status message displays
- `ShowConfigurationSummary()` - Configuration preview before cluster creation
- `ShowNoResourcesMessage()` - Helpful message when no clusters exist

## Usage Example

```go
// Create UI instance
ui := NewOperationsUI()

// Select cluster for general operation
clusterName, err := ui.SelectClusterForOperation(clusters, args, "start")
if err != nil {
    return err
}

// Select cluster for deletion with confirmation
clusterName, err := ui.SelectClusterForDelete(clusters, args, false)
if err != nil {
    return err
}

// Show operation progress
ui.ShowOperationStart("cleanup", clusterName)
// ... perform operation ...
ui.ShowOperationSuccess("cleanup", clusterName)

// Display configuration before creation
ui.ShowConfigurationSummary(config, false, false)
```

The UI handles both command-line argument input and interactive selection, validates cluster existence, and provides consistent user feedback with colored output using pterm for enhanced terminal experience.
