Provides specialized UI interfaces for cluster operations including selection, confirmation dialogs, and status reporting with enhanced user experience through colored terminal output.

## Key Components

- **OperationsUI**: Main struct providing user-friendly cluster operation interfaces
- **SelectClusterForOperation()**: Handles cluster selection for general operations with argument parsing and interactive selection
- **SelectClusterForDelete()**: Specialized cluster selection with deletion confirmation and force flag support
- **SelectClusterForCleanup()**: Cluster selection for cleanup operations with aggressive cleanup warnings
- **ShowOperationStart/Success/Error()**: Progress reporting methods with operation-specific messaging
- **ShowConfigurationSummary()**: Displays cluster configuration in a clean, readable format
- **confirmCleanup/confirmDeletion()**: Internal confirmation dialog helpers

## Usage Example

```go
// Create operations UI service
opsUI := NewOperationsUI()

// Select cluster for deletion with confirmation
clusterName, err := opsUI.SelectClusterForDelete(clusters, args, false)
if err != nil {
    return err
}
if clusterName == "" {
    return nil // User cancelled
}

// Show operation progress
opsUI.ShowOperationStart("delete", clusterName)

// Perform deletion...

// Show success with detailed summary
opsUI.ShowOperationSuccess("delete", clusterName)
```

The operations UI provides consistent user experience across cluster management commands with proper validation, confirmation dialogs, and detailed feedback including troubleshooting tips on errors.
