Provides user-friendly interfaces for cluster operations including selection, confirmation dialogs, and status messaging. Handles cluster selection for various operations like deletion and cleanup with appropriate validation and user confirmations.

## Key Components

- **OperationsUI**: Main service struct for cluster operation interfaces
- **NewOperationsUI()**: Constructor function
- **SelectClusterForOperation()**: Generic cluster selection for any operation
- **SelectClusterForDelete()**: Cluster selection with deletion confirmation
- **SelectClusterForCleanup()**: Cluster selection with cleanup confirmation  
- **ShowOperationStart/Success/Error()**: Status messaging methods
- **ShowConfigurationSummary()**: Displays cluster config details
- **ShowNoResourcesMessage()**: Handles empty cluster list scenarios

## Usage Example

```go
// Create operations UI service
opsUI := NewOperationsUI()

// Select cluster for a generic operation
clusterName, err := opsUI.SelectClusterForOperation(
    clusters, 
    []string{"my-cluster"}, 
    "restart",
)

// Select cluster for deletion with confirmation
clusterName, err := opsUI.SelectClusterForDelete(
    clusters, 
    []string{}, 
    false, // force flag
)

// Show operation progress
opsUI.ShowOperationStart("cleanup", "my-cluster")
// ... perform operation ...
opsUI.ShowOperationSuccess("cleanup", "my-cluster")

// Display configuration summary
opsUI.ShowConfigurationSummary(config, false, false)
```
