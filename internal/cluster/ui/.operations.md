Provides user-friendly interfaces and interactive prompts for cluster operation workflows. Handles cluster selection, confirmation dialogs, and operation status messaging with consistent UI patterns.

## Key Components

- **OperationsUI**: Main UI service for cluster operation interfaces
- **SelectClusterForOperation()**: Interactive cluster selection for general operations
- **SelectClusterForDelete()**: Specialized cluster selection with deletion confirmation
- **SelectClusterForCleanup()**: Cluster selection with cleanup confirmation (including aggressive mode)
- **ShowOperationStart/Success/Error()**: Standardized operation status messaging
- **ShowConfigurationSummary()**: Displays cluster configuration before creation
- **ShowNoResourcesMessage()**: Handles empty cluster list scenarios

## Usage Example

```go
// Create operations UI service
operationsUI := NewOperationsUI()

// Select cluster for general operation
clusterName, err := operationsUI.SelectClusterForOperation(
    clusters, 
    []string{"my-cluster"}, 
    "restart",
)
if err != nil {
    operationsUI.ShowOperationError("restart", clusterName, err)
    return
}

// Show operation progress
operationsUI.ShowOperationStart("restart", clusterName)

// Perform operation...

// Show success
operationsUI.ShowOperationSuccess("restart", clusterName)

// Delete cluster with confirmation
clusterName, err = operationsUI.SelectClusterForDelete(
    clusters,
    args,
    force, // Skip confirmation if true
)
if err != nil {
    return err
}
```
