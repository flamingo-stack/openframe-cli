Provides user-friendly interfaces for cluster operation workflows, including selection, confirmation, and status feedback. This UI service handles interactive cluster selection for various operations like delete, cleanup, and general operations.

## Key Components

- **OperationsUI**: Main service struct for cluster operation interfaces
- **SelectClusterForOperation**: Generic cluster selection for any operation
- **SelectClusterForDelete**: Cluster selection with deletion confirmation
- **SelectClusterForCleanup**: Cluster selection with cleanup confirmation
- **ShowOperationStart/Success/Error**: Status feedback methods
- **ShowConfigurationSummary**: Displays cluster config before creation

## Usage Example

```go
// Create operations UI service
opsUI := NewOperationsUI()

// Select cluster for general operation
clusterName, err := opsUI.SelectClusterForOperation(clusters, args, "restart")
if err != nil {
    return err
}

// Select cluster for deletion with confirmation
clusterName, err = opsUI.SelectClusterForDelete(clusters, args, false)
if err != nil {
    return err
}

// Show operation feedback
opsUI.ShowOperationStart("cleanup", clusterName)
// ... perform operation ...
opsUI.ShowOperationSuccess("cleanup", clusterName)

// Display configuration summary
opsUI.ShowConfigurationSummary(config, false, false)
```

The service handles both command-line argument input and interactive selection, validates cluster existence, and provides consistent user feedback throughout cluster operations.
