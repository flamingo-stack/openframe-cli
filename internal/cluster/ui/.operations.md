Provides user-friendly interfaces for cluster operations including selection, confirmation dialogs, and status messaging with interactive prompts and colorized output.

## Key Components

**OperationsUI Struct**
- `SelectClusterForOperation()` - Interactive cluster selection for general operations
- `SelectClusterForDelete()` - Cluster selection with deletion confirmation workflow
- `SelectClusterForCleanup()` - Cluster selection with cleanup confirmation (supports force mode)
- `ShowOperationStart()` - Display operation start messages with cluster context
- `ShowOperationSuccess()` - Show detailed success messages with operation summaries
- `ShowOperationError()` - Display formatted error messages with troubleshooting tips
- `ShowConfigurationSummary()` - Display cluster configuration before creation
- `ShowNoResourcesMessage()` - Handle empty cluster list scenarios

**Helper Functions**
- `confirmDeletion()` - Confirmation dialog for cluster deletion
- `confirmCleanup()` - Confirmation dialog for cleanup operations (with force mode warnings)

## Usage Example

```go
// Initialize operations UI
opsUI := NewOperationsUI()

// Select cluster for general operation
clusterName, err := opsUI.SelectClusterForOperation(
    clusters, 
    []string{"my-cluster"}, 
    "restart",
)

// Select cluster for deletion with confirmation
clusterName, err := opsUI.SelectClusterForDelete(
    clusters, 
    []string{}, // no args - will prompt for selection
    false, // not forced - will show confirmation
)

// Show operation progress
opsUI.ShowOperationStart("cleanup", "my-cluster")
// ... perform operation ...
opsUI.ShowOperationSuccess("cleanup", "my-cluster")

// Display configuration summary
opsUI.ShowConfigurationSummary(config, false, false)
```
