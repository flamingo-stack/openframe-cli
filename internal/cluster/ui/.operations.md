This file provides user interface handlers for cluster operations, managing interactive cluster selection, confirmations, and user feedback throughout the operation lifecycle.

## Key Components

### OperationsUI
- **`SelectClusterForOperation`** - Handles cluster selection for general operations with argument validation and interactive fallback
- **`SelectClusterForDelete`** - Specialized cluster selection for deletion with optional confirmation prompts
- **`SelectClusterForCleanup`** - Cluster selection for cleanup operations with mandatory confirmation (supports force mode)
- **`confirmCleanup`** - Prompts user confirmation for cleanup operations with different messaging for force mode
- **`confirmDeletion`** - Prompts user confirmation for cluster deletion
- **`ShowOperationStart`** - Displays operation start messages with context-specific formatting
- **`ShowOperationSuccess`** - Shows detailed success messages with operation-specific summaries and cleanup details
- **`ShowOperationError`** - Displays formatted error messages with troubleshooting tips
- **`ShowConfigurationSummary`** - Presents cluster configuration in a clean, readable format
- **`ShowNoResourcesMessage`** - Shows helpful message when no clusters are available

## Usage Example

```go
// Create operations UI
opsUI := NewOperationsUI()

// Select cluster for general operation
clusterName, err := opsUI.SelectClusterForOperation(clusters, args, "start")
if err != nil {
    return err
}

// Select cluster for deletion with confirmation
clusterName, err := opsUI.SelectClusterForDelete(clusters, args, force)
if err != nil {
    return err
}

// Show operation progress
opsUI.ShowOperationStart("delete", clusterName)
// ... perform operation ...
opsUI.ShowOperationSuccess("delete", clusterName)

// Display configuration summary
opsUI.ShowConfigurationSummary(config, dryRun, skipWizard)
```

The UI provides consistent user experience across cluster operations with proper validation, interactive selection fallbacks, and rich feedback including troubleshooting guidance for errors.
