Provides user-friendly interfaces for cluster operations including selection, confirmation dialogs, and status messaging.

## Key Components

- **OperationsUI**: Main struct providing cluster operation UI methods
- **SelectClusterForOperation**: Interactive cluster selection for general operations
- **SelectClusterForDelete**: Cluster selection with deletion confirmation
- **SelectClusterForCleanup**: Cluster selection with cleanup confirmation
- **ShowOperationStart/Success/Error**: Status messaging for operations
- **ShowConfigurationSummary**: Display cluster configuration details
- **ShowNoResourcesMessage**: Handle empty cluster list scenarios

## Usage Example

```go
// Create operations UI
opUI := NewOperationsUI()

// Select cluster for operation
clusterName, err := opUI.SelectClusterForOperation(
    clusters,
    []string{"my-cluster"}, // optional args
    "restart",
)
if err != nil {
    log.Fatal(err)
}

// Show operation progress
opUI.ShowOperationStart("restart", clusterName)

// Perform operation...

// Show success
opUI.ShowOperationSuccess("restart", clusterName)

// Delete with confirmation
clusterName, err = opUI.SelectClusterForDelete(
    clusters,
    []string{},
    false, // force flag
)
```

The UI provides consistent, interactive interfaces for cluster operations with confirmation dialogs, validation, and friendly status messages using the pterm library for enhanced terminal output.
