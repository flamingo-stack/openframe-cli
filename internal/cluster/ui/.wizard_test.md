Test file for the configuration wizard UI component that validates cluster creation and selection functionality.

## Key Components

- **TestNewConfigWizard** - Tests wizard initialization with default values
- **TestClusterConfig** - Validates cluster configuration structure and different cluster types (K3d, GKE)
- **TestFormatClusterOption** - Tests cluster option display formatting for UI selection
- **TestGetClusterNameOrDefault** - Tests cluster name resolution logic with fallback handling
- **TestSelectCluster** - Tests cluster selection functionality and error handling
- **TestWizardValidation** - Tests input validation for cluster names and node counts
- **TestConfigWizardState** - Tests wizard state management and isolation

## Usage Example

```go
// Test wizard creation
wizard := NewConfigWizard()
assert.Equal(t, "openframe-dev", wizard.config.Name)
assert.Equal(t, ClusterTypeK3d, wizard.config.Type)

// Test cluster name resolution
clusterName := GetClusterNameOrDefault([]string{"my-cluster"}, "default")
assert.Equal(t, "my-cluster", clusterName)

// Test cluster formatting
clusterInfo := ClusterInfo{Name: "test", Status: "running"}
formatted := formatClusterOption(clusterInfo)
assert.Contains(t, formatted, "test - running")
```
