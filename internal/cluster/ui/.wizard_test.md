Test file for the configuration wizard UI component, containing comprehensive test coverage for cluster configuration creation, validation, and user interaction helpers.

## Key Components

- **TestNewConfigWizard** - Tests wizard initialization with default values (name: "openframe-dev", type: K3d, nodes: 3)
- **TestClusterConfig** - Validates ClusterConfig struct creation with different cluster types (K3d, GKE)
- **TestFormatClusterOption** - Tests cluster information formatting for display (name + status with separator)
- **TestGetClusterNameOrDefault** - Tests command-line argument parsing with fallback to default names
- **TestSelectCluster** - Tests cluster selection logic and error handling for empty cluster lists
- **TestWizardValidation** - Mock validation tests for cluster names and node counts without UI interaction
- **TestConfigWizardState** - Tests wizard state management and instance independence

## Usage Example

```go
// Run specific test suite
go test -run TestNewConfigWizard

// Test cluster name validation
func validateClusterName(input string) error {
    if len(strings.TrimSpace(input)) < 1 {
        return errors.New("cluster name cannot be empty")
    }
    return nil
}

// Test cluster selection with mock data
clusters := []ClusterInfo{
    {Name: "prod-cluster", Status: "running"},
    {Name: "dev-cluster", Status: "stopped"},
}
_, err := SelectCluster(clusters, "Choose cluster:")
```
