A utility module providing display and formatting functions for the OpenFrame CLI user interface, with most functions acting as deprecated wrappers around shared UI components.

## Key Components

- **Deprecated wrapper functions**: `GetStatusColor()`, `RenderTableWithFallback()`, `ShowSuccessBox()`, `FormatAge()` - legacy functions that delegate to `sharedUI` package
- **Table rendering**: `RenderOverviewTable()`, `RenderNodeTable()` - specialized table renderers for cluster data
- **Post-creation guidance**: `ShowClusterCreationNextSteps()` - displays formatted next steps after cluster creation

## Usage Example

```go
package main

import (
    "time"
    "github.com/pterm/pterm"
    "your-project/ui"
)

func main() {
    // Render cluster overview
    overviewData := pterm.TableData{
        {"Cluster Name", "my-cluster"},
        {"Status", "Running"},
        {"Node Count", "3"},
    }
    ui.RenderOverviewTable(overviewData)

    // Show next steps after cluster creation
    ui.ShowClusterCreationNextSteps("my-cluster")

    // Format time duration (deprecated - use sharedUI instead)
    age := ui.FormatAge(time.Now().Add(-2 * time.Hour))
    fmt.Printf("Cluster age: %s\n", age)
}
```

Note: Most functions are deprecated and should be replaced with their `sharedUI` equivalents for new development.
