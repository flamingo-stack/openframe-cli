This file provides UI display utilities for the OpenFrame CLI, including table rendering, status formatting, and user guidance messages with graceful fallbacks for different terminal environments.

## Key Components

- **Deprecated Wrapper Functions**: `GetStatusColor`, `RenderTableWithFallback`, `ShowSuccessBox`, `FormatAge` - Legacy functions that delegate to shared UI components
- **Table Rendering**: `RenderOverviewTable`, `RenderNodeTable` - Specialized table renderers for cluster and node information
- **User Guidance**: `ShowClusterCreationNextSteps` - Displays formatted next steps after cluster creation with fallback handling

## Usage Example

```go
package main

import (
    "time"
    "github.com/flamingo-stack/openframe-cli/internal/ui"
    "github.com/pterm/pterm"
)

func main() {
    // Display cluster overview
    overviewData := pterm.TableData{
        {"Cluster Name", "my-cluster"},
        {"Status", "Running"},
        {"Nodes", "3"},
    }
    ui.RenderOverviewTable(overviewData)

    // Show next steps after cluster creation
    ui.ShowClusterCreationNextSteps("my-cluster")

    // Render node information
    nodeData := pterm.TableData{
        {"NAME", "STATUS", "ROLES", "AGE"},
        {"node-1", "Ready", "control-plane", "5m"},
        {"node-2", "Ready", "worker", "3m"},
    }
    ui.RenderNodeTable(nodeData)
}
```

The file includes robust error handling with fallback rendering when terminal table display fails, ensuring consistent user experience across different environments.
