This package provides UI display utilities for the OpenFrame CLI, serving as a bridge to shared UI components with some cluster-specific display functions.

## Key Components

- **GetStatusColor** - Returns color functions for status display (deprecated, delegates to sharedUI)
- **RenderTableWithFallback** - Renders tables with fallback to simple output (deprecated, delegates to sharedUI)
- **RenderOverviewTable** - Renders cluster overview information using key-value table format
- **RenderNodeTable** - Renders node information tables
- **ShowSuccessBox** - Displays formatted success messages (deprecated, delegates to sharedUI)
- **FormatAge** - Formats time durations into human-readable age strings (deprecated, delegates to sharedUI)
- **ShowClusterCreationNextSteps** - Displays next steps after successful cluster creation

## Usage Example

```go
package main

import (
    "time"
    "github.com/pterm/pterm"
    "your-project/ui"
)

func main() {
    // Display cluster overview
    overviewData := pterm.TableData{
        {"Cluster Name", "my-cluster"},
        {"Status", "Running"},
        {"Nodes", "3"},
    }
    ui.RenderOverviewTable(overviewData)

    // Show next steps after cluster creation
    ui.ShowClusterCreationNextSteps("my-cluster")

    // Format time age (deprecated)
    age := ui.FormatAge(time.Now().Add(-2 * time.Hour))
    fmt.Printf("Cluster age: %s\n", age)
}
```

Most functions are deprecated wrappers around shared UI components, with the package primarily serving cluster-specific display logic like post-creation guidance.
