This file provides UI display utilities for the OpenFrame CLI, primarily serving as a wrapper around shared UI components with some deprecated legacy functions.

## Key Components

- **Legacy Functions** (Deprecated): `GetStatusColor`, `RenderTableWithFallback`, `ShowSuccessBox`, `FormatAge` - wrapper functions that delegate to `sharedUI` package
- **Active Table Renderers**: `RenderOverviewTable`, `RenderNodeTable` - render cluster and node information tables
- **Next Steps Display**: `ShowClusterCreationNextSteps` - shows post-creation guidance with commands

## Usage Example

```go
import (
    "github.com/pterm/pterm"
    "your-project/ui"
)

// Render cluster overview
overviewData := pterm.TableData{
    {"Cluster Name", "my-cluster"},
    {"Status", "Running"},
    {"Nodes", "3"},
}
ui.RenderOverviewTable(overviewData)

// Show next steps after cluster creation
ui.ShowClusterCreationNextSteps("my-cluster")

// Render node information
nodeData := pterm.TableData{
    {"Name", "Status", "Role"},
    {"node-1", "Ready", "master"},
    {"node-2", "Ready", "worker"},
}
ui.RenderNodeTable(nodeData)
```

The file is transitioning away from deprecated wrapper functions toward using the shared UI package directly, while maintaining cluster-specific display functionality.
