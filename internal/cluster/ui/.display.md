This file provides UI display utilities for the OpenFrame CLI, serving as a bridge to shared UI components while adding cluster-specific display functionality.

## Key Components

- **Deprecated wrapper functions**: `GetStatusColor`, `RenderTableWithFallback`, `ShowSuccessBox`, `FormatAge` - redirect to `sharedUI` equivalents
- **Table rendering functions**: `RenderOverviewTable`, `RenderNodeTable` - specialized table renderers for cluster data
- **Next steps display**: `ShowClusterCreationNextSteps` - shows post-creation guidance with fallback formatting

## Usage Example

```go
// Render cluster overview data
overviewData := pterm.TableData{
    {"Cluster Name", "my-cluster"},
    {"Status", "Running"},
    {"Nodes", "3"},
}
err := ui.RenderOverviewTable(overviewData)

// Show next steps after cluster creation
ui.ShowClusterCreationNextSteps("my-cluster")

// Render node information (delegated to shared UI)
nodeData := pterm.TableData{
    {"Name", "Status", "Age"},
    {"node-1", "Ready", "2d"},
    {"node-2", "Ready", "2d"},
}
err = ui.RenderNodeTable(nodeData)
```

The file primarily acts as a compatibility layer, with most functions deprecated in favor of shared UI components, while providing cluster-specific display logic like the next steps guidance.
