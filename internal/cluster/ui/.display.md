This file provides UI display utilities and formatting functions for the openframe CLI, primarily serving as a wrapper around shared UI components with some deprecated legacy functions.

## Key Components

- **GetStatusColor()** - Returns color function for status strings (deprecated, delegates to shared UI)
- **RenderTableWithFallback()** - Renders tables with fallback to simple output (deprecated)
- **RenderOverviewTable()** - Renders cluster overview information as key-value table
- **RenderNodeTable()** - Renders node information in table format
- **ShowSuccessBox()** - Displays formatted success messages (deprecated)
- **FormatAge()** - Formats time durations to human-readable strings (deprecated)
- **ShowClusterCreationNextSteps()** - Displays post-creation guidance with command examples

## Usage Example

```go
package main

import (
    "time"
    "github.com/pterm/pterm"
    "your-project/ui"
)

func main() {
    // Display cluster overview
    overviewData := pterm.TableData{
        {"Name", "my-cluster"},
        {"Status", "Running"},
        {"Nodes", "3"},
    }
    ui.RenderOverviewTable(overviewData)
    
    // Show next steps after cluster creation
    ui.ShowClusterCreationNextSteps("my-cluster")
    
    // Format node data
    nodeData := pterm.TableData{
        {"NAME", "STATUS", "AGE"},
        {"node-1", "Ready", "2h30m"},
    }
    ui.RenderNodeTable(nodeData)
}
```

Note: Most functions are deprecated wrappers around shared UI components and should be replaced with their `sharedUI` equivalents in new code.
