Provides display and formatting utilities for the OpenFrame CLI, with most functions deprecated in favor of the shared UI package.

## Key Components

- **GetStatusColor** - Returns color function for status strings (deprecated)
- **RenderTableWithFallback** - Renders tables with simple output fallback (deprecated)
- **RenderOverviewTable** - Renders cluster overview as key-value table
- **RenderNodeTable** - Renders node information table
- **ShowSuccessBox** - Displays formatted success messages (deprecated)
- **FormatAge** - Formats time duration to human-readable age (deprecated)
- **ShowClusterCreationNextSteps** - Displays post-creation workflow guidance

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/ui"
    "github.com/pterm/pterm"
)

func main() {
    // Render cluster overview
    overviewData := pterm.TableData{
        {"Cluster Name", "my-cluster"},
        {"Status", "Running"},
        {"Nodes", "3"},
    }
    ui.RenderOverviewTable(overviewData)

    // Show next steps after cluster creation
    ui.ShowClusterCreationNextSteps("my-cluster")

    // Render node information
    nodeData := pterm.TableData{
        {"NAME", "STATUS", "ROLES", "AGE"},
        {"node-1", "Ready", "control-plane", "5m"},
        {"node-2", "Ready", "worker", "4m"},
    }
    ui.RenderNodeTable(nodeData)
}
```

Most functions are deprecated and redirect to the `sharedUI` package for centralized UI handling. The package focuses on cluster-specific display operations and post-creation user guidance.
