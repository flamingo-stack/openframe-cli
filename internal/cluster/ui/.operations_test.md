Test file for the OperationsUI component that validates cluster operation selection, UI display methods, and error handling for cluster management operations.

## Key Components

- **TestOperationsUI_SelectClusterForOperation**: Tests cluster selection logic with various input scenarios (provided args, empty names, no clusters available, whitespace handling)
- **TestNewOperationsUI**: Validates proper initialization of OperationsUI instances
- **TestOperationsUI_ShowOperationStart**: Tests operation start message display without panics
- **TestOperationsUI_ShowOperationSuccess**: Tests success message display for completed operations
- **TestOperationsUI_ShowOperationError**: Tests error message display with proper error handling
- **TestOperationsUI_ShowNoResourcesMessage**: Tests display of "no resources" messages

## Usage Example

```go
func TestClusterOperations(t *testing.T) {
    ui := NewOperationsUI()
    
    // Test cluster selection
    clusters := []models.ClusterInfo{
        {Name: "my-cluster", Type: models.ClusterTypeK3d},
    }
    args := []string{"my-cluster"}
    
    selected, err := ui.SelectClusterForOperation(clusters, args, "cleanup")
    if err != nil {
        t.Errorf("cluster selection failed: %v", err)
    }
    
    // Test UI display methods
    ui.ShowOperationStart("cleanup", selected)
    ui.ShowOperationSuccess("cleanup", selected)
}
```
