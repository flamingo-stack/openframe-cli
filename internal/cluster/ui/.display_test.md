This file contains comprehensive unit tests for UI display functions, verifying color coding, table rendering, and message display functionality.

## Key Components

- **TestGetStatusColor**: Tests status-based color mapping (green for running/ready, yellow for stopped/pending, red for error/failed, gray for unknown)
- **TestRenderTableWithFallback**: Tests general table rendering with various column configurations
- **TestRenderOverviewTable**: Tests cluster overview table rendering
- **TestRenderNodeTable**: Tests node information table rendering  
- **TestShowSuccessBox**: Tests success message display functionality
- **TestFormatAge**: Tests time duration formatting (days, hours, minutes, seconds)
- **TestShowClusterCreationNextSteps**: Tests post-creation guidance display
- **TestShowNoResourcesMessage**: Tests empty state message display

## Usage Example

```go
func TestExample(t *testing.T) {
    // Test status color mapping
    colorFunc := GetStatusColor("running")
    result := colorFunc("test")
    expected := pterm.Green("test")
    assert.Equal(t, expected, result)
    
    // Test table rendering
    data := pterm.TableData{
        {"Header1", "Header2"},
        {"Row1Col1", "Row1Col2"},
    }
    err := RenderTableWithFallback(data, true)
    assert.NoError(t, err)
    
    // Test age formatting
    pastTime := time.Now().Add(-2 * time.Hour)
    result := FormatAge(pastTime)
    assert.Equal(t, "2h", result)
}
```

The tests ensure robust handling of edge cases including empty data, missing columns, and case-insensitive status matching.
