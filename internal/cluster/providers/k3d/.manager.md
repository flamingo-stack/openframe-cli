Manages K3D cluster lifecycle operations including creation, deletion, and status monitoring with intelligent port allocation and configuration management.

## Key Components

- **K3dManager**: Core manager struct that implements cluster operations
- **ClusterManager**: Interface defining cluster management contract
- **NewK3dManager**: Constructor with default timeout settings
- **NewK3dManagerWithTimeout**: Constructor with custom timeout configuration

**Core Operations:**
- `CreateCluster`: Creates clusters using temporary config files with port allocation
- `DeleteCluster`: Removes existing clusters with validation
- `StartCluster`: Starts stopped clusters
- `ListClusters`: Returns cluster information with node status
- `GetClusterStatus`: Retrieves detailed status for specific clusters
- `DetectClusterType`: Identifies if a cluster is K3D-managed
- `GetKubeconfig`: Extracts kubeconfig for cluster access

**Configuration Management:**
- `validateClusterConfig`: Ensures cluster configuration is valid
- `createK3dConfigFile`: Generates temporary YAML config files
- `findAvailablePorts`: Intelligent port allocation avoiding conflicts
- `getUsedPortsByExistingClusters`: Tracks ports used by existing clusters

## Usage Example

```go
// Create manager with verbose output
executor := executor.NewRealExecutor()
manager := NewK3dManager(executor, true)

// Create a new cluster
config := models.ClusterConfig{
    Name:       "my-cluster",
    Type:       models.ClusterTypeK3d,
    NodeCount:  3,
    K8sVersion: "v1.31.5-k3s1",
}

ctx := context.Background()
if err := manager.CreateCluster(ctx, config); err != nil {
    log.Fatal(err)
}

// List all clusters
clusters, err := manager.ListClusters(ctx)
if err != nil {
    log.Fatal(err)
}

// Get cluster status
status, err := manager.GetClusterStatus(ctx, "my-cluster")
if err != nil {
    log.Fatal(err)
}

// Clean up
err = manager.DeleteCluster(ctx, "my-cluster", models.ClusterTypeK3d, false)
```
