This file implements a K3D cluster manager for creating and managing Kubernetes clusters using K3D (Kubernetes in Docker).

## Key Components

**Main Types:**
- `ClusterManager` - Interface for cluster management operations
- `K3dManager` - Core manager implementing K3D cluster operations
- Configuration constants for default ports, timeouts, and K3s image versions

**Primary Methods:**
- `NewK3dManager()` / `NewK3dManagerWithTimeout()` - Constructor functions
- `CreateCluster()` - Creates new K3D cluster from configuration
- `DeleteCluster()` / `StartCluster()` - Lifecycle management
- `ListClusters()` / `GetClusterStatus()` - Cluster discovery and status
- `GetKubeconfig()` - Retrieves cluster authentication config
- `DetectClusterType()` - Identifies if cluster is K3D-managed

**Helper Functions:**
- `findAvailablePorts()` - Intelligent port allocation avoiding conflicts
- `createK3dConfigFile()` - Generates K3D YAML configuration files
- `validateClusterConfig()` - Input validation

## Usage Example

```go
// Create manager with command executor
exec := executor.NewRealCommandExecutor()
manager := NewK3dManager(exec, true) // verbose mode

// Create cluster configuration
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
    NodeCount: 3,
    K8sVersion: "v1.31.5-k3s1",
}

// Create the cluster
ctx := context.Background()
err := manager.CreateCluster(ctx, config)

// List existing clusters
clusters, err := manager.ListClusters(ctx)

// Get kubeconfig
kubeconfig, err := manager.GetKubeconfig(ctx, "my-cluster", models.ClusterTypeK3d)
```
