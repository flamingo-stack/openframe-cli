K3D cluster manager that handles creation, deletion, and management of K3D Kubernetes clusters with extensive Windows/WSL support and cross-platform compatibility.

## Key Components

**ClusterManager Interface**
- `DetectClusterType()` - Identifies cluster type by name
- `ListClusters()` - Returns K3D clusters only  
- `ListAllClusters()` - Returns all available clusters

**K3dManager Struct**
- `CreateCluster()` - Creates new K3D cluster with config file approach
- `DeleteCluster()` - Removes cluster with fallback Docker cleanup
- `GetRestConfig()` - Returns Kubernetes REST config for existing cluster
- `StartCluster()` - Starts stopped cluster
- `ListClusters()` - Lists all K3D clusters with status

**Platform-Specific Features**
- Windows/WSL2 support with IP detection and path conversion
- Inotify limits configuration for file-watching applications
- Kubeconfig permission fixes and lock file cleanup
- Docker container force cleanup for failed deletions

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
)

// Create manager
exec := executor.NewCommandExecutor()
manager := NewK3dManager(exec, true) // verbose=true

// Create cluster
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
    K3d: &models.K3dConfig{
        Image: "rancher/k3s:v1.31.5-k3s1",
        Ports: []models.PortMapping{{Host: 8080, Container: 80}},
    },
}

ctx := context.Background()
restConfig, err := manager.CreateCluster(ctx, config)
if err != nil {
    return err
}

// List clusters
clusters, err := manager.ListClusters(ctx)
if err != nil {
    return err
}

// Delete cluster
err = manager.DeleteCluster(ctx, "my-cluster", models.ClusterTypeK3d, false)
```
