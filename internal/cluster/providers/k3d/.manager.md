This file implements a K3D cluster manager that provides comprehensive cluster lifecycle management operations for the K3D Kubernetes distribution.

## Key Components

- **K3dManager**: Main struct implementing cluster management operations with command execution and configuration handling
- **ClusterManager**: Interface defining core cluster management methods (DetectClusterType, ListClusters, ListAllClusters)
- **CreateCluster**: Creates new K3D clusters using dynamic configuration file generation with automatic port allocation
- **DeleteCluster/StartCluster**: Basic cluster lifecycle operations with validation
- **ListClusters/GetClusterStatus**: Cluster discovery and status reporting with JSON parsing
- **GetKubeconfig**: Retrieves cluster authentication configuration
- **Port Management**: Intelligent port allocation system to avoid conflicts between clusters

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
)

// Create manager
exec := executor.NewCommandExecutor()
manager := NewK3dManager(exec, true) // verbose mode

// Create cluster
config := models.ClusterConfig{
    Name:      "my-cluster",
    Type:      models.ClusterTypeK3d,
    NodeCount: 3,
    K8sVersion: "v1.31.5-k3s1",
}

ctx := context.Background()
err := manager.CreateCluster(ctx, config)

// List clusters
clusters, err := manager.ListClusters(ctx)

// Get kubeconfig
kubeconfig, err := manager.GetKubeconfig(ctx, "my-cluster", models.ClusterTypeK3d)
```
