This file provides a comprehensive K3D cluster management implementation for creating, managing, and operating Kubernetes clusters using K3D.

## Key Components

**Main Types:**
- `ClusterManager` - Interface for cluster management operations
- `K3dManager` - Concrete implementation for K3D cluster operations
- `k3dClusterInfo` - Internal structure for parsing K3D cluster data

**Constructor Functions:**
- `NewK3dManager()` - Creates manager with default timeout
- `NewK3dManagerWithTimeout()` - Creates manager with custom timeout

**Core Operations:**
- `CreateCluster()` - Creates new K3D clusters with dynamic port allocation
- `DeleteCluster()` - Removes K3D clusters
- `StartCluster()` - Starts stopped clusters
- `ListClusters()` - Lists all K3D clusters with status
- `GetClusterStatus()` - Gets detailed status for specific cluster
- `GetKubeconfig()` - Retrieves kubeconfig for cluster access

**Utility Functions:**
- `DetectClusterType()` - Identifies cluster type
- `findAvailablePorts()` - Intelligent port allocation to avoid conflicts
- `createK3dConfigFile()` - Generates K3D configuration files

## Usage Example

```go
// Create K3D manager
executor := executor.NewRealCommandExecutor()
manager := NewK3dManager(executor, true)

// Create a cluster
config := models.ClusterConfig{
    Name:      "my-cluster",
    Type:      models.ClusterTypeK3d,
    NodeCount: 3,
    K8sVersion: "v1.31.5-k3s1",
}

ctx := context.Background()
err := manager.CreateCluster(ctx, config)
if err != nil {
    log.Fatal(err)
}

// List all clusters
clusters, err := manager.ListClusters(ctx)
if err != nil {
    log.Fatal(err)
}

// Get kubeconfig
kubeconfig, err := manager.GetKubeconfig(ctx, "my-cluster", models.ClusterTypeK3d)
```
