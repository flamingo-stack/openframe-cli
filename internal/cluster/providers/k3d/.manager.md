Manages K3D (K3s in Docker) cluster operations including creation, deletion, lifecycle management, and configuration. Provides a complete interface for working with lightweight Kubernetes clusters in Docker containers.

## Key Components

- **K3dManager**: Main cluster management struct with command execution and configuration
- **ClusterManager Interface**: Defines contract for cluster detection and listing operations
- **Configuration Constants**: Default values for K3s image, ports, and timeouts
- **Port Management**: Dynamic port allocation to prevent conflicts between clusters
- **Config File Generation**: Creates temporary K3D configuration files for cluster creation

## Key Methods

- `CreateCluster()`: Creates new K3D cluster using configuration file approach
- `DeleteCluster()`: Removes K3D cluster and cleans up resources
- `StartCluster()`: Starts stopped K3D cluster
- `ListClusters()`: Returns all K3D clusters with status information
- `GetClusterStatus()`: Gets detailed status for specific cluster
- `DetectClusterType()`: Determines if cluster is K3D type
- `GetKubeconfig()`: Retrieves kubeconfig for cluster access

## Usage Example

```go
// Create K3D manager
executor := executor.NewRealExecutor()
manager := NewK3dManager(executor, true) // verbose mode

// Create cluster configuration
config := models.ClusterConfig{
    Name:      "my-cluster",
    Type:      models.ClusterTypeK3d,
    NodeCount: 3,
    K8sVersion: "v1.31.5-k3s1",
}

// Create the cluster
ctx := context.Background()
err := manager.CreateCluster(ctx, config)
if err != nil {
    log.Fatal(err)
}

// List all clusters
clusters, err := manager.ListClusters(ctx)
if err != nil {
    log.Fatal(err)
}

// Get kubeconfig
kubeconfig, err := manager.GetKubeconfig(ctx, "my-cluster", models.ClusterTypeK3d)
if err != nil {
    log.Fatal(err)
}
```
