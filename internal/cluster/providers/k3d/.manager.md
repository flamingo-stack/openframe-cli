Manages K3D Kubernetes cluster lifecycle operations including creation, deletion, and configuration through the k3d CLI tool. Provides intelligent port allocation and cluster status monitoring capabilities.

## Key Components

**Core Types:**
- `K3dManager` - Main manager struct for K3D operations
- `ClusterManager` interface - Defines cluster management operations
- `k3dClusterInfo` - Internal struct for parsing k3d JSON responses

**Primary Methods:**
- `CreateCluster()` - Creates new K3D clusters using YAML configuration files
- `DeleteCluster()` - Removes existing K3D clusters
- `StartCluster()` - Starts stopped clusters
- `ListClusters()` - Returns all K3D clusters with status information
- `GetKubeconfig()` - Retrieves kubeconfig for specific clusters
- `DetectClusterType()` - Identifies if a cluster is managed by K3D

**Configuration Management:**
- `createK3dConfigFile()` - Generates temporary YAML config files
- `findAvailablePorts()` - Intelligent port allocation to avoid conflicts
- `validateClusterConfig()` - Input validation for cluster configurations

## Usage Example

```go
// Create a new K3D manager with command executor
exec := executor.NewCommandExecutor()
manager := NewK3dManager(exec, true) // verbose mode enabled

// Create cluster configuration
config := models.ClusterConfig{
    Name:      "my-cluster",
    Type:      models.ClusterTypeK3d,
    NodeCount: 3,
    K8sVersion: "v1.31.5-k3s1",
}

// Create the cluster
ctx := context.Background()
if err := manager.CreateCluster(ctx, config); err != nil {
    log.Fatal("Failed to create cluster:", err)
}

// List all clusters
clusters, err := manager.ListClusters(ctx)
if err != nil {
    log.Fatal("Failed to list clusters:", err)
}

// Get kubeconfig for the cluster
kubeconfig, err := manager.GetKubeconfig(ctx, "my-cluster", models.ClusterTypeK3d)
if err != nil {
    log.Fatal("Failed to get kubeconfig:", err)
}
```

The manager automatically handles port allocation (starting from 6550, 8080, 8443), creates temporary configuration files, and manages kubectl context switching after cluster creation.
