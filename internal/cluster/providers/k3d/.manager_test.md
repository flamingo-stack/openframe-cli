This is a comprehensive test file for the K3d cluster manager implementation. It provides thorough test coverage for all cluster management operations including creation, deletion, starting, listing, and status checking using mocked command execution.

## Key Components

- **MockExecutor**: Mock implementation of CommandExecutor for isolated testing
- **Test Functions**:
  - `TestNewK3dManager`: Tests manager initialization with different configurations
  - `TestCreateClusterManagerWithExecutor`: Tests factory function for manager creation
  - `TestK3dManager_CreateCluster`: Tests cluster creation with various configurations and error scenarios
  - `TestK3dManager_DeleteCluster`: Tests cluster deletion functionality
  - `TestK3dManager_StartCluster`: Tests cluster startup operations
  - `TestK3dManager_ListClusters`: Tests cluster listing with JSON parsing
  - `TestK3dManager_GetClusterStatus`: Tests status retrieval functionality

## Usage Example

```go
// Create a mock executor for testing
executor := &MockExecutor{}
executor.On("Execute", mock.Anything, "k3d", mock.Anything).
    Return(&execPkg.CommandResult{Stdout: "success"}, nil)

// Test cluster creation
manager := NewK3dManager(executor, false)
config := models.ClusterConfig{
    Name:      "test-cluster",
    Type:      models.ClusterTypeK3d,
    NodeCount: 3,
}

err := manager.CreateCluster(context.Background(), config)
assert.NoError(t, err)
executor.AssertExpectations(t)
```

The tests cover both happy path scenarios and error conditions, including validation errors, command execution failures, and JSON parsing issues. Each test uses the testify framework for assertions and mocking.
