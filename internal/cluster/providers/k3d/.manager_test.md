This test file provides comprehensive unit testing for the K3D cluster manager implementation. It includes mock implementations and test cases to verify proper cluster management functionality.

## Key Components

- **MockExecutor**: Mock implementation of CommandExecutor interface for testing
- **Constructor Tests**: Tests for `NewK3dManager`, `CreateClusterManagerWithExecutor`, and `CreateDefaultClusterManager`
- **Cluster Operations Tests**: Comprehensive test suites for:
  - `CreateCluster` - Cluster creation with various configurations and error scenarios
  - `DeleteCluster` - Cluster deletion operations  
  - `StartCluster` - Starting existing clusters
  - `ListClusters` - Retrieving cluster information with JSON parsing
  - `GetClusterStatus` - Fetching status of specific clusters

## Usage Example

```go
// Run specific test
go test -run TestK3dManager_CreateCluster

// Run all K3D manager tests
go test ./...

// Example of how tests verify cluster creation
func TestK3dManager_CreateCluster(t *testing.T) {
    executor := &MockExecutor{}
    executor.On("Execute", mock.Anything, "k3d", mock.Anything).
        Return(&execPkg.CommandResult{Stdout: "success"}, nil)
    
    manager := NewK3dManager(executor, false)
    config := models.ClusterConfig{
        Name:      "test-cluster",
        Type:      models.ClusterTypeK3d,
        NodeCount: 3,
    }
    
    err := manager.CreateCluster(context.Background(), config)
    assert.NoError(t, err)
}
```

The tests cover success scenarios, error conditions, input validation, and edge cases like verbose mode and JSON parsing failures.
