This file implements an automated installer for OpenFrame CLI prerequisites including Docker, kubectl, and k3d. It provides both interactive and non-interactive installation modes with comprehensive user feedback and error handling.

## Key Components

- **`Installer`** - Main installer struct that orchestrates prerequisite installation
- **`NewInstaller()`** - Factory function that creates a new installer with integrated prerequisite checker
- **`InstallMissingPrerequisites()`** - Installs all missing tools with progress tracking
- **`CheckAndInstallNonInteractive()`** - Handles installation with optional non-interactive mode for automation
- **`installTool()`** - Dispatches installation to tool-specific installers
- **`showManualInstructions()`** - Displays platform-specific manual installation guidance

## Usage Example

```go
// Create installer and check/install prerequisites interactively
installer := prerequisites.NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Non-interactive mode for automation
if err := installer.CheckAndInstallNonInteractive(true); err != nil {
    log.Fatal(err)
}

// Install only missing prerequisites with progress tracking
if err := installer.InstallMissingPrerequisites(); err != nil {
    log.Fatal(err)
}
```

The installer features a two-phase approach: first installing missing tools, then handling Docker startup separately. It provides rich terminal UI with spinners, progress indicators, and platform-specific instructions for manual installation when needed.
