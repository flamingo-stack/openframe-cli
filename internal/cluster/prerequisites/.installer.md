Manages the installation of required development tools including Docker, kubectl, and k3d with interactive and non-interactive modes. Provides a complete workflow for checking prerequisites and installing missing tools with user feedback via spinners and prompts.

## Key Components

- `Installer` - Main struct that orchestrates tool installation and verification
- `NewInstaller()` - Creates a new installer instance with prerequisite checker
- `InstallMissingPrerequisites()` - Installs all missing prerequisites automatically
- `CheckAndInstall()` - Interactive mode for checking and installing prerequisites
- `CheckAndInstallNonInteractive(bool)` - Non-interactive mode with auto-approval option
- `installTool(string)` - Installs a specific tool (docker, kubectl, or k3d)
- `showManualInstructions()` - Displays manual installation instructions in table format

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/cluster/prerequisites"
)

func main() {
    installer := prerequisites.NewInstaller()
    
    // Interactive installation with user prompts
    if err := installer.CheckAndInstall(); err != nil {
        log.Fatal(err)
    }
    
    // Non-interactive installation (auto-approve)
    if err := installer.CheckAndInstallNonInteractive(true); err != nil {
        log.Fatal(err)
    }
    
    // Install only missing prerequisites
    if err := installer.InstallMissingPrerequisites(); err != nil {
        log.Fatal(err)
    }
}
```
