Manages prerequisite tool installation and verification for the OpenFrame CLI, handling Docker, kubectl, and k3d installations across different operating systems.

## Key Components

- **Installer** - Main struct that orchestrates prerequisite installation with integrated checking capabilities
- **NewInstaller()** - Creates a new installer instance with a prerequisite checker
- **InstallMissingPrerequisites()** - Detects and installs all missing tools with progress feedback
- **CheckAndInstall()** - Interactive flow for checking and installing prerequisites
- **CheckAndInstallNonInteractive()** - Automated installation flow for CI/CD environments
- **installTool()** - Delegates installation to tool-specific installers (Docker, kubectl, k3d)

## Usage Example

```go
// Create installer and check prerequisites
installer := prerequisites.NewInstaller()

// Interactive installation with user prompts
if err := installer.CheckAndInstall(); err != nil {
    log.Fatalf("Failed to install prerequisites: %v", err)
}

// Non-interactive installation for automation
if err := installer.CheckAndInstallNonInteractive(true); err != nil {
    log.Printf("Some prerequisites failed: %v", err)
    // Continue execution in non-interactive mode
}

// Install only missing tools
if err := installer.InstallMissingPrerequisites(); err != nil {
    log.Fatalf("Installation failed: %v", err)
}
```

The installer handles complex scenarios like Docker being installed but not running, provides platform-specific manual installation instructions, and supports both interactive and automated workflows with comprehensive error handling and progress indicators.
