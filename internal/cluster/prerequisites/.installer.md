Manages the automatic installation and verification of OpenFrame CLI prerequisites including Docker, kubectl, and k3d. Provides both interactive and non-interactive installation modes with progress feedback and error handling.

## Key Components

- **Installer**: Main struct that orchestrates prerequisite installation
- **NewInstaller()**: Creates a new installer instance
- **InstallMissingPrerequisites()**: Installs all missing prerequisites automatically
- **CheckAndInstall()**: Interactive mode for checking and installing prerequisites
- **CheckAndInstallNonInteractive()**: Non-interactive mode for automated environments
- **installTool()**: Installs a specific tool (Docker, kubectl, or k3d)
- **showManualInstructions()**: Displays manual installation instructions when auto-install is declined

## Usage Example

```go
// Create installer and check/install prerequisites interactively
installer := NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Or use non-interactive mode for CI/CD environments
if err := installer.CheckAndInstallNonInteractive(true); err != nil {
    log.Fatal(err)
}

// Install only missing prerequisites without prompting
if err := installer.InstallMissingPrerequisites(); err != nil {
    log.Fatal(err)
}
```

The installer provides visual feedback through spinners and colored output, handles Docker startup separately from installation, and gracefully falls back to manual instructions when automatic installation fails or is declined.
