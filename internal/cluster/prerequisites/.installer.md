Handles automated installation and management of OpenFrame CLI prerequisites including Docker, kubectl, and k3d tools.

## Key Components

- **Installer**: Main struct that orchestrates prerequisite installation and verification
- **NewInstaller()**: Creates a new installer instance with prerequisite checker
- **InstallMissingPrerequisites()**: Installs all missing prerequisites with progress feedback
- **CheckAndInstall()**: Interactive check and install flow
- **CheckAndInstallNonInteractive()**: Non-interactive installation mode for automated environments
- **installTool()**: Delegates installation to specific tool installers (docker, kubectl, k3d)

## Usage Example

```go
// Basic installation
installer := NewInstaller()
if err := installer.InstallMissingPrerequisites(); err != nil {
    log.Fatalf("Failed to install prerequisites: %v", err)
}

// Interactive check and install
if err := installer.CheckAndInstall(); err != nil {
    log.Fatalf("Installation failed: %v", err)
}

// Non-interactive mode (for CI/automated environments)
if err := installer.CheckAndInstallNonInteractive(true); err != nil {
    log.Printf("Warning: Some prerequisites may be missing: %v", err)
}
```

The installer provides both interactive prompts for user confirmation and automated installation modes, with proper error handling and progress indicators using pterm spinners.
