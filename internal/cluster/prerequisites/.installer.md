This Go file implements a prerequisite installer that automatically checks and installs missing development tools required for the OpenFrame CLI cluster operations. It provides both interactive and non-interactive installation modes with user-friendly progress indicators and error handling.

## Key Components

- **`Installer`**: Main struct that orchestrates the installation process using a `PrerequisiteChecker`
- **`InstallMissingPrerequisites()`**: Checks for missing tools and installs them automatically
- **`CheckAndInstallNonInteractive()`**: Handles both interactive and automated installation workflows
- **`installTool()`**: Routes installation requests to specific tool installers (Docker, kubectl, k3d, helm)
- **Manual instruction helpers**: Provides platform-specific guidance when automatic installation fails

## Usage Example

```go
// Create installer and check/install prerequisites interactively
installer := prerequisites.NewInstaller()
err := installer.CheckAndInstall()
if err != nil {
    log.Fatal(err)
}

// Non-interactive mode for CI/automation
err = installer.CheckAndInstallNonInteractive(true)
if err != nil {
    log.Fatal(err)
}

// Install only missing prerequisites
err = installer.InstallMissingPrerequisites()
if err != nil {
    log.Fatal(err)
}
```

The installer handles Docker startup separately from installation, provides platform-specific manual instructions, and includes comprehensive error handling with progress spinners for better user experience.
