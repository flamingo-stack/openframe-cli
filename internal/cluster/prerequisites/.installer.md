This file provides a comprehensive installation manager for Docker, kubectl, and k3d prerequisites required by the OpenFrame CLI tool.

## Key Components

- **Installer** - Main struct that orchestrates prerequisite installation with progress tracking
- **NewInstaller()** - Factory function that creates a new installer with integrated prerequisite checker
- **InstallMissingPrerequisites()** - Automatically detects and installs all missing tools with user feedback
- **CheckAndInstallNonInteractive()** - Supports both interactive and automated installation modes
- **installTool()** - Tool-specific installation dispatcher for Docker, kubectl, and k3d
- **showManualInstructions()** - Displays platform-specific manual installation guides

## Usage Example

```go
// Create installer and install missing prerequisites
installer := NewInstaller()
err := installer.CheckAndInstall()
if err != nil {
    log.Fatalf("Failed to install prerequisites: %v", err)
}

// Non-interactive mode for CI/automated environments
err = installer.CheckAndInstallNonInteractive(true)
if err != nil {
    log.Printf("Some prerequisites failed: %v", err)
}

// Install only specific tools
installer := NewInstaller()
err = installer.installSpecificTools([]string{"docker", "kubectl"})
```

The installer features a two-phase approach: first installing missing tools, then handling Docker startup separately. It provides rich terminal UI with spinners, progress indicators, and cross-platform manual installation instructions. The system gracefully handles both interactive user prompts and automated CI environments.
