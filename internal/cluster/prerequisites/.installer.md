Manages automated installation of OpenFrame CLI prerequisites including Docker, kubectl, and k3d with interactive and non-interactive modes.

## Key Components

- **`Installer`** - Main installation orchestrator with prerequisite checker integration
- **`NewInstaller()`** - Creates installer instance with embedded prerequisite checker
- **`InstallMissingPrerequisites()`** - Installs all detected missing tools with progress tracking
- **`CheckAndInstall()`** - Interactive flow for checking and installing prerequisites
- **`CheckAndInstallNonInteractive()`** - Non-interactive installation mode for automation
- **`installTool()`** - Tool-specific installation dispatcher
- **Manual instruction methods** - Provides platform-specific installation guidance

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/cluster/prerequisites"
)

func main() {
    installer := prerequisites.NewInstaller()
    
    // Interactive installation
    if err := installer.CheckAndInstall(); err != nil {
        log.Fatal(err)
    }
    
    // Non-interactive installation (for CI/automation)
    if err := installer.CheckAndInstallNonInteractive(true); err != nil {
        log.Fatal(err)
    }
    
    // Install only missing tools
    if err := installer.InstallMissingPrerequisites(); err != nil {
        log.Fatal(err)
    }
}
```

The installer handles Docker service management separately from installation, provides platform-specific manual instructions when automatic installation fails, and uses spinners for visual feedback during installation progress.
