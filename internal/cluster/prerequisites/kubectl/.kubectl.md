This package provides cross-platform kubectl installation and detection functionality for Kubernetes command-line tool management.

## Key Components

- **KubectlInstaller**: Main struct that handles kubectl installation across different operating systems
- **NewKubectlInstaller()**: Constructor function that returns a new installer instance
- **IsInstalled()**: Checks if kubectl is already installed and functional
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs kubectl using the appropriate package manager for the current OS

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Get installation help for manual installation
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please follow the manual installation instructions above")
    } else {
        fmt.Println("kubectl installed successfully")
    }
}
```

The installer supports macOS (via Homebrew), various Linux distributions (Ubuntu/Debian, CentOS/RHEL, Fedora, Arch), and provides manual installation guidance for Windows. It automatically detects the appropriate package manager and handles the complete installation process including repository setup and GPG key management.
