This package provides cross-platform kubectl installation and verification functionality, automatically detecting the operating system and using appropriate package managers or binary downloads to install kubectl.

## Key Components

- **KubectlInstaller**: Main struct that handles kubectl installation operations
- **commandExists()**: Checks if a command is available in the system PATH  
- **isKubectlInstalled()**: Verifies kubectl is installed and working
- **kubectlInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs kubectl based on the detected OS and package manager

## Platform Support

### Automatic Installation
- **macOS**: Uses Homebrew (`brew install kubectl`)
- **Linux**: Detects and uses appropriate package manager:
  - Ubuntu/Debian: `apt` with Kubernetes repository
  - CentOS/RHEL: `yum` with Kubernetes repository  
  - Fedora: `dnf` with Kubernetes repository
  - Arch Linux: `pacman`
  - Fallback: Direct binary download
- **Windows**: Manual installation only (provides help text)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Try automatic installation
    if err := installer.Install(); err != nil {
        // Fall back to manual installation help
        fmt.Println("Automatic installation failed:")
        fmt.Println(installer.GetInstallHelp())
        log.Fatal(err)
    }
    
    fmt.Println("kubectl installed successfully!")
}
```

The installer runs commands silently to avoid cluttering output and handles architecture detection for binary downloads on unsupported Linux distributions.
