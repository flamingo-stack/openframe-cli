This package provides cross-platform detection and automatic installation of the kubectl command-line tool for Kubernetes cluster management.

## Key Components

- **KubectlInstaller**: Main struct that handles kubectl installation and detection
- **commandExists()**: Checks if a command is available in system PATH
- **isKubectlInstalled()**: Verifies kubectl installation and functionality
- **kubectlInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Performs automatic kubectl installation based on the operating system
- Platform-specific installers for macOS (Homebrew), Linux distributions (apt, yum, dnf, pacman), and binary download fallback

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Get manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Auto-installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
    } else {
        fmt.Println("kubectl installed successfully")
    }
}
```

The installer automatically detects the operating system and package manager, then uses the appropriate installation method (Homebrew on macOS, native package managers on Linux, or direct binary download as fallback).
