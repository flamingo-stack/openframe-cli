Cross-platform kubectl installer and detector for Kubernetes CLI tools. Provides automatic installation capabilities for macOS (Homebrew), Linux distributions (package managers + binary fallback), and Windows (WSL2-based).

## Key Components

- **KubectlInstaller**: Main installer struct with platform-specific installation methods
- **isKubectlInstalled()**: Detects kubectl presence with timeout protection and WSL2 support
- **kubectlInstallHelp()**: Platform-specific installation guidance
- **Install()**: Automatic installation dispatcher based on OS detection
- **Platform installers**: Dedicated methods for macOS, Linux distros, and Windows/WSL2

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is installed
    if !installer.IsInstalled() {
        fmt.Println("kubectl not found")
        
        // Get platform-specific help
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation
        if err := installer.Install(); err != nil {
            log.Fatalf("Failed to install kubectl: %v", err)
        }
        
        fmt.Println("kubectl installed successfully!")
    } else {
        fmt.Println("kubectl is already installed")
    }
}
```

The installer supports multiple Linux package managers (apt, yum, dnf, pacman) and includes a binary download fallback. Windows installation uses WSL2 Ubuntu with automatic wrapper script generation for seamless Windows integration.
