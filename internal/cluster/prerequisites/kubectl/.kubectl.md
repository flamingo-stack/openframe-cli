Provides cross-platform kubectl installation and detection functionality with support for automatic installation on macOS and various Linux distributions.

## Key Components

- **KubectlInstaller**: Main struct that handles kubectl installation across different operating systems
- **IsInstalled()**: Checks if kubectl is already installed and functional
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs kubectl using the appropriate package manager
- **commandExists()**: Utility function to check if a command is available in PATH
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Ubuntu/Debian (apt), CentOS/RHEL (yum), Fedora (dnf), Arch Linux (pacman), and binary installation

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Get platform-specific help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually:", installer.GetInstallHelp())
    } else {
        fmt.Println("kubectl installed successfully")
    }
}
```

The installer automatically detects the operating system and package manager, then uses the most appropriate installation method. On Windows, it provides manual installation instructions since automatic installation is not supported.
