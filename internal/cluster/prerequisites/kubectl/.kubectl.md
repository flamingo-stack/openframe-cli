Cross-platform kubectl installer that handles kubectl installation and verification across macOS, Linux, and Windows (with WSL2 support).

## Key Components

- **KubectlInstaller** - Main installer struct with platform-specific installation methods
- **isKubectlInstalled()** - Verifies kubectl installation with timeout protection and WSL2 support
- **commandExists()** - Utility function to check if a command is available in PATH
- **kubectlInstallHelp()** - Provides platform-specific installation guidance
- **Platform-specific installers**:
  - `installMacOS()` - Uses Homebrew for installation
  - `installLinux()` - Direct binary download with retry logic
  - `installWindows()` - WSL2-based installation with Windows wrapper

## Usage Example

```go
// Create a new kubectl installer
installer := kubectl.NewKubectlInstaller()

// Check if kubectl is already installed
if installer.IsInstalled() {
    fmt.Println("kubectl is already installed")
} else {
    fmt.Println("kubectl not found")
    
    // Get platform-specific installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        fmt.Printf("Failed to install kubectl: %v\n", err)
    } else {
        fmt.Println("kubectl installed successfully!")
    }
}
```

The installer includes special handling for Windows environments by installing kubectl inside WSL2 Ubuntu and creating a Windows batch wrapper that bridges Windows kubectl calls to the WSL2 installation. It also features retry logic for downloads and fallback version support for reliable installation across different network conditions.
