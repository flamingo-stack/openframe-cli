A Go package that provides cross-platform kubectl installation and verification capabilities. It detects the operating system and package manager to automatically install kubectl using the most appropriate method for each platform.

## Key Components

- **KubectlInstaller**: Main struct that implements installation methods for different platforms
- **IsInstalled()**: Checks if kubectl is available and functional on the system
- **GetInstallHelp()**: Returns platform-specific installation instructions as a string
- **Install()**: Automatically installs kubectl using the detected OS and package manager
- **commandExists()**: Utility function to check if a command is available in PATH
- **Platform-specific installers**: Dedicated methods for macOS (Homebrew), Ubuntu/Debian (apt), CentOS/RHEL (yum), Fedora (dnf), Arch Linux (pacman), and binary installation

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "your-project/kubectl"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Get installation help for manual installation
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
        return
    }
    
    fmt.Println("kubectl installed successfully!")
}
```

The installer automatically detects the platform and uses the appropriate package manager (brew, apt, yum, dnf, pacman) or falls back to binary installation. On Windows, it only provides installation instructions since automatic installation is not supported.
