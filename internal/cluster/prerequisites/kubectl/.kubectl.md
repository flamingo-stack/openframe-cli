Provides cross-platform kubectl installation functionality for Kubernetes CLI tool management. The package detects system architecture and package managers to automatically install kubectl using platform-appropriate methods.

## Key Components

- **KubectlInstaller**: Main struct that provides kubectl installation and management capabilities
- **commandExists()**: Utility function to check if a command is available in the system PATH
- **isKubectlInstalled()**: Verifies if kubectl is properly installed and functional
- **kubectlInstallHelp()**: Returns platform-specific installation instructions as fallback guidance
- **Install()**: Orchestrates automatic kubectl installation based on the detected operating system
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Ubuntu/Debian (apt), RHEL/CentOS (yum), Fedora (dnf), Arch Linux (pacman), and binary installation

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Get platform-specific help instructions
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
        return
    }
    
    fmt.Println("kubectl installed successfully!")
}
```

The installer handles platform detection automatically and uses the most appropriate installation method for each system, from package managers to direct binary downloads when repositories aren't available.
