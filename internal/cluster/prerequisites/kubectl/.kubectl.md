This file provides cross-platform kubectl installation detection and automation for Kubernetes command-line tool management. It includes methods to check if kubectl is installed, provide platform-specific installation instructions, and automatically install kubectl on supported systems.

## Key Components

- **KubectlInstaller**: Main struct providing kubectl management functionality
- **commandExists()**: Utility function to check if a command exists in PATH
- **isKubectlInstalled()**: Verifies kubectl installation and functionality
- **kubectlInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs kubectl based on the operating system
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Ubuntu/Debian (apt), CentOS/RHEL (yum), Fedora (dnf), Arch Linux (pacman), and binary installation

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Get installation help for current platform
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
    } else {
        fmt.Println("kubectl installed successfully!")
    }
}
```
