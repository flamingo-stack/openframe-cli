Test suite for Docker installer functionality that validates cross-platform installation support and help text generation.

## Key Components

- **TestNewDockerInstaller**: Tests Docker installer instantiation
- **TestDockerInstaller_GetInstallHelp**: Validates platform-specific installation help text
- **TestDockerInstaller_Install**: Tests installation error handling across different operating systems
- **containsSubstring**: Helper function for substring matching in test assertions

## Usage Example

```go
// Run the test suite
go test -v ./docker

// Test individual functions
func TestExample(t *testing.T) {
    installer := NewDockerInstaller()
    
    // Verify installer creation
    if installer == nil {
        t.Error("Expected Docker installer to be created")
    }
    
    // Check help text contains platform-specific guidance
    help := installer.GetInstallHelp()
    if runtime.GOOS == "darwin" {
        if !containsSubstring(help, "brew") {
            t.Error("macOS help should mention Homebrew")
        }
    }
}
```

The tests verify that the Docker installer provides appropriate platform-specific installation guidance and handles unsupported scenarios gracefully without actually performing installations during CI runs.
