Test suite for the Docker installer functionality, validating installation commands and help text across different operating systems.

## Key Components

- **TestNewDockerInstaller**: Verifies that the Docker installer can be created successfully
- **TestDockerInstaller_GetInstallHelp**: Tests platform-specific installation help text for macOS, Linux, and Windows
- **TestDockerInstaller_Install**: Validates installation behavior and error handling across different platforms
- **containsSubstring**: Helper function for substring checking in test assertions

## Usage Example

```go
func TestDockerFunctionality(t *testing.T) {
    // Create installer instance
    installer := NewDockerInstaller()
    
    // Test help text generation
    help := installer.GetInstallHelp()
    if help == "" {
        t.Error("Help text should not be empty")
    }
    
    // Test installation (with error handling)
    err := installer.Install()
    if runtime.GOOS == "windows" {
        // Expect manual installation suggestion
        if !containsSubstring(err.Error(), "Docker Desktop") {
            t.Error("Windows should suggest Docker Desktop")
        }
    }
}

// Use the helper function for string validation
if containsSubstring(output, "expected text") {
    // String contains expected content
}
```
