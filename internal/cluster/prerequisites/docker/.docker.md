This package provides comprehensive Docker installation and management functionality across different operating systems. It handles detection, installation, and startup of Docker with platform-specific implementations.

## Key Components

- **`DockerInstaller`**: Main struct that implements cross-platform Docker installation
- **`IsDockerRunning()`**: Checks if Docker daemon is accessible
- **`IsDockerInstalledButNotRunning()`**: Detects when Docker is installed but not running
- **`StartDocker()`**: Attempts to start Docker service/application based on OS
- **`WaitForDocker()`**: Waits up to 30 seconds for Docker daemon to become available
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Linux distributions (apt/yum/dnf/pacman), and Windows

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "your-module/docker"
)

func main() {
    installer := docker.NewDockerInstaller()
    
    // Check if Docker is installed
    if !installer.IsInstalled() {
        fmt.Println("Docker not found. Installing...")
        if err := installer.Install(); err != nil {
            log.Fatal(err)
        }
    }
    
    // Check if Docker is running
    if !docker.IsDockerRunning() {
        fmt.Println("Starting Docker...")
        if err := docker.StartDocker(); err != nil {
            log.Fatal(err)
        }
        
        // Wait for Docker to be ready
        if err := docker.WaitForDocker(); err != nil {
            log.Fatal(err)
        }
    }
    
    fmt.Println("Docker is ready!")
}
```
