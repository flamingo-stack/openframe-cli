Provides Docker installation and management functionality across multiple operating systems with automatic platform detection and installation methods.

## Key Components

- **DockerInstaller**: Main struct that implements cross-platform Docker installation
- **IsDockerRunning()**: Checks if Docker daemon is accessible and running
- **IsDockerInstalledButNotRunning()**: Detects when Docker is installed but daemon is not active
- **StartDocker()**: Platform-specific Docker daemon/Desktop startup
- **WaitForDocker()**: Waits up to 30 seconds for Docker daemon to become available

## Usage Example

```go
// Check Docker status
if !IsDockerRunning() {
    if IsDockerInstalledButNotRunning() {
        // Start existing Docker installation
        if err := StartDocker(); err != nil {
            log.Printf("Failed to start Docker: %v", err)
        }
        // Wait for Docker to be ready
        if err := WaitForDocker(); err != nil {
            log.Printf("Docker failed to start: %v", err)
        }
    } else {
        // Install Docker automatically
        installer := NewDockerInstaller()
        if !installer.IsInstalled() {
            fmt.Println(installer.GetInstallHelp())
            if err := installer.Install(); err != nil {
                log.Printf("Installation failed: %v", err)
            }
        }
    }
}
```

The installer supports automatic installation on macOS (via Homebrew), Linux (Ubuntu/Debian, CentOS/RHEL, Fedora, Arch), and provides guidance for Windows. It handles package manager detection, repository setup, and post-installation configuration including adding users to the docker group.
