This file provides a comprehensive Docker installation and management utility for cross-platform environments. It handles Docker detection, installation, and startup across macOS, Linux distributions, and Windows.

## Key Components

- **DockerInstaller**: Main struct implementing installation methods for different operating systems
- **IsDockerRunning()**: Checks if Docker daemon is accessible
- **IsDockerInstalledButNotRunning()**: Detects when Docker is installed but not running
- **StartDocker()**: Attempts to start Docker service/application based on OS
- **WaitForDocker()**: Waits up to 30 seconds for Docker daemon to become available
- **Install()**: Automatically installs Docker using platform-specific package managers

## Usage Example

```go
// Check Docker status
if !IsDockerRunning() {
    if IsDockerInstalledButNotRunning() {
        // Start existing Docker installation
        if err := StartDocker(); err != nil {
            return fmt.Errorf("failed to start Docker: %w", err)
        }
        
        // Wait for daemon to be ready
        if err := WaitForDocker(); err != nil {
            return fmt.Errorf("Docker failed to start: %w", err)
        }
    } else {
        // Install Docker
        installer := NewDockerInstaller()
        if !installer.IsInstalled() {
            if err := installer.Install(); err != nil {
                fmt.Println(installer.GetInstallHelp())
                return err
            }
        }
    }
}
```
