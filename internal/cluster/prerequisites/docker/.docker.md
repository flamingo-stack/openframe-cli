A comprehensive Docker installation and management utility that provides cross-platform Docker installation, status checking, and daemon management capabilities.

## Key Components

- **DockerInstaller**: Main struct providing Docker installation methods across different operating systems
- **Status Functions**: `IsDockerRunning()`, `IsDockerInstalledButNotRunning()`, `isDockerInstalled()` for checking Docker state
- **Installation Methods**: Platform-specific installers for macOS, Linux distributions (Ubuntu, CentOS, Fedora, Arch), and Windows
- **Docker Management**: `StartDocker()` and `WaitForDocker()` for starting and waiting for Docker daemon
- **Utility Functions**: `commandExists()` for checking command availability and helper methods for running system commands

## Usage Example

```go
// Check Docker status
if !IsDockerRunning() {
    if IsDockerInstalledButNotRunning() {
        // Start existing Docker installation
        if err := StartDocker(); err != nil {
            log.Printf("Failed to start Docker: %v", err)
        }
        
        // Wait for Docker to be ready
        if err := WaitForDocker(); err != nil {
            log.Printf("Docker failed to start: %v", err)
        }
    } else {
        // Install Docker
        installer := NewDockerInstaller()
        if !installer.IsInstalled() {
            fmt.Println(installer.GetInstallHelp())
            if err := installer.Install(); err != nil {
                log.Printf("Installation failed: %v", err)
            }
        }
    }
}
```

The installer automatically detects the operating system and package manager, handling platform-specific installation procedures including repository setup, GPG keys, and service management.
