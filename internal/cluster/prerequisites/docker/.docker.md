A comprehensive Docker installer utility that provides cross-platform Docker installation and management capabilities, with special support for WSL2 on Windows.

## Key Components

- **DockerInstaller struct** - Main installer implementation with platform-specific methods
- **commandExists()** - Checks if a command is available in the system PATH
- **isDockerInstalled()** - Verifies Docker installation across platforms
- **IsDockerRunning()** - Checks if Docker daemon is accessible and running
- **IsDockerInstalledButNotRunning()** - Detects Docker installation without running daemon
- **Platform-specific installers** - Separate methods for macOS, Linux distributions, and Windows/WSL2

## Key Functions

### Detection Functions
```go
// Check if Docker is installed
installer := NewDockerInstaller()
if installer.IsInstalled() {
    fmt.Println("Docker is installed")
}

// Check if Docker daemon is running
if IsDockerRunning() {
    fmt.Println("Docker is running")
}

// Check for installation without running daemon
if IsDockerInstalledButNotRunning() {
    fmt.Println("Docker installed but not running")
}
```

### Installation
```go
// Create installer and attempt installation
installer := NewDockerInstaller()
if err := installer.Install(); err != nil {
    fmt.Printf("Installation failed: %v\n", err)
    fmt.Println(installer.GetInstallHelp())
}
```

## Platform Support

- **macOS** - Installs Docker Desktop via Homebrew
- **Linux** - Supports Ubuntu/Debian (apt), CentOS/RHEL (yum), Fedora (dnf), and Arch Linux (pacman)
- **Windows** - Installs Docker CE in WSL2 Ubuntu without Docker Desktop requirement

## Special Features

- **WSL2 Integration** - Automatic WSL2 setup and Ubuntu installation on Windows
- **Docker Wrapper Creation** - Creates Windows command wrapper for seamless Docker usage
- **User Group Management** - Automatically adds users to docker group on Linux
- **Service Management** - Enables and starts Docker systemd services on Linux
- **Timeout Handling** - Uses context timeouts for daemon connectivity checks
