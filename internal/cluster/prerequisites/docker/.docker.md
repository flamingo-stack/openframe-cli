This file provides a cross-platform Docker installation and management system with automatic detection, installation, and startup capabilities for different operating systems.

## Key Components

- **DockerInstaller**: Main struct that implements platform-specific Docker installation
- **Docker Status Functions**: `IsDockerRunning()`, `IsDockerInstalledButNotRunning()`, `isDockerInstalled()`
- **Installation Methods**: Platform-specific installers for macOS (Homebrew), Linux (apt/yum/dnf/pacman), and Windows
- **Docker Management**: `StartDocker()` and `WaitForDocker()` for daemon control
- **Utility Functions**: `commandExists()` for checking command availability

## Usage Example

```go
// Check Docker status
if !IsDockerRunning() {
    if IsDockerInstalledButNotRunning() {
        // Docker installed but not running - try to start it
        if err := StartDocker(); err != nil {
            log.Printf("Failed to start Docker: %v", err)
        }
        // Wait for Docker to be ready
        if err := WaitForDocker(); err != nil {
            log.Printf("Docker failed to start: %v", err)
        }
    } else {
        // Docker not installed - install it
        installer := NewDockerInstaller()
        if !installer.IsInstalled() {
            fmt.Println(installer.GetInstallHelp())
            if err := installer.Install(); err != nil {
                log.Printf("Failed to install Docker: %v", err)
            }
        }
    }
}
```

The installer automatically detects the operating system and package manager, then uses the appropriate installation method. On Linux, it supports Ubuntu/Debian (apt), CentOS/RHEL (yum), Fedora (dnf), and Arch Linux (pacman).
