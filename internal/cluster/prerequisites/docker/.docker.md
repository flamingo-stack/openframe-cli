This package provides cross-platform Docker installation and management functionality, with special handling for Windows WSL2 environments. It detects Docker installation status, checks if the Docker daemon is running, and provides automated installation methods for different operating systems.

## Key Components

**Core Functions:**
- `commandExists()` - Checks if a command is available in system PATH
- `isDockerInstalled()` - Detects Docker installation (including WSL2 on Windows)
- `IsDockerRunning()` - Verifies Docker daemon accessibility with timeout
- `IsDockerInstalledButNotRunning()` - Identifies Docker daemon issues

**DockerInstaller Struct:**
- `IsInstalled()` - Wrapper for installation detection
- `GetInstallHelp()` - Returns OS-specific installation instructions
- `Install()` - Automated installation dispatcher for different platforms

**Platform-Specific Installers:**
- `installMacOS()` - Uses Homebrew to install Docker Desktop
- `installLinux()` - Auto-detects package manager (apt/yum/dnf/pacman)
- `installWindows()` - Sets up Docker CE in WSL2 Ubuntu
- `installUbuntu()`, `installRedHat()`, `installFedora()`, `installArch()` - Distro-specific implementations

## Usage Example

```go
package main

import (
    "fmt"
    "your-project/docker"
)

func main() {
    installer := docker.NewDockerInstaller()
    
    // Check installation status
    if !installer.IsInstalled() {
        fmt.Println("Docker not found")
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation
        if err := installer.Install(); err != nil {
            fmt.Printf("Installation failed: %v\n", err)
            return
        }
    }
    
    // Verify Docker daemon is running
    if docker.IsDockerInstalledButNotRunning() {
        fmt.Println("Docker is installed but not running")
        fmt.Println("Please start Docker service")
    } else if docker.IsDockerRunning() {
        fmt.Println("Docker is ready!")
    }
}
```

The package handles platform differences automatically, including Windows-specific WSL2 Docker installation and Linux distribution detection for appropriate package managers.
