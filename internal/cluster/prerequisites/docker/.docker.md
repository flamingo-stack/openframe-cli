Provides Docker installation and management functionality across different operating systems, including detection, installation, and startup capabilities.

## Key Components

- **DockerInstaller struct**: Main installer with platform-specific installation methods
- **IsDockerRunning()**: Checks if Docker daemon is accessible
- **IsDockerInstalledButNotRunning()**: Detects when Docker is installed but not running
- **StartDocker()**: Attempts to start Docker based on OS
- **WaitForDocker()**: Waits for Docker daemon to become available

## Usage Example

```go
// Check Docker status
if !IsDockerRunning() {
    if IsDockerInstalledButNotRunning() {
        // Start existing Docker installation
        if err := StartDocker(); err != nil {
            log.Fatal(err)
        }
        // Wait for Docker to be ready
        if err := WaitForDocker(); err != nil {
            log.Fatal(err)
        }
    } else {
        // Install Docker
        installer := NewDockerInstaller()
        if !installer.IsInstalled() {
            if err := installer.Install(); err != nil {
                fmt.Println("Installation failed:", installer.GetInstallHelp())
                log.Fatal(err)
            }
        }
    }
}
```

The installer supports macOS (via Homebrew), Linux distributions (Ubuntu/Debian, CentOS/RHEL, Fedora, Arch), and provides installation guidance for Windows.
