This package provides Docker installation detection, automatic installation, and daemon management functionality across multiple operating systems.

## Key Components

- **DockerInstaller**: Main struct that handles Docker installation and management
- **Detection Functions**: `IsDockerRunning()`, `IsDockerInstalledButNotRunning()`, `commandExists()`
- **Installation Methods**: Platform-specific installers for macOS, Linux distributions, and Windows
- **Docker Management**: `StartDocker()`, `WaitForDocker()` for daemon lifecycle control

## Usage Example

```go
// Check Docker status
if !IsDockerRunning() {
    if IsDockerInstalledButNotRunning() {
        // Docker is installed but not running - try to start it
        if err := StartDocker(); err != nil {
            fmt.Printf("Failed to start Docker: %v\n", err)
        }
        // Wait for Docker to become available
        if err := WaitForDocker(); err != nil {
            fmt.Printf("Docker failed to start: %v\n", err)
        }
    } else {
        // Docker not installed - install it
        installer := NewDockerInstaller()
        if !installer.IsInstalled() {
            fmt.Println(installer.GetInstallHelp())
            if err := installer.Install(); err != nil {
                fmt.Printf("Installation failed: %v\n", err)
            }
        }
    }
}
```

The package supports automatic installation on macOS (via Homebrew), Ubuntu/Debian, CentOS/RHEL, Fedora, and Arch Linux. Windows installation provides helpful guidance but requires manual setup of Docker Desktop.
