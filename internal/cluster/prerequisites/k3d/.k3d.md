Provides a cross-platform installer for k3d (lightweight Kubernetes distribution) with automatic installation capabilities for macOS, Linux, and Windows (via WSL2).

## Key Components

- **K3dInstaller**: Main installer struct with platform-specific installation methods
- **isK3dInstalled()**: Checks if k3d is already installed, with special handling for Windows/WSL2
- **commandExists()**: Utility function to verify if a command is available in PATH
- **Install()**: Platform-aware installation dispatcher that routes to appropriate OS-specific installers
- **installMacOS()**: Uses Homebrew to install k3d on macOS
- **installLinux()**: Detects package manager and installs via official script with retry logic
- **installWindows()**: Installs k3d in WSL2 Ubuntu and creates Windows wrapper batch file
- **installScript()**: Robust installation using official k3d script with fallback binary download

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "your-project/k3d"
)

func main() {
    installer := k3d.NewK3dInstaller()
    
    // Check if k3d is already installed
    if installer.IsInstalled() {
        fmt.Println("k3d is already installed")
        return
    }
    
    // Get installation help for current platform
    fmt.Println("Installation needed:")
    fmt.Println(installer.GetInstallHelp())
    
    // Perform automatic installation
    fmt.Println("Installing k3d...")
    if err := installer.Install(); err != nil {
        log.Fatalf("Installation failed: %v", err)
    }
    
    fmt.Println("k3d installed successfully!")
}
```

The installer includes robust error handling with retry logic and fallback versions to handle GitHub API rate limits and network issues during installation.
