Provides k3d (lightweight Kubernetes) installation management with cross-platform support and automatic detection of available package managers.

## Key Components

- **K3dInstaller** - Main installer struct implementing the installer interface
- **IsInstalled()** - Checks if k3d is already installed and functional
- **Install()** - Performs automatic installation based on the detected OS
- **GetInstallHelp()** - Returns platform-specific manual installation instructions

## Platform Support

### macOS
- Uses Homebrew (`brew install k3d`) when available
- Falls back to manual installation instructions

### Linux
- Detects package manager (apt, yum, dnf, pacman) 
- Uses AUR packages (k3d-bin) on Arch Linux with yay/paru
- Falls back to official installation script for all distributions

### Windows
- Provides manual installation instructions only
- Suggests Chocolatey or direct download from releases

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := NewK3dInstaller()
    
    if installer.IsInstalled() {
        fmt.Println("k3d is already installed")
        return
    }
    
    fmt.Println("Installing k3d...")
    if err := installer.Install(); err != nil {
        log.Printf("Auto-install failed: %v", err)
        fmt.Println("Manual installation required:")
        fmt.Println(installer.GetInstallHelp())
        return
    }
    
    fmt.Println("k3d installed successfully!")
}
```
