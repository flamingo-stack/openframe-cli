A cross-platform installer for k3d (a lightweight Kubernetes distribution) that provides automatic installation capabilities and platform-specific installation guidance.

## Key Components

**K3dInstaller struct**: Main installer type providing installation methods for different operating systems.

**Core Functions**:
- `NewK3dInstaller()`: Creates a new installer instance
- `IsInstalled()`: Checks if k3d is already installed on the system
- `GetInstallHelp()`: Returns platform-specific manual installation instructions
- `Install()`: Automatically installs k3d based on the detected operating system

**Platform Support**:
- **macOS**: Uses Homebrew (`brew install k3d`)
- **Linux**: Supports multiple distributions (Ubuntu, RHEL, Fedora, Arch) with fallback to official install script
- **Windows**: Manual installation only (returns error with instructions)

## Usage Example

```go
// Create installer instance
installer := k3d.NewK3dInstaller()

// Check if k3d is already installed
if !installer.IsInstalled() {
    fmt.Println("k3d not found, attempting installation...")
    
    // Try automatic installation
    if err := installer.Install(); err != nil {
        // Fall back to manual installation help
        fmt.Println("Auto-install failed:", err)
        fmt.Println(installer.GetInstallHelp())
    } else {
        fmt.Println("k3d installed successfully!")
    }
} else {
    fmt.Println("k3d is already installed")
}
```

The installer automatically detects the platform and package manager, providing a seamless installation experience across different environments.
