A Go package that provides cross-platform installation and detection capabilities for k3d (Kubernetes in Docker), offering automatic installation methods tailored to different operating systems and package managers.

## Key Components

- **K3dInstaller**: Main struct implementing installation methods
- **NewK3dInstaller()**: Factory function to create a new installer instance
- **IsInstalled()**: Checks if k3d is already installed on the system
- **GetInstallHelp()**: Returns OS-specific installation instructions
- **Install()**: Automatically installs k3d based on the detected OS and available package managers
- **commandExists()**: Utility function to check if a command is available in PATH
- **Platform-specific installers**: Separate methods for macOS, Linux distributions, and Windows

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := NewK3dInstaller()
    
    // Check if k3d is already installed
    if installer.IsInstalled() {
        fmt.Println("k3d is already installed")
        return
    }
    
    // Get installation help for manual installation
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Auto-installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
    } else {
        fmt.Println("k3d installed successfully!")
    }
}
```

The installer automatically detects the operating system and uses appropriate package managers (Homebrew on macOS, AUR helpers on Arch Linux) or falls back to the official installation script.
