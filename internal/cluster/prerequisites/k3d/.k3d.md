Provides a cross-platform installer for k3d (Kubernetes in Docker) with automatic detection and installation capabilities. The package handles platform-specific installation methods including Homebrew on macOS, package managers on Linux, and fallback to the official install script.

## Key Components

- **K3dInstaller**: Main struct that implements installation logic
- **NewK3dInstaller()**: Constructor for creating a new installer instance
- **IsInstalled()**: Checks if k3d is already installed on the system
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs k3d using the appropriate method for the current OS
- **commandExists()**: Utility function to check if a command is available in PATH
- **Platform-specific installers**: Separate methods for macOS, Linux distributions, and binary installation

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := k3d.NewK3dInstaller()
    
    // Check if k3d is already installed
    if installer.IsInstalled() {
        fmt.Println("k3d is already installed")
        return
    }
    
    // Get installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Fatalf("Failed to install k3d: %v", err)
    }
    
    fmt.Println("k3d installed successfully")
}
```

The installer automatically detects the operating system and uses the most appropriate installation method, falling back to the official install script when package managers aren't available.
