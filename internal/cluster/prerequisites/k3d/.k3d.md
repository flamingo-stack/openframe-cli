Provides cross-platform k3d (lightweight Kubernetes) installation and management functionality with automatic fallback mechanisms and retry logic for reliable deployment.

## Key Components

- **K3dInstaller struct**: Main installer with platform-specific installation methods
- **commandExists()**: Checks if a command is available in the system PATH
- **isK3dInstalled()**: Detects existing k3d installations with timeout protection
- **k3dInstallHelp()**: Returns platform-specific installation instructions
- **Platform installers**: Separate methods for macOS, Linux, Windows, and various Linux distributions

## Usage Example

```go
// Create a new k3d installer instance
installer := k3d.NewK3dInstaller()

// Check if k3d is already installed
if installer.IsInstalled() {
    fmt.Println("k3d is already installed")
} else {
    // Get platform-specific help
    fmt.Println(installer.GetInstallHelp())
    
    // Automatically install k3d
    if err := installer.Install(); err != nil {
        fmt.Printf("Installation failed: %v\n", err)
    } else {
        fmt.Println("k3d installed successfully")
    }
}
```

The installer handles platform-specific requirements:
- **macOS**: Uses Homebrew
- **Linux**: Detects package manager (apt, yum, dnf, pacman) or uses install script
- **Windows**: Installs in WSL2 Ubuntu with Windows wrapper batch file
- **Fallback**: Uses direct binary download with retry logic for network failures
