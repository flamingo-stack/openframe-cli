K3dInstaller provides cross-platform installation and validation functionality for k3d (Kubernetes in Docker), supporting macOS, Linux, and Windows with platform-specific installation methods.

## Key Components

- **K3dInstaller**: Main installer struct with methods for installation management
- **commandExists()**: Utility function to check if a command is available in PATH
- **isK3dInstalled()**: Validates k3d installation by checking version command
- **k3dInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Performs automated installation based on detected platform
- **Platform-specific installers**: Methods for macOS (Homebrew), Linux (script/AUR), and Windows

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := k3d.NewK3dInstaller()
    
    // Check if k3d is already installed
    if installer.IsInstalled() {
        fmt.Println("k3d is already installed")
        return
    }
    
    // Get installation help for manual installation
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automated installation
    if err := installer.Install(); err != nil {
        log.Printf("Automated installation failed: %v", err)
        fmt.Println("Please follow manual installation instructions above")
    } else {
        fmt.Println("k3d installed successfully!")
    }
}
```

The installer detects the operating system and uses appropriate package managers (Homebrew, AUR helpers) or falls back to the official installation script. Windows requires manual installation due to platform limitations.
