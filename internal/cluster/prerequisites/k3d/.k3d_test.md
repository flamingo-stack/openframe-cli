This test file validates the functionality of k3d (lightweight Kubernetes) installer components, ensuring cross-platform installation support and proper error handling.

## Key Components

- **TestNewK3dInstaller**: Validates k3d installer creation
- **TestK3dInstaller_GetInstallHelp**: Tests platform-specific installation help messages
- **TestK3dInstaller_Install**: Tests installation process and error handling across platforms
- **TestCommandExists**: Validates command existence checking utility
- **TestInstallScript**: Linux-specific installation script testing
- **containsSubstring**: Helper function for string matching in tests

## Usage Example

```go
// Run all tests
go test ./k3d

// Run specific test
go test -run TestNewK3dInstaller ./k3d

// Test with verbose output
go test -v ./k3d

// Test installation help on current platform
func ExampleInstaller() {
    installer := NewK3dInstaller()
    help := installer.GetInstallHelp()
    fmt.Println(help)
}
```

The tests verify that the k3d installer provides appropriate installation instructions for macOS (brew), Linux (curl/script), and Windows (chocolatey/manual), while gracefully handling unsupported platforms and missing dependencies.
