This file contains comprehensive unit tests for the K3d installer functionality, validating installation methods, help text generation, and platform-specific behaviors.

## Key Components

- **TestNewK3dInstaller**: Tests constructor for K3d installer instance creation
- **TestK3dInstaller_GetInstallHelp**: Validates platform-specific installation help text (brew for macOS, curl for Linux, chocolatey for Windows)
- **TestK3dInstaller_Install**: Tests installation behavior across different platforms and error conditions
- **TestCommandExists**: Validates command existence checking utility
- **TestInstallScript**: Linux-specific test for script-based installation
- **containsSubstring**: Helper function for string matching in tests

## Usage Example

```go
func TestK3dInstaller_GetInstallHelp(t *testing.T) {
    installer := NewK3dInstaller()
    help := installer.GetInstallHelp()
    
    if help == "" {
        t.Error("Install help should not be empty")
    }
    
    switch runtime.GOOS {
    case "darwin":
        if !containsSubstring(help, "brew") {
            t.Errorf("macOS help should contain brew reference")
        }
    }
}
```

The tests verify platform-specific installation methods, error handling for unsupported platforms, and command availability checks while gracefully handling CI/test environment limitations.
