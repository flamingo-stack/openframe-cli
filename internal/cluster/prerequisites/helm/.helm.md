This file provides a cross-platform Helm installer that automatically detects and installs the Helm package manager, with special handling for Windows through WSL2 integration.

## Key Components

- **`HelmInstaller`** - Main struct providing installation methods across operating systems
- **`isHelmInstalled()`** - Detects existing Helm installations with platform-specific checks
- **`helmInstallHelp()`** - Returns platform-specific installation guidance
- **Platform installers** - Separate methods for macOS (Homebrew), Linux (curl script), and Windows (WSL2)
- **Windows wrapper creation** - Creates a batch file wrapper for seamless Helm execution on Windows

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := helm.NewHelmInstaller()
    
    // Check if Helm is already installed
    if installer.IsInstalled() {
        fmt.Println("Helm is already installed")
        return
    }
    
    // Try automatic installation
    if err := installer.Install(); err != nil {
        // Fall back to manual instructions
        fmt.Println("Automatic installation failed:")
        fmt.Println(installer.GetInstallHelp())
        log.Fatal(err)
    }
    
    fmt.Println("Helm installed successfully!")
}
```

The installer handles platform-specific requirements: Homebrew on macOS, curl scripts on Linux, and WSL2 integration on Windows with PATH management and wrapper script creation for seamless cross-platform usage.
