This file provides cross-platform Helm installation functionality with special handling for Windows WSL2 environments. It implements automatic Helm installation and detection across different operating systems.

## Key Components

- **HelmInstaller struct**: Main installer implementation
- **commandExists()**: Utility function to check if a command is available in PATH
- **isHelmInstalled()**: Cross-platform Helm detection with special Windows WSL2 handling
- **NewHelmInstaller()**: Constructor for creating new installer instances
- **Install()**: Platform-specific installation dispatcher
- **installMacOS()**: Homebrew-based installation for macOS
- **installLinux()**: Script-based installation using official Helm install script
- **installWindows()**: WSL2-based installation with Windows wrapper creation
- **createHelmWrapper()**: Creates Windows batch wrapper for WSL2 Helm commands

## Usage Example

```go
// Create a new Helm installer
installer := helm.NewHelmInstaller()

// Check if Helm is already installed
if !installer.IsInstalled() {
    fmt.Println("Helm not found, installing...")
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        // Fall back to manual installation instructions
        fmt.Println(installer.GetInstallHelp())
        return err
    }
    
    fmt.Println("Helm installed successfully!")
} else {
    fmt.Println("Helm is already installed")
}
```

The installer handles platform-specific requirements including Homebrew for macOS, curl-based scripts for Linux, and WSL2 integration with PATH management for Windows environments.
