This file contains unit tests for the prerequisites installer functionality, testing tool installation delegation and command execution capabilities.

## Key Components

- **TestNewInstaller**: Validates that the installer is properly initialized with a non-nil checker component
- **TestInstallTool**: Tests tool installation for valid tools (docker, kubectl, k3d) and error handling for unknown tools
- **TestRunCommand**: Verifies command execution functionality using a simple echo command
- **containsSubstring**: Helper function that checks if a string contains a specific substring

## Usage Example

```go
// Run all tests
go test

// Run specific test
go test -run TestNewInstaller

// Run with verbose output
go test -v

// The tests validate installer behavior:
func TestNewInstaller(t *testing.T) {
    installer := NewInstaller()
    
    if installer == nil {
        t.Error("Expected installer to be created")
    }
    
    if installer.checker == nil {
        t.Error("Expected installer to have a checker")
    }
}
```

The tests ensure proper error handling for unknown tools, validate that supported tools (docker, kubectl, k3d) are handled appropriately, and confirm basic command execution works across different systems.
