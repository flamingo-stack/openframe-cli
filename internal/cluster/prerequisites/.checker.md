Provides a comprehensive prerequisite checking system for validating and managing required development tools (Docker, kubectl, k3d) before cluster operations.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of requirements and provides checking functionality
- **Requirement**: Defines a prerequisite with name, command, installation check function, and help text generator
- **NewPrerequisiteChecker()**: Factory function that creates a checker with predefined Docker, kubectl, and k3d requirements
- **CheckAll()**: Validates all prerequisites and returns status plus list of missing tools
- **GetInstallInstructions()**: Generates installation help text for missing tools
- **CheckPrerequisites()**: Convenience function for automated checking and installation

## Usage Example

```go
// Create and use the prerequisite checker
checker := NewPrerequisiteChecker()

// Check if all prerequisites are installed
allPresent, missing := checker.CheckAll()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function for automated handling
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
