Orchestrates prerequisite checking for OpenFrame CLI cluster operations, validating that required tools (Docker, kubectl, k3d) are installed and properly configured.

## Key Components

- **PrerequisiteChecker** - Main checker struct that validates required tools
- **Requirement** - Defines a prerequisite with name, command, installation check, and help functions
- **NewPrerequisiteChecker()** - Factory function creating checker with Docker, kubectl, and k3d requirements
- **CheckAll()** - Validates all prerequisites and returns status with missing tools list
- **GetInstallInstructions()** - Retrieves installation help for specific missing tools
- **CheckPrerequisites()** - Convenience function for complete prerequisite validation

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Quick prerequisite check
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites not met:", err)
}
```

The checker validates that Docker is running (not just installed), kubectl is available, and k3d is installed, providing tailored installation guidance for each missing tool.
