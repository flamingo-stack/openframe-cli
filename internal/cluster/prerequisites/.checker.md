Provides a unified prerequisite checking system for OpenFrame cluster deployment tools, validating that Docker, kubectl, k3d, and Helm are properly installed and configured.

## Key Components

- **PrerequisiteChecker**: Main checker that validates all required tools
- **Requirement**: Struct defining tool validation logic and install instructions  
- **CheckAll()**: Validates all prerequisites and returns missing tools
- **GetInstallInstructions()**: Provides installation help for missing tools
- **CheckPrerequisites()**: Entry point function with CI environment detection

## Usage Example

```go
// Create a new prerequisite checker
checker := NewPrerequisiteChecker()

// Check all required tools
allPresent, missing := checker.CheckAll()

if !allPresent {
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    
    for i, tool := range missing {
        fmt.Printf("Missing: %s\n", tool)
        fmt.Printf("Install: %s\n", instructions[i])
    }
}

// Or use the convenience function with CI detection
err := CheckPrerequisites()
if err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```

The checker validates Docker daemon status, kubectl availability, k3d installation, and Helm presence. It automatically detects CI environments (GitHub Actions, GitLab CI, CircleCI) to adjust behavior for non-interactive installation flows.
