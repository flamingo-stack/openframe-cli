A system for checking and managing prerequisites (Docker, kubectl, k3d) required for cluster operations, providing installation detection and help instructions.

## Key Components

- **PrerequisiteChecker**: Main orchestrator that manages requirement validation
- **Requirement**: Structure defining each prerequisite with name, command, detection logic, and install help
- **NewPrerequisiteChecker()**: Factory function that creates a checker with predefined Docker, kubectl, and k3d requirements
- **CheckAll()**: Validates all prerequisites and returns missing tools
- **GetInstallInstructions()**: Provides installation help for missing prerequisites
- **CheckPrerequisites()**: Convenience function for complete prerequisite validation

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Simple prerequisite check with automatic installation prompts
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites not met:", err)
}
```

The checker validates that Docker is running (not just installed), kubectl is available, and k3d is installed, providing specific installation guidance for each missing component.
