A prerequisite validation system that checks for required tools (Docker, kubectl, k3d) and provides installation guidance for missing components.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of system requirements
- **Requirement**: Defines a tool requirement with name, command, installation check, and help text
- **NewPrerequisiteChecker()**: Factory function that initializes checker with Docker, kubectl, and k3d requirements
- **CheckAll()**: Validates all prerequisites and returns missing tools
- **GetInstallInstructions()**: Retrieves installation help for specific missing tools
- **CheckPrerequisites()**: Convenience function for complete prerequisite validation

## Usage Example

```go
// Create a new prerequisite checker
checker := NewPrerequisiteChecker()

// Check all requirements at once
allPresent, missing := checker.CheckAll()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```

The checker handles Docker's special case where the tool may be installed but not running, providing appropriate guidance for both scenarios.
