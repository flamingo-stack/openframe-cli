Manages and validates OpenFrame CLI prerequisites by checking the availability of required tools like Docker, kubectl, k3d, and Helm. This checker verifies that all necessary tools are installed and provides installation guidance when dependencies are missing.

## Key Components

- **PrerequisiteChecker**: Main checker struct that validates all required tools
- **Requirement**: Defines a tool requirement with name, command, installation check, and help function
- **CheckAll()**: Validates all prerequisites and returns missing tools list
- **GetInstallInstructions()**: Provides installation guidance for missing tools
- **CheckPrerequisites()**: Entry point function that handles CI environment detection

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/prerequisites"
)

func main() {
    // Create a new prerequisite checker
    checker := prerequisites.NewPrerequisiteChecker()
    
    // Check all required tools
    allPresent, missing := checker.CheckAll()
    
    if !allPresent {
        fmt.Printf("Missing tools: %v\n", missing)
        
        // Get installation instructions for missing tools
        instructions := checker.GetInstallInstructions(missing)
        for _, instruction := range instructions {
            fmt.Println(instruction)
        }
    }
    
    // Alternative: Use the simplified check function
    err := prerequisites.CheckPrerequisites()
    if err != nil {
        fmt.Printf("Prerequisites check failed: %v\n", err)
    }
}
```
