Provides prerequisite checking functionality for cluster setup, validating that required tools (Docker, kubectl, k3d) are installed and available before cluster operations.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a list of requirements and provides validation methods
- **Requirement**: Struct defining a prerequisite with name, command, installation check, and help text
- **NewPrerequisiteChecker()**: Creates a checker instance with Docker, kubectl, and k3d requirements pre-configured
- **CheckAll()**: Validates all requirements and returns missing tools
- **GetInstallInstructions()**: Provides installation help for missing prerequisites
- **CheckPrerequisites()**: Convenience function for checking and installing missing tools

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
