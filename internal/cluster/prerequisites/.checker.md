Provides a prerequisite checking system for validating required development tools (Docker, kubectl, k3d) before cluster operations. The checker can detect missing tools and provide installation guidance.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of requirements and provides checking functionality
- **Requirement**: Defines a tool requirement with name, command, installation check, and help text
- **NewPrerequisiteChecker()**: Factory function that creates a checker with predefined requirements for Docker, kubectl, and k3d
- **CheckAll()**: Validates all requirements and returns status with list of missing tools
- **GetInstallInstructions()**: Retrieves installation help text for specified missing tools
- **CheckPrerequisites()**: Convenience function that performs complete check and installation flow

## Usage Example

```go
// Create and use the prerequisite checker
checker := prerequisites.NewPrerequisiteChecker()

// Check all requirements
allPresent, missing := checker.CheckAll()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := prerequisites.CheckPrerequisites(); err != nil {
    log.Fatal(err)
}
```
