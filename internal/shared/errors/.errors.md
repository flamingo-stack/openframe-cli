This file provides a comprehensive error handling system with custom error types and standardized display formatting for a CLI application.

## Key Components

- **ValidationError** - Custom error for field validation failures with field, value, and message details
- **CommandError** - Error wrapper for failed command executions, includes command name, arguments, and underlying error
- **BranchNotFoundError** - Specific error for Git branch not found scenarios
- **AlreadyHandledError** - Wrapper for errors already displayed to prevent duplicate messaging
- **ErrorHandler** - Central error processor with verbose mode support and type-specific formatting
- **Helper Functions** - Error constructors, type checkers, and global error handling utilities

## Usage Example

```go
// Create and handle validation errors
validationErr := CreateValidationError("cluster-name", "my-cluster", "name must be lowercase")
handler := NewErrorHandler(true)
handler.HandleError(validationErr)

// Handle command errors
cmdErr := CreateCommandError("kubectl", []string{"get", "pods"}, fmt.Errorf("connection refused"))
handler.HandleError(cmdErr)

// Global error handling in CLI commands
func MyCommandRun(cmd *cobra.Command, args []string) error {
    if err := someOperation(); err != nil {
        return HandleGlobalError(err, verbose)
    }
    return nil
}

// Type checking
if IsValidationError(err) {
    // Handle validation-specific logic
}
```
