This file provides a comprehensive error handling system for CLI applications with standardized error types, formatted display output, and user-friendly error processing.

## Key Components

- **ValidationError**: Handles field validation failures with field name, value, and error message
- **CommandError**: Wraps command execution errors with command details and underlying error  
- **BranchNotFoundError**: Specific error type for Git branch lookup failures
- **AlreadyHandledError**: Wrapper for errors that have already been displayed to users
- **ErrorHandler**: Core error processor with verbose/quiet output modes and formatted display
- **Helper Functions**: Error creation utilities and type checking functions
- **HandleGlobalError**: Entry point function for consistent CLI error handling

## Usage Example

```go
// Create specific error types
valErr := CreateValidationError("port", "abc", "must be numeric")
cmdErr := CreateCommandError("kubectl", []string{"apply", "-f", "config.yaml"}, err)

// Set up error handler
handler := NewErrorHandler(true) // verbose mode

// Process errors with formatted output
handler.HandleError(valErr)

// Global error handling for CLI commands
func (cmd *Command) RunE(cmd *cobra.Command, args []string) error {
    if err := doSomething(); err != nil {
        return HandleGlobalError(err, verbose)
    }
    return nil
}

// Type checking
if IsValidationError(err) {
    // Handle validation-specific logic
}
```
