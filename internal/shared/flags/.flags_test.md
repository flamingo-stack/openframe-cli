This file contains comprehensive unit tests for a command-line flags management system that integrates with the Cobra CLI framework. It tests the functionality of common flags (verbose, dry-run, force) and their management across CLI commands.

## Key Components

- **TestCommonFlags**: Tests the CommonFlags struct initialization and field assignment
- **TestNewFlagManager**: Validates FlagManager creation and initialization
- **TestFlagManager_AddCommonFlags**: Tests adding common flags to Cobra commands with proper shorthand and descriptions
- **TestFlagManager_AddCommonFlags_Integration**: End-to-end testing of flag parsing and value setting
- **TestValidateCommonFlags**: Validates flag validation logic with various combinations
- **TestGetFlagDescription**: Tests flag description retrieval functionality
- **Edge case tests**: Handle nil values, multiple commands, and flag conflicts
- **Benchmark tests**: Performance testing for key operations

## Usage Example

```go
// Run all tests
go test ./flags

// Run specific test
go test -run TestCommonFlags ./flags

// Run benchmarks
go test -bench=. ./flags

// Test with coverage
go test -cover ./flags

// Run integration tests only
go test -run Integration ./flags
```

The tests ensure the flags package correctly handles CLI flag management, validation, and integration with Cobra commands, covering both happy path scenarios and edge cases like nil handling and flag conflicts.
