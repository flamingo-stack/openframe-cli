This file provides a centralized system for managing common command-line flags across the CLI application, ensuring consistent flag handling and descriptions throughout different commands.

## Key Components

- **`CommonFlags`** - Struct containing standard flags (Verbose, DryRun, Force) used across non-cluster commands
- **`FlagManager`** - Main manager struct that handles consistent flag setup and attachment to cobra commands
- **`NewFlagManager()`** - Constructor function that creates a new flag manager instance
- **`AddCommonFlags()`** - Method that attaches common flags to cobra commands with fallback handling
- **`ValidateCommonFlags()`** - Validation function for flag combinations (currently placeholder)
- **`GetFlagDescription()`** - Utility function providing standardized descriptions for common flags

## Usage Example

```go
// Create common flags structure
commonFlags := &flags.CommonFlags{}

// Initialize flag manager
flagManager := flags.NewFlagManager(commonFlags)

// Add flags to a cobra command
var myCmd = &cobra.Command{
    Use: "example",
    Run: func(cmd *cobra.Command, args []string) {
        if commonFlags.Verbose {
            fmt.Println("Verbose mode enabled")
        }
        if commonFlags.DryRun {
            fmt.Println("Dry run - no changes will be made")
        }
    },
}

// Attach common flags to the command
flagManager.AddCommonFlags(myCmd)

// Get standardized flag descriptions
description := flags.GetFlagDescription("verbose")
```

The system gracefully handles nil flag structures by creating placeholder flags, ensuring commands work even without explicit flag binding.
