A centralized flag management package that provides consistent command-line flag handling across the application using Cobra commands. It standardizes common flags like verbose, dry-run, and force options.

## Key Components

- **CommonFlags**: Struct containing standard boolean flags (Verbose, DryRun, Force)
- **FlagManager**: Manages consistent flag setup across different commands
- **NewFlagManager()**: Factory function to create a new flag manager instance
- **AddCommonFlags()**: Adds standard flags to any Cobra command with consistent naming and descriptions
- **ValidateCommonFlags()**: Placeholder for future flag validation logic
- **GetFlagDescription()**: Returns standardized descriptions for common flag types

## Usage Example

```go
import (
    "github.com/spf13/cobra"
    "your-app/pkg/flags"
)

// Create common flags structure
commonFlags := &flags.CommonFlags{}

// Initialize flag manager
flagManager := flags.NewFlagManager(commonFlags)

// Create a command and add common flags
cmd := &cobra.Command{
    Use: "example",
    Run: func(cmd *cobra.Command, args []string) {
        if commonFlags.Verbose {
            fmt.Println("Verbose mode enabled")
        }
        if commonFlags.DryRun {
            fmt.Println("Dry run - would execute command")
            return
        }
    },
}

// Add standard flags to the command
flagManager.AddCommonFlags(cmd)

// Validate flags if needed
if err := flags.ValidateCommonFlags(commonFlags); err != nil {
    return err
}
```
