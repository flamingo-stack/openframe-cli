This file provides a centralized system for managing common command-line flags across CLI commands using the Cobra framework. It ensures consistent flag behavior and descriptions throughout the application.

## Key Components

- **`CommonFlags`** - Struct containing standard flags (Verbose, DryRun, Force) for non-cluster commands
- **`FlagManager`** - Main struct that handles consistent flag setup across commands
- **`NewFlagManager()`** - Constructor that creates a new flag manager instance
- **`AddCommonFlags()`** - Method that adds the common flags to any Cobra command
- **`ValidateCommonFlags()`** - Validation function for flag combinations (currently a placeholder)
- **`GetFlagDescription()`** - Returns standardized descriptions for common flag types

## Usage Example

```go
package main

import (
    "github.com/spf13/cobra"
    "your-project/internal/flags"
)

func main() {
    // Create common flags structure
    commonFlags := &flags.CommonFlags{}
    
    // Initialize flag manager
    flagManager := flags.NewFlagManager(commonFlags)
    
    // Create a new command
    cmd := &cobra.Command{
        Use: "mycommand",
        Run: func(cmd *cobra.Command, args []string) {
            if commonFlags.Verbose {
                fmt.Println("Verbose mode enabled")
            }
            if commonFlags.DryRun {
                fmt.Println("Dry run - no changes will be made")
            }
        },
    }
    
    // Add common flags to the command
    flagManager.AddCommonFlags(cmd)
    
    // Execute command
    cmd.Execute()
}
```
