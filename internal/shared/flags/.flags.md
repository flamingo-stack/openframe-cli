This file provides a centralized flag management system for CLI commands, ensuring consistent flag behavior across the application.

## Key Components

- **CommonFlags**: Struct containing standard flags (Verbose, DryRun, Force) used across non-cluster commands
- **FlagManager**: Core manager that handles consistent flag setup and attachment to Cobra commands
- **NewFlagManager()**: Constructor function that creates a new flag manager instance
- **AddCommonFlags()**: Method that attaches common flags to any Cobra command with fallback support
- **ValidateCommonFlags()**: Validation function for flag combinations (currently a placeholder)
- **GetFlagDescription()**: Helper function that returns standardized descriptions for common flags

## Usage Example

```go
package main

import (
    "github.com/spf13/cobra"
    "your-app/internal/flags"
)

func main() {
    // Create common flags structure
    commonFlags := &flags.CommonFlags{}
    
    // Create flag manager
    flagManager := flags.NewFlagManager(commonFlags)
    
    // Create a command
    cmd := &cobra.Command{
        Use:   "deploy",
        Short: "Deploy application",
        Run: func(cmd *cobra.Command, args []string) {
            if commonFlags.Verbose {
                fmt.Println("Verbose mode enabled")
            }
            if commonFlags.DryRun {
                fmt.Println("Dry run - would deploy application")
                return
            }
            // Actual deployment logic
        },
    }
    
    // Add common flags to command
    flagManager.AddCommonFlags(cmd)
    
    // Validate flags before execution
    if err := flags.ValidateCommonFlags(commonFlags); err != nil {
        log.Fatal(err)
    }
}
```
