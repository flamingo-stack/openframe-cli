A centralized flag management system for CLI commands using Cobra, providing consistent flag handling and descriptions across the application.

## Key Components

- **`CommonFlags`** - Struct containing standard CLI flags (Verbose, DryRun, Force)
- **`FlagManager`** - Main type for managing flag setup and binding to Cobra commands
- **`NewFlagManager()`** - Constructor for creating flag managers with common flags
- **`AddCommonFlags()`** - Method to add standard flags to any Cobra command
- **`ValidateCommonFlags()`** - Validation function for flag combinations
- **`GetFlagDescription()`** - Centralized flag description lookup

## Usage Example

```go
// Create common flags struct
commonFlags := &CommonFlags{}

// Initialize flag manager
flagManager := NewFlagManager(commonFlags)

// Add flags to a Cobra command
var myCmd = &cobra.Command{
    Use:   "example",
    Short: "Example command",
    RunE: func(cmd *cobra.Command, args []string) error {
        // Access flags through commonFlags
        if commonFlags.Verbose {
            fmt.Println("Verbose mode enabled")
        }
        
        if commonFlags.DryRun {
            fmt.Println("Dry run mode - no changes will be made")
            return nil
        }
        
        return nil
    },
}

// Apply common flags
flagManager.AddCommonFlags(myCmd)

// Get standard descriptions
desc := GetFlagDescription("verbose")
// Returns: "Enable verbose output with detailed information"
```
