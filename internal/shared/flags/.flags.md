This package provides a centralized system for managing common command-line flags across a CLI application using the Cobra library. It ensures consistent flag behavior and descriptions throughout different commands.

## Key Components

- **`CommonFlags`** - Struct containing standard flags (Verbose, DryRun, Force) used across non-cluster commands
- **`FlagManager`** - Central handler for setting up flags consistently across commands
- **`NewFlagManager()`** - Constructor for creating a new flag manager instance
- **`AddCommonFlags()`** - Adds standard flags to any Cobra command with consistent naming and shortcuts
- **`ValidateCommonFlags()`** - Validates flag combinations (placeholder for future validation logic)
- **`GetFlagDescription()`** - Returns standardized descriptions for common flag names

## Usage Example

```go
package main

import (
    "github.com/spf13/cobra"
    "yourapp/flags"
)

func main() {
    // Create common flags struct
    commonFlags := &flags.CommonFlags{}
    
    // Create flag manager
    flagManager := flags.NewFlagManager(commonFlags)
    
    // Create a command
    cmd := &cobra.Command{
        Use: "deploy",
        Run: func(cmd *cobra.Command, args []string) {
            if commonFlags.Verbose {
                fmt.Println("Verbose mode enabled")
            }
            if commonFlags.DryRun {
                fmt.Println("Dry run mode - no changes will be made")
            }
        },
    }
    
    // Add common flags to the command
    flagManager.AddCommonFlags(cmd)
    
    // Execute command
    cmd.Execute()
}
```
