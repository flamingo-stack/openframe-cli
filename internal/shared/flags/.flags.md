This file provides a centralized flag management system for CLI commands using the Cobra framework, ensuring consistent flag handling across non-cluster commands.

## Key Components

- **`CommonFlags`** - Struct containing standard flags (Verbose, DryRun, Force) used across commands
- **`FlagManager`** - Main struct that handles consistent flag setup for Cobra commands
- **`NewFlagManager()`** - Constructor that creates a new flag manager instance
- **`AddCommonFlags()`** - Method that adds standard flags to any Cobra command
- **`ValidateCommonFlags()`** - Validation function for flag combinations (placeholder for future logic)
- **`GetFlagDescription()`** - Helper that returns standardized descriptions for common flags

## Usage Example

```go
package main

import (
    "github.com/spf13/cobra"
    "your-project/flags"
)

func main() {
    commonFlags := &flags.CommonFlags{}
    flagManager := flags.NewFlagManager(commonFlags)
    
    cmd := &cobra.Command{
        Use: "example",
        Run: func(cmd *cobra.Command, args []string) {
            if commonFlags.Verbose {
                fmt.Println("Verbose mode enabled")
            }
            if commonFlags.DryRun {
                fmt.Println("Dry run mode - no changes made")
                return
            }
        },
    }
    
    flagManager.AddCommonFlags(cmd)
    cmd.Execute()
}
```

The flag manager automatically adds `-v/--verbose`, `-f/--force`, and `--dry-run` flags with consistent descriptions and behavior.
