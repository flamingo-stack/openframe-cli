This file contains comprehensive unit tests for the file cleanup functionality, testing temporary file management, backup/restore operations, and different cleanup modes.

## Key Components

- **TestRegisterTempFile_NoBackupCreated**: Tests that temporary files are registered without creating backup files
- **TestRegisterTempFile_MultipleFiles**: Verifies handling of multiple temporary files simultaneously
- **TestFileCleanup_CleanupOnSuccessOnly_Behavior**: Tests conditional cleanup based on operation success
- **TestFileCleanup_RegularMode_AlwaysCleanup**: Ensures regular mode always performs cleanup regardless of success
- **TestFileCleanup_SignalInterruption_Scenario**: Simulates CTRL-C interruption handling
- **TestFileCleanup_BackupFile_MemoryOnly**: Tests in-memory backup functionality
- **TestFileCleanup_BackupFile_PhysicalBackup**: Tests physical backup file creation and restoration

## Usage Example

```go
func TestCustomCleanup(t *testing.T) {
    cleanup := NewFileCleanup()
    tmpDir := t.TempDir()
    testFile := filepath.Join(tmpDir, "test.yaml")
    
    // Create and register temp file
    os.WriteFile(testFile, []byte("content"), 0644)
    cleanup.RegisterTempFile(testFile)
    
    // Verify file exists before cleanup
    assert.FileExists(t, testFile)
    
    // Perform cleanup
    cleanup.RestoreFiles(false)
    
    // Verify file was removed
    assert.NoFileExists(t, testFile)
}
```
