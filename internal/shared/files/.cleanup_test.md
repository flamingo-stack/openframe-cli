This test file contains comprehensive unit tests for a file cleanup utility that manages temporary files and file backups with restoration capabilities.

## Key Components

- **TestRegisterTempFile_NoBackupCreated**: Tests that temporary files are registered without creating backup files and are properly cleaned up
- **TestRegisterTempFile_MultipleFiles**: Validates handling of multiple temporary files simultaneously
- **TestFileCleanup_CleanupOnSuccessOnly_Behavior**: Tests conditional cleanup mode that only removes files on successful operations
- **TestFileCleanup_RegularMode_AlwaysCleanup**: Verifies default behavior that always cleans up files regardless of success/failure
- **TestFileCleanup_SignalInterruption_Scenario**: Tests cleanup behavior during CTRL-C interruption scenarios
- **TestFileCleanup_BackupFile_MemoryOnly**: Tests in-memory file backup without creating physical backup files
- **TestFileCleanup_BackupFile_PhysicalBackup**: Tests physical file backup creation and restoration

## Usage Example

```go
func TestMyCleanupScenario(t *testing.T) {
    cleanup := NewFileCleanup()
    
    // Register temporary file for cleanup
    tempFile := "/tmp/myfile.txt"
    os.WriteFile(tempFile, []byte("content"), 0644)
    err := cleanup.RegisterTempFile(tempFile)
    assert.NoError(t, err)
    
    // Enable success-only cleanup mode
    cleanup.SetCleanupOnSuccessOnly(true)
    
    // Cleanup only on success
    err = cleanup.RestoreFilesWithResult(false, true)
    assert.NoError(t, err)
    assert.NoFileExists(t, tempFile)
}
```
