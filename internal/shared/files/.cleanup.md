Handles backup and restoration of files modified by the OpenFrame CLI, providing automatic cleanup capabilities for temporary and generated files.

## Key Components

**Constants:**
- `CLI_GENERATED_MARKER` - Comment marker for CLI-generated files
- `CLI_GENERATED_TIMESTAMP_PREFIX` - Timestamp tracking prefix  
- `BACKUP_SUFFIX` - Suffix for backup files

**Types:**
- `FileBackup` - Represents a single file backup operation with original/backup paths and content
- `FileCleanup` - Main cleanup manager handling multiple file backups and restoration

**Key Methods:**
- `NewFileCleanup()` - Creates a new cleanup manager
- `BackupFile()` - Creates backup (memory or disk-based) before modification
- `RegisterTempFile()` - Registers temporary files for deletion
- `RestoreFiles()` - Restores all files to original state
- `AddCLIMarkerToFile()` - Adds generation markers to files
- `GetSafeFileName()` - Creates timestamped temporary filenames

## Usage Example

```go
// Create cleanup manager and backup files before modification
cleanup := NewFileCleanup()
defer cleanup.RestoreFiles(true) // Ensure cleanup on exit

// Backup existing file before modifying
if err := cleanup.BackupFile("config.yaml", false); err != nil {
    return err
}

// Register temporary file for cleanup
tempFile := GetSafeFileName("temp-config.yaml")
if err := cleanup.RegisterTempFile(tempFile); err != nil {
    return err
}

// Add CLI marker to generated content
content := AddCLIMarkerToFile(tempFile, "generated config content...")

// On success, clean up temporary files
cleanup.RestoreFilesOnSuccess(true)
```
