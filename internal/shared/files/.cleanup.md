Provides file backup, cleanup, and restoration utilities for managing CLI-generated files during operations. This package ensures clean rollback capabilities for file modifications made by the OpenFrame CLI.

## Key Components

- **FileCleanup**: Main cleanup manager that tracks and restores file backups
- **FileBackup**: Represents a backup operation with original/backup paths and content storage
- **Constants**: CLI generation markers, timestamp prefixes, and backup file suffixes
- **Helper Functions**: File marker addition, safe filename generation, and file copying utilities

## Usage Example

```go
// Create cleanup manager
cleanup := NewFileCleanup()

// Backup existing file before modification
err := cleanup.BackupFile("app.js", false) // false = create physical backup
if err != nil {
    log.Fatal(err)
}

// Register a temporary file we're creating
err = cleanup.RegisterTempFile("temp-config.json")
if err != nil {
    log.Fatal(err)
}

// Configure to only cleanup on success
cleanup.SetCleanupOnSuccessOnly(true)

// ... perform operations ...

// Restore files based on operation result
if operationSuccessful {
    cleanup.RestoreFilesOnSuccess(true) // Clean up temp files
} else {
    cleanup.RestoreFiles(true) // Restore all backups
}

// Add CLI markers to generated files
markedContent := AddCLIMarkerToFile("config.js", originalContent)
```

The cleanup system supports both memory-only backups for small files and physical backup files for larger content, with automatic restoration and temporary file management.
