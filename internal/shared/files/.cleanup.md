This module provides file backup and cleanup functionality for the OpenFrame CLI, handling temporary files and ensuring proper restoration after operations complete or fail.

## Key Components

- **FileCleanup**: Main struct managing backup and restore operations
- **FileBackup**: Represents a single file backup with original/backup paths and content
- **Constants**: Markers for CLI-generated files (`CLI_GENERATED_MARKER`, `BACKUP_SUFFIX`)
- **AddCLIMarkerToFile()**: Adds generation markers and timestamps to files
- **GetSafeFileName()**: Creates timestamped temporary filenames

## Usage Example

```go
// Create cleanup manager
cleanup := files.NewFileCleanup()

// Backup existing file before modification
err := cleanup.BackupFile("config.yaml", false) // false = use physical backup
if err != nil {
    return fmt.Errorf("backup failed: %w", err)
}

// Register a temporary file for cleanup
err = cleanup.RegisterTempFile("temp-script.sh")
if err != nil {
    return fmt.Errorf("registration failed: %w", err)
}

// Add CLI marker to generated content
content := files.AddCLIMarkerToFile("script.sh", originalScript)

// Restore files after operation
if operationSucceeded {
    cleanup.RestoreFilesOnSuccess(true) // verbose output
} else {
    cleanup.RestoreFiles(true) // restore on failure
}

// Create safe temporary filename
tempFile := files.GetSafeFileName("config.yaml")
// Results in: "config-generated-20241201-143025.yaml"
```

The module supports both memory-only and physical file backups, handles cleanup on success/failure scenarios, and provides utilities for marking CLI-generated files with timestamps and warnings.
