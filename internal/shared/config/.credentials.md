Provides a unified interface for prompting and managing different types of authentication credentials across various services in the CLI application.

## Key Components

- **CredentialsPrompter** - Main struct that handles credential prompting across different contexts
- **GitHubCredentials** - Stores GitHub username and personal access token
- **DockerCredentials** - Contains Docker registry authentication details
- **GenericCredentials** - Basic username/password credentials for general use
- **APIKeyCredentials** - API key and secret pairs for service authentication
- **CredentialsOptions** - Configuration options for customizing credential prompts

## Usage Example

```go
package main

import (
    "log"
    "github.com/flamingo-stack/openframe-cli/internal/config"
)

func main() {
    prompter := config.NewCredentialsPrompter()
    
    // Prompt for GitHub credentials
    githubCreds, err := prompter.PromptForGitHubCredentials("https://github.com/private/repo")
    if err != nil {
        log.Fatal(err)
    }
    
    // Prompt for Docker registry credentials
    dockerCreds, err := prompter.PromptForDockerCredentials("registry.example.com")
    if err != nil {
        log.Fatal(err)
    }
    
    // Validate existing credentials
    if prompter.IsCredentialsRequired(username, password) {
        genericCreds, err := prompter.PromptForGenericCredentials("MyService")
        if err != nil {
            log.Fatal(err)
        }
        
        // Validate the credentials
        err = prompter.ValidateCredentials(genericCreds.Username, genericCreds.Password)
        if err != nil {
            log.Fatal(err)
        }
    }
    
    // Custom prompting with options
    opts := config.CredentialsOptions{
        AllowEmpty:      false,
        DefaultUsername: "admin",
        CustomMessage:   "Enter your admin credentials:",
    }
    
    customCreds, err := prompter.PromptWithOptions("Admin Panel", opts)
    if err != nil {
        log.Fatal(err)
    }
}
```
