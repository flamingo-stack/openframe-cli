Handles interactive credential collection for various authentication contexts including GitHub, Docker registries, and API services.

## Key Components

- **CredentialsPrompter**: Main handler for prompting credentials across different contexts
- **GitHubCredentials**: Username and personal access token for GitHub authentication
- **DockerCredentials**: Registry authentication with username/password
- **GenericCredentials**: Basic username/password credentials for general services
- **APIKeyCredentials**: API key and secret pair for API-based authentication
- **CredentialsOptions**: Configuration options for customizing credential prompting behavior

## Core Methods

- `PromptForGitHubCredentials()`: Prompts for GitHub username and token for private repository access
- `PromptForDockerCredentials()`: Collects Docker registry authentication credentials
- `PromptForGenericCredentials()`: Generic username/password collection for any service
- `PromptForAPIKeyCredentials()`: API key and secret collection for API services
- `PromptWithOptions()`: Flexible credential prompting with custom options
- `ValidateCredentials()`: Validates credential format and minimum requirements
- `IsCredentialsRequired()`: Checks if credentials need to be prompted based on existing values

## Usage Example

```go
// Create a credentials prompter
prompter := config.NewCredentialsPrompter()

// Prompt for GitHub credentials
githubCreds, err := prompter.PromptForGitHubCredentials("https://github.com/private/repo")
if err != nil {
    log.Fatal(err)
}

// Prompt for Docker registry credentials
dockerCreds, err := prompter.PromptForDockerCredentials("docker.io")
if err != nil {
    log.Fatal(err)
}

// Custom credential prompting with options
opts := config.CredentialsOptions{
    AllowEmpty:      true,
    DefaultUsername: "admin",
    CustomMessage:   "Please login to continue",
}
creds, err := prompter.PromptWithOptions("MyService", opts)
if err != nil {
    log.Fatal(err)
}

// Validate existing credentials
err = prompter.ValidateCredentials("user123", "pass456")
if err != nil {
    log.Printf("Invalid credentials: %v", err)
}
```
