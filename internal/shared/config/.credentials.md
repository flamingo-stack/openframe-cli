This file provides a unified credentials prompting system for the OpenFrame CLI, handling authentication across different services like GitHub, Docker registries, and generic APIs with interactive user input validation.

## Key Components

- **CredentialsPrompter**: Main struct that orchestrates credential collection with validation
- **Credential Types**: 
  - `GitHubCredentials` - GitHub username and personal access token
  - `DockerCredentials` - Docker registry authentication 
  - `GenericCredentials` - Basic username/password pairs
  - `APIKeyCredentials` - API key and secret combinations
- **CredentialsOptions**: Customization struct for flexible prompting behavior
- **Validation Methods**: Input validation and requirement checking functions

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/flamingo-stack/openframe-cli/internal/config"
)

func main() {
    prompter := config.NewCredentialsPrompter()
    
    // Prompt for GitHub credentials
    githubCreds, err := prompter.PromptForGitHubCredentials("https://github.com/private/repo")
    if err != nil {
        fmt.Printf("Error: %v\n", err)
        return
    }
    
    // Prompt for Docker credentials
    dockerCreds, err := prompter.PromptForDockerCredentials("docker.io")
    if err != nil {
        fmt.Printf("Error: %v\n", err)
        return
    }
    
    // Custom prompting with options
    opts := config.CredentialsOptions{
        AllowEmpty:      false,
        DefaultUsername: "admin",
        CustomMessage:   "Login to staging environment",
    }
    
    creds, err := prompter.PromptWithOptions("staging-api", opts)
    if err != nil {
        fmt.Printf("Error: %v\n", err)
        return
    }
    
    // Validate existing credentials
    err = prompter.ValidateCredentials(creds.Username, creds.Password)
    if err != nil {
        fmt.Printf("Invalid credentials: %v\n", err)
        return
    }
}
```
