This file provides system-level configuration management for the OpenFrame deployment system, handling initialization tasks like logging directory setup and system-wide configuration.

## Key Components

- **`SystemService`** - Main service struct that manages system configuration and log directory
- **`NewSystemService()`** - Constructor that creates a service with default temp directory logging
- **`NewSystemServiceWithOptions()`** - Constructor allowing custom log directory specification
- **`Initialize()`** - Performs system initialization including directory setup
- **`GetLogDirectory()`** - Returns the configured log directory path

## Usage Example

```go
// Create service with default configuration
sysService := NewSystemService()

// Or create with custom log directory
sysService := NewSystemServiceWithOptions("/var/log/openframe")

// Initialize the system
if err := sysService.Initialize(); err != nil {
    log.Fatalf("System initialization failed: %v", err)
}

// Get the log directory path for other components
logDir := sysService.GetLogDirectory()
fmt.Printf("Logs will be written to: %s\n", logDir)
```

The service automatically creates the log directory with 0755 permissions during initialization, making it ready for logging operations across the application.
