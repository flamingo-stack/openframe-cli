Provides system-level configuration and initialization services for the OpenFrame deployment system, handling CLI setup, logging directories, and system-wide settings.

## Key Components

- **SystemService** - Main service struct that manages system-level configuration
- **NewSystemService()** - Constructor that creates a service with default log directory in system temp
- **NewSystemServiceWithOptions()** - Constructor that allows custom log directory configuration
- **Initialize()** - Performs system initialization tasks including directory setup
- **GetLogDirectory()** - Getter for the configured log directory path

## Usage Example

```go
// Create system service with default configuration
systemSvc := config.NewSystemService()

// Or with custom log directory
systemSvc := config.NewSystemServiceWithOptions("/var/log/openframe")

// Initialize the system (creates log directories)
if err := systemSvc.Initialize(); err != nil {
    log.Fatalf("System initialization failed: %v", err)
}

// Get the log directory for use by other components
logDir := systemSvc.GetLogDirectory()
fmt.Printf("Logs will be written to: %s\n", logDir)
```

The service ensures proper directory structure exists before other components attempt to write logs, making it a foundational component for system startup.
