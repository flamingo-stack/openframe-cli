Provides system-level configuration and initialization services for CLI setup, logging directories, and system-wide settings management.

## Key Components

- **SystemService**: Main struct that manages system initialization and configuration
- **NewSystemService()**: Constructor that creates a service instance with default temp directory logging
- **NewSystemServiceWithOptions()**: Constructor allowing custom log directory specification
- **Initialize()**: Performs system setup tasks including directory creation
- **GetLogDirectory()**: Returns the configured logging directory path

## Usage Example

```go
// Create with default configuration
sysService := NewSystemService()

// Or create with custom log directory
customSysService := NewSystemServiceWithOptions("/var/log/openframe")

// Initialize the system (creates directories, etc.)
if err := sysService.Initialize(); err != nil {
    log.Fatalf("System initialization failed: %v", err)
}

// Get the configured log directory
logDir := sysService.GetLogDirectory()
fmt.Printf("Logs will be written to: %s\n", logDir)
```

The service automatically creates the logging directory structure during initialization with appropriate permissions (0755), defaulting to the system temp directory under "openframe-deployment-logs" unless overridden.
