Provides interfaces and base implementations for building command adapters with the Cobra CLI framework, including flag management, validation, and command execution patterns.

## Key Components

**Core Interfaces:**
- `CommandHandler` - Interface for command execution logic
- `FlagsProvider` - Interface for defining and adding command flags
- `CommandAdapter` - Combined interface for complete command handling

**Base Implementation:**
- `BaseCommandAdapter` - Common functionality including flag extraction and validation
- `FlagDefinition` - Structure for defining command flags with metadata

**Utility Types:**
- `CommandMetadata` - Complete command configuration (usage, description, examples)
- `ExampleBuilder` - Helper for building formatted command examples
- `PreRunEChain` - Chainable pre-execution functions
- `RequiredFlagError`, `InvalidFlagError` - Structured error types

## Usage Example

```go
// Define a command adapter
type MyCommandAdapter struct {
    *BaseCommandAdapter
}

func (a *MyCommandAdapter) Execute(cmd *cobra.Command, args []string) error {
    flags := a.ExtractFlags(cmd)
    if result := a.ValidateRequired(flags.GetAll()); !result.IsValid() {
        return result.FirstError()
    }
    // Command logic here
    return nil
}

func (a *MyCommandAdapter) GetFlagDefinitions() []FlagDefinition {
    return []FlagDefinition{
        {Name: "output", Shorthand: "o", DefaultValue: "", Usage: "Output file", Required: true},
    }
}

// Build examples and pre-run chain
examples := NewExampleBuilder().
    Add("Generate output", "--output result.txt").
    Build("mycommand")

preRun := NewPreRunEChain().
    Add(validateConfig).
    Add(checkPermissions).
    Build()
```
