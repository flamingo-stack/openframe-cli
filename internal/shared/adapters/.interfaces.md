Defines the core interfaces and base types for building command adapters in the Cobra CLI framework, providing a structured approach to command handling, flag management, and validation.

## Key Components

**Core Interfaces:**
- `CommandHandler` - Interface for command execution logic
- `FlagsProvider` - Interface for flag definition and registration
- `CommandAdapter` - Combined interface extending both handler and provider

**Base Implementation:**
- `BaseCommandAdapter` - Common functionality with flag extraction and validation
- `FlagDefinition` - Structure defining command flag properties
- `CommandMetadata` - Container for command metadata (usage, description, examples)

**Utility Types:**
- `ExampleBuilder` - Helper for building formatted command examples
- `PreRunEChain` - Chainable pre-execution validation functions
- `RequiredFlagError` / `InvalidFlagError` - Specific error types for flag validation

## Usage Example

```go
// Implement a custom command adapter
type MyCommandAdapter struct {
    *BaseCommandAdapter
}

func (a *MyCommandAdapter) Execute(cmd *cobra.Command, args []string) error {
    // Extract and validate flags
    extractor := a.ExtractFlags(cmd)
    flags := extractor.GetAllFlags()
    
    if result := a.ValidateRequired(flags); !result.IsValid() {
        return result.GetFirstError()
    }
    
    // Command logic here
    return nil
}

func (a *MyCommandAdapter) GetFlagDefinitions() []FlagDefinition {
    return []FlagDefinition{
        {Name: "output", Shorthand: "o", DefaultValue: "", Usage: "Output file", Required: true},
    }
}

// Build command examples
examples := NewExampleBuilder().
    Add("Basic usage", "--output file.txt").
    Add("With verbose", "--output file.txt --verbose").
    Build("mycommand")
```
