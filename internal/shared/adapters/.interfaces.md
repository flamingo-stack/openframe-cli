Defines a comprehensive set of interfaces and utilities for building CLI command adapters in a Go application using the Cobra framework. This package provides abstractions for command execution, flag management, validation, and metadata handling.

## Key Components

- **CommandHandler** - Interface for executing commands with Cobra
- **FlagsProvider** - Interface for defining and adding command flags
- **CommandAdapter** - Combined interface merging command handling and flag management
- **BaseCommandAdapter** - Concrete implementation providing common adapter functionality
- **FlagDefinition** - Structure defining command flag properties and requirements
- **ValidationResult** - Error handling for flag validation with specific error types
- **ExampleBuilder** - Utility for constructing formatted command usage examples
- **PreRunEChain** - Chain multiple pre-execution functions for command setup

## Usage Example

```go
// Create a command adapter
type MyCommandAdapter struct {
    *BaseCommandAdapter
}

func (m *MyCommandAdapter) GetFlagDefinitions() []FlagDefinition {
    return []FlagDefinition{
        {
            Name:         "name",
            Shorthand:    "n",
            DefaultValue: "",
            Usage:        "Name of the resource",
            Required:     true,
        },
    }
}

func (m *MyCommandAdapter) Execute(cmd *cobra.Command, args []string) error {
    extractor := m.ExtractFlags(cmd)
    name := extractor.GetString("name")
    
    // Validate required flags
    flags := map[string]interface{}{"name": name}
    if result := m.ValidateRequired(flags); !result.IsValid() {
        return result.FirstError()
    }
    
    // Execute command logic
    return nil
}

// Build examples for help text
examples := NewExampleBuilder().
    Add("Create with name", "--name myapp").
    Add("Create with alias", "-n myapp").
    Build("mycommand")
```
