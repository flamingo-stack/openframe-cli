This file defines interfaces and utilities for building Cobra command adapters with standardized flag handling, validation, and command structure management.

## Key Components

**Core Interfaces:**
- `CommandHandler` - Interface for command execution
- `FlagsProvider` - Interface for command flag management
- `CommandAdapter` - Combined interface extending both handler and provider

**Base Implementation:**
- `BaseCommandAdapter` - Common functionality for command adapters with flag extraction and validation

**Utilities:**
- `FlagDefinition` - Structure for defining command flags
- `CommandMetadata` - Command metadata container
- `ExampleBuilder` - Helper for building formatted command examples
- `PreRunEChain` - Utility for chaining multiple PreRunE functions

**Error Types:**
- `RequiredFlagError` - Missing required flag error
- `InvalidFlagError` - Invalid flag value error

## Usage Example

```go
// Create a command adapter
type MyCommandAdapter struct {
    *BaseCommandAdapter
}

func NewMyCommandAdapter() *MyCommandAdapter {
    return &MyCommandAdapter{
        BaseCommandAdapter: NewBaseCommandAdapter(),
    }
}

func (a *MyCommandAdapter) Execute(cmd *cobra.Command, args []string) error {
    flags := a.ExtractFlags(cmd)
    
    // Validate required flags
    flagValues := map[string]interface{}{
        "name": flags.GetString("name"),
        "count": flags.GetInt("count"),
    }
    
    if result := a.ValidateRequired(flagValues); !result.IsValid() {
        return result.GetFirstError()
    }
    
    // Command logic here
    return nil
}

// Build command examples
examples := NewExampleBuilder().
    Add("Create item", "create --name myitem").
    Add("Create with count", "create --name myitem --count 5").
    Build("mycommand")
```
