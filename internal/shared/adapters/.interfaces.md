This file defines interfaces and supporting types for building command line applications using the Cobra framework, providing a structured approach to command handling, flag management, and validation.

## Key Components

**Interfaces:**
- `CommandHandler` - Defines command execution contract
- `FlagsProvider` - Manages flag definitions and registration
- `CommandAdapter` - Combines command handling and flag management

**Core Types:**
- `BaseCommandAdapter` - Base implementation with flag extraction and validation
- `FlagDefinition` - Represents command flag metadata
- `CommandMetadata` - Contains command information (usage, description, examples)

**Utilities:**
- `ExampleBuilder` - Helps construct formatted command examples
- `PreRunEChain` - Chains multiple pre-execution functions
- `RequiredFlagError`/`InvalidFlagError` - Custom error types for validation

## Usage Example

```go
// Create a command adapter
type MyCommandAdapter struct {
    *BaseCommandAdapter
}

func (m *MyCommandAdapter) Execute(cmd *cobra.Command, args []string) error {
    flags := m.ExtractFlags(cmd)
    
    // Validate required flags
    flagValues := map[string]interface{}{
        "output": flags.GetString("output"),
        "count": flags.GetInt("count"),
    }
    
    if result := m.ValidateRequired(flagValues); !result.IsValid() {
        return result.Errors[0]
    }
    
    // Execute command logic
    return nil
}

// Build examples
examples := NewExampleBuilder().
    Add("basic usage", "command --output file.txt").
    Add("with count", "command --output file.txt --count 5").
    Build("myapp")
```
