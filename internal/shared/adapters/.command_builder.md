A utility package that provides fluent interfaces for building and working with Cobra CLI commands, including command construction, flag extraction, and validation result handling.

## Key Components

- **CommandBuilder**: Fluent interface for constructing cobra.Command instances with method chaining
- **FlagExtractor**: Safe utilities for extracting flag values from commands with error handling
- **ValidationResult**: Structure for collecting and managing validation errors

## Usage Example

```go
// Build a command with fluent interface
cmd := NewCommandBuilder("server", "Start the server").
    Long("Starts the HTTP server on the specified port").
    Aliases([]string{"serve", "start"}).
    AddStringFlag("port", "p", "8080", "Server port").
    AddBoolFlag("debug", "d", false, "Enable debug mode").
    RunE(func(cmd *cobra.Command, args []string) error {
        // Extract flags safely
        extractor := NewFlagExtractor(cmd)
        port, _ := extractor.GetString("port")
        debug, _ := extractor.GetBool("debug")
        
        // Validation example
        result := NewValidationResult()
        if port == "" {
            result.AddError(fmt.Errorf("port cannot be empty"))
        }
        
        if result.HasErrors() {
            return result.GetFirstError()
        }
        
        // Server logic here
        return nil
    }).
    Build()
```
