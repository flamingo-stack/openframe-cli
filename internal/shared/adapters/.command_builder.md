Provides utilities for building and working with Cobra CLI commands using fluent interfaces and type-safe flag extraction.

## Key Components

### CommandBuilder
- **NewCommandBuilder()** - Creates a fluent builder for cobra commands
- **Long()**, **Aliases()**, **Args()** - Sets command metadata and validation
- **PreRunE()**, **RunE()**, **PersistentPreRunE()** - Configures command execution hooks
- **AddBoolFlag()**, **AddStringFlag()**, **AddIntFlag()** - Adds typed flags with optional shorthand
- **AddSubcommand()** - Adds nested commands
- **Build()** - Returns the constructed cobra.Command

### FlagExtractor
- **NewFlagExtractor()** - Creates a safe flag value extractor
- **GetBool()**, **GetString()**, **GetInt()** - Type-safe flag value retrieval
- **GetRootBool()**, **GetRootString()** - Extracts persistent flags from root command
- **FlagChanged()** - Checks if flag was explicitly set by user

### ValidationResult
- **NewValidationResult()** - Creates validation state tracker
- **AddError()** - Records validation errors and marks as invalid
- **HasErrors()**, **GetFirstError()** - Query validation state

## Usage Example

```go
// Build a command with flags
cmd := NewCommandBuilder("serve", "Start the server").
    Long("Starts the HTTP server on the specified port").
    AddStringFlag("host", "h", "localhost", "Server host").
    AddIntFlag("port", "p", 8080, "Server port").
    AddBoolFlag("verbose", "v", false, "Enable verbose logging").
    RunE(func(cmd *cobra.Command, args []string) error {
        // Extract flags safely
        extractor := NewFlagExtractor(cmd)
        host, _ := extractor.GetString("host")
        port, _ := extractor.GetInt("port")
        verbose, _ := extractor.GetBool("verbose")
        
        // Start server with extracted values
        return startServer(host, port, verbose)
    }).
    Build()

// Validate flags
result := NewValidationResult()
if port < 1 || port > 65535 {
    result.AddError(errors.New("port must be between 1 and 65535"))
}
```
