Utility functions for the UI package that provide common formatting and display helpers for user interface components.

## Key Components

- **FormatAge(createdAt time.Time) string** - Converts a timestamp into a human-readable age string showing time elapsed since creation

## Usage Example

```go
package main

import (
    "fmt"
    "time"
    "your-project/ui"
)

func main() {
    // Format various timestamps
    now := time.Now()
    
    // Recent timestamp (30 seconds ago)
    recent := now.Add(-30 * time.Second)
    fmt.Println(ui.FormatAge(recent)) // Output: "30s"
    
    // Older timestamp (2 hours ago)
    older := now.Add(-2 * time.Hour)
    fmt.Println(ui.FormatAge(older)) // Output: "2h"
    
    // Very old timestamp (5 days ago)
    ancient := now.Add(-5 * 24 * time.Hour)
    fmt.Println(ui.FormatAge(ancient)) // Output: "5d"
    
    // Zero time
    var zero time.Time
    fmt.Println(ui.FormatAge(zero)) // Output: "unknown"
}
```

The function automatically selects the most appropriate time unit (days, hours, minutes, or seconds) and handles edge cases like zero timestamps gracefully.
