Utility functions for the UI package that provide common formatting and helper operations for user interface components.

## Key Components

- **FormatAge** - Converts a timestamp into a human-readable relative age string (e.g., "2d", "5h", "30m", "45s")

## Usage Example

```go
package main

import (
    "fmt"
    "time"
    "your-project/ui"
)

func main() {
    // Format recent timestamps
    now := time.Now()
    oneHourAgo := now.Add(-1 * time.Hour)
    twoDaysAgo := now.Add(-48 * time.Hour)
    
    fmt.Println(ui.FormatAge(oneHourAgo))  // Output: "1h"
    fmt.Println(ui.FormatAge(twoDaysAgo))  // Output: "2d"
    
    // Handle zero/invalid timestamps
    var zeroTime time.Time
    fmt.Println(ui.FormatAge(zeroTime))    // Output: "unknown"
}
```

The `FormatAge` function prioritizes the largest time unit (days > hours > minutes > seconds) and returns a concise format suitable for displaying resource ages, timestamps, or durations in terminal interfaces and status displays.
