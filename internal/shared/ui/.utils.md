Utility functions for the UI package that provide common formatting and display helpers for user interface components.

## Key Components

**Functions:**
- `FormatAge(time.Time) string` - Converts a timestamp into a human-readable relative age string (e.g., "2d", "5h", "30m", "45s")

## Usage Example

```go
package main

import (
    "fmt"
    "time"
    "your-project/ui"
)

func main() {
    // Format recent timestamp
    recent := time.Now().Add(-2 * time.Hour)
    fmt.Println(ui.FormatAge(recent)) // Output: "2h"
    
    // Format older timestamp
    old := time.Now().Add(-3 * 24 * time.Hour)
    fmt.Println(ui.FormatAge(old)) // Output: "3d"
    
    // Handle zero time
    var zero time.Time
    fmt.Println(ui.FormatAge(zero)) // Output: "unknown"
    
    // Format very recent timestamp
    justNow := time.Now().Add(-30 * time.Second)
    fmt.Println(ui.FormatAge(justNow)) // Output: "30s"
}
```

The `FormatAge` function is particularly useful for displaying timestamps in dashboards, lists, or logs where you need concise, at-a-glance age information. It automatically selects the most appropriate unit (days, hours, minutes, or seconds) and handles edge cases like zero timestamps gracefully.
