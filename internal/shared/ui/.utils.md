This file provides utility functions for formatting time-related data in user interfaces, specifically converting time durations into human-readable age strings.

## Key Components

- **FormatAge(createdAt time.Time) string** - Converts a timestamp into a concise, human-readable age format (e.g., "2d", "5h", "30m", "45s")

## Usage Example

```go
package main

import (
    "fmt"
    "time"
    "your-project/ui"
)

func main() {
    // Format various timestamps
    now := time.Now()
    
    // 2 days ago
    twoDaysAgo := now.Add(-48 * time.Hour)
    fmt.Println(ui.FormatAge(twoDaysAgo)) // Output: "2d"
    
    // 3 hours ago
    threeHoursAgo := now.Add(-3 * time.Hour)
    fmt.Println(ui.FormatAge(threeHoursAgo)) // Output: "3h"
    
    // 30 minutes ago
    thirtyMinutesAgo := now.Add(-30 * time.Minute)
    fmt.Println(ui.FormatAge(thirtyMinutesAgo)) // Output: "30m"
    
    // Handle zero time
    var zeroTime time.Time
    fmt.Println(ui.FormatAge(zeroTime)) // Output: "unknown"
}
```

The function prioritizes larger time units and returns the most significant unit only, making it ideal for displaying compact timestamps in lists, tables, or status displays where space is limited.
