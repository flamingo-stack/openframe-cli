Provides utility functions for formatting time and duration values in user interfaces, particularly for displaying human-readable "age" information for resources or objects.

## Key Components

- **FormatAge**: Converts a timestamp into a compact, human-readable duration string (e.g., "3d", "2h", "45m", "12s")

## Usage Example

```go
package main

import (
    "fmt"
    "time"
    "your-project/ui"
)

func main() {
    // Format various timestamps
    now := time.Now()
    
    // Recent creation (30 seconds ago)
    recent := now.Add(-30 * time.Second)
    fmt.Println(ui.FormatAge(recent)) // Output: "30s"
    
    // Created 2 hours ago
    hoursAgo := now.Add(-2 * time.Hour)
    fmt.Println(ui.FormatAge(hoursAgo)) // Output: "2h"
    
    // Created 3 days ago
    daysAgo := now.Add(-3 * 24 * time.Hour)
    fmt.Println(ui.FormatAge(daysAgo)) // Output: "3d"
    
    // Handle zero time
    var zeroTime time.Time
    fmt.Println(ui.FormatAge(zeroTime)) // Output: "unknown"
}
```

The function automatically selects the most appropriate time unit (days, hours, minutes, or seconds) and returns the largest non-zero unit for compact display. Zero timestamps are handled gracefully by returning "unknown".
