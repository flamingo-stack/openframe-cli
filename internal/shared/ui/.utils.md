Provides utility functions for formatting time-related data in user interfaces. This package helps convert time values into human-readable strings for display purposes.

## Key Components

- **FormatAge(createdAt time.Time) string** - Converts a timestamp into a human-readable age string (e.g., "2d", "3h", "45m", "30s")

## Usage Example

```go
package main

import (
    "fmt"
    "time"
    "your-project/ui"
)

func main() {
    // Format different ages
    now := time.Now()
    
    // 2 days ago
    twoDaysAgo := now.Add(-48 * time.Hour)
    fmt.Println(ui.FormatAge(twoDaysAgo)) // Output: "2d"
    
    // 3 hours ago
    threeHoursAgo := now.Add(-3 * time.Hour)
    fmt.Println(ui.FormatAge(threeHoursAgo)) // Output: "3h"
    
    // 30 minutes ago
    thirtyMinsAgo := now.Add(-30 * time.Minute)
    fmt.Println(ui.FormatAge(thirtyMinsAgo)) // Output: "30m"
    
    // Zero time
    var zeroTime time.Time
    fmt.Println(ui.FormatAge(zeroTime)) // Output: "unknown"
}
```

The function returns the largest applicable time unit (days > hours > minutes > seconds) and handles zero-value timestamps gracefully.
