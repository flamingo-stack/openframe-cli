This file provides a comprehensive progress tracking system for multi-step, long-running operations with visual feedback using spinners and progress bars.

## Key Components

- **`Tracker`** - Main struct for tracking operation progress with thread-safe operations
- **`Step`** - Represents individual steps with status, timing, and error handling
- **`StepStatus`** - Enum for step states (Pending, Running, Completed, Failed, Skipped)
- **`NewTracker()`** - Creates a new tracker with operation name and steps
- **Step Management** - `StartStep()`, `CompleteStep()`, `FailStep()`, `SkipStep()`
- **Progress Updates** - `UpdateProgress()` for real-time progress tracking
- **Operation Control** - `Start()`, `Complete()`, `Fail()`, `Cancel()`
- **Status Queries** - `GetProgress()`, `IsCompleted()`, `IsCancelled()`

## Usage Example

```go
// Define operation steps
steps := []Step{
    {Name: "Initialize", Weight: 1.0},
    {Name: "Process Data", Weight: 3.0},
    {Name: "Finalize", Weight: 1.0},
}

// Create and start tracker
tracker := NewTracker("Data Processing", steps)
tracker.Start()

// Execute steps
tracker.StartStep(0)
tracker.CompleteStep(0)

tracker.StartStep(1)
tracker.UpdateProgress(50.0) // 50% through current step
tracker.CompleteStep(1)

tracker.StartStep(2)
tracker.CompleteStep(2)

tracker.Complete() // Shows summary with timing details
```
