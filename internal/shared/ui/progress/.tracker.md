Provides comprehensive progress tracking for long-running operations with multi-step workflows, visual indicators, and cancellation support. Includes spinner and progress bar displays using the pterm library.

## Key Components

- **Tracker**: Main progress tracking struct with thread-safe operations
- **Step**: Represents individual operation steps with status, timing, and weight
- **StepStatus**: Enum for step states (Pending, Running, Completed, Failed, Skipped)
- **NewTracker()**: Creates a new tracker with operation name and steps
- **Start/Complete/Fail/Cancel**: Lifecycle management methods
- **StartStep/CompleteStep/FailStep/SkipStep**: Step-specific operations
- **UpdateProgress()**: Updates visual progress indicators
- **Context()**: Returns cancellation context for graceful shutdown

## Usage Example

```go
// Define operation steps
steps := []progress.Step{
    {Name: "Initialize", Description: "Setting up", Weight: 1.0},
    {Name: "Process", Description: "Main work", Weight: 3.0},
    {Name: "Cleanup", Description: "Finalizing", Weight: 1.0},
}

// Create and start tracker
tracker := progress.NewTracker("Data Migration", steps)
tracker.Start()

// Execute steps
tracker.StartStep(0)
// ... do work ...
tracker.CompleteStep(0)

tracker.StartStep(1)
tracker.UpdateProgress(50.0) // 50% through current step
// ... do more work ...
tracker.CompleteStep(1)

// Handle cancellation
if tracker.IsCancelled() {
    return
}

tracker.Complete()
```
