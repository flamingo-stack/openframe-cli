Test suite for the UI package's time formatting utilities. Contains comprehensive test cases for the `FormatAge` function that converts timestamps to human-readable age strings.

## Key Components

- **TestFormatAge**: Main test function covering standard age formatting scenarios (seconds, minutes, hours, days)
- **TestFormatAgeEdgeCases**: Additional tests for edge cases like very recent times and future timestamps
- **Test cases**: Structured test data with expected outputs for different time durations

## Usage Example

```go
// Run all tests
go test -v

// Run specific test
go test -run TestFormatAge

// The tests validate FormatAge function behavior:
// FormatAge(time.Time{}) -> "unknown"
// FormatAge(30 seconds ago) -> "30s" 
// FormatAge(5 minutes ago) -> "5m"
// FormatAge(2 hours ago) -> "2h"
// FormatAge(3 days ago) -> "3d"
```

The tests use table-driven testing patterns with time tolerance handling for real-time calculations. Edge case testing ensures the function handles zero times, very recent timestamps, and future times gracefully without panicking.
