A comprehensive message templating system that provides standardized, emoji-rich console output for CLI operations using the pterm library. It offers type-safe message formatting with predefined templates for common scenarios like installations, validations, and progress updates.

## Key Components

### Core Types
- `MessageType` - Enum defining message categories (Info, Success, Warning, Error, Progress, Completion)
- `Templates` - Main template container with predefined message formats
- `CustomTemplates` - Extended templates allowing custom message additions
- `Formatter` - High-level formatting interface with specialized sub-formatters

### Template Categories
- **InfoMessage** - Operation starts, progress updates, connections
- **SuccessMessage** - Completions, validations, established connections
- **WarningMessage** - Cancellations, skipped steps, deprecation notices
- **ErrorMessage** - Failures, timeouts, permission issues, not found errors
- **ProgressMessage** - Bootstrapping, health checks, resource waiting
- **CompletionMessage** - Troubleshooting, summaries, statistics

### Specialized Formatters
- `InstallationFormatter` - Installation-specific messaging (starting, complete, failed)
- `ClusterFormatter` - Cluster operation messaging
- `NewFormatter()` - Factory for accessing specialized formatters

## Usage Example

```go
// Basic template usage
templates := NewTemplates()
templates.ShowOperationStart("ArgoCD installation", "production")
templates.ShowStepComplete("Configuration", 2*time.Second)
templates.ShowOperationComplete("ArgoCD installation")

// Custom templates
custom := NewCustomTemplates()
custom.AddTemplate(InfoMessage, "custom_step", "ðŸ”§ Custom step: %s")
custom.ShowInfo("custom_step", "Configuring networking")

// High-level formatter
formatter := NewFormatter()
installer := formatter.Installation()
installer.Starting("ArgoCD", "production-cluster")
installer.Complete("ArgoCD", []string{"Access UI at https://argocd.example.com", "Login with provided credentials"})

// Error handling with troubleshooting
installer.Failed("ArgoCD", errors.New("timeout"), []string{
    "Check cluster connectivity",
    "Verify resource quotas",
    "Review installation logs",
})
```
