Provides a comprehensive template system for standardized console messaging with emoji-enhanced formatting and multiple message types (info, success, warning, error, progress, completion).

## Key Components

- **MessageType** - Enum for categorizing messages (Info, Success, Warning, Error, Progress, Completion)
- **Templates** - Core template manager with predefined message formats for common operations
- **CustomTemplates** - Extends Templates with ability to add custom message formats
- **Formatter** - Provides convenient access to specialized formatters
- **InstallationFormatter** - Specialized formatter for installation-related messages
- **ClusterFormatter** - Specialized formatter for cluster-related messages

## Usage Example

```go
// Basic template usage
templates := NewTemplates()
templates.ShowOperationStart("deployment", "my-cluster")
templates.ShowSuccess("operation_complete", "deployment")

// Custom templates
custom := NewCustomTemplates()
custom.AddTemplate(InfoMessage, "custom_msg", "ðŸŽ¯ Custom: %s")
custom.ShowInfo("custom_msg", "my message")

// Specialized formatters
formatter := NewFormatter()
installer := formatter.Installation()
installer.Starting("nginx", "production")
installer.Complete("nginx", []string{"Access at http://localhost", "Check logs"})

// Error handling with troubleshooting
installer.Failed("nginx", errors.New("port conflict"), []string{
    "Check if port 80 is available",
    "Review configuration settings",
})
```

The template system uses `pterm` for colorized console output and includes built-in emoji icons for visual clarity. All messages support printf-style formatting arguments.
