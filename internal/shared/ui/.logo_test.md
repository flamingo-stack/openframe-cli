This file contains comprehensive test cases for the UI logo display functionality, testing both plain and fancy logo rendering modes with various environment configurations.

## Key Components

- **`TestShowLogo_*` functions**: Test the main `ShowLogo()` function in different modes (test, plain, fancy)
- **`TestCenterText`**: Validates text centering logic with edge cases like zero width and long text
- **`TestIsTerminalEnvironment`**: Ensures terminal detection doesn't panic
- **`TestLogoConstants`**: Verifies logo art constants and Unicode character validation
- **`captureOutput` helper**: Captures stdout for testing output functions
- **Benchmark tests**: Performance validation for logo rendering and text centering
- **Environment variable tests**: Validates behavior with different `OPENFRAME_FANCY_LOGO` settings

## Usage Example

```go
// Run all logo tests
go test -v ./ui -run TestShowLogo

// Run performance benchmarks
go test -bench=BenchmarkShowPlainLogo ./ui

// Test text centering functionality
func TestMyCustomCentering(t *testing.T) {
    result := centerText("Hello", 10)
    assert.Equal(t, "  Hello   ", result)
}

// Capture output for testing
output := captureOutput(func() {
    ShowLogo()
})
assert.Contains(t, output, "OpenFrame")
```

The tests ensure logo display works correctly across different terminal environments and configuration modes while maintaining performance standards.
