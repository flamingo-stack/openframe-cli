Comprehensive test suite for the UI logo functionality, validating different display modes, text centering utilities, and environment-based configuration.

## Key Components

- **`TestShowLogo_*`** - Tests for the main `ShowLogo()` function in test, plain, and fancy modes
- **`TestCenterText`** - Validates text centering logic with various edge cases
- **`TestIsTerminalEnvironment`** - Checks terminal detection functionality
- **`TestShowPlainLogo`/`TestShowFancyLogo`** - Tests individual logo display modes
- **`TestLogoConstants`** - Validates logo constants and Unicode art structure
- **`captureOutput`** - Helper function to capture stdout for testing
- **Benchmark tests** - Performance validation for logo rendering and text operations

## Usage Example

```go
// Run all logo tests
go test -v ./ui -run TestShowLogo

// Run performance benchmarks
go test -bench=BenchmarkShowPlainLogo ./ui

// Test with environment variables
OPENFRAME_FANCY_LOGO=false go test ./ui -run TestShowLogo_PlainMode

// Capture output in custom tests
output := captureOutput(func() {
    ShowLogo()
})
assert.Contains(t, output, "OpenFrame Platform Bootstrapper")
```

The tests cover various scenarios including test mode suppression, environment variable handling, terminal detection, and text formatting edge cases with comprehensive validation of logo constants and Unicode characters.
