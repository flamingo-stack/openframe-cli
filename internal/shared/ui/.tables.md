Provides table rendering utilities with fallback mechanisms for CLI applications that display tabular data using pterm.

## Key Components

- **RenderTableWithFallback** - Renders a 5-column table with pterm, falls back to formatted printf output if pterm fails
- **RenderKeyValueTable** - Displays key-value pairs in table format with simple fallback
- **RenderNodeTable** - Specialized table renderer for node information (name, role, status, age)
- **GetStatusColor** - Returns appropriate color functions based on status strings (green for running/ready, yellow for pending, red for errors)
- **ShowSuccessBox** - Displays success messages in a formatted box with title

## Usage Example

```go
// Render cluster information table
data := pterm.TableData{
    {"Name", "Status", "Type", "Port", "Version"},
    {"cluster-1", "running", "control-plane", "6443", "v1.28.0"},
    {"cluster-2", "stopped", "worker", "6444", "v1.27.2"},
}
RenderTableWithFallback(data, true)

// Display key-value configuration
config := pterm.TableData{
    {"Key", "Value"},
    {"API Server", "https://localhost:6443"},
    {"Cluster Name", "my-cluster"},
}
RenderKeyValueTable(config)

// Color status text
colorFunc := GetStatusColor("running")
fmt.Println(colorFunc("Cluster is running"))

// Show success message
ShowSuccessBox("Deployment Complete", "All services are running successfully")
```
