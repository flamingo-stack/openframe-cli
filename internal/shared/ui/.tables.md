Provides table rendering utilities with automatic fallback to plain text when styled output fails, designed for CLI applications displaying cluster and node information.

## Key Components

- **RenderTableWithFallback**: Renders formatted tables with 5-column layout fallback for cluster commands
- **RenderKeyValueTable**: Displays data in key-value format with simple text fallback
- **RenderNodeTable**: Specialized table renderer for node information with fixed column layout
- **GetStatusColor**: Returns appropriate color functions based on status strings (green for success, yellow for pending, red for errors)
- **ShowSuccessBox**: Displays formatted success messages in a bordered box

## Usage Example

```go
// Render cluster information
data := pterm.TableData{
    {"Name", "Status", "Type", "Nodes", "Age"},
    {"cluster-1", "Running", "production", "3", "2d"},
}
ui.RenderTableWithFallback(data, true)

// Display key-value configuration
config := pterm.TableData{
    {"Key", "Value"},
    {"Version", "1.0.0"},
    {"Region", "us-west-2"},
}
ui.RenderKeyValueTable(config)

// Color status based on state
colorFn := ui.GetStatusColor("running")
fmt.Println(colorFn("Service is running"))

// Show success message
ui.ShowSuccessBox("Deployment Complete", "All services started successfully")
```
