This file provides utility functions for rendering tables and status information in a CLI application using the pterm library, with graceful fallbacks for environments where advanced terminal formatting is not available.

## Key Components

- **RenderTableWithFallback** - Renders formatted tables with automatic fallback to plain text for 5-column layouts
- **RenderKeyValueTable** - Displays key-value pairs in table format with simple text fallback
- **RenderNodeTable** - Specialized table renderer for node information (name, role, status, age)
- **GetStatusColor** - Returns appropriate color functions based on status strings (green for success, yellow for pending, red for errors)
- **ShowSuccessBox** - Displays success messages in a formatted box with centered title

## Usage Example

```go
// Render a cluster information table
data := pterm.TableData{
    {"Name", "Status", "Role", "CPU", "Memory"},
    {"node-1", "Running", "Master", "45%", "2.1GB"},
    {"node-2", "Ready", "Worker", "32%", "1.8GB"},
}
err := RenderTableWithFallback(data, true)

// Display key-value configuration
configData := pterm.TableData{
    {"Key", "Value"},
    {"Cluster", "production"},
    {"Version", "v1.2.3"},
}
err = RenderKeyValueTable(configData)

// Show colored status with appropriate color
colorFunc := GetStatusColor("running")
fmt.Println(colorFunc("System is running"))

// Display success message
ShowSuccessBox("Deployment Complete", "All services are now running successfully")
```
