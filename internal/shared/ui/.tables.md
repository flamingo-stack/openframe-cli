This Go package provides table rendering utilities for CLI applications with automatic fallback to plain text when styled tables fail to render.

## Key Components

### Table Rendering Functions
- **`RenderTableWithFallback`** - Renders formatted tables with 5-column layout fallback for cluster commands
- **`RenderKeyValueTable`** - Displays data in key-value format with simple text fallback
- **`RenderNodeTable`** - Specialized renderer for node information with predefined column layout

### Utility Functions
- **`GetStatusColor`** - Returns appropriate color functions based on status values (green for running/ready, yellow for pending, red for errors)
- **`ShowSuccessBox`** - Displays success messages in formatted boxes with centered titles

## Usage Example

```go
package main

import (
    "github.com/pterm/pterm"
    "your-project/ui"
)

func main() {
    // Render a cluster status table
    clusterData := pterm.TableData{
        {"NAME", "STATUS", "VERSION", "NODES", "AGE"},
        {"prod-cluster", "Running", "v1.28.0", "3", "5d"},
        {"dev-cluster", "Stopped", "v1.27.0", "1", "2d"},
    }
    ui.RenderTableWithFallback(clusterData, true)

    // Show node information
    nodeData := pterm.TableData{
        {"NAME", "ROLE", "STATUS", "AGE"},
        {"node-1", "control-plane", "Ready", "5d"},
        {"node-2", "worker", "NotReady", "3d"},
    }
    ui.RenderNodeTable(nodeData)

    // Display key-value configuration
    configData := pterm.TableData{
        {"Key", "Value"},
        {"API Server", "https://api.cluster.local"},
        {"Registry", "registry.local:5000"},
    }
    ui.RenderKeyValueTable(configData)

    // Color status text
    colorFunc := ui.GetStatusColor("running")
    fmt.Println(colorFunc("Service is running"))

    // Show success message
    ui.ShowSuccessBox("Deployment Complete", "All services are running successfully!")
}
```
