This file contains comprehensive unit tests for a UI prompts package that provides interactive CLI functionality using the promptui library.

## Key Components

- **Validation Function Tests**: Tests for input validation logic used in confirmation prompts (Y/N responses)
- **Result Parsing Tests**: Tests for converting user input strings to boolean values  
- **Template Configuration Tests**: Verification of promptui template setup for select lists
- **Multi-choice Error Handling**: Tests for validating array length consistency in multi-select prompts
- **Validation Helpers**: Tests for `ValidateNonEmpty` and `ValidateIntRange` input validators
- **Benchmark Tests**: Performance testing for validation and parsing functions
- **Package Export Verification**: Compile-time checks ensuring all expected functions are available

## Usage Example

```go
// Test the confirmation prompt validation
func TestConfirmValidation(t *testing.T) {
    validateFunc := func(input string) error {
        input = strings.ToLower(strings.TrimSpace(input))
        if input == "" || input == "y" || input == "yes" || input == "n" || input == "no" {
            return nil
        }
        return errors.New("please enter Y/y/yes or N/n/no")
    }
    
    assert.NoError(t, validateFunc("y"))
    assert.Error(t, validateFunc("maybe"))
}

// Test custom validators
func TestValidators(t *testing.T) {
    validator := ValidateIntRange(1, 10, "port")
    assert.NoError(t, validator("5"))
    assert.Error(t, validator("15"))
}
```

The tests focus on the validation logic and configuration rather than actual user interaction, since promptui requires stdin/stdout which is difficult to mock in unit tests.
