A UI utility package that renders the OpenFrame Platform Bootstrapper ASCII logo with automatic terminal detection and both fancy (colored) and plain text display modes.

## Key Components

- **ShowLogo()** - Displays the ASCII logo with automatic environment detection
- **ShowLogoConditional(suppress bool)** - Conditional logo display with suppression option
- **ShowLogoWithContext(ctx context.Context)** - Context-aware logo display with suppression support
- **WithSuppressedLogo(ctx context.Context)** - Returns context with logo suppression enabled
- **TestMode** - Global flag to suppress logo output during testing
- **logoArt** - Unicode ASCII art array containing the OpenFrame logo design

## Usage Example

```go
package main

import (
    "context"
    "os"
    "your-project/ui"
)

func main() {
    // Basic logo display with auto-detection
    ui.ShowLogo()
    
    // Conditional display
    ui.ShowLogoConditional(false) // Always show
    ui.ShowLogoConditional(true)  // Always suppress
    
    // Context-based suppression
    ctx := context.Background()
    ui.ShowLogoWithContext(ctx) // Shows logo
    
    suppressedCtx := ui.WithSuppressedLogo(ctx)
    ui.ShowLogoWithContext(suppressedCtx) // Suppresses logo
    
    // Environment control
    os.Setenv("OPENFRAME_FANCY_LOGO", "true")  // Force fancy mode
    os.Setenv("OPENFRAME_FANCY_LOGO", "false") // Force plain mode
    
    // Testing mode
    ui.TestMode = true
    ui.ShowLogo() // No output in test mode
}
```

The package automatically detects terminal capabilities and uses pterm for colored output in fancy mode, or plain Unicode box drawing characters in basic mode. Logo display can be controlled via environment variables, context suppression, or the global TestMode flag.
