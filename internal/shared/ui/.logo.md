This file provides logo display functionality for the OpenFrame Platform Bootstrapper, supporting both fancy terminal output with colors and plain text fallbacks.

## Key Components

- **ShowLogo()**: Simple function to display the logo
- **ShowLogoConditional(suppress bool)**: Displays logo with optional suppression
- **ShowLogoWithContext(ctx context.Context)**: Context-aware logo display with suppression support
- **WithSuppressedLogo(ctx context.Context)**: Returns context with logo suppression enabled
- **TestMode**: Global variable to suppress logo output during testing
- **logoArt**: Unicode ASCII art array containing the OpenFrame logo

## Usage Example

```go
package main

import (
    "context"
    "your-package/ui"
)

func main() {
    // Simple logo display
    ui.ShowLogo()
    
    // Conditional display
    ui.ShowLogoConditional(false)
    
    // Context-based suppression
    ctx := context.Background()
    ui.ShowLogoWithContext(ctx)
    
    // Suppress logo via context
    suppressedCtx := ui.WithSuppressedLogo(ctx)
    ui.ShowLogoWithContext(suppressedCtx) // Won't display
    
    // Suppress during testing
    ui.TestMode = true
    ui.ShowLogo() // Won't display
}
```

The logo automatically detects terminal capabilities and environment variables like `OPENFRAME_FANCY_LOGO` and `NO_COLOR` to choose between fancy colored output or plain text display.
