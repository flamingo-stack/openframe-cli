A UI component that displays the OpenFrame platform ASCII art logo with adaptive styling based on terminal capabilities and environment detection.

## Key Components

### Constants and Variables
- **Logo Configuration**: Border characters, dimensions, and styling constants for consistent display
- **logoArt**: Unicode ASCII art array containing the OpenFrame logo graphics
- **TestMode**: Global flag to suppress logo output during testing

### Main Functions
- **ShowLogo()**: Default logo display function
- **ShowLogoConditional(suppress bool)**: Logo display with optional suppression flag
- **ShowLogoWithContext(ctx context.Context)**: Context-aware logo display with suppression support
- **WithSuppressedLogo(ctx context.Context)**: Creates context with logo suppression enabled

### Display Functions
- **showFancyLogo()**: Enhanced display using pterm library with colors and box styling
- **showPlainLogo()**: Fallback plain text display for non-terminal environments
- **isTerminalEnvironment()**: Detects if running in an interactive terminal

## Usage Example

```go
package main

import (
    "context"
    "os"
    "your-project/ui"
)

func main() {
    // Basic logo display
    ui.ShowLogo()
    
    // Context-based suppression
    ctx := ui.WithSuppressedLogo(context.Background())
    ui.ShowLogoWithContext(ctx) // Won't display
    
    // Environment-based control
    os.Setenv("OPENFRAME_FANCY_LOGO", "true")
    ui.ShowLogoConditional(false) // Uses fancy styling
    
    // Test mode suppression
    ui.TestMode = true
    ui.ShowLogo() // Won't display in test mode
}
```

The component automatically detects terminal capabilities and environment variables to choose between fancy colored output (using pterm) or plain text fallback, ensuring consistent branding across different deployment environments.
