A UI utility for displaying the OpenFrame Platform ASCII logo with support for both fancy terminal output and plain text fallback modes.

## Key Components

- **ShowLogo()** - Displays the OpenFrame ASCII logo with automatic formatting detection
- **ShowLogoConditional(suppress bool)** - Conditionally displays the logo with suppression option
- **ShowLogoWithContext(ctx context.Context)** - Context-aware logo display that respects suppression flags
- **WithSuppressedLogo(ctx context.Context)** - Returns a context with logo suppression enabled
- **TestMode** - Global variable to suppress logo output during testing
- **logoArt** - Unicode ASCII art array containing the OpenFrame logo design

## Usage Example

```go
package main

import (
    "context"
    "github.com/openframe/ui"
)

func main() {
    // Simple logo display
    ui.ShowLogo()
    
    // Conditional display with suppression
    ui.ShowLogoConditional(false) // shows logo
    ui.ShowLogoConditional(true)  // suppresses logo
    
    // Context-based suppression
    ctx := context.Background()
    ui.ShowLogoWithContext(ctx) // shows logo
    
    suppressedCtx := ui.WithSuppressedLogo(ctx)
    ui.ShowLogoWithContext(suppressedCtx) // suppresses logo
    
    // Disable for testing
    ui.TestMode = true
    ui.ShowLogo() // no output
}
```

The package automatically detects terminal capabilities and environment variables (`OPENFRAME_FANCY_LOGO`, `NO_COLOR`, `TERM`) to choose between fancy pterm-styled output with colors and borders, or plain text ASCII art suitable for basic terminals and CI environments.
