Provides ASCII art logo display functionality for the OpenFrame Platform Bootstrapper with support for both fancy terminal rendering and plain text fallback based on environment detection.

## Key Components

- **ShowLogo()** - Main function to display the OpenFrame logo
- **ShowLogoConditional(suppress bool)** - Display logo with optional suppression
- **ShowLogoWithContext(ctx context.Context)** - Context-aware logo display with suppression support
- **WithSuppressedLogo(ctx context.Context)** - Returns context with logo suppression enabled
- **TestMode** - Global variable to suppress logo output during testing
- **logoArt** - Pre-defined Unicode ASCII art for the OpenFrame logo

The package automatically detects terminal capabilities and environment variables (`OPENFRAME_FANCY_LOGO`, `NO_COLOR`, `TERM`) to choose between fancy styled output using pterm or plain text with Unicode box drawing characters.

## Usage Example

```go
package main

import (
    "context"
    "your-project/ui"
)

func main() {
    // Basic logo display
    ui.ShowLogo()
    
    // Conditional display
    ui.ShowLogoConditional(false) // shows logo
    ui.ShowLogoConditional(true)  // suppresses logo
    
    // Context-based suppression
    ctx := context.Background()
    ui.ShowLogoWithContext(ctx) // shows logo
    
    suppressedCtx := ui.WithSuppressedLogo(ctx)
    ui.ShowLogoWithContext(suppressedCtx) // suppresses logo
    
    // Disable during testing
    ui.TestMode = true
    ui.ShowLogo() // no output
}
```
