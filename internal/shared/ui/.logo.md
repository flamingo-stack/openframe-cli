This file provides a flexible ASCII logo display system for the OpenFrame Platform Bootstrapper with support for both fancy terminal rendering and plain text fallback modes.

## Key Components

- **ShowLogo()** - Main function to display the logo with automatic environment detection
- **ShowLogoConditional(suppress bool)** - Display logo with explicit suppression option
- **ShowLogoWithContext(ctx context.Context)** - Context-aware logo display with suppression support
- **WithSuppressedLogo(ctx context.Context)** - Creates a context that suppresses logo output
- **TestMode** - Global variable to disable logo output during testing
- **logoArt** - Unicode box-drawing character array containing the "OPENFRAME" ASCII art

The system automatically detects terminal capabilities and environment variables (`OPENFRAME_FANCY_LOGO`, `NO_COLOR`) to choose between fancy pterm-styled output or plain text with custom box borders.

## Usage Example

```go
package main

import (
    "context"
    "your-project/ui"
)

func main() {
    // Simple logo display with auto-detection
    ui.ShowLogo()
    
    // Explicitly suppress logo
    ui.ShowLogoConditional(true)
    
    // Context-based suppression
    ctx := ui.WithSuppressedLogo(context.Background())
    ui.ShowLogoWithContext(ctx) // Won't display logo
    
    // Disable during testing
    ui.TestMode = true
    ui.ShowLogo() // Won't display
}
```

The logo supports environment variable configuration (`OPENFRAME_FANCY_LOGO=true/false`) and automatically falls back to plain text in non-interactive environments like CI/CD pipelines or when `NO_COLOR` is set.
