This file provides ASCII art logo display functionality for the OpenFrame Platform Bootstrapper with support for both fancy terminal output and plain text fallback based on environment detection.

## Key Components

**Constants & Variables:**
- `logoTitle` - Application title text
- `logoArt` - ASCII art logo stored as string slice
- `TestMode` - Global flag to suppress logo output during testing
- Border and formatting characters for plain text display

**Main Functions:**
- `ShowLogo()` - Basic logo display function
- `ShowLogoConditional(suppress bool)` - Logo display with optional suppression
- `ShowLogoWithContext(ctx context.Context)` - Context-aware logo display
- `WithSuppressedLogo(ctx context.Context)` - Creates context with logo suppression

**Helper Functions:**
- `isTerminalEnvironment()` - Detects if running in a proper terminal
- `showFancyLogo()` - Displays colored logo using pterm library
- `showPlainLogo()` - Displays simple bordered text logo
- `centerText(text, width)` - Text centering utility

## Usage Example

```go
package main

import (
    "context"
    "your-project/ui"
)

func main() {
    // Basic usage
    ui.ShowLogo()
    
    // With conditional suppression
    ui.ShowLogoConditional(false)
    
    // Context-based control
    ctx := context.Background()
    ui.ShowLogoWithContext(ctx)
    
    // Suppress logo via context
    suppressedCtx := ui.WithSuppressedLogo(ctx)
    ui.ShowLogoWithContext(suppressedCtx) // Won't display
    
    // Control via environment
    // Set OPENFRAME_FANCY_LOGO=true for colored output
}
```
