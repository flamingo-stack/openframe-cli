This file provides a set of user-friendly messaging functions for displaying CLI operation feedback with formatted output using the pterm library.

## Key Components

- **ShowNoResourcesMessage()** - Displays a helpful message box when no resources are found, including suggested commands for creating and listing resources
- **ShowOperationStart()** - Shows an info message when beginning an operation, with support for custom messages
- **ShowOperationSuccess()** - Displays a success message upon operation completion with customizable content
- **ShowOperationError()** - Shows detailed error information with optional troubleshooting tips in a formatted table
- **TroubleshootingTip** - Struct containing description and command pairs for error guidance

## Usage Example

```go
package main

import "github.com/yourorg/yourapp/ui"

func main() {
    // Show no resources message
    ui.ShowNoResourcesMessage(
        "deployments", 
        "delete", 
        "kubectl create deployment", 
        "kubectl get deployments",
    )
    
    // Show operation progress
    customMsgs := map[string]string{
        "deploy": "Deploying application to cluster...",
    }
    ui.ShowOperationStart("deploy", "my-app", customMsgs)
    
    // Show error with troubleshooting
    tips := []ui.TroubleshootingTip{
        {Description: "Check cluster status", Command: "kubectl cluster-info"},
        {Description: "Verify permissions", Command: "kubectl auth can-i create pods"},
    }
    ui.ShowOperationError("deploy", "my-app", err, tips)
}
```
