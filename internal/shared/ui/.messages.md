This file provides user-friendly console messaging functions for CLI operations, using the pterm library for styled terminal output.

## Key Components

- **`ShowNoResourcesMessage`** - Displays a formatted warning when no resources are found, with helpful next steps in a bordered box
- **`ShowOperationStart`** - Shows an info message when beginning an operation, with support for custom messages
- **`ShowOperationSuccess`** - Displays a success message when operations complete successfully
- **`ShowOperationError`** - Shows detailed error information with optional troubleshooting tips in a table format
- **`TroubleshootingTip`** - Struct for organizing troubleshooting suggestions with description and command

## Usage Example

```go
import "github.com/yourorg/yourproject/ui"

// Show no resources found
ui.ShowNoResourcesMessage("clusters", "delete", "kubectl create cluster", "kubectl get clusters")

// Track operation progress
ui.ShowOperationStart("deploy", "my-app", nil)

// Handle success
ui.ShowOperationSuccess("deploy", "my-app", map[string]string{
    "deploy": "Successfully deployed my-app to production!",
})

// Handle errors with troubleshooting
tips := []ui.TroubleshootingTip{
    {Description: "Check cluster status", Command: "kubectl cluster-info"},
    {Description: "Verify credentials", Command: "kubectl auth whoami"},
}
ui.ShowOperationError("deploy", "my-app", err, tips)
```
