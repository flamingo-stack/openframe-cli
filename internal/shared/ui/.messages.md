A UI utility package that provides standardized, user-friendly console messages for CLI operations using the pterm library for enhanced terminal output with colors and formatting.

## Key Components

- **ShowNoResourcesMessage**: Displays a formatted warning box when no resources are found, including helpful creation and listing commands
- **ShowOperationStart**: Shows an informational message when beginning an operation, with support for custom messaging
- **ShowOperationSuccess**: Displays a success message upon operation completion with optional custom text
- **ShowOperationError**: Shows detailed error information with optional troubleshooting tips in a formatted table
- **TroubleshootingTip**: Struct for organizing troubleshooting suggestions with description and command pairs

## Usage Example

```go
import "your-project/ui"

// Show no resources found
ui.ShowNoResourcesMessage(
    "projects", 
    "deploy", 
    "flamingo create project", 
    "flamingo list projects",
)

// Show operation progress
ui.ShowOperationStart("deploy", "my-app", nil)

// Show success
ui.ShowOperationSuccess("deploy", "my-app", nil)

// Show error with troubleshooting
tips := []ui.TroubleshootingTip{
    {Description: "Check network connectivity", Command: "ping api.example.com"},
    {Description: "Verify credentials", Command: "flamingo auth status"},
}
ui.ShowOperationError("deploy", "my-app", err, tips)
```
