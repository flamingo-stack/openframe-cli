This file provides user interface utilities for displaying formatted messages and feedback during CLI operations, with consistent styling using the pterm library.

## Key Components

**Main Functions:**
- `ShowNoResourcesMessage()` - Displays a helpful message when no resources are found, with suggested next steps
- `ShowOperationStart()` - Shows a progress message when beginning an operation
- `ShowOperationSuccess()` - Displays success confirmation after operation completion
- `ShowOperationError()` - Shows error details with troubleshooting suggestions

**Type:**
- `TroubleshootingTip` - Struct containing description and command for error resolution guidance

## Usage Example

```go
package main

import "your-project/ui"

func main() {
    // Show when no resources exist
    ui.ShowNoResourcesMessage(
        "clusters", 
        "deploy", 
        "kubectl create cluster", 
        "kubectl get clusters",
    )
    
    // Operation lifecycle messages
    ui.ShowOperationStart("deploy", "my-app", nil)
    
    // Handle errors with troubleshooting tips
    tips := []ui.TroubleshootingTip{
        {Description: "Check connection", Command: "kubectl cluster-info"},
        {Description: "Verify permissions", Command: "kubectl auth can-i create pods"},
    }
    ui.ShowOperationError("deploy", "my-app", err, tips)
    
    // Success message
    ui.ShowOperationSuccess("deploy", "my-app", nil)
}
```
