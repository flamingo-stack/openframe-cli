This file provides a UI messaging system for displaying user-friendly terminal messages with consistent formatting and helpful guidance during CLI operations.

## Key Components

- **ShowNoResourcesMessage**: Displays a boxed warning when no resources are available for an operation, with helpful next steps
- **ShowOperationStart**: Shows an info message when beginning an operation, with support for custom messages
- **ShowOperationSuccess**: Displays a success message when operations complete successfully
- **ShowOperationError**: Shows detailed error information with troubleshooting tips in a formatted table
- **TroubleshootingTip**: Struct containing description and command pairs for error guidance

## Usage Example

```go
// Show no resources found message
ShowNoResourcesMessage(
    "applications", 
    "deployment", 
    "create app myapp", 
    "list apps",
)

// Show operation progress
ShowOperationStart("deploy", "myapp", nil)

// Show success with custom message
customMessages := map[string]string{
    "deploy": "Successfully deployed application to production!",
}
ShowOperationSuccess("deploy", "myapp", customMessages)

// Show error with troubleshooting tips
tips := []TroubleshootingTip{
    {"Check connection", "ping example.com"},
    {"Verify credentials", "auth status"},
}
ShowOperationError("deploy", "myapp", errors.New("connection failed"), tips)
```

The package uses pterm for colorized terminal output and provides consistent messaging patterns across CLI operations with helpful user guidance.
