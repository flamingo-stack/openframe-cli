A collection of user-friendly message display functions for CLI operations, providing consistent formatting and helpful feedback using pterm styling.

## Key Components

- **ShowNoResourcesMessage** - Displays a styled box when no resources are available for an operation
- **ShowOperationStart** - Shows informational messages when beginning operations
- **ShowOperationSuccess** - Displays success confirmations with custom or default messages
- **ShowOperationError** - Shows detailed error information with troubleshooting suggestions
- **TroubleshootingTip** - Struct for organizing help suggestions with descriptions and commands

## Usage Example

```go
package main

import "your-project/ui"

func main() {
    // Show no resources available
    ui.ShowNoResourcesMessage(
        "projects", 
        "delete", 
        "openframe project create", 
        "openframe project list",
    )
    
    // Show operation start
    customMessages := map[string]string{
        "deploy": "Deploying your awesome project...",
    }
    ui.ShowOperationStart("deploy", "my-project", customMessages)
    
    // Show success
    ui.ShowOperationSuccess("deploy", "my-project", customMessages)
    
    // Show error with troubleshooting
    tips := []ui.TroubleshootingTip{
        {Description: "Check connection", Command: "ping example.com"},
        {Description: "Verify credentials", Command: "auth status"},
    }
    ui.ShowOperationError("deploy", "my-project", err, tips)
}
```

The functions provide a consistent CLI experience with colored output, styled boxes, and helpful troubleshooting guidance when operations fail.
