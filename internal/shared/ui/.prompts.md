This file provides a comprehensive set of interactive prompt utilities for building polished CLI interfaces with consistent styling and user experience patterns.

## Key Components

- **ConfirmActionInteractive**: Interactive confirmation prompts using pterm with colored styling
- **ConfirmDeletion**: Specialized deletion confirmation with resource highlighting
- **ConfirmAction**: Simple Y/n confirmation with single-key input (no Enter required)
- **SelectFromList**: Basic list selection with arrow navigation
- **SelectFromListWithSearch**: List selection with real-time search/filtering
- **GetInput**: Text input prompts with validation support
- **GetMultiChoice**: Multiple item selection from a list
- **HandleResourceSelection**: Pattern for CLI args vs interactive selection
- **Validation functions**: `ValidateNonEmpty`, `ValidateIntRange` for input validation

## Usage Example

```go
// Simple confirmation
confirmed, err := ui.ConfirmAction("Continue with deployment?")
if err != nil {
    return err
}

// Resource deletion with highlighting
deleted, err := ui.ConfirmDeletion("database", "production-db")

// Interactive list selection with search
options := []string{"development", "staging", "production"}
_, env, err := ui.SelectFromListWithSearch("Select environment", options)

// Text input with validation
name, err := ui.GetInput("Project name", "", ui.ValidateNonEmpty("Project name"))

// Handle CLI args or interactive selection
projectName, err := ui.HandleResourceSelection(
    os.Args[1:], 
    availableProjects, 
    "Select project"
)
```

The module seamlessly handles both terminal and non-terminal input scenarios, providing fallbacks for automated environments while maintaining rich interactive experiences for human users.
