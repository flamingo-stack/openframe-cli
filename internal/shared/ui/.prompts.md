A comprehensive UI prompt library that provides interactive command-line interfaces for user input, confirmations, and selections with terminal-aware functionality and rich styling.

## Key Components

- **ConfirmActionInteractive/ConfirmAction** - Interactive confirmation prompts with polished styling and smart terminal detection
- **ConfirmDeletion** - Specialized deletion confirmation with resource type highlighting
- **SelectFromList/SelectFromListWithSearch** - Interactive list selection with optional search filtering
- **GetInput** - Text input prompts with validation and default values
- **GetMultiChoice** - Multiple selection interface for choosing several items
- **HandleResourceSelection** - Common pattern handler for CLI args vs interactive selection
- **ValidateNonEmpty/ValidateIntRange** - Input validation helper functions

## Usage Example

```go
// Basic confirmation
confirmed, err := ui.ConfirmAction("Deploy to production")
if err != nil {
    return err
}

// Interactive list selection with search
_, selected, err := ui.SelectFromListWithSearch(
    "Select environment", 
    []string{"development", "staging", "production"},
)

// Resource selection from args or interactive
resourceName, err := ui.HandleResourceSelection(
    os.Args[2:], // Skip command and subcommand
    availableApps,
    "Select application",
)

// Text input with validation
name, err := ui.GetInput(
    "Enter application name",
    "",
    ui.ValidateNonEmpty("Application name"),
)
```

The library automatically detects terminal capabilities and falls back to simpler input methods for non-terminal contexts like pipes or tests. All prompts use consistent styling with pterm and promptui for a polished user experience.
