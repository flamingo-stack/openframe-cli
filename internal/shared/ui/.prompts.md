This file provides a comprehensive set of user interaction utilities for CLI applications, offering both simple and interactive prompts with consistent styling and UX patterns.

## Key Components

**Confirmation Functions:**
- `ConfirmActionInteractive()` - Interactive confirmation with pterm styling
- `ConfirmDeletion()` - Specialized deletion confirmation with resource highlighting
- `ConfirmAction()` - Single-key confirmation (y/n/Enter) with terminal support

**Selection Functions:**
- `SelectFromList()` - Basic list selection with styled templates
- `SelectFromListWithSearch()` - List selection with search/filter capability
- `SelectFromListWithCustomTemplates()` - Customizable selection styling
- `HandleResourceSelection()` - Handles arg-based or interactive resource selection

**Input Functions:**
- `GetInput()` - Text input with validation support
- `GetMultiChoice()` - Multiple item selection from a list

**Validation Helpers:**
- `ValidateNonEmpty()` - Ensures non-empty input
- `ValidateIntRange()` - Validates integer within specified range

## Usage Example

```go
// Simple confirmation
confirmed, err := ui.ConfirmAction("Continue with deployment?")
if err != nil || !confirmed {
    return
}

// Interactive list selection with search
options := []string{"prod", "staging", "dev"}
_, env, err := ui.SelectFromListWithSearch("Select environment", options)

// Resource selection (args or interactive)
resource, err := ui.HandleResourceSelection(args, availableApps, "Select application")

// Input with validation
name, err := ui.GetInput("App name", "", ui.ValidateNonEmpty("name"))
```
