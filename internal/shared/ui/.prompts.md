This file provides a comprehensive suite of interactive user interface utilities for command-line applications, featuring confirmation prompts, list selection, text input, and validation functions.

## Key Components

**Confirmation Functions:**
- `ConfirmActionInteractive()` - Polished interactive confirmation using pterm
- `ConfirmDeletion()` - Specialized deletion confirmation with resource highlighting
- `ConfirmAction()` - Single-key confirmation with Enter/y/n support

**Selection Functions:**
- `SelectFromList()` - Basic list selection with styled templates
- `SelectFromListWithSearch()` - List selection with search/filter capability
- `SelectFromListWithCustomTemplates()` - Customizable selection styling

**Input Functions:**
- `GetInput()` - Text input with validation support
- `GetMultiChoice()` - Multiple selection from a list
- `HandleResourceSelection()` - Resource selection from args or interactive list

**Validation Functions:**
- `ValidateNonEmpty()` - Ensures non-empty input
- `ValidateIntRange()` - Validates integer within specified range

## Usage Example

```go
// Confirm an action
confirmed, err := ui.ConfirmAction("Deploy to production")
if err != nil || !confirmed {
    return
}

// Select from a list with search
items := []string{"apple", "banana", "cherry"}
_, selected, err := ui.SelectFromListWithSearch("Choose fruit", items)

// Get validated input
name, err := ui.GetInput("Enter name", "", ui.ValidateNonEmpty("Name"))

// Handle resource selection (from args or interactive)
resource, err := ui.HandleResourceSelection(os.Args[1:], availableResources, "Select resource")
```
