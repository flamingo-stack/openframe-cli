This file provides a comprehensive set of interactive user interface utilities for command-line applications, offering both simple confirmation dialogs and complex selection workflows with search capabilities.

## Key Components

### Confirmation Functions
- **`ConfirmActionInteractive`** - Polished interactive confirmation using pterm with colored styling
- **`ConfirmDeletion`** - Specialized deletion confirmation with resource type and name formatting
- **`ConfirmAction`** - Simple Y/n confirmation with single-key response (no Enter required)

### Selection Functions
- **`SelectFromList`** - Basic list selection with arrow navigation
- **`SelectFromListWithSearch`** - Enhanced list selection with filtering capability
- **`SelectFromListWithCustomTemplates`** - Customizable selection interface
- **`GetMultiChoice`** - Multiple item selection from a list

### Input Functions
- **`GetInput`** - Text input with validation support
- **`HandleResourceSelection`** - Handles CLI args or interactive selection pattern

### Validation Helpers
- **`ValidateNonEmpty`** - Ensures input is not empty
- **`ValidateIntRange`** - Validates integer input within bounds

## Usage Example

```go
// Simple confirmation
confirmed, err := ui.ConfirmAction("Deploy to production?")
if err != nil {
    return err
}

// Interactive list selection with search
options := []string{"development", "staging", "production"}
_, environment, err := ui.SelectFromListWithSearch("Select environment", options)
if err != nil {
    return err
}

// Get validated text input
name, err := ui.GetInput("Project name", "", ui.ValidateNonEmpty("Project name"))
if err != nil {
    return err
}

// Handle CLI args or interactive selection
projectName, err := ui.HandleResourceSelection(args, availableProjects, "Select project")
if err != nil {
    return err
}
```

The module handles both terminal and non-terminal environments gracefully, making it suitable for automated scripts and interactive CLI tools.
