This file provides interactive command-line prompts and user input utilities for Go applications, combining `pterm` for styled confirmations and `promptui` for selections and text input.

## Key Components

- **ConfirmActionInteractive/ConfirmAction** - Confirmation prompts with different UX patterns (interactive vs immediate)
- **ConfirmDeletion** - Specialized deletion confirmation with resource highlighting
- **SelectFromList/SelectFromListWithSearch** - List selection prompts with optional search filtering
- **GetInput** - Text input prompt with validation support
- **GetMultiChoice** - Multiple selection prompt for choosing several items
- **HandleResourceSelection** - Handles argument vs interactive selection pattern
- **Validation functions** - ValidateNonEmpty, ValidateIntRange for input validation

## Usage Example

```go
// Simple confirmation
confirmed, err := ui.ConfirmAction("Deploy to production")
if err != nil {
    log.Fatal(err)
}

// Resource selection with fallback
items := []string{"app1", "app2", "app3"}
selected, err := ui.HandleResourceSelection(args, items, "Select application")
if err != nil {
    log.Fatal(err)
}

// Text input with validation
name, err := ui.GetInput(
    "Enter service name",
    "my-service",
    ui.ValidateNonEmpty("service name"),
)
if err != nil {
    log.Fatal(err)
}

// List selection with search
index, value, err := ui.SelectFromListWithSearch(
    "Choose environment",
    []string{"dev", "staging", "prod"},
)
if err != nil {
    log.Fatal(err)
}
```
