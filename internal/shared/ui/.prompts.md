This file provides a comprehensive UI toolkit for interactive command-line prompts, offering confirmation dialogs, selection menus, and input validation with polished styling and user experience.

## Key Components

**Confirmation Functions:**
- `ConfirmActionInteractive` - Interactive confirmation with pterm styling
- `ConfirmDeletion` - Deletion-specific confirmation with highlighted resource names
- `ConfirmAction` - Friendly Y/n confirmation with single-key response

**Selection Functions:**
- `SelectFromList` - Basic list selection with styled templates
- `SelectFromListWithSearch` - Searchable/filterable list selection
- `SelectFromListWithCustomTemplates` - Customizable selection styling
- `GetMultiChoice` - Multiple item selection

**Input Functions:**
- `GetInput` - Text input with validation and defaults
- `HandleResourceSelection` - Resource selection from args or interactive list

**Validation Helpers:**
- `ValidateNonEmpty` - Ensures non-empty input after trimming
- `ValidateIntRange` - Validates integer input within specified bounds

## Usage Example

```go
// Confirm an action
confirmed, err := ui.ConfirmAction("Deploy to production")
if err != nil || !confirmed {
    return
}

// Select from a list with search
items := []string{"dev", "staging", "production"}
_, env, err := ui.SelectFromListWithSearch("Select environment", items)

// Get validated input
name, err := ui.GetInput("Enter name", "", ui.ValidateNonEmpty("name"))

// Handle resource selection (args or interactive)
resource, err := ui.HandleResourceSelection(args, availableItems, "Select resource")
```
