Test file for the executor package that validates command execution functionality. Contains comprehensive unit tests for command execution, dry-run mode, timeout handling, and environment variable processing.

## Key Components

- **TestCommandResult_Output**: Tests the `Output()` method combining stdout and stderr
- **TestNewRealCommandExecutor**: Validates executor creation with different configurations
- **TestRealCommandExecutor_Execute_***: Tests basic command execution, dry-run mode, and error scenarios
- **TestRealCommandExecutor_ExecuteWithOptions_***: Tests advanced execution with custom options
- **TestRealCommandExecutor_buildEnvStrings**: Tests environment variable string formatting
- **Interface compliance tests**: Ensures proper implementation of `CommandExecutor` interface

## Usage Example

```go
func TestMyCommand(t *testing.T) {
    // Test dry-run mode
    executor := NewRealCommandExecutor(true, false)
    result, err := executor.Execute(ctx, "echo", "hello")
    assert.NoError(t, err)
    assert.Equal(t, 0, result.ExitCode)

    // Test with options
    options := ExecuteOptions{
        Command: "echo",
        Args:    []string{"test"},
        Env:     map[string]string{"VAR": "value"},
        Timeout: 5 * time.Second,
    }
    result, err = executor.ExecuteWithOptions(ctx, options)
    assert.NoError(t, err)
}
```
