Test suite for the executor package that validates command execution functionality including dry-run mode, real command execution, and various configuration options.

## Key Components

- **CommandResult.Output()** - Tests the output formatting method that combines stdout and stderr
- **RealCommandExecutor** - Tests the main executor implementation with dry-run and verbose modes
- **Execute()** - Tests basic command execution with context handling
- **ExecuteWithOptions()** - Tests advanced execution with timeout, environment variables, and working directory
- **buildEnvStrings()** - Tests environment variable formatting helper
- **Interface Compliance** - Validates that RealCommandExecutor implements CommandExecutor interface

## Usage Example

```go
// Test basic command execution
func TestCommandExecution(t *testing.T) {
    executor := NewRealCommandExecutor(false, false)
    ctx := context.Background()
    
    result, err := executor.Execute(ctx, "echo", "hello")
    assert.NoError(t, err)
    assert.Equal(t, 0, result.ExitCode)
    assert.Contains(t, result.Stdout, "hello")
}

// Test with execution options
func TestWithOptions(t *testing.T) {
    executor := NewRealCommandExecutor(false, true)
    
    options := ExecuteOptions{
        Command: "echo",
        Args:    []string{"$TEST_VAR"},
        Env:     map[string]string{"TEST_VAR": "value"},
        Timeout: 5 * time.Second,
    }
    
    result, err := executor.ExecuteWithOptions(ctx, options)
    assert.NoError(t, err)
    assert.Contains(t, result.Stdout, "value")
}
```
