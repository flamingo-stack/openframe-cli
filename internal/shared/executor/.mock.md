A mock implementation of CommandExecutor for testing purposes that simulates command execution without running actual external commands, providing configurable responses and execution tracking.

## Key Components

- **MockCommandExecutor**: Main mock struct that implements CommandExecutor interface
- **NewMockCommandExecutor()**: Factory function to create new mock instances
- **SetResponse()**: Configure specific responses for command patterns
- **SetShouldFail()**: Configure the mock to simulate command failures
- **GetExecutedCommands()**: Retrieve log of all executed commands
- **WasCommandExecuted()**: Check if a command pattern was executed
- **Reset()**: Clear execution history and reset mock state

## Usage Example

```go
// Create mock executor
mock := NewMockCommandExecutor()

// Configure custom response for specific command
mock.SetResponse("git status", &CommandResult{
    ExitCode: 0,
    Stdout:   "nothing to commit, working tree clean",
    Duration: 50 * time.Millisecond,
})

// Execute command
result, err := mock.Execute(ctx, "git", "status")

// Verify execution
if mock.WasCommandExecuted("git status") {
    fmt.Printf("Command executed: %s\n", mock.GetLastCommand())
}

// Configure failure simulation
mock.SetShouldFail(true, "network timeout")
result, err = mock.Execute(ctx, "curl", "example.com")
// Returns error with "mock command failure: network timeout"

// Reset for clean test state
mock.Reset()
```
