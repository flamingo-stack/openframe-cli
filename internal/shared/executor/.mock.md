A mock implementation of CommandExecutor for testing purposes that simulates command execution without running external processes.

## Key Components

- **MockCommandExecutor**: Main mock struct that implements CommandExecutor interface
- **NewMockCommandExecutor()**: Constructor that creates a new mock instance with defaults
- **SetShouldFail()**: Configures the mock to simulate command failures
- **SetResponse()**: Sets predefined responses for specific command patterns
- **Execute/ExecuteWithOptions()**: Implements the CommandExecutor interface methods
- **GetExecutedCommands()**: Returns log of all executed commands for verification
- **Reset()**: Clears all state for test cleanup
- **WasCommandExecuted()**: Checks if a command pattern was executed

## Usage Example

```go
// Create mock executor
mock := NewMockCommandExecutor()

// Set custom response for specific command
mock.SetResponse("git status", &CommandResult{
    ExitCode: 0,
    Stdout:   "nothing to commit",
    Duration: 50 * time.Millisecond,
})

// Execute command
result, err := mock.Execute(context.Background(), "git", "status")

// Verify execution
if mock.WasCommandExecuted("git status") {
    fmt.Println("Git status was called")
}

// Simulate failure
mock.SetShouldFail(true, "network error")
result, err := mock.Execute(context.Background(), "curl", "example.com")
// err will contain "mock command failure: network error"

// Reset for next test
mock.Reset()
```
