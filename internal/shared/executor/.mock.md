A mock implementation of the CommandExecutor interface designed for testing purposes, allowing simulation of command execution without running actual external processes.

## Key Components

- **MockCommandExecutor**: Main struct implementing CommandExecutor interface with configurable behavior
- **NewMockCommandExecutor()**: Factory function creating a new mock executor with default settings
- **SetShouldFail()**: Configure the mock to simulate command failures
- **SetResponse()**: Set specific responses for command patterns
- **Execute/ExecuteWithOptions()**: Simulate command execution with logging and response matching
- **GetExecutedCommands()**: Retrieve logged commands for test verification
- **WasCommandExecuted()**: Check if a command pattern was executed
- **Reset()**: Clear all state for test isolation

## Usage Example

```go
// Create mock executor for testing
mockExec := NewMockCommandExecutor()

// Configure specific responses
mockExec.SetResponse("git status", &CommandResult{
    ExitCode: 0,
    Stdout:   "On branch main\nnothing to commit, working tree clean",
    Duration: 50 * time.Millisecond,
})

// Simulate command failure
mockExec.SetShouldFail(true, "network error")

// Execute mock command
result, err := mockExec.Execute(ctx, "git", "status")

// Verify in tests
if !mockExec.WasCommandExecuted("git status") {
    t.Error("Expected git status to be executed")
}

commands := mockExec.GetExecutedCommands()
assert.Equal(t, 1, len(commands))
```
