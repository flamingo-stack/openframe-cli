A mock implementation of the CommandExecutor interface designed for testing, allowing you to simulate command execution without running actual external commands.

## Key Components

- **MockCommandExecutor**: Main struct implementing CommandExecutor interface
- **NewMockCommandExecutor()**: Constructor function
- **SetResponse()**: Configure specific responses for command patterns
- **SetShouldFail()**: Simulate command failures
- **Execute/ExecuteWithOptions()**: Mock command execution methods
- **GetExecutedCommands()**: Retrieve logged commands for verification
- **WasCommandExecuted()**: Check if a command pattern was executed

## Usage Example

```go
// Create mock executor
mock := NewMockCommandExecutor()

// Set up specific responses
mock.SetResponse("git status", &CommandResult{
    ExitCode: 0,
    Stdout:   "nothing to commit, working tree clean",
    Duration: 50 * time.Millisecond,
})

// Execute commands
result, err := mock.Execute(ctx, "git", "status")

// Verify execution in tests
assert.True(t, mock.WasCommandExecuted("git status"))
assert.Equal(t, 1, mock.GetCommandCount())

// Simulate failures
mock.SetShouldFail(true, "network timeout")
_, err = mock.Execute(ctx, "curl", "example.com")
assert.Error(t, err)

// Reset for next test
mock.Reset()
```
