This file provides a robust abstraction layer for executing external system commands with support for dry-run mode, timeouts, custom environments, and comprehensive logging.

## Key Components

- **`CommandExecutor`** - Interface defining command execution methods for dependency injection and testing
- **`CommandResult`** - Struct containing execution results including exit code, stdout/stderr, and duration
- **`ExecuteOptions`** - Configuration struct for fine-grained control over command execution
- **`RealCommandExecutor`** - Production implementation that executes actual system commands
- **`NewRealCommandExecutor()`** - Factory function creating executor instances with dry-run and verbose options

## Usage Example

```go
package main

import (
    "context"
    "fmt"
    "time"
)

func main() {
    // Create executor with verbose logging
    executor := NewRealCommandExecutor(false, true)
    
    // Simple command execution
    result, err := executor.Execute(context.Background(), "ls", "-la")
    if err != nil {
        fmt.Printf("Error: %v\n", err)
        return
    }
    fmt.Printf("Output: %s\n", result.Stdout)
    
    // Advanced execution with options
    options := ExecuteOptions{
        Command: "git",
        Args:    []string{"status"},
        Dir:     "/path/to/repo",
        Env:     map[string]string{"GIT_PAGER": "cat"},
        Timeout: 30 * time.Second,
    }
    
    result, err = executor.ExecuteWithOptions(context.Background(), options)
    if err != nil {
        fmt.Printf("Git command failed: %v\n", err)
        return
    }
    fmt.Printf("Git status took: %v\n", result.Duration)
}
```

The executor supports dry-run mode for testing, automatic timeout handling, environment variable injection, and detailed logging for debugging command execution workflows.
