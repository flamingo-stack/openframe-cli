This package provides a robust abstraction layer for executing external system commands with support for dry-run mode, timeout handling, custom environments, and comprehensive result tracking.

## Key Components

- **CommandExecutor** - Interface defining command execution methods for dependency injection and testing
- **RealCommandExecutor** - Concrete implementation that executes actual system commands
- **CommandResult** - Struct containing execution results including exit code, output streams, and duration
- **ExecuteOptions** - Configuration struct for fine-grained control over command execution parameters

## Usage Example

```go
// Create executor with dry-run and verbose options
executor := NewRealCommandExecutor(false, true)

// Simple execution
ctx := context.Background()
result, err := executor.Execute(ctx, "git", "status")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Exit code: %d\nOutput: %s\n", result.ExitCode, result.Output())

// Advanced execution with options
options := ExecuteOptions{
    Command: "docker",
    Args:    []string{"build", "-t", "myapp", "."},
    Dir:     "/path/to/project",
    Env:     map[string]string{"DOCKER_BUILDKIT": "1"},
    Timeout: 5 * time.Minute,
}
result, err = executor.ExecuteWithOptions(ctx, options)
```

The executor supports dry-run mode for testing, verbose logging for debugging, custom working directories, environment variable injection, and execution timeouts with proper context cancellation.
