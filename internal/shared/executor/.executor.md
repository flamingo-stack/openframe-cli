Command execution utilities for cross-platform system operations with specialized Windows Subsystem for Linux (WSL) support. Provides abstraction layer for running external commands with proper error handling and platform-specific optimizations.

## Key Components

### Core Types
- **`CommandExecutor`** - Interface for executing external commands with dependency injection support
- **`RealCommandExecutor`** - Implementation using actual system commands with dry-run and verbose modes
- **`CommandResult`** - Execution result containing exit code, stdout, stderr, and duration
- **`WSLError`** - Specialized error type for WSL-specific failures with suggestions
- **`ExecuteOptions`** - Configuration for command execution with timeout, environment, and directory settings

### WSL Management Functions
- **`IsWSLAvailable()`** - Cached availability check for WSL
- **`IsWSLUbuntuAvailable()`** - Checks Ubuntu distribution accessibility
- **`WakeUpWSL()`** - Ensures WSL responsiveness before critical operations
- **`TryRecoverWSL()`** - Last-resort recovery by terminating and restarting WSL
- **`RestartDockerInWSL()`** - Restarts Docker daemon after WSL recovery

## Usage Example

```go
// Create command executor
executor := NewRealCommandExecutor(false, true) // dryRun=false, verbose=true

// Basic command execution
ctx := context.Background()
result, err := executor.Execute(ctx, "kubectl", "get", "pods")
if err != nil {
    if wslErr, ok := err.(*WSLError); ok {
        fmt.Printf("WSL Error: %s\nSuggestion: %s\n", wslErr.Error(), wslErr.Suggestion)
    }
    return err
}

// Advanced execution with options
options := ExecuteOptions{
    Command: "docker",
    Args:    []string{"ps", "-a"},
    Dir:     "/tmp",
    Env:     map[string]string{"DOCKER_HOST": "unix:///var/run/docker.sock"},
    Timeout: 30 * time.Second,
}
result, err := executor.ExecuteWithOptions(ctx, options)

// WSL management
if IsWSLAvailable() && !IsWSLUbuntuAvailable() {
    if err := TryRecoverWSL(); err != nil {
        log.Printf("WSL recovery failed: %v", err)
    }
}
```
