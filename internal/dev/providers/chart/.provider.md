Provides chart installation and management operations for OpenFrame development workflows with Helm values customization and Skaffold integration.

## Key Components

- **Provider**: Main struct that handles chart operations with command execution and verbose logging
- **NewProvider()**: Factory function to create a new Provider instance
- **InstallCharts()**: Installs charts on a cluster with custom Helm values for development
- **InstallChartsWithContext()**: Context-aware chart installation with cancellation support
- **PrepareDevHelmValues()**: Creates development-specific Helm values configuration
- **createDevHelmValuesFile()**: Generates Helm values with autoSync disabled for Skaffold workflows

## Usage Example

```go
// Create a new chart provider
exec := executor.NewCommandExecutor()
provider := chart.NewProvider(exec, true) // verbose=true

// Install charts with custom values
err := provider.InstallCharts("my-cluster", "custom-values.yaml")
if err != nil {
    log.Fatal("Failed to install charts:", err)
}

// Context-aware installation with timeout
ctx, cancel := context.WithTimeout(context.Background(), 5*time.Minute)
defer cancel()

err = provider.InstallChartsWithContext(ctx, "my-cluster", "dev-values.yaml")
if err != nil {
    log.Fatal("Chart installation failed:", err)
}

// Prepare development Helm values
devValuesPath, err := provider.PrepareDevHelmValues("base-values.yaml")
if err != nil {
    log.Fatal("Failed to prepare dev values:", err)
}
```

The Provider automatically disables autoSync in Helm values for Skaffold development workflows and provides aggressive cancellation handling to prevent hanging background processes during ArgoCD application management.
