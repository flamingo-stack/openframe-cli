Manages chart installation operations for development environments with Helm values customization and development-specific configurations. Provides context-aware installation with cancellation support and automatic creation of development Helm values files.

## Key Components

- **Provider**: Main struct providing chart installation operations with command execution and verbose logging
- **NewProvider()**: Constructor for creating a new chart provider instance
- **InstallCharts()**: Installs charts on a cluster using custom Helm values
- **InstallChartsWithContext()**: Context-aware chart installation with cancellation support
- **PrepareDevHelmValues()**: Prepares development-specific Helm values files
- **GetDefaultDevValues()**: Returns path to default development Helm values
- **createDevHelmValuesFile()**: Creates Helm values with development settings (autoSync disabled)

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
)

// Create a new chart provider
exec := executor.NewRealExecutor()
provider := NewProvider(exec, true) // verbose=true

// Install charts with custom values
err := provider.InstallCharts("my-cluster", "custom-values.yaml")
if err != nil {
    log.Fatal(err)
}

// Install with context for cancellation
ctx, cancel := context.WithTimeout(context.Background(), 5*time.Minute)
defer cancel()
err = provider.InstallChartsWithContext(ctx, "my-cluster", "values.yaml")

// Prepare development Helm values
devValuesPath, err := provider.PrepareDevHelmValues("base-values.yaml")
```
