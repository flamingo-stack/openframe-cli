This file provides chart installation operations specifically designed for development workflows, focusing on Helm chart management with Skaffold integration.

## Key Components

- **Provider struct**: Main chart provider with command executor and verbose logging support
- **NewProvider()**: Factory function to create a new chart provider instance
- **InstallCharts()**: Installs charts on a cluster with custom Helm values for development
- **InstallChartsWithContext()**: Context-aware chart installation with cancellation support
- **PrepareDevHelmValues()**: Creates development-specific Helm values files
- **validateHelmValuesFile()**: Validates existence and accessibility of Helm values files
- **GetDefaultDevValues()**: Returns the path to default development Helm values
- **createDevHelmValuesFile()**: Creates/merges Helm values with development settings (autoSync disabled)

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
    "github.com/flamingo-stack/openframe-cli/internal/chart"
)

// Create a new chart provider
exec := executor.NewRealCommandExecutor()
provider := chart.NewProvider(exec, true) // verbose=true

// Install charts for development
err := provider.InstallCharts("my-cluster", "custom-values.yaml")
if err != nil {
    log.Fatal(err)
}

// Install with context for cancellation
ctx, cancel := context.WithTimeout(context.Background(), 5*time.Minute)
defer cancel()

err = provider.InstallChartsWithContext(ctx, "my-cluster", "")
if err != nil {
    log.Fatal(err)
}

// Prepare development values
devValuesPath, err := provider.PrepareDevHelmValues("base-values.yaml")
```
