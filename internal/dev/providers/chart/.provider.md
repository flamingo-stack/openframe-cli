A chart provider implementation that manages Helm chart installations for development environments with autoSync disabled for Skaffold compatibility.

## Key Components

- **Provider** - Main struct providing chart installation operations with context support
- **NewProvider** - Constructor that accepts a command executor and verbose flag
- **InstallCharts/InstallChartsWithContext** - Installs charts with development-specific configurations
- **PrepareDevHelmValues** - Prepares development-specific Helm values files
- **createDevHelmValuesFile** - Creates/modifies Helm values with autoSync disabled for Skaffold workflow
- **validateHelmValuesFile** - Validates existence and accessibility of Helm values files

## Usage Example

```go
// Create a new chart provider
exec := executor.NewRealCommandExecutor()
provider := chart.NewProvider(exec, true)

// Install charts with custom values for development
err := provider.InstallCharts("my-cluster", "custom-values.yaml")
if err != nil {
    log.Fatal("Failed to install charts:", err)
}

// Prepare development values file
devValuesPath, err := provider.PrepareDevHelmValues("base-values.yaml")
if err != nil {
    log.Fatal("Failed to prepare dev values:", err)
}

// Install with context for cancellation support
ctx, cancel := context.WithTimeout(context.Background(), 5*time.Minute)
defer cancel()

err = provider.InstallChartsWithContext(ctx, "my-cluster", devValuesPath)
if err != nil {
    log.Fatal("Chart installation failed:", err)
}
```

The provider automatically disables autoSync in Helm values for development workflows and handles graceful cancellation of long-running chart installations.
