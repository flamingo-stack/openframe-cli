A chart management provider that handles Helm chart installations with development-specific configurations, particularly for disabling autoSync during Skaffold workflows.

## Key Components

- **Provider**: Main struct that manages chart installation operations with configurable verbosity
- **NewProvider()**: Constructor function that initializes a new Provider instance
- **InstallCharts()/InstallChartsWithContext()**: Core methods for installing charts with context cancellation support
- **PrepareDevHelmValues()**: Prepares development-specific Helm values files
- **validateHelmValuesFile()**: Validates existence and accessibility of Helm values files
- **GetDefaultDevValues()**: Returns the appropriate default development values file path
- **createDevHelmValuesFile()**: Creates/modifies Helm values files with development settings (autoSync disabled)

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
)

// Create a new chart provider
exec := executor.NewRealExecutor()
provider := NewProvider(exec, true) // verbose mode enabled

// Install charts with custom values
err := provider.InstallCharts("my-cluster", "custom-values.yaml")
if err != nil {
    log.Fatal("Chart installation failed:", err)
}

// Install charts with context cancellation
ctx, cancel := context.WithTimeout(context.Background(), 5*time.Minute)
defer cancel()

err = provider.InstallChartsWithContext(ctx, "my-cluster", "helm-values.yaml")
if err != nil {
    log.Printf("Installation error: %v", err)
}

// Prepare development values
devValuesPath, err := provider.PrepareDevHelmValues("base-values.yaml")
if err != nil {
    log.Fatal("Failed to prepare dev values:", err)
}
```
