Test suite for the chart provider functionality, covering Helm chart management operations including validation, file handling, and installation workflows.

## Key Components

- **TestNewProvider**: Tests provider initialization with executor and verbose mode configuration
- **TestProvider_ValidateHelmValuesFile**: Validates Helm values file existence and path handling
- **TestProvider_PrepareDevHelmValues**: Tests development-specific Helm values preparation logic
- **TestProvider_GetDefaultDevValues**: Tests default development values file resolution
- **TestProvider_InstallCharts**: Tests chart installation workflow with validation
- **TestProvider_VerboseLogging**: Validates verbose mode behavior across operations

## Usage Example

```go
func TestExample(t *testing.T) {
    testutil.InitializeTestMode()
    
    mockExecutor := testutil.NewTestMockExecutor()
    provider := NewProvider(mockExecutor, true)
    
    // Test values file validation
    err := provider.validateHelmValuesFile("values.yaml")
    assert.Error(t, err) // File doesn't exist
    
    // Test default values resolution
    defaultPath := provider.GetDefaultDevValues()
    assert.Equal(t, "helm-values.yaml", defaultPath)
    
    // Test development values preparation
    devPath, err := provider.PrepareDevHelmValues("")
    assert.NoError(t, err)
    assert.Equal(t, "helm-values.yaml", devPath)
}
```
