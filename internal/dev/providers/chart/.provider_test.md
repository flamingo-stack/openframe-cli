Test file for the chart provider functionality, covering validation, configuration, and installation logic for Helm chart operations.

## Key Components

- **NewProvider Tests**: Validates provider initialization with executor and verbose flag settings
- **ValidateHelmValuesFile Tests**: Tests file existence validation for Helm values files with various scenarios (empty paths, missing files, valid files)
- **PrepareDevHelmValues Tests**: Verifies development values file preparation and fallback behavior
- **GetDefaultDevValues Tests**: Checks default values file detection logic, including dev-specific file preference
- **InstallCharts Tests**: Basic validation testing for chart installation workflow
- **Verbose Logging Tests**: Ensures verbose mode configuration works correctly

## Usage Example

```go
func TestMyChart(t *testing.T) {
    testutil.InitializeTestMode()
    mockExecutor := testutil.NewTestMockExecutor()
    provider := NewProvider(mockExecutor, true)

    // Test values file validation
    err := provider.validateHelmValuesFile("my-values.yaml")
    assert.NoError(t, err)

    // Test default values detection
    defaultFile := provider.GetDefaultDevValues()
    assert.Equal(t, "helm-values.yaml", defaultFile)

    // Test dev values preparation
    valuesPath, err := provider.PrepareDevHelmValues("custom-values.yaml")
    assert.NoError(t, err)
}
```
