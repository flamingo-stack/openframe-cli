Unit test file for the Telepresence provider implementation, containing comprehensive test coverage for all provider functionality including intercept creation, connection management, and error handling scenarios.

## Key Components

- **TestNewProvider** - Tests provider initialization with mock executor
- **TestProvider_CheckTelepresenceInstallation** - Validates Telepresence installation detection
- **TestProvider_ConnectToCluster** - Tests Kubernetes cluster connection
- **TestProvider_CreateIntercept** - Tests service intercept creation with various configurations
- **TestProvider_CreateInterceptWithEnvFile** - Tests intercept creation with environment files
- **TestProvider_TeardownIntercept** - Tests intercept cleanup operations
- **TestProvider_Disconnect** - Tests Telepresence disconnection
- **TestProvider_ShowInterceptStatus** - Tests intercept status display
- **TestProvider_SetupIntercept_Integration** - End-to-end integration test
- **TestProvider_VerboseLogging** - Tests verbose mode configuration

## Usage Example

```go
func TestMyProvider(t *testing.T) {
    testutil.InitializeTestMode()
    mockExecutor := testutil.NewTestMockExecutor()
    
    // Create provider instance
    provider := NewProvider(mockExecutor, true)
    
    // Mock successful telepresence version check
    mockExecutor.SetResponse("telepresence version", &executor.CommandResult{
        ExitCode: 0,
        Stdout:   "Telepresence 2.19.1",
    })
    
    // Test installation check
    err := provider.checkTelepresenceInstallation(context.Background())
    assert.NoError(t, err)
}
```
