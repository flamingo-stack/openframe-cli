Provides utilities for managing and discovering Kubernetes namespaces, including namespace validation and automatic resource-to-namespace resolution for service discovery.

## Key Components

- **GetNamespaces** - Retrieves all available namespaces in the current cluster
- **ValidateNamespace** - Verifies that a specified namespace exists
- **FindResourceNamespace** - Automatically discovers which namespace contains resources matching a service name
- **searchResourceByType** - Helper method that searches for specific resource types across all namespaces

## Usage Example

```go
// Get all namespaces
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Available namespaces: %v\n", namespaces)

// Validate a specific namespace exists
if err := provider.ValidateNamespace(ctx, "production"); err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Find which namespace contains resources for a service
namespace, err := provider.FindResourceNamespace(ctx, "web-service")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service 'web-service' found in namespace: %s\n", namespace)
```

The `FindResourceNamespace` method is particularly useful for service discovery, as it searches through deployments, statefulsets, and pods to automatically determine where a service's resources are located, falling back to the "default" namespace if no matches are found.
