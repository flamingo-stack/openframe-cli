This file provides Kubernetes namespace utilities for discovering and validating namespaces and locating resources by service name.

## Key Components

- **GetNamespaces()** - Retrieves all namespaces in the current cluster
- **ValidateNamespace()** - Verifies if a specific namespace exists
- **FindResourceNamespace()** - Searches for Kubernetes resources matching a service name and returns their namespace
- **searchResourceByType()** - Helper method that searches for specific resource types (deployments, statefulsets, pods) across all namespaces

## Usage Example

```go
// Get all namespaces
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Println("Available namespaces:", namespaces)

// Validate a namespace exists
err = provider.ValidateNamespace(ctx, "production")
if err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Find which namespace contains resources for a service
namespace, err := provider.FindResourceNamespace(ctx, "my-app")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service 'my-app' found in namespace: %s\n", namespace)
```

The `FindResourceNamespace` method intelligently searches across deployments, statefulsets, and pods to locate resources matching the service name, supporting both exact matches and prefix matching patterns.
