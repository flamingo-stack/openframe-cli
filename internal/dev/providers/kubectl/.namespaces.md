A Kubernetes namespace discovery utility that provides methods for listing, validating, and finding namespaces for resources in a cluster.

## Key Components

- **`GetNamespaces()`** - Retrieves all namespaces in the current cluster
- **`ValidateNamespace()`** - Checks if a specific namespace exists
- **`FindResourceNamespace()`** - Searches for Kubernetes resources by service name and returns their namespace
- **`searchResourceByType()`** - Internal helper that searches for specific resource types (deployments, statefulsets, pods) across all namespaces

## Usage Example

```go
// Get all namespaces
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Available namespaces: %v\n", namespaces)

// Validate a namespace exists
err = provider.ValidateNamespace(ctx, "production")
if err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Find namespace containing a service
namespace, err := provider.FindResourceNamespace(ctx, "web-server")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service 'web-server' found in namespace: %s\n", namespace)
```

The `FindResourceNamespace()` method intelligently searches for deployments, statefulsets, and pods that match the service name (exact match or prefix match with hyphen), returning "default" if no matching resources are found.
