Manages Kubernetes namespace operations including namespace discovery, validation, and automatic resource-to-namespace mapping for kubectl commands.

## Key Components

- **GetNamespaces()** - Retrieves all available namespaces in the current cluster
- **ValidateNamespace()** - Verifies if a specified namespace exists
- **FindResourceNamespace()** - Automatically discovers the namespace for a service by searching deployments, statefulsets, and pods
- **searchResourceByType()** - Internal helper that searches for specific resource types across all namespaces
- **resourceList** - JSON unmarshaling struct for parsing kubectl output

## Usage Example

```go
// Get all namespaces in the cluster
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatalf("Failed to get namespaces: %v", err)
}
fmt.Printf("Available namespaces: %v\n", namespaces)

// Validate a specific namespace exists
err = provider.ValidateNamespace(ctx, "production")
if err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Automatically find which namespace contains a service
namespace, err := provider.FindResourceNamespace(ctx, "web-app")
if err != nil {
    log.Fatalf("Failed to find namespace: %v", err)
}
fmt.Printf("Service 'web-app' found in namespace: %s\n", namespace)
```

The namespace discovery feature is particularly useful for OpenFrame's AI-driven operations, allowing automatic resource location without requiring users to specify explicit namespace parameters. The search logic checks for exact matches and common naming patterns (service-name prefix matching).
