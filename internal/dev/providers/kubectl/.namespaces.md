Provides Kubernetes namespace management functionality including listing, validation, and intelligent service-to-namespace discovery for OpenFrame's kubectl provider.

## Key Components

- **GetNamespaces()** - Retrieves all namespaces in the current cluster
- **ValidateNamespace()** - Checks if a specific namespace exists
- **FindResourceNamespace()** - Intelligently discovers which namespace contains resources for a given service name
- **searchResourceByType()** - Internal helper that searches for specific resource types across all namespaces
- **resourceList** - JSON parsing structure for Kubernetes resource metadata

## Usage Example

```go
provider := &Provider{
    executor: someExecutor,
    verbose: true,
}

// List all namespaces
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatal(err)
}

// Validate a namespace exists
err = provider.ValidateNamespace(ctx, "production")
if err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Find namespace for a service (searches deployments, statefulsets, pods)
namespace, err := provider.FindResourceNamespace(ctx, "my-app")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service 'my-app' found in namespace: %s\n", namespace)
```

The `FindResourceNamespace` method is particularly useful for auto-discovery scenarios where you need to locate resources by service name without knowing the exact namespace, supporting both exact matches and prefix-based matching patterns.
