This file provides namespace management functionality for Kubernetes clusters, including retrieval, validation, and resource-based namespace discovery.

## Key Components

- **GetNamespaces**: Retrieves all available namespaces in the cluster
- **ValidateNamespace**: Verifies if a specific namespace exists
- **FindResourceNamespace**: Intelligently locates the namespace containing resources for a given service name
- **searchResourceByType**: Helper function that searches for specific resource types across all namespaces

## Usage Example

```go
// Get all namespaces
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Println("Available namespaces:", namespaces)

// Validate a namespace exists
err = provider.ValidateNamespace(ctx, "production")
if err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Find namespace for a service
namespace, err := provider.FindResourceNamespace(ctx, "my-service")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service found in namespace: %s", namespace)
```

The `FindResourceNamespace` function is particularly useful for automatically discovering where services are deployed by searching through deployments, statefulsets, and pods with name matching or prefix matching logic. It defaults to the "default" namespace when no matching resources are found.
