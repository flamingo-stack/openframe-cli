This file provides Kubernetes namespace discovery and validation functionality using kubectl commands through a Provider interface.

## Key Components

- **GetNamespaces**: Retrieves all available namespaces in the current Kubernetes cluster
- **ValidateNamespace**: Checks if a specific namespace exists
- **FindResourceNamespace**: Intelligently discovers the namespace containing resources for a given service name
- **searchResourceByType**: Internal helper that searches for specific resource types (deployments, statefulsets, pods) across all namespaces

## Usage Example

```go
provider := &Provider{executor: executor, verbose: true}

// Get all namespaces
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Available namespaces: %v\n", namespaces)

// Validate a namespace exists
err = provider.ValidateNamespace(ctx, "production")
if err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Find namespace for a service
namespace, err := provider.FindResourceNamespace(ctx, "my-app")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service 'my-app' found in namespace: %s\n", namespace)
```

The FindResourceNamespace method is particularly useful for auto-discovery scenarios where you need to locate Kubernetes resources by service name without knowing the exact namespace.
