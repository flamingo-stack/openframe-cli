This file provides service management functionality for Kubernetes clusters through kubectl commands, enabling retrieval and validation of services with fallback mechanisms for reliable operation.

## Key Components

- **GetServices()** - Retrieves all services in a specified namespace, returning `[]intercept.ServiceInfo`
- **GetService()** - Fetches details for a specific service by name and namespace
- **ValidateService()** - Checks if a service exists in a given namespace
- **getServicesSimple()** - Fallback method using simpler kubectl commands when JSON parsing fails
- **convertJSONToServiceInfo()** - Converts JSON service data to `intercept.ServiceInfo` struct

## Usage Example

```go
// Initialize provider (assuming kubectl.Provider exists)
provider := &kubectl.Provider{...}

// Get all services in a namespace
services, err := provider.GetServices(ctx, "default")
if err != nil {
    log.Fatal(err)
}

// Get specific service details
service, err := provider.GetService(ctx, "default", "my-service")
if err != nil {
    log.Fatal(err)
}

// Validate service exists
err = provider.ValidateService(ctx, "default", "my-service")
if err != nil {
    log.Printf("Service validation failed: %v", err)
}
```

The implementation uses JSON output parsing with automatic fallback to simpler kubectl commands for enhanced reliability across different Kubernetes environments.
