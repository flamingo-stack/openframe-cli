Kubernetes service management utilities that provide high-level operations for retrieving and validating services through kubectl commands with JSON parsing and fallback mechanisms.

## Key Components

- **GetServices()** - Retrieves all services in a namespace with detailed port information
- **GetService()** - Gets details for a specific service by name
- **ValidateService()** - Checks if a service exists in a given namespace
- **getServicesSimple()** - Fallback method using basic kubectl commands when JSON parsing fails
- **getServiceSimple()** - Fallback method for individual service retrieval
- **convertJSONToServiceInfo()** - Converts kubectl JSON output to structured ServiceInfo objects

## Usage Example

```go
provider := &kubectl.Provider{
    executor: commandExecutor,
    verbose:  true,
}

// Get all services in a namespace
services, err := provider.GetServices(ctx, "default")
if err != nil {
    log.Fatal(err)
}

// Get specific service details
service, err := provider.GetService(ctx, "default", "my-service")
if err != nil {
    log.Fatal(err)
}

// Validate service exists
err = provider.ValidateService(ctx, "default", "my-service")
if err != nil {
    log.Printf("Service not found: %v", err)
}
```

The implementation uses JSON output parsing for structured data retrieval with automatic fallback to simpler kubectl commands if JSON parsing fails, ensuring reliability across different Kubernetes configurations.
