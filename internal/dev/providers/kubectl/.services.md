This service provides Kubernetes service discovery functionality through kubectl commands, offering both JSON-based parsing and fallback methods for retrieving service information.

## Key Components

- **GetServices()** - Retrieves all services in a namespace with JSON parsing and fallback support
- **GetService()** - Gets detailed information about a specific service
- **ValidateService()** - Checks if a service exists in a namespace
- **getServicesSimple()** - Fallback method using basic kubectl commands when JSON parsing fails
- **getServiceSimple()** - Fallback method for single service retrieval
- **convertJSONToServiceInfo()** - Converts JSON service data to standardized ServiceInfo format

## Usage Example

```go
// Get all services in a namespace
services, err := provider.GetServices(ctx, "default")
if err != nil {
    log.Fatal(err)
}

// Get a specific service
service, err := provider.GetService(ctx, "default", "my-service")
if err != nil {
    log.Fatal(err)
}

// Validate service exists
err = provider.ValidateService(ctx, "default", "my-service")
if err != nil {
    log.Printf("Service not found: %v", err)
}
```

The service implements robust error handling with JSON parsing as the primary method and simple kubectl commands as fallbacks, ensuring compatibility across different Kubernetes environments and kubectl versions.
