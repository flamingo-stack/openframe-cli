This file implements Kubernetes service management functionality for the kubectl provider, offering both JSON-based and fallback parsing methods for retrieving service information.

## Key Components

- **GetServices()** - Retrieves all services in a namespace with JSON parsing and fallback support
- **GetService()** - Gets detailed information about a specific service
- **ValidateService()** - Checks if a service exists in the given namespace
- **getServicesSimple()** - Fallback method for service listing when JSON parsing fails
- **getServiceSimple()** - Fallback method for single service queries
- **convertJSONToServiceInfo()** - Converts kubectl JSON output to ServiceInfo structs

## Usage Example

```go
// Get all services in a namespace
services, err := provider.GetServices(ctx, "default")
if err != nil {
    return fmt.Errorf("failed to list services: %w", err)
}

// Get specific service details
service, err := provider.GetService(ctx, "default", "my-service")
if err != nil {
    return fmt.Errorf("service not found: %w", err)
}

// Validate service exists before operations
if err := provider.ValidateService(ctx, "default", "my-service"); err != nil {
    return fmt.Errorf("validation failed: %w", err)
}

// Access service information
for _, svc := range services {
    fmt.Printf("Service: %s, Type: %s, Ports: %v\n", 
        svc.Name, svc.Type, svc.Ports)
}
```

The implementation provides robust error handling with automatic fallback to simpler kubectl commands when JSON parsing fails, ensuring compatibility across different Kubernetes versions and configurations.
