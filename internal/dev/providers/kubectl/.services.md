This file provides Kubernetes service management functionality, enabling retrieval and validation of services within namespaces using kubectl commands.

## Key Components

- **GetServices**: Retrieves all services in a namespace, with JSON parsing and fallback to simple parsing
- **GetService**: Gets detailed information about a specific service by name
- **ValidateService**: Checks if a service exists in a given namespace
- **getServicesSimple/getServiceSimple**: Fallback methods using simpler kubectl commands when JSON parsing fails
- **convertJSONToServiceInfo**: Converts JSON service data to structured ServiceInfo objects

## Usage Example

```go
provider := &Provider{executor: executor, verbose: true}

// Get all services in a namespace
services, err := provider.GetServices(ctx, "default")
if err != nil {
    return fmt.Errorf("failed to get services: %w", err)
}

// Get specific service details
service, err := provider.GetService(ctx, "default", "my-service")
if err != nil {
    return fmt.Errorf("service not found: %w", err)
}

// Validate service exists
if err := provider.ValidateService(ctx, "default", "my-service"); err != nil {
    return fmt.Errorf("service validation failed: %w", err)
}
```

The implementation uses JSON output for reliable parsing with graceful fallbacks to simple kubectl commands when needed, ensuring robust service discovery across different Kubernetes configurations.
