Provides Kubernetes service discovery and management functionality through kubectl commands, with robust JSON parsing and fallback mechanisms for service information retrieval.

## Key Components

- **GetServices()** - Retrieves all services in a namespace with full details
- **GetService()** - Fetches information about a specific service
- **ValidateService()** - Checks if a service exists in a namespace
- **getServicesSimple()** - Fallback method using simpler kubectl commands
- **getServiceSimple()** - Fallback method for single service retrieval
- **convertJSONToServiceInfo()** - Converts kubectl JSON output to ServiceInfo structs

## Usage Example

```go
// Initialize provider (assuming it exists)
provider := &kubectl.Provider{}

// Get all services in a namespace
services, err := provider.GetServices(ctx, "default")
if err != nil {
    log.Fatal(err)
}

for _, svc := range services {
    fmt.Printf("Service: %s, Type: %s, Ports: %v\n", 
        svc.Name, svc.Type, svc.Ports)
}

// Get specific service details
service, err := provider.GetService(ctx, "default", "my-service")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service %s has %d ports\n", service.Name, len(service.Ports))

// Validate service exists
if err := provider.ValidateService(ctx, "default", "my-service"); err != nil {
    fmt.Printf("Service validation failed: %v\n", err)
}
```

The implementation prioritizes JSON parsing for reliability but includes fallback methods for environments where JSON output might fail, ensuring robust service discovery across different Kubernetes configurations.
