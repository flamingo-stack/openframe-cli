This file defines JSON structures for parsing Kubernetes service data returned by kubectl commands.

## Key Components

- **`serviceJSON`** - Structure for parsing individual Kubernetes service objects with metadata (name, namespace) and spec (type, ports configuration)
- **`serviceListJSON`** - Wrapper structure for parsing lists of services returned by kubectl list operations

## Usage Example

```go
// Parse a single service from kubectl get service output
var service serviceJSON
err := json.Unmarshal(kubectlOutput, &service)
if err != nil {
    return err
}

fmt.Printf("Service: %s in namespace %s\n", 
    service.Metadata.Name, 
    service.Metadata.Namespace)

// Parse multiple services from kubectl get services output
var serviceList serviceListJSON
err = json.Unmarshal(kubectlListOutput, &serviceList)
if err != nil {
    return err
}

for _, svc := range serviceList.Items {
    fmt.Printf("Found service: %s with %d ports\n", 
        svc.Metadata.Name, 
        len(svc.Spec.Ports))
}
```

The structures handle the common kubectl JSON output format, with `TargetPort` as `interface{}` to accommodate both string and numeric port specifications that Kubernetes allows.
