This file contains comprehensive unit tests for JSON unmarshaling of Kubernetes service structures, validating that the package's data types can properly deserialize kubectl command output.

## Key Components

- **TestServiceJSON_UnmarshalJSON**: Tests unmarshaling of individual Kubernetes service JSON objects
- **TestServiceListJSON_UnmarshalJSON**: Tests unmarshaling of service list responses from kubectl
- **TestServiceJSON_RealWorldExamples**: Tests with realistic service configurations including web apps, databases, and headless services

## Usage Example

Run the tests to verify JSON parsing functionality:

```go
// Run all tests
go test -v ./kubectl

// Run specific test cases
go test -run TestServiceJSON_UnmarshalJSON -v
go test -run TestServiceListJSON_UnmarshalJSON -v
go test -run TestServiceJSON_RealWorldExamples -v
```

The tests cover various scenarios including:
- Single and multi-port services
- Different service types (ClusterIP, NodePort, LoadBalancer)
- Named and unnamed ports
- Numeric and string target ports
- Empty service lists and malformed JSON
- Real-world service configurations for web applications, databases, and distributed systems

Each test validates that the `serviceJSON` and `serviceListJSON` structures correctly unmarshal kubectl's JSON output format.
