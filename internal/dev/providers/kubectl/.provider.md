Provides a kubectl-based implementation for Kubernetes cluster operations and context management within the OpenFrame CLI development environment.

## Key Components

- **Provider**: Main struct that wraps kubectl commands through a command executor interface
- **NewProvider()**: Constructor function that creates a new Provider instance with executor and verbose options
- **CheckConnection()**: Verifies kubectl connectivity to a Kubernetes cluster
- **GetCurrentContext()**: Retrieves the active kubectl context name
- **SetContext()**: Switches kubectl to a different context
- **parseTargetPort()**: Helper method for parsing port values from various data types

The Provider implements both `intercept.KubernetesClient` and `intercept.ServiceClient` interfaces, making it suitable for service interception and Kubernetes operations.

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
    "github.com/flamingo-stack/openframe-cli/internal/dev/services/kubectl"
)

// Create a new kubectl provider
exec := executor.NewCommandExecutor()
provider := kubectl.NewProvider(exec, true) // verbose mode enabled

// Check cluster connectivity
ctx := context.Background()
if err := provider.CheckConnection(ctx); err != nil {
    log.Fatal("No cluster connection:", err)
}

// Get current context
context, err := provider.GetCurrentContext(ctx)
if err != nil {
    log.Fatal("Failed to get context:", err)
}

// Switch contexts
err = provider.SetContext(ctx, "production-cluster")
if err != nil {
    log.Fatal("Failed to switch context:", err)
}
```
