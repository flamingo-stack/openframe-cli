A kubectl provider that implements Kubernetes cluster operations through the kubectl command-line tool, serving as an adapter for executing cluster management commands.

## Key Components

- **Provider** - Main struct implementing both `KubernetesClient` and `ServiceClient` interfaces
- **NewProvider()** - Constructor function accepting a command executor and verbose flag
- **CheckConnection()** - Verifies kubectl connectivity to a Kubernetes cluster
- **GetCurrentContext()** - Retrieves the active kubectl context name
- **SetContext()** - Switches kubectl to a different context
- **parseTargetPort()** - Helper method for converting various port types to strings

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
)

// Create a new kubectl provider
exec := executor.NewCommandExecutor()
provider := NewProvider(exec, true) // verbose mode enabled

// Check cluster connection
ctx := context.Background()
if err := provider.CheckConnection(ctx); err != nil {
    log.Fatal("No cluster connection:", err)
}

// Get current context
context, err := provider.GetCurrentContext(ctx)
if err != nil {
    log.Fatal("Failed to get context:", err)
}
fmt.Printf("Current context: %s\n", context)

// Switch contexts
err = provider.SetContext(ctx, "production-cluster")
if err != nil {
    log.Fatal("Failed to switch context:", err)
}
```

The provider abstracts kubectl operations behind a clean interface, enabling dependency injection and testability while maintaining compatibility with standard Kubernetes tooling.
