This file provides a kubectl-based implementation for Kubernetes operations, serving as a provider that wraps kubectl command execution for cluster management and service operations.

## Key Components

- **`Provider`** - Main struct that implements Kubernetes operations using kubectl commands
- **`NewProvider()`** - Factory function to create a new kubectl provider instance
- **`CheckConnection()`** - Verifies kubectl connectivity to a Kubernetes cluster
- **`GetCurrentContext()`** - Retrieves the currently active kubectl context
- **`SetContext()`** - Switches between different kubectl contexts
- **`parseTargetPort()`** - Helper method to normalize port values from different data types

The provider implements both `intercept.KubernetesClient` and `intercept.ServiceClient` interfaces, making it suitable for service interception and general Kubernetes operations.

## Usage Example

```go
package main

import (
    "context"
    "log"
    
    "github.com/flamingo-stack/openframe-cli/internal/dev/providers/kubectl"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
)

func main() {
    // Create executor and provider
    exec := executor.NewCommandExecutor()
    provider := kubectl.NewProvider(exec, true) // verbose mode
    
    ctx := context.Background()
    
    // Check cluster connection
    if err := provider.CheckConnection(ctx); err != nil {
        log.Fatalf("Cluster connection failed: %v", err)
    }
    
    // Get current context
    context, err := provider.GetCurrentContext(ctx)
    if err != nil {
        log.Fatalf("Failed to get context: %v", err)
    }
    
    log.Printf("Current context: %s", context)
}
```
