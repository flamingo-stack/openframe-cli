This file implements a kubectl provider that wraps kubectl commands for Kubernetes operations and service management within the OpenFrame CLI development tools.

## Key Components

- **Provider struct**: Main implementation that wraps a command executor and verbose flag
- **NewProvider()**: Constructor function for creating a new kubectl provider instance
- **CheckConnection()**: Verifies kubectl connectivity to a Kubernetes cluster
- **GetCurrentContext()**: Retrieves the current kubectl context name
- **SetContext()**: Switches kubectl to a different context
- **parseTargetPort()**: Helper method for parsing port values from various types

The Provider implements both `intercept.KubernetesClient` and `intercept.ServiceClient` interfaces for service interception capabilities.

## Usage Example

```go
// Create a new kubectl provider
exec := executor.NewCommandExecutor()
provider := kubectl.NewProvider(exec, true)

// Check cluster connection
ctx := context.Background()
if err := provider.CheckConnection(ctx); err != nil {
    log.Fatal("Not connected to cluster:", err)
}

// Get current context
context, err := provider.GetCurrentContext(ctx)
if err != nil {
    log.Fatal("Failed to get context:", err)
}
fmt.Println("Current context:", context)

// Switch context
err = provider.SetContext(ctx, "production-cluster")
if err != nil {
    log.Fatal("Failed to switch context:", err)
}
```
