This file implements a kubectl provider that wraps Kubernetes operations using the kubectl command-line tool. It serves as an adapter between the OpenFrame CLI's internal interfaces and kubectl commands.

## Key Components

### Types
- **`Provider`** - Main struct that implements Kubernetes operations via kubectl
  - `executor` - Command executor for running kubectl commands
  - `verbose` - Flag for detailed output

### Functions
- **`NewProvider()`** - Constructor for creating new kubectl provider instances
- **`CheckConnection()`** - Verifies kubectl can connect to a cluster
- **`GetCurrentContext()`** - Retrieves the current kubectl context name
- **`SetContext()`** - Switches to a different kubectl context
- **`parseTargetPort()`** - Helper method for parsing port values from different types

### Interface Implementations
- Implements `intercept.KubernetesClient` interface
- Implements `intercept.ServiceClient` interface

## Usage Example

```go
// Create executor and provider
exec := executor.NewCommandExecutor()
provider := NewProvider(exec, true) // verbose=true

// Check cluster connection
err := provider.CheckConnection(ctx)
if err != nil {
    log.Fatal("Cannot connect to cluster:", err)
}

// Get current context
context, err := provider.GetCurrentContext(ctx)
if err != nil {
    log.Fatal("Failed to get context:", err)
}
fmt.Printf("Current context: %s\n", context)

// Switch contexts
err = provider.SetContext(ctx, "production")
if err != nil {
    log.Fatal("Failed to switch context:", err)
}
```
