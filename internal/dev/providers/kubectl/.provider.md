This file implements a kubectl provider that wraps kubectl command execution to provide Kubernetes cluster operations through a structured Go interface.

## Key Components

- **Provider struct**: Main implementation that wraps a command executor for kubectl operations
- **NewProvider()**: Constructor function that creates a new kubectl provider instance
- **CheckConnection()**: Verifies kubectl connectivity to a Kubernetes cluster
- **GetCurrentContext()**: Retrieves the currently active kubectl context
- **SetContext()**: Switches to a different kubectl context
- **parseTargetPort()**: Helper method for parsing port values from various data types

The provider implements both `intercept.KubernetesClient` and `intercept.ServiceClient` interfaces.

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
    "github.com/flamingo-stack/openframe-cli/internal/dev/services/kubectl"
)

// Create a kubectl provider
exec := executor.NewCommandExecutor()
provider := kubectl.NewProvider(exec, true) // verbose=true

ctx := context.Background()

// Check cluster connection
if err := provider.CheckConnection(ctx); err != nil {
    log.Fatal("Cannot connect to cluster:", err)
}

// Get current context
context, err := provider.GetCurrentContext(ctx)
if err != nil {
    log.Fatal("Failed to get context:", err)
}

// Switch context
err = provider.SetContext(ctx, "production-cluster")
```
