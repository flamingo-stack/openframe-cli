A Kubernetes operations provider that implements kubectl-based interactions for cluster management and service operations.

## Key Components

- **Provider**: Main struct that wraps a command executor to perform kubectl operations
- **NewProvider()**: Constructor function that creates a new kubectl provider instance
- **CheckConnection()**: Verifies kubectl connectivity to a Kubernetes cluster
- **GetCurrentContext()**: Retrieves the active kubectl context name
- **SetContext()**: Switches kubectl to a different context
- **parseTargetPort()**: Helper method to convert various port types to string format

The provider implements both `intercept.KubernetesClient` and `intercept.ServiceClient` interfaces.

## Usage Example

```go
// Create a new kubectl provider
exec := executor.NewCommandExecutor()
provider := kubectl.NewProvider(exec, true)

// Check cluster connection
ctx := context.Background()
if err := provider.CheckConnection(ctx); err != nil {
    log.Fatal("Not connected to cluster:", err)
}

// Get current context
context, err := provider.GetCurrentContext(ctx)
if err != nil {
    log.Fatal("Failed to get context:", err)
}
fmt.Printf("Current context: %s\n", context)

// Switch context
err = provider.SetContext(ctx, "production-cluster")
if err != nil {
    log.Fatal("Failed to switch context:", err)
}
```
