Test suite for kubectl namespace operations, validating the Provider's GetNamespaces and ValidateNamespace methods using mocked command execution.

## Key Components

- **TestProvider_GetNamespaces**: Tests namespace retrieval functionality including multiple namespaces, single namespace, empty clusters, command failures, and malformed output scenarios
- **TestProvider_ValidateNamespace**: Tests namespace validation for existing/non-existing namespaces, connection errors, and system namespaces
- **Mock Setup Functions**: Configure mock executor responses for different test scenarios
- **Test Utilities**: Uses `testutil` package for test initialization and mock creation

## Usage Example

```go
// Example test structure for namespace operations
func TestCustomNamespaceOperation(t *testing.T) {
    testutil.InitializeTestMode()
    mockExecutor := testutil.NewTestMockExecutor()
    provider := NewProvider(mockExecutor, false)

    // Setup mock response
    mockExecutor.SetResponse("kubectl get namespaces", &executor.CommandResult{
        ExitCode: 0,
        Stdout:   "default production staging",
    })

    // Test namespace retrieval
    namespaces, err := provider.GetNamespaces(context.Background())
    assert.NoError(t, err)
    assert.Equal(t, []string{"default", "production", "staging"}, namespaces)

    // Test namespace validation
    err = provider.ValidateNamespace(context.Background(), "production")
    assert.NoError(t, err)
}
```
