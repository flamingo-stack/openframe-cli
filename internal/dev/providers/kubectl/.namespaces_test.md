Test suite for the kubectl provider's namespace management functionality. This file contains comprehensive unit tests for retrieving and validating Kubernetes namespaces using mocked command execution.

## Key Components

- **TestProvider_GetNamespaces**: Tests the `GetNamespaces()` method that retrieves all available namespaces from a Kubernetes cluster
- **TestProvider_ValidateNamespace**: Tests the `ValidateNamespace()` method that checks if a specific namespace exists
- **Mock setup functions**: Configure test scenarios with various kubectl command responses and failure conditions
- **Test scenarios**: Cover multiple namespaces, single namespace, empty clusters, command failures, malformed output, and whitespace handling

## Usage Example

```go
func TestProvider_GetNamespaces(t *testing.T) {
    testutil.InitializeTestMode()
    mockExecutor := testutil.NewTestMockExecutor()
    provider := NewProvider(mockExecutor, false)

    // Setup mock response for kubectl command
    mockExecutor.SetResponse("kubectl get namespaces", &executor.CommandResult{
        ExitCode: 0,
        Stdout:   "default kube-system openframe",
    })

    namespaces, err := provider.GetNamespaces(context.Background())
    
    assert.NoError(t, err)
    assert.Equal(t, []string{"default", "kube-system", "openframe"}, namespaces)
}
```

The tests validate proper parsing of kubectl output, error handling for non-existent namespaces, and correct command execution with appropriate arguments.
