This file implements the core Skaffold development workflow service for the OpenFrame CLI, orchestrating the complete development cycle including prerequisite checks, cluster setup, chart installation, and Skaffold execution.

## Key Components

- **Service** - Main service struct managing Skaffold workflow orchestration
- **NewService()** - Constructor creating a new scaffold service instance
- **RunScaffoldWorkflow()** - Primary method executing the complete development workflow
- **checkPrerequisites()** - Validates and optionally installs Skaffold
- **runSkaffoldDev()** - Executes Skaffold dev command with retry logic and signal handling
- **ShowSkaffoldConfigInfoAndSelectService()** - Interactive service selection interface
- **setupSignalHandler()** - Graceful shutdown handling for development sessions

## Usage Example

```go
// Create a scaffold service
executor := executor.NewRealCommandExecutor()
service := scaffold.NewService(executor, true) // verbose mode

// Define flags
flags := &models.ScaffoldFlags{
    Namespace:      "my-namespace",
    HelmValuesFile: "values.yaml",
    SkipBootstrap:  false,
}

// Run complete workflow
ctx := context.Background()
args := []string{"my-cluster"} // optional cluster name
err := service.RunScaffoldWorkflow(ctx, args, flags)
if err != nil {
    log.Fatal(err)
}

// Check if running
if service.IsRunning() {
    service.Stop()
}
```

The service provides a complete development workflow with prerequisite validation, interactive configuration selection, chart installation with timeouts, and robust Skaffold execution with retry logic and signal handling.
