Provides Skaffold development workflow orchestration for OpenFrame CLI, managing the complete development lifecycle from prerequisite checks to cluster deployment.

## Key Components

- **Service**: Main orchestrator that coordinates Skaffold development workflows
- **RunScaffoldWorkflow**: Executes the complete development pipeline including service selection, cluster setup, and Skaffold execution
- **checkPrerequisites**: Validates and optionally installs Skaffold binary
- **runSkaffoldDev**: Manages Skaffold development session with retry logic and signal handling

## Usage Example

```go
// Create a new scaffold service
executor := &executor.RealCommandExecutor{}
service := NewService(executor, true) // verbose=true

// Run the complete workflow
ctx := context.Background()
args := []string{"my-cluster"}
flags := &models.ScaffoldFlags{
    SkipBootstrap: false,
    HelmValuesFile: "values.yaml",
    Namespace: "development",
}

err := service.RunScaffoldWorkflow(ctx, args, flags)
if err != nil {
    log.Fatal(err)
}

// Check if service is running
if service.IsRunning() {
    fmt.Println("Skaffold session is active")
}

// Stop the session
err = service.Stop()
```

The service handles prerequisite installation, interactive cluster selection, Helm chart deployment with timeouts, and Skaffold development execution with automatic retry logic and graceful shutdown capabilities.
