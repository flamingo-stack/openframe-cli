Manages Skaffold development workflows for the OpenFrame CLI, providing service discovery, cluster integration, chart installation, and development session orchestration.

## Key Components

- **Service** - Main service struct that orchestrates Skaffold development workflows
- **NewService()** - Creates a new scaffold service instance with executor and verbose settings
- **RunScaffoldWorkflow()** - Executes the complete development workflow including service selection, cluster setup, and Skaffold execution
- **checkPrerequisites()** - Validates and optionally installs Skaffold if missing
- **runSkaffoldDev()** - Runs the Skaffold development session with retry logic and signal handling

## Usage Example

```go
// Create a new scaffold service
executor := executor.NewCommandExecutor()
service := NewService(executor, true)

// Run the complete scaffold workflow
flags := &models.ScaffoldFlags{
    SkipBootstrap: false,
    HelmValuesFile: "values.yaml",
    Namespace: "development",
}

ctx := context.Background()
args := []string{"my-cluster"}

err := service.RunScaffoldWorkflow(ctx, args, flags)
if err != nil {
    log.Fatal(err)
}

// Check if service is running
if service.IsRunning() {
    fmt.Println("Skaffold session is active")
}
```

The service handles prerequisite checking, interactive service selection, cluster management, Helm chart installation with timeout handling, and Skaffold development session management with graceful shutdown capabilities.
