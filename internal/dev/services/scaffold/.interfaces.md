This file defines the core interfaces for the scaffold package, establishing contracts for bootstrap operations, prerequisite checking, and Skaffold command execution.

## Key Components

### Interfaces

- **`BootstrapService`** - Interface for executing bootstrap commands with arguments
- **`PrerequisiteChecker`** - Interface for checking, installing, and managing software prerequisites
- **`ScaffoldRunner`** - Interface for running Skaffold development operations (dev, build, deploy)

## Usage Example

```go
package main

import (
    "context"
    "fmt"
)

// Example implementation of PrerequisiteChecker
type DockerChecker struct{}

func (d *DockerChecker) IsInstalled() bool {
    // Check if Docker is installed
    return true
}

func (d *DockerChecker) GetInstallHelp() string {
    return "Install Docker from https://docker.com"
}

func (d *DockerChecker) Install() error {
    // Installation logic
    return nil
}

func (d *DockerChecker) GetVersion() (string, error) {
    return "20.10.0", nil
}

// Example usage
func main() {
    checker := &DockerChecker{}
    
    if !checker.IsInstalled() {
        fmt.Println(checker.GetInstallHelp())
        if err := checker.Install(); err != nil {
            fmt.Printf("Installation failed: %v\n", err)
        }
    }
    
    version, _ := checker.GetVersion()
    fmt.Printf("Docker version: %s\n", version)
}
```

These interfaces enable dependency injection and testing by providing clear contracts for scaffold operations, prerequisite management, and Skaffold workflow execution.
