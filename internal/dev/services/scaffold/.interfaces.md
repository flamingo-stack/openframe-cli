Core interface definitions for the scaffold package that provide contracts for bootstrap operations, prerequisite checking, and Skaffold command execution.

## Key Components

- **BootstrapService** - Interface for executing bootstrap commands with arguments
- **PrerequisiteChecker** - Interface for validating and managing system prerequisites
- **ScaffoldRunner** - Interface for executing Skaffold development operations

## Interface Details

### BootstrapService
Provides a contract for bootstrap command execution:
- `Execute(cmd interface{}, args []string) error` - Executes bootstrap commands with provided arguments

### PrerequisiteChecker
Manages system prerequisite validation and installation:
- `IsInstalled() bool` - Checks if prerequisite is installed
- `GetInstallHelp() string` - Returns installation instructions
- `Install() error` - Attempts automatic installation
- `GetVersion() (string, error)` - Retrieves installed version

### ScaffoldRunner
Handles Skaffold workflow operations with context support:
- `RunDev(ctx context.Context, args []string) error` - Runs development mode
- `Build(ctx context.Context, args []string) error` - Builds container images
- `Deploy(ctx context.Context, args []string) error` - Deploys to Kubernetes

## Usage Example

```go
// Implementing a prerequisite checker
type DockerChecker struct{}

func (d *DockerChecker) IsInstalled() bool {
    _, err := exec.LookPath("docker")
    return err == nil
}

func (d *DockerChecker) GetInstallHelp() string {
    return "Install Docker from https://docker.com"
}

// Using the interfaces
var checker PrerequisiteChecker = &DockerChecker{}
if !checker.IsInstalled() {
    fmt.Println(checker.GetInstallHelp())
}
```
