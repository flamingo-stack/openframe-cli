Defines core interfaces for the scaffold package, providing contracts for bootstrap operations, prerequisite checking, and Skaffold command execution. These interfaces enable dependency injection and testing by abstracting external tool interactions.

## Key Components

- **BootstrapService**: Interface for executing bootstrap commands with arguments
- **PrerequisiteChecker**: Interface for validating, installing, and managing external dependencies
- **ScaffoldRunner**: Interface for executing Skaffold development workflow commands

## Usage Example

```go
// Implementing a prerequisite checker
type DockerChecker struct{}

func (d *DockerChecker) IsInstalled() bool {
    // Check if Docker is available
    return true
}

func (d *DockerChecker) GetInstallHelp() string {
    return "Install Docker from https://docker.com"
}

func (d *DockerChecker) Install() error {
    // Automated installation logic
    return nil
}

func (d *DockerChecker) GetVersion() (string, error) {
    return "20.10.0", nil
}

// Using the interfaces together
func setupDevelopment(checker PrerequisiteChecker, runner ScaffoldRunner) error {
    if !checker.IsInstalled() {
        return fmt.Errorf("dependency not installed: %s", checker.GetInstallHelp())
    }
    
    ctx := context.Background()
    return runner.RunDev(ctx, []string{"--port-forward"})
}
```
