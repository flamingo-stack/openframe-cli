This file defines core interfaces for a scaffold system that manages development workflows, particularly around Skaffold operations for Kubernetes development.

## Key Components

- **BootstrapService**: Interface for executing bootstrap commands with flexible command and argument handling
- **PrerequisiteChecker**: Interface for managing tool prerequisites including installation checks, help text, automated installation, and version retrieval
- **ScaffoldRunner**: Interface for executing Skaffold operations including development mode, building, and deployment

## Usage Example

```go
// Implement a prerequisite checker for Docker
type DockerChecker struct{}

func (d *DockerChecker) IsInstalled() bool {
    // Check if Docker is installed
    return checkDockerInstallation()
}

func (d *DockerChecker) GetInstallHelp() string {
    return "Install Docker from https://docker.com/get-started"
}

// Use the scaffold runner
func runDevEnvironment(runner ScaffoldRunner) error {
    ctx := context.Background()
    args := []string{"--port-forward"}
    return runner.RunDev(ctx, args)
}

// Execute bootstrap operations
func bootstrap(service BootstrapService) error {
    cmd := "init-project"
    args := []string{"--template", "golang"}
    return service.Execute(cmd, args)
}
```
