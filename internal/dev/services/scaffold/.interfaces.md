This file defines the core interfaces for the scaffold package, establishing contracts for bootstrap operations, prerequisite checking, and Skaffold execution workflows.

## Key Components

### BootstrapService
- **Execute(cmd interface{}, args []string) error** - Executes bootstrap commands with provided arguments

### PrerequisiteChecker  
- **IsInstalled() bool** - Checks if a prerequisite tool is installed
- **GetInstallHelp() string** - Returns installation instructions
- **Install() error** - Attempts automatic installation
- **GetVersion() (string, error)** - Retrieves the installed version

### ScaffoldRunner
- **RunDev(ctx context.Context, args []string) error** - Runs Skaffold in development mode
- **Build(ctx context.Context, args []string) error** - Builds container images
- **Deploy(ctx context.Context, args []string) error** - Deploys to Kubernetes cluster
