This file defines core interfaces and data structures for Kubernetes service interception operations, providing abstractions for working with services and namespaces.

## Key Components

**Data Types:**
- `ServicePort` - Configuration for individual service ports (name, port number, target port, protocol)
- `ServiceInfo` - Complete service metadata including name, namespace, type, and port configurations

**Interfaces:**
- `KubernetesClient` - Operations for namespace discovery and validation
- `ServiceClient` - Operations for service discovery, retrieval, and validation

## Usage Example

```go
// Implementing a service client
type myServiceClient struct {
    // client implementation
}

func (c *myServiceClient) GetServices(ctx context.Context, namespace string) ([]ServiceInfo, error) {
    // Retrieve services from Kubernetes API
    services := []ServiceInfo{
        {
            Name:      "web-service",
            Namespace: "default",
            Type:      "ClusterIP",
            Ports: []ServicePort{
                {Name: "http", Port: 80, TargetPort: "8080", Protocol: "TCP"},
            },
        },
    }
    return services, nil
}

// Using the interfaces
var kubeClient KubernetesClient
var serviceClient ServiceClient

namespaces, err := kubeClient.GetNamespaces(ctx)
services, err := serviceClient.GetServices(ctx, "default")
```
