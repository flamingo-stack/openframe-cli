Defines interfaces and data structures for Kubernetes service and namespace operations within an interceptor system. This file establishes contracts for interacting with Kubernetes resources and represents service metadata.

## Key Components

- **ServicePort**: Struct representing a service port with name, port number, target port, and protocol
- **ServiceInfo**: Struct containing complete service metadata including name, namespace, type, and ports
- **KubernetesClient**: Interface for namespace-level operations (listing and validating namespaces)
- **ServiceClient**: Interface for service-level operations (retrieving and validating services)

## Usage Example

```go
// Implementing a service client
type myServiceClient struct {
    // client implementation
}

func (c *myServiceClient) GetService(ctx context.Context, namespace, serviceName string) (*ServiceInfo, error) {
    // Implementation to fetch service from Kubernetes
    return &ServiceInfo{
        Name:      serviceName,
        Namespace: namespace,
        Type:      "ClusterIP",
        Ports: []ServicePort{
            {
                Name:       "http",
                Port:       80,
                TargetPort: "8080",
                Protocol:   "TCP",
            },
        },
    }, nil
}

// Using the interfaces
var client ServiceClient = &myServiceClient{}
service, err := client.GetService(ctx, "default", "my-service")
```
