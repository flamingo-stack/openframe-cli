Defines core interfaces and data structures for Kubernetes service interception operations, providing abstractions for namespace and service management.

## Key Components

- **ServicePort** - Configuration struct for service port details including name, port number, target port, and protocol
- **ServiceInfo** - Metadata struct representing a Kubernetes service with name, namespace, type, and associated ports
- **KubernetesClient** - Interface for namespace-level operations like listing and validating namespaces
- **ServiceClient** - Interface for service-level operations including retrieving and validating services

## Usage Example

    // client implementation

    // Implementation to retrieve services from Kubernetes
    return []ServiceInfo{

            Type:      "ClusterIP",

                    Port:       80,
                    TargetPort: "8080",

            },
        },
    }, nil
}
