Defines core interfaces and data structures for Kubernetes service interception operations, providing abstractions for namespace and service management.

## Key Components

**Data Types:**
- `ServicePort` - Represents a service port configuration with name, port, target port, and protocol
- `ServiceInfo` - Contains Kubernetes service metadata including name, namespace, type, and ports

**Interfaces:**
- `KubernetesClient` - Interface for namespace operations (listing and validation)
- `ServiceClient` - Interface for service operations (retrieval and validation)

## Usage Example

```go
// Implementing a service client
type myServiceClient struct{}

func (c *myServiceClient) GetServices(ctx context.Context, namespace string) ([]ServiceInfo, error) {
    // Implementation to fetch services from Kubernetes
    return []ServiceInfo{
        {
            Name:      "web-service",
            Namespace: "default",
            Type:      "ClusterIP",
            Ports: []ServicePort{
                {
                    Name:       "http",
                    Port:       80,
                    TargetPort: "8080",
                    Protocol:   "TCP",
                },
            },
        },
    }, nil
}

// Using the interfaces
var serviceClient ServiceClient = &myServiceClient{}
services, err := serviceClient.GetServices(ctx, "default")
```
