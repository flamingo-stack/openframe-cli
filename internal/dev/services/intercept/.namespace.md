Provides functionality for managing Telepresence namespace operations, including retrieving the current namespace and switching between namespaces while preserving traffic manager connections.

## Key Components

### Methods
- **`getCurrentNamespace(ctx Context) (string, error)`** - Retrieves the current Telepresence namespace using JSON output parsing, defaults to "default" if none found
- **`switchNamespace(ctx Context, current, target string) error`** - Switches Telepresence from current namespace to target namespace with graceful disconnect/reconnect

## Usage Example

```go
// Get current namespace
namespace, err := service.getCurrentNamespace(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Current namespace: %s\n", namespace)

// Switch to a different namespace
err = service.switchNamespace(ctx, "current-ns", "target-ns")
if err != nil {
    log.Fatal(err)
}
```

The namespace switching operation performs a controlled disconnect followed by reconnection to the target namespace, preserving the traffic manager connection while avoiding the overhead of a full quit/restart cycle. The current namespace detection uses the same JSON parsing approach as the original bash implementation for consistency.
