Test file providing mock implementations and comprehensive unit tests for Kubernetes client interfaces. Contains mock objects for testing interactions with Kubernetes namespaces and services, along with tests for data structures.

## Key Components

- **MockKubernetesClient**: Mock implementation of KubernetesClient interface for testing namespace operations
- **MockServiceClient**: Mock implementation of ServiceClient interface for testing service operations  
- **ServicePort/ServiceInfo Structure Tests**: Validation of data structure fields and serialization
- **Interface Compliance Tests**: Compile-time verification that mocks implement required interfaces
- **Integration Test Scenarios**: Complex service configurations with multiple ports and edge cases

## Usage Example

```go
// Testing namespace validation
func TestMyFunction(t *testing.T) {
    ctx := context.Background()
    mockClient := new(MockKubernetesClient)
    
    // Setup mock expectations
    mockClient.On("GetNamespaces", ctx).Return([]string{"default", "kube-system"}, nil)
    
    // Use in your code under test
    namespaces, err := mockClient.GetNamespaces(ctx)
    
    assert.NoError(t, err)
    assert.Equal(t, []string{"default", "kube-system"}, namespaces)
    mockClient.AssertExpectations(t)
}

// Testing service retrieval
func TestServiceLookup(t *testing.T) {
    mockServiceClient := new(MockServiceClient)
    
    expectedService := &ServiceInfo{
        Name: "my-service",
        Namespace: "default",
        Ports: []ServicePort{{Name: "http", Port: 8080}},
    }
    
    mockServiceClient.On("GetService", mock.Anything, "default", "my-service").
        Return(expectedService, nil)
}
```
