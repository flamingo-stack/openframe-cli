A Go service that provides Telepresence intercept functionality for Kubernetes services, implementing enhanced capabilities based on bash scripts from develop.sh.

## Key Components

- **Service**: Main struct managing Telepresence intercepts with state tracking and cleanup handlers
- **TelepresenceStatus**: JSON structure for parsing Telepresence daemon status  
- **StartIntercept()**: Validates inputs, sets up Kubernetes context, and creates service intercepts
- **StopIntercept()**: Manually terminates active intercepts with proper cleanup
- **validateInputs()**: Comprehensive validation for service names, ports, namespaces, and configuration files
- **checkKubernetesContext()**: Verifies kubectl availability and cluster connectivity
- **ensureCorrectNamespace()**: Manages namespace switching and Telepresence reconnection

## Usage Example

```go
// Create intercept service
exec := executor.NewCommandExecutor()
service := intercept.NewService(exec, true) // verbose mode

// Configure intercept flags
flags := &models.InterceptFlags{
    Port:      8080,
    Namespace: "development", 
    EnvFile:   ".env.local",
    Header:    []string{"x-user=developer"},
}

// Start intercepting a service
err := service.StartIntercept("my-service", flags)
if err != nil {
    log.Fatal(err)
}

// Check status
if service.IsIntercepting() {
    fmt.Printf("Intercepting: %s in %s\n", 
        service.GetCurrentService(),
        service.GetCurrentNamespace())
}

// Stop intercept
err = service.StopIntercept("my-service")
```
