This file contains the core telepresence intercept creation logic for the OpenFrame CLI development tools.

## Key Components

- **`createIntercept`** - Main function that builds and executes telepresence intercept commands with configurable options
- **`getRemotePortName`** - Helper function that determines the remote port name, defaulting to the port number if not specified

## Usage Example

```go
// Create an intercept service instance
service := &Service{executor: cmdExecutor}

// Configure intercept flags
flags := &models.InterceptFlags{
    Port:           8080,
    RemotePortName: "http",
    EnvFile:        ".env.local",
    Global:         false,
    Header:         []string{"x-user-id: 123"},
    Replace:        true,
}

// Create the intercept
ctx := context.Background()
err := service.createIntercept(ctx, "my-service", flags)
if err != nil {
    log.Fatalf("Failed to create intercept: %v", err)
}
```

The implementation follows telepresence CLI patterns, automatically disabling file mounts and supporting standard intercept options like port mapping, environment files, HTTP headers, and global/replace modes.
