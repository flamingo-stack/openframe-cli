Provides a centralized system for checking and managing development tool prerequisites for the OpenFrame CLI. The checker validates that required tools like Telepresence, jq, and Skaffold are installed on the system.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of requirements and provides validation methods
- **Requirement**: Represents a single prerequisite with name, command, installation check, and help functions
- **NewPrerequisiteChecker()**: Factory function that creates a checker with predefined requirements (Telepresence, jq, Skaffold)
- **CheckAll()**: Validates all requirements and returns status with list of missing tools
- **GetInstallInstructions()**: Retrieves installation help text for missing tools
- **CheckPrerequisites()**: Convenience function that creates an installer and performs check/install

## Usage Example

```go
// Create a new prerequisite checker
checker := prerequisites.NewPrerequisiteChecker()

// Check all requirements
allPresent, missing := checker.CheckAll()
if !allPresent {
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    
    for i, tool := range missing {
        fmt.Printf("Missing: %s\n", tool)
        fmt.Printf("Install: %s\n\n", instructions[i])
    }
}

// Or use the convenience function
if err := prerequisites.CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
