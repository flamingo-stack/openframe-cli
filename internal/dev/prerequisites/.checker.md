A prerequisite checker for the OpenFrame CLI that validates and manages required development tools like Telepresence, jq, and Skaffold.

## Key Components

- **`PrerequisiteChecker`**: Main struct that manages a list of development tool requirements
- **`Requirement`**: Defines a tool with its name, command, installation check, and help instructions
- **`NewPrerequisiteChecker()`**: Factory function that creates a checker with predefined tools (Telepresence, jq, Skaffold)
- **`CheckAll()`**: Validates all requirements and returns missing tools
- **`GetInstallInstructions()`**: Retrieves installation help for missing tools
- **`CheckPrerequisites()`**: Convenience function for checking and installing prerequisites

## Usage Example

```go
// Create a prerequisite checker
checker := NewPrerequisiteChecker()

// Check all required tools
allPresent, missing := checker.CheckAll()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Failed to check prerequisites:", err)
}
```

The checker uses dependency injection pattern where each tool's installer (telepresence, jq, scaffold packages) provides its own installation check and help methods.
