Provides a prerequisite checking system for validating that required development tools (Telepresence, jq, Skaffold) are installed on the system. The checker can identify missing tools and provide installation instructions.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of tool requirements
- **Requirement**: Defines a tool with its name, command, installation check, and help instructions
- **NewPrerequisiteChecker()**: Factory function that creates a checker with predefined tool requirements
- **CheckAll()**: Validates all requirements and returns missing tools
- **GetInstallInstructions()**: Retrieves installation help for specific missing tools
- **CheckPrerequisites()**: Convenience function for checking and installing prerequisites

## Usage Example

```go
checker := prerequisites.NewPrerequisiteChecker()

// Check which tools are missing
allPresent, missing := checker.CheckAll()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := prerequisites.CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
