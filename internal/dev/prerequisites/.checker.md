This file provides a centralized system for checking and managing development tool prerequisites required by the OpenFrame CLI, including Telepresence, jq, and Skaffold.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a list of requirements and provides checking functionality
- **Requirement**: Defines the structure for each prerequisite tool with name, command, installation check, and help functions
- **NewPrerequisiteChecker()**: Factory function that initializes the checker with predefined requirements
- **CheckAll()**: Validates all prerequisites and returns status with missing tools
- **GetInstallInstructions()**: Retrieves installation help for specific missing tools
- **CheckPrerequisites()**: Convenience function for the complete check and install workflow

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Quick check and install using convenience function
if err := CheckPrerequisites(); err != nil {
    log.Printf("Failed to check prerequisites: %v", err)
}
```

The checker uses a modular design where each tool (Telepresence, jq, Skaffold) has its own installer package that implements the `IsInstalled()` and `GetInstallHelp()` methods, making it easy to add new prerequisites or modify existing ones.
