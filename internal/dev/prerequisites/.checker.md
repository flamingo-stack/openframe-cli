Validates and manages required development tools for the OpenFrame CLI, ensuring Telepresence, jq, and Skaffold are installed before development operations.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of development tool requirements
- **Requirement**: Struct defining each tool with name, command, installation check, and help functions
- **NewPrerequisiteChecker()**: Factory function that initializes checker with predefined tools (Telepresence, jq, Skaffold)
- **CheckAll()**: Validates all requirements and returns missing tools
- **GetInstallInstructions()**: Provides installation guidance for missing tools
- **CheckPrerequisites()**: Convenience function for complete prerequisite validation

## Usage Example

```go
// Create a new prerequisite checker
checker := NewPrerequisiteChecker()

// Check if all required tools are installed
allPresent, missingTools := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missingTools)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missingTools)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Alternative: Use convenience function
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```

The checker leverages dedicated installer modules for each tool (telepresence, jq, scaffold packages) to provide platform-specific installation detection and guidance.
