This file provides a comprehensive tool installer for development prerequisites, managing the installation and verification of required tools like Telepresence, jq, and Skaffold. It offers both automatic installation and manual instruction fallbacks with interactive user prompts.

## Key Components

- **Installer struct**: Main installer with embedded PrerequisiteChecker
- **ToolInstaller interface**: Common interface for tool installers with `IsInstalled()`, `GetInstallHelp()`, and `Install()` methods
- **NewInstaller()**: Factory function returning configured installer instance
- **CheckAndInstall()**: Checks all prerequisites and offers automatic installation
- **CheckSpecificTools()**: Validates only specified tools without installation
- **CheckInterceptPrerequisites()**: Specialized check for intercept command (Telepresence + jq + cluster)
- **CheckScaffoldPrerequisites()**: Specialized check for scaffold command (Skaffold + cluster)

## Usage Example

```go
// Check and install all development tools
installer := NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Check specific tools only
if err := installer.CheckSpecificTools([]string{"telepresence", "jq"}); err != nil {
    log.Fatal(err)
}

// Check prerequisites for intercept command
if err := CheckInterceptPrerequisites(); err != nil {
    log.Fatal(err)
}

// Check and install specific tools with user interaction
if err := installer.CheckAndInstallSpecificTools([]string{"skaffold"}); err != nil {
    log.Fatal(err)
}
```
