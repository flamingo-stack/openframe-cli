This file implements an automated installer for development prerequisites required by the OpenFrame CLI, managing tools like Telepresence, jq, and Skaffold with interactive prompts and progress feedback.

## Key Components

- **`Installer`** - Main struct that manages prerequisite checking and installation
- **`ToolInstaller` interface** - Defines contract for tool installers with install, check, and help methods
- **Installation methods** - Various specialized methods for different tool combinations:
  - `CheckAndInstall()` - Full prerequisite check and installation
  - `CheckSpecificTools()` - Validation for specific tool sets
  - `CheckInterceptPrerequisites()` - Prerequisites for intercept commands
  - `CheckScaffoldPrerequisites()` - Prerequisites for scaffold operations

## Usage Example

```go
// Initialize installer and check all prerequisites
installer := NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal("Failed to install prerequisites:", err)
}

// Check specific tools only
requiredTools := []string{"telepresence", "jq"}
if err := installer.CheckSpecificTools(requiredTools); err != nil {
    log.Fatal("Required tools missing:", err)
}

// Check intercept prerequisites (tools + cluster)
if err := CheckInterceptPrerequisites(); err != nil {
    log.Fatal("Intercept prerequisites not met:", err)
}
```

The installer supports interactive mode with user confirmation prompts, silent checking for automation, and provides detailed installation instructions when automatic installation fails. It integrates with the UI system to respect test mode and provide consistent user experience across the CLI.
