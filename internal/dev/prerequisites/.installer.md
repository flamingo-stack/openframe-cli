This file provides an installer system for managing development tool prerequisites including telepresence, jq, and skaffold. It handles automatic installation with user confirmation and provides fallback manual installation instructions.

## Key Components

- **`Installer`** - Main installer struct with prerequisite checking functionality
- **`ToolInstaller`** - Interface defining installation methods for tools
- **`NewInstaller()`** - Creates a new installer instance
- **`CheckAndInstall()`** - Checks all prerequisites and offers to install missing tools
- **`CheckSpecificTools()`** - Validates specific tools are installed
- **`CheckAndInstallSpecificTools()`** - Checks and installs specific tools with user confirmation
- **`CheckInterceptPrerequisites()`** - Validates telepresence and jq for intercept commands
- **`CheckScaffoldPrerequisites()`** - Validates skaffold for scaffold commands

## Usage Example

```go
// Create installer and check all prerequisites
installer := NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Check specific tools only
tools := []string{"telepresence", "jq"}
if err := installer.CheckSpecificTools(tools); err != nil {
    log.Fatal(err)
}

// Check intercept prerequisites (telepresence + jq + cluster)
if err := CheckInterceptPrerequisites(); err != nil {
    log.Fatal(err)
}

// Silent check without user interaction
allPresent, missing := installer.CheckSilent()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
}
```
