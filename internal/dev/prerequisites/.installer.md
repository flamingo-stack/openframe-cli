This file manages prerequisite tool installation for the OpenFrame CLI, automatically detecting and installing required development tools like Telepresence, jq, and Skaffold.

## Key Components

- **Installer**: Main struct that orchestrates prerequisite checking and installation
- **ToolInstaller**: Interface defining installation capabilities for individual tools
- **installMissingTools()**: Core installation logic with progress indicators
- **CheckAndInstall()**: Full prerequisite check with user confirmation
- **CheckSpecificTools()**: Targeted checking for specific tool subsets
- **CheckInterceptPrerequisites()**: Specialized check for service intercept functionality
- **CheckScaffoldPrerequisites()**: Specialized check for scaffolding functionality

## Usage Example

```go
// Create installer and check all prerequisites
installer := NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal("Prerequisites not met:", err)
}

// Check specific tools only
if err := installer.CheckSpecificTools([]string{"telepresence", "jq"}); err != nil {
    log.Fatal("Required tools missing:", err)
}

// Check prerequisites for intercept command
if err := CheckInterceptPrerequisites(); err != nil {
    log.Fatal("Intercept prerequisites not met:", err)
}

// Silent check without user interaction
allPresent, missing := installer.CheckSilent()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
}
```

The installer provides interactive prompts for automatic installation, fallback to manual instructions, and specialized checking for different CLI command workflows with comprehensive progress feedback.
