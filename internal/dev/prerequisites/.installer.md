Manages the installation of development prerequisite tools for the OpenFrame CLI, providing automated installation with user confirmation and fallback manual instructions.

## Key Components

- **Installer struct**: Main coordinator that wraps PrerequisiteChecker functionality
- **ToolInstaller interface**: Defines contract for tool-specific installers (IsInstalled, GetInstallHelp, Install)
- **Tool registry**: Maps tool names to their respective installer implementations (telepresence, jq, skaffold)
- **Installation methods**: Handles different installation scenarios with user interaction and progress feedback
- **Prerequisite checkers**: Specialized functions for command-specific tool requirements

## Main Methods

- `CheckAndInstall()` - Checks all prerequisites and offers automatic installation
- `CheckSpecificTools()` - Validates only specified tools without installation
- `CheckAndInstallSpecificTools()` - Checks and installs specific tools with user confirmation
- `CheckInterceptPrerequisites()` - Validates prerequisites for intercept commands (telepresence + jq)
- `CheckScaffoldPrerequisites()` - Validates prerequisites for scaffold commands (skaffold)

## Usage Example

```go
// Check and install all development tools
installer := NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal("Prerequisites not met:", err)
}

// Check specific tools for a command
if err := installer.CheckSpecificTools([]string{"telepresence", "jq"}); err != nil {
    return fmt.Errorf("missing required tools: %w", err)
}

// Check intercept prerequisites (includes cluster availability)
if err := CheckInterceptPrerequisites(); err != nil {
    return fmt.Errorf("intercept prerequisites not met: %w", err)
}
```

The installer provides interactive installation with progress spinners, graceful fallback to manual instructions, and test mode compatibility for automated environments.
