Manages the installation and verification of development tools required by the OpenFrame CLI. This installer handles tools like Telepresence, jq, and Skaffold with interactive prompts and automatic installation capabilities.

## Key Components

- **Installer**: Main installer struct that orchestrates tool verification and installation
- **ToolInstaller**: Interface defining methods for checking installation status and installing tools
- **NewInstaller()**: Creates a new installer instance with prerequisite checker
- **CheckAndInstall()**: Checks all prerequisites and offers automatic installation
- **CheckSpecificTools()**: Verifies installation of specified tools only
- **CheckInterceptPrerequisites()**: Specialized check for intercept command tools (Telepresence + jq)
- **CheckScaffoldPrerequisites()**: Specialized check for scaffold command tools

## Usage Example

```go
// Check and install all development tools
installer := prerequisites.NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Check specific tools only
err := installer.CheckSpecificTools([]string{"telepresence", "jq"})

// Check intercept prerequisites (telepresence + jq + cluster)
if err := prerequisites.CheckInterceptPrerequisites(); err != nil {
    return err
}

// Check scaffold prerequisites (skaffold + cluster)
if err := prerequisites.CheckScaffoldPrerequisites(); err != nil {
    return err
}
```

The installer provides interactive prompts for automatic installation and falls back to manual installation instructions when needed. It includes spinner animations and progress tracking for better user experience.
