Manages installation and verification of development prerequisites including Telepresence, jq, and Skaffold. Provides automated installation with user confirmation and fallback manual instructions.

## Key Components

- **Installer**: Main installer struct with prerequisite checking capabilities
- **ToolInstaller**: Interface defining installation contract for tools (IsInstalled, GetInstallHelp, Install)
- **installMissingTools**: Automated installation with progress spinners
- **CheckAndInstall**: Full prerequisite check with user prompt for installation
- **CheckSpecificTools**: Validates only specified tools are installed
- **CheckInterceptPrerequisites**: Validates tools needed for intercept command (Telepresence + jq)
- **CheckScaffoldPrerequisites**: Validates tools needed for scaffold command (Skaffold)

## Usage Example

```go
// Create installer and check all prerequisites
installer := NewInstaller()
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Check specific tools only
tools := []string{"telepresence", "jq"}
if err := installer.CheckSpecificTools(tools); err != nil {
    log.Fatal(err)
}

// Check prerequisites for intercept command
if err := CheckInterceptPrerequisites(); err != nil {
    log.Fatal(err)
}

// Silent check without user interaction
allPresent, missing := installer.CheckSilent()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
}
```
