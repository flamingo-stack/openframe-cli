A Go package that provides automated Skaffold installation and management functionality across different operating systems (macOS, Linux, Windows).

## Key Components

- **ScaffoldInstaller**: Main struct that handles Skaffold installation operations
- **commandExists()**: Utility function to check if a command is available in PATH
- **isScaffoldInstalled()** / **IsScaffoldRunning()**: Functions to verify Skaffold availability
- **scaffoldInstallHelp()**: Platform-specific installation instructions
- **Install()**: Automated installation method with OS-specific implementations
- **GetVersion()**: Retrieves the installed Skaffold version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := scaffold.NewScaffoldInstaller()
    
    // Check if Skaffold is already installed
    if !installer.IsInstalled() {
        fmt.Println("Skaffold not found. Installing...")
        
        if err := installer.Install(); err != nil {
            log.Fatalf("Failed to install Skaffold: %v", err)
        }
        
        fmt.Println("Skaffold installed successfully!")
    }
    
    // Get version information
    version, err := installer.GetVersion()
    if err != nil {
        log.Printf("Error getting version: %v", err)
    } else {
        fmt.Printf("Skaffold version: %s\n", version)
    }
    
    // Check if Skaffold is running properly
    if scaffold.IsScaffoldRunning() {
        fmt.Println("Skaffold is ready to use!")
    }
}
```

The installer supports automatic installation on macOS (via Homebrew) and Linux (via curl/wget), while providing helpful installation instructions for Windows users.
