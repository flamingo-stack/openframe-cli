This package provides automated detection and installation of Skaffold, a Kubernetes development tool, with cross-platform support for macOS, Linux, and Windows.

## Key Components

- **ScaffoldInstaller**: Main struct that encapsulates Skaffold installation functionality
- **NewScaffoldInstaller()**: Constructor function returning a new installer instance
- **IsInstalled()**: Checks if Skaffold is currently installed on the system
- **IsScaffoldRunning()**: Global function to verify Skaffold is available and functional
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs Skaffold using platform-appropriate package managers
- **GetVersion()**: Retrieves the installed Skaffold version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := scaffold.NewScaffoldInstaller()
    
    // Check if Skaffold is installed
    if !installer.IsInstalled() {
        fmt.Println("Skaffold not found. Installing...")
        
        if err := installer.Install(); err != nil {
            log.Printf("Auto-install failed: %v", err)
            fmt.Println("Manual installation help:", installer.GetInstallHelp())
            return
        }
    }
    
    // Verify installation and get version
    if scaffold.IsScaffoldRunning() {
        version, _ := installer.GetVersion()
        fmt.Printf("Skaffold is ready: %s\n", version)
    }
}
```
