Provides cross-platform installation and management functionality for Skaffold, Google's Kubernetes development tool, with automatic detection and platform-specific installation methods.

## Key Components

- **ScaffoldInstaller**: Main struct that handles Skaffold installation and management
- **commandExists()**: Utility function to check if a command is available in the system PATH
- **isScaffoldInstalled()/IsScaffoldRunning()**: Functions to detect if Skaffold is installed and operational
- **Install()**: Cross-platform installation method with OS-specific implementations
- **GetVersion()**: Retrieves the installed Skaffold version
- **scaffoldInstallHelp()**: Provides platform-specific installation instructions

## Platform Support

| Platform | Installation Method | Requirements |
|----------|-------------------|--------------|
| **macOS** | Homebrew (`brew install skaffold`) | Homebrew required |
| **Linux** | Direct binary download via curl/wget | curl or wget required |
| **Windows** | Manual installation only | Chocolatey recommended |

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := scaffold.NewScaffoldInstaller()
    
    // Check if Skaffold is already installed
    if !installer.IsInstalled() {
        fmt.Println("Skaffold not found. Installation help:")
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation (Linux/macOS only)
        if err := installer.Install(); err != nil {
            log.Fatalf("Installation failed: %v", err)
        }
    }
    
    // Verify installation and get version
    if version, err := installer.GetVersion(); err == nil {
        fmt.Printf("Skaffold version: %s\n", version)
    }
    
    // Check if Skaffold is operational
    if scaffold.IsScaffoldRunning() {
        fmt.Println("Skaffold is ready to use!")
    }
}
```

The installer handles platform detection automatically and provides appropriate error messages with manual installation instructions when automatic installation isn't supported.
