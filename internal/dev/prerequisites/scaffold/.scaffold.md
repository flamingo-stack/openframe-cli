This package provides utilities for detecting, installing, and managing Skaffold, a command-line tool for continuous development with Kubernetes. It offers cross-platform installation support and version checking capabilities.

## Key Components

- `ScaffoldInstaller` - Main struct providing Skaffold management functionality
- `NewScaffoldInstaller()` - Constructor function returning a new installer instance
- `IsScaffoldRunning()` - Global function to check if Skaffold is available and functional
- `IsInstalled()` - Method to verify Skaffold installation
- `Install()` - Method providing automatic installation across different operating systems
- `GetInstallHelp()` - Method returning platform-specific installation instructions
- `GetVersion()` - Method to retrieve the installed Skaffold version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "scaffold"
)

func main() {
    installer := scaffold.NewScaffoldInstaller()
    
    // Check if Skaffold is installed
    if !installer.IsInstalled() {
        fmt.Println("Skaffold not found")
        fmt.Println("Install help:", installer.GetInstallHelp())
        
        // Attempt automatic installation
        if err := installer.Install(); err != nil {
            log.Fatal("Installation failed:", err)
        }
    }
    
    // Verify installation and get version
    if version, err := installer.GetVersion(); err == nil {
        fmt.Println("Skaffold version:", version)
    }
    
    // Check if Skaffold is running properly
    if scaffold.IsScaffoldRunning() {
        fmt.Println("Skaffold is ready to use")
    }
}
```
