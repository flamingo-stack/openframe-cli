This package provides cross-platform detection and installation utilities for Skaffold, the Kubernetes development tool. It includes platform-specific installation methods and status checking functionality.

## Key Components

- **ScaffoldInstaller**: Main struct providing installation and management methods
- **IsScaffoldRunning()**: Global function to check if Skaffold is available and working
- **commandExists()**: Helper to detect if a command is available in PATH
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Linux (curl/wget), and Windows (manual guidance)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := scaffold.NewScaffoldInstaller()
    
    // Check if Skaffold is already installed
    if installer.IsInstalled() {
        version, err := installer.GetVersion()
        if err != nil {
            log.Fatal(err)
        }
        fmt.Printf("Skaffold is installed: %s\n", version)
        return
    }
    
    // Get platform-specific installation help
    fmt.Printf("Installation help: %s\n", installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        fmt.Printf("Automatic installation failed: %v\n", err)
        fmt.Printf("Please install manually: %s\n", installer.GetInstallHelp())
        return
    }
    
    fmt.Println("Skaffold installed successfully!")
}
```

The installer handles platform detection automatically and provides fallback guidance for unsupported scenarios like Windows automatic installation.
