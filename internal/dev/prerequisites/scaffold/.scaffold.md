This file provides a cross-platform Skaffold installer that can detect, install, and manage Skaffold installations programmatically. It supports macOS, Linux, and Windows with platform-specific installation methods.

## Key Components

- **ScaffoldInstaller**: Main struct that provides installation and management methods
- **NewScaffoldInstaller()**: Constructor function that returns a new installer instance
- **IsInstalled()**: Checks if Skaffold is installed on the system
- **IsScaffoldRunning()**: Global function to verify Skaffold is available and functional
- **Install()**: Automatically installs Skaffold using platform-appropriate package managers
- **GetInstallHelp()**: Returns platform-specific manual installation instructions
- **GetVersion()**: Retrieves the installed Skaffold version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := scaffold.NewScaffoldInstaller()
    
    // Check if Skaffold is installed
    if !installer.IsInstalled() {
        fmt.Println("Skaffold not found. Installing...")
        
        if err := installer.Install(); err != nil {
            fmt.Printf("Auto-install failed: %v\n", err)
            fmt.Printf("Manual installation: %s\n", installer.GetInstallHelp())
            return
        }
        
        fmt.Println("Skaffold installed successfully!")
    }
    
    // Get version information
    version, err := installer.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Skaffold version: %s\n", version)
    
    // Check if running properly
    if scaffold.IsScaffoldRunning() {
        fmt.Println("Skaffold is ready to use!")
    }
}
```
