This file provides a cross-platform Skaffold installer and manager for development environments. It handles detection, installation, and version management of the Skaffold Kubernetes development tool across macOS, Linux, and Windows.

## Key Components

- **ScaffoldInstaller**: Main struct that provides installation and management functionality
- **commandExists()**: Utility function to check if a command is available in PATH
- **IsScaffoldRunning()**: Checks if Skaffold is installed and functional
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Linux (curl/wget), and Windows (manual)
- **GetVersion()**: Retrieves the installed Skaffold version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "your-project/scaffold"
)

func main() {
    installer := scaffold.NewScaffoldInstaller()
    
    // Check if Skaffold is installed
    if !installer.IsInstalled() {
        fmt.Println("Skaffold not found. Installation help:")
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation
        if err := installer.Install(); err != nil {
            log.Fatalf("Installation failed: %v", err)
        }
    }
    
    // Check if Skaffold is running properly
    if scaffold.IsScaffoldRunning() {
        version, err := installer.GetVersion()
        if err == nil {
            fmt.Printf("Skaffold is ready: %s\n", version)
        }
    }
}
```

The installer automatically detects the operating system and uses the appropriate installation method: Homebrew on macOS, direct binary download on Linux, and provides manual instructions for Windows.
