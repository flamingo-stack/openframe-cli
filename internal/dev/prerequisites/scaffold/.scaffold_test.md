This test file provides comprehensive unit tests for the Scaffold (Skaffold) installer functionality, testing installation detection, version retrieval, and help text generation.

## Key Components

- **TestNewScaffoldInstaller**: Tests creation of new installer instances
- **TestScaffoldInstaller_GetInstallHelp**: Validates help text contains installation methods (brew, curl, choco)
- **TestScaffoldInstaller_IsInstalled**: Verifies installation detection returns boolean
- **TestIsScaffoldRunning**: Tests standalone function for detecting running Skaffold processes
- **TestScaffoldInstaller_GetVersion**: Tests version retrieval with conditional assertions based on installation status
- **TestCommandExists**: Utility test for command existence checking
- **TestScaffoldInstallHelp**: Tests platform-specific help text generation
- **TestScaffoldInstaller_Install**: Basic structure validation for install method

## Usage Example

```go
func TestCustomScaffoldBehavior(t *testing.T) {
    installer := NewScaffoldInstaller()
    
    // Test installation status
    if installer.IsInstalled() {
        version, err := installer.GetVersion()
        assert.NoError(t, err)
        assert.NotEmpty(t, version)
    }
    
    // Test help text
    help := installer.GetInstallHelp()
    assert.Contains(t, help, "skaffold")
    
    // Test running status
    isRunning := IsScaffoldRunning()
    assert.IsType(t, true, isRunning)
}
```
