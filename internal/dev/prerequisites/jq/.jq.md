A cross-platform installer and validator for the `jq` command-line JSON processor, providing automatic installation support across macOS, Linux distributions, and Windows.

## Key Components

- **JqInstaller**: Main struct providing jq installation and management capabilities
- **IsJqRunning()**: Global function to check if jq is available and functional
- **commandExists()**: Utility to verify command availability in PATH
- **Platform-specific installers**: Dedicated methods for macOS (Homebrew), Linux (apt/yum/dnf/pacman), and Windows (manual)
- **Distribution detection**: Smart Linux distribution detection using `/etc/os-release`

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    // Check if jq is running
    if !IsJqRunning() {
        fmt.Println("jq is not available")
    }
    
    // Create installer
    installer := NewJqInstaller()
    
    // Check installation status
    if !installer.IsInstalled() {
        fmt.Println("Installing jq...")
        if err := installer.Install(); err != nil {
            // Fall back to manual installation help
            fmt.Println(installer.GetInstallHelp())
            log.Fatal(err)
        }
    }
    
    // Get version info
    version, err := installer.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("jq version: %s\n", version)
}
```

The installer automatically detects the operating system and uses the appropriate package manager (brew, apt, yum, dnf, pacman) or provides manual installation instructions for unsupported platforms.
