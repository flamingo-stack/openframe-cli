A cross-platform utility package for detecting, installing, and managing the jq JSON processor command-line tool. It provides platform-specific installation methods and system compatibility checks.

## Key Components

- **JqInstaller**: Main struct providing jq installation and management functionality
- **IsJqRunning()**: Global function to check if jq is available and functional
- **NewJqInstaller()**: Constructor function that returns a new installer instance
- **commandExists()**: Helper function to check if a command is available in PATH

### JqInstaller Methods

- **IsInstalled()**: Checks if jq is installed on the system
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs jq using the appropriate package manager
- **GetVersion()**: Retrieves the installed jq version

## Usage Example

```go
// Check if jq is running before using it
if !jq.IsJqRunning() {
    installer := jq.NewJqInstaller()
    
    if !installer.IsInstalled() {
        fmt.Println("jq not found. Installation help:")
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation
        if err := installer.Install(); err != nil {
            return fmt.Errorf("failed to install jq: %w", err)
        }
    }
}

// Get version information
installer := jq.NewJqInstaller()
version, err := installer.GetVersion()
if err != nil {
    return err
}
fmt.Printf("jq version: %s\n", version)
```

The package supports automatic installation on macOS (via Homebrew), various Linux distributions (apt, yum, dnf, pacman), and provides manual installation guidance for Windows. It includes Linux distribution detection and fallback installation methods.
