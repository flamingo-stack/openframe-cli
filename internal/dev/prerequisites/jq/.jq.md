A cross-platform Go package that provides utilities for checking, installing, and managing the jq JSON processor command-line tool. The package automatically detects the operating system and uses appropriate package managers or direct downloads to install jq.

## Key Components

- **`JqInstaller`** - Main struct providing installation and management functionality
- **`NewJqInstaller()`** - Constructor that returns a new JqInstaller instance
- **`IsJqRunning()`** - Global function to check if jq is available and working
- **`IsInstalled()`** - Method to check if jq is installed on the system
- **`Install()`** - Method that automatically installs jq based on the detected OS
- **`GetInstallHelp()`** - Method that returns platform-specific installation instructions
- **`GetVersion()`** - Method to retrieve the installed jq version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "your-project/jq"
)

func main() {
    installer := jq.NewJqInstaller()
    
    // Check if jq is installed
    if !installer.IsInstalled() {
        fmt.Println("jq not found. Installing...")
        
        // Attempt automatic installation
        if err := installer.Install(); err != nil {
            // If automatic install fails, show manual instructions
            fmt.Println("Auto-install failed:", err)
            fmt.Println(installer.GetInstallHelp())
            return
        }
    }
    
    // Verify jq is working and get version
    if version, err := installer.GetVersion(); err == nil {
        fmt.Printf("jq version: %s\n", version)
    }
    
    // Check if jq is running properly
    if jq.IsJqRunning() {
        fmt.Println("jq is ready to use!")
    }
}
```
