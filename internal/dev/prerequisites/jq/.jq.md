A cross-platform jq command-line tool installer and manager that handles automatic installation, version checking, and platform-specific installation commands.

## Key Components

- **`JqInstaller`** - Main installer struct providing cross-platform jq installation capabilities
- **`NewJqInstaller()`** - Constructor for creating a new JqInstaller instance
- **`IsJqRunning()`** - Global function to check if jq is properly installed and functional
- **Installation Methods** - Platform-specific installers for macOS, Linux distributions, and Windows
- **System Detection** - Functions to identify Debian-like and RHEL-like Linux distributions
- **Version Management** - Methods to check installation status and retrieve version information

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := NewJqInstaller()
    
    // Check if jq is already installed
    if installer.IsInstalled() {
        version, _ := installer.GetVersion()
        fmt.Printf("jq is installed: %s\n", version)
        return
    }
    
    // Get platform-specific installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation (Unix systems only)
    if err := installer.Install(); err != nil {
        log.Printf("Auto-install failed: %v", err)
        fmt.Println("Please install manually:", installer.GetInstallHelp())
        return
    }
    
    // Verify installation
    if IsJqRunning() {
        fmt.Println("jq installed and working correctly")
    }
}
```

The installer supports automatic installation on macOS (via Homebrew), various Linux distributions (apt, yum, dnf, pacman), and provides manual installation guidance for Windows and unsupported platforms.
