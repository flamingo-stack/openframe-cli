A cross-platform jq installer package that provides automated installation and management of the jq JSON processor command-line tool across different operating systems.

## Key Components

**Types:**
- `JqInstaller` - Main struct for managing jq installation operations

**Functions:**
- `NewJqInstaller()` - Creates a new JqInstaller instance
- `IsJqRunning()` - Checks if jq is available and functional
- `commandExists(cmd)` - Utility to check if a command is available in PATH

**JqInstaller Methods:**
- `IsInstalled()` - Checks if jq is installed on the system
- `GetInstallHelp()` - Returns platform-specific installation instructions
- `Install()` - Automatically installs jq using platform-appropriate package managers
- `GetVersion()` - Retrieves the installed jq version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := jq.NewJqInstaller()
    
    if !installer.IsInstalled() {
        fmt.Println("jq not found, installing...")
        if err := installer.Install(); err != nil {
            log.Printf("Installation failed: %v", err)
            fmt.Println(installer.GetInstallHelp())
            return
        }
    }
    
    version, err := installer.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("jq version: %s\n", version)
}
```

The installer supports automatic installation on macOS (via Homebrew), Linux (via apt, yum, dnf, or pacman), and provides manual installation guidance for Windows.
