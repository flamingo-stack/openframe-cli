A cross-platform jq installer and checker that provides automatic installation capabilities for the jq JSON processor on macOS, Linux, and Windows systems.

## Key Components

- **JqInstaller struct**: Main installer with platform-specific installation methods
- **IsJqRunning()**: Checks if jq is installed and functional by running version command
- **commandExists()**: Utility function to check if a command is available in PATH
- **NewJqInstaller()**: Factory function to create a new installer instance
- **Install()**: Platform-aware installation dispatcher
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **GetVersion()**: Retrieves the installed jq version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    // Check if jq is already running
    if !IsJqRunning() {
        fmt.Println("jq is not available")
        
        // Create installer
        installer := NewJqInstaller()
        
        // Get installation help
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation (macOS/Linux only)
        if err := installer.Install(); err != nil {
            log.Printf("Installation failed: %v", err)
            return
        }
    }
    
    // Get version
    installer := NewJqInstaller()
    version, err := installer.GetVersion()
    if err != nil {
        log.Printf("Failed to get version: %v", err)
        return
    }
    
    fmt.Printf("jq version: %s\n", version)
}
```

The installer supports automatic installation via Homebrew on macOS, and various package managers on Linux (apt, yum, dnf, pacman). Windows requires manual installation with helpful instructions provided.
