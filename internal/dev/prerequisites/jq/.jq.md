This package provides cross-platform installation and management utilities for the `jq` command-line JSON processor. It supports automatic installation on macOS, Linux, and provides guidance for Windows users.

## Key Components

- **JqInstaller**: Main struct for managing jq installation and detection
- **IsJqRunning()**: Checks if jq is installed and functional
- **NewJqInstaller()**: Creates a new installer instance
- **Install()**: Automatically installs jq based on the operating system
- **IsInstalled()**: Checks if jq is available on the system
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **GetVersion()**: Retrieves the installed jq version

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := jq.NewJqInstaller()
    
    // Check if jq is installed
    if !installer.IsInstalled() {
        fmt.Println("jq is not installed")
        
        // Get installation help
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation (macOS/Linux)
        if err := installer.Install(); err != nil {
            log.Printf("Failed to install jq: %v", err)
        }
    }
    
    // Check version
    if version, err := installer.GetVersion(); err == nil {
        fmt.Printf("jq version: %s\n", version)
    }
    
    // Verify jq is running
    if jq.IsJqRunning() {
        fmt.Println("jq is ready to use")
    }
}
```
