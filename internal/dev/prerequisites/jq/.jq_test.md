This file contains unit tests for the jq command-line tool installer functionality, validating installation detection, version checking, OS detection, and installation help generation.

## Key Components

- **TestNewJqInstaller**: Tests the creation of a new JqInstaller instance
- **TestJqInstaller_IsInstalled**: Verifies the installer can detect if jq is already installed
- **TestIsJqRunning**: Tests the standalone function that checks if jq process is running
- **TestJqInstaller_GetVersion**: Validates version retrieval for installed jq instances
- **TestCommandExists**: Tests utility function for command existence detection
- **TestJqInstallHelp**: Verifies installation help text generation
- **TestJqInstaller_OSDetection**: Tests OS detection methods (Debian-like, RHEL-like systems)
- **TestJqInstaller_Install**: Basic validation of the install method structure

## Usage Example

```go
func TestJqFunctionality(t *testing.T) {
    // Run tests with: go test -v
    installer := NewJqInstaller()
    
    // Check if jq is installed
    if installer.IsInstalled() {
        version, err := installer.GetVersion()
        assert.NoError(t, err)
        assert.NotEmpty(t, version)
    }
    
    // Test command existence
    assert.True(t, commandExists("echo"))
    assert.False(t, commandExists("nonexistent-command"))
}
```

The tests use system-agnostic approaches to avoid making actual system changes during testing while still validating the installer's core functionality.
