This file contains comprehensive unit tests for the jq installer package, validating the functionality of jq installation detection, version checking, and system compatibility methods.

## Key Components

- **TestNewJqInstaller** - Tests constructor for JqInstaller instance creation
- **TestJqInstaller_GetInstallHelp** - Validates installation help text generation
- **TestJqInstaller_IsInstalled** - Checks jq installation detection logic
- **TestIsJqRunning** - Tests standalone function for jq process detection
- **TestJqInstaller_GetVersion** - Validates version retrieval with proper error handling
- **TestCommandExists** - Tests utility function for command availability checking
- **TestJqInstaller_OSDetection** - Validates OS-specific detection methods (Debian/RHEL)

## Usage Example

```go
func TestJqFunctionality(t *testing.T) {
    // Test installer creation
    installer := NewJqInstaller()
    assert.NotNil(t, installer)
    
    // Test installation check
    isInstalled := installer.IsInstalled()
    assert.IsType(t, true, isInstalled)
    
    // Test version retrieval with conditional logic
    version, err := installer.GetVersion()
    if installer.IsInstalled() {
        assert.NoError(t, err)
        assert.NotEmpty(t, version)
    } else {
        assert.Error(t, err)
    }
    
    // Test OS detection
    assert.NotPanics(t, func() {
        installer.isDebianLike()
        installer.isRhelLike()
    })
}
```
