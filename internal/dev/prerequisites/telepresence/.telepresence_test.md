Test file that validates the functionality of the Telepresence installer package, ensuring proper detection, version checking, and help text generation.

## Key Components

- **TestNewTelepresenceInstaller**: Verifies installer instantiation
- **TestTelepresenceInstaller_GetInstallHelp**: Validates installation help text content
- **TestTelepresenceInstaller_IsInstalled**: Tests installation detection logic
- **TestIsTelepresenceRunning**: Checks runtime status detection
- **TestTelepresenceInstaller_GetVersion**: Validates version retrieval functionality
- **TestCommandExists**: Tests command availability checking utility
- **TestTelepresenceInstallHelp**: Verifies help text generation
- **TestTelepresenceInstaller_Install**: Basic install method structure validation

## Usage Example

```go
// Run specific test
go test -run TestNewTelepresenceInstaller

// Run all tests with verbose output
go test -v

// Run tests with coverage
go test -cover

// Run specific test pattern
go test -run "TestTelepresence.*Install"
```

The tests use testify/assert for assertions and handle both installed and non-installed scenarios gracefully. They avoid making actual system changes during testing while still validating the core functionality of command detection, version checking, and help text generation.
