This file contains comprehensive unit tests for the Telepresence installer functionality, verifying installation detection, version checking, and help text generation capabilities.

## Key Components

- **TestNewTelepresenceInstaller** - Tests creation of new installer instances
- **TestTelepresenceInstaller_GetInstallHelp** - Validates installation help text contains proper URLs and content
- **TestTelepresenceInstaller_IsInstalled** - Checks installation detection without system dependencies
- **TestIsTelepresenceRunning** - Tests standalone function for detecting running Telepresence processes
- **TestTelepresenceInstaller_GetVersion** - Validates version retrieval with proper error handling
- **TestCommandExists** - Tests utility function for command availability detection
- **TestTelepresenceInstallHelp** - Verifies help text generation functionality

## Usage Example

```go
func TestTelepresenceFeatures(t *testing.T) {
    // Test installer creation and basic functionality
    installer := NewTelepresenceInstaller()
    assert.NotNil(t, installer)
    
    // Check installation help
    help := installer.GetInstallHelp()
    assert.Contains(t, help, "telepresence")
    
    // Verify installation status
    isInstalled := installer.IsInstalled()
    if isInstalled {
        version, err := installer.GetVersion()
        assert.NoError(t, err)
        assert.NotEmpty(t, version)
    }
}
```
