This package provides installation and management functionality for Telepresence, a tool for local development against remote Kubernetes clusters. It handles platform-specific installation methods and daemon lifecycle management.

## Key Components

- **TelepresenceInstaller**: Main struct providing installation and management methods
- **commandExists()**: Utility function to check if a command is available in PATH
- **IsTelepresenceRunning()**: Checks if the Telepresence daemon is actively running
- **telepresenceInstallHelp()**: Returns platform-specific installation instructions

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := telepresence.NewTelepresenceInstaller()
    
    // Check if Telepresence is installed
    if !installer.IsInstalled() {
        fmt.Println("Telepresence not found")
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation (macOS/Linux)
        if err := installer.Install(); err != nil {
            log.Fatal(err)
        }
    }
    
    // Check version
    version, err := installer.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Telepresence version: %s\n", version)
    
    // Check if daemon is running
    if telepresence.IsTelepresenceRunning() {
        fmt.Println("Telepresence daemon is running")
    }
    
    // Initialize daemon for first-time setup
    if err := installer.EnsureConfig(); err != nil {
        log.Fatal(err)
    }
}
```

The installer supports automatic installation on macOS (via Homebrew) and Linux (direct binary download), while providing manual installation guidance for Windows and other platforms.
