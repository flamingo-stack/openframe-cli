A cross-platform installer and manager for Telepresence, providing automated installation and status checking across macOS, Linux, and Windows platforms.

## Key Components

- **`TelepresenceInstaller`** - Main struct providing installation and management functionality
- **`IsTelepresenceRunning()`** - Checks if the telepresence daemon is actively running
- **`NewTelepresenceInstaller()`** - Constructor for the installer instance
- **Platform-specific installers** - `installMacOS()`, `installLinux()`, `installLinuxCurl()`, `installLinuxWget()`
- **`GetVersion()`** - Retrieves the installed Telepresence version
- **`EnsureConfig()`** - Initializes the telepresence daemon configuration

## Usage Example

```go
// Create a new installer instance
installer := telepresence.NewTelepresenceInstaller()

// Check if Telepresence is installed
if !installer.IsInstalled() {
    fmt.Println("Installing Telepresence...")
    if err := installer.Install(); err != nil {
        fmt.Printf("Installation failed: %v\n", err)
        fmt.Println(installer.GetInstallHelp())
        return
    }
}

// Check if daemon is running
if telepresence.IsTelepresenceRunning() {
    fmt.Println("Telepresence is active")
} else {
    fmt.Println("Telepresence daemon not running")
}

// Get version information
version, err := installer.GetVersion()
if err == nil {
    fmt.Printf("Telepresence version: %s\n", version)
}
```

The installer handles platform-specific installation methods (Homebrew for macOS, direct binary download for Linux) and automatically initializes the telepresence daemon to handle sudo requirements upfront.
