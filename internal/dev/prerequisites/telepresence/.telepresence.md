This file provides a cross-platform installer and manager for Telepresence, a Kubernetes development tool that enables local development with remote cluster connectivity.

## Key Components

**Main Type:**
- `TelepresenceInstaller` - Core installer struct with methods for installation and management

**Key Functions:**
- `NewTelepresenceInstaller()` - Creates new installer instance
- `IsInstalled()` - Checks if Telepresence is installed on the system
- `Install()` - Automatically installs Telepresence based on the operating system
- `IsTelepresenceRunning()` - Checks if the Telepresence daemon is currently running
- `GetVersion()` - Returns the installed Telepresence version
- `GetInstallHelp()` - Provides platform-specific installation instructions
- `EnsureConfig()` - Initializes the Telepresence daemon

## Usage Example

```go
installer := NewTelepresenceInstaller()

// Check if already installed
if !installer.IsInstalled() {
    // Install automatically
    if err := installer.Install(); err != nil {
        fmt.Printf("Installation failed: %v\n", err)
        fmt.Println(installer.GetInstallHelp())
        return
    }
}

// Check version
version, err := installer.GetVersion()
if err != nil {
    fmt.Printf("Error getting version: %v\n", err)
}

// Ensure daemon is initialized
if err := installer.EnsureConfig(); err != nil {
    fmt.Printf("Failed to initialize: %v\n", err)
}

// Check if running
if IsTelepresenceRunning() {
    fmt.Println("Telepresence daemon is active")
}
```
