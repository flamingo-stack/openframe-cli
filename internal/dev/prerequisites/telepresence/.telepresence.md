Provides cross-platform installation and management utilities for Telepresence, a tool for local development against remote Kubernetes clusters. The package handles automatic installation, version checking, and daemon initialization across macOS, Linux, and Windows platforms.

## Key Components

- **TelepresenceInstaller**: Main struct providing installation and management methods
- **IsTelepresenceRunning()**: Global function to check if Telepresence daemon is active
- **NewTelepresenceInstaller()**: Constructor for creating installer instances
- **Install()**: Automated installation based on operating system
- **IsInstalled()**: Check if Telepresence is available in PATH
- **GetInstallHelp()**: Platform-specific installation instructions
- **GetVersion()**: Retrieve installed Telepresence version
- **EnsureConfig()**: Initialize Telepresence daemon

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := NewTelepresenceInstaller()
    
    // Check if already installed
    if !installer.IsInstalled() {
        fmt.Println("Installing Telepresence...")
        if err := installer.Install(); err != nil {
            log.Fatalf("Installation failed: %v", err)
        }
    }
    
    // Check version and status
    version, _ := installer.GetVersion()
    fmt.Printf("Telepresence version: %s\n", version)
    fmt.Printf("Daemon running: %t\n", IsTelepresenceRunning())
    
    // Ensure daemon is initialized
    installer.EnsureConfig()
}
```
