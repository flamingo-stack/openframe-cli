A cross-platform installer and utility package for managing Telepresence, which provides platform-specific installation methods and status checking for the Kubernetes development tool.

## Key Components

- **TelepresenceInstaller**: Main struct providing installation and management functionality
- **commandExists()**: Utility function to check if a command is available in PATH
- **IsTelepresenceRunning()**: Global function to check if Telepresence daemon is active
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Linux (curl/wget), and Windows (manual)
- **initializeTelepresence()**: Handles daemon initialization and sudo requirements

## Usage Example

```go
// Create installer instance
installer := telepresence.NewTelepresenceInstaller()

// Check if already installed
if !installer.IsInstalled() {
    fmt.Println("Installing Telepresence...")
    if err := installer.Install(); err != nil {
        fmt.Printf("Installation failed: %v\n", err)
        fmt.Println(installer.GetInstallHelp())
        return
    }
}

// Check if running
if telepresence.IsTelepresenceRunning() {
    fmt.Println("Telepresence daemon is running")
} else {
    // Initialize daemon if needed
    if err := installer.EnsureConfig(); err != nil {
        fmt.Printf("Failed to initialize: %v\n", err)
    }
}

// Get version info
version, err := installer.GetVersion()
if err == nil {
    fmt.Printf("Telepresence version: %s\n", version)
}
```
