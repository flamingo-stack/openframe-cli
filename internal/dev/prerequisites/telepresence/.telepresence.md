This package provides utilities for installing and managing Telepresence, a tool for local Kubernetes development. It handles platform-specific installation methods and daemon initialization.

## Key Components

- **TelepresenceInstaller**: Main struct providing installation and management functionality
- **IsTelepresenceRunning()**: Checks if Telepresence daemon is currently running
- **NewTelepresenceInstaller()**: Creates a new installer instance
- **Install()**: Automatically installs Telepresence based on the operating system
- **IsInstalled()**: Checks if Telepresence is available in the system PATH
- **GetVersion()**: Returns the installed Telepresence version
- **EnsureConfig()**: Initializes the Telepresence daemon

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := telepresence.NewTelepresenceInstaller()
    
    // Check if Telepresence is installed
    if !installer.IsInstalled() {
        fmt.Println("Telepresence not found, installing...")
        if err := installer.Install(); err != nil {
            log.Fatal("Installation failed:", err)
        }
    }
    
    // Check version
    version, err := installer.GetVersion()
    if err != nil {
        log.Fatal("Failed to get version:", err)
    }
    fmt.Printf("Telepresence version: %s\n", version)
    
    // Check if daemon is running
    if telepresence.IsTelepresenceRunning() {
        fmt.Println("Telepresence daemon is active")
    }
}
```
