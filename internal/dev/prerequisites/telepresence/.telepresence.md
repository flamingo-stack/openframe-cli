This package provides cross-platform installation and management functionality for Telepresence, a Kubernetes development tool that enables local development against remote clusters.

## Key Components

- **TelepresenceInstaller**: Main struct that handles installation, version checking, and daemon management
- **commandExists()**: Utility function to check if a command is available in PATH
- **isTelepresenceInstalled()**: Checks if Telepresence is installed on the system
- **IsTelepresenceRunning()**: Public function to verify if the Telepresence daemon is running
- **telepresenceInstallHelp()**: Platform-specific installation instructions

## Usage Example

```go
// Create installer instance
installer := NewTelepresenceInstaller()

// Check if already installed
if !installer.IsInstalled() {
    fmt.Println("Installing Telepresence...")
    if err := installer.Install(); err != nil {
        log.Fatalf("Installation failed: %v", err)
    }
}

// Check version
version, err := installer.GetVersion()
if err != nil {
    log.Printf("Could not get version: %v", err)
} else {
    fmt.Printf("Telepresence version: %s\n", version)
}

// Check if daemon is running
if IsTelepresenceRunning() {
    fmt.Println("Telepresence is active")
}

// Initialize daemon if needed
if err := installer.EnsureConfig(); err != nil {
    log.Printf("Failed to initialize: %v", err)
}
```

The installer supports automatic installation on macOS (via Homebrew) and Linux (via direct download), while providing helpful instructions for Windows. It includes daemon initialization to handle sudo requirements upfront and provides methods to check installation status and version information.
