This file provides a user interface layer for setting up Kubernetes service intercepts through interactive CLI prompts, handling service discovery, port selection, and input validation.

## Key Components

- **InterceptUI**: Main struct that orchestrates user interactions for intercept configuration
- **ServiceInfo**: Data structure containing service details including name, namespace, and available ports
- **PromptForService**: Interactive service discovery across all namespaces with validation
- **PromptForKubernetesPort**: Port selection interface with automatic single-port handling
- **PromptForLocalPort**: Local port configuration with validation
- **findServiceInCluster**: Cross-namespace service search functionality
- **validatePort**: Port number validation for user inputs

## Usage Example

```go
// Initialize the intercept UI
kubernetesClient := // ... your k8s client
serviceClient := // ... your service client
ui := NewInterceptUI(kubernetesClient, serviceClient)

// Interactive service setup
ctx := context.Background()

// Get service from user
serviceInfo, err := ui.PromptForService(ctx)
if err != nil {
    log.Fatal(err)
}

// Select Kubernetes port
kubernetesPort, err := ui.PromptForKubernetesPort(serviceInfo.Ports)
if err != nil {
    log.Fatal(err)
}

// Get local port
localPort, err := ui.PromptForLocalPort(kubernetesPort)
if err != nil {
    log.Fatal(err)
}

// Use the configured ports for intercept setup
fmt.Printf("Intercepting %s:%d -> localhost:%d\n", 
    serviceInfo.Name, kubernetesPort.Port, localPort)
```
