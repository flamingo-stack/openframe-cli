This file provides a user interface handler for setting up Kubernetes service intercepts, facilitating interactive service discovery and port configuration through CLI prompts.

## Key Components

- **`InterceptUI`** - Main struct handling intercept setup interactions with Kubernetes and service clients
- **`ServiceInfo`** - Contains service metadata including name, namespace, ports, and availability status
- **`PromptForService()`** - Interactive service name input with cluster-wide validation
- **`PromptForKubernetesPort()`** - Port selection interface for services with multiple exposed ports
- **`PromptForLocalPort()`** - Local port configuration with validation
- **`findServiceInCluster()`** - Cross-namespace service discovery functionality

## Usage Example

```go
// Initialize the UI handler
kubeClient := // ... your Kubernetes client
serviceClient := // ... your service client
ui := NewInterceptUI(kubeClient, serviceClient)

// Interactive service setup
service, err := ui.PromptForService(ctx)
if err != nil {
    log.Fatal(err)
}

// Select port for intercept
kubernetesPort, err := ui.PromptForKubernetesPort(service.Ports)
if err != nil {
    log.Fatal(err)
}

// Configure local target port
localPort, err := ui.PromptForLocalPort(kubernetesPort)
if err != nil {
    log.Fatal(err)
}

// Now ready to set up intercept with selected configuration
```
