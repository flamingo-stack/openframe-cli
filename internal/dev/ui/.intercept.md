Provides an interactive user interface for setting up Kubernetes service intercepts with validation and port selection capabilities.

## Key Components

- **InterceptUI**: Main struct handling user interactions for intercept configuration
- **ServiceInfo**: Data structure containing service details including name, namespace, and available ports
- **PromptForService()**: Interactive service name input with cluster-wide validation
- **PromptForKubernetesPort()**: Port selection interface with automatic single-port handling
- **PromptForLocalPort()**: Local port configuration with validation
- **findServiceInCluster()**: Cross-namespace service discovery functionality
- **validatePort()**: Port number validation (1-65535 range)

## Usage Example

```go
package main

import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/dev/ui"
    "github.com/flamingo-stack/openframe-cli/internal/dev/services/intercept"
)

func main() {
    // Initialize clients
    kubeClient := intercept.NewKubernetesClient()
    serviceClient := intercept.NewServiceClient()
    
    // Create UI handler
    interceptUI := ui.NewInterceptUI(kubeClient, serviceClient)
    
    ctx := context.Background()
    
    // Interactive service selection
    service, err := interceptUI.PromptForService(ctx)
    if err != nil {
        log.Fatal(err)
    }
    
    // Select Kubernetes port
    kubePort, err := interceptUI.PromptForKubernetesPort(service.Ports)
    if err != nil {
        log.Fatal(err)
    }
    
    // Configure local port
    localPort, err := interceptUI.PromptForLocalPort(kubePort)
    if err != nil {
        log.Fatal(err)
    }
    
    // Use configuration for intercept setup
    fmt.Printf("Intercepting %s:%d -> localhost:%d\n", 
        service.Name, kubePort.Port, localPort)
}
```
