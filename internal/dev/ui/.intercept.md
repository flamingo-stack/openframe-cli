Provides a user-friendly interface for setting up Kubernetes service intercepts, handling service discovery, port selection, and user input validation with interactive prompts.

## Key Components

- **InterceptUI** - Main UI handler for intercept operations with Kubernetes and service clients
- **ServiceInfo** - Struct containing service details including name, namespace, ports, and discovery status
- **NewInterceptUI()** - Constructor function that initializes the UI with required clients
- **PromptForService()** - Interactive service name input with cluster-wide validation
- **PromptForKubernetesPort()** - Port selection interface with automatic single-port handling
- **PromptForLocalPort()** - Local port configuration with validation
- **findServiceInCluster()** - Cross-namespace service discovery functionality
- **validatePort()** - Port number validation (1-65535 range)

## Usage Example

```go
// Initialize the intercept UI
kubeClient := // ... your Kubernetes client
serviceClient := // ... your service client
ui := NewInterceptUI(kubeClient, serviceClient)

// Get service information from user
serviceInfo, err := ui.PromptForService(ctx)
if err != nil {
    log.Fatal(err)
}

// Select Kubernetes port to intercept
selectedPort, err := ui.PromptForKubernetesPort(serviceInfo.Ports)
if err != nil {
    log.Fatal(err)
}

// Configure local port for traffic forwarding
localPort, err := ui.PromptForLocalPort(selectedPort)
if err != nil {
    log.Fatal(err)
}

// Now you have all the configuration needed for intercept setup
fmt.Printf("Intercepting %s:%d -> localhost:%d\n", 
    serviceInfo.Name, selectedPort.Port, localPort)
```
