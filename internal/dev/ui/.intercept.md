This file provides a user interface layer for setting up Kubernetes service intercepts, handling user prompts and validation for service selection and port configuration.

## Key Components

- **InterceptUI**: Main struct that orchestrates intercept setup through user interactions
- **ServiceInfo**: Data structure containing service details including name, namespace, and available ports
- **PromptForService()**: Prompts user for service name and validates its existence across all namespaces
- **PromptForKubernetesPort()**: Handles port selection from available service ports
- **PromptForLocalPort()**: Prompts for local target port with validation
- **findServiceInCluster()**: Searches for services across all namespaces in the cluster

## Usage Example

```go
// Initialize the intercept UI
interceptUI := NewInterceptUI(kubernetesClient, serviceClient)

// Get service from user input
serviceInfo, err := interceptUI.PromptForService(ctx)
if err != nil {
    log.Fatal(err)
}

// Select Kubernetes port to intercept
kubernetesPort, err := interceptUI.PromptForKubernetesPort(serviceInfo.Ports)
if err != nil {
    log.Fatal(err)
}

// Get local port for forwarding
localPort, err := interceptUI.PromptForLocalPort(kubernetesPort)
if err != nil {
    log.Fatal(err)
}

// Now use serviceInfo.Name, serviceInfo.Namespace, kubernetesPort, and localPort
// to set up the actual intercept
```
