Provides UI interactions for setting up service intercepts in Kubernetes, handling service discovery, port selection, and user input validation.

## Key Components

- **`InterceptUI`** - Main struct that handles intercept configuration through user prompts
- **`ServiceInfo`** - Contains discovered service details including name, namespace, and available ports
- **`PromptForService`** - Validates service existence across all namespaces and prompts for service name
- **`PromptForKubernetesPort`** - Handles port selection from available service ports
- **`PromptForLocalPort`** - Prompts for local forwarding port with validation

## Usage Example

```go
// Initialize the UI handler
ui := NewInterceptUI(kubernetesClient, serviceClient)

// Get service information from user
serviceInfo, err := ui.PromptForService(ctx)
if err != nil {
    log.Fatal(err)
}

// Select Kubernetes port to intercept
kubernetesPort, err := ui.PromptForKubernetesPort(serviceInfo.Ports)
if err != nil {
    log.Fatal(err)
}

// Get local port for forwarding
localPort, err := ui.PromptForLocalPort(kubernetesPort)
if err != nil {
    log.Fatal(err)
}

// Now use serviceInfo, kubernetesPort, and localPort for intercept setup
```
