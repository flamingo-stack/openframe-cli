Provides a terminal UI interface for setting up Kubernetes service intercepts, allowing users to select services, ports, and configure local forwarding through interactive prompts.

## Key Components

- **InterceptUI**: Main struct that handles user interactions for intercept configuration
- **ServiceInfo**: Contains service details including name, namespace, and available ports
- **PromptForService()**: Interactive service name input with cluster-wide validation
- **PromptForKubernetesPort()**: Port selection from available service ports
- **PromptForLocalPort()**: Local port configuration with validation
- **findServiceInCluster()**: Searches for services across all namespaces
- **validatePort()**: Port number validation (1-65535 range)

## Usage Example

```go
// Create intercept UI with clients
kubeClient := // ... kubernetes client
serviceClient := // ... service client
ui := NewInterceptUI(kubeClient, serviceClient)

// Interactive service selection
serviceInfo, err := ui.PromptForService(ctx)
if err != nil {
    log.Fatal(err)
}

// Select Kubernetes port to intercept
kubePort, err := ui.PromptForKubernetesPort(serviceInfo.Ports)
if err != nil {
    log.Fatal(err)
}

// Configure local forwarding port
localPort, err := ui.PromptForLocalPort(kubePort)
if err != nil {
    log.Fatal(err)
}

// Now ready to set up intercept with selected configuration
fmt.Printf("Intercepting %s:%d -> localhost:%d\n", 
    serviceInfo.Name, kubePort.Port, localPort)
```

The UI automatically handles single-port services and provides selection menus for multi-port services, with comprehensive validation and user-friendly error messages.
