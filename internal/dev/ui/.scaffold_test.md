Test suite for the SkaffoldUI component that validates Skaffold YAML file discovery, categorization, and service name extraction functionality. Includes comprehensive tests for file system operations, path parsing, and UI organization features.

## Key Components

- **TestNewSkaffoldUI**: Tests SkaffoldUI constructor with verbose flag options
- **TestSkaffoldUI_findSkaffoldYamlFiles**: Validates discovery of skaffold.yaml/.yml files in directory structures
- **TestSkaffoldUI_extractServiceName**: Tests service name extraction from file paths with various patterns
- **TestSkaffoldUI_categorizeSkaffoldFiles**: Tests organization of Skaffold files into categorized groups
- **Error/Edge Case Tests**: Validates handling of non-existent paths, empty inputs, and malformed data
- **Structure Tests**: Verifies SkaffoldFile and SkaffoldCategory struct definitions

## Usage Example

```go
func TestCustomScenario(t *testing.T) {
    // Create SkaffoldUI instance for testing
    ui := NewSkaffoldUI(false)
    
    // Test file discovery in temp directory
    tmpDir := t.TempDir()
    err := os.MkdirAll(filepath.Join(tmpDir, "services", "api"), 0755)
    require.NoError(t, err)
    
    // Create test skaffold file
    skaffoldPath := filepath.Join(tmpDir, "services", "api", "skaffold.yaml")
    err = os.WriteFile(skaffoldPath, []byte("# test"), 0644)
    require.NoError(t, err)
    
    // Test file discovery
    files := ui.findSkaffoldYamlFiles(tmpDir)
    assert.Len(t, files, 1)
    assert.Equal(t, "api", files[0].ServiceName)
}
```
