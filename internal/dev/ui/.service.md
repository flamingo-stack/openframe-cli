This service provides a unified interface for interactive development UI operations, primarily focused on Kubernetes service intercept workflows with user-friendly prompting and validation.

## Key Components

- **Service**: Main service struct that orchestrates UI interactions and intercept operations
- **NewService/NewServiceWithExecutor**: Factory functions for creating service instances with different configurations
- **InteractiveInterceptSetup**: Guides users through service, port, and configuration selection
- **RunFullInteractiveIntercept**: Complete workflow for setting up and starting service intercepts
- **InterceptSetup**: Configuration struct containing intercept parameters

## Usage Example

```go
// Create service with executor for interactive workflows
executor := executor.NewCommandExecutor()
service := ui.NewService(executor, true) // verbose mode

// Run complete interactive intercept
ctx := context.Background()
err := service.RunFullInteractiveIntercept(ctx)
if err != nil {
    log.Fatal(err)
}

// Or use step-by-step approach
setup, err := service.InteractiveInterceptSetup(ctx)
if err != nil {
    log.Fatal(err)
}

// Setup contains selected service, ports, and configuration
fmt.Printf("Intercepting %s:%d -> localhost:%d\n", 
    setup.ServiceName, setup.KubernetesPort.Port, setup.LocalPort)
```

The service handles kubectl context validation, cluster connectivity checks, and converts user selections into appropriate intercept configurations for the underlying intercept service.
