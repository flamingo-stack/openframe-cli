This service provides a unified interface for managing development UI interactions, specifically for interactive Kubernetes service interception workflows.

## Key Components

- **Service**: Main service struct that orchestrates UI interactions and command execution
- **NewService()**: Constructor for basic service with intercept UI capabilities
- **NewServiceWithExecutor()**: Constructor for service with interactive command execution
- **InteractiveInterceptSetup()**: Prompts user for service, port, and configuration details
- **RunFullInteractiveIntercept()**: Complete workflow for setting up service interception
- **InterceptSetup**: Configuration struct containing service name, namespace, and port mappings

## Usage Example

```go
// Create service with executor for interactive workflows
executor := executor.NewCommandExecutor()
service := NewServiceWithExecutor(executor, true) // verbose mode

// Run complete interactive intercept workflow
ctx := context.Background()
err := service.RunFullInteractiveIntercept(ctx)
if err != nil {
    log.Fatal(err)
}

// Or use step-by-step approach
setup, err := service.InteractiveInterceptSetup(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Intercepting %s on port %d\n", setup.ServiceName, setup.LocalPort)
```

The service validates kubectl connectivity, guides users through service selection, and configures port forwarding for local development against Kubernetes services.
