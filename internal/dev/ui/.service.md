This service provides a unified interface for managing interactive development workflows, particularly focused on Kubernetes service intercept operations.

## Key Components

- **Service**: Main service struct that orchestrates UI interactions and intercept workflows
- **NewService()**: Constructor for basic service with intercept UI capabilities
- **NewServiceWithExecutor()**: Constructor that includes command executor for interactive workflows
- **InteractiveInterceptSetup()**: Guides users through interactive intercept configuration
- **RunFullInteractiveIntercept()**: Complete workflow for setting up and starting service intercepts
- **InterceptSetup**: Configuration struct containing service details and port mappings

## Usage Example

```go
// Create service with executor for full interactive workflow
executor := executor.NewCommandExecutor()
service := NewServiceWithExecutor(executor, true)

// Run complete interactive intercept setup
ctx := context.Background()
if err := service.RunFullInteractiveIntercept(ctx); err != nil {
    log.Fatal(err)
}

// Or use step-by-step approach
setup, err := service.InteractiveInterceptSetup(ctx)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Intercepting %s:%d -> localhost:%d\n", 
    setup.ServiceName, setup.KubernetesPort.Port, setup.LocalPort)
```

The service handles kubectl context validation, cluster connectivity checks, and user prompts for service selection and port configuration, making it easy to set up development intercepts with Kubernetes services.
