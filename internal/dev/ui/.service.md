A service layer that provides unified interfaces for dev UI interactions, primarily focused on interactive Kubernetes service intercept workflows.

## Key Components

- **Service**: Main service struct containing interceptUI, executor, and verbose flag
- **NewService**: Constructor accepting Kubernetes and service clients
- **NewServiceWithExecutor**: Constructor accepting command executor for interactive workflows
- **InteractiveInterceptSetup**: Prompts user for service, ports, and returns configuration
- **RunFullInteractiveIntercept**: Complete workflow for setting up intercepts with context checking
- **InterceptSetup**: Configuration struct holding service name, namespace, and port mappings

## Usage Example

```go
// Create service with executor for interactive workflows
executor := executor.NewCommandExecutor()
uiService := ui.NewServiceWithExecutor(executor, true)

// Run complete interactive intercept
ctx := context.Background()
err := uiService.RunFullInteractiveIntercept(ctx)
if err != nil {
    log.Fatal(err)
}

// Or use step-by-step setup
setup, err := uiService.InteractiveInterceptSetup(ctx)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Service: %s, Local Port: %d\n", 
    setup.ServiceName, setup.LocalPort)
```

The service handles kubectl context validation, cluster connectivity checks, and converts user input into intercept configurations for the underlying intercept service.
