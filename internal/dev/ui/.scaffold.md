Handles Skaffold configuration file discovery and user selection for development environments. Provides a terminal UI for browsing and selecting from available services with categorization and search capabilities.

## Key Components

- **SkaffoldUI**: Main struct that manages skaffold file discovery and user interaction
- **ServiceSelection**: Represents a selected service with name, file path, and directory
- **SkaffoldCategory**: Groups skaffold files by type (OpenFrame Services, Integrated Tools, etc.)
- **DiscoverAndSelectService()**: Primary method that finds skaffold files and prompts user selection
- **findSkaffoldYamlFiles()**: Recursively searches for `skaffold.yaml`/`skaffold.yml` files
- **categorizeSkaffoldFiles()**: Organizes files into logical categories based on directory structure
- **extractServiceName()**: Derives clean service names from file paths

## Usage Example

```go
// Create a new SkaffoldUI instance
ui := NewSkaffoldUI(true) // verbose mode enabled

// Discover and let user select a service
selection, err := ui.DiscoverAndSelectService()
if err != nil {
    if err == ErrNoSkaffoldFiles {
        log.Fatal("No skaffold.yaml files found in project")
    }
    log.Fatalf("Selection failed: %v", err)
}

// Use the selected service
fmt.Printf("Selected: %s\n", selection.ServiceName)
fmt.Printf("Path: %s\n", selection.FilePath)
fmt.Printf("Directory: %s\n", selection.Directory)
```
