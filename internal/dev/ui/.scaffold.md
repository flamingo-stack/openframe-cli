This file provides a comprehensive UI system for discovering, categorizing, and selecting Skaffold configuration files within OpenFrame projects.

## Key Components

### Core Types
- **`ServiceSelection`** - Represents a selected Skaffold service with name, file path, and directory
- **`SkaffoldUI`** - Main UI handler with verbose logging support
- **`SkaffoldCategory`** - Groups Skaffold files by project type (OpenFrame Services, Integrated Tools, etc.)
- **`SkaffoldFile`** - Individual Skaffold configuration file representation

### Main Functions
- **`NewSkaffoldUI(verbose bool)`** - Creates a new SkaffoldUI instance
- **`DiscoverAndSelectService()`** - Main workflow: discovers files, categorizes them, and prompts user selection
- **`findSkaffoldYamlFiles(rootPath)`** - Recursively searches for skaffold.yaml/yml files
- **`categorizeSkaffoldFiles(files)`** - Organizes files into logical categories
- **`extractServiceName(filePath)`** - Derives clean service names from file paths

## Usage Example

```go
// Create UI instance with verbose logging
skaffoldUI := ui.NewSkaffoldUI(true)

// Discover and select a service
selection, err := skaffoldUI.DiscoverAndSelectService()
if err != nil {
    if errors.Is(err, ui.ErrNoSkaffoldFiles) {
        log.Println("No Skaffold files found in project")
        return
    }
    log.Fatalf("Service selection failed: %v", err)
}

// Use the selected service
fmt.Printf("Selected service: %s\n", selection.ServiceName)
fmt.Printf("File path: %s\n", selection.FilePath)
fmt.Printf("Directory: %s\n", selection.Directory)
```

The system automatically categorizes services into OpenFrame Services, Integrated Tools, Client Applications, and Other Services, providing an intuitive selection interface with search functionality.
