Manages Skaffold configuration discovery and service selection for the OpenFrame CLI, providing an interactive interface to find and choose from available skaffold.yaml files in a project directory.

## Key Components

- **SkaffoldUI** - Main struct for handling Skaffold configuration discovery and user interaction
- **ServiceSelection** - Represents a selected Skaffold service with name, file path, and directory
- **SkaffoldCategory/SkaffoldFile** - Data structures for organizing discovered configuration files
- **DiscoverAndSelectService()** - Core method that finds skaffold.yaml files and prompts user selection
- **findSkaffoldYamlFiles()** - Recursively searches for Skaffold configuration files
- **categorizeSkaffoldFiles()** - Groups files by type (OpenFrame Services, Integrated Tools, etc.)

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/ui"
)

// Create a new SkaffoldUI instance
skaffoldUI := ui.NewSkaffoldUI(true) // verbose mode enabled

// Discover and select a service
selection, err := skaffoldUI.DiscoverAndSelectService()
if err != nil {
    if err == ui.ErrNoSkaffoldFiles {
        fmt.Println("No Skaffold files found in project")
        return
    }
    log.Fatal(err)
}

// Use the selected service
fmt.Printf("Selected: %s at %s\n", selection.ServiceName, selection.FilePath)
fmt.Printf("Working directory: %s\n", selection.Directory)
```
