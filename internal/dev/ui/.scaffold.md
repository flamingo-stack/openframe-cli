Provides UI functionality for discovering and selecting Skaffold configuration files in a project. Handles file discovery, categorization by service type, and user selection through an interactive interface.

## Key Components

**Types:**
- `ServiceSelection` - Represents a selected Skaffold service with name, file path, and directory
- `SkaffoldUI` - Main struct for handling Skaffold configuration discovery and selection
- `SkaffoldCategory` - Groups Skaffold files by category (OpenFrame Services, Integrated Tools, etc.)
- `SkaffoldFile` - Represents individual skaffold.yaml file metadata

**Key Methods:**
- `NewSkaffoldUI(verbose bool)` - Creates new SkaffoldUI instance
- `DiscoverAndSelectService()` - Main method to find and prompt user to select a service
- `findSkaffoldYamlFiles(rootPath string)` - Recursively searches for skaffold.yaml files
- `categorizeSkaffoldFiles()` - Organizes files by location/purpose
- `extractServiceName()` - Derives clean service names from file paths

## Usage Example

```go
// Create UI instance
skaffoldUI := NewSkaffoldUI(true) // verbose mode

// Discover and select service
selection, err := skaffoldUI.DiscoverAndSelectService()
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Selected: %s\n", selection.ServiceName)
fmt.Printf("Path: %s\n", selection.FilePath)
fmt.Printf("Directory: %s\n", selection.Directory)
```

The component automatically categorizes services (OpenFrame Services, Integrated Tools, Client Applications) and provides an interactive selection interface with search capabilities.
