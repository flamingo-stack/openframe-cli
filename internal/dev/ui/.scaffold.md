A CLI utility for discovering, categorizing, and selecting Skaffold configuration files within a project directory. Provides an interactive interface for users to choose from available services organized by type.

## Key Components

- **ServiceSelection**: Struct containing selected service metadata (name, file path, directory)
- **SkaffoldUI**: Main interface for Skaffold file discovery and selection operations
- **SkaffoldCategory**: Groups related Skaffold files by project type (OpenFrame Services, Integrated Tools, etc.)
- **DiscoverAndSelectService()**: Primary method that finds all `skaffold.yaml` files and prompts user selection
- **findSkaffoldYamlFiles()**: Recursively searches directory tree for Skaffold configuration files
- **categorizeSkaffoldFiles()**: Organizes discovered files into logical groups based on file path patterns
- **extractServiceName()**: Parses file paths to generate clean, user-friendly service names

## Usage Example

```go
// Create a new SkaffoldUI instance with verbose logging
ui := NewSkaffoldUI(true)

// Discover and select a service interactively
selection, err := ui.DiscoverAndSelectService()
if err != nil {
    if err == ErrNoSkaffoldFiles {
        // Handle case where no skaffold.yaml files exist
        log.Fatal("No Skaffold configurations found")
    }
    log.Fatalf("Selection failed: %v", err)
}

// Use the selected service
fmt.Printf("Selected: %s at %s\n", selection.ServiceName, selection.FilePath)
fmt.Printf("Working directory: %s\n", selection.Directory)
```
