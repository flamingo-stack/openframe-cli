This file provides a UI component for discovering and selecting Skaffold configuration files within a project directory structure.

## Key Components

**Types:**
- `SkaffoldUI` - Main UI handler with verbose logging support
- `ServiceSelection` - Represents a selected service with name, file path, and directory
- `SkaffoldCategory` - Groups skaffold files by category (OpenFrame Services, Integrated Tools, etc.)
- `SkaffoldFile` - Represents individual skaffold configuration files

**Key Functions:**
- `NewSkaffoldUI(verbose bool)` - Creates a new SkaffoldUI instance
- `DiscoverAndSelectService()` - Main method to find and select skaffold files
- `findSkaffoldYamlFiles(rootPath string)` - Recursively searches for skaffold.yaml/yml files
- `categorizeSkaffoldFiles()` - Organizes files by project structure (services, tools, clients)
- `extractServiceName()` - Derives clean service names from file paths

## Usage Example

```go
// Create UI instance with verbose logging
skaffoldUI := NewSkaffoldUI(true)

// Discover and select a service
selection, err := skaffoldUI.DiscoverAndSelectService()
if err != nil {
    if err == ErrNoSkaffoldFiles {
        // Handle no files found
        return
    }
    log.Fatal(err)
}

// Use the selected service
fmt.Printf("Selected: %s at %s\n", selection.ServiceName, selection.FilePath)
fmt.Printf("Working directory: %s\n", selection.Directory)
```

The component automatically categorizes services by their location in the project structure and provides an interactive selection interface with search capabilities.
