This file provides UI functionality for discovering and selecting Skaffold configuration files in a project directory.

## Key Components

- **SkaffoldUI**: Main struct that handles Skaffold file discovery and user selection with verbose logging support
- **ServiceSelection**: Represents a selected service with name, file path, and directory information
- **SkaffoldCategory**: Groups Skaffold files by category (OpenFrame Services, Integrated Tools, Client Applications, Other)
- **SkaffoldFile**: Represents an individual Skaffold configuration file with service name and path
- **DiscoverAndSelectService()**: Main method that discovers all `skaffold.yaml` files and prompts user selection
- **findSkaffoldYamlFiles()**: Recursively searches for Skaffold configuration files in the project
- **categorizeSkaffoldFiles()**: Organizes discovered files by their location and purpose
- **extractServiceName()**: Extracts clean service names from file paths using path-based heuristics

## Usage Example

```go
// Create a new SkaffoldUI instance
skaffoldUI := NewSkaffoldUI(true) // verbose mode enabled

// Discover and select a service
selection, err := skaffoldUI.DiscoverAndSelectService()
if err != nil {
    if err == ErrNoSkaffoldFiles {
        // Handle case where no skaffold.yaml files found
        return
    }
    log.Fatal(err)
}

// Use the selected service
fmt.Printf("Selected service: %s\n", selection.ServiceName)
fmt.Printf("File path: %s\n", selection.FilePath)
fmt.Printf("Directory: %s\n", selection.Directory)
```

The UI automatically categorizes services, provides search functionality, and handles the complete workflow from discovery to selection.
