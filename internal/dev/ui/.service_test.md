This file contains unit tests for the UI service component, testing service initialization, interceptUI retrieval, and intercept setup structure validation.

## Key Components

- **TestNewService**: Tests the creation of a new UI service with mock Kubernetes clients
- **TestService_GetInterceptUI**: Verifies the GetInterceptUI method returns the correct interceptUI instance
- **TestInterceptSetup_Structure**: Validates the structure and field assignments of InterceptSetup configuration
- **InterceptSetup**: Test data structure for service interception configuration with service name, namespace, ports

## Usage Example

```go
func TestUIService(t *testing.T) {
    // Initialize test mode and create mock client
    testutil.InitializeTestMode()
    client := devMocks.NewMockKubernetesClient()
    
    // Create service instance
    service := NewService(client, client)
    assert.NotNil(t, service)
    
    // Test intercept UI retrieval
    interceptUI := service.GetInterceptUI()
    assert.NotNil(t, interceptUI)
    
    // Test intercept setup structure
    setup := &InterceptSetup{
        ServiceName: "my-service",
        Namespace:   "production",
        LocalPort:   3000,
        KubernetesPort: &intercept.ServicePort{
            Name: "http",
            Port: 8080,
        },
    }
    assert.Equal(t, "my-service", setup.ServiceName)
}
```
