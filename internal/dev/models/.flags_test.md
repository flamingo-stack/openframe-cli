This test file validates the flag structures and global flag functionality used throughout the application's command-line interface. It ensures proper default values, field assignments, and flag registration behavior.

## Key Components

- **TestInterceptFlags_DefaultValues** - Tests default zero values for InterceptFlags struct
- **TestInterceptFlags_WithValues** - Validates proper assignment of InterceptFlags fields
- **TestScaffoldFlags_DefaultValues** - Tests default zero values for ScaffoldFlags struct  
- **TestScaffoldFlags_WithValues** - Validates proper assignment of ScaffoldFlags fields
- **TestAddGlobalFlags** - Tests global flag registration with cobra commands
- **TestFlags_EdgeCases** - Tests edge cases like empty slices and negative values
- **TestFlags_StringRepresentation** - Validates string formatting of flag structures

## Usage Example

```go
// Running the tests
go test -v ./models -run TestInterceptFlags

// Example of testing custom flag values
func TestCustomFlags(t *testing.T) {
    flags := &InterceptFlags{
        Port:      8080,
        Namespace: "production",
        Global:    true,
    }
    
    assert.Equal(t, 8080, flags.Port)
    assert.Equal(t, "production", flags.Namespace)
    assert.True(t, flags.Global)
}

// Testing global flags on a command
cmd := &cobra.Command{Use: "test"}
AddGlobalFlags(cmd)
verbose, _ := cmd.PersistentFlags().GetBool("verbose")
```
