This test file validates flag-related data structures and functionality, ensuring proper default values, field assignments, and global flag registration for CLI commands.

## Key Components

- **InterceptFlags Tests**: Validates default values and field assignments for intercept command configuration
- **ScaffoldFlags Tests**: Tests scaffold command flag structure initialization and value setting
- **AddGlobalFlags Tests**: Verifies global CLI flags (verbose, silent, dry-run) are properly registered with cobra commands
- **Edge Case Tests**: Covers boundary conditions like empty headers, zero/negative ports
- **String Representation Tests**: Ensures flag structs can be formatted for debugging

## Usage Example

```go
// Test default flag values
func TestMyFlags_Defaults(t *testing.T) {
    flags := &InterceptFlags{}
    assert.Equal(t, 0, flags.Port)
    assert.Equal(t, "", flags.Namespace)
    assert.False(t, flags.Global)
}

// Test flag assignment
func TestMyFlags_WithValues(t *testing.T) {
    flags := &InterceptFlags{
        Port:      8080,
        Namespace: "production",
        Header:    []string{"auth=token"},
    }
    assert.Equal(t, 8080, flags.Port)
    assert.Equal(t, "production", flags.Namespace)
}

// Test global flag registration
func TestGlobalFlagSetup(t *testing.T) {
    cmd := &cobra.Command{Use: "test"}
    AddGlobalFlags(cmd)
    
    verbose, err := cmd.PersistentFlags().GetBool("verbose")
    assert.NoError(t, err)
    assert.False(t, verbose) // default value
}
```
