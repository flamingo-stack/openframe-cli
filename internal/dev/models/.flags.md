Defines flag structures and utilities for CLI commands in a Kubernetes development tool, providing configuration options for intercepting and scaffolding services.

## Key Components

- **`InterceptFlags`** - Configuration struct for traffic interception commands with options for port forwarding, namespaces, volume mounts, and header-based routing
- **`ScaffoldFlags`** - Configuration struct for service scaffolding commands with Docker image settings, sync directories, and Kubernetes resources
- **`AddGlobalFlags()`** - Helper function that adds common CLI flags (verbose, silent, dry-run) to cobra commands

## Usage Example

```go
package main

import (
    "github.com/spf13/cobra"
    "your-project/models"
)

func createInterceptCommand() *cobra.Command {
    var flags models.InterceptFlags
    
    cmd := &cobra.Command{
        Use: "intercept",
        Run: func(cmd *cobra.Command, args []string) {
            // Use flags.Port, flags.Namespace, etc.
        },
    }
    
    cmd.Flags().IntVarP(&flags.Port, "port", "p", 8080, "Local port")
    cmd.Flags().StringVar(&flags.Namespace, "namespace", "default", "K8s namespace")
    cmd.Flags().BoolVar(&flags.Global, "global", false, "Intercept all traffic")
    
    // Add global flags
    models.AddGlobalFlags(cmd)
    
    return cmd
}
```
