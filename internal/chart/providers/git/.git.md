This file defines data structures for Git operations, specifically for handling the results of repository cloning.

## Key Components

- **CloneResult**: A struct that encapsulates the outcome of a git clone operation, containing paths to the temporary directory and chart location.

## Usage Example

```go
package main

import "your-module/git"

func main() {
    // Create a CloneResult after cloning a repository
    result := git.CloneResult{
        TempDir:   "/tmp/repo-12345",
        ChartPath: "/tmp/repo-12345/charts/mychart",
    }
    
    // Access the cloned repository location
    fmt.Printf("Repository cloned to: %s\n", result.TempDir)
    fmt.Printf("Chart found at: %s\n", result.ChartPath)
    
    // Use the paths for further processing
    if err := processChart(result.ChartPath); err != nil {
        // Clean up temporary directory
        os.RemoveAll(result.TempDir)
    }
}
```

The CloneResult struct provides a standardized way to return both the temporary directory where a Git repository was cloned and the specific path to a chart within that repository, making it easier to manage cleanup and chart processing operations.
