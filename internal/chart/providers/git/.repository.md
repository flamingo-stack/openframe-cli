Handles git repository operations for cloning and managing chart repositories with optimized shallow cloning and automatic cleanup functionality.

## Key Components

- **Repository**: Main struct that wraps a command executor for git operations
- **NewRepository()**: Constructor function that creates a new Repository instance
- **CloneChartRepository()**: Clones a GitHub repository with depth 1 optimization and validates chart paths
- **Cleanup()**: Removes temporary directories with error handling

## Usage Example

```go
// Create a repository handler
executor := executor.NewCommandExecutor()
repo := git.NewRepository(executor)

// Configure repository details
config := &models.AppOfAppsConfig{
    GitHubRepo:   "https://github.com/user/charts",
    GitHubBranch: "main",
    ChartPath:    "charts/myapp",
}

// Clone repository
result, err := repo.CloneChartRepository(context.Background(), config)
if err != nil {
    log.Fatal(err)
}
defer repo.Cleanup(result.TempDir)

// Use the cloned chart
fmt.Printf("Chart available at: %s\n", result.ChartPath)
```

The implementation uses shallow cloning (`--depth 1`) for performance and includes comprehensive error handling for common scenarios like missing branches or invalid chart paths.
