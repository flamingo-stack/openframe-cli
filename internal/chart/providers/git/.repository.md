This file provides Git repository operations specifically for cloning and managing Helm chart repositories in the OpenFrame CLI tool.

## Key Components

- **Repository**: Main struct that handles Git operations using a command executor
- **NewRepository**: Constructor function that creates a new Repository instance
- **CloneChartRepository**: Clones a GitHub repository with optimizations (depth 1, single branch) and returns chart path information
- **Cleanup**: Utility method for removing temporary directories created during clone operations
- **CloneResult**: Return type containing temporary directory path and chart path (referenced but not defined in this file)

## Usage Example

```go
// Create a repository handler
exec := executor.NewCommandExecutor()
repo := git.NewRepository(exec)

// Configure the app of apps
config := &models.AppOfAppsConfig{
    GitHubRepo:   "https://github.com/user/charts",
    GitHubBranch: "main", 
    ChartPath:    "charts/myapp",
}

// Clone the repository
ctx := context.Background()
result, err := repo.CloneChartRepository(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Use the chart
fmt.Printf("Chart available at: %s\n", result.ChartPath)

// Clean up when done
defer repo.Cleanup(result.TempDir)
```

The implementation includes error handling for common Git issues like missing branches and non-existent chart paths.
