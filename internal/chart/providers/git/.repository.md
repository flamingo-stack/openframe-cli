Provides git repository operations for cloning and managing Helm chart repositories, with optimized cloning and automatic cleanup functionality.

## Key Components

- **Repository**: Main struct that handles git operations using a command executor
- **NewRepository()**: Constructor function that creates a new repository handler
- **CloneChartRepository()**: Clones a GitHub repository to a temporary directory with shallow clone optimizations
- **Cleanup()**: Removes temporary directories created during clone operations
- **CloneResult**: Return type containing temporary directory and chart path information

## Usage Example

```go
// Create a repository handler
executor := executor.NewCommandExecutor()
repo := git.NewRepository(executor)

// Configure the app-of-apps
config := &models.AppOfAppsConfig{
    GitHubRepo:   "https://github.com/user/helm-charts",
    GitHubBranch: "main",
    ChartPath:    "charts/my-app",
}

// Clone the repository
result, err := repo.CloneChartRepository(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Use the cloned chart
fmt.Printf("Chart available at: %s\n", result.ChartPath)

// Clean up when done
defer repo.Cleanup(result.TempDir)
```

The implementation uses shallow cloning (`--depth 1`) and other git optimizations for faster performance, includes error handling for common issues like missing branches, and provides automatic cleanup of temporary resources.
