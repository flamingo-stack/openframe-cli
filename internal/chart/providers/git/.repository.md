This file provides Git repository operations for the OpenFrame CLI, specifically handling the cloning and cleanup of Helm chart repositories from GitHub.

## Key Components

- **Repository**: Main struct that wraps a command executor for Git operations
- **NewRepository()**: Constructor function that creates a new Repository instance
- **CloneChartRepository()**: Clones a GitHub repository to a temporary directory with optimizations for speed
- **Cleanup()**: Removes temporary directories created during clone operations
- **CloneResult**: Return type containing paths to the cloned repository and chart location

## Usage Example

```go
// Create a repository handler
executor := executor.NewCommandExecutor()
repo := git.NewRepository(executor)

// Configure the repository to clone
config := &models.AppOfAppsConfig{
    GitHubRepo:   "https://github.com/user/charts",
    GitHubBranch: "main", 
    ChartPath:    "charts/myapp",
}

// Clone the repository
ctx := context.Background()
result, err := repo.CloneChartRepository(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Use the cloned chart
fmt.Printf("Chart available at: %s\n", result.ChartPath)

// Always cleanup when done
defer repo.Cleanup(result.TempDir)
```

The implementation includes error handling for common scenarios like missing branches and invalid chart paths, and uses Git optimizations (`--depth 1`, `--single-branch`) for faster cloning of chart repositories.
