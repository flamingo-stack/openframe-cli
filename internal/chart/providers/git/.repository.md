Provides git repository operations for cloning and managing Helm chart repositories from GitHub with optimized shallow clones and cleanup capabilities.

## Key Components

- **Repository**: Main struct that handles git operations using a command executor
- **NewRepository**: Constructor function that creates a new repository handler
- **CloneChartRepository**: Clones a GitHub repository with depth 1 optimization and validates chart paths
- **Cleanup**: Removes temporary directories safely with error logging
- **CloneResult**: Return type containing temporary directory and chart path information

## Usage Example

```go
// Create a repository handler with a command executor
executor := executor.NewCommandExecutor()
repo := git.NewRepository(executor)

// Configure the app of apps
config := &models.AppOfAppsConfig{
    GitHubRepo:   "https://github.com/user/charts",
    GitHubBranch: "main",
    ChartPath:    "helm-charts/myapp",
}

// Clone the repository
ctx := context.Background()
result, err := repo.CloneChartRepository(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Use the cloned chart
fmt.Printf("Chart available at: %s\n", result.ChartPath)

// Clean up when done
defer repo.Cleanup(result.TempDir)
```

The repository handler performs shallow clones for performance, validates branch existence, and provides detailed error messages for common issues like missing branches or invalid chart paths.
