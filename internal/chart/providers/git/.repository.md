Provides Git repository operations for cloning and managing Helm chart repositories with optimized shallow cloning and error handling.

## Key Components

- **Repository**: Main struct handling git operations with command executor dependency
- **NewRepository()**: Constructor function for creating repository handlers
- **CloneChartRepository()**: Clones GitHub repositories to temporary directories with depth-1 optimization
- **Cleanup()**: Safely removes temporary directories with error logging
- **CloneResult**: Return type containing temporary directory and chart paths

## Usage Example

```go
// Create repository handler with command executor
exec := executor.NewCommandExecutor()
repo := git.NewRepository(exec)

// Clone chart repository
config := &models.AppOfAppsConfig{
    GitHubRepo:   "https://github.com/user/charts.git",
    GitHubBranch: "main",
    ChartPath:    "charts/myapp",
}

result, err := repo.CloneChartRepository(context.Background(), config)
if err != nil {
    log.Fatal(err)
}

// Use the cloned chart
fmt.Printf("Chart available at: %s\n", result.ChartPath)

// Cleanup when done
defer repo.Cleanup(result.TempDir)
```

The implementation includes optimizations like shallow cloning (`--depth 1`) and provides detailed error messages for common issues like missing branches or invalid chart paths.
