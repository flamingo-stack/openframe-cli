This file contains unit tests for the `CloneResult` struct in the git package, validating both default initialization and value assignment behaviors.

## Key Components

- **TestCloneResult_DefaultValues**: Tests that a newly initialized `CloneResult` struct has empty string values for all fields
- **TestCloneResult_WithValues**: Tests that a `CloneResult` struct properly stores assigned values for `TempDir` and `ChartPath` fields

## Usage Example

```go
// Run tests to validate CloneResult behavior
go test ./git

// The tests verify this struct usage:
result := &CloneResult{}
// result.TempDir and result.ChartPath are empty strings

result = &CloneResult{
    TempDir:   "/tmp/clone-12345",
    ChartPath: "/tmp/clone-12345/manifests/app-of-apps",
}
// Fields contain the assigned values
```

These tests ensure the `CloneResult` struct behaves correctly when used to store temporary directory paths and chart locations from git repository cloning operations.
