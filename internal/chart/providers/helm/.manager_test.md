Test file providing comprehensive unit tests for the Helm manager component, including mock command executor functionality for testing Helm operations without executing actual commands.

## Key Components

- **MockExecutor**: Test double implementing `CommandExecutor` interface with configurable responses
- **TestHelmManager_IsHelmInstalled**: Tests Helm installation detection
- **TestHelmManager_IsChartInstalled**: Tests chart installation status checking
- **TestHelmManager_InstallArgoCD**: Tests ArgoCD chart installation with various configurations
- **TestHelmManager_GetChartStatus**: Tests chart status retrieval

## Usage Example

```go
// Create mock executor for testing
mockExec := NewMockExecutor()

// Configure expected command result
mockExec.SetResult("helm version --short", &executor.CommandResult{
    ExitCode: 0,
    Stdout:   "v3.12.0+g4f11b4a",
})

// Test the manager
manager := NewHelmManager(mockExec)
err := manager.IsHelmInstalled(context.Background())

// Configure error scenarios
mockExec.SetError("helm list -q -n argocd -f argocd", errors.New("command failed"))

// Verify executed commands
commands := mockExec.GetCommands()
```

The mock executor supports both success and failure scenarios, command tracking, and flexible response configuration for thorough testing of Helm operations.
