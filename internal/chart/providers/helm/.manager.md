This file implements a Helm client wrapper that provides high-level operations for managing Helm charts, with specific support for ArgoCD and app-of-apps installations.

## Key Components

- **HelmManager**: Main struct that wraps a command executor for Helm operations
- **NewHelmManager()**: Constructor function that creates a new HelmManager instance
- **IsHelmInstalled()**: Validates Helm CLI availability on the system
- **IsChartInstalled()**: Checks if a specific Helm release exists in a namespace
- **InstallArgoCD()**: Installs ArgoCD using Helm with predefined configuration
- **InstallArgoCDWithProgress()**: ArgoCD installation with progress indicators and verbose output
- **InstallAppOfAppsFromLocal()**: Installs app-of-apps chart from local filesystem with optional TLS certificates
- **GetChartStatus()**: Retrieves status information for deployed Helm releases

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/config"
)

// Initialize Helm manager
exec := executor.NewCommandExecutor()
helmManager := NewHelmManager(exec)

// Check if Helm is available
ctx := context.Background()
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    log.Fatal("Helm not available:", err)
}

// Install ArgoCD with progress indicators
installConfig := config.ChartInstallConfig{
    Silent:         false,
    NonInteractive: false,
    DryRun:         false,
    Verbose:        true,
}

err := helmManager.InstallArgoCDWithProgress(ctx, installConfig)
if err != nil {
    log.Fatal("Failed to install ArgoCD:", err)
}
```
