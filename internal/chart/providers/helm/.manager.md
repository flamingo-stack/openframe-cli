HelmManager provides a comprehensive interface for managing Helm chart operations, specifically focused on ArgoCD installations and app-of-apps deployments. It wraps the Helm CLI with error handling, progress indicators, and context cancellation support.

## Key Components

- **HelmManager**: Main struct that encapsulates Helm operations using a command executor
- **NewHelmManager()**: Factory function for creating new manager instances
- **IsHelmInstalled()**: Validates Helm CLI availability
- **IsChartInstalled()**: Checks if a specific chart release exists in a namespace
- **InstallArgoCD()**: Basic ArgoCD installation without progress indicators
- **InstallArgoCDWithProgress()**: Enhanced ArgoCD installation with spinner and verbose output
- **InstallAppOfAppsFromLocal()**: Deploys app-of-apps charts from local filesystem with TLS certificate support
- **GetChartStatus()**: Retrieves deployment status information for chart releases

## Usage Example

```go
// Create a new Helm manager with command executor
exec := executor.NewRealCommandExecutor()
helmManager := NewHelmManager(exec)

// Check if Helm is available
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    return fmt.Errorf("helm not available: %w", err)
}

// Install ArgoCD with progress indicators
config := config.ChartInstallConfig{
    Silent:         false,
    NonInteractive: false,
    Verbose:        true,
    DryRun:        false,
}

if err := helmManager.InstallArgoCDWithProgress(ctx, config); err != nil {
    return fmt.Errorf("argocd installation failed: %w", err)
}

// Install app-of-apps from local chart
appConfig := config
appConfig.AppOfApps = &config.AppOfAppsConfig{
    ChartPath:  "./charts/app-of-apps",
    Namespace:  "argocd",
    Timeout:    "10m",
    ValuesFile: "values.yaml",
}

err := helmManager.InstallAppOfAppsFromLocal(ctx, appConfig, "cert.pem", "key.pem")
```
