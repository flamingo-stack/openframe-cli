A comprehensive Helm operations manager that provides high-level abstractions for Helm CLI operations, focusing on ArgoCD installation and application management with support for progress indicators and configuration options.

## Key Components

- **HelmManager** - Main struct that wraps a command executor for Helm operations
- **NewHelmManager** - Constructor function that creates a new Helm manager instance
- **IsHelmInstalled** - Validates Helm CLI availability on the system
- **IsChartInstalled** - Checks if a specific Helm release exists in a namespace
- **InstallArgoCD** - Basic ArgoCD installation without progress indicators
- **InstallArgoCDWithProgress** - Enhanced ArgoCD installation with progress feedback and UI controls
- **InstallAppOfAppsFromLocal** - Installs app-of-apps pattern charts from local file paths
- **GetChartStatus** - Retrieves deployment status information for Helm releases

## Usage Example

```go
// Initialize with command executor
executor := &executor.RealCommandExecutor{}
helmManager := helm.NewHelmManager(executor)

// Check if Helm is available
ctx := context.Background()
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    log.Fatal("Helm not available:", err)
}

// Install ArgoCD with progress indicators
config := config.ChartInstallConfig{
    DryRun:         false,
    Silent:         false,
    NonInteractive: false,
    Verbose:        true,
}

err := helmManager.InstallArgoCDWithProgress(ctx, config)
if err != nil {
    log.Fatal("Failed to install ArgoCD:", err)
}

// Check if a chart is installed
installed, err := helmManager.IsChartInstalled(ctx, "argo-cd", "argocd")
if err != nil {
    log.Fatal("Error checking installation:", err)
}
fmt.Printf("ArgoCD installed: %v\n", installed)
```

The manager handles temporary file creation for Helm values, repository management, and provides both silent and interactive installation modes with proper error handling and context cancellation support.
