A Kubernetes-native Helm manager that handles ArgoCD installation with robust client initialization, TLS configuration, and cross-platform support for Windows/WSL environments.

## Key Components

**Core Structure:**
- `HelmManager` - Main struct with Kubernetes clients (core, dynamic, CRD) and command executor
- `NewHelmManager()` - Constructor with automatic TLS bypass for local k3d clusters and graceful client degradation
- `getHelmEnv()` - Cross-platform environment setup for Helm directories

**Installation Methods:**
- `InstallArgoCD()` - Basic ArgoCD installation with repository management and values file handling
- `InstallArgoCDWithProgress()` - Enhanced installation with progress indicators and cluster connectivity retries
- `IsChartInstalled()` - Release existence verification
- `IsHelmInstalled()` - Helm availability check

**Platform Support:**
- Windows path conversion for WSL integration
- Context-aware kubectl operations
- Graceful fallback to kubectl when native clients fail

## Usage Example

```go
// Initialize with Kubernetes config
config := &rest.Config{Host: "https://localhost:6443"}
executor := executor.NewCommandExecutor()
helmMgr, err := NewHelmManager(executor, config, true)
if err != nil {
    return err
}

// Check if ArgoCD is already installed
installed, err := helmMgr.IsChartInstalled(ctx, "argo-cd", "argocd")
if err != nil {
    return err
}

if !installed {
    // Install ArgoCD with progress tracking
    installConfig := config.ChartInstallConfig{
        ClusterName: "my-cluster",
        Verbose:     true,
        Silent:      false,
    }
    
    err = helmMgr.InstallArgoCDWithProgress(ctx, installConfig)
    if err != nil {
        return fmt.Errorf("ArgoCD installation failed: %w", err)
    }
}
```

The manager automatically handles TLS configuration for local clusters, provides multiple Kubernetes client interfaces, and includes comprehensive error handling with diagnostic information for troubleshooting installation issues.
