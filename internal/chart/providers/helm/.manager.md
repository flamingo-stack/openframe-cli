A Helm operations manager that provides high-level abstractions for installing and managing Helm charts, with specialized support for ArgoCD and app-of-apps patterns. It wraps Helm CLI commands with progress indicators, error handling, and configuration management.

## Key Components

- **HelmManager**: Main struct that wraps a command executor for Helm operations
- **NewHelmManager()**: Factory function for creating manager instances
- **IsHelmInstalled()**: Validates Helm CLI availability
- **IsChartInstalled()**: Checks if a specific chart release exists
- **InstallArgoCD()**: Basic ArgoCD installation with standard values
- **InstallArgoCDWithProgress()**: ArgoCD installation with UI progress indicators
- **InstallAppOfAppsFromLocal()**: Installs app-of-apps chart from local filesystem
- **GetChartStatus()**: Retrieves status information for deployed charts

## Usage Example

```go
// Initialize the manager
executor := executor.NewRealCommandExecutor()
helmManager := NewHelmManager(executor)

// Check if Helm is available
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    log.Fatal("Helm not found:", err)
}

// Install ArgoCD with progress
config := config.ChartInstallConfig{
    DryRun:         false,
    Silent:         false,
    NonInteractive: false,
    Verbose:        true,
}

err := helmManager.InstallArgoCDWithProgress(ctx, config)
if err != nil {
    log.Fatal("ArgoCD installation failed:", err)
}

// Check if a chart is installed
installed, err := helmManager.IsChartInstalled(ctx, "argo-cd", "argocd")
if err != nil {
    log.Fatal("Failed to check chart status:", err)
}
```
