Manages Helm chart operations including installation, status checking, and ArgoCD deployment. Provides a comprehensive interface for executing Helm commands with progress tracking and error handling.

## Key Components

- **HelmManager**: Main struct that wraps a command executor for Helm operations
- **NewHelmManager()**: Constructor that creates a new Helm manager instance
- **IsHelmInstalled()**: Validates Helm CLI availability in the system
- **IsChartInstalled()**: Checks if a specific Helm release exists in a namespace
- **InstallArgoCD()**: Basic ArgoCD installation without progress indicators
- **InstallArgoCDWithProgress()**: ArgoCD installation with progress spinners and verbose output
- **InstallAppOfAppsFromLocal()**: Installs app-of-apps pattern from local chart path with TLS certificate support
- **GetChartStatus()**: Retrieves status information for deployed charts

## Usage Example

```go
// Create Helm manager
executor := executor.NewCommandExecutor()
helmManager := helm.NewHelmManager(executor)

// Check if Helm is available
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    log.Fatal("Helm not found:", err)
}

// Install ArgoCD with progress tracking
config := config.ChartInstallConfig{
    Silent:         false,
    NonInteractive: false,
    DryRun:        false,
    Verbose:       true,
}

if err := helmManager.InstallArgoCDWithProgress(ctx, config); err != nil {
    log.Fatal("ArgoCD installation failed:", err)
}

// Check if a chart is installed
installed, err := helmManager.IsChartInstalled(ctx, "argo-cd", "argocd")
if err != nil {
    log.Fatal("Failed to check installation:", err)
}
```
