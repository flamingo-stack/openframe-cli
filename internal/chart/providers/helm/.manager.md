A Helm operations manager that provides a high-level interface for executing Helm commands, specifically focused on ArgoCD and app-of-apps chart installations with progress tracking and error handling.

## Key Components

- **HelmManager**: Main struct that wraps a command executor for Helm operations
- **NewHelmManager**: Constructor function that creates a new HelmManager instance
- **IsHelmInstalled**: Validates Helm availability on the system
- **IsChartInstalled**: Checks if a specific Helm release exists in a namespace
- **InstallArgoCD**: Basic ArgoCD installation without progress indicators
- **InstallArgoCDWithProgress**: ArgoCD installation with spinner progress and verbose logging
- **InstallAppOfAppsFromLocal**: Installs app-of-apps chart from local filesystem with TLS certificate support
- **GetChartStatus**: Retrieves status information for deployed Helm releases

## Usage Example

```go
// Create a new Helm manager
executor := executor.NewCommandExecutor()
helmManager := NewHelmManager(executor)

// Check if Helm is available
ctx := context.Background()
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    log.Fatal("Helm not available:", err)
}

// Install ArgoCD with progress tracking
config := config.ChartInstallConfig{
    DryRun:         false,
    Silent:         false,
    NonInteractive: false,
    Verbose:        true,
}

err := helmManager.InstallArgoCDWithProgress(ctx, config)
if err != nil {
    log.Fatal("ArgoCD installation failed:", err)
}

// Check if a chart is installed
installed, err := helmManager.IsChartInstalled(ctx, "argo-cd", "argocd")
if err != nil {
    log.Fatal("Failed to check installation:", err)
}
```
