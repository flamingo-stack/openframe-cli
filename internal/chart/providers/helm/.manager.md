Manages Helm operations for the OpenFrame CLI, providing a wrapper around Helm commands with Kubernetes integration. It handles ArgoCD installation with CRD management, cluster connectivity verification, and cross-platform compatibility (Windows/WSL support).

## Key Components

**Core Type:**
- `HelmManager` - Main manager struct with Kubernetes clients and command executor

**Client Integrations:**
- `kubeConfig` - Kubernetes cluster connection configuration
- `dynamicClient` - Dynamic client for programmatic resource management
- `kubeClient` - Typed client for deployment verification
- `crdClient` - Custom Resource Definition client
- `executor` - Command executor for Helm CLI operations

**Primary Methods:**
- `NewHelmManager()` - Creates manager with TLS bypass for local k3d clusters
- `IsHelmInstalled()` - Verifies Helm CLI availability
- `IsChartInstalled()` - Checks if Helm release exists
- `InstallArgoCD()` - Installs ArgoCD with exact configuration
- `InstallArgoCDWithProgress()` - ArgoCD installation with progress indicators
- `getHelmEnv()` - Sets up writable directories for CI environments

## Usage Example

```go
// Create Helm manager with Kubernetes config
helmManager, err := helm.NewHelmManager(executor, kubeConfig, true)
if err != nil {
    return fmt.Errorf("failed to create helm manager: %w", err)
}

// Check if Helm is available
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    return fmt.Errorf("helm not available: %w", err)
}

// Install ArgoCD with configuration
config := config.ChartInstallConfig{
    ClusterName:    "my-cluster",
    DryRun:        false,
    SkipCRDs:      false,
    Verbose:       true,
}

err = helmManager.InstallArgoCDWithProgress(ctx, config)
if err != nil {
    return fmt.Errorf("argocd installation failed: %w", err)
}
```

The manager provides robust error handling, cross-platform path conversion for Windows/WSL environments, and intelligent fallback mechanisms when Kubernetes clients are unavailable.
