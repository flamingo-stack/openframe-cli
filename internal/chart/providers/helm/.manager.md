This file provides a Helm client wrapper for managing chart installations and operations within the OpenFrame CLI tool.

## Key Components

- **HelmManager**: Main struct that wraps a command executor for running Helm commands
- **NewHelmManager()**: Factory function to create a new HelmManager instance
- **IsHelmInstalled()**: Verifies Helm availability on the system
- **IsChartInstalled()**: Checks if a specific chart release exists in a namespace
- **InstallArgoCD()**: Basic ArgoCD installation without progress indicators
- **InstallArgoCDWithProgress()**: ArgoCD installation with progress spinners and verbose output
- **InstallAppOfAppsFromLocal()**: Installs app-of-apps chart from local filesystem with optional TLS certificates
- **GetChartStatus()**: Retrieves the status of a deployed chart

## Usage Example

```go
// Create a new Helm manager
executor := executor.NewCommandExecutor()
helmManager := NewHelmManager(executor)

// Check if Helm is available
ctx := context.Background()
if err := helmManager.IsHelmInstalled(ctx); err != nil {
    log.Fatal("Helm not available:", err)
}

// Install ArgoCD with progress indicators
config := config.ChartInstallConfig{
    DryRun:         false,
    Silent:         false,
    NonInteractive: false,
    Verbose:        true,
}
if err := helmManager.InstallArgoCDWithProgress(ctx, config); err != nil {
    log.Fatal("Failed to install ArgoCD:", err)
}

// Check if a chart is installed
installed, err := helmManager.IsChartInstalled(ctx, "argo-cd", "argocd")
if err != nil {
    log.Fatal("Error checking chart status:", err)
}
```
