This file contains unit tests for the `HelmManager.InstallAppOfAppsFromLocal` method, which installs Helm charts for app-of-apps configurations from local file paths.

## Key Components

- **TestHelmManager_InstallAppOfAppsFromLocal**: Main test function that validates various scenarios for local app-of-apps installation
- **Test cases**: Cover nil configuration, empty chart paths, successful installations, and dry-run mode
- **MockExecutor**: Used to simulate helm command execution without actually running helm commands
- **Validation scenarios**: Tests error handling for invalid configurations and success cases with proper helm command construction

## Usage Example

```go
func TestLocalInstall(t *testing.T) {
    // Create a mock executor
    mockExec := NewMockExecutor()
    
    // Set up expected helm command result
    command := "helm upgrade --install app-of-apps /path/to/chart --namespace argocd"
    result := &executor.CommandResult{
        ExitCode: 0,
        Stdout:   "Release installed successfully",
    }
    mockExec.SetResult(command, result)
    
    // Create helm manager with mock
    manager := NewHelmManager(mockExec)
    
    // Test the installation
    config := config.ChartInstallConfig{
        AppOfApps: &models.AppOfAppsConfig{
            ChartPath: "/path/to/chart",
            Namespace: "argocd",
        },
    }
    
    err := manager.InstallAppOfAppsFromLocal(context.Background(), config, "cert.pem", "key.pem")
    assert.NoError(t, err)
}
```
