Test suite for the Helm manager's local app-of-apps installation functionality. Validates installation scenarios including configuration validation, successful deployments, and dry-run operations.

## Key Components

- **TestHelmManager_InstallAppOfAppsFromLocal**: Main test function that validates the `InstallAppOfAppsFromLocal` method
- **MockExecutor**: Test utility for simulating command execution without actual Helm operations
- **Test scenarios**: Covers nil configuration, empty chart path, successful installation, and dry-run mode

## Usage Example

```go
// Run tests for local app-of-apps installation
go test -v ./internal/chart/helm -run TestHelmManager_InstallAppOfAppsFromLocal

// Test structure demonstrates expected behavior
mockExec := NewMockExecutor()
manager := NewHelmManager(mockExec)

config := config.ChartInstallConfig{
    AppOfApps: &models.AppOfAppsConfig{
        ChartPath:  "/tmp/chart/manifests/app-of-apps",
        ValuesFile: "/path/to/values.yaml",
        Namespace:  "argocd",
        Timeout:    "60m",
    },
}

err := manager.InstallAppOfAppsFromLocal(context.Background(), config, certFile, keyFile)
```

The tests verify error handling for invalid configurations and successful execution of Helm upgrade/install commands with proper TLS certificate handling and namespace management.
