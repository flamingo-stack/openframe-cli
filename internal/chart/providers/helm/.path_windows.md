Windows-specific path expansion utility that converts Windows 8.3 short filenames to their full long path names, essential for WSL compatibility where short filenames are not recognized.

## Key Components

- **expandShortPath()** - Main function that expands Windows short paths to full paths
- **kernel32 syscalls** - Direct Windows API calls to `GetLongPathNameW`
- **UTF-16 conversion** - Handles Windows Unicode string format requirements

## Usage Example

```go
// Expand a Windows short path
shortPath := "C:\\Users\\RUNNER~1\\AppData\\Local"
longPath, err := expandShortPath(shortPath)
if err != nil {
    return err
}
// Result: "C:\\Users\\runneradmin\\AppData\\Local"

// Handle empty paths safely
emptyPath := ""
result, _ := expandShortPath(emptyPath)
// Returns: "" (no error)

// Fallback behavior for invalid paths
invalidPath := "C:\\NonExistent\\PATH~1"
fallback, _ := expandShortPath(invalidPath)
// Returns: original path if expansion fails
```

This utility is critical for Helm operations on Windows when working with WSL environments, as WSL cannot interpret Windows 8.3 short filename format. The function gracefully handles errors by returning the original path as a fallback, ensuring Helm operations continue even if path expansion fails.
