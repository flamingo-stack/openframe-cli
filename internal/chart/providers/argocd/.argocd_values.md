Provides ArgoCD Helm chart configuration values as a YAML string for deployment setup.

## Key Components

- **GetArgoCDValues()** - Returns a pre-configured YAML string containing ArgoCD Helm chart values with custom resource health checks and repository server timeout settings

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/your-org/project/argocd"
)

func main() {
    // Get ArgoCD Helm values for deployment
    values := argocd.GetArgoCDValues()
    
    // Use with Helm deployment
    fmt.Println("ArgoCD Values:")
    fmt.Println(values)
    
    // Typically used with Helm Go SDK or kubectl apply
    // helm.Install("argocd", "argo/argo-cd", values)
}
```

The configuration includes:
- Custom health check logic for ArgoCD Application resources
- Repository server timeout set to 180 seconds for long-running operations
- Placeholder for image pull secrets (commented out)
- Full name override set to "argocd"

This is commonly used in GitOps deployment pipelines where ArgoCD needs to be installed with specific operational parameters.
