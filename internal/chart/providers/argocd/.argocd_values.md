This file provides default Helm chart values for deploying ArgoCD with optimized configurations for resource health checks and execution timeouts.

## Key Components

- **`GetArgoCDValues()`** - Returns a pre-configured YAML string containing ArgoCD Helm chart values with custom health check logic and timeout settings

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/your-org/your-project/argocd"
)

func main() {
    // Get ArgoCD Helm values for deployment
    values := argocd.GetArgoCDValues()
    
    // Use with Helm deployment
    fmt.Println(values)
    
    // Or save to file for helm install
    // helm install argocd argo/argo-cd -f values.yaml
}
```

The returned values include:
- Custom health check configuration for ArgoCD Applications
- Extended execution timeout (180s) for repository operations
- Fullname override to simplify resource naming
- Commented image pull secrets template for private registries
