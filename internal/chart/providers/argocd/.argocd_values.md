Provides ArgoCD Helm chart configuration values optimized for lightweight installations in CI/K3d environments with resource constraints and disabled non-essential components.

## Key Components

- **GetArgoCDValues()** - Returns a complete YAML configuration string for ArgoCD Helm deployment
- **Resource Optimization** - Configured CPU/memory limits suitable for development and CI environments
- **Component Management** - Selectively disables Dex, notifications, and ApplicationSet for minimal footprint
- **Health Checks** - Custom Application health check configuration using Lua scripting
- **Timeout Configuration** - Extended timeouts for reconciliation and repository operations

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/flamingo-stack/openframe/pkg/argocd"
)

func deployArgoCD() {
    // Get the ArgoCD Helm values
    values := argocd.GetArgoCDValues()
    
    // Use with Helm deployment
    fmt.Printf("ArgoCD Configuration:\n%s", values)
    
    // The values can be written to a file or passed directly to Helm
    // helm install argocd argo/argo-cd --values <(echo "$values")
}
```

The configuration includes resource constraints optimized for K3d clusters, custom health checks for ArgoCD Applications, and disabled components (Dex, notifications, ApplicationSet) to reduce resource usage in development environments.
