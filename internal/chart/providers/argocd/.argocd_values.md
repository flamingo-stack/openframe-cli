This file provides pre-configured Helm chart values for deploying ArgoCD with optimized settings for application health monitoring and extended timeout configurations.

## Key Components

- **GetArgoCDValues()**: Returns a YAML string containing ArgoCD Helm chart values with custom health check configurations and repository server settings

## Configuration Features

The returned YAML includes:
- **Custom Health Checks**: Lua script for monitoring ArgoCD Application resources with proper status and message handling
- **Extended Timeouts**: 180-second execution timeout for repository operations
- **Optional Image Pull Secrets**: Commented configuration for private registry authentication
- **Fullname Override**: Sets consistent naming for ArgoCD resources

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/openframe/argocd"
)

func main() {
    // Get ArgoCD values for Helm deployment
    values := argocd.GetArgoCDValues()
    
    // Write to values file
    err := os.WriteFile("argocd-values.yaml", []byte(values), 0644)
    if err != nil {
        log.Fatal(err)
    }
    
    // Or use directly with Helm client
    fmt.Println(values)
}
```

The configuration is optimized for production deployments with robust health monitoring and sufficient timeouts for complex repository operations.
