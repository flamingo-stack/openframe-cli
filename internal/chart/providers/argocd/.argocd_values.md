Returns a pre-configured YAML string containing ArgoCD Helm chart values with customized health checks and timeout settings.

## Key Components

- **GetArgoCDValues()** - Function that returns ArgoCD Helm chart configuration as a YAML string

## Usage Example

```go
package main

import (
    "fmt"
    "github.com/yourorg/argocd"
)

func main() {
    // Get the ArgoCD configuration values
    values := argocd.GetArgoCDValues()
    
    // Use with Helm deployment
    fmt.Println(values)
    
    // Or write to file for Helm installation
    // helm install argocd argo/argo-cd -f values.yaml
}
```

The returned YAML includes:
- Custom health check configuration for ArgoCD Applications
- Repository server timeout settings (180s)
- Fullname override set to "argocd" 
- Commented image pull secrets configuration for easy customization

This configuration is typically used when deploying ArgoCD via Helm charts to ensure proper application health monitoring and extended timeout handling.
