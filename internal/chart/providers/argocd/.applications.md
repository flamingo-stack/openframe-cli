This file provides ArgoCD application management functionality for monitoring and parsing ArgoCD application states within a Kubernetes cluster.

## Key Components

**Types:**
- `Manager` - Main ArgoCD operations handler with command executor dependency
- `Application` - Represents ArgoCD application with Name, Health, and Sync status

**Methods:**
- `NewManager(executor.CommandExecutor) *Manager` - Creates new ArgoCD manager instance
- `getTotalExpectedApplications(ctx, config) int` - Estimates total applications using multiple detection methods
- `parseApplications(ctx, verbose) ([]Application, error)` - Retrieves current application states via kubectl

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
)

// Create manager
exec := executor.NewCommandExecutor()
manager := NewManager(exec)

// Get current applications
ctx := context.Background()
apps, err := manager.parseApplications(ctx, true)
if err != nil {
    log.Fatal(err)
}

// Process applications
for _, app := range apps {
    fmt.Printf("App: %s, Health: %s, Sync: %s\n", 
        app.Name, app.Health, app.Sync)
}

// Estimate total expected applications
config := config.ChartInstallConfig{Verbose: true}
total := manager.getTotalExpectedApplications(ctx, config)
fmt.Printf("Expected total: %d applications\n", total)
```
