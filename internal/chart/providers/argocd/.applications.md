This Go file manages ArgoCD application operations within the OpenFrame MSP platform, providing high-level abstractions for interacting with ArgoCD applications in Kubernetes clusters.

## Key Components

- **Manager struct** - Main manager for ArgoCD operations with Kubernetes client support
- **Application struct** - Represents ArgoCD application status with health, sync, and repository information
- **Constructor functions** - `NewManager()`, `NewManagerWithCluster()`, `NewManagerWithConfig()` for different initialization patterns
- **Client initialization** - Lazy loading of Kubernetes, API extensions, and ArgoCD native clients
- **Platform-specific handling** - Windows/WSL2 compatibility with kubectl fallback mechanisms
- **Application counting** - Methods to determine expected application counts for progress tracking

## Usage Example

```go
// Create manager with command executor
exec := executor.NewCommandExecutor()
manager := argocd.NewManager(exec)

// Or create with specific cluster context
manager := argocd.NewManagerWithCluster(exec, "openframe")

// Or create with pre-configured Kubernetes client
config := &rest.Config{Host: "https://127.0.0.1:6443"}
manager, err := argocd.NewManagerWithConfig(exec, config)
if err != nil {
    log.Fatal(err)
}

// Set cluster context if needed
manager.SetClusterName("k3d-openframe")

// Get expected application count for progress tracking
ctx := context.Background()
chartConfig := config.ChartInstallConfig{ClusterName: "openframe", Verbose: true}
totalApps := manager.getTotalExpectedApplications(ctx, chartConfig)
```

The manager automatically handles TLS configuration for local k3d clusters and provides fallback mechanisms for Windows environments where native Kubernetes clients may have connectivity issues.
