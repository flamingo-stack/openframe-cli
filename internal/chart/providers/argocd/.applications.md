Handles ArgoCD application management and monitoring, providing methods to discover applications, track their status, and estimate deployment progress during OpenFrame installation.

## Key Components

- **Manager**: Main struct that wraps a command executor for ArgoCD operations
- **Application**: Represents an ArgoCD application with Name, Health, and Sync status fields
- **getTotalExpectedApplications()**: Attempts to determine the total number of applications that will be deployed using multiple discovery methods
- **parseApplications()**: Retrieves current ArgoCD application status via kubectl commands

## Usage Example

```go
// Create an ArgoCD manager
exec := executor.NewCommandExecutor()
manager := argocd.NewManager(exec)

// Get current applications and their status
ctx := context.Background()
apps, err := manager.parseApplications(ctx, true)
if err != nil {
    log.Fatal(err)
}

// Display application status
for _, app := range apps {
    fmt.Printf("App: %s, Health: %s, Sync: %s\n", 
        app.Name, app.Health, app.Sync)
}

// Estimate total expected applications for progress tracking
config := config.ChartInstallConfig{Verbose: true}
total := manager.getTotalExpectedApplications(ctx, config)
fmt.Printf("Expected total applications: %d\n", total)
```

The manager uses multiple fallback strategies to discover applications, including checking the app-of-apps status, querying the ArgoCD server directly, and analyzing Helm values to provide robust application tracking during OpenFrame deployments.
