ArgoCD manager that handles application monitoring, status tracking, and deployment counting for OpenFrame CLI operations.

## Key Components

- **Manager**: Main struct that manages ArgoCD operations using a command executor
- **Application**: Represents an ArgoCD application with name, health, and sync status
- **NewManager()**: Factory function to create a new ArgoCD manager instance
- **getTotalExpectedApplications()**: Estimates total application count using multiple detection methods
- **parseApplications()**: Retrieves and parses current ArgoCD application statuses

## Usage Example

```go
// Create ArgoCD manager
exec := executor.NewCommandExecutor()
manager := argocd.NewManager(exec)

// Get current applications and their status
apps, err := manager.parseApplications(ctx, true)
if err != nil {
    log.Fatal(err)
}

// Display application status
for _, app := range apps {
    fmt.Printf("App: %s, Health: %s, Sync: %s\n", 
        app.Name, app.Health, app.Sync)
}

// Estimate total expected applications
config := config.ChartInstallConfig{Verbose: true}
total := manager.getTotalExpectedApplications(ctx, config)
fmt.Printf("Expected applications: %d\n", total)
```

The manager uses multiple fallback strategies to detect applications through kubectl commands, helm values inspection, and ArgoCD server queries, making it resilient to various deployment states and configurations.
