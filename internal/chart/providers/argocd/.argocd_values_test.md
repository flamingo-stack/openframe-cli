This file contains unit tests for the `GetArgoCDValues()` function, verifying that it returns properly formatted ArgoCD configuration values in YAML format.

## Key Components

- **TestGetArgoCDValues()** - Main test function that validates the function returns non-empty YAML content with expected configuration keys
- **TestGetArgoCDValuesStructure()** - Additional test that verifies the YAML structure has sufficient content and includes Lua health check scripts

## Usage Example

```go
func TestExample(t *testing.T) {
    // Test the ArgoCD values function
    values := GetArgoCDValues()
    
    // Verify it returns YAML content
    if values == "" {
        t.Error("Expected non-empty values")
    }
    
    // Check for specific configuration
    if !strings.Contains(values, "fullnameOverride: argocd") {
        t.Error("Missing expected ArgoCD configuration")
    }
    
    // Verify Lua health check script exists
    if !strings.Contains(values, "if obj.status ~= nil then") {
        t.Error("Missing health check script")
    }
}
```

The tests ensure the configuration includes essential ArgoCD settings like fullname override, custom resource health checks, and Lua scripts for application status monitoring.
