This test file validates the `GetArgoCDValues()` function, ensuring it returns properly formatted Helm values YAML for ArgoCD configuration with health check scripts.

## Key Components

- **`TestGetArgoCDValues`** - Main test function that validates the returned YAML contains required configuration elements like fullname override, configs section, and health check definitions
- **`TestGetArgoCDValuesStructure`** - Structural validation test that checks the YAML has sufficient content length and includes Lua health check scripts

## Usage Example

```go
func TestCustomArgoCDValidation(t *testing.T) {
    values := GetArgoCDValues()
    
    // Verify custom health checks are present
    if !strings.Contains(values, "hs.status = \"Progressing\"") {
        t.Error("Missing expected health check status")
    }
    
    // Validate YAML structure
    lines := strings.Split(values, "\n")
    if len(lines) < 20 {
        t.Error("Insufficient YAML content")
    }
}
```

The tests ensure the ArgoCD values include essential configurations like custom resource health checks, proper YAML formatting, and adequate content depth for a functional ArgoCD deployment.
