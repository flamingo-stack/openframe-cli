This file implements ArgoCD application synchronization monitoring with graceful cancellation support and comprehensive progress tracking.

## Key Components

- **`WaitForApplications(ctx, config)`**: Main function that waits for all ArgoCD applications to reach "Healthy" and "Synced" status
- **Context handling**: Supports cancellation via context and direct interrupt signals (Ctrl+C)
- **Progress tracking**: Visual spinner with timer and optional verbose logging
- **Timeout management**: 60-minute timeout with early exit for short-deadline contexts
- **Bootstrap phase**: 30-second initial wait for application discovery
- **Debug diagnostics**: Detailed pod status reporting for stuck applications after 7 minutes

## Usage Example

```go
import (
    "context"
    "time"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/config"
)

// Create ArgoCD manager
manager := &Manager{
    executor: executor,
    // other fields...
}

// Configure wait behavior
cfg := config.ChartInstallConfig{
    Verbose: true,    // Enable detailed progress logging
    Silent:  false,   // Show spinner UI
    DryRun:  false,   // Actually wait for sync
}

// Wait for applications with timeout
ctx, cancel := context.WithTimeout(context.Background(), 45*time.Minute)
defer cancel()

err := manager.WaitForApplications(ctx, cfg)
if err != nil {
    // Handle timeout, cancellation, or other errors
    log.Fatal(err)
}
```
