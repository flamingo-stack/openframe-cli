This file contains unit tests for a Helm installer component that provides platform-specific installation instructions and capabilities.

## Key Components

- **TestNewHelmInstaller()** - Tests creation of new Helm installer instances
- **TestHelmInstaller_GetInstallHelp()** - Validates platform-specific installation help text for macOS, Linux, and Windows
- **TestHelmInstaller_Install()** - Tests basic installer structure without performing actual installation
- **TestInstallScript()** - Linux-specific test for script-based installation with error handling
- **containsSubstring()** - Helper function for substring matching in test assertions

## Usage Example

```go
func TestCustomHelmInstaller(t *testing.T) {
    // Create installer instance
    installer := NewHelmInstaller()
    if installer == nil {
        t.Fatal("Failed to create installer")
    }

    // Get platform-specific help
    help := installer.GetInstallHelp()
    if help == "" {
        t.Error("Help text should not be empty")
    }

    // Check for expected platform commands
    if runtime.GOOS == "darwin" {
        if !containsSubstring(help, "brew") {
            t.Error("Expected brew reference for macOS")
        }
    }
}
```

The tests validate that the installer provides appropriate guidance for different operating systems and handles installation errors gracefully without requiring actual Helm installation during testing.
