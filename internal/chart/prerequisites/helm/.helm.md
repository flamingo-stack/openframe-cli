Provides cross-platform Helm installation management with automatic detection and platform-specific installation methods. Supports macOS (Homebrew), various Linux distributions (APT, YUM, DNF, Pacman), and provides manual installation guidance for Windows.

## Key Components

- **HelmInstaller**: Main struct implementing the installer interface
- **IsInstalled()**: Checks if Helm is available and functional on the system
- **Install()**: Performs automatic installation based on detected OS and package manager
- **GetInstallHelp()**: Returns platform-specific manual installation instructions
- **commandExists()**: Utility function to verify if a command is available in PATH
- **Platform-specific installers**: Separate methods for macOS, Ubuntu, RedHat, Fedora, and Arch Linux

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := NewHelmInstaller()
    
    // Check if Helm is already installed
    if installer.IsInstalled() {
        fmt.Println("Helm is already installed")
        return
    }
    
    // Get manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
    } else {
        fmt.Println("Helm installed successfully")
    }
}
```

The installer automatically detects the operating system and available package managers, falling back to the official Helm installation script when native package managers are unavailable. Windows requires manual installation due to platform limitations.
