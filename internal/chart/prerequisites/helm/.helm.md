A cross-platform Helm installer that detects if Helm is installed and provides automatic installation capabilities for macOS and Linux systems.

## Key Components

- **HelmInstaller**: Main struct providing Helm installation functionality
- **NewHelmInstaller()**: Constructor function returning a new installer instance
- **IsInstalled()**: Checks if Helm is available on the system
- **GetInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Performs automatic Helm installation based on the operating system
- **commandExists()**: Utility function to check if a command is available in PATH

## Usage Example

```go
installer := helm.NewHelmInstaller()

// Check if Helm is already installed
if !installer.IsInstalled() {
    fmt.Println("Helm not found")
    
    // Get installation help for manual installation
    fmt.Println(installer.GetInstallHelp())
    
    // Or attempt automatic installation (macOS/Linux only)
    if err := installer.Install(); err != nil {
        fmt.Printf("Auto-install failed: %v\n", err)
        fmt.Println("Please install manually using the instructions above")
    }
} else {
    fmt.Println("Helm is installed and ready to use")
}
```

The installer supports automatic installation on macOS (via Homebrew) and various Linux distributions (Ubuntu/Debian, RHEL/CentOS, Fedora, Arch Linux), while providing manual installation guidance for Windows and unsupported platforms.
