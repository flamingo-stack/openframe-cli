This file implements a cross-platform Helm installer that can detect, validate, and automatically install Helm (the Kubernetes package manager) across different operating systems.

## Key Components

- **HelmInstaller**: Main struct providing installation methods
- **commandExists()**: Utility function to check if a command is available in PATH
- **isHelmInstalled()**: Validates Helm installation by running `helm version`
- **helmInstallHelp()**: Provides platform-specific manual installation instructions
- **Install()**: Orchestrates automatic installation based on the operating system
- **Platform-specific installers**: Separate methods for macOS, Linux distributions, and Windows
- **Command execution helpers**: `runCommand()` and `runShellCommand()` for silent execution

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := helm.NewHelmInstaller()
    
    // Check if Helm is already installed
    if installer.IsInstalled() {
        fmt.Println("Helm is already installed")
        return
    }
    
    // Get manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation (Linux/macOS only)
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
    } else {
        fmt.Println("Helm installed successfully!")
    }
}
```

The installer supports multiple Linux distributions (Ubuntu, Red Hat, Fedora, Arch) with distribution-specific package managers, falls back to Homebrew on macOS, and provides manual installation guidance for Windows. All command execution is silenced to prevent cluttering the output during installation.
