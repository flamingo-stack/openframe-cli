A Helm package manager installer that provides cross-platform installation capabilities with OS-specific methods and installation verification.

## Key Components

- **HelmInstaller**: Main struct providing Helm installation functionality
- **NewHelmInstaller()**: Factory function to create installer instance
- **IsInstalled()**: Checks if Helm is already installed on the system
- **GetInstallHelp()**: Returns OS-specific installation instructions
- **Install()**: Performs automatic Helm installation based on the operating system

## Usage Example

```go
// Create installer instance
installer := helm.NewHelmInstaller()

// Check if Helm is already installed
if !installer.IsInstalled() {
    fmt.Println("Helm not found")
    
    // Get installation help for current OS
    help := installer.GetInstallHelp()
    fmt.Println(help)
    
    // Attempt automatic installation
    err := installer.Install()
    if err != nil {
        fmt.Printf("Installation failed: %v\n", err)
        return
    }
    
    fmt.Println("Helm installed successfully")
} else {
    fmt.Println("Helm is already installed")
}
```

The installer supports automatic installation on macOS (via Homebrew) and various Linux distributions (Ubuntu/Debian, Red Hat/CentOS, Fedora, Arch Linux). Windows installation requires manual intervention with helpful guidance provided.
