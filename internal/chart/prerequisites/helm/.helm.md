A Helm installer utility that detects, validates, and automatically installs the Helm package manager across different operating systems with platform-specific installation methods.

## Key Components

- **HelmInstaller**: Main struct providing installation management functionality
- **commandExists()**: Checks if a command is available in the system PATH  
- **isHelmInstalled()**: Verifies Helm installation by running `helm version`
- **helmInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automatically installs Helm using the appropriate method for the OS
- **Platform-specific installers**: Separate methods for macOS, Linux distributions, and Windows

## Usage Example

```go
installer := helm.NewHelmInstaller()

// Check if Helm is already installed
if !installer.IsInstalled() {
    fmt.Println("Helm not found")
    
    // Get installation help for manual installation
    fmt.Println(installer.GetInstallHelp())
    
    // Or attempt automatic installation
    if err := installer.Install(); err != nil {
        fmt.Printf("Auto-install failed: %v\n", err)
        fmt.Println("Please install manually using the instructions above")
    } else {
        fmt.Println("Helm installed successfully")
    }
} else {
    fmt.Println("Helm is already installed")
}
```

The installer supports automatic installation on macOS (via Homebrew), various Linux distributions (Ubuntu, Red Hat, Fedora, Arch), and provides manual installation guidance for Windows.
