A Helm package manager installer that provides cross-platform detection, installation instructions, and automated installation capabilities for Kubernetes Helm charts.

## Key Components

- **HelmInstaller struct**: Main installer implementation with platform-specific installation methods
- **commandExists()**: Utility function to check if a command is available in PATH
- **isHelmInstalled()**: Verifies Helm installation by checking command availability and version
- **helmInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Automated installation with fallback methods for different Linux distributions

## Platform Support

The installer supports:
- **macOS**: Uses Homebrew (`brew install helm`)
- **Linux**: Detects distribution and uses appropriate package manager (apt, yum, dnf, pacman) with fallback to official script
- **Windows**: Provides manual installation instructions only

## Usage Example

```go
// Create a new Helm installer
installer := helm.NewHelmInstaller()

// Check if Helm is already installed
if installer.IsInstalled() {
    fmt.Println("Helm is already installed")
} else {
    // Get installation instructions
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation (Linux/macOS only)
    if err := installer.Install(); err != nil {
        fmt.Printf("Installation failed: %v\n", err)
    }
}
```

The installer handles edge cases like missing package managers, provides helpful error messages, and falls back to the official Helm installation script when distribution-specific packages aren't available.
