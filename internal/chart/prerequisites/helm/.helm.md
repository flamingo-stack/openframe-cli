This file provides a cross-platform Helm installer that can detect, install, and manage the Helm Kubernetes package manager across different operating systems.

## Key Components

- **`HelmInstaller`** - Main struct that handles Helm installation operations
- **`NewHelmInstaller()`** - Constructor function that returns a new HelmInstaller instance
- **`IsInstalled()`** - Checks if Helm is already installed on the system
- **`GetInstallHelp()`** - Returns OS-specific installation instructions
- **`Install()`** - Automatically installs Helm based on the detected operating system
- **Helper functions** - `commandExists()`, `isHelmInstalled()`, and various OS-specific installers

## Usage Example

```go
installer := helm.NewHelmInstaller()

// Check if Helm is already installed
if installer.IsInstalled() {
    fmt.Println("Helm is already installed")
} else {
    // Get manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Or attempt automatic installation
    if err := installer.Install(); err != nil {
        fmt.Printf("Installation failed: %v\n", err)
    }
}
```

The installer supports automatic installation on macOS (via Homebrew) and various Linux distributions (Ubuntu, Red Hat, Fedora, Arch), while providing manual installation guidance for Windows and unsupported platforms.
