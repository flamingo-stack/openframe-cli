A cross-platform Helm installer that automatically detects and installs Helm on macOS, Linux, and Windows systems, with special handling for Windows WSL2 environments.

## Key Components

- **HelmInstaller** - Main installer struct with platform-specific installation methods
- **commandExists()** - Utility function to check if a command is available in PATH
- **isHelmInstalled()** - Detects existing Helm installations with timeout protection
- **helmInstallHelp()** - Provides platform-specific manual installation instructions
- **Platform installers** - Dedicated methods for macOS (Homebrew), Linux (package managers), and Windows (WSL2)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := helm.NewHelmInstaller()
    
    // Check if Helm is already installed
    if installer.IsInstalled() {
        fmt.Println("Helm is already installed")
        return
    }
    
    // Get manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
        return
    }
    
    fmt.Println("Helm installed successfully!")
}
```

The installer supports multiple Linux package managers (apt, yum, dnf, pacman) and falls back to the official Helm installation script. On Windows, it installs Helm within WSL2 Ubuntu and creates a wrapper batch file for seamless integration with the Windows command line.
