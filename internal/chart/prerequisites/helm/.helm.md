A cross-platform Helm installer that provides automated installation capabilities for the Helm Kubernetes package manager across different operating systems.

## Key Components

- **HelmInstaller**: Main struct providing installation methods
- **commandExists()**: Utility function to check if a command is available in PATH
- **isHelmInstalled()**: Verifies if Helm is already installed and functional
- **helmInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Performs automated installation based on the detected OS
- Platform-specific installers: `installMacOS()`, `installLinux()`, `installUbuntu()`, etc.

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := NewHelmInstaller()
    
    // Check if Helm is already installed
    if installer.IsInstalled() {
        fmt.Println("Helm is already installed")
        return
    }
    
    // Get installation help for manual installation
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Auto-install failed: %v", err)
        fmt.Println("Please install manually:", installer.GetInstallHelp())
    } else {
        fmt.Println("Helm installed successfully")
    }
}
```

The installer supports macOS (via Homebrew), various Linux distributions (Ubuntu, Red Hat, Fedora, Arch), and provides manual installation guidance for Windows.
