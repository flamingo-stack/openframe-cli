Cross-platform Helm installer that automatically detects and installs Helm on macOS, Linux, and Windows (with WSL2 support). Provides platform-specific installation methods and help messages for manual installation.

## Key Components

- **HelmInstaller**: Main installer struct with methods for detection and installation
- **isHelmInstalled()**: Checks if Helm is available on the system (with WSL2 support on Windows)
- **helmInstallHelp()**: Provides platform-specific manual installation instructions
- **Install()**: Automatically installs Helm using the appropriate method for each platform
- **Platform-specific installers**: Separate methods for macOS (Homebrew), Linux (distro-specific), and Windows (WSL2)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := helm.NewHelmInstaller()
    
    // Check if Helm is already installed
    if installer.IsInstalled() {
        fmt.Println("Helm is already installed")
        return
    }
    
    // Get manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually using the instructions above")
        return
    }
    
    fmt.Println("Helm installed successfully!")
}
```

The installer supports multiple Linux distributions (Ubuntu, Red Hat, Fedora, Arch) and creates a Windows wrapper script that enables Helm usage from Windows command prompt by routing commands through WSL2.
