Provides cross-platform Helm installation detection and automated installation capabilities with OS-specific package manager integration.

## Key Components

- **HelmInstaller**: Main struct implementing installation methods
- **NewHelmInstaller()**: Constructor function
- **IsInstalled()**: Checks if Helm is available on the system
- **GetInstallHelp()**: Returns OS-specific manual installation instructions
- **Install()**: Performs automated installation based on detected OS and package managers

## Usage Example

```go
installer := helm.NewHelmInstaller()

// Check if Helm is already installed
if !installer.IsInstalled() {
    fmt.Println("Helm not found")
    
    // Get manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Or attempt automatic installation
    if err := installer.Install(); err != nil {
        fmt.Printf("Failed to install Helm: %v\n", err)
    }
}
```

The installer automatically detects the operating system and uses appropriate package managers (Homebrew on macOS, apt/yum/dnf/pacman on Linux) or falls back to the official Helm installation script. Windows requires manual installation.
