This file implements a cross-platform memory checker that verifies if the system has sufficient RAM for application requirements. It checks available physical memory against a recommended threshold of 15GB.

## Key Components

- **MemoryChecker**: Main struct implementing memory validation functionality
- **RecommendedMemoryMB**: Constant defining 15GB (15,360 MB) minimum memory requirement
- **IsInstalled()**: Checks if system meets memory requirements
- **getTotalMemoryMB()**: Platform-specific memory detection for macOS, Linux, and Windows
- **GetMemoryInfo()**: Returns current memory, recommended amount, and sufficiency status
- **GetInstallHelp()**: Provides user-friendly memory status information

## Usage Example

```go
checker := memory.NewMemoryChecker()

// Check if system has sufficient memory
if checker.IsInstalled() {
    fmt.Println("System has sufficient memory")
} else {
    // Get detailed memory information
    current, recommended, sufficient := checker.GetMemoryInfo()
    fmt.Printf("Memory check failed: %d MB available, %d MB required\n", 
               current, recommended)
    
    // Show help message
    fmt.Println(checker.GetInstallHelp())
}

// Attempt installation (will return error for memory)
if err := checker.Install(); err != nil {
    fmt.Printf("Cannot install memory: %v\n", err)
}
```

The checker uses platform-specific commands (`sysctl` for macOS, `/proc/meminfo` for Linux, `PowerShell` for Windows) to retrieve accurate physical memory information and determine if the system meets the 15GB requirement.
