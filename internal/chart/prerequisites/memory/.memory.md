A cross-platform system memory checker that validates if the machine has sufficient RAM for application requirements, with a recommended minimum of 15GB.

## Key Components

- **MemoryChecker**: Main struct implementing memory validation functionality
- **NewMemoryChecker()**: Constructor function returning a new MemoryChecker instance
- **IsInstalled()**: Checks if sufficient memory is available (aliases HasSufficientMemory)
- **HasSufficientMemory()**: Validates if current memory meets the 15GB recommendation
- **GetInstallHelp()**: Provides formatted message with current vs recommended memory
- **Install()**: Returns error since memory cannot be automatically installed
- **GetMemoryInfo()**: Returns current memory, recommended memory, and sufficiency status
- **getTotalMemoryMB()**: Cross-platform memory detection using OS-specific commands

## Usage Example

```go
package main

import (
    "fmt"
    "your-app/memory"
)

func main() {
    checker := memory.NewMemoryChecker()
    
    // Check if system has sufficient memory
    if checker.IsInstalled() {
        fmt.Println("System has sufficient memory")
    } else {
        fmt.Println(checker.GetInstallHelp())
    }
    
    // Get detailed memory information
    current, recommended, sufficient := checker.GetMemoryInfo()
    fmt.Printf("Current: %d MB, Recommended: %d MB, Sufficient: %v\n", 
               current, recommended, sufficient)
}
```

The checker uses platform-specific commands: `sysctl` on macOS, `/proc/meminfo` on Linux, and PowerShell on Windows to retrieve total physical memory information.
