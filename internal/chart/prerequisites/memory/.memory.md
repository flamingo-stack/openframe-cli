Cross-platform memory checker that validates system RAM meets minimum requirements for resource-intensive applications.

## Key Components

- **`MemoryChecker`**: Main struct implementing memory validation interface
- **`NewMemoryChecker()`**: Constructor function
- **`HasSufficientMemory()`**: Checks if system meets 15GB RAM requirement
- **`GetMemoryInfo()`**: Returns current, recommended memory and sufficiency status
- **`getTotalMemoryMB()`**: Cross-platform memory detection (macOS/Linux/Windows)
- **`RecommendedMemoryMB`**: Constant defining 15GB minimum requirement

## Usage Example

```go
checker := memory.NewMemoryChecker()

// Check if system has sufficient memory
if checker.HasSufficientMemory() {
    fmt.Println("System has sufficient memory")
} else {
    fmt.Println(checker.GetInstallHelp())
}

// Get detailed memory information
current, recommended, sufficient := checker.GetMemoryInfo()
fmt.Printf("Memory: %dMB/%dMB (sufficient: %v)\n", 
    current, recommended, sufficient)

// Use as part of installation checker
if !checker.IsInstalled() {
    fmt.Println("Memory requirements not met")
}
```

The checker uses platform-specific commands (`sysctl` on macOS, `/proc/meminfo` on Linux, PowerShell on Windows) to detect total system RAM and validates against the 15GB threshold.
