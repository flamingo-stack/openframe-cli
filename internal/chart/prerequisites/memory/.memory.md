Cross-platform memory checker that validates system RAM meets minimum requirements for resource-intensive applications. Detects available memory on Windows, macOS, and Linux systems using platform-specific commands.

## Key Components

- **MemoryChecker**: Main struct implementing memory validation interface
- **RecommendedMemoryMB**: Constant defining 15GB minimum requirement
- **IsInstalled()**: Checks if system has sufficient memory
- **GetInstallHelp()**: Returns memory status and recommendations
- **GetMemoryInfo()**: Returns current, recommended, and sufficiency status
- **Platform-specific methods**: getMacOSMemory(), getLinuxMemory(), getWindowsMemory()

## Usage Example

```go
checker := memory.NewMemoryChecker()

// Check if system meets memory requirements
if checker.IsInstalled() {
    fmt.Println("System has sufficient memory")
} else {
    fmt.Println(checker.GetInstallHelp())
    // Output: Memory: 8192 MB available, 15360 MB recommended...
}

// Get detailed memory information
current, recommended, sufficient := checker.GetMemoryInfo()
fmt.Printf("Current: %d MB, Required: %d MB, Sufficient: %t\n", 
    current, recommended, sufficient)

// Attempt installation (will return error)
if err := checker.Install(); err != nil {
    fmt.Println("Cannot auto-install memory:", err)
}
```
