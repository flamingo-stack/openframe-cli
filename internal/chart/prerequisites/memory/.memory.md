A cross-platform memory checker that validates system RAM meets the recommended 15GB requirement for the application.

## Key Components

- **`MemoryChecker`** - Main struct implementing memory validation
- **`NewMemoryChecker()`** - Constructor function
- **`HasSufficientMemory()`** - Checks if system meets 15GB RAM requirement
- **`GetMemoryInfo()`** - Returns current, recommended, and sufficiency status
- **`IsInstalled()`** - Interface method that aliases memory check
- **`GetInstallHelp()`** - Provides user guidance when memory is insufficient
- **Platform-specific methods** - `getMacOSMemory()`, `getLinuxMemory()`, `getWindowsMemory()`

## Usage Example

```go
package main

import (
    "fmt"
    "your-project/memory"
)

func main() {
    checker := memory.NewMemoryChecker()
    
    // Check if system has sufficient memory
    if checker.HasSufficientMemory() {
        fmt.Println("System meets memory requirements")
    } else {
        fmt.Println(checker.GetInstallHelp())
    }
    
    // Get detailed memory information
    current, recommended, sufficient := checker.GetMemoryInfo()
    fmt.Printf("Current: %d MB, Recommended: %d MB, Sufficient: %v\n", 
               current, recommended, sufficient)
}
```

The checker uses platform-specific system commands (`sysctl` on macOS, `/proc/meminfo` on Linux, PowerShell on Windows) to accurately detect total system RAM and compare against the 15GB minimum requirement.
