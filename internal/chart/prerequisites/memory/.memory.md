A cross-platform memory checker utility that validates whether a system has sufficient RAM (15GB recommended) for resource-intensive applications. It provides system memory detection across macOS, Linux, and Windows platforms.

## Key Components

- **`MemoryChecker`** - Main struct implementing memory validation logic
- **`NewMemoryChecker()`** - Constructor function returning a new MemoryChecker instance
- **`IsInstalled()`** - Checks if system meets memory requirements
- **`HasSufficientMemory()`** - Validates if available memory meets the 15GB threshold
- **`GetInstallHelp()`** - Provides user-friendly guidance for memory upgrades
- **`GetMemoryInfo()`** - Returns current, recommended, and sufficiency status
- **`RecommendedMemoryMB`** - Constant defining 15GB memory requirement

## Usage Example

```go
checker := memory.NewMemoryChecker()

// Check if system has sufficient memory
if checker.IsInstalled() {
    fmt.Println("System has sufficient memory")
} else {
    fmt.Println(checker.GetInstallHelp())
}

// Get detailed memory information
current, recommended, sufficient := checker.GetMemoryInfo()
fmt.Printf("Current: %d MB, Recommended: %d MB, Sufficient: %v\n", 
    current, recommended, sufficient)

// Simple memory validation
if !checker.HasSufficientMemory() {
    fmt.Println("Warning: Insufficient memory detected")
}
```

The checker automatically detects the operating system and uses appropriate system commands to retrieve memory information.
