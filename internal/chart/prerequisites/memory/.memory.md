Cross-platform memory detection and validation utility that checks if the system has sufficient RAM for application requirements using OS-specific commands. This package provides a standardized interface for memory checking across macOS, Linux, and Windows platforms.

## Key Components

- **MemoryChecker**: Main struct implementing memory validation interface
- **RecommendedMemoryMB**: Constant defining minimum required memory (15GB)
- **NewMemoryChecker()**: Constructor function returning a new MemoryChecker instance
- **IsInstalled()**: Checks if sufficient memory is available
- **HasSufficientMemory()**: Core validation method comparing current vs required memory
- **GetMemoryInfo()**: Returns current memory, recommended memory, and sufficiency status
- **getTotalMemoryMB()**: Platform-agnostic memory detection with OS-specific implementations

## Usage Example

```go
import "github.com/openframe/openframe/pkg/memory"

// Create memory checker
checker := memory.NewMemoryChecker()

// Check if system has sufficient memory
if checker.HasSufficientMemory() {
    fmt.Println("System has sufficient memory")
} else {
    fmt.Println(checker.GetInstallHelp())
}

// Get detailed memory information
current, recommended, sufficient := checker.GetMemoryInfo()
fmt.Printf("Current: %dMB, Recommended: %dMB, Sufficient: %v\n", 
    current, recommended, sufficient)

// Use as part of installation checks
if !checker.IsInstalled() {
    return fmt.Errorf("insufficient memory: %s", checker.GetInstallHelp())
}
```

The checker uses platform-specific commands: `sysctl hw.memsize` on macOS, `/proc/meminfo` on Linux, and PowerShell's `Get-CimInstance` on Windows to retrieve total physical memory information.
