Test suite for the memory checker system that validates memory detection, sufficiency checks, and error handling across different operating systems.

## Key Components

- **TestNewMemoryChecker** - Validates proper instantiation of MemoryChecker
- **TestMemoryChecker_GetMemoryInfo** - Tests memory information retrieval and sufficiency logic
- **TestMemoryChecker_GetInstallHelp** - Verifies help text generation for memory requirements
- **TestMemoryChecker_Install** - Confirms proper error handling for unsupported installation
- **TestGetTotalMemoryMB** - Tests cross-platform memory detection methods
- **TestHasSufficientMemory** - Validates memory sufficiency checking logic
- **containsSubstring** - Helper function for string matching in tests

## Usage Example

```go
// Run all memory tests
go test ./memory

// Run specific test
go test -run TestMemoryChecker_GetMemoryInfo ./memory

// Test with verbose output
go test -v ./memory

// Test platform-specific memory detection
go test -run TestGetTotalMemoryMB ./memory
```

The tests verify that the memory checker correctly detects system memory across macOS, Linux, and Windows platforms, properly compares against recommended requirements, and provides appropriate error messages when memory cannot be automatically installed.
