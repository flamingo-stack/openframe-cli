This file contains comprehensive unit tests for the memory checking functionality, validating memory detection, system requirements evaluation, and error handling across different operating systems.

## Key Components

- **TestNewMemoryChecker** - Tests memory checker instance creation
- **TestMemoryChecker_GetMemoryInfo** - Validates memory information retrieval and sufficiency calculations
- **TestMemoryChecker_GetInstallHelp** - Verifies help message content and formatting
- **TestMemoryChecker_Install** - Tests error handling for memory installation attempts
- **TestGetTotalMemoryMB** - Tests memory detection across platforms (macOS, Linux, Windows)
- **TestHasSufficientMemory** - Validates memory requirement checking logic
- **containsSubstring** - Helper function for string validation in tests

## Usage Example

```go
// Run all memory-related tests
go test -v ./memory

// Run specific test
go test -run TestMemoryChecker_GetMemoryInfo ./memory

// Test with coverage
go test -cover ./memory

// The tests validate:
checker := NewMemoryChecker()
current, recommended, sufficient := checker.GetMemoryInfo()
// Ensures current > 0, recommended matches constant, sufficient is accurate

// Platform-specific memory detection is tested automatically
// based on runtime.GOOS (darwin, linux, windows)
```
