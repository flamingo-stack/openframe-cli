Test suite for the prerequisite checker system that validates system requirements for the OpenFrame CLI. Contains comprehensive tests for checking and managing dependencies like Git, Helm, memory, and certificates.

## Key Components

- **TestNewPrerequisiteChecker**: Validates the checker initializes with all 4 required dependencies (Git, Helm, Memory, Certificates) in correct order
- **TestInstallHelp**: Ensures all prerequisite checkers provide non-empty installation instructions
- **TestCheckAllWithMissingTools**: Tests detection of missing tools and proper reporting
- **TestCheckAllWithAllTools**: Verifies behavior when all prerequisites are satisfied
- **TestGetInstallInstructions**: Validates installation instruction generation for missing tools

## Usage Example

```go
func TestCustomPrerequisite(t *testing.T) {
    checker := NewPrerequisiteChecker()
    
    // Mock some requirements as missing
    checker.requirements[0].IsInstalled = func() bool { return false }
    checker.requirements[1].IsInstalled = func() bool { return true }
    
    allPresent, missing := checker.CheckAll()
    
    if allPresent {
        t.Error("Expected missing prerequisites")
    }
    
    if len(missing) != 1 {
        t.Errorf("Expected 1 missing tool, got %d", len(missing))
    }
    
    instructions := checker.GetInstallInstructions(missing)
    // Verify instructions are provided
}
```
