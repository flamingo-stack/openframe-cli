This file implements a comprehensive prerequisite installer that manages installation and verification of required tools for the OpenFrame CLI application.

## Key Components

- **Installer struct**: Main installer with embedded PrerequisiteChecker
- **NewInstaller()**: Constructor that initializes a new installer instance
- **InstallMissingPrerequisites()**: Checks and installs all missing prerequisites
- **CheckAndInstallNonInteractive()**: Handles prerequisite installation with optional non-interactive mode for CI/CD
- **RegenerateCertificatesOnly()**: Specifically regenerates certificates without checking other tools
- **installToolNonInteractive()**: Core method that installs individual tools (git, helm, certificates)

## Usage Example

```go
package main

import "github.com/flamingo-stack/openframe-cli/internal/chart/prerequisites"

func main() {
    installer := prerequisites.NewInstaller()
    
    // Interactive installation with user prompts
    if err := installer.CheckAndInstall(); err != nil {
        panic(err)
    }
    
    // Non-interactive installation for CI/CD
    if err := installer.CheckAndInstallNonInteractive(true); err != nil {
        panic(err)
    }
    
    // Just regenerate certificates
    if err := installer.RegenerateCertificatesOnly(); err != nil {
        panic(err)
    }
}
```

The installer provides both interactive and non-interactive modes, handles memory warnings gracefully, and can install git, helm, and certificates while providing clear progress feedback through spinners and success/failure messages.
