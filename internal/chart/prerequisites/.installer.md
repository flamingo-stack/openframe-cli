Manages the automated installation of OpenFrame prerequisites including Git, Helm, and SSL certificates. The installer can operate in both interactive and non-interactive modes for CI/CD environments.

## Key Components

- **`Installer`** - Main installer struct that orchestrates prerequisite installation
- **`NewInstaller()`** - Creates a new installer instance with an embedded prerequisite checker
- **`InstallMissingPrerequisites()`** - Installs all missing prerequisites automatically
- **`CheckAndInstall()`/`CheckAndInstallNonInteractive()`** - Checks prerequisites and prompts for installation
- **`RegenerateCertificatesOnly()`** - Regenerates SSL certificates without checking other prerequisites
- **`installToolNonInteractive()`** - Installs individual tools (git, helm, certificates) with CI/CD support

## Usage Example

```go
// Basic installation
installer := NewInstaller()
if err := installer.InstallMissingPrerequisites(); err != nil {
    log.Fatal(err)
}

// Interactive check and install
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Non-interactive mode for CI/CD
if err := installer.CheckAndInstallNonInteractive(true); err != nil {
    log.Fatal(err)
}

// Regenerate certificates only
if err := installer.RegenerateCertificatesOnly(); err != nil {
    log.Printf("Certificate regeneration failed: %v", err)
}
```

The installer handles memory warnings gracefully (cannot install RAM automatically) and provides manual installation instructions when automatic installation is declined. In non-interactive mode, it continues execution even if some prerequisites fail to install.
