This file implements a prerequisite installer that automatically downloads and installs required tools for the OpenFrame CLI, with support for both interactive and non-interactive modes.

## Key Components

- **Installer struct**: Main installer that manages prerequisite installation using a PrerequisiteChecker
- **InstallMissingPrerequisites()**: Primary method that checks and installs all missing prerequisites
- **CheckAndInstallNonInteractive()**: Entry point that supports both interactive prompts and CI/CD automation
- **installToolNonInteractive()**: Core installation logic for individual tools (git, helm, certificates)
- **RegenerateCertificatesOnly()**: Specialized method for certificate regeneration during deployment

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/prerequisites"
)

func main() {
    installer := prerequisites.NewInstaller()
    
    // Interactive mode with user prompts
    if err := installer.CheckAndInstall(); err != nil {
        log.Fatal("Failed to install prerequisites:", err)
    }
    
    // Non-interactive mode for CI/CD
    if err := installer.CheckAndInstallNonInteractive(true); err != nil {
        log.Fatal("Failed to install prerequisites:", err)
    }
    
    // Regenerate certificates only (for install command)
    if err := installer.RegenerateCertificatesOnly(); err != nil {
        log.Fatal("Failed to regenerate certificates:", err)
    }
}
```

The installer handles memory checks as warnings only (since RAM cannot be programmatically increased), provides detailed installation instructions for manual setup, and continues gracefully in CI/CD environments even when some tools fail to install.
