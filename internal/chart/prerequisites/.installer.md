This file provides the main installation orchestration logic for managing system prerequisites required by the OpenFrame CLI tool.

## Key Components

- **`Installer`** - Core struct that manages prerequisite installation workflow
- **`NewInstaller()`** - Constructor that creates installer with prerequisite checker
- **`CheckAndInstall()`/`CheckAndInstallNonInteractive()`** - Main entry points for checking and installing missing prerequisites
- **`InstallMissingPrerequisites()`** - Installs all detected missing tools
- **`RegenerateCertificatesOnly()`** - Specialized method for certificate refresh during install command

## Usage Example

```go
// Basic usage - interactive mode
installer := prerequisites.NewInstaller()
err := installer.CheckAndInstall()
if err != nil {
    log.Fatal(err)
}

// Non-interactive mode (for CI/CD)
err = installer.CheckAndInstallNonInteractive(true)
if err != nil {
    log.Fatal(err)
}

// Just regenerate certificates
err = installer.RegenerateCertificatesOnly()
if err != nil {
    log.Fatal(err)
}
```

The installer handles Git, Helm, memory checks, and certificate generation, providing user-friendly prompts in interactive mode and automatic installation in non-interactive environments. Memory warnings are non-blocking, while other missing tools can be auto-installed or require manual setup.
