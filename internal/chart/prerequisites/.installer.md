Manages the installation of prerequisites required for chart deployment including Git, Helm, certificates, and memory validation. Provides both interactive and non-interactive installation modes with progress feedback.

## Key Components

- **Installer**: Main struct that orchestrates prerequisite installation
- **NewInstaller()**: Creates a new installer instance with prerequisite checker
- **InstallMissingPrerequisites()**: Identifies and installs missing tools
- **CheckAndInstall()**: Checks prerequisites and prompts for installation
- **CheckAndInstallNonInteractive()**: Non-interactive version for CI/CD environments
- **RegenerateCertificatesOnly()**: Regenerates certificates without checking other tools

## Usage Example

```go
package main

import "github.com/flamingo-stack/openframe-cli/internal/chart/prerequisites"

func main() {
    installer := prerequisites.NewInstaller()
    
    // Interactive mode with user prompts
    if err := installer.CheckAndInstall(); err != nil {
        log.Fatal(err)
    }
    
    // Non-interactive mode for CI/CD
    if err := installer.CheckAndInstallNonInteractive(true); err != nil {
        log.Fatal(err)
    }
    
    // Just regenerate certificates
    if err := installer.RegenerateCertificatesOnly(); err != nil {
        log.Printf("Certificate warning: %v", err)
    }
}
```

The installer handles Git, Helm, certificates, and memory validation with user-friendly spinners and progress indicators. Memory issues show warnings but don't block installation.
