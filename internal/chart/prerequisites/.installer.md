Manages the installation and verification of OpenFrame CLI prerequisites, providing both interactive and non-interactive modes for automated tool setup.

## Key Components

- **`Installer`** - Main struct that orchestrates prerequisite installation
- **`NewInstaller()`** - Constructor that initializes installer with prerequisite checker
- **`InstallMissingPrerequisites()`** - Checks and installs all missing prerequisites
- **`CheckAndInstallNonInteractive()`** - Handles prerequisite installation with CI/CD support
- **`RegenerateCertificatesOnly()`** - Specifically regenerates certificates for install command

## Usage Example

```go
// Create installer and check/install prerequisites
installer := NewInstaller()

// Interactive mode (prompts user)
if err := installer.CheckAndInstall(); err != nil {
    log.Fatal(err)
}

// Non-interactive mode (for CI/CD)
if err := installer.CheckAndInstallNonInteractive(true); err != nil {
    log.Fatal(err)
}

// Regenerate certificates only
if err := installer.RegenerateCertificatesOnly(); err != nil {
    log.Printf("Certificate warning: %v", err)
}
```

The installer handles Git, Helm, memory validation, and certificate generation with appropriate fallbacks for automated environments. Memory issues are treated as warnings rather than blocking errors, and the non-interactive mode skips certificate generation for CI/CD environments.
