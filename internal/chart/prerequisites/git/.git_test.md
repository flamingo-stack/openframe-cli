This file contains comprehensive unit tests for the Git checker functionality, testing Git installation detection, version retrieval, and validation methods.

## Key Components

- **TestNewGitChecker** - Tests creation of new Git checker instances
- **TestGitChecker_GetInstallInstructions** - Validates installation instruction content for different platforms
- **TestGitChecker_IsInstalled** - Tests Git installation detection logic
- **TestGitChecker_GetVersion** - Tests Git version retrieval when installed
- **TestGitChecker_Validate** - Tests overall Git validation functionality
- **TestGitChecker_GetVersion_NotInstalled** - Tests error handling when Git is not installed

## Usage Example

Run these tests to verify Git checker functionality:

```go
// Run all tests
go test ./git

// Run specific test
go test -run TestNewGitChecker ./git

// Run with verbose output
go test -v ./git
```

The tests handle different Git installation states automatically using conditional skipping:

```go
if !checker.IsInstalled() {
    t.Skip("Git not installed, skipping version test")
}
```

Tests validate that installation instructions contain platform-specific information (macOS, Ubuntu) and verification commands, ensuring the Git checker provides complete setup guidance across different operating systems.
