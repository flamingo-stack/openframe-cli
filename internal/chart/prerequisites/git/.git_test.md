This file contains comprehensive unit tests for the Git checker functionality, validating Git installation detection, version retrieval, and installation instructions.

## Key Components

- **TestNewGitChecker**: Tests the creation of a new GitChecker instance
- **TestGitChecker_GetInstallInstructions**: Validates installation instructions contain platform-specific guidance (macOS, Ubuntu) and verification commands
- **TestGitChecker_IsInstalled**: Tests the Git installation detection method
- **TestGitChecker_GetVersion**: Verifies version retrieval when Git is installed
- **TestGitChecker_Validate**: Tests the validation method for installed Git
- **TestGitChecker_GetVersion_NotInstalled**: Tests error handling when Git is not installed

## Usage Example

Run the tests to validate Git checker functionality:

```go
// Run all tests
go test ./git

// Run specific test
go test -run TestNewGitChecker ./git

// Run with verbose output
go test -v ./git
```

The tests automatically adapt to the environment - they skip version-related tests if Git isn't installed and test error handling appropriately. Tests verify that installation instructions include platform-specific information and that version strings follow expected Git format patterns.
