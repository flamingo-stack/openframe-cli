This file provides a git prerequisite checker for validating git installation and version requirements in a development environment.

## Key Components

- **GitChecker**: Main struct for validating git prerequisites
- **NewGitChecker()**: Factory function creating a new checker instance
- **IsInstalled()**: Checks if git binary is available in system PATH
- **GetVersion()**: Retrieves and returns the installed git version string
- **GetInstallInstructions()**: Provides cross-platform installation guidance
- **Validate()**: Performs comprehensive validation including minimum version check (git 2.0+)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    checker := NewGitChecker()
    
    // Quick installation check
    if !checker.IsInstalled() {
        fmt.Println("Git not found!")
        fmt.Println(checker.GetInstallInstructions())
        return
    }
    
    // Get version information
    version, err := checker.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Found: %s\n", version)
    
    // Comprehensive validation
    if err := checker.Validate(); err != nil {
        log.Fatalf("Git validation failed: %v", err)
    }
    
    fmt.Println("Git validation passed!")
}
```

The checker ensures git 2.0+ is installed and working properly before proceeding with git-dependent operations like chart repository cloning.
