A Git prerequisite checker that validates Git installation and version requirements for chart repository operations.

## Key Components

- **GitChecker**: Main struct that provides Git validation functionality
- **NewGitChecker()**: Constructor that creates a new Git checker instance
- **IsInstalled()**: Checks if Git is available in the system PATH
- **GetVersion()**: Retrieves the installed Git version string
- **GetInstallInstructions()**: Returns platform-specific installation guidance
- **Validate()**: Performs comprehensive Git validation including minimum version check (Git 2.0+)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    checker := NewGitChecker()
    
    // Check if Git is installed
    if !checker.IsInstalled() {
        fmt.Println("Git not found!")
        fmt.Println(checker.GetInstallInstructions())
        return
    }
    
    // Get Git version
    version, err := checker.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Found: %s\n", version)
    
    // Comprehensive validation
    if err := checker.Validate(); err != nil {
        fmt.Printf("Git validation failed: %s\n", err)
        fmt.Println(checker.GetInstallInstructions())
    } else {
        fmt.Println("Git validation passed!")
    }
}
```

The checker ensures Git 2.0+ is installed and functional before attempting chart repository operations, providing clear installation instructions when prerequisites aren't met.
