This file provides Git installation validation and prerequisite checking functionality for applications that depend on Git being available in the system PATH.

## Key Components

- **GitChecker**: Main struct for validating Git prerequisites
- **NewGitChecker()**: Constructor function that creates a new GitChecker instance
- **IsInstalled()**: Checks if Git is available in the system PATH
- **GetVersion()**: Retrieves the installed Git version string
- **GetInstallInstructions()**: Returns platform-specific Git installation instructions
- **Validate()**: Performs comprehensive validation including version checking (requires Git 2.0+)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    checker := NewGitChecker()
    
    // Basic installation check
    if !checker.IsInstalled() {
        fmt.Println("Git not found!")
        fmt.Println(checker.GetInstallInstructions())
        return
    }
    
    // Get version information
    version, err := checker.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Found: %s\n", version)
    
    // Comprehensive validation
    if err := checker.Validate(); err != nil {
        log.Fatalf("Git validation failed: %v", err)
    }
    
    fmt.Println("Git is properly installed and ready to use")
}
```
