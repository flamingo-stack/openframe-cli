This file provides a Git prerequisite checker that validates Git installation and compatibility for applications that depend on Git functionality.

## Key Components

- **GitChecker**: Main struct that encapsulates Git validation logic
- **NewGitChecker()**: Constructor function for creating a new GitChecker instance
- **IsInstalled()**: Checks if Git is available in the system PATH
- **GetVersion()**: Retrieves the installed Git version string
- **GetInstallInstructions()**: Returns platform-specific installation guidance
- **Validate()**: Performs comprehensive validation including minimum version requirements (Git 2.0+)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    checker := NewGitChecker()
    
    // Basic installation check
    if !checker.IsInstalled() {
        fmt.Println("Git not found!")
        fmt.Println(checker.GetInstallInstructions())
        return
    }
    
    // Get version information
    version, err := checker.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Found: %s\n", version)
    
    // Comprehensive validation
    if err := checker.Validate(); err != nil {
        log.Fatalf("Git validation failed: %v", err)
    }
    
    fmt.Println("Git is properly installed and ready!")
}
```
