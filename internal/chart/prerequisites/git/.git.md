Validates Git installation and version requirements as a system prerequisite checker for chart repository operations.

## Key Components

- **GitChecker**: Main struct for Git validation operations
- **NewGitChecker()**: Constructor that creates a new Git checker instance
- **IsInstalled()**: Checks if Git is available in the system PATH
- **GetVersion()**: Retrieves the installed Git version string
- **GetInstallInstructions()**: Provides platform-specific installation guidance
- **Validate()**: Performs comprehensive Git validation including minimum version checks (requires Git 2.0+)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    checker := NewGitChecker()
    
    // Quick installation check
    if !checker.IsInstalled() {
        fmt.Println("Git not found!")
        fmt.Println(checker.GetInstallInstructions())
        return
    }
    
    // Get version information
    version, err := checker.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Found: %s\n", version)
    
    // Full validation with version requirements
    if err := checker.Validate(); err != nil {
        log.Fatalf("Git validation failed: %v", err)
    }
    
    fmt.Println("Git validation passed!")
}
```
