A prerequisite checker that validates Git installation and version requirements for chart repository operations.

## Key Components

- **`GitChecker`** - Main struct for validating Git prerequisites
- **`NewGitChecker()`** - Constructor function that creates a new GitChecker instance
- **`IsInstalled()`** - Checks if Git is available in the system PATH
- **`GetVersion()`** - Retrieves the installed Git version string
- **`GetInstallInstructions()`** - Returns platform-specific installation commands
- **`Validate()`** - Performs comprehensive validation including minimum version check (Git 2.0+)

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    checker := NewGitChecker()
    
    // Check if git is installed
    if !checker.IsInstalled() {
        fmt.Println(checker.GetInstallInstructions())
        return
    }
    
    // Get version information
    version, err := checker.GetVersion()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Found: %s\n", version)
    
    // Perform full validation
    if err := checker.Validate(); err != nil {
        log.Fatalf("Git validation failed: %v", err)
    }
    
    fmt.Println("Git is ready for use!")
}
```
