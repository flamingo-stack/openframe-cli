This file provides a Git prerequisite validator that checks for Git installation, version compatibility, and provides installation guidance for chart repository operations.

## Key Components

- **GitChecker**: Main struct that encapsulates Git validation functionality
- **NewGitChecker()**: Constructor function that creates a new GitChecker instance
- **IsInstalled()**: Checks if Git is available in the system PATH
- **GetVersion()**: Retrieves the installed Git version string
- **GetInstallInstructions()**: Returns platform-specific installation commands
- **Validate()**: Performs comprehensive Git validation including minimum version checking (requires Git 2.0+)

## Usage Example

```go
checker := git.NewGitChecker()

// Basic installation check
if !checker.IsInstalled() {
    fmt.Println("Git not found!")
    fmt.Println(checker.GetInstallInstructions())
    return
}

// Get version information
version, err := checker.GetVersion()
if err != nil {
    fmt.Printf("Error getting version: %v\n", err)
    return
}
fmt.Printf("Found: %s\n", version)

// Full validation with version requirements
if err := checker.Validate(); err != nil {
    fmt.Printf("Git validation failed: %v\n", err)
    return
}
fmt.Println("Git is properly installed and ready!")
```

The validator ensures Git 2.0+ is available for reliable chart repository cloning operations, providing clear error messages and installation guidance when prerequisites aren't met.
