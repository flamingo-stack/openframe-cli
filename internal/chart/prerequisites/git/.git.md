A Git prerequisite checker that validates Git installation, version compatibility, and provides installation guidance for chart repository operations.

## Key Components

- **GitChecker**: Main struct for Git validation operations
- **NewGitChecker()**: Constructor for creating a new Git checker instance
- **IsInstalled()**: Checks if Git is available in system PATH
- **GetVersion()**: Retrieves the installed Git version string
- **GetInstallInstructions()**: Returns platform-specific installation commands
- **Validate()**: Performs comprehensive Git validation including minimum version check

## Usage Example

```go
// Create a Git checker
checker := NewGitChecker()

// Basic installation check
if !checker.IsInstalled() {
    fmt.Println("Git not found!")
    fmt.Println(checker.GetInstallInstructions())
    return
}

// Get version information
version, err := checker.GetVersion()
if err != nil {
    fmt.Printf("Error getting version: %v\n", err)
    return
}
fmt.Printf("Git version: %s\n", version)

// Full validation (includes version compatibility)
if err := checker.Validate(); err != nil {
    fmt.Printf("Git validation failed: %v\n", err)
    return
}
fmt.Println("Git is ready for use!")
```

The checker enforces a minimum Git 2.0 requirement and provides clear error messages with installation instructions when validation fails.
