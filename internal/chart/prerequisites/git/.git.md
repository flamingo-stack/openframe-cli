A Git prerequisite checker that validates Git installation and version compatibility for the system.

## Key Components

- **GitChecker**: Main struct that provides Git validation functionality
- **NewGitChecker()**: Constructor function that creates a new GitChecker instance
- **IsInstalled()**: Checks if Git is available in the system PATH
- **GetVersion()**: Retrieves the installed Git version string
- **GetInstallInstructions()**: Returns platform-specific installation guidance
- **Validate()**: Performs comprehensive Git validation including minimum version requirements

## Usage Example

```go
checker := git.NewGitChecker()

// Check if Git is installed
if !checker.IsInstalled() {
    fmt.Println("Git not found")
    fmt.Println(checker.GetInstallInstructions())
    return
}

// Get version information
version, err := checker.GetVersion()
if err != nil {
    fmt.Printf("Error getting version: %v\n", err)
    return
}
fmt.Printf("Git version: %s\n", version)

// Perform full validation (includes version check)
if err := checker.Validate(); err != nil {
    fmt.Printf("Git validation failed: %v\n", err)
    return
}

fmt.Println("Git is properly installed and ready to use")
```

The checker enforces a minimum Git version of 2.0 and provides helpful installation instructions for multiple platforms (macOS, Ubuntu, CentOS, Windows).
