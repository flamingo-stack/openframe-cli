This package provides cross-platform SSL certificate management using mkcert for local development environments. It handles automatic installation of mkcert and generation of localhost certificates with proper system trust store integration.

## Key Components

- **CertificateInstaller**: Main struct that manages certificate operations
- **NewCertificateInstaller()**: Factory function to create a new installer instance
- **Install()**: Installs mkcert (if needed) and generates certificates
- **ForceRegenerate()**: Regenerates certificates even if they already exist
- **IsInstalled()**: Checks if mkcert is available
- **GetInstallHelp()**: Returns platform-specific installation guidance

## Usage Example

```go
// Create a new certificate installer
installer := certificates.NewCertificateInstaller()

// Check if mkcert is already installed
if !installer.IsInstalled() {
    fmt.Println(installer.GetInstallHelp())
    
    // Install mkcert and generate certificates
    if err := installer.Install(); err != nil {
        log.Fatal("Failed to install certificates:", err)
    }
}

// Force regenerate certificates (useful for updates)
if err := installer.ForceRegenerate(); err != nil {
    log.Fatal("Failed to regenerate certificates:", err)
}
```

The installer automatically handles platform-specific differences (macOS via Homebrew, Linux via direct download, Windows requires manual installation) and integrates certificates with system trust stores and browser NSS databases for seamless HTTPS development.
