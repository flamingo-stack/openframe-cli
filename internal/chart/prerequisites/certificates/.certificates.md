This package provides cross-platform installation and management of local development SSL certificates using mkcert for the openframe project.

## Key Components

- **CertificateInstaller**: Main struct that handles certificate operations across different operating systems
- **IsInstalled()**: Checks if mkcert is available on the system
- **Install()**: Automatically installs mkcert and generates localhost certificates
- **ForceRegenerate()**: Regenerates certificates even if they already exist
- **GetInstallHelp()**: Returns platform-specific installation guidance

## Usage Example

```go
package main

import (
    "log"
    "github.com/openframe/openframe/certificates"
)

func main() {
    installer := certificates.NewCertificateInstaller()
    
    // Check if certificates are ready
    if !installer.IsInstalled() {
        log.Println(installer.GetInstallHelp())
        
        // Install mkcert and generate certificates
        if err := installer.Install(); err != nil {
            log.Fatal("Failed to install certificates:", err)
        }
    }
    
    // Force regenerate certificates if needed
    if err := installer.ForceRegenerate(); err != nil {
        log.Fatal("Failed to regenerate certificates:", err)
    }
}
```

The package handles platform-specific installation (Homebrew on macOS, direct download on Linux) and automatically configures system trust stores for browsers. Certificates are stored in `~/.config/openframe/certs/`.
