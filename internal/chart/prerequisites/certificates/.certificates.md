This package provides cross-platform SSL certificate management using mkcert for local development environments. It handles automatic installation of mkcert and generation of localhost certificates with proper system trust store integration.

## Key Components

- **CertificateInstaller**: Main struct providing certificate management functionality
- **NewCertificateInstaller()**: Constructor returning a new installer instance
- **IsInstalled()**: Checks if mkcert is available on the system
- **Install()**: Installs mkcert and generates certificates automatically
- **ForceRegenerate()**: Forces regeneration of certificates even if they exist
- **GetInstallHelp()**: Returns platform-specific installation guidance

## Usage Example

```go
installer := certificates.NewCertificateInstaller()

// Check if mkcert is installed
if !installer.IsInstalled() {
    fmt.Println(installer.GetInstallHelp())
    
    // Install mkcert and generate certificates
    if err := installer.Install(); err != nil {
        log.Fatal(err)
    }
}

// Force regenerate certificates if needed
if err := installer.ForceRegenerate(); err != nil {
    log.Fatal(err)
}
```

The installer automatically handles platform differences (macOS via Homebrew, Linux direct download, Windows manual installation) and integrates certificates with system trust stores and browser NSS databases. Generated certificates are stored in `~/.config/openframe/certs/` and include localhost, 127.0.0.1, and ::1 domains.
