This package provides cross-platform SSL certificate management using mkcert for local HTTPS development. It handles automatic installation of mkcert and generation of localhost certificates with proper trust store configuration.

## Key Components

- **CertificateInstaller**: Main struct that manages certificate installation and generation
- **IsInstalled()**: Checks if mkcert is available on the system
- **Install()**: Installs mkcert (if needed) and generates certificates
- **ForceRegenerate()**: Regenerates certificates even if they already exist
- **GetInstallHelp()**: Returns platform-specific installation instructions

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := certificates.NewCertificateInstaller()
    
    // Check if certificates are ready
    if !installer.IsInstalled() {
        fmt.Println(installer.GetInstallHelp())
        
        // Install mkcert and generate certificates
        if err := installer.Install(); err != nil {
            log.Fatalf("Failed to install certificates: %v", err)
        }
    }
    
    // Force regenerate certificates if needed
    if err := installer.ForceRegenerate(); err != nil {
        log.Fatalf("Failed to regenerate certificates: %v", err)
    }
    
    fmt.Println("Certificates ready at ~/.config/openframe/certs/")
}
```

The installer automatically handles platform differences (macOS via Homebrew, Linux via direct download, Windows requires manual installation) and configures trust stores for browsers and system-wide certificate validation.
