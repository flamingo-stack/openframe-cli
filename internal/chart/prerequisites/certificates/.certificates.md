Provides SSL certificate management functionality using mkcert for local HTTPS development. Handles cross-platform installation of mkcert and generates localhost certificates with proper trust store integration.

## Key Components

- **CertificateInstaller**: Main struct that manages mkcert installation and certificate generation
- **commandExists()**: Checks if a command is available in PATH
- **fileExists()**: Verifies file existence at given path
- **isMkcertInstalled()**: Validates mkcert tool availability
- **areCertificatesGenerated()**: Checks for existing localhost certificates
- **certificateInstallHelp()**: Returns platform-specific installation guidance

## Usage Example

```go
// Create certificate installer
installer := certificates.NewCertificateInstaller()

// Check if mkcert is available
if !installer.IsInstalled() {
    fmt.Println(installer.GetInstallHelp())
    
    // Install mkcert and generate certificates
    if err := installer.Install(); err != nil {
        log.Fatal("Failed to install certificates:", err)
    }
}

// Force regenerate certificates even if they exist
if err := installer.ForceRegenerate(); err != nil {
    log.Fatal("Failed to regenerate certificates:", err)
}
```

The installer automatically handles platform-specific mkcert installation (Homebrew on macOS, direct download on Linux) and generates trusted localhost certificates in `~/.config/openframe/certs/`. On macOS and Linux, it integrates with system trust stores and browser NSS databases for seamless HTTPS development.
