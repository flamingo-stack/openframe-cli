Cross-platform SSL certificate management utility that automates mkcert installation and localhost certificate generation for HTTPS development environments.

## Key Components

- **CertificateInstaller**: Main struct providing certificate management functionality
- **Platform Detection**: Runtime OS detection for platform-specific installation methods
- **Certificate Generation**: Automated localhost certificate creation with proper trust store integration
- **Trust Store Management**: Platform-specific certificate trust handling for browsers and system stores

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := NewCertificateInstaller()
    
    // Check if mkcert is already installed
    if !installer.IsInstalled() {
        fmt.Println(installer.GetInstallHelp())
        
        // Install mkcert and generate certificates
        if err := installer.Install(); err != nil {
            log.Fatal("Installation failed:", err)
        }
    }
    
    // Force regenerate certificates (useful for updates)
    if err := installer.ForceRegenerate(); err != nil {
        log.Fatal("Certificate generation failed:", err)
    }
    
    fmt.Println("Certificates ready at ~/.config/openframe/certs/")
}
```

The installer handles platform-specific mkcert installation (Homebrew on macOS, direct download on Linux, manual on Windows) and manages certificate trust stores including NSS databases for Firefox and system certificate stores. Generated certificates are stored in `~/.config/openframe/certs/` and include localhost, 127.0.0.1, and ::1 addresses.
