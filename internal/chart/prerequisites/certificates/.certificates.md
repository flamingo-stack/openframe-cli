This package provides cross-platform SSL certificate management using mkcert for local development environments. It automatically installs mkcert and generates localhost certificates with proper system trust store integration.

## Key Components

- **CertificateInstaller**: Main struct that handles mkcert installation and certificate generation
- **NewCertificateInstaller()**: Constructor function
- **IsInstalled()**: Checks if mkcert is available
- **Install()**: Installs mkcert (if needed) and generates certificates
- **ForceRegenerate()**: Regenerates certificates even if they exist
- **GetInstallHelp()**: Returns platform-specific installation guidance

Helper functions include `commandExists()`, `fileExists()`, and `areCertificatesGenerated()` for system checks.

## Usage Example

```go
// Create installer instance
installer := certificates.NewCertificateInstaller()

// Check if mkcert is installed
if !installer.IsInstalled() {
    fmt.Println(installer.GetInstallHelp())
    
    // Install mkcert and generate certificates
    if err := installer.Install(); err != nil {
        log.Fatal("Failed to install certificates:", err)
    }
}

// Force regenerate certificates
if err := installer.ForceRegenerate(); err != nil {
    log.Fatal("Failed to regenerate certificates:", err)
}
```

The package handles platform-specific installation (Homebrew on macOS, direct download on Linux) and integrates certificates with system trust stores and browser NSS databases. Generated certificates are stored in `~/.config/openframe/certs/`.
