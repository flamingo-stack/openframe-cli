This package provides cross-platform SSL certificate management using mkcert for local development environments. It handles automatic installation of mkcert and generates trusted localhost certificates for HTTPS development.

## Key Components

- **CertificateInstaller**: Main struct providing certificate installation and management functionality
- **NewCertificateInstaller()**: Constructor for creating a new installer instance
- **IsInstalled()**: Checks if mkcert is available on the system
- **Install()**: Installs mkcert (if needed) and generates certificates
- **ForceRegenerate()**: Regenerates certificates even if they already exist
- **GetInstallHelp()**: Returns platform-specific installation instructions

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "certificates"
)

func main() {
    installer := certificates.NewCertificateInstaller()
    
    // Check if certificates are ready
    if !installer.IsInstalled() {
        fmt.Println("Installing certificates...")
        if err := installer.Install(); err != nil {
            log.Fatal(err)
        }
    }
    
    // Force regeneration if needed
    if err := installer.ForceRegenerate(); err != nil {
        log.Printf("Failed to regenerate: %v", err)
    }
    
    // Get platform-specific help
    fmt.Println(installer.GetInstallHelp())
}
```

The installer automatically handles platform differences (macOS via Homebrew, Linux via direct download) and manages system trust stores for browser compatibility. Certificates are stored in `~/.config/openframe/certs/`.
