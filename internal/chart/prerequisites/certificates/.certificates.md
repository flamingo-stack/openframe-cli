This package provides cross-platform SSL certificate installation and management using mkcert for local development environments.

## Key Components

- **CertificateInstaller** - Main struct that handles certificate installation and management
- **NewCertificateInstaller()** - Constructor function that returns a new installer instance
- **IsInstalled()** - Checks if mkcert is available on the system
- **Install()** - Installs mkcert and generates localhost certificates
- **ForceRegenerate()** - Regenerates certificates even if they already exist
- **GetInstallHelp()** - Returns platform-specific installation instructions

## Platform Support

The installer provides different installation strategies:
- **macOS**: Uses Homebrew to install mkcert
- **Linux**: Downloads mkcert binary directly and handles NSS/system trust stores
- **Windows**: Requires manual installation with helpful error messages

## Certificate Management

Certificates are stored in `~/.config/openframe/certs/` and include:
- `localhost.pem` - Certificate file
- `localhost-key.pem` - Private key file

The installer automatically handles platform-specific trust store integration, including macOS Keychain and Linux NSS databases for browser compatibility.

## Usage Example

```go
installer := certificates.NewCertificateInstaller()

// Check if mkcert is available
if !installer.IsInstalled() {
    fmt.Println(installer.GetInstallHelp())
    // Install mkcert and generate certificates
    if err := installer.Install(); err != nil {
        log.Fatal(err)
    }
}

// Force regenerate certificates
if err := installer.ForceRegenerate(); err != nil {
    log.Fatal(err)
}
```
