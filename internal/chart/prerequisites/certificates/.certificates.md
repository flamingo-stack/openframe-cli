This package provides cross-platform SSL certificate generation and installation using mkcert for local HTTPS development. It handles platform-specific installation methods and manages certificate trust stores for browsers.

## Key Components

- **`CertificateInstaller`** - Main struct that implements certificate management functionality
- **`NewCertificateInstaller()`** - Constructor function returning a new installer instance
- **`IsInstalled()`** - Checks if mkcert is available on the system
- **`Install()`** - Installs mkcert and generates certificates if needed
- **`ForceRegenerate()`** - Regenerates certificates even if they already exist
- **`GetInstallHelp()`** - Returns platform-specific installation guidance

### Helper Functions

- **`commandExists()`** - Verifies if a command is available in PATH
- **`fileExists()`** - Checks if a file exists at the given path
- **`isMkcertInstalled()`** - Specifically checks for mkcert availability
- **`areCertificatesGenerated()`** - Verifies if certificates exist in expected location

## Usage Example

```go
package main

import (
    "fmt"
    "log"
    "your-project/certificates"
)

func main() {
    installer := certificates.NewCertificateInstaller()
    
    // Check if mkcert is installed
    if !installer.IsInstalled() {
        fmt.Println("Mkcert not found. Installation help:")
        fmt.Println(installer.GetInstallHelp())
        
        // Install mkcert and generate certificates
        if err := installer.Install(); err != nil {
            log.Fatalf("Failed to install: %v", err)
        }
    }
    
    // Force regenerate certificates
    if err := installer.ForceRegenerate(); err != nil {
        log.Fatalf("Failed to regenerate certificates: %v", err)
    }
    
    fmt.Println("Certificates ready at ~/.config/openframe/certs/")
}
```

The installer automatically handles platform differences - using Homebrew on macOS, direct downloads on Linux, and providing manual instructions for Windows. Generated certificates include localhost, 127.0.0.1, and IPv6 loopback support.
