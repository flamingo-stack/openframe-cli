This package provides cross-platform SSL certificate management using mkcert for local development environments. It handles automatic installation of mkcert and generation of localhost certificates for secure local development.

## Key Components

- **CertificateInstaller**: Main struct that manages certificate installation and generation
- **NewCertificateInstaller()**: Factory function to create a new installer instance
- **Install()**: Installs mkcert (if needed) and generates certificates
- **ForceRegenerate()**: Regenerates certificates even if they already exist
- **IsInstalled()**: Checks if mkcert is available on the system
- **GetInstallHelp()**: Returns platform-specific installation instructions

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := certificates.NewCertificateInstaller()
    
    // Check if mkcert is installed
    if !installer.IsInstalled() {
        fmt.Println(installer.GetInstallHelp())
        
        // Install mkcert and generate certificates
        if err := installer.Install(); err != nil {
            log.Fatal("Installation failed:", err)
        }
    }
    
    // Force regenerate certificates if needed
    if err := installer.ForceRegenerate(); err != nil {
        log.Fatal("Certificate regeneration failed:", err)
    }
    
    fmt.Println("Certificates ready for localhost development")
}
```

The installer automatically handles platform-specific certificate trust store integration (macOS Keychain, Linux NSS databases) and creates certificates in `~/.config/openframe/certs/`.
