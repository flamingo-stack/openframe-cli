This file contains comprehensive unit tests for the certificate installation functionality, validating certificate installer creation, help message generation, and cross-platform mkcert installation logic.

## Key Components

- **TestNewCertificateInstaller**: Tests creation of certificate installer instances
- **TestCertificateInstaller_GetInstallHelp**: Validates platform-specific installation help messages (macOS/Homebrew, Linux/download, Windows/manual)
- **TestCertificateInstaller_Install**: Basic installer structure validation without performing actual installations
- **TestAreCertificatesGenerated**: Tests certificate detection logic
- **TestIsMkcertInstalled**: Validates mkcert installation detection
- **TestInstallMkcert**: Tests mkcert installation with graceful error handling
- **containsSubstring**: Helper function for string matching in test assertions

## Usage Example

```go
// Run all certificate tests
go test ./certificates

// Run specific test
go test -run TestNewCertificateInstaller ./certificates

// Test with verbose output to see platform-specific help messages
go test -v ./certificates

// Example of what the tests validate:
installer := NewCertificateInstaller()
help := installer.GetInstallHelp()
// Ensures help contains "mkcert" and platform-specific instructions

// Tests handle cross-platform scenarios gracefully
err := installer.installMkcert()
// Validates proper error handling for unsupported environments
```

The tests focus on validation logic rather than actual certificate installation to avoid environment dependencies and slow execution times.
