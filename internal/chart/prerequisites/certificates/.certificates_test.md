This file contains unit tests for the certificate management system, specifically testing the CertificateInstaller functionality across different operating systems.

## Key Components

- **TestNewCertificateInstaller** - Tests creation of certificate installer instances
- **TestCertificateInstaller_GetInstallHelp** - Validates platform-specific installation help text
- **TestCertificateInstaller_Install** - Tests basic installer functionality without actual installation
- **TestAreCertificatesGenerated** - Tests certificate detection logic
- **TestIsMkcertInstalled** - Validates mkcert installation detection
- **TestInstallMkcert** - Tests mkcert installation process with error handling
- **containsSubstring** - Helper function for string matching in tests

## Usage Example

```go
// Run all certificate installer tests
go test ./certificates -v

// Run specific test
go test ./certificates -run TestNewCertificateInstaller

// Test with coverage
go test ./certificates -cover

// The tests validate cross-platform behavior:
// - macOS: Checks for Homebrew references
// - Linux: Validates download instructions  
// - Windows: Ensures manual installation guidance
```

The tests are designed to be safe for CI/CD environments, avoiding actual certificate installation while validating the installer's logic and error handling across platforms.
