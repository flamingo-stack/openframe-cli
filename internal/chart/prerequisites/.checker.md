Provides a centralized system for checking and managing prerequisite dependencies required by the OpenFrame CLI, including Git, Helm, system memory, and certificates.

## Key Components

- **`PrerequisiteChecker`** - Main struct that manages a collection of prerequisite requirements
- **`Requirement`** - Defines a single prerequisite with name, command, installation check, and help instructions
- **`NewPrerequisiteChecker()`** - Factory function that creates a checker with all standard prerequisites
- **`CheckAll()`** - Validates all prerequisites and returns missing tools
- **`GetInstallInstructions()`** - Retrieves installation help for specific missing tools
- **`CheckPrerequisites()`** - Convenience function for automated prerequisite checking and installation

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing prerequisites: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function for automated checking
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```

The checker validates Git, Helm, system memory requirements, and certificate installations through modular checker implementations in their respective packages.
