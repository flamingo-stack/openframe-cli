Provides a centralized system for checking and managing prerequisites required by the OpenFrame CLI, including Git, Helm, memory requirements, and certificates.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of system requirements
- **Requirement**: Defines a prerequisite with name, command, installation check function, and help instructions
- **NewPrerequisiteChecker()**: Factory function that initializes checker with predefined requirements (Git, Helm, Memory, Certificates)
- **CheckAll()**: Validates all prerequisites and returns missing tools
- **GetInstallInstructions()**: Retrieves installation help for missing prerequisites
- **CheckPrerequisites()**: Convenience function for complete prerequisite validation

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    
    for i, tool := range missing {
        fmt.Printf("Missing: %s\n", tool)
        fmt.Printf("Install: %s\n\n", instructions[i])
    }
}

// Simple check using convenience function
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites not met:", err)
}
```
