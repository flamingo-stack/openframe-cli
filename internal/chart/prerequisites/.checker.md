A unified prerequisite checking system that validates required tools and dependencies for the OpenFrame CLI installation and operation.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of system requirements
- **Requirement**: Defines a prerequisite with name, command, installation check, and help instructions
- **NewPrerequisiteChecker()**: Factory function that initializes checker with Git, Helm, Memory, and Certificate requirements
- **CheckAll()**: Validates all prerequisites and returns status with list of missing tools
- **GetInstallInstructions()**: Provides installation help for missing prerequisites
- **CheckPrerequisites()**: Convenience function for complete prerequisite validation

## Usage Example

```go
// Create and use prerequisite checker
checker := NewPrerequisiteChecker()

// Check all requirements
allPresent, missing := checker.CheckAll()
if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use convenience function
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```

The checker validates Git (version control), Helm (Kubernetes package manager), system memory requirements, and SSL certificates needed for OpenFrame deployment.
