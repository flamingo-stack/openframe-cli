Provides a unified prerequisite validation system that checks for required system dependencies (Git, Helm, memory, certificates) before chart operations. The checker evaluates each requirement and provides installation guidance for missing components.

## Key Components

- **PrerequisiteChecker**: Main struct that manages and validates system requirements
- **Requirement**: Struct defining each prerequisite with name, command, validation function, and install help
- **NewPrerequisiteChecker()**: Factory function that initializes checker with all required dependencies
- **CheckAll()**: Validates all prerequisites and returns missing items
- **GetInstallInstructions()**: Retrieves installation help for specific missing tools
- **CheckPrerequisites()**: Convenience function for automated checking and installation

## Usage Example

```go
// Check all prerequisites
checker := prerequisites.NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Quick check with automatic installation attempt
if err := prerequisites.CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
