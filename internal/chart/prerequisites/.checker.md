A centralized prerequisite checker that validates system requirements for OpenFrame CLI installation, including Git, Helm, memory, and SSL certificates.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of system requirements
- **Requirement**: Struct defining each prerequisite with name, command, installation check, and help instructions
- **NewPrerequisiteChecker()**: Factory function that initializes checker with predefined requirements (Git, Helm, Memory, Certificates)
- **CheckAll()**: Validates all requirements and returns missing tools list
- **GetInstallInstructions()**: Provides installation help for missing prerequisites
- **CheckPrerequisites()**: Convenience function for complete prerequisite validation

## Usage Example

```go
// Create prerequisite checker
checker := prerequisites.NewPrerequisiteChecker()

// Check all requirements
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := prerequisites.CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```

The checker uses a modular design where each prerequisite (Git, Helm, Memory, Certificates) is implemented in separate packages with consistent interfaces for installation detection and help instructions.
