Provides a centralized system for checking and managing prerequisites required by the OpenFrame CLI, including Git, Helm, memory requirements, and certificates.

## Key Components

**PrerequisiteChecker**: Main struct that manages a collection of requirements and provides validation functionality.

**Requirement**: Defines the structure for each prerequisite with name, command, installation check, and help instructions.

**NewPrerequisiteChecker()**: Factory function that creates a checker pre-configured with Git, Helm, memory, and certificate requirements.

**CheckAll()**: Validates all prerequisites and returns success status plus list of missing tools.

**GetInstallInstructions()**: Retrieves installation help for specific missing prerequisites.

**CheckPrerequisites()**: Convenience function that performs full prerequisite check and installation.

## Usage Example

```go
// Check all prerequisites
checker := prerequisites.NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing prerequisites: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := prerequisites.CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
