A centralized prerequisite validation system that checks for required tools and provides installation guidance for the OpenFrame CLI chart deployment process.

## Key Components

- **PrerequisiteChecker**: Main struct that orchestrates validation of multiple system requirements
- **Requirement**: Configuration structure defining each prerequisite with validation and help functions
- **NewPrerequisiteChecker()**: Factory function that initializes checker with Git, Helm, Memory, and Certificate requirements
- **CheckAll()**: Validates all prerequisites and returns status with list of missing tools
- **GetInstallInstructions()**: Retrieves installation help text for specific missing tools
- **CheckPrerequisites()**: Convenience function for complete check and install workflow

## Usage Example

```go
// Check all prerequisites
checker := prerequisites.NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := prerequisites.CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```

The checker integrates with specialized validators for Git availability, Helm installation, system memory requirements, and certificate configuration to ensure a complete deployment environment.
