Validates system prerequisites for Helm chart deployment by checking required tools and providing installation guidance. The checker verifies Git, Helm, memory requirements, and certificates before allowing chart operations.

## Key Components

- **PrerequisiteChecker**: Main struct that manages prerequisite validation
- **Requirement**: Defines a prerequisite with installation check and help functions
- **NewPrerequisiteChecker()**: Factory function that initializes checker with default requirements
- **CheckAll()**: Validates all prerequisites and returns status with missing tools
- **GetInstallInstructions()**: Provides installation help for missing prerequisites
- **CheckPrerequisites()**: Convenience function for complete prerequisite check and install

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing prerequisites: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Quick check and install
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
