A prerequisite validation system that checks for required tools (Git, Helm, Memory, Certificates) and provides installation guidance when dependencies are missing.

## Key Components

- **PrerequisiteChecker**: Main struct that manages multiple prerequisite requirements
- **Requirement**: Defines a single prerequisite with name, command, installation check, and help instructions
- **NewPrerequisiteChecker()**: Factory function that creates a checker with predefined requirements
- **CheckAll()**: Validates all prerequisites and returns missing tools
- **GetInstallInstructions()**: Retrieves installation help for specific missing tools
- **CheckPrerequisites()**: Convenience function for complete prerequisite validation

## Usage Example

```go
// Create and use prerequisite checker
checker := NewPrerequisiteChecker()

// Check all prerequisites
allInstalled, missing := checker.CheckAll()
if !allInstalled {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions for missing tools
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Or use the convenience function
if err := CheckPrerequisites(); err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
