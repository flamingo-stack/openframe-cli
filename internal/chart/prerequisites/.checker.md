This file provides a centralized system for checking and validating prerequisites required for chart operations. It orchestrates checks for Git, Helm, memory requirements, and certificates through a unified interface.

## Key Components

- **PrerequisiteChecker**: Main struct that manages a collection of requirements and provides validation methods
- **Requirement**: Struct defining each prerequisite with name, command, installation check, and help functions
- **NewPrerequisiteChecker()**: Factory function that initializes the checker with predefined requirements (Git, Helm, Memory, Certificates)
- **CheckAll()**: Validates all requirements and returns status with missing tools list
- **GetInstallInstructions()**: Retrieves installation help for specific missing tools
- **CheckPrerequisites()**: Convenience function for automated checking and installation

## Usage Example

```go
// Check all prerequisites
checker := NewPrerequisiteChecker()
allPresent, missing := checker.CheckAll()

if !allPresent {
    fmt.Printf("Missing tools: %v\n", missing)
    
    // Get installation instructions
    instructions := checker.GetInstallInstructions(missing)
    for _, instruction := range instructions {
        fmt.Println(instruction)
    }
}

// Quick check with auto-install
err := CheckPrerequisites()
if err != nil {
    log.Fatal("Prerequisites check failed:", err)
}
```
