This file contains unit tests for the prerequisites installer functionality, testing installer creation, tool installation behavior, and command execution.

## Key Components

- **TestNewInstaller**: Validates that the installer is properly initialized with required components
- **TestInstallTool**: Tests tool installation behavior for edge cases (memory tool, unknown tools)
- **TestRunCommand**: Verifies command execution functionality using a simple echo command
- **containsSubstring**: Helper function for substring matching in error messages

## Usage Example

```go
// Run all tests
go test -v

// Run specific test
go test -run TestNewInstaller

// The tests verify installer behavior without performing actual installations:
func TestInstallTool(t *testing.T) {
    installer := NewInstaller()
    
    // Memory tool should always fail
    err := installer.installTool("memory")
    if err == nil {
        t.Error("Expected error for memory tool")
    }
    
    // Unknown tools should return specific error
    err = installer.installTool("unknown-tool")
    expectedError := "unknown tool: unknown-tool"
    if err.Error() != expectedError {
        t.Errorf("Expected error '%s', got '%s'", expectedError, err.Error())
    }
}
```
