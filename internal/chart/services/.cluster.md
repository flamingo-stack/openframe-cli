Handles cluster selection logic for Helm chart operations, providing an interactive interface to choose from available Kubernetes clusters.

## Key Components

- **ClusterSelector**: Main service that orchestrates cluster selection workflow
- **NewClusterSelector()**: Constructor function that initializes the selector with required dependencies
- **SelectCluster()**: Core method that lists available clusters and handles user selection

## Usage Example

```go
// Initialize cluster selector with dependencies
clusterService := &myClusterService{} // implements types.ClusterLister
operationsUI := chartUI.NewOperationsUI()
selector := NewClusterSelector(clusterService, operationsUI)

// Select cluster for deployment
clusterName, err := selector.SelectCluster(os.Args[1:], true)
if err != nil {
    log.Fatalf("Cluster selection failed: %v", err)
}

if clusterName == "" {
    log.Println("No cluster selected or available")
    return
}

log.Printf("Selected cluster: %s", clusterName)
```

The service handles error cases gracefully, displaying appropriate messages when no clusters are available or when listing fails. Verbose mode provides detailed logging of the selection process and available clusters with their status information.
