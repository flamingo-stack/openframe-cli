This file provides cluster selection functionality for chart operations, handling the logic to list available clusters and present them to users for selection.

## Key Components

- **ClusterSelector** - Main struct that manages cluster selection workflow
- **NewClusterSelector()** - Constructor function that initializes a new cluster selector with required dependencies
- **SelectCluster()** - Core method that orchestrates the cluster selection process, including listing clusters, error handling, and user interaction

## Usage Example

```go
// Initialize cluster selector with dependencies
clusterService := // ... your cluster service implementation
operationsUI := chartUI.NewOperationsUI()
selector := NewClusterSelector(clusterService, operationsUI)

// Select a cluster for operations
args := []string{"my-cluster"} // optional cluster name argument
verbose := true
clusterName, err := selector.SelectCluster(args, verbose)
if err != nil {
    log.Fatal(err)
}

if clusterName != "" {
    fmt.Printf("Selected cluster: %s\n", clusterName)
} else {
    fmt.Println("No cluster selected or available")
}
```

The selector handles cases where no clusters exist, provides verbose logging when enabled, and delegates the actual selection UI to the operations interface.
