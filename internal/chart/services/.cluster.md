This service handles cluster selection logic for chart operations, providing a unified interface for listing and selecting Kubernetes clusters.

## Key Components

- **ClusterSelector**: Main struct that orchestrates cluster selection workflow
- **NewClusterSelector()**: Constructor function that initializes the selector with required dependencies
- **SelectCluster()**: Primary method that manages the complete cluster selection process including listing, validation, and user interaction

## Usage Example

```go
// Initialize cluster selector
clusterService := &myClusterService{} // implements types.ClusterLister
operationsUI := chartUI.NewOperationsUI()
selector := NewClusterSelector(clusterService, operationsUI)

// Select a cluster for chart operations
args := []string{"my-cluster"} // optional cluster name argument
selectedCluster, err := selector.SelectCluster(args, true)
if err != nil {
    log.Fatal(err)
}

if selectedCluster != "" {
    fmt.Printf("Selected cluster: %s\n", selectedCluster)
} else {
    fmt.Println("No cluster selected or available")
}
```

The service automatically handles scenarios where no clusters are available, provides verbose logging when enabled, and delegates the actual selection UI to the operations interface.
