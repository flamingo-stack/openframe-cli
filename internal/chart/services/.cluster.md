Provides cluster selection functionality for chart operations, handling the retrieval and interactive selection of available clusters with verbose logging support.

## Key Components

- **ClusterSelector**: Main struct that orchestrates cluster selection logic
- **NewClusterSelector()**: Constructor function for creating a new cluster selector instance
- **SelectCluster()**: Core method that manages the complete cluster selection workflow

## Usage Example

```go
// Initialize cluster selector with dependencies
clusterService := // your cluster service implementation
operationsUI := chartUI.NewOperationsUI()
selector := services.NewClusterSelector(clusterService, operationsUI)

// Select cluster with command line arguments and verbose output
selectedCluster, err := selector.SelectCluster([]string{"my-cluster"}, true)
if err != nil {
    log.Fatal(err)
}

if selectedCluster != "" {
    fmt.Printf("Selected cluster: %s\n", selectedCluster)
} else {
    fmt.Println("No cluster selected or available")
}
```

The selector automatically handles cases where no clusters are available, displays appropriate messages through the UI, and provides detailed logging when verbose mode is enabled. It returns an empty string when no cluster is selected or available, making it safe to use in conditional logic.
