Handles cluster selection logic for chart operations with UI integration and verbose logging capabilities.

## Key Components

- **ClusterSelector** - Main service struct that orchestrates cluster selection workflow
- **NewClusterSelector()** - Constructor function that initializes the selector with required dependencies
- **SelectCluster()** - Core method that manages the complete cluster selection process including listing, filtering, and UI interaction

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/services"
    "github.com/flamingo-stack/openframe-cli/internal/chart/ui"
)

// Initialize dependencies
clusterService := myClusterLister // implements types.ClusterLister
operationsUI := chartUI.NewOperationsUI()

// Create cluster selector
selector := services.NewClusterSelector(clusterService, operationsUI)

// Select a cluster with command line arguments and verbose output
selectedCluster, err := selector.SelectCluster([]string{"my-cluster"}, true)
if err != nil {
    // Handle selection error
}

if selectedCluster != "" {
    // Proceed with selected cluster
    fmt.Printf("Selected cluster: %s\n", selectedCluster)
}
```

The service handles error cases gracefully by showing appropriate UI messages when no clusters are available or when listing fails, making it suitable for interactive CLI applications.
