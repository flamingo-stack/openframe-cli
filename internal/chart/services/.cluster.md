This file provides cluster selection functionality for chart operations in the OpenFrame CLI, handling cluster listing, validation, and interactive selection through a clean service abstraction.

## Key Components

- **ClusterSelector**: Main service struct that orchestrates cluster selection workflow
- **NewClusterSelector()**: Constructor function that initializes the selector with required dependencies
- **SelectCluster()**: Primary method that manages the complete cluster selection process including listing, validation, and user interaction

## Usage Example

```go
// Initialize cluster selector with dependencies
clusterService := // your cluster listing implementation
operationsUI := chartUI.NewOperationsUI()
selector := NewClusterSelector(clusterService, operationsUI)

// Select cluster with optional arguments and verbose output
selectedCluster, err := selector.SelectCluster([]string{"my-cluster"}, true)
if err != nil {
    log.Fatal(err)
}

if selectedCluster != "" {
    fmt.Printf("Selected cluster: %s\n", selectedCluster)
} else {
    fmt.Println("No cluster selected or available")
}
```

The service handles error cases gracefully by displaying appropriate UI messages when no clusters are found or when listing fails, making it suitable for both automated and interactive use cases.
