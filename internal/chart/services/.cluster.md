Manages cluster selection workflow for chart deployment operations, providing a unified interface for listing and selecting target clusters.

## Key Components

- **ClusterSelector** - Main service struct that orchestrates cluster selection logic
- **NewClusterSelector()** - Constructor function that initializes the selector with required dependencies
- **SelectCluster()** - Core method that handles the complete cluster selection workflow including listing, validation, and user interaction

## Usage Example

```go
// Initialize the cluster selector
clusterService := myClusterService // implements types.ClusterLister
operationsUI := chartUI.NewOperationsUI()
selector := NewClusterSelector(clusterService, operationsUI)

// Select a cluster for deployment
args := []string{"prod-cluster"} // optional cluster name argument
verbose := true

selectedCluster, err := selector.SelectCluster(args, verbose)
if err != nil {
    log.Fatal(err)
}

if selectedCluster != "" {
    fmt.Printf("Selected cluster: %s\n", selectedCluster)
} else {
    fmt.Println("No cluster selected or available")
}
```

The service handles error scenarios gracefully, displaying appropriate messages when no clusters are available or when listing fails.
