Manages ArgoCD deployment and lifecycle operations through Helm integration, providing installation, status monitoring, and application synchronization capabilities for the OpenFrame platform.

## Key Components

- **ArgoCD struct**: Main service managing ArgoCD operations with Helm, path resolution, and command execution
- **NewArgoCD()**: Constructor that creates a non-verbose executor for internal ArgoCD operations
- **Install()**: Deploys ArgoCD using Helm with progress tracking and error handling
- **WaitForApplications()**: Monitors ArgoCD applications until they reach ready state
- **IsInstalled()**: Checks if ArgoCD is currently deployed in the cluster
- **GetStatus()**: Retrieves current installation status and chart information

## Usage Example

```go
// Initialize ArgoCD service
helmManager := helm.NewHelmManager(executor)
pathResolver := config.NewPathResolver()
executor := executor.NewRealCommandExecutor(true, true)

argoCDService := NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
ctx := context.Background()
cfg := config.ChartInstallConfig{
    ClusterName: "production",
    Verbose:     true,
}

err := argoCDService.Install(ctx, cfg)
if err != nil {
    log.Fatalf("ArgoCD installation failed: %v", err)
}

// Wait for applications to be ready
err = argoCDService.WaitForApplications(ctx, cfg)
if err != nil {
    log.Printf("Application sync timeout: %v", err)
}

// Check status
status, err := argoCDService.GetStatus(ctx)
```
