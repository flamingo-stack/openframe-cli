Service for managing ArgoCD installation and lifecycle operations using Helm and ArgoCD-specific management capabilities.

## Key Components

- **`ArgoCD`** - Main service struct that orchestrates ArgoCD operations through Helm and ArgoCD managers
- **`NewArgoCD()`** - Constructor that initializes the service with non-verbose command execution for internal operations
- **`Install()`** - Installs or upgrades ArgoCD using Helm with progress indication
- **`WaitForApplications()`** - Waits for all ArgoCD applications to reach ready state with configurable verbosity
- **`IsInstalled()`** - Checks if ArgoCD is currently installed in the cluster
- **`GetStatus()`** - Retrieves current status information for the ArgoCD installation

## Usage Example

```go
// Initialize the ArgoCD service
helmManager := helm.NewHelmManager(executor)
pathResolver := config.NewPathResolver()
executor := executor.NewRealCommandExecutor(true, true)

argoCDService := NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose:     true,
}

err := argoCDService.Install(ctx, config)
if err != nil {
    log.Fatal("Failed to install ArgoCD:", err)
}

// Wait for applications to be ready
err = argoCDService.WaitForApplications(ctx, config)
if err != nil {
    log.Fatal("ArgoCD applications not ready:", err)
}
```

The service automatically handles error wrapping with cluster context and uses recoverable errors for waiting operations with a 60-second timeout.
