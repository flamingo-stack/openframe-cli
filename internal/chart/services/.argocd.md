This file provides a service layer for managing ArgoCD installations through Helm, handling installation, status checking, and application readiness waiting.

## Key Components

- **ArgoCD struct**: Main service that orchestrates ArgoCD operations using Helm and ArgoCD managers
- **NewArgoCD()**: Constructor that initializes the service with required dependencies
- **Install()**: Installs ArgoCD using Helm with progress indication
- **WaitForApplications()**: Waits for all ArgoCD applications to become ready
- **IsInstalled()**: Checks if ArgoCD is currently installed in the cluster
- **GetStatus()**: Retrieves current status information about the ArgoCD installation

## Usage Example

```go
// Initialize the service
helmManager := helm.NewHelmManager(executor)
pathResolver := config.NewPathResolver()
argoCDService := NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose:     true,
}

ctx := context.Background()
if err := argoCDService.Install(ctx, config); err != nil {
    log.Fatal(err)
}

// Wait for applications to be ready
if err := argoCDService.WaitForApplications(ctx, config); err != nil {
    log.Fatal(err)
}

// Check installation status
installed, err := argoCDService.IsInstalled(ctx)
if err != nil {
    log.Fatal(err)
}
```
