This service provides high-level orchestration for ArgoCD installation and management operations, wrapping lower-level Helm and ArgoCD managers with error handling and progress reporting.

## Key Components

- **ArgoCD struct**: Main service containing Helm manager, path resolver, and ArgoCD manager dependencies
- **NewArgoCD()**: Factory function that creates a non-verbose command executor for internal ArgoCD operations
- **Install()**: Installs or upgrades ArgoCD using Helm with progress indication and error wrapping
- **WaitForApplications()**: Waits for all ArgoCD applications to become ready with configurable verbosity
- **IsInstalled()**: Checks if ArgoCD is currently installed via Helm chart detection
- **GetStatus()**: Retrieves current installation status and chart information

## Usage Example

```go
// Create ArgoCD service
helmManager := helm.NewManager(executor)
pathResolver := config.NewPathResolver()
argoCDService := services.NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose:     true,
}

err := argoCDService.Install(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Wait for applications to be ready
err = argoCDService.WaitForApplications(ctx, config)
if err != nil {
    log.Printf("Applications not ready: %v", err)
}

// Check status
status, err := argoCDService.GetStatus(ctx)
if err == nil {
    fmt.Printf("ArgoCD version: %s\n", status.Version)
}
```
