This service provides high-level management of ArgoCD installations through Helm, handling installation, status monitoring, and application synchronization.

## Key Components

- **ArgoCD struct**: Main service that orchestrates ArgoCD operations using Helm and ArgoCD managers
- **NewArgoCD()**: Constructor that initializes the service with required dependencies
- **Install()**: Installs or upgrades ArgoCD using Helm with progress indication
- **WaitForApplications()**: Waits for all ArgoCD applications to reach ready state
- **IsInstalled()**: Checks if ArgoCD is currently installed in the cluster
- **GetStatus()**: Retrieves detailed status information about the ArgoCD installation

## Usage Example

```go
// Initialize the service
helmManager := helm.NewHelmManager(executor)
pathResolver := config.NewPathResolver()
argoCDService := NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
ctx := context.Background()
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose:     true,
}

err := argoCDService.Install(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Wait for applications to be ready
err = argoCDService.WaitForApplications(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Check installation status
status, err := argoCDService.GetStatus(ctx)
```
