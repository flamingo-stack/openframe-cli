This service provides high-level ArgoCD management operations, wrapping Helm and ArgoCD-specific functionality for installation and monitoring within the OpenFrame CLI.

## Key Components

- **ArgoCD**: Main service struct that orchestrates ArgoCD operations through Helm and ArgoCD managers
- **NewArgoCD()**: Constructor that creates a new ArgoCD service with non-verbose command execution
- **Install()**: Installs ArgoCD using Helm with progress indication and error handling
- **WaitForApplications()**: Waits for all ArgoCD applications to reach ready state
- **IsInstalled()**: Checks if ArgoCD is currently installed in the cluster
- **GetStatus()**: Retrieves the current status of the ArgoCD installation

## Usage Example

```go
// Create ArgoCD service
helmManager := helm.NewHelmManager(executor)
pathResolver := config.NewPathResolver()
argoCDService := services.NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose:     true,
}

err := argoCDService.Install(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Wait for applications to be ready
err = argoCDService.WaitForApplications(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Check status
status, err := argoCDService.GetStatus(ctx)
```
