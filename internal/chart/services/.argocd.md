ArgoCD service manages the installation and lifecycle of ArgoCD using Helm charts and provides application waiting capabilities for the OpenFrame platform.

## Key Components

- **ArgoCD struct**: Main service containing Helm manager, path resolver, ArgoCD manager, and command executor
- **NewArgoCD()**: Constructor that creates a new ArgoCD service with a non-verbose executor for internal operations
- **Install()**: Installs or upgrades ArgoCD using Helm with progress indication
- **WaitForApplications()**: Waits for all ArgoCD applications to reach ready state
- **IsInstalled()**: Checks if ArgoCD is currently installed in the cluster
- **GetStatus()**: Returns detailed status information about the ArgoCD installation

## Usage Example

```go
// Create ArgoCD service
helmManager := helm.NewHelmManager(executor, pathResolver)
pathResolver := config.NewPathResolver()
executor := executor.NewRealCommandExecutor(true, false)
argoCDService := NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
cfg := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose:     true,
}
err := argoCDService.Install(ctx, cfg)
if err != nil {
    log.Fatal(err)
}

// Wait for applications to be ready
err = argoCDService.WaitForApplications(ctx, cfg)
if err != nil {
    log.Printf("Warning: %v", err)
}

// Check installation status
installed, err := argoCDService.IsInstalled(ctx)
if err == nil && installed {
    status, _ := argoCDService.GetStatus(ctx)
    fmt.Printf("ArgoCD Status: %+v\n", status)
}
```
