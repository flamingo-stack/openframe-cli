This service manages ArgoCD installation and lifecycle operations within the OpenFrame CLI, providing a higher-level interface for ArgoCD deployment and application management through Helm and dedicated ArgoCD managers.

## Key Components

- **ArgoCD struct**: Main service that orchestrates ArgoCD operations using Helm and ArgoCD-specific managers
- **NewArgoCD()**: Factory function that creates a new ArgoCD service with non-verbose command execution for internal operations
- **Install()**: Installs ArgoCD using Helm with progress tracking and error handling
- **WaitForApplications()**: Waits for all ArgoCD applications to reach ready state with configurable verbosity
- **IsInstalled()**: Checks if ArgoCD is currently installed in the cluster
- **GetStatus()**: Retrieves current status information for the ArgoCD installation

## Usage Example

```go
// Create ArgoCD service
helmManager := helm.NewHelmManager(executor, pathResolver)
pathResolver := config.NewPathResolver()
exec := executor.NewRealCommandExecutor(true, false)
argoCDService := NewArgoCD(helmManager, pathResolver, exec)

// Install ArgoCD
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose:     true,
}

err := argoCDService.Install(ctx, config)
if err != nil {
    log.Fatal("Failed to install ArgoCD:", err)
}

// Wait for applications to be ready
err = argoCDService.WaitForApplications(ctx, config)
if err != nil {
    log.Fatal("Applications not ready:", err)
}

// Check installation status
installed, err := argoCDService.IsInstalled(ctx)
if err != nil {
    log.Fatal("Failed to check status:", err)
}
```
