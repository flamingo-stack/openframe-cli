Provides ArgoCD installation and management capabilities through Helm and native ArgoCD operations, handling the complete lifecycle from installation to application synchronization.

## Key Components

- **`ArgoCD`** - Main service struct that orchestrates ArgoCD operations using Helm and ArgoCD managers
- **`NewArgoCD()`** - Factory function that creates an ArgoCD service with a non-verbose executor for internal operations
- **`Install()`** - Installs ArgoCD using Helm with progress indication and error handling
- **`WaitForApplications()`** - Waits for all ArgoCD applications to reach ready state with configurable timeout
- **`IsInstalled()`** - Checks if ArgoCD is currently installed in the cluster
- **`GetStatus()`** - Retrieves current status information about the ArgoCD installation

## Usage Example

```go
// Create ArgoCD service
helmManager := helm.NewHelmManager(executor, pathResolver)
pathResolver := config.NewPathResolver()
exec := executor.NewRealCommandExecutor(true, true)
argoService := NewArgoCD(helmManager, pathResolver, exec)

// Install ArgoCD
config := config.ChartInstallConfig{
    ClusterName: "production",
    Verbose: true,
}

err := argoService.Install(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Wait for applications to be ready
err = argoService.WaitForApplications(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Check status
status, err := argoService.GetStatus(ctx)
```
