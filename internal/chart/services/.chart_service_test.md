Test suite for chart service functionality that validates chart installation workflows and cluster listing operations.

## Key Components

**MockClusterLister**
- Test implementation of `ClusterLister` interface
- `NewMockClusterLister()` - Creates new mock instance
- `SetClusters(clusters)` - Sets clusters to return
- `SetError(err)` - Sets error to return
- `ListClusters()` - Returns configured clusters or error

**Test Functions**
- `TestNewChartService*` - Tests chart service initialization with different configurations
- `TestInstallationWorkflow*` - Tests workflow creation and request validation
- `TestMockClusterLister*` - Tests mock cluster lister functionality

## Usage Example

```go
// Create a mock cluster lister for testing
mockLister := NewMockClusterLister()
mockLister.SetClusters([]clusterDomain.ClusterInfo{
    {Name: "test-cluster", Status: "running"},
})

// Test chart service creation
service := NewChartService(false, false)
assert.NotNil(t, service)

// Test installation request validation
req := types.InstallationRequest{
    Args:         []string{"test-cluster"},
    GitHubRepo:   "https://github.com/test/repo",
    GitHubBranch: "main",
}

// Create workflow for testing
workflow := &InstallationWorkflow{
    chartService:   service,
    clusterService: mockLister,
}
```

The file provides comprehensive test coverage for chart service initialization, mock cluster operations, and installation request handling.
