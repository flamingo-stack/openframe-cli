Test suite for the chart service module, providing unit tests and mocking capabilities for chart installation workflows.

## Key Components

- **MockClusterLister**: Test double implementing ClusterLister interface for controlled testing scenarios
- **TestNewChartService**: Tests chart service initialization with different configurations (dry-run, verbose)
- **TestInstallationWorkflow**: Tests for installation workflow creation and validation
- **TestMockClusterLister**: Comprehensive tests for mock cluster lister functionality
- **TestInstallationWorkflow_ValidateRequest**: Tests for installation request validation

## Usage Example

```go
// Create a mock cluster lister for testing
mockLister := NewMockClusterLister()
mockLister.SetClusters([]clusterDomain.ClusterInfo{
    {Name: "test-cluster", Status: "running"},
})

// Test chart service creation
service := NewChartService(true, false) // dry-run enabled
assert.NotNil(t, service)

// Test installation request validation
req := types.InstallationRequest{
    Args:         []string{"my-cluster"},
    GitHubRepo:   "https://github.com/my/repo",
    GitHubBranch: "main",
    DryRun:       true,
}

// Mock error scenarios
mockLister.SetError(errors.New("cluster unavailable"))
clusters, err := mockLister.ListClusters()
assert.Error(t, err)
```
