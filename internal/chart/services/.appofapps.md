Provides app-of-apps installation and management functionality, handling Git repository cloning and Helm chart deployment for ArgoCD-style application management.

## Key Components

- **AppOfApps struct**: Main service with dependencies for Helm management, Git operations, and path resolution
- **NewAppOfApps()**: Constructor function for creating new service instances
- **Install()**: Clones a Git repository and installs the app-of-apps Helm chart with validation and cleanup
- **IsInstalled()**: Checks if the app-of-apps chart is currently installed in a namespace
- **GetStatus()**: Retrieves detailed status information about the app-of-apps installation

## Usage Example

```go
// Create service instance
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()

appOfApps := services.NewAppOfApps(helmManager, gitRepo, pathResolver)

// Install app-of-apps
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/myorg/my-apps",
        GitHubBranch: "main",
        ChartPath:    "./charts/app-of-apps",
    },
}

err := appOfApps.Install(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Check installation status
installed, err := appOfApps.IsInstalled(ctx, "argocd")
```
