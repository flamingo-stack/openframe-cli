This service provides app-of-apps installation functionality for OpenFrame, handling the complete workflow of cloning Git repositories and deploying Helm charts for managing multiple applications as a single unit.

## Key Components

- **AppOfApps** - Core service struct that orchestrates app-of-apps installations
- **NewAppOfApps()** - Constructor function that initializes the service with required dependencies
- **Install()** - Main installation method that clones Git repos and deploys charts
- **IsInstalled()** - Checks if an app-of-apps deployment exists in a namespace
- **GetStatus()** - Retrieves current status information for app-of-apps installations

## Usage Example

```go
// Initialize the service
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()
appOfApps := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Install app-of-apps from a Git repository
config := config.ChartInstallConfig{
    ClusterName: "production",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/org/charts",
        GitHubBranch: "main",
        ChartPath:    "app-of-apps",
        ValuesFile:   "values.yaml",
    },
}

ctx := context.Background()
err := appOfApps.Install(ctx, config)
if err != nil {
    log.Fatal(err)
}

// Check installation status
installed, err := appOfApps.IsInstalled(ctx, "argocd")
status, err := appOfApps.GetStatus(ctx, "argocd")
```

The service handles error recovery, temporary directory cleanup, branch validation, and provides detailed logging with optional verbose output for debugging installations.
