Manages ArgoCD app-of-apps installation by cloning Git repositories and deploying Helm charts for GitOps-based application management.

## Key Components

- **AppOfApps**: Main service that orchestrates app-of-apps installation
- **NewAppOfApps()**: Constructor that initializes the service with required dependencies
- **Install()**: Clones a Git repository and installs the app-of-apps chart via Helm
- **IsInstalled()**: Checks if the app-of-apps chart is already deployed
- **GetStatus()**: Retrieves current deployment status and information

## Usage Example

```go
// Initialize the app-of-apps service
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()

appOfApps := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Configure app-of-apps installation
config := config.ChartInstallConfig{
    ClusterName: "production",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/company/k8s-apps",
        GitHubBranch: "main",
        ChartPath:    "./charts/app-of-apps",
        ValuesFile:   "values.yaml",
    },
}

// Install app-of-apps
err := appOfApps.Install(context.Background(), config)
if err != nil {
    log.Fatal(err)
}

// Check installation status
installed, err := appOfApps.IsInstalled(context.Background(), "argocd")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("App-of-apps installed: %v\n", installed)
```

The service handles Git repository cloning, path resolution, and Helm chart deployment with proper error handling and cleanup. It supports branch selection with automatic fallback to `main` branch and provides detailed logging in verbose mode.
