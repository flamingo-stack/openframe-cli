This service manages the installation and status of "app-of-apps" charts by cloning Git repositories and using Helm for deployment.

## Key Components

- **AppOfApps struct**: Main service with dependencies for Helm management, Git operations, and path resolution
- **NewAppOfApps()**: Constructor function for creating new service instances
- **Install()**: Clones a Git repository and installs the app-of-apps chart using Helm
- **IsInstalled()**: Checks if the app-of-apps chart is currently installed
- **GetStatus()**: Retrieves the current status of the app-of-apps installation

## Usage Example

```go
// Create service dependencies
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()

// Initialize the service
appOfApps := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Install app-of-apps from a GitHub repository
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/org/charts",
        GitHubBranch: "main",
        ChartPath:    "charts/app-of-apps",
    },
}

err := appOfApps.Install(context.Background(), config)

// Check installation status
installed, err := appOfApps.IsInstalled(context.Background(), "default")
```
