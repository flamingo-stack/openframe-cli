Manages app-of-apps deployment pattern by orchestrating Git repository cloning and Helm chart installation for multi-application configurations.

## Key Components

- **AppOfApps**: Main service struct that coordinates Git and Helm operations
- **Install()**: Clones GitHub repository and deploys app-of-apps chart via Helm
- **IsInstalled()**: Checks if app-of-apps is currently deployed
- **GetStatus()**: Retrieves deployment status and metadata
- **NewAppOfApps()**: Constructor function for service initialization

## Usage Example

```go
// Initialize the service
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()

appOfApps := services.NewAppOfApps(helmManager, gitRepo, pathResolver)

// Install app-of-apps from GitHub repository
config := config.ChartInstallConfig{
    ClusterName: "production",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/org/app-configs",
        GitHubBranch: "main",
        ChartPath:    "charts/app-of-apps",
        ValuesFile:   "values.yaml",
    },
}

err := appOfApps.Install(context.Background(), config)
if err != nil {
    log.Fatal(err)
}

// Check installation status
installed, err := appOfApps.IsInstalled(context.Background(), "argocd")
```

The service handles temporary directory cleanup, branch validation, and provides detailed error messages for troubleshooting deployment issues.
