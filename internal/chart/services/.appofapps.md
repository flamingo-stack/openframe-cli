Manages app-of-apps pattern installations by cloning charts from GitHub repositories and deploying them using Helm with proper validation and error handling.

## Key Components

### AppOfApps Service
- **AppOfApps**: Main service struct managing app-of-apps installations with Helm, Git, and path resolution dependencies
- **NewAppOfApps()**: Constructor function creating a new app-of-apps service instance

### Core Methods
- **Install()**: Clones chart repository from GitHub and installs using Helm with comprehensive error handling
- **IsInstalled()**: Checks if app-of-apps chart is currently installed in the specified namespace
- **GetStatus()**: Retrieves detailed status information about the app-of-apps installation

## Usage Example

```go
// Initialize dependencies
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()

// Create app-of-apps service
appService := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Configure installation
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    AppOfApps: &models.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/myorg/charts",
        GitHubBranch: "main",
        ChartPath:    "app-of-apps",
        ValuesFile:   "values.yaml",
    },
}

// Install app-of-apps
ctx := context.Background()
err := appService.Install(ctx, config)

// Check installation status
installed, err := appService.IsInstalled(ctx, "argocd")
status, err := appService.GetStatus(ctx, "argocd")
```

The service handles automatic cleanup of temporary directories, branch validation with user-friendly error messages, and supports both verbose and quiet installation modes.
