This service manages the installation and status of "app-of-apps" Helm charts from GitHub repositories, handling the complete workflow from git cloning to Helm deployment.

## Key Components

- **AppOfApps**: Main service struct that orchestrates git cloning and Helm installation
- **NewAppOfApps**: Constructor function that initializes the service with required dependencies
- **Install**: Downloads chart from GitHub repository and installs it via Helm
- **IsInstalled**: Checks if the app-of-apps chart is currently installed
- **GetStatus**: Retrieves current status information for the installed chart

## Usage Example

```go
// Initialize the service
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()
appOfApps := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Configure installation
installConfig := config.ChartInstallConfig{
    ClusterName: "production",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/myorg/charts",
        GitHubBranch: "main",
        ChartPath:    "app-of-apps",
        ValuesFile:   "values.yaml",
    },
}

// Install the app-of-apps chart
ctx := context.Background()
if err := appOfApps.Install(ctx, installConfig); err != nil {
    log.Fatal(err)
}

// Check installation status
isInstalled, err := appOfApps.IsInstalled(ctx, "argocd")
if err != nil {
    log.Fatal(err)
}
```

The service provides automatic cleanup of temporary directories, branch validation with user-friendly error messages, and supports both verbose and quiet installation modes.
