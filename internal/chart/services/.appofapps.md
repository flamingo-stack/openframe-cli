This service handles the installation and management of app-of-apps patterns, which enable managing multiple applications as a single unit by cloning chart repositories from GitHub and installing them via Helm.

## Key Components

- **AppOfApps**: Main service struct that orchestrates git repository cloning and Helm chart installation
- **NewAppOfApps**: Constructor function that initializes the service with required dependencies
- **Install**: Core method that clones a GitHub repository and installs the app-of-apps chart
- **IsInstalled**: Checks if an app-of-apps instance is currently installed
- **GetStatus**: Retrieves the current status of an app-of-apps installation

## Usage Example

```go
// Initialize the service
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()
appOfApps := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Configure installation
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/org/charts",
        GitHubBranch: "main",
        ChartPath:    "./app-of-apps",
        ValuesFile:   "values.yaml",
    },
}

// Install app-of-apps
ctx := context.Background()
err := appOfApps.Install(ctx, config)
if err != nil {
    // Handle installation error
}

// Check installation status
installed, err := appOfApps.IsInstalled(ctx, "default")
if installed {
    status, _ := appOfApps.GetStatus(ctx, "default")
    fmt.Printf("Status: %+v\n", status)
}
```

The service handles git repository cloning, validates configuration, manages temporary directories, and provides detailed error handling including branch-not-found scenarios. It supports verbose logging and automatic cleanup of temporary resources.
