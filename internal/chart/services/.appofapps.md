Manages app-of-apps pattern installations by cloning chart repositories from GitHub and deploying them using Helm. This service handles the complete workflow from repository cloning to chart installation with proper error handling and cleanup.

## Key Components

- **AppOfApps**: Main service struct containing Helm manager, Git repository handler, and path resolver
- **NewAppOfApps()**: Constructor function for creating new app-of-apps service instances
- **Install()**: Core method that clones a GitHub repository and installs the app-of-apps chart
- **IsInstalled()**: Checks if app-of-apps is currently installed in a namespace
- **GetStatus()**: Retrieves the current status of an app-of-apps installation

## Usage Example

```go
// Create service dependencies
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()

// Create app-of-apps service
appOfApps := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Configure installation
config := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    AppOfApps: &config.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/org/app-of-apps",
        GitHubBranch: "main",
        ValuesFile:   "values.yaml",
    },
}

// Install app-of-apps
err := appOfApps.Install(context.Background(), config)
if err != nil {
    log.Fatal(err)
}

// Check installation status
installed, _ := appOfApps.IsInstalled(ctx, "argocd")
```
