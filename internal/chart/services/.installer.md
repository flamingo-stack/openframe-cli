Orchestrates the complete chart installation process for ArgoCD and app-of-apps components. Manages the sequential installation flow with proper error handling and context support.

## Key Components

- **Installer** - Main service struct that coordinates chart installations
- **InstallCharts()** - Simplified installation method using background context
- **InstallChartsWithContext()** - Full installation method with context support for cancellation/timeouts
- **argoCDService** - Service dependency for ArgoCD operations
- **appOfAppsService** - Service dependency for app-of-apps pattern management

## Usage Example

```go
// Create installer with service dependencies
installer := &Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Configure installation
config := config.ChartInstallConfig{
    ClusterName: "production",
    // ... other config options
}

// Install charts with context
ctx, cancel := context.WithTimeout(context.Background(), 10*time.Minute)
defer cancel()

err := installer.InstallChartsWithContext(ctx, config)
if err != nil {
    log.Fatalf("Installation failed: %v", err)
}
```

The installer follows a specific sequence: ArgoCD installation first, followed by optional app-of-apps deployment, then waiting for all applications to reach ready state. Includes specialized error handling for branch-not-found scenarios and wraps other errors with cluster context.
