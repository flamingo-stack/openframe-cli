Orchestrates the complete chart installation process for OpenFrame clusters, managing ArgoCD and app-of-apps deployment with proper error handling and context support.

## Key Components

**Installer Struct**
- `argoCDService` - Service for ArgoCD installation and management
- `appOfAppsService` - Service for app-of-apps pattern deployment

**Core Methods**
- `InstallCharts()` - Main installation method using background context
- `InstallChartsWithContext()` - Installation with custom context for cancellation/timeout

## Usage Example

```go
import (
    "context"
    "time"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/config"
)

// Create installer with required services
installer := &services.Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Configure installation
installConfig := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    // Additional configuration...
}

// Install with timeout context
ctx, cancel := context.WithTimeout(context.Background(), 10*time.Minute)
defer cancel()

if err := installer.InstallChartsWithContext(ctx, installConfig); err != nil {
    log.Fatal("Installation failed:", err)
}
```

The installer follows a sequential process: installs ArgoCD first, then optionally deploys app-of-apps from GitHub repositories, and finally waits for all applications to be ready. It provides specialized error handling for recoverable scenarios and branch not found errors.
