Manages the orchestrated installation of charts, specifically ArgoCD and app-of-apps configurations, with context support and proper error handling.

## Key Components

- **Installer**: Main service struct that coordinates chart installations
- **InstallCharts()**: Simple chart installation without context
- **InstallChartsWithContext()**: Context-aware chart installation with timeout support

## Usage Example

```go
// Initialize installer with required services
installer := &Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Install charts with configuration
config := config.ChartInstallConfig{
    ClusterName: "production",
    // ... other config fields
}

// Simple installation
err := installer.InstallCharts(config)

// Or with context and timeout
ctx, cancel := context.WithTimeout(context.Background(), 10*time.Minute)
defer cancel()
err = installer.InstallChartsWithContext(ctx, config)
```

The installer follows a specific sequence: first installing ArgoCD, then optionally installing app-of-apps from a GitHub repository, and finally waiting for all ArgoCD applications to become ready. It includes specialized error handling for branch not found scenarios and wraps other errors with cluster context information.
