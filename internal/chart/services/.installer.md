Orchestrates the complete Helm chart installation process for the OpenFrame platform, managing ArgoCD deployment and application of applications (app-of-apps) pattern with proper error handling and context support.

## Key Components

- **Installer struct**: Main orchestrator with `argoCDService` and `appOfAppsService` dependencies
- **InstallCharts()**: Simplified installation method using background context
- **InstallChartsWithContext()**: Full installation method with custom context support and sequential deployment logic

## Usage Example

```go
// Create installer with required services
installer := &Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Basic installation
config := config.ChartInstallConfig{
    ClusterName: "production",
    // ... other config options
}

err := installer.InstallCharts(config)
if err != nil {
    log.Fatal("Installation failed:", err)
}

// Installation with custom context
ctx, cancel := context.WithTimeout(context.Background(), 30*time.Minute)
defer cancel()

err = installer.InstallChartsWithContext(ctx, config)
if err != nil {
    // Handle specific error types
    if branchErr, ok := err.(*sharedErrors.BranchNotFoundError); ok {
        log.Printf("Branch not found: %v", branchErr)
    } else {
        log.Printf("Installation error: %v", err)
    }
}
```

The installation follows a strict sequence: ArgoCD installation, optional app-of-apps deployment, and application readiness verification. Branch-related errors are preserved as-is, while other errors are wrapped with cluster context for better debugging.
