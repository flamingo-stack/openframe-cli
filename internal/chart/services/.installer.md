Service that orchestrates the complete Helm chart installation process for OpenFrame, handling ArgoCD setup and app-of-apps pattern deployment with proper error handling and context support.

## Key Components

**`Installer` struct**: Main orchestrator containing ArgoCD and app-of-apps service dependencies

**`InstallCharts(config)`**: Convenience method for chart installation without context

**`InstallChartsWithContext(ctx, config)`**: Core installation method with context support for cancellation and timeouts

## Usage Example

```go
// Create installer with required services
installer := &Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Install charts with configuration
config := config.ChartInstallConfig{
    ClusterName: "production",
    // ... other config fields
}

// Simple installation
if err := installer.InstallCharts(config); err != nil {
    log.Fatalf("Installation failed: %v", err)
}

// Installation with context and timeout
ctx, cancel := context.WithTimeout(context.Background(), 30*time.Minute)
defer cancel()

if err := installer.InstallChartsWithContext(ctx, config); err != nil {
    log.Fatalf("Installation failed: %v", err)
}
```

The installer follows a two-phase approach: first installing ArgoCD as the GitOps controller, then deploying the app-of-apps pattern if configured. It includes sophisticated error handling that preserves branch-not-found errors while wrapping other failures as chart errors with cluster context.
