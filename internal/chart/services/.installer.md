This file orchestrates the Helm chart installation process by coordinating ArgoCD and app-of-apps deployments with proper error handling and context support.

## Key Components

- **Installer**: Main struct that orchestrates chart installations using ArgoCD and app-of-apps services
- **InstallCharts()**: Simplified installation method using background context
- **InstallChartsWithContext()**: Full installation method with custom context support for cancellation/timeouts

## Usage Example

```go
// Create installer with required services
installer := &Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Simple installation
config := config.ChartInstallConfig{
    ClusterName: "production",
    // other config fields...
}
err := installer.InstallCharts(config)

// Installation with timeout context
ctx, cancel := context.WithTimeout(context.Background(), 10*time.Minute)
defer cancel()
err = installer.InstallChartsWithContext(ctx, config)
```

The installer follows a two-phase approach: first installing ArgoCD, then optionally deploying app-of-apps from a GitHub repository. It includes specialized error handling for branch-not-found scenarios and implements application readiness waiting with recoverable error types.
