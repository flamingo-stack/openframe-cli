High-level service that orchestrates the complete Helm chart installation process, managing the sequential installation of ArgoCD and app-of-apps patterns with proper error handling and context support.

## Key Components

- **Installer**: Main orchestrator struct containing ArgoCD and app-of-apps service dependencies
- **InstallCharts**: Convenience method for chart installation using background context
- **InstallChartsWithContext**: Core installation method with context support and cancellation

## Usage Example

```go
// Create installer with required services
installer := &services.Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Install charts with default context
config := config.ChartInstallConfig{
    ClusterName: "production",
    // ... other configuration
}

err := installer.InstallCharts(config)
if err != nil {
    log.Fatalf("Installation failed: %v", err)
}

// Install with custom context and timeout
ctx, cancel := context.WithTimeout(context.Background(), 10*time.Minute)
defer cancel()

err = installer.InstallChartsWithContext(ctx, config)
```

The installer follows a sequential process: installs ArgoCD first, then optionally installs app-of-apps if configured, and finally waits for all applications to become ready. It provides specialized error handling for branch not found scenarios and wraps other errors with cluster context information.
