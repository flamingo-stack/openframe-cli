This file implements the chart installation orchestrator that coordinates ArgoCD and app-of-apps deployments with proper error handling and context support.

## Key Components

- **Installer**: Main orchestrator struct that manages chart installation workflows
- **InstallCharts()**: Simple installation method using default background context
- **InstallChartsWithContext()**: Context-aware installation method with timeout/cancellation support

## Usage Example

```go
// Initialize installer with required services
installer := &Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Simple installation
config := config.ChartInstallConfig{
    ClusterName: "production",
    // ... other config fields
}

err := installer.InstallCharts(config)
if err != nil {
    log.Fatal("Installation failed:", err)
}

// Installation with context and timeout
ctx, cancel := context.WithTimeout(context.Background(), 10*time.Minute)
defer cancel()

err = installer.InstallChartsWithContext(ctx, config)
if err != nil {
    log.Fatal("Installation failed:", err)
}
```

The installer follows a specific sequence: installs ArgoCD first, then optionally deploys app-of-apps from a GitHub repository, and finally waits for all applications to be ready. It provides specialized error handling for different failure scenarios like branch-not-found errors.
