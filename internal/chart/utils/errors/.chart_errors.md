This file provides a comprehensive error handling system specifically designed for Helm chart operations, with support for error recovery, troubleshooting guidance, and contextual information.

## Key Components

### Core Error Types
- **`ChartError`** - Base error type with operation context, cluster information, and recovery metadata
- **`InstallationError`** - Installation-specific errors with phase tracking and troubleshooting steps
- **`ValidationError`** - Configuration validation errors with field-level details
- **`ConfigurationError`** - Configuration file errors with missing key tracking
- **`SkippedInstallationError`** - Non-error type for intentionally skipped installations

### Constructor Functions
- **`NewChartError`** - Creates basic chart errors
- **`NewRecoverableChartError`** - Creates retryable errors with delay
- **`NewInstallationError`** - Creates installation-specific errors
- **`NewValidationError`** - Creates validation errors with constraint details
- **`NewConfigurationError`** - Creates configuration errors

### Utility Functions
- **`IsTimeout`**, **`IsRecoverable`**, **`IsSkippedInstallation`** - Error type checking
- **`GetRetryDelay`** - Extracts retry timing from recoverable errors
- **`WrapAsChartError`** - Converts generic errors to chart errors
- **`CombineErrors`** - Merges multiple errors into one

### Predefined Error Constants
Common chart operation errors like `ErrChartNotFound`, `ErrClusterNotReady`, `ErrInsufficientResources`, etc.

## Usage Example

```go
// Create a recoverable installation error
err := NewRecoverableChartError("install", "prometheus", 
    ErrClusterNotReady, 30*time.Second)
err.WithCluster("production")

// Create installation error with troubleshooting
installErr := NewInstallationError("nginx", "validation", ErrInvalidConfiguration)
installErr.WithSuggestions([]string{
    "Check values.yaml syntax",
    "Verify required fields are present",
})

// Check if error is recoverable and get retry delay
if IsRecoverable(err) {
    delay := GetRetryDelay(err)
    time.Sleep(delay)
    // retry operation
}

// Handle validation errors
validationErr := NewValidationError("replicas", "invalid", "must be positive integer")
fmt.Println(validationErr.Error()) // "validation failed for field 'replicas': must be positive integer (value: 'invalid')"
```
