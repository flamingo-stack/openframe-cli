Test suite for chart error handling types and utilities. Contains comprehensive test coverage for chart-specific error types, validation, configuration errors, and helper functions.

## Key Components

- **ChartError Tests** - Validates core chart error functionality including creation, error messages, clustering, and recovery options
- **InstallationError Tests** - Tests Helm installation error handling with troubleshooting steps and suggestions
- **ValidationError Tests** - Verifies field validation error creation and formatting
- **ConfigurationError Tests** - Tests configuration file error handling with missing keys support
- **Helper Function Tests** - Tests utility functions like `IsTimeout`, `IsRecoverable`, `GetRetryDelay`, `WrapAsChartError`, and `CombineErrors`

## Usage Example

```go
// Test creating a recoverable chart error
func TestRecoverableError(t *testing.T) {
    cause := errors.New("temporary failure")
    retryAfter := 30 * time.Second
    
    chartErr := NewRecoverableChartError("installation", "Helm", cause, retryAfter)
    
    assert.True(t, chartErr.IsRecoverable())
    assert.Equal(t, retryAfter, chartErr.GetRetryAfter())
}

// Test validation error with field details
func TestValidation(t *testing.T) {
    valErr := NewValidationError("database.host", "invalid", "must be valid hostname")
    
    errorMsg := valErr.Error()
    assert.Contains(t, errorMsg, "database.host")
    assert.Contains(t, errorMsg, "invalid")
}
```
