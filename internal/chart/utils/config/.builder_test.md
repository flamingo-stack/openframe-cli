Test suite for the config builder component that constructs installation configurations for the OpenFrame CLI tool.

## Key Components

- **TestNewBuilder** - Tests builder initialization with operations UI
- **TestNewBuilder_WithNilOperationsUI** - Validates builder handles nil UI gracefully
- **TestBuilder_ImplementsConfigBuilderInterface** - Verifies interface compliance
- **TestBuilder_BuildInstallConfig_*** - Comprehensive tests for configuration building with various parameter combinations

## Usage Example

```go
func TestBasicConfiguration(t *testing.T) {
    operationsUI := chartUI.NewOperationsUI()
    builder := NewBuilder(operationsUI)
    
    config, err := builder.BuildInstallConfig(
        false, false, false, // force, dryRun, verbose
        "my-cluster",
        "https://github.com/user/repo", "main", "/certs",
    )
    
    assert.NoError(t, err)
    assert.Equal(t, "my-cluster", config.ClusterName)
    assert.NotNil(t, config.AppOfApps)
    assert.True(t, config.HasAppOfApps())
}
```

The tests cover builder initialization, configuration building with various flag combinations, GitHub repository settings, certificate directories, and edge cases like empty parameters. They ensure the builder is stateless and can handle multiple configuration builds independently.
