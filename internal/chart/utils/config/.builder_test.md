Test suite for the configuration builder component that validates the creation and configuration of install configurations for CLI operations.

## Key Components

- **TestNewBuilder**: Validates proper initialization of the Builder with required dependencies
- **TestBuilder_BuildInstallConfig_***: Comprehensive test suite covering various configuration scenarios including basic setup, flags, GitHub repository integration, and certificate directory handling
- **Interface compatibility tests**: Ensures the Builder implements expected methods without import cycles

## Usage Example

```go
func TestBuilder_BuildInstallConfig_WithGitHub(t *testing.T) {
    operationsUI := chartUI.NewOperationsUI()
    builder := NewBuilder(operationsUI)

    config, err := builder.BuildInstallConfig(
        true, false, true,  // force, dryRun, verbose flags
        "production-cluster",
        "https://github.com/company/argocd-apps", 
        "main", 
        "/etc/ssl/certs",
    )

    assert.NoError(t, err)
    assert.Equal(t, "production-cluster", config.ClusterName)
    assert.True(t, config.Force)
    assert.True(t, config.HasAppOfApps())
    assert.Equal(t, "main", config.AppOfApps.GitHubBranch)
}
```

The test suite validates builder functionality across different scenarios including nil handling, flag combinations, GitHub integration, and stateless operation across multiple builds.
