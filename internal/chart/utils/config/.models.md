Configuration model for chart installation operations, defining the main configuration structure and app-of-apps detection logic.

## Key Components

- **ChartInstallConfig**: Main configuration struct containing installation parameters including cluster settings, execution flags, and optional app-of-apps configuration
- **HasAppOfApps()**: Method to determine if app-of-apps configuration is present and valid

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/config"
    "github.com/flamingo-stack/openframe-cli/internal/chart/models"
)

func main() {
    // Create basic installation config
    installConfig := &config.ChartInstallConfig{
        ClusterName:    "production",
        Force:          false,
        DryRun:         true,
        Verbose:        true,
        Silent:         false,
        NonInteractive: false,
    }

    // Add app-of-apps configuration
    installConfig.AppOfApps = &models.AppOfAppsConfig{
        GitHubRepo: "myorg/gitops-repo",
        // ... other app-of-apps fields
    }

    // Check if app-of-apps is configured
    if installConfig.HasAppOfApps() {
        // Proceed with app-of-apps installation
        fmt.Println("Installing with app-of-apps pattern")
    } else {
        // Standard chart installation
        fmt.Println("Installing standard chart")
    }
}
```

The configuration supports both standard chart installations and the app-of-apps GitOps pattern, with flags for controlling execution behavior like dry runs, verbosity, and interactive elements.
