Defines configuration structures and utilities for Helm chart installation operations within the OpenFrame CLI tool.

## Key Components

- **`ChartInstallConfig`** - Main configuration struct containing installation parameters including cluster name, execution flags (force, dry-run, verbose), and optional app-of-apps configuration
- **`HasAppOfApps()`** - Method that validates whether app-of-apps configuration is properly set with a GitHub repository

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/config"
    "github.com/flamingo-stack/openframe-cli/internal/chart/models"
)

func installChart() {
    cfg := &config.ChartInstallConfig{
        ClusterName:    "production",
        Force:          false,
        DryRun:         true,
        Verbose:        true,
        NonInteractive: false,
        AppOfApps: &models.AppOfAppsConfig{
            GitHubRepo: "my-org/argocd-apps",
        },
    }
    
    // Check if app-of-apps is configured
    if cfg.HasAppOfApps() {
        // Proceed with app-of-apps installation
    } else {
        // Standard chart installation
    }
}
```
