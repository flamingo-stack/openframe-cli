Configuration model for Helm chart installation operations, providing options for deployment behavior and app-of-apps pattern support.

## Key Components

- **`ChartInstallConfig`** - Main configuration struct containing installation parameters including cluster targeting, execution modes (dry-run, force, verbose), UI preferences, and optional app-of-apps configuration
- **`HasAppOfApps()`** - Validation method that checks if valid app-of-apps configuration is present by verifying both the config exists and has a GitHub repository specified

## Usage Example

```go
// Basic chart installation configuration
config := &ChartInstallConfig{
    ClusterName:    "production",
    Force:          false,
    DryRun:         true,
    Verbose:        true,
    Silent:         false,
    NonInteractive: false,
}

// Configuration with app-of-apps pattern
config.AppOfApps = &models.AppOfAppsConfig{
    GitHubRepo: "my-org/argocd-apps",
    // ... other app-of-apps fields
}

// Check if app-of-apps is configured
if config.HasAppOfApps() {
    // Proceed with app-of-apps installation
    fmt.Println("Installing with app-of-apps pattern")
}
```
