This file defines configuration structures for chart installation operations in the OpenFrame CLI, providing settings for both basic chart installations and app-of-apps deployments.

## Key Components

- **`ChartInstallConfig`** - Main configuration struct containing installation parameters like cluster name, execution flags (force, dry-run, verbose), and optional app-of-apps settings
- **`HasAppOfApps()`** - Helper method to check if app-of-apps configuration is properly configured with a GitHub repository

## Usage Example

```go
package main

import "github.com/flamingo-stack/openframe-cli/internal/config"

func main() {
    // Basic chart installation config
    config := &config.ChartInstallConfig{
        ClusterName:    "my-cluster",
        Force:          true,
        DryRun:         false,
        Verbose:        true,
        NonInteractive: true,
    }

    // Check if app-of-apps is configured
    if config.HasAppOfApps() {
        // Handle app-of-apps installation
        fmt.Println("App-of-apps configuration detected")
    } else {
        // Handle standard chart installation
        fmt.Println("Standard chart installation")
    }
}
```
