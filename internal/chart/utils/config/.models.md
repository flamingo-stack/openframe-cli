Defines configuration structures for Helm chart installation operations within the openframe-cli tool, providing options for deployment behavior and app-of-apps pattern support.

## Key Components

- **`ChartInstallConfig`** - Main configuration struct containing installation parameters including cluster targeting, execution modes (dry-run, force, verbose), UI behavior controls, and optional app-of-apps configuration
- **`HasAppOfApps()`** - Method that validates whether app-of-apps configuration is properly set with a valid GitHub repository

## Usage Example

```go
config := &ChartInstallConfig{
    ClusterName:    "production",
    Force:          false,
    DryRun:         true,
    Verbose:        true,
    NonInteractive: true,
    AppOfApps: &models.AppOfAppsConfig{
        GitHubRepo: "org/my-apps-repo",
    },
}

// Check if app-of-apps pattern is configured
if config.HasAppOfApps() {
    // Proceed with app-of-apps installation
    fmt.Println("Installing using app-of-apps pattern")
} else {
    // Standard chart installation
    fmt.Println("Installing single chart")
}
```
