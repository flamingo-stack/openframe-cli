Defines configuration structures and utilities for Helm chart installation operations in the OpenFrame CLI, providing settings for deployment behavior and app-of-apps pattern support.

## Key Components

**ChartInstallConfig** - Main configuration struct containing:
- Cluster identification (`ClusterName`)
- Installation behavior flags (`Force`, `DryRun`, `Verbose`, `Silent`)
- UI control options (`NonInteractive`, `SkipCRDs`) 
- App-of-apps pattern configuration (`AppOfApps`)

**HasAppOfApps()** - Method to check if valid app-of-apps configuration is present

## Usage Example

```go
// Create chart installation configuration
config := &ChartInstallConfig{
    ClusterName:    "production-cluster",
    Force:          false,
    DryRun:         true,
    Verbose:        true,
    Silent:         false,
    NonInteractive: true,
    SkipCRDs:       false,
    AppOfApps: &models.AppOfAppsConfig{
        GitHubRepo: "flamingo-stack/production-apps",
        // ... other app-of-apps settings
    },
}

// Check if app-of-apps pattern is configured
if config.HasAppOfApps() {
    fmt.Println("App-of-apps configuration detected")
}
```

This configuration drives the Helm chart installation process, allowing fine-grained control over deployment behavior while supporting both traditional single-chart installs and the app-of-apps GitOps pattern for managing multiple applications.
