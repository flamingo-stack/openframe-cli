This file provides a centralized path resolution system for managing file and directory locations within the OpenFrame CLI application, particularly for certificates, Helm charts, and Kubernetes manifests.

## Key Components

- **PathResolver**: Main struct that handles all path resolution operations
- **NewPathResolver()**: Constructor function that creates a new PathResolver instance
- **GetCertificateDirectory()**: Returns the directory for storing SSL certificates (`~/.config/openframe/certs`)
- **GetManifestsDirectory()**: Returns the relative path to Kubernetes manifests directory
- **GetHelmValuesFile()**: Returns the path to the main Helm values configuration file
- **GetArgocdValuesFile()**: Returns the path to ArgoCD-specific values file
- **GetCertificateFiles()**: Returns both certificate and key file paths for localhost SSL

## Usage Example

```go
package main

import (
    "fmt"
    "config"
)

func main() {
    resolver := config.NewPathResolver()
    
    // Get certificate directory
    certDir := resolver.GetCertificateDirectory()
    fmt.Printf("Certificates stored in: %s\n", certDir)
    
    // Get certificate files
    certFile, keyFile := resolver.GetCertificateFiles()
    fmt.Printf("Cert file: %s\n", certFile)
    fmt.Printf("Key file: %s\n", keyFile)
    
    // Get configuration files
    helmValues := resolver.GetHelmValuesFile()
    argocdValues := resolver.GetArgocdValuesFile()
    fmt.Printf("Helm values: %s\n", helmValues)
    fmt.Printf("ArgoCD values: %s\n", argocdValues)
}
```

The PathResolver automatically creates the certificate directory if it doesn't exist and provides fallback behavior when the user's home directory cannot be determined.
