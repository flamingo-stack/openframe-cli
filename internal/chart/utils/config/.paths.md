Provides centralized path resolution for chart-related files and directories, handling certificate storage locations, manifest paths, and configuration files with proper fallback mechanisms.

## Key Components

- **PathResolver**: Main struct that handles all path resolution operations
- **NewPathResolver()**: Factory function to create a new path resolver instance
- **GetCertificateDirectory()**: Resolves certificate storage directory with home directory fallback
- **GetManifestsDirectory()**: Returns relative path to chart manifests
- **GetHelmValuesFile()**: Provides path to Helm values configuration file
- **GetArgocdValuesFile()**: Returns path to ArgoCD-specific values file
- **GetCertificateFiles()**: Returns both certificate and key file paths

## Usage Example

```go
package main

import (
    "fmt"
    "path/config"
)

func main() {
    resolver := config.NewPathResolver()
    
    // Get certificate directory (creates if doesn't exist)
    certDir := resolver.GetCertificateDirectory()
    fmt.Printf("Certificates stored in: %s\n", certDir)
    
    // Get certificate file paths
    certFile, keyFile := resolver.GetCertificateFiles()
    fmt.Printf("Cert: %s, Key: %s\n", certFile, keyFile)
    
    // Get configuration file paths
    helmValues := resolver.GetHelmValuesFile()
    argocdValues := resolver.GetArgocdValuesFile()
    fmt.Printf("Helm values: %s\n", helmValues)
    fmt.Printf("ArgoCD values: %s\n", argocdValues)
}
```

The resolver automatically creates the certificate directory (`~/.config/openframe/certs`) if it doesn't exist and provides fallback logic when the home directory cannot be determined.
