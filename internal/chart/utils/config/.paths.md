Provides centralized path resolution for chart-related files, directories, and certificates used throughout the OpenFrame application. Handles cross-platform path construction and automatic directory creation for certificate storage.

## Key Components

- **PathResolver**: Main struct for handling all path resolution operations
- **NewPathResolver()**: Factory function to create a new path resolver instance
- **GetCertificateDirectory()**: Returns certificate storage directory (~/.config/openframe/certs)
- **GetManifestsDirectory()**: Returns path to Kubernetes manifests directory
- **GetHelmValuesFile()**: Returns path to Helm values configuration file
- **GetArgocdValuesFile()**: Returns path to ArgoCD values configuration file
- **GetCertificateFiles()**: Returns certificate and key file paths as a tuple

## Usage Example

```go
package main

import (
    "fmt"
    "config"
)

func main() {
    resolver := config.NewPathResolver()
    
    // Get certificate directory (creates if doesn't exist)
    certDir := resolver.GetCertificateDirectory()
    fmt.Printf("Certificates stored in: %s\n", certDir)
    
    // Get certificate file paths
    certFile, keyFile := resolver.GetCertificateFiles()
    fmt.Printf("Cert: %s, Key: %s\n", certFile, keyFile)
    
    // Get configuration file paths
    helmValues := resolver.GetHelmValuesFile()
    argoValues := resolver.GetArgocdValuesFile()
    fmt.Printf("Helm values: %s\n", helmValues)
    fmt.Printf("ArgoCD values: %s\n", argoValues)
}
```
