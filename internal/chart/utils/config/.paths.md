This file provides centralized path resolution functionality for OpenFrame's chart-related files and directories, including certificates, manifests, and configuration files.

## Key Components

- **`PathResolver`** - Main struct that handles all path resolution logic
- **`NewPathResolver()`** - Constructor function that returns a new PathResolver instance
- **`GetCertificateDirectory()`** - Resolves certificate storage location with automatic directory creation
- **`GetManifestsDirectory()`** - Returns relative path to Kubernetes manifests
- **`GetHelmValuesFile()`** - Provides path to the main Helm values configuration
- **`GetArgocdValuesFile()`** - Returns path to ArgoCD-specific values file
- **`GetCertificateFiles()`** - Returns both certificate and key file paths as a tuple

## Usage Example

```go
package main

import (
    "fmt"
    "yourproject/config"
)

func main() {
    resolver := config.NewPathResolver()
    
    // Get certificate directory (creates if doesn't exist)
    certDir := resolver.GetCertificateDirectory()
    fmt.Printf("Certificates stored in: %s\n", certDir)
    
    // Get certificate file paths
    certFile, keyFile := resolver.GetCertificateFiles()
    fmt.Printf("Cert: %s, Key: %s\n", certFile, keyFile)
    
    // Get configuration file paths
    helmValues := resolver.GetHelmValuesFile()
    argoValues := resolver.GetArgocdValuesFile()
    
    fmt.Printf("Helm values: %s\n", helmValues)
    fmt.Printf("ArgoCD values: %s\n", argoValues)
}
```

The resolver handles cross-platform path construction and includes fallback logic for certificate directory resolution when the user's home directory is unavailable.
