This file provides a centralized path resolution service for chart-related files and directories in the OpenFrame configuration system. It handles dynamic path construction for certificates, manifests, and configuration files across different execution contexts.

## Key Components

- **PathResolver**: Main struct that encapsulates path resolution logic
- **NewPathResolver()**: Constructor function for creating a new path resolver instance
- **GetCertificateDirectory()**: Returns the user's certificate storage directory with automatic creation
- **GetManifestsDirectory()**: Returns the relative path to chart manifests
- **GetHelmValuesFile()**: Returns the path to the main Helm values configuration file
- **GetArgocdValuesFile()**: Returns the path to ArgoCD-specific values file
- **GetCertificateFiles()**: Returns both certificate and key file paths as a tuple

## Usage Example

```go
// Create a path resolver instance
resolver := config.NewPathResolver()

// Get certificate directory (creates if doesn't exist)
certDir := resolver.GetCertificateDirectory()
// Returns: ~/.config/openframe/certs

// Get certificate file paths
certFile, keyFile := resolver.GetCertificateFiles()
// Returns: (~/.config/openframe/certs/localhost.pem, ~/.config/openframe/certs/localhost-key.pem)

// Get configuration file paths
manifestsDir := resolver.GetManifestsDirectory()
helmValues := resolver.GetHelmValuesFile()
argocdValues := resolver.GetArgocdValuesFile()
```
