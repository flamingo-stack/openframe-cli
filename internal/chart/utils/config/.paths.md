Provides path resolution utilities for chart-related files, certificates, and configuration directories. This utility centralizes file path management for the OpenFrame CLI tool.

## Key Components

- **PathResolver**: Main struct that handles all path resolution operations
- **NewPathResolver()**: Constructor function for creating a new path resolver instance
- **GetCertificateDirectory()**: Returns the user's certificate storage directory (`~/.config/openframe/certs`)
- **GetManifestsDirectory()**: Returns the relative path to chart manifests
- **GetHelmValuesFile()**: Returns the path to the main Helm values configuration file
- **GetArgocdValuesFile()**: Returns the path to ArgoCD-specific values file
- **GetCertificateFiles()**: Returns paths to both certificate and key files

## Usage Example

```go
// Create a path resolver
resolver := config.NewPathResolver()

// Get certificate directory (creates if doesn't exist)
certDir := resolver.GetCertificateDirectory()

// Get certificate file paths
certFile, keyFile := resolver.GetCertificateFiles()

// Get configuration file paths
helmValues := resolver.GetHelmValuesFile()
argoValues := resolver.GetArgocdValuesFile()
manifests := resolver.GetManifestsDirectory()

// Use paths for file operations
if _, err := os.Stat(certFile); err == nil {
    // Certificate exists, proceed with configuration
}
```
