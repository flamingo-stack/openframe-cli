This file provides a centralized path resolution service for OpenFrame chart-related files and directories, handling both user-specific certificate storage and relative paths to manifests and configuration files.

## Key Components

- **`PathResolver`**: Main struct that provides path resolution methods
- **`NewPathResolver()`**: Constructor function that creates a new path resolver instance
- **`GetCertificateDirectory()`**: Resolves user's certificate directory (`~/.config/openframe/certs`) with fallback handling
- **`GetManifestsDirectory()`**: Returns relative path to chart manifests
- **`GetHelmValuesFile()`**: Returns path to Helm values configuration file
- **`GetArgocdValuesFile()`**: Returns path to ArgoCD values file within manifests directory
- **`GetCertificateFiles()`**: Returns full paths to SSL certificate and key files

## Usage Example

```go
package main

import (
    "fmt"
    "your-project/config"
)

func main() {
    resolver := config.NewPathResolver()
    
    // Get certificate directory (creates if doesn't exist)
    certDir := resolver.GetCertificateDirectory()
    fmt.Printf("Certificates stored in: %s\n", certDir)
    
    // Get certificate file paths
    certFile, keyFile := resolver.GetCertificateFiles()
    fmt.Printf("Cert: %s, Key: %s\n", certFile, keyFile)
    
    // Get configuration file paths
    helmValues := resolver.GetHelmValuesFile()
    argocdValues := resolver.GetArgocdValuesFile()
    fmt.Printf("Helm values: %s\n", helmValues)
    fmt.Printf("ArgoCD values: %s\n", argocdValues)
}
```

The resolver automatically creates the certificate directory if it doesn't exist and provides consistent path resolution across the OpenFrame deployment system.
