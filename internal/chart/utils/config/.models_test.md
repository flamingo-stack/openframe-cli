Test suite for the `ChartInstallConfig` model that validates configuration behavior and field initialization.

## Key Components

- **TestChartInstallConfig_HasAppOfApps**: Tests the `HasAppOfApps()` method logic with various AppOfApps configurations
- **TestChartInstallConfig_DefaultValues**: Verifies default zero-value initialization of all configuration fields  
- **TestChartInstallConfig_WithValues**: Tests configuration with explicitly set values and validates field assignment

## Usage Example

```go
// Run all tests for ChartInstallConfig
go test -v ./config -run TestChartInstallConfig

// Run specific test cases
go test -v ./config -run TestChartInstallConfig_HasAppOfApps
go test -v ./config -run TestChartInstallConfig_DefaultValues

// The tests validate behavior like:
config := &ChartInstallConfig{}
hasAppOfApps := config.HasAppOfApps() // Should return false for empty config

config.AppOfApps = &models.AppOfAppsConfig{
    GitHubRepo: "https://github.com/test/repo",
}
hasAppOfApps = config.HasAppOfApps() // Should return true with valid repo
```

This test file ensures the `ChartInstallConfig` model behaves correctly across different initialization scenarios and validates the AppOfApps detection logic.
