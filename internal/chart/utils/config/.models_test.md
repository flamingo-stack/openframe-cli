Test file that validates the behavior and functionality of the `ChartInstallConfig` struct and its methods.

## Key Components

- **TestChartInstallConfig_HasAppOfApps**: Tests the `HasAppOfApps()` method logic with different AppOfApps configurations
- **TestChartInstallConfig_DefaultValues**: Verifies default/zero values of ChartInstallConfig fields
- **TestChartInstallConfig_WithValues**: Tests ChartInstallConfig with explicitly set values

## Usage Example

```go
// Run tests for the config package
go test ./internal/config

// Run specific test function
go test -run TestChartInstallConfig_HasAppOfApps ./internal/config

// Run with verbose output
go test -v ./internal/config
```

The test suite covers three main scenarios: validating the AppOfApps detection logic (checking nil vs empty vs valid configurations), ensuring proper default values for all struct fields, and confirming that explicitly set values are correctly stored and accessible through the HasAppOfApps method.
