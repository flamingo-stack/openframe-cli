Configuration builder for constructing Helm chart installation configurations with branch detection from values files.

## Key Components

- **Builder**: Main struct that handles configuration construction with UI operations support
- **HelmValues**: YAML structure representing Helm values file with deployment modes (OSS/SaaS)
- **NewBuilder()**: Creates a new Builder instance with operations UI integration
- **BuildInstallConfig()**: Constructs standard installation configuration with GitHub repo support
- **BuildInstallConfigWithCustomHelmPath()**: Builds configuration using custom Helm values path and deployment modes
- **getBranchForDeploymentMode()**: Extracts appropriate Git branch based on deployment mode (saas-shared, saas-tenant, oss-tenant)
- **getBranchFromHelmValues()**: Reads branch configuration from default Helm values file

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/ui"
    "github.com/flamingo-stack/openframe-cli/internal/config"
)

// Create builder
operationsUI := &ui.OperationsUI{}
builder := config.NewBuilder(operationsUI)

// Build standard install config
config, err := builder.BuildInstallConfig(
    false,                    // force
    false,                    // dryRun  
    true,                     // verbose
    "my-cluster",             // clusterName
    "owner/repo",             // githubRepo
    "main",                   // githubBranch
    "/path/to/certs",         // certDir
)

// Build config with custom Helm values
customConfig, err := builder.BuildInstallConfigWithCustomHelmPath(
    false,                    // force
    false,                    // dryRun
    true,                     // verbose
    false,                    // nonInteractive
    "my-cluster",             // clusterName
    "owner/repo",             // githubRepo
    "main",                   // githubBranch
    "/path/to/certs",         // certDir
    "/custom/values.yaml",    // helmValuesPath
    "saas-shared",            // deploymentMode
)
```
