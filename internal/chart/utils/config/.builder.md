This file implements a builder pattern for constructing installation configurations, handling Helm values parsing and branch selection for different deployment modes (OSS, SaaS Tenant, SaaS Shared).

## Key Components

**Types:**
- `Builder` - Main configuration builder with config service and UI dependencies
- `HelmValues` - Struct representing Helm values file structure with deployment configurations

**Functions:**
- `NewBuilder()` - Creates a new configuration builder instance
- `getBranchForDeploymentMode()` - Extracts appropriate Git branch based on deployment mode
- `getBranchFromHelmValues()` - Reads branch from default Helm values file
- `BuildInstallConfig()` - Constructs standard installation configuration
- `BuildInstallConfigWithCustomHelmPath()` - Builds config with custom Helm values file path

## Usage Example

```go
// Create a builder with UI operations
operationsUI := chartUI.NewOperationsUI()
builder := NewBuilder(operationsUI)

// Build standard installation config
config, err := builder.BuildInstallConfig(
    false,          // force
    false,          // dryRun  
    true,           // verbose
    "my-cluster",   // clusterName
    "github.com/my/repo", // githubRepo
    "main",         // githubBranch
    "/certs",       // certDir
)

// Build config with custom Helm values
config, err := builder.BuildInstallConfigWithCustomHelmPath(
    false, false, true, false,  // flags
    "my-cluster",               // clusterName
    "github.com/my/repo",       // githubRepo
    "main",                     // githubBranch
    "/certs",                   // certDir
    "/custom/values.yaml",      // helmValuesPath
    "saas-shared",              // deploymentMode
)
```
