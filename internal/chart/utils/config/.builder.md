Configuration builder for OpenFrame installation setups, managing Helm values parsing and deployment mode branching logic for OSS, SaaS tenant, and SaaS shared environments.

## Key Components

- **Builder** - Main struct handling configuration construction with chart UI integration
- **HelmValues** - YAML structure representing Helm deployment configurations with OSS/SaaS branches
- **NewBuilder()** - Constructor creating builder with operations UI and config service
- **getBranchForDeploymentMode()** - Deployment mode-aware branch resolution (saas-shared vs oss/saas-tenant)
- **BuildInstallConfig()** - Standard configuration builder with auto-detected Helm values
- **BuildInstallConfigWithCustomHelmPath()** - Configuration builder supporting custom Helm values files and deployment modes

## Usage Example

```go
// Create builder with UI operations
operationsUI := chartUI.NewOperationsUI()
builder := config.NewBuilder(operationsUI)

// Build standard installation config
config, err := builder.BuildInstallConfig(
    false,                    // force
    false,                    // dryRun  
    true,                     // verbose
    "my-cluster",            // clusterName
    "github.com/org/repo",   // githubRepo
    "main",                  // githubBranch
    "/path/to/certs",        // certDir
)

// Build config with custom Helm path and deployment mode
config, err := builder.BuildInstallConfigWithCustomHelmPath(
    false,                    // force
    false,                    // dryRun
    true,                     // verbose
    false,                    // nonInteractive
    "prod-cluster",          // clusterName
    "github.com/org/repo",   // githubRepo
    "main",                  // githubBranch
    "/certs",                // certDir
    "/custom/values.yaml",   // helmValuesPath
    "saas-shared",           // deploymentMode
)
```

The builder automatically detects branch overrides from Helm values and handles deployment mode-specific branching for different OpenFrame environments.
