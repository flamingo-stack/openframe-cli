This file provides a configuration builder for Helm chart installations, handling different deployment modes and branch selection based on Helm values files.

## Key Components

- **Builder**: Main struct that orchestrates configuration building with UI operations
- **HelmValues**: Struct representing the structure of Helm values YAML files
- **NewBuilder()**: Factory function for creating configuration builders
- **getBranchForDeploymentMode()**: Determines the appropriate Git branch based on deployment mode (OSS, SaaS Shared, SaaS Tenant)
- **BuildInstallConfig()**: Constructs installation configuration with default settings
- **BuildInstallConfigWithCustomHelmPath()**: Constructs configuration using custom Helm values file paths

## Usage Example

```go
// Create a builder with UI operations
operationsUI := chartUI.NewOperationsUI()
builder := config.NewBuilder(operationsUI)

// Build basic install configuration
config, err := builder.BuildInstallConfig(
    false,              // force
    false,              // dryRun
    true,               // verbose
    "my-cluster",       // clusterName
    "github.com/repo",  // githubRepo
    "main",             // githubBranch
    "/path/to/certs",   // certDir
)

// Build with custom Helm values path
config, err := builder.BuildInstallConfigWithCustomHelmPath(
    false,              // force
    false,              // dryRun
    true,               // verbose
    false,              // nonInteractive
    "my-cluster",       // clusterName
    "github.com/repo",  // githubRepo
    "main",             // githubBranch
    "/path/to/certs",   // certDir
    "/custom/values.yaml", // helmValuesPath
    "saas-shared",      // deploymentMode
)
```

The builder intelligently selects Git branches based on deployment mode configuration found in Helm values files, supporting OSS tenant, SaaS tenant, and SaaS shared deployment scenarios.
