Provides functionality for building installation configurations with support for Helm values parsing and branch selection based on deployment modes.

## Key Components

- **Builder**: Main struct that handles construction of installation configurations
- **HelmValues**: Struct representing the structure of Helm values files with deployment configurations
- **NewBuilder()**: Constructor function that creates a new configuration builder
- **BuildInstallConfig()**: Constructs standard installation configuration
- **BuildInstallConfigWithCustomHelmPath()**: Constructs installation configuration with custom Helm values file
- **getBranchForDeploymentMode()**: Extracts appropriate Git branch based on deployment mode (OSS, SaaS Shared, SaaS Tenant)

## Usage Example

```go
// Create a new builder
operationsUI := &chartUI.OperationsUI{}
builder := NewBuilder(operationsUI)

// Build standard install config
config, err := builder.BuildInstallConfig(
    false,                    // force
    false,                    // dryRun  
    true,                     // verbose
    "my-cluster",            // clusterName
    "github.com/user/repo",  // githubRepo
    "main",                  // githubBranch
    "/path/to/certs",        // certDir
)

// Build config with custom Helm values
config, err := builder.BuildInstallConfigWithCustomHelmPath(
    false,                    // force
    false,                    // dryRun
    true,                     // verbose
    false,                    // nonInteractive
    "my-cluster",            // clusterName
    "github.com/user/repo",  // githubRepo
    "main",                  // githubBranch
    "/path/to/certs",        // certDir
    "/custom/values.yaml",   // helmValuesPath
    "saas-shared",           // deploymentMode
)
```
