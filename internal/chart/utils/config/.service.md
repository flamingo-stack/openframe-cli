Provides centralized configuration management for chart operations, handling paths, certificates, and installation configurations.

## Key Components

- **Service**: Main configuration service that orchestrates path resolution and system configuration
- **NewService()**: Factory function that creates a new configuration service instance
- **GetCertificateDirectory()**: Returns the directory path for SSL certificates
- **GetCertificateFiles()**: Returns paths to both certificate and key files
- **GetHelmValuesFile()**: Returns the path to the Helm values configuration file
- **Initialize()**: Sets up required directories and initializes the system service
- **BuildInstallConfig()**: Creates complete installation configuration for chart deployments
- **GetDefaultManifestsPath()**: Resolves the default path for Kubernetes manifests

## Usage Example

```go
// Create and initialize the configuration service
configService := config.NewService()
if err := configService.Initialize(); err != nil {
    log.Fatalf("Failed to initialize config: %v", err)
}

// Get certificate paths
certDir := configService.GetCertificateDirectory()
certFile, keyFile := configService.GetCertificateFiles()

// Build installation configuration
installConfig := configService.BuildInstallConfig(
    false,           // force
    false,           // dryRun  
    true,            // verbose
    "my-cluster",    // clusterName
    nil,             // appOfAppsConfig
)

// Get manifests path
manifestsPath := configService.GetDefaultManifestsPath()
```

The service integrates with shared system configuration and provides a unified interface for accessing all chart-related configuration paths and settings.
