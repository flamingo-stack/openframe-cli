Provides centralized configuration management for Helm chart operations, handling paths, certificates, and installation configurations. Integrates with shared system configuration and path resolution services.

## Key Components

- **Service**: Main configuration service with embedded system service and path resolver
- **NewService()**: Constructor that initializes the service with default dependencies
- **Initialize()**: Sets up system configuration and ensures required directories exist
- **BuildInstallConfig()**: Creates complete installation configurations for chart deployments
- **Path Methods**: Certificate directory, Helm values file, and log directory path retrieval
- **GetDefaultManifestsPath()**: Locates manifests directory with fallback logic

## Usage Example

```go
// Create and initialize configuration service
configService := config.NewService()
if err := configService.Initialize(); err != nil {
    log.Fatal(err)
}

// Get certificate paths
certFile, keyFile := configService.GetCertificateFiles()

// Build installation configuration
installConfig := configService.BuildInstallConfig(
    false,      // force
    true,       // dryRun
    true,       // verbose
    "my-cluster",
    appOfAppsConfig,
)

// Get manifests location
manifestsPath := configService.GetDefaultManifestsPath()
```
