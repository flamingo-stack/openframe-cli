Provides centralized configuration management for chart operations, handling paths, certificates, and installation configurations. This service acts as a facade over system configuration and path resolution services.

## Key Components

- **Service**: Main configuration service with embedded system service and path resolver
- **NewService()**: Constructor that initializes the configuration service
- **GetCertificateDirectory()**: Returns certificate directory path
- **GetCertificateFiles()**: Returns certificate and key file paths
- **GetHelmValuesFile()**: Returns Helm values file path
- **Initialize()**: Sets up necessary directories and initializes dependencies
- **BuildInstallConfig()**: Creates complete chart installation configuration
- **GetDefaultManifestsPath()**: Resolves default manifests directory path

## Usage Example

```go
// Create and initialize configuration service
configService := config.NewService()
if err := configService.Initialize(); err != nil {
    log.Fatal(err)
}

// Get certificate paths
certFile, keyFile := configService.GetCertificateFiles()

// Build installation configuration
installConfig := configService.BuildInstallConfig(
    false,    // force
    false,    // dryRun
    true,     // verbose
    "my-cluster",
    appOfAppsConfig,
)

// Get manifests path
manifestsPath := configService.GetDefaultManifestsPath()
```
