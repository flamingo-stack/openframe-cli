This service provides centralized configuration management for chart operations, handling paths, certificates, and installation configurations within the OpenFrame CLI application.

## Key Components

- **Service**: Main configuration service that orchestrates system configuration and path resolution
- **NewService()**: Constructor that initializes the service with system service and path resolver dependencies
- **GetCertificateDirectory/GetCertificateFiles()**: Methods for managing SSL certificate paths
- **GetHelmValuesFile()**: Returns the path to Helm values configuration file
- **Initialize()**: Sets up required directories and initializes the system service
- **BuildInstallConfig()**: Creates complete installation configurations for chart deployments
- **GetDefaultManifestsPath()**: Resolves the default location for Kubernetes manifests

## Usage Example

```go
// Create and initialize the configuration service
configSvc := NewService()
if err := configSvc.Initialize(); err != nil {
    log.Fatal("Failed to initialize config:", err)
}

// Get certificate paths
certFile, keyFile := configSvc.GetCertificateFiles()
fmt.Printf("Cert: %s, Key: %s\n", certFile, keyFile)

// Build installation configuration
installConfig := configSvc.BuildInstallConfig(
    false, // force
    true,  // dryRun
    true,  // verbose
    "my-cluster",
    &models.AppOfAppsConfig{},
)

// Get manifests path
manifestsPath := configSvc.GetDefaultManifestsPath()
```
