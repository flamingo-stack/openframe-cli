This file contains comprehensive unit tests for configuration-related types and structures used in the system. It validates the behavior of ingress types, Docker registry configurations, Ngrok configurations, and chart configurations.

## Key Components

- **IngressType Tests**: Validates constants for localhost and ngrok ingress types
- **DockerRegistryConfig Tests**: Tests creation and field validation for Docker registry credentials
- **NgrokConfig Tests**: Comprehensive testing of Ngrok configuration including authentication, domains, IP allowlists, and registration fields
- **IngressConfig Tests**: Tests ingress configuration with different types (localhost/ngrok)
- **ChartConfiguration Tests**: Tests Helm chart configuration including paths, values, and modification tracking
- **NgrokRegistrationURLs Tests**: Validates constant URLs for Ngrok registration processes

## Usage Example

```go
// Test ingress type validation
func TestMyIngressType(t *testing.T) {
    assert.Equal(t, IngressType("localhost"), IngressTypeLocalhost)
    assert.Equal(t, IngressType("ngrok"), IngressTypeNgrok)
}

// Test Docker registry configuration
func TestDockerConfig(t *testing.T) {
    config := &DockerRegistryConfig{
        Username: "testuser",
        Password: "testpass", 
        Email:    "test@example.com",
    }
    assert.Equal(t, "testuser", config.Username)
}

// Test Ngrok configuration with IP restrictions
func TestNgrokWithIPs(t *testing.T) {
    config := &NgrokConfig{
        AuthToken:     "token_123",
        UseAllowedIPs: true,
        AllowedIPs:    []string{"192.168.1.1"},
    }
    assert.True(t, config.UseAllowedIPs)
    assert.Contains(t, config.AllowedIPs, "192.168.1.1")
}
```
