Test suite for configuration types and constants used in a Helm chart management system. Validates the behavior of ingress configurations, Docker registry settings, and Ngrok tunnel configurations.

## Key Components

- **IngressType Constants Tests**: Validates `IngressTypeLocalhost` and `IngressTypeNgrok` string constants
- **DockerRegistryConfig Tests**: Tests creation and validation of Docker registry credentials
- **NgrokConfig Tests**: Comprehensive testing of Ngrok tunnel configuration including IP allowlisting and registration
- **IngressConfig Tests**: Tests ingress configuration for both localhost and Ngrok types
- **ChartConfiguration Tests**: Tests complete Helm chart configuration with all nested components
- **NgrokRegistrationURLs Tests**: Validates Ngrok dashboard URL constants

## Usage Example

```go
// Testing DockerRegistryConfig creation
config := &DockerRegistryConfig{
    Username: "testuser",
    Password: "testpass", 
    Email:    "test@example.com",
}
assert.Equal(t, "testuser", config.Username)

// Testing NgrokConfig with IP restrictions
ngrokConfig := &NgrokConfig{
    AuthToken:     "auth_token_123",
    Domain:        "example.ngrok.io",
    UseAllowedIPs: true,
    AllowedIPs:    []string{"192.168.1.1", "10.0.0.1"},
}

// Testing complete ChartConfiguration
fullConfig := &ChartConfiguration{
    BaseHelmValuesPath: "/path/to/values.yaml",
    ModifiedSections:   []string{"branch", "docker"},
    IngressConfig:      ingressConfig,
    DockerRegistry:     dockerConfig,
}
```
