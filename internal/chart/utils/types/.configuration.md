This file defines configuration types and constants for managing OpenFrame deployment settings, including Docker registry credentials, ingress configurations, and deployment modes.

## Key Components

- **DeploymentMode**: Enum for OSS, SaaS, and SaaS Shared deployment types
- **IngressType**: Enum for localhost, ngrok, and GCP ingress options
- **DockerRegistryConfig**: Docker registry authentication settings
- **NgrokConfig**: Complete ngrok configuration with auth tokens, domains, and IP allowlists
- **SaaSConfig**: SaaS-specific repository and branch settings
- **ChartConfiguration**: Main configuration struct for Helm chart installations
- **GetRepositoryURL()**: Helper function to resolve repository URLs by deployment mode

## Usage Example

```go
// Create a chart configuration for SaaS deployment
config := &ChartConfiguration{
    BaseHelmValuesPath: "/path/to/helm-values.yaml",
    TempHelmValuesPath: "/tmp/values.yaml",
    DeploymentMode:     &DeploymentModeSaaS,
    DockerRegistry: &DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass",
        Email:    "user@example.com",
    },
    IngressConfig: &IngressConfig{
        Type: IngressTypeNgrok,
        NgrokConfig: &NgrokConfig{
            AuthToken: "your-token",
            Domain:    "myapp.ngrok.io",
            UseAllowedIPs: true,
            AllowedIPs: []string{"192.168.1.0/24"},
        },
    },
}

// Get repository URL for deployment mode
repoURL := GetRepositoryURL(DeploymentModeSaaS)
// Returns: "https://github.com/flamingo-stack/openframe-oss-tenant"
```
