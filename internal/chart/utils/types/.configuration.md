Defines configuration types and structures for managing OpenFrame deployment settings, including Docker registry, ingress, and SaaS configurations. This file serves as the central configuration schema for different deployment modes and external service integrations.

## Key Components

- **DeploymentMode**: Constants for OSS, SaaS, and SaaS Shared deployment types
- **IngressType**: Enumeration for ingress options (localhost, Ngrok, GCP)
- **NgrokConfig**: Complete Ngrok tunnel configuration with authentication and IP allowlisting
- **ChartConfiguration**: Main configuration structure for Helm chart installations
- **DockerRegistryConfig**: Docker registry authentication settings
- **SaaSConfig**: SaaS-specific deployment parameters
- **GetRepositoryURL()**: Function to resolve repository URLs based on deployment mode

## Usage Example

```go
// Create a new chart configuration
config := &ChartConfiguration{
    BaseHelmValuesPath: "/path/to/helm-values.yaml",
    TempHelmValuesPath: "/tmp/values.yaml",
    DeploymentMode:     &DeploymentModeOSS,
}

// Configure Docker registry
dockerConfig := &DockerRegistryConfig{
    Username: "myuser",
    Password: "mypass",
    Email:    "user@example.com",
}
config.DockerRegistry = dockerConfig

// Set up Ngrok ingress
ngrokConfig := &NgrokConfig{
    AuthToken: "your-token",
    APIKey:    "your-api-key",
    Domain:    "myapp.ngrok.io",
    UseAllowedIPs: true,
    AllowedIPs: []string{"192.168.1.0/24"},
}
config.IngressConfig = &IngressConfig{
    Type:        IngressTypeNgrok,
    NgrokConfig: ngrokConfig,
}

// Get repository URL for deployment mode
repoURL := GetRepositoryURL(DeploymentModeOSS)
// Returns: "https://github.com/flamingo-stack/openframe-oss-tenant"
```
