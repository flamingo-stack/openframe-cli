Defines configuration types and constants for OpenFrame deployment management, including Docker registry settings, ingress configuration, and deployment modes (OSS, SaaS, SaaS Shared).

## Key Components

- **Configuration Types**: `DockerRegistryConfig`, `NgrokConfig`, `SaaSConfig`, `IngressConfig` for deployment settings
- **Enums**: `DeploymentMode` (OSS/SaaS/SaaS Shared) and `IngressType` (localhost/ngrok/GCP) constants
- **Main Config**: `ChartConfiguration` struct that aggregates all deployment configuration options
- **Helper Function**: `GetRepositoryURL()` returns appropriate Git repository based on deployment mode
- **Constants**: `NgrokRegistrationURLs` struct with Ngrok dashboard and documentation URLs

## Usage Example

```go
// Create configuration for SaaS deployment with Ngrok ingress
config := &ChartConfiguration{
    BaseHelmValuesPath: "/path/to/helm-values.yaml",
    TempHelmValuesPath: "/tmp/modified-values.yaml",
    DeploymentMode: &[]DeploymentMode{DeploymentModeSaaS}[0],
    IngressConfig: &IngressConfig{
        Type: IngressTypeNgrok,
        NgrokConfig: &NgrokConfig{
            AuthToken: "your-token",
            Domain: "your-domain.ngrok.io",
            UseAllowedIPs: true,
            AllowedIPs: []string{"192.168.1.0/24"},
        },
    },
    DockerRegistry: &DockerRegistryConfig{
        Username: "user",
        Password: "pass",
        Email: "user@example.com",
    },
}

// Get repository URL for deployment
repoURL := GetRepositoryURL(DeploymentModeSaaS)
// Returns: "https://github.com/flamingo-stack/openframe-oss-tenant"
```
