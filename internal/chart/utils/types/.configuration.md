Defines configuration types and structures for OpenFrame deployment settings, supporting multiple deployment modes (OSS, SaaS, SaaS Shared) and ingress options (localhost, ngrok, GCP).

## Key Components

- **DeploymentMode**: Constants for OSS, SaaS, and SaaS Shared deployment types
- **IngressType**: Enum for localhost, ngrok, and GCP ingress configurations
- **NgrokConfig**: Complete ngrok tunnel configuration including auth tokens, domains, and IP allowlists
- **SaaSConfig**: SaaS-specific settings for repository passwords and branch selection
- **ChartConfiguration**: Main configuration struct for Helm chart installation with deployment parameters
- **GetRepositoryURL()**: Function that returns appropriate GitHub repository URL based on deployment mode

## Usage Example

```go
// Create a new chart configuration for SaaS deployment
config := &ChartConfiguration{
    BaseHelmValuesPath: "./helm-values.yaml",
    TempHelmValuesPath: "/tmp/values.yaml",
    DeploymentMode: &[]DeploymentMode{DeploymentModeSaaS}[0],
    DockerRegistry: &DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass",
        Email:    "user@example.com",
    },
    IngressConfig: &IngressConfig{
        Type: IngressTypeNgrok,
        NgrokConfig: &NgrokConfig{
            AuthToken: "your_token_here",
            Domain:    "myapp.ngrok.io",
            UseAllowedIPs: true,
            AllowedIPs: []string{"192.168.1.0/24"},
        },
    },
}

// Get repository URL for the deployment mode
repoURL := GetRepositoryURL(DeploymentModeSaaS)
// Returns: "https://github.com/flamingo-stack/openframe-saas-tenant"
```
