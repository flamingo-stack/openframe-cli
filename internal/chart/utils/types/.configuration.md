Defines configuration types and constants for managing deployment modes, ingress settings, and Helm chart configurations in a multi-tenant application deployment system.

## Key Components

**Types:**
- `DockerRegistryConfig` - Docker registry authentication settings
- `DeploymentMode` - Enum for deployment types (OSS, SaaS, SaaS Shared)
- `IngressType` - Enum for ingress types (localhost, ngrok, GCP)
- `NgrokConfig` - Ngrok tunnel configuration with IP allowlisting
- `SaaSConfig` - SaaS-specific repository settings
- `IngressConfig` - Ingress configuration wrapper
- `ChartConfiguration` - Complete Helm chart configuration state

**Constants:**
- Deployment mode constants (`DeploymentModeOSS`, `DeploymentModeSaaS`, etc.)
- Ingress type constants (`IngressTypeLocalhost`, `IngressTypeNgrok`, etc.)
- `NgrokRegistrationURLs` - Predefined Ngrok documentation URLs

**Functions:**
- `GetRepositoryURL(mode DeploymentMode)` - Returns repository URL based on deployment mode

## Usage Example

```go
// Configure for SaaS deployment with Ngrok ingress
config := &ChartConfiguration{
    DeploymentMode: &[]DeploymentMode{DeploymentModeSaaS}[0],
    IngressConfig: &IngressConfig{
        Type: IngressTypeNgrok,
        NgrokConfig: &NgrokConfig{
            AuthToken: "your-auth-token",
            Domain:    "your-domain.ngrok.io",
            UseAllowedIPs: true,
            AllowedIPs: []string{"192.168.1.0/24"},
        },
    },
}

// Get repository URL
repoURL := GetRepositoryURL(DeploymentModeSaaS)
```
