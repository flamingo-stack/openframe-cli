Defines comprehensive interfaces for managing Kubernetes chart installations, ArgoCD deployments, and Git operations within the OpenFrame CLI system. This file establishes the contract layer that enables dependency injection and modular architecture across different chart management components.

## Key Components

### Core Service Interfaces
- **ChartInstaller** - Orchestrates the complete chart installation process
- **ArgoCDService** - Manages ArgoCD installation and lifecycle operations
- **AppOfAppsService** - Handles app-of-apps pattern installations

### Provider Interfaces
- **HelmProvider** - Manages Helm chart operations and status checking
- **GitProvider** - Handles Git repository cloning and cleanup
- **ClusterLister** - Provides Kubernetes cluster listing capabilities

### Configuration & UI Interfaces
- **ConfigBuilder** - Constructs installation configurations
- **PathResolver** - Resolves certificate and configuration file paths
- **OperationsUI** - Provides user interaction capabilities for installations
- **ClusterSelector** - Handles cluster selection logic

### Orchestration & Factory Interfaces
- **ServiceOrchestrator** - Coordinates service interactions
- **WorkflowExecutor** - Executes complex workflows with step tracking
- **ServiceFactory** - Creates service instances with dependency injection

### Result Types
- **WorkflowResult** - Contains workflow execution results and timing
- **StepResult** - Represents individual workflow step outcomes
- **InstallationRequest** - Encapsulates all installation parameters

## Usage Example

```go
// Using the interfaces for dependency injection
type ChartManager struct {
    installer    ChartInstaller
    configBuilder ConfigBuilder
    ui           OperationsUI
}

func (cm *ChartManager) InstallCharts(req InstallationRequest) error {
    // Build configuration
    config, err := cm.configBuilder.BuildInstallConfig(
        req.Force, req.DryRun, req.Verbose, 
        "cluster-name", req.GitHubRepo, req.GitHubBranch, req.CertDir,
    )
    if err != nil {
        return err
    }

    // Execute installation
    return cm.installer.InstallCharts(config)
}

// Factory usage
func NewChartManager(factory ServiceFactory) *ChartManager {
    return &ChartManager{
        installer:     factory.CreateInstaller(),
        configBuilder: factory.CreateConfigBuilder(),
        ui:           factory.GetOperationsUI(),
    }
}
```
