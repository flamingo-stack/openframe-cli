This file defines the complete interface architecture for OpenFrame's chart installation system, providing contracts for Helm operations, ArgoCD management, Git repository handling, and user interactions.

## Key Components

**Core Services:**
- `ChartInstaller` - Main orchestrator for chart installation workflows
- `ArgoCDService` - Manages ArgoCD installation and application lifecycle
- `AppOfAppsService` - Handles app-of-apps pattern deployment
- `ServiceOrchestrator` - Coordinates service interactions during installation

**Provider Interfaces:**
- `HelmProvider` - Helm chart operations and status checking
- `GitProvider` - Git repository cloning and cleanup
- `ClusterLister` - Kubernetes cluster discovery

**Configuration & UI:**
- `ConfigBuilder` - Constructs installation configurations
- `PathResolver` - Resolves certificate and values file paths
- `OperationsUI` - User interaction prompts and feedback
- `ClusterSelector` - Interactive cluster selection

**Workflow Management:**
- `WorkflowExecutor` - Step-by-step workflow execution with tracking
- `ServiceFactory` - Dependency injection and service creation

## Usage Example

```go
// Create services using factory pattern
factory := &serviceFactory{}
installer := factory.CreateInstaller()

// Build installation configuration
configBuilder := factory.CreateConfigBuilder()
config, err := configBuilder.BuildInstallConfig(
    false, // force
    false, // dryRun
    true,  // verbose
    "my-cluster",
    "https://github.com/user/charts",
    "main",
    "/path/to/certs",
)

// Execute installation
if err := installer.InstallCharts(config); err != nil {
    log.Fatal(err)
}
```
