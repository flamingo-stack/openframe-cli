This file defines the complete interface contract for the OpenFrame CLI chart management system, establishing clear boundaries between different components and their responsibilities.

## Key Components

**Core Service Interfaces:**
- `ChartInstaller` - Main orchestrator for chart installation process
- `ArgoCDService` - Manages ArgoCD installation and lifecycle
- `AppOfAppsService` - Handles app-of-apps pattern installations

**Provider Interfaces:**
- `HelmProvider` - Abstracts Helm chart operations and status checking
- `GitProvider` - Manages Git repository cloning and cleanup
- `ClusterLister` - Provides cluster discovery capabilities

**Configuration & UI Interfaces:**
- `ConfigBuilder` - Constructs installation configurations
- `PathResolver` - Resolves certificate and configuration file paths
- `OperationsUI` - Handles user interactions and prompts
- `ClusterSelector` - Manages cluster selection logic

**Orchestration Types:**
- `WorkflowExecutor` - Executes multi-step workflows with tracking
- `InstallationRequest` - Contains all installation parameters
- `WorkflowResult`/`StepResult` - Capture execution results and timing

## Usage Example

```go
// Using the interfaces for dependency injection
func NewChartCommand(factory ServiceFactory, clusterLister ClusterLister) *cobra.Command {
    installer := factory.CreateInstaller()
    ui := factory.GetOperationsUI()
    selector := factory.CreateClusterSelector(clusterLister)
    
    return &cobra.Command{
        RunE: func(cmd *cobra.Command, args []string) error {
            // Build installation request
            req := InstallationRequest{
                Args:         args,
                Force:        force,
                DryRun:       dryRun,
                GitHubRepo:   repo,
                GitHubBranch: branch,
            }
            
            // Execute through orchestrator
            orchestrator := NewServiceOrchestrator(installer, ui, selector)
            return orchestrator.ExecuteInstallation(req)
        },
    }
}
```
