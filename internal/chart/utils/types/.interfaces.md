This file defines the complete interface layer for the OpenFrame CLI chart management system, providing contracts for service orchestration, Helm operations, Git management, and user interactions.

## Key Components

**Core Services:**
- `ChartInstaller` - Main orchestrator for chart installation workflows
- `ServiceOrchestrator` - Manages service coordination and execution
- `WorkflowExecutor` - Handles complex multi-step workflows with tracking

**Provider Interfaces:**
- `HelmProvider` - Manages Helm chart operations (ArgoCD, app-of-apps)
- `GitProvider` - Handles Git repository cloning and cleanup
- `ClusterLister` - Provides cluster discovery capabilities

**Service Components:**
- `ArgoCDService` - ArgoCD lifecycle management (install, status, wait)
- `AppOfAppsService` - App-of-apps pattern management

**Configuration & UI:**
- `ConfigBuilder` - Constructs installation configurations
- `PathResolver` - Resolves certificate and configuration paths
- `OperationsUI` - User interaction interface for installations
- `ClusterSelector` - Interactive cluster selection

**Result Types:**
- `WorkflowResult` - Complete workflow execution results with timing
- `StepResult` - Individual step execution details
- `InstallationRequest` - Installation parameters container

## Usage Example

```go
// Implement a service orchestrator
type orchestrator struct {
    installer ChartInstaller
    ui        OperationsUI
}

func (o *orchestrator) ExecuteInstallation(req InstallationRequest) error {
    // Use the installer through the interface
    config, err := o.configBuilder.BuildInstallConfig(
        req.Force, req.DryRun, req.Verbose,
        req.ClusterName, req.GitHubRepo, req.GitHubBranch, req.CertDir,
    )
    if err != nil {
        return err
    }
    
    return o.installer.InstallCharts(config)
}
```
