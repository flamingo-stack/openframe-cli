Defines the core interfaces for the OpenFrame CLI chart installation system, providing contracts for Helm operations, Git repository management, ArgoCD services, and workflow orchestration.

## Key Components

**Core Service Interfaces:**
- `ChartInstaller` - Main orchestrator for chart installation processes
- `ServiceOrchestrator` - Coordinates service interactions during installation
- `WorkflowExecutor` - Executes multi-step workflows with progress tracking

**Provider Interfaces:**
- `HelmProvider` - Manages Helm chart operations and status checks
- `GitProvider` - Handles Git repository cloning and cleanup
- `ClusterLister` - Provides Kubernetes cluster discovery capabilities

**Service Components:**
- `ArgoCDService` - Manages ArgoCD installation and lifecycle
- `AppOfAppsService` - Handles app-of-apps pattern deployment
- `ConfigBuilder` - Constructs installation configurations
- `PathResolver` - Resolves file paths for certificates and configurations

**UI & Factory Interfaces:**
- `OperationsUI` - Provides user interaction capabilities
- `ClusterSelector` - Handles cluster selection logic
- `ServiceFactory` - Creates service instances with dependency injection

    gitProvider  GitProvider
    ui          OperationsUI
}

func (c *ChartInstallerImpl) InstallCharts(config config.ChartInstallConfig) error {
    // Installation logic using injected providers
    return nil
}

// Create installation request
req := InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          false,
    DryRun:         false,
    Verbose:        true,
    GitHubRepo:     "flamingo-stack/charts",
    GitHubBranch:   "main",

    NonInteractive: false,

// Execute through orchestrator
orchestrator.ExecuteInstallation(req)
