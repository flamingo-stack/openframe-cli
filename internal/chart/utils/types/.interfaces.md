This file defines the complete interface architecture for the OpenFrame CLI chart installation system, providing contracts for services, providers, and orchestration components.

## Key Components

### Core Service Interfaces
- **ChartInstaller** - Main orchestrator for chart installation workflows
- **ServiceOrchestrator** - Coordinates service interactions during installation
- **WorkflowExecutor** - Executes multi-step workflows with progress tracking

### Provider Interfaces
- **HelmProvider** - Manages Helm operations (ArgoCD, app-of-apps installation)
- **GitProvider** - Handles Git repository cloning and cleanup
- **ClusterLister** - Provides Kubernetes cluster discovery

### Service Components
- **ArgoCDService** - ArgoCD lifecycle management (install, status, wait for apps)
- **AppOfAppsService** - App-of-apps pattern installation and monitoring
- **ConfigBuilder** - Constructs installation configurations
- **PathResolver** - Resolves certificate and configuration file paths

### UI Interfaces
- **ClusterSelector** - Interactive cluster selection
- **OperationsUI** - User interaction for confirmations and credential prompts

### Result Types
- **WorkflowResult** - Complete workflow execution results with timing
- **StepResult** - Individual step outcomes with error handling
- **InstallationRequest** - Installation parameters and configuration

## Usage Example

```go
// Service factory creates configured instances
factory := NewServiceFactory()
installer := factory.CreateInstaller()

// Installation request with all parameters
req := InstallationRequest{
    Args:           []string{"my-cluster"},
    Force:          false,
    DryRun:         false,
    Verbose:        true,
    GitHubRepo:     "company/charts",
    GitHubBranch:   "main",
    CertDir:        "./certs",
    DeploymentMode: "oss-tenant",
    NonInteractive: false,
}

// Execute installation through orchestrator
orchestrator := factory.CreateOrchestrator()
err := orchestrator.ExecuteInstallation(req)
```
