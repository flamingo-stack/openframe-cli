This file provides a service for displaying chart-related UI operations with contextual icons and formatted messages using the pterm library.

## Key Components

- **DisplayService**: Main struct handling all UI display operations
- **NewDisplayService()**: Constructor function returning a new DisplayService instance
- **ShowInstallProgress()**: Displays progress messages with chart-type specific icons (üöÄ for ArgoCD, üì¶ for App-of-Apps)
- **ShowInstallSuccess()**: Handles successful installation display (currently a no-op)
- **ShowInstallError()**: Shows installation error messages
- **ShowSkippedInstallation()**: Displays when component installation is skipped
- **ShowPreInstallCheck()**: Shows pre-installation check messages with üîç icon
- **ShowDryRunResults()**: Outputs dry-run results to specified writer
- **getChartDisplayName()**: Returns user-friendly names for chart types

## Usage Example

```go
// Create a new display service
displayService := NewDisplayService()

// Show installation progress
displayService.ShowInstallProgress(models.ChartTypeArgoCD, "Installing ArgoCD components...")

// Display pre-installation check
displayService.ShowPreInstallCheck("Checking cluster connectivity")

// Show error if installation fails
displayService.ShowInstallError(models.ChartTypeArgoCD, fmt.Errorf("connection timeout"))

// Show skipped installation
displayService.ShowSkippedInstallation("Prometheus", "already installed")

// Display dry-run results
var results []string
displayService.ShowDryRunResults(os.Stdout, results)
```
