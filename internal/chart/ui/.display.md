This service provides UI display functionality for chart installation operations, handling progress messages, success/error states, and dry-run outputs with visual formatting.

## Key Components

- **DisplayService**: Main service struct for chart-related UI operations
- **NewDisplayService()**: Factory function creating new display service instances
- **ShowInstallProgress()**: Displays progress messages with chart-type-specific icons
- **ShowInstallSuccess()**: Handles successful installation display (placeholder implementation)
- **ShowInstallError()**: Shows error messages for failed installations
- **ShowSkippedInstallation()**: Displays when component installation is skipped
- **ShowPreInstallCheck()**: Shows pre-installation validation messages
- **ShowDryRunResults()**: Formats and displays dry-run operation results
- **getChartDisplayName()**: Helper method converting chart types to user-friendly names

## Usage Example

```go
// Create display service
display := NewDisplayService()

// Show installation progress
display.ShowInstallProgress(models.ChartTypeArgoCD, "Installing ArgoCD components...")

// Display pre-installation checks
display.ShowPreInstallCheck("Validating Kubernetes cluster connectivity")

// Show skipped installation
display.ShowSkippedInstallation("ArgoCD", "already installed")

// Display dry-run results
results := []string{
    "ArgoCD namespace would be created",
    "ConfigMap argocd-config would be applied",
}
display.ShowDryRunResults(os.Stdout, results)

// Show installation error
display.ShowInstallError(models.ChartTypeArgoCD, fmt.Errorf("timeout waiting for pods"))
```
