A service that provides chart-specific UI display functionality for installation progress, status updates, and error messages using the pterm library for enhanced terminal output.

## Key Components

**DisplayService**: Main service struct that handles all chart-related display operations

**Core Methods**:
- `NewDisplayService()` - Factory function that creates a new display service instance
- `ShowInstallProgress()` - Displays installation progress with chart-type-specific icons
- `ShowInstallSuccess()` - Handles successful installation display (delegated to service layer)
- `ShowInstallError()` - Shows formatted error messages for failed installations
- `ShowSkippedInstallation()` - Displays when component installation is bypassed
- `ShowPreInstallCheck()` - Shows pre-installation validation messages
- `ShowDryRunResults()` - Outputs dry-run simulation results to specified writer

**Helper Methods**:
- `getChartDisplayName()` - Converts chart types to user-friendly display names

## Usage Example

```go
package main

import (
    "os"
    "github.com/flamingo-stack/openframe-cli/internal/chart/models"
    "github.com/flamingo-stack/openframe-cli/internal/ui"
)

func main() {
    display := ui.NewDisplayService()
    
    // Show installation progress
    display.ShowInstallProgress(models.ChartTypeArgoCD, "Installing ArgoCD components...")
    
    // Show pre-installation checks
    display.ShowPreInstallCheck("Validating Kubernetes cluster connection")
    
    // Display dry-run results
    results := []string{"ArgoCD namespace created", "RBAC policies applied"}
    display.ShowDryRunResults(os.Stdout, results)
    
    // Show skipped installation
    display.ShowSkippedInstallation("Prometheus", "already exists in cluster")
}
```
