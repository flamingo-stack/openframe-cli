Provides UI display functionality for chart installation operations, handling progress indicators, success/error messages, and formatted output using the pterm library.

## Key Components

- **DisplayService**: Main service struct for handling UI display operations
- **NewDisplayService()**: Constructor function returning a new DisplayService instance
- **ShowInstallProgress()**: Displays installation progress with chart-type specific icons
- **ShowInstallSuccess()**: Handles successful installation display (placeholder implementation)
- **ShowInstallError()**: Shows installation error messages
- **ShowSkippedInstallation()**: Displays when installations are skipped with reasons
- **ShowPreInstallCheck()**: Shows pre-installation validation messages
- **ShowDryRunResults()**: Formats and displays dry-run output results
- **getChartDisplayName()**: Returns user-friendly names for chart types

## Usage Example

```go
// Create display service
displayService := NewDisplayService()

// Show installation progress
displayService.ShowInstallProgress(models.ChartTypeArgoCD, "Installing ArgoCD chart...")

// Display pre-installation checks
displayService.ShowPreInstallCheck("Validating cluster connectivity")

// Show skipped installation
displayService.ShowSkippedInstallation("ArgoCD", "already installed")

// Display dry-run results
results := []string{"Chart validation passed", "Dependencies resolved"}
displayService.ShowDryRunResults(os.Stdout, results)

// Show installation error
err := errors.New("connection timeout")
displayService.ShowInstallError(models.ChartTypeArgoCD, err)
```
