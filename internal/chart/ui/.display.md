This file provides a UI display service for handling chart installation progress and status messages with context-specific icons and formatting.

## Key Components

**DisplayService**: Main service struct that handles all chart-related UI display operations.

**Core Methods**:
- `NewDisplayService()` - Creates a new display service instance
- `ShowInstallProgress()` - Shows installation progress with chart-type specific icons (üöÄ for ArgoCD, üì¶ for App-of-Apps)
- `ShowInstallSuccess()` - Placeholder for success messages (actual display handled by service layer)
- `ShowInstallError()` - Displays installation errors with error formatting
- `ShowSkippedInstallation()` - Shows when installations are skipped with reason
- `ShowPreInstallCheck()` - Displays pre-installation check messages with üîç icon
- `ShowDryRunResults()` - Outputs dry-run results to specified writer
- `getChartDisplayName()` - Returns user-friendly names for chart types

## Usage Example

```go
package main

import (
    "os"
    "github.com/flamingo-stack/openframe-cli/internal/chart/models"
    "github.com/flamingo-stack/openframe-cli/internal/ui"
)

func main() {
    display := ui.NewDisplayService()
    
    // Show installation progress
    display.ShowInstallProgress(models.ChartTypeArgoCD, "Installing ArgoCD components...")
    
    // Show pre-installation check
    display.ShowPreInstallCheck("Checking cluster connectivity")
    
    // Show skipped installation
    display.ShowSkippedInstallation("Prometheus", "already exists")
    
    // Display dry-run results
    results := []string{"helm install argocd", "kubectl apply -f manifests/"}
    display.ShowDryRunResults(os.Stdout, results)
}
```
