Provides user interface components and operations for managing chart installations in the OpenFrame CLI tool. This package handles user interactions for cluster selection, installation confirmations, progress displays, and credential prompting.

## Key Components

- **OperationsUI**: Main struct that orchestrates chart operation user interfaces
- **NewOperationsUI()**: Factory function that creates a new operations UI service
- **SelectClusterForInstall()**: Handles cluster selection for chart installations
- **ConfirmInstallation()/ConfirmInstallationOnCluster()**: User confirmation prompts before installation
- **ShowInstallationStart()/ShowInstallationComplete()**: Installation progress messaging
- **ShowInstallationError()**: Error handling with troubleshooting steps
- **ShowCloneProgress()/ShowCloneComplete()**: Git repository cloning progress
- **PromptForGitHubCredentials()**: Interactive credential collection for GitHub access

## Usage Example

```go
// Create operations UI service
opsUI := NewOperationsUI()

// Select cluster for installation
clusters := []models.ClusterInfo{...}
clusterName, err := opsUI.SelectClusterForInstall(clusters, args)
if err != nil {
    return err
}

// Confirm installation with user
confirmed, err := opsUI.ConfirmInstallation(clusterName)
if err != nil || !confirmed {
    opsUI.ShowOperationCancelled("installation")
    return nil
}

// Show installation progress
opsUI.ShowInstallationStart(clusterName)

// Handle installation completion or errors
if installErr != nil {
    opsUI.ShowInstallationError(installErr)
} else {
    opsUI.ShowInstallationComplete()
}
```
