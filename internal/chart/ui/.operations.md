Provides user interface components for managing Helm chart operations including installation, cluster selection, and progress feedback with consistent messaging and user prompts.

## Key Components

- **OperationsUI**: Main struct that orchestrates chart operation UI components including cluster selector, credentials prompter, and message templates
- **SelectClusterForInstall()**: Handles interactive cluster selection for chart installations
- **ConfirmInstallation()**: Prompts user for confirmation before starting chart installation with time estimates
- **ShowInstallationComplete()**: Displays success message with helpful next steps for accessing ArgoCD and applications
- **ShowInstallationError()**: Shows formatted error messages with troubleshooting steps for failed installations
- **PromptForGitHubCredentials()**: Handles secure credential collection for GitHub repository access

## Usage Example

```go
// Create operations UI service
ui := NewOperationsUI()

// Select cluster for installation
clusters := []models.ClusterInfo{{Name: "dev-cluster"}, {Name: "prod-cluster"}}
selectedCluster, err := ui.SelectClusterForInstall(clusters, []string{"dev-cluster"})
if err != nil {
    return err
}

// Confirm installation with user
confirmed, err := ui.ConfirmInstallation(selectedCluster)
if !confirmed {
    ui.ShowOperationCancelled("installation")
    return nil
}

// Show installation progress
ui.ShowInstallationStart(selectedCluster)

// Handle completion or errors
if err := performInstallation(); err != nil {
    ui.ShowInstallationError(err)
} else {
    ui.ShowInstallationComplete()
}
```
