Provides user interface components for chart operations including cluster selection, installation confirmation, and progress messaging.

## Key Components

- **OperationsUI**: Main UI service that orchestrates chart operation interfaces
- **SelectClusterForInstall**: Interactive cluster selection for chart installations
- **ConfirmInstallation/ConfirmInstallationOnCluster**: User confirmation prompts with installation time estimates
- **ShowInstallationStart/Complete/Error**: Progress and status messaging for chart installations
- **ShowCloneProgress/Complete**: Repository cloning status updates
- **PromptForGitHubCredentials**: Interactive GitHub authentication prompt

## Usage Example

```go
// Initialize chart operations UI
ui := NewOperationsUI()

// Select cluster for installation
clusterName, err := ui.SelectClusterForInstall(clusters, args)
if err != nil {
    return err
}

// Confirm installation
confirmed, err := ui.ConfirmInstallation(clusterName)
if err != nil {
    return err
}

if !confirmed {
    ui.ShowOperationCancelled("installation")
    return nil
}

// Show installation progress
ui.ShowInstallationStart(clusterName)

// After installation completes
ui.ShowInstallationComplete()

// Handle GitHub authentication if needed
username, token, err := ui.PromptForGitHubCredentials("https://github.com/user/repo")
```

The service integrates cluster selection, credential prompting, and messaging templates to provide a consistent user experience across chart operations. It includes helpful troubleshooting information and next-step guidance for successful installations.
