Provides user interface components for Helm chart operations including installation workflows, cluster selection, and user confirmations with consistent messaging and error handling.

## Key Components

- **OperationsUI**: Main struct that orchestrates chart operation workflows
- **NewOperationsUI()**: Constructor for creating operations UI instances
- **SelectClusterForInstall()**: Handles cluster selection for chart installations
- **ConfirmInstallation()**: User confirmation prompts for installation operations
- **ShowInstallationStart/Complete/Error()**: Status messages for installation lifecycle
- **PromptForGitHubCredentials()**: Interactive credential collection for private repositories

## Usage Example

```go
// Create operations UI
ui := NewOperationsUI()

// Select cluster for installation
clusters := []models.ClusterInfo{...}
clusterName, err := ui.SelectClusterForInstall(clusters, []string{"my-cluster"})
if err != nil {
    return err
}

// Confirm installation
confirmed, err := ui.ConfirmInstallation(clusterName)
if err != nil {
    return err
}

if confirmed {
    ui.ShowInstallationStart(clusterName)
    // Perform installation...
    ui.ShowInstallationComplete()
} else {
    ui.ShowOperationCancelled("installation")
}
```

The UI provides consistent messaging patterns and integrates with cluster selection, credential prompting, and progress indicators for a cohesive user experience during chart operations.
