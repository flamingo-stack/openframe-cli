Provides user interface components for managing chart operations, including cluster selection, installation confirmations, and progress feedback for OpenFrame chart deployments.

## Key Components

- **OperationsUI**: Main struct containing cluster selector, credentials prompter, and message templates
- **NewOperationsUI()**: Factory function to create a new operations UI instance
- **SelectClusterForInstall()**: Handles cluster selection for chart installations
- **ConfirmInstallation()** / **ConfirmInstallationOnCluster()**: User confirmation prompts before installation
- **ShowInstallationStart()** / **ShowInstallationComplete()** / **ShowInstallationError()**: Installation status messages
- **ShowCloneProgress()** / **ShowCloneComplete()**: Git repository cloning feedback
- **PromptForGitHubCredentials()**: Interactive GitHub authentication prompt

## Usage Example

```go
// Create operations UI
ui := NewOperationsUI()

// Select cluster for installation
clusters := []models.ClusterInfo{...}
clusterName, err := ui.SelectClusterForInstall(clusters, os.Args)

// Confirm installation
confirmed, err := ui.ConfirmInstallation(clusterName)
if !confirmed {
    ui.ShowOperationCancelled("installation")
    return
}

// Show installation progress
ui.ShowInstallationStart(clusterName)
// ... perform installation ...
ui.ShowInstallationComplete()

// Handle GitHub authentication if needed
username, token, err := ui.PromptForGitHubCredentials("https://github.com/user/repo")
```
