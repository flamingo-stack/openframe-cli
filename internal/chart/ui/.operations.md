Provides user interface components for managing chart operations in OpenFrame clusters, including installation confirmations, progress tracking, and credential handling.

## Key Components

- **OperationsUI**: Main struct that orchestrates chart operation UI workflows
- **NewOperationsUI()**: Factory function that creates a new operations UI instance
- **SelectClusterForInstall()**: Handles cluster selection for chart installations
- **ConfirmInstallation/ConfirmInstallationOnCluster()**: Interactive confirmation prompts
- **ShowInstallationStart/Complete/Error()**: Status display methods for installation lifecycle
- **ShowCloneProgress/Complete()**: Git repository cloning progress indicators
- **PromptForGitHubCredentials()**: Interactive GitHub credential collection

## Usage Example

```go
// Create operations UI
opsUI := NewOperationsUI()

// Select cluster for installation
clusters := []models.ClusterInfo{...}
clusterName, err := opsUI.SelectClusterForInstall(clusters, []string{"dev-cluster"})
if err != nil {
    return err
}

// Confirm installation
confirmed, err := opsUI.ConfirmInstallation(clusterName)
if err != nil || !confirmed {
    opsUI.ShowOperationCancelled("installation")
    return nil
}

// Show installation progress
opsUI.ShowInstallationStart(clusterName)

// After successful installation
opsUI.ShowInstallationComplete()

// Or on error
opsUI.ShowInstallationError(err)
```

The UI provides consistent messaging, interactive prompts, and helpful troubleshooting information for chart operations across the OpenFrame CLI.
