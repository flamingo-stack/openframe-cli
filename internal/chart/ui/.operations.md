This file provides a user interface layer for chart operations, specifically handling user interactions during OpenFrame chart installations on Kubernetes clusters.

## Key Components

- **OperationsUI**: Main struct that orchestrates chart operation workflows
- **NewOperationsUI()**: Factory function for creating UI service instances
- **SelectClusterForInstall()**: Interactive cluster selection for chart installations
- **ConfirmInstallation/ConfirmInstallationOnCluster()**: User confirmation dialogs before installation
- **ShowInstallationStart/Complete/Error()**: Installation status messaging
- **ShowCloneProgress/Complete()**: Git repository cloning progress indicators
- **PromptForGitHubCredentials()**: Interactive GitHub authentication prompts

## Usage Example

```go
// Create operations UI service
ui := NewOperationsUI()

// Select cluster for installation
clusterName, err := ui.SelectClusterForInstall(clusters, args)
if err != nil {
    return err
}

// Confirm installation
confirmed, err := ui.ConfirmInstallation(clusterName)
if err != nil || !confirmed {
    ui.ShowOperationCancelled("installation")
    return nil
}

// Show installation progress
ui.ShowInstallationStart(clusterName)

// After installation completes
ui.ShowInstallationComplete()

// Handle GitHub authentication if needed
username, token, err := ui.PromptForGitHubCredentials("https://github.com/repo")
```
