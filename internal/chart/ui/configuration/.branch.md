Handles Git branch configuration for Helm chart deployments, providing an interactive interface to select or specify OSS repository branches for non-SaaS deployment modes.

## Key Components

- **BranchConfigurator**: Main struct that handles Git branch configuration with an embedded `HelmValuesModifier`
- **NewBranchConfigurator()**: Factory function that creates a new branch configurator instance
- **Configure()**: Interactive method that prompts users to configure Git branch settings for OSS deployments

## Usage Example

```go
// Create a branch configurator with a Helm values modifier
modifier := templates.NewHelmValuesModifier()
branchConfig := configuration.NewBranchConfigurator(modifier)

// Configure branch settings for a chart
chartConfig := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    ExistingValues: map[string]interface{}{},
}

err := branchConfig.Configure(chartConfig)
if err != nil {
    log.Fatalf("Branch configuration failed: %v", err)
}

// Check if branch was modified
if chartConfig.Branch != nil {
    fmt.Printf("Selected branch: %s\n", *chartConfig.Branch)
}
```

The configurator automatically skips branch configuration for SaaS modes and only prompts for OSS deployments, allowing users to either keep the current branch or specify a custom one through an interactive terminal interface.
