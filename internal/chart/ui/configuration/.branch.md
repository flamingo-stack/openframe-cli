Handles Git branch configuration for OpenFrame Chart deployments based on the deployment mode.

## Key Components

- **BranchConfigurator**: Main configurator struct that manages Git branch settings
- **NewBranchConfigurator**: Constructor that creates a new branch configurator with a Helm values modifier
- **Configure**: Interactive method that prompts users to configure Git branch settings for OSS deployments

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/ui/templates"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/types"
)

// Create a branch configurator
modifier := templates.NewHelmValuesModifier()
branchConfig := NewBranchConfigurator(modifier)

// Configure branch for a chart
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
}

err := branchConfig.Configure(config)
if err != nil {
    log.Printf("Branch configuration failed: %v", err)
}

// Check if branch was modified
if config.Branch != nil {
    fmt.Printf("Selected branch: %s", *config.Branch)
}
```

The configurator automatically skips branch configuration for SaaS deployments and allows users to either keep the current OSS repository branch or specify a custom one through an interactive interface.
