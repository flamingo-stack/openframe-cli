Handles Git branch configuration for OSS deployments, allowing users to specify which branch to use for the OpenFrame repository.

## Key Components

**BranchConfigurator**
- `NewBranchConfigurator(modifier *templates.HelmValuesModifier) *BranchConfigurator` - Creates a new branch configurator instance
- `Configure(config *types.ChartConfiguration) error` - Interactive branch configuration workflow

## Usage Example

```go
// Create branch configurator
modifier := templates.NewHelmValuesModifier()
branchConfig := configuration.NewBranchConfigurator(modifier)

// Configure branch settings
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
}

err := branchConfig.Configure(config)
if err != nil {
    log.Fatal(err)
}

// Branch configuration is stored in config.Branch
if config.Branch != nil {
    fmt.Printf("Selected branch: %s\n", *config.Branch)
}
```

The configurator skips branch selection for SaaS deployments (handled elsewhere) and presents OSS users with options to keep the current branch or specify a custom one. Modified branch settings are tracked in the configuration's `ModifiedSections` field.
