Handles Git branch configuration for OpenFrame deployments, allowing users to specify which OSS repository branch to use based on deployment mode.

## Key Components

- **BranchConfigurator**: Main struct that manages Git branch configuration with a Helm values modifier
- **NewBranchConfigurator**: Factory function that creates a new branch configurator instance
- **Configure**: Interactive method that prompts users to configure the OSS repository branch, with automatic skipping for SaaS deployments

## Usage Example

```go
// Create a branch configurator with Helm values modifier
modifier := templates.NewHelmValuesModifier()
branchConfigurator := NewBranchConfigurator(modifier)

// Configure branch settings interactively
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
}

err := branchConfigurator.Configure(config)
if err != nil {
    log.Fatalf("Failed to configure branch: %v", err)
}

// Check if branch was modified
if config.Branch != nil {
    fmt.Printf("Selected branch: %s\n", *config.Branch)
}
```

The configurator automatically skips branch configuration for SaaS and SaaS Shared deployment modes since those are handled elsewhere. For OSS deployments, it presents the current branch and allows users to either keep it or specify a custom branch name.
