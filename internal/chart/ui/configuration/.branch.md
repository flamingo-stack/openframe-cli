This file provides branch configuration functionality for the OpenFrame CLI, allowing users to specify Git repository branches for OSS deployments while skipping this step for SaaS modes.

## Key Components

- **BranchConfigurator**: Main struct that handles Git branch configuration with an embedded `HelmValuesModifier`
- **NewBranchConfigurator()**: Constructor function that creates a new branch configurator instance
- **Configure()**: Interactive method that prompts users to configure Git branch settings, with conditional logic for different deployment modes

## Usage Example

```go
// Create a branch configurator
modifier := templates.NewHelmValuesModifier()
configurator := configuration.NewBranchConfigurator(modifier)

// Configure branch settings
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
}

err := configurator.Configure(config)
if err != nil {
    log.Fatalf("Branch configuration failed: %v", err)
}

// Check if branch was modified
if config.Branch != nil {
    fmt.Printf("Using branch: %s\n", *config.Branch)
}
```

The configurator automatically skips branch configuration for SaaS deployments and provides an interactive interface for OSS deployments to either keep the current branch or specify a custom one.
