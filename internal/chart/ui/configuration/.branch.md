This file provides branch configuration functionality for managing Git repository branches in chart deployments.

## Key Components

- **BranchConfigurator**: Main struct that handles branch configuration with interactive UI prompts
- **NewBranchConfigurator()**: Constructor function that creates a new configurator instance
- **Configure()**: Primary method that presents branch selection options and updates configuration

## Usage Example

```go
// Create a branch configurator
modifier := templates.NewHelmValuesModifier()
configurator := NewBranchConfigurator(modifier)

// Configure branch settings
config := &types.ChartConfiguration{
    DeploymentMode: &deploymentMode,
}

err := configurator.Configure(config)
if err != nil {
    log.Fatalf("Branch configuration failed: %v", err)
}

// Check if branch was modified
if config.Branch != nil {
    fmt.Printf("Branch set to: %s", *config.Branch)
}
```

The configurator automatically skips branch configuration for SaaS deployment modes and provides an interactive interface for OSS deployments, allowing users to keep the current branch or specify a custom one. Modified branches are tracked in the configuration's ModifiedSections slice.
