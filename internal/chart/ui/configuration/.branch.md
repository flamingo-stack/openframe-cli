This file provides branch configuration functionality for Git repositories in the OpenFrame CLI, handling different deployment modes and allowing users to specify custom branches.

## Key Components

- **BranchConfigurator**: Main struct that handles Git branch configuration with a Helm values modifier
- **NewBranchConfigurator()**: Constructor function that creates a new branch configurator instance
- **Configure()**: Interactive method that prompts users to configure Git branch settings, with special handling for SaaS deployment modes

## Usage Example

```go
// Create a branch configurator with Helm values modifier
modifier := templates.NewHelmValuesModifier()
branchConfig := NewBranchConfigurator(modifier)

// Configure branch settings for a chart
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    ExistingValues: existingHelmValues,
}

err := branchConfig.Configure(config)
if err != nil {
    log.Fatalf("Branch configuration failed: %v", err)
}

// The config.Branch will be set if user chose a custom branch
if config.Branch != nil {
    fmt.Printf("Using custom branch: %s", *config.Branch)
}
```

The configurator automatically skips branch configuration for SaaS modes and provides an interactive UI for OSS deployments to either keep the current branch or specify a custom one.
