This test file validates the proper initialization and structure of the ConfigurationWizard component, ensuring all internal configurators are correctly instantiated and isolated between instances.

## Key Components

- **TestNewConfigurationWizard**: Verifies basic wizard creation and component initialization
- **TestConfigurationWizard_Structure**: Validates that all wizard components (modifier, branchConfig, dockerConfig, ingressConfig) are properly initialized
- **TestConfigurationWizard_Components**: Ensures wizard instances are independent and don't share internal state

## Usage Example

```go
func TestCustomWizardBehavior(t *testing.T) {
    // Create a new wizard instance
    wizard := NewConfigurationWizard()
    
    // Verify initialization
    assert.NotNil(t, wizard)
    assert.NotNil(t, wizard.modifier)
    
    // Test instance isolation
    wizard1 := NewConfigurationWizard()
    wizard2 := NewConfigurationWizard()
    assert.NotSame(t, wizard1, wizard2)
}
```

These tests follow the standard Go testing pattern using testify assertions to validate the ConfigurationWizard constructor creates properly isolated instances with all required components initialized.
