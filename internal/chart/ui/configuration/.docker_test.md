Test suite for Docker registry configuration functionality in the OpenFrame CLI chart management system. This file validates Docker credential handling, configuration updates, and edge cases for Helm chart value modifications.

## Key Components

- **TestNewDockerConfigurator** - Tests Docker configurator initialization
- **TestDockerConfigurator_Configure_DefaultCredentials** - Validates handling of existing default Docker credentials
- **TestDockerConfigurator_Configure_CustomCredentials** - Tests custom Docker credential configuration
- **TestDockerConfigurator_Configure_WithEmptyValues** - Validates behavior with empty/missing registry values
- **TestDockerConfigurator_promptForDockerSettings_Validation** - Tests various Docker configuration scenarios
- **TestDockerConfigurator_Configure_NoChangesWhenSameValues** - Verifies handling when values remain unchanged
- **TestDockerConfigurator_Configure_SpecialCharacters** - Tests special character handling in credentials
- **TestDockerConfigurator_Configure_EdgeCases** - Validates edge cases like long strings and Unicode

## Usage Example

```go
func TestDockerConfiguration(t *testing.T) {
    // Create modifier and configurator
    modifier := templates.NewHelmValuesModifier()
    configurator := NewDockerConfigurator(modifier)

    // Test configuration with custom credentials
    config := &types.ChartConfiguration{
        DockerRegistry: &types.DockerRegistryConfig{
            Username: "myuser",
            Password: "mypass",
            Email:    "user@example.com",
        },
        ModifiedSections: []string{"docker"},
        ExistingValues:   map[string]interface{}{},
    }

    err := modifier.ApplyConfiguration(config.ExistingValues, config)
    assert.NoError(t, err)
}
```
