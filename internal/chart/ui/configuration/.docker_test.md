Test file for Docker registry configuration functionality. Contains comprehensive unit tests for the DockerConfigurator component, verifying Docker credential management, Helm values modification, and edge cases.

## Key Components

- **TestNewDockerConfigurator**: Tests constructor initialization
- **TestDockerConfigurator_Configure_DefaultCredentials**: Validates handling of default Docker credentials
- **TestDockerConfigurator_Configure_CustomCredentials**: Tests custom credential configuration
- **TestDockerConfigurator_Configure_WithEmptyValues**: Verifies configuration with empty initial values
- **TestDockerConfigurator_promptForDockerSettings_Validation**: Tests configuration validation scenarios
- **Edge case tests**: Special characters, unicode, length limits, and duplicate values

## Usage Example

```go
// Run all Docker configurator tests
go test -run TestDockerConfigurator

// Run specific test for custom credentials
go test -run TestDockerConfigurator_Configure_CustomCredentials

// Test with verbose output
go test -v ./... -run Docker
```

The tests validate Docker registry configuration including username, password, and email handling, ensuring proper Helm values modification and support for various input scenarios including special characters and edge cases.
