Test file for SaaS branch extraction functionality within the configuration wizard. Contains comprehensive unit tests that validate the `getSaaSBranchFromValues` method's ability to parse nested configuration structures and extract SaaS repository branch information.

## Key Components

- **TestConfigurationWizard_GetSaaSBranchFromValues**: Main test function covering edge cases for branch extraction from configuration values
- **TestConfigurationWizard_SaaSBranchExtraction**: Tests complex nested structures and cross-validates OSS branch extraction
- **TestConfigurationWizard_SaaSConfigStructure**: Validates proper handling of configuration structure variations

## Usage Example

```go
// Test the getSaaSBranchFromValues method
wizard := NewConfigurationWizard()

values := map[string]interface{}{
    "deployment": map[string]interface{}{
        "saas": map[string]interface{}{
            "repository": map[string]interface{}{
                "branch": "develop",
            },
        },
    },
}

branch := wizard.getSaaSBranchFromValues(values)
// Returns "develop" if found, "main" as default

// Run tests
go test -run TestConfigurationWizard_GetSaaSBranchFromValues
```

The tests ensure robust handling of nil values, empty maps, incomplete structures, and proper extraction of SaaS branch information while defaulting to "main" when configuration is missing or malformed.
