Provides interactive configuration for Docker registry credentials in Helm chart deployments, allowing users to set or update authentication settings through a command-line interface.

## Key Components

- **DockerConfigurator**: Main struct that handles Docker registry configuration workflows
- **NewDockerConfigurator()**: Constructor function that creates a new configurator instance
- **Configure()**: Interactive method that prompts users to choose Docker credential options and updates configuration
- **promptForDockerSettings()**: Private method that collects Docker registry credentials (username, password, email) from user input

## Usage Example

```go
// Initialize with a Helm values modifier
modifier := templates.NewHelmValuesModifier()
dockerConfig := configuration.NewDockerConfigurator(modifier)

// Configure Docker settings interactively
chartConfig := &types.ChartConfiguration{
    ExistingValues: existingHelmValues,
}

err := dockerConfig.Configure(chartConfig)
if err != nil {
    log.Fatal(err)
}

// Check if Docker settings were modified
if contains(chartConfig.ModifiedSections, "docker") {
    fmt.Println("Docker registry credentials updated")
}
```

The configurator integrates with the pterm library to provide a user-friendly CLI experience, displaying current settings and only updating the configuration when values actually change.
