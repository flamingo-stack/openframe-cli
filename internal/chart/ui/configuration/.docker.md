This file provides interactive configuration management for Docker registry credentials within the OpenFrame CLI's chart configuration system.

## Key Components

- **DockerConfigurator**: Main struct that handles Docker registry configuration with Helm values modification
- **NewDockerConfigurator()**: Constructor function that creates a configurator with a Helm values modifier
- **Configure()**: Interactive method that prompts users to configure Docker credentials or skip configuration
- **promptForDockerSettings()**: Internal method that collects Docker username, password/token, and email from user input

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/ui/templates"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/types"
    "github.com/flamingo-stack/openframe-cli/internal/chart/ui/configuration"
)

// Create a Helm values modifier
modifier := templates.NewHelmValuesModifier()

// Create Docker configurator
dockerConfig := configuration.NewDockerConfigurator(modifier)

// Configure Docker settings interactively
config := &types.ChartConfiguration{
    ExistingValues: map[string]interface{}{},
}

err := dockerConfig.Configure(config)
if err != nil {
    log.Fatal(err)
}

// Docker credentials are now stored in config.DockerRegistry
// and "docker" is added to config.ModifiedSections if changes were made
```

The configurator intelligently detects when Docker settings have actually changed and only updates the configuration when necessary, preventing unnecessary modifications to Helm values.
