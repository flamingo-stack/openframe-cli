Manages Docker registry configuration for Helm charts through interactive user prompts, allowing users to set or update Docker credentials including username, password, and email.

## Key Components

- **DockerConfigurator**: Main struct that handles Docker registry configuration with a Helm values modifier
- **NewDockerConfigurator()**: Constructor function that creates a new DockerConfigurator instance
- **Configure()**: Primary method that presents configuration options and handles user choices
- **promptForDockerSettings()**: Internal method that collects Docker credentials through interactive prompts

## Usage Example

```go
// Initialize with a Helm values modifier
modifier := templates.NewHelmValuesModifier()
dockerConfig := NewDockerConfigurator(modifier)

// Configure Docker settings for a chart
config := &types.ChartConfiguration{
    ExistingValues: existingHelmValues,
}

err := dockerConfig.Configure(config)
if err != nil {
    log.Fatal("Docker configuration failed:", err)
}

// Check if Docker settings were modified
for _, section := range config.ModifiedSections {
    if section == "docker" {
        fmt.Printf("Docker credentials updated: %s", config.DockerRegistry.Username)
    }
}
```

The configurator only updates the configuration if the user provides different values than what currently exists, tracking changes in the `ModifiedSections` slice.
