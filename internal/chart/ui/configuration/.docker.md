Provides interactive configuration management for Docker registry credentials in Helm chart deployments, allowing users to set or skip Docker authentication settings.

## Key Components

- **DockerConfigurator**: Main struct that handles Docker registry configuration with Helm values modification support
- **NewDockerConfigurator()**: Constructor function that creates a new configurator instance
- **Configure()**: Interactive method that presents options for Docker credentials and processes user choices
- **promptForDockerSettings()**: Private method that collects Docker registry credentials (username, password, email) from user input

## Usage Example

```go
// Initialize the configurator
modifier := templates.NewHelmValuesModifier()
dockerConfig := NewDockerConfigurator(modifier)

// Configure Docker settings interactively
chartConfig := &types.ChartConfiguration{
    ExistingValues: existingHelmValues,
}

err := dockerConfig.Configure(chartConfig)
if err != nil {
    log.Fatal("Docker configuration failed:", err)
}

// Check if Docker settings were modified
if contains(chartConfig.ModifiedSections, "docker") {
    fmt.Println("Docker registry configured:", chartConfig.DockerRegistry.Username)
}
```

The configurator integrates with the broader chart configuration system, only updating settings when changes are detected, and tracking modifications for downstream processing.
