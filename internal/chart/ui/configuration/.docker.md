Provides interactive Docker registry credential configuration for OpenFrame CLI chart management, with options to skip or input custom registry authentication details.

## Key Components

- **`DockerConfigurator`** - Main struct that handles Docker registry configuration with Helm values integration
- **`NewDockerConfigurator`** - Constructor that initializes the configurator with a Helm values modifier
- **`Configure`** - Primary method that presents options for Docker credentials (none or custom input)
- **`promptForDockerSettings`** - Internal method that collects Docker registry credentials through interactive prompts

## Usage Example

```go
// Initialize with a Helm values modifier
modifier := &templates.HelmValuesModifier{}
dockerConfig := NewDockerConfigurator(modifier)

// Configure Docker settings interactively
config := &types.ChartConfiguration{}
err := dockerConfig.Configure(config)
if err != nil {
    log.Fatalf("Docker configuration failed: %v", err)
}

// Check if Docker settings were modified
for _, section := range config.ModifiedSections {
    if section == "docker" {
        fmt.Printf("Docker registry configured: %s", config.DockerRegistry.Username)
    }
}
```

The configurator only updates the chart configuration if the user provides different values than the current settings, tracking changes in the `ModifiedSections` slice for efficient Helm value updates.
