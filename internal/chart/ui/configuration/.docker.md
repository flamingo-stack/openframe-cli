Handles interactive Docker registry credential configuration for the OpenFrame CLI chart management system.

## Key Components

- **DockerConfigurator**: Main struct that manages Docker registry configuration with Helm values integration
- **NewDockerConfigurator()**: Constructor function that initializes the configurator with a Helm values modifier
- **Configure()**: Interactive method that presents user choices for Docker credentials and handles configuration updates
- **promptForDockerSettings()**: Private method that collects Docker registry credentials from user input

## Usage Example

```go
// Initialize with a Helm values modifier
modifier := templates.NewHelmValuesModifier()
dockerConfig := NewDockerConfigurator(modifier)

// Configure Docker settings interactively
config := &types.ChartConfiguration{
    ExistingValues: map[string]interface{}{},
}

err := dockerConfig.Configure(config)
if err != nil {
    log.Fatal("Docker configuration failed:", err)
}

// Check if Docker settings were modified
for _, section := range config.ModifiedSections {
    if section == "docker" {
        fmt.Printf("Docker credentials updated: %+v", config.DockerRegistry)
    }
}
```

The configurator provides an interactive terminal interface for setting up Docker registry authentication, supporting both credential input and no-authentication scenarios. It integrates with the existing Helm values system and only marks sections as modified when actual changes occur.
