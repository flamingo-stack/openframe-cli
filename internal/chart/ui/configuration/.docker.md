Provides interactive configuration for Docker registry credentials during Helm chart setup, allowing users to input or skip Docker authentication settings.

## Key Components

- **DockerConfigurator**: Main struct that handles Docker registry configuration workflow
- **NewDockerConfigurator()**: Constructor function that creates a new Docker configurator instance
- **Configure()**: Primary method that presents Docker configuration options and handles user choices
- **promptForDockerSettings()**: Helper method that collects Docker credentials through interactive prompts

## Usage Example

```go
// Initialize the Docker configurator
modifier := templates.NewHelmValuesModifier()
dockerConfig := NewDockerConfigurator(modifier)

// Configure Docker settings for a chart
config := &types.ChartConfiguration{
    ExistingValues: existingHelmValues,
}

err := dockerConfig.Configure(config)
if err != nil {
    log.Fatalf("Docker configuration failed: %v", err)
}

// Check if Docker settings were modified
for _, section := range config.ModifiedSections {
    if section == "docker" {
        fmt.Printf("Docker credentials updated: %s", config.DockerRegistry.Username)
    }
}
```

The configurator presents users with options to either skip Docker credentials or input custom registry authentication details. It only updates the configuration when values actually change from existing settings, and uses masked input for sensitive password fields.
