A wizard that orchestrates the Helm chart configuration process by coordinating branch, Docker, and ingress configurations through an interactive or default setup workflow.

## Key Components

- **ConfigurationWizard**: Main orchestrator struct that manages the configuration workflow
- **NewConfigurationWizard()**: Factory function that initializes the wizard with all required configurators
- **ConfigureHelmValues()**: Primary method that guides users through deployment and configuration mode selection
- **ConfigureHelmValuesWithMode()**: Alternative entry point that skips deployment mode selection when mode is pre-determined

## Usage Example

```go
// Create a new configuration wizard
wizard := NewConfigurationWizard()

// Interactive configuration with mode selection
config, err := wizard.ConfigureHelmValues()
if err != nil {
    log.Fatal("Configuration failed:", err)
}

// Or configure with predetermined deployment mode
config, err := wizard.ConfigureHelmValuesWithMode(types.ProductionMode)
if err != nil {
    log.Fatal("Configuration with mode failed:", err)
}

// The returned ChartConfiguration contains all user selections
fmt.Printf("Chart configured for %s deployment\n", config.DeploymentMode)
```

The wizard provides a unified interface for configuring OpenFrame deployments, handling the complexity of coordinating multiple configuration aspects while offering both interactive and default configuration paths.
