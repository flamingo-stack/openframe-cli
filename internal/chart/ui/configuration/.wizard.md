Interactive wizard for configuring OpenFrame chart deployments with multiple configuration modes and specialized configurators for different deployment aspects.

## Key Components

- **ConfigurationWizard** - Main wizard struct that orchestrates the configuration workflow
- **NewConfigurationWizard()** - Factory function that creates a wizard with initialized configurators
- **ConfigureHelmValues()** - Primary method that guides users through deployment and configuration mode selection
- **ConfigureHelmValuesWithMode()** - Alternative entry point that accepts a pre-selected deployment mode
- **Specialized configurators** - Branch, Docker, and Ingress configurators for specific configuration domains

## Usage Example

```go
// Create a new configuration wizard
wizard := configuration.NewConfigurationWizard()

// Full interactive configuration
chartConfig, err := wizard.ConfigureHelmValues()
if err != nil {
    log.Fatal("Configuration failed:", err)
}

// Configure with predetermined deployment mode
chartConfig, err = wizard.ConfigureHelmValuesWithMode(types.ProductionMode)
if err != nil {
    log.Fatal("Configuration with mode failed:", err)
}

// Use the resulting chart configuration
fmt.Printf("Configured for deployment mode: %v\n", chartConfig.DeploymentMode)
```

The wizard provides a structured approach to Helm chart configuration by separating concerns into specialized configurators while maintaining a unified interface for the overall configuration workflow.
