Orchestrates the Helm chart configuration workflow by coordinating branch, Docker, and ingress configuration steps through an interactive wizard interface.

## Key Components

**ConfigurationWizard**: Main struct that manages the configuration workflow through specialized configurators and a Helm values modifier.

**Core Methods**:
- `NewConfigurationWizard()` - Creates a new wizard instance with all configurators
- `ConfigureHelmValues()` - Full interactive configuration with deployment mode selection
- `ConfigureHelmValuesWithMode()` - Configuration with pre-selected deployment mode

**Internal Components**:
- `HelmValuesModifier` - Handles Helm values manipulation
- `BranchConfigurator` - Manages branch-specific settings
- `DockerConfigurator` - Handles Docker configuration
- `IngressConfigurator` - Manages ingress settings

## Usage Example

```go
// Create and run full interactive wizard
wizard := configuration.NewConfigurationWizard()
config, err := wizard.ConfigureHelmValues()
if err != nil {
    log.Fatal(err)
}

// Use wizard with predetermined deployment mode
config, err = wizard.ConfigureHelmValuesWithMode(types.ProductionMode)
if err != nil {
    log.Fatal(err)
}

// Access the resulting chart configuration
fmt.Printf("Deployment mode: %v\n", config.DeploymentMode)
```

The wizard follows a step-by-step approach: deployment mode selection → configuration mode choice → either default or interactive configuration flow.
