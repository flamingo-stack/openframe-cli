This file provides a centralized wizard for configuring Helm chart values through interactive prompts and default configurations.

## Key Components

- **ConfigurationWizard** - Main struct that orchestrates the configuration workflow using specialized configurators for different aspects (branch, docker, ingress)
- **NewConfigurationWizard()** - Factory function that creates a wizard with all necessary configurators
- **ConfigureHelmValues()** - Primary method that guides users through deployment mode selection and configuration
- **ConfigureHelmValuesWithMode()** - Alternative method that skips deployment mode selection when mode is pre-determined

## Usage Example

```go
// Create a new configuration wizard
wizard := NewConfigurationWizard()

// Configure with interactive prompts
config, err := wizard.ConfigureHelmValues()
if err != nil {
    log.Fatal(err)
}

// Or configure with a specific deployment mode
config, err = wizard.ConfigureHelmValuesWithMode(types.ProductionMode)
if err != nil {
    log.Fatal(err)
}
```

The wizard provides two configuration paths: default configuration for quick setup, or interactive mode for detailed customization. It integrates branch, Docker, and ingress configuration through specialized components while maintaining a unified interface.
