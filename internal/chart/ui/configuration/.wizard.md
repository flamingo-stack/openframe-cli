This file implements a wizard-style interface for configuring Helm chart values through interactive prompts and default settings.

## Key Components

- **ConfigurationWizard**: Main struct that orchestrates the configuration workflow using specialized configurators for different aspects (branches, Docker, ingress)
- **NewConfigurationWizard()**: Factory function that creates a wizard with all necessary configurators and a Helm values modifier
- **ConfigureHelmValues()**: Primary method that guides users through deployment mode selection and configuration options
- **ConfigureHelmValuesWithMode()**: Alternative method that accepts a pre-selected deployment mode and proceeds with configuration

## Usage Example

```go
// Create a new configuration wizard
wizard := configuration.NewConfigurationWizard()

// Configure with user prompts for all options
config, err := wizard.ConfigureHelmValues()
if err != nil {
    log.Fatal(err)
}

// Or configure with a specific deployment mode
config, err := wizard.ConfigureHelmValuesWithMode(types.ProductionMode)
if err != nil {
    log.Fatal(err)
}

// Use the returned ChartConfiguration for deployment
fmt.Printf("Configuration complete: %+v\n", config)
```

The wizard provides both fully interactive configuration and streamlined configuration with pre-selected deployment modes, making it flexible for different use cases.
