Test suite for the configuration wizard's summary display functionality, ensuring the ShowConfigurationSummary method handles various configuration scenarios without panicking.

## Key Components

- **TestConfigurationWizard_ShowConfigurationSummary_NoChanges**: Tests summary display with empty configuration
- **TestConfigurationWizard_ShowConfigurationSummary_WithChanges**: Tests summary with standard configuration changes (branch, deployment mode, docker registry, ingress)
- **TestConfigurationWizard_ShowConfigurationSummary_WithNgrokConfig**: Tests summary with ngrok-specific ingress configuration
- **TestConfigurationWizard_ShowConfigurationSummary_WithSaaSConfig**: Tests summary with SaaS deployment configuration

## Usage Example

```go
func TestCustomConfiguration(t *testing.T) {
    wizard := NewConfigurationWizard()
    
    config := &types.ChartConfiguration{
        Branch: &[]string{"feature-branch"}[0],
        DeploymentMode: &types.DeploymentModeOSS,
        ModifiedSections: []string{"branch", "deployment"},
        ExistingValues: map[string]interface{}{},
    }
    
    // Verify summary display doesn't crash
    assert.NotPanics(t, func() {
        wizard.ShowConfigurationSummary(config)
    })
}
```

These tests validate that the configuration wizard can safely display summaries for different deployment scenarios including localhost, ngrok, and SaaS configurations with various modified sections.
