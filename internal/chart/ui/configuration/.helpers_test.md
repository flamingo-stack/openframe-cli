Test suite for the configuration wizard's summary display functionality, validating that configuration summaries are properly rendered without errors.

## Key Components

- **TestConfigurationWizard_ShowConfigurationSummary_NoChanges**: Tests summary display with empty configuration
- **TestConfigurationWizard_ShowConfigurationSummary_WithChanges**: Tests summary with standard configuration changes (branch, deployment mode, Docker registry, ingress)
- **TestConfigurationWizard_ShowConfigurationSummary_WithNgrokConfig**: Tests summary display for ngrok ingress configurations
- **TestConfigurationWizard_ShowConfigurationSummary_WithSaaSConfig**: Tests summary display for SaaS deployment configurations

## Usage Example

```go
func TestYourConfigurationWizard(t *testing.T) {
    wizard := NewConfigurationWizard()
    
    // Test with basic configuration
    config := &types.ChartConfiguration{
        ModifiedSections: []string{"deployment"},
        ExistingValues:   map[string]interface{}{},
    }
    
    // Verify no panic occurs during summary display
    assert.NotPanics(t, func() {
        wizard.ShowConfigurationSummary(config)
    })
}
```

These tests ensure the `ShowConfigurationSummary` method handles various configuration scenarios gracefully, from empty configurations to complex setups with Docker registries, ngrok tunneling, and SaaS deployments.
