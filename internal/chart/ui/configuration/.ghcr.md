This file provides interactive configuration functionality for GitHub Container Registry (GHCR) credentials within a chart configuration wizard.

## Key Components

- **configureGHCRCredentials**: Method on `ConfigurationWizard` that handles GHCR credential collection through an interactive UI. It supports both new credential setup and updating existing credentials while preserving user data where appropriate.

## Usage Example

```go
// Configure GHCR credentials interactively
wizard := &ConfigurationWizard{}
config := &types.ChartConfiguration{
    ExistingValues: map[string]interface{}{
        "registry": map[string]interface{}{
            "ghcr": map[string]interface{}{
                "username": "existing-user",
                "email": "user@example.com",
            },
        },
    },
}

username, password, email, err := wizard.configureGHCRCredentials(config)
if err != nil {
    log.Fatal(err)
}

// Use the collected credentials for registry authentication
fmt.Printf("Configured GHCR: %s <%s>\n", username, email)
```

The function intelligently detects existing credentials and provides options to either keep or update them, while always requiring password input for security reasons.
