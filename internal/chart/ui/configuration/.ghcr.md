Handles GitHub Container Registry (GHCR) credential configuration with interactive user prompts and support for existing credential preservation.

## Key Components

- **`configureGHCRCredentials`** - Main method that prompts users to configure GHCR registry credentials (username, password/token, email) with options to keep or update existing values

## Usage Example

```go
wizard := &ConfigurationWizard{}
config := &types.ChartConfiguration{
    ExistingValues: map[string]interface{}{
        "registry": map[string]interface{}{
            "ghcr": map[string]interface{}{
                "username": "existing-user",
                "email": "user@example.com",
            },
        },
    },
}

username, password, email, err := wizard.configureGHCRCredentials(config)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Configured GHCR credentials for %s\n", username)
```

The method intelligently detects existing GHCR credentials from the chart configuration and offers users the choice to keep existing credentials or configure new ones. Password/token input is always required as it's never stored in plain text for security reasons.
