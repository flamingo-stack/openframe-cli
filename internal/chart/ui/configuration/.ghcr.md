Provides interactive configuration for GitHub Container Registry (GHCR) credentials within a chart configuration wizard. The file handles both new credential setup and updates to existing configurations with a user-friendly selection interface.

## Key Components

- **`configureGHCRCredentials`** - Main method that guides users through GHCR credential configuration
- **Credential detection** - Automatically detects existing GHCR credentials from chart configuration
- **Interactive UI** - Uses pterm for masked password input and selection menus
- **Configuration management** - Handles both new setups and credential updates

## Usage Example

```go
// Initialize configuration wizard
wizard := &ConfigurationWizard{}

// Configure GHCR credentials for a chart
config := &types.ChartConfiguration{
    ExistingValues: map[string]interface{}{
        "registry": map[string]interface{}{
            "ghcr": map[string]interface{}{
                "username": "existing-user",
                "email": "user@example.com",
            },
        },
    },
}

// Get GHCR credentials through interactive prompts
username, password, email, err := wizard.configureGHCRCredentials(config)
if err != nil {
    log.Fatal("Failed to configure GHCR credentials:", err)
}

fmt.Printf("Configured GHCR for user: %s\n", username)
```
