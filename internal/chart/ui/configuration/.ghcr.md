Handles GitHub Container Registry (GHCR) credential configuration through an interactive wizard interface, allowing users to set up or update authentication details for GHCR access.

## Key Components

- **configureGHCRCredentials()** - Main method that manages GHCR credential collection with options to keep existing credentials or configure new ones
- **Credential Detection** - Automatically detects existing GHCR credentials from chart configuration values
- **Interactive UI** - Uses pterm for user input with masked password fields and default value suggestions
- **Input Validation** - Trims whitespace and handles empty/default values appropriately

## Usage Example

```go
// Initialize configuration wizard
wizard := &ConfigurationWizard{}

// Configure GHCR credentials interactively
username, password, email, err := wizard.configureGHCRCredentials(config)
if err != nil {
    log.Fatalf("Failed to configure GHCR credentials: %v", err)
}

// Use credentials for registry authentication
fmt.Printf("Configured GHCR access for user: %s\n", username)
```

The method provides a user-friendly flow that:
- Detects existing credentials and offers to keep them
- Prompts for username, password/token, and email with appropriate defaults
- Masks sensitive password input for security
- Returns sanitized credential values for use in registry operations
