Handles GHCR (GitHub Container Registry) credential configuration through an interactive UI wizard. The function provides options to configure new credentials or update existing ones with proper masking for sensitive data.

## Key Components

- `configureGHCRCredentials()` - Main function that manages GHCR credential collection and configuration
- Interactive credential input with password masking
- Existing credential detection and preservation options
- Input validation and error handling

## Usage Example

```go
wizard := &ConfigurationWizard{}
config := &types.ChartConfiguration{
    ExistingValues: map[string]interface{}{
        "registry": map[string]interface{}{
            "ghcr": map[string]interface{}{
                "username": "existinguser",
                "email": "user@example.com",
            },
        },
    },
}

username, password, email, err := wizard.configureGHCRCredentials(config)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Configured GHCR credentials for user: %s\n", username)
```

The function presents users with options to either keep existing credentials (if found) or configure new ones, automatically handling the secure collection of passwords and tokens through masked input fields.
