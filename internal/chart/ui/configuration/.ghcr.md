This file implements GHCR (GitHub Container Registry) credential configuration functionality for the OpenFrame CLI's chart configuration wizard.

## Key Components

- **`configureGHCRCredentials`** - Main method that handles GHCR credential collection through an interactive UI, supporting both new configuration and updates to existing credentials

## Usage Example

```go
wizard := &ConfigurationWizard{}
config := &types.ChartConfiguration{
    ExistingValues: map[string]interface{}{
        "registry": map[string]interface{}{
            "ghcr": map[string]interface{}{
                "username": "existing-user",
                "email": "user@example.com",
            },
        },
    },
}

username, password, email, err := wizard.configureGHCRCredentials(config)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Configured GHCR credentials for: %s (%s)", username, email)
```

The method intelligently detects existing GHCR credentials from the chart configuration and provides options to either keep existing credentials (requiring only password re-entry) or configure new ones. It uses interactive prompts with masked password input for security and returns the complete credential set needed for GHCR authentication.
