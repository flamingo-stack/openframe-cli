This test file validates the configuration wizard functionality for OpenFrame CLI, specifically testing Helm values file manipulation and deployment mode configurations.

## Key Components

- **TestConfigurationWizard_ConfigureWithDefaults_OSS**: Tests OSS mode configuration with default values and temporary file creation
- **TestConfigurationWizard_ConfigureWithExistingFile**: Validates loading existing Helm values files with pre-configured settings
- **TestConfigurationWizard_Integration_LoadAndApply**: Integration test for the complete flow of loading, modifying, and writing Helm values
- **TestConfigurationWizard_DeploymentModes**: Tests deployment mode constants (OSS and SaaS)
- **TestConfigurationWizard_LoadBaseValues**: Tests base configuration loading functionality

## Usage Example

```go
// Create a configuration wizard instance
wizard := NewConfigurationWizard()

// Load base values for configuration
config, err := wizard.loadBaseValues()
if err != nil {
    log.Fatal(err)
}

// Create configuration for OSS deployment
deploymentMode := types.DeploymentModeOSS
branch := "develop"
config := &types.ChartConfiguration{
    Branch:         &branch,
    DeploymentMode: &deploymentMode,
    DockerRegistry: &types.DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass",
        Email:    "user@example.com",
    },
    ModifiedSections: []string{"branch", "docker"},
}

// Apply configuration using HelmValuesModifier
modifier := templates.NewHelmValuesModifier()
values, _ := modifier.LoadOrCreateBaseValues()
modifier.ApplyConfiguration(values, config)
```
