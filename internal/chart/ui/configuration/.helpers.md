This file provides helper functions for the ConfigurationWizard to manage Helm chart configurations, handle temporary files, and display configuration summaries.

## Key Components

- **`loadBaseValues()`**: Loads existing Helm values from the current directory or creates default values, returning a ChartConfiguration struct
- **`createTemporaryValuesFile()`**: Applies configuration changes and creates a temporary Helm values file for installation
- **`ShowConfigurationSummary()`**: Displays a formatted summary of all modified configuration sections using pterm for colored output

## Usage Example

```go
wizard := &ConfigurationWizard{modifier: someModifier}

// Load base configuration
config, err := wizard.loadBaseValues()
if err != nil {
    log.Fatal(err)
}

// Apply changes and create temporary file
if err := wizard.createTemporaryValuesFile(config); err != nil {
    log.Fatal(err)
}

// Display what was changed
wizard.ShowConfigurationSummary(config)
```

The helpers work together to provide a complete configuration workflow: loading existing values, applying modifications through the wizard, creating temporary files for Helm deployment, and providing user feedback about the changes made.
