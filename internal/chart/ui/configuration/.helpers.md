A utility module providing helper functions for the ConfigurationWizard to manage chart configurations, temporary files, and display summaries.

## Key Components

- **`loadBaseValues()`** - Loads existing helm values from the current directory or creates default values, returning a `ChartConfiguration` struct
- **`createTemporaryValuesFile()`** - Applies configuration changes and creates a temporary helm values file for installation
- **`ShowConfigurationSummary()`** - Displays a formatted summary of all modified configuration sections with checkmarks

## Usage Example

```go
// Load base configuration
config, err := wizard.loadBaseValues()
if err != nil {
    return fmt.Errorf("failed to load base values: %w", err)
}

// Apply changes and create temporary file
err = wizard.createTemporaryValuesFile(config)
if err != nil {
    return fmt.Errorf("failed to create temporary file: %w", err)
}

// Display summary of changes
wizard.ShowConfigurationSummary(config)
```

The helper functions work together in the configuration workflow: first loading existing values, then applying modifications and creating temporary files for helm deployment, and finally providing user feedback through formatted summaries that show what was changed (deployment mode, SaaS settings, branch, Docker registry, ingress configuration, etc.).
