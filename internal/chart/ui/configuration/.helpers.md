Provides helper functions for the configuration wizard to manage Helm chart values and display configuration changes.

## Key Components

- **`loadBaseValues()`** - Loads existing Helm values from the current directory or creates default values, returning a `ChartConfiguration` struct with base file path and existing values
- **`createTemporaryValuesFile()`** - Applies configuration changes to values and creates a temporary Helm values file for installation, updating the config with the temp file path
- **`ShowConfigurationSummary()`** - Displays a formatted summary of all modified configuration sections using colored terminal output via pterm

## Usage Example

```go
// Initialize configuration wizard
wizard := &ConfigurationWizard{
    modifier: chartModifier,
}

// Load base values from current directory
config, err := wizard.loadBaseValues()
if err != nil {
    return fmt.Errorf("failed to load base values: %w", err)
}

// Apply changes and create temporary file
if err := wizard.createTemporaryValuesFile(config); err != nil {
    return fmt.Errorf("failed to create temp values: %w", err)
}

// Show what was configured
wizard.ShowConfigurationSummary(config)
```

The helper functions work together to manage the configuration workflow: loading base values, applying modifications through the chart modifier, creating temporary installation files, and providing user feedback on changes made during the configuration process.
