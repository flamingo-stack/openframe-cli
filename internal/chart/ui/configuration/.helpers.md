This file provides helper methods for the ConfigurationWizard to manage Helm chart configurations, including loading values, creating temporary files, and displaying summaries.

## Key Components

- **`loadBaseValues()`** - Loads existing Helm values from the current directory or creates default values, returning a ChartConfiguration struct
- **`createTemporaryValuesFile()`** - Applies configuration changes and creates a temporary Helm values file for installation
- **`ShowConfigurationSummary()`** - Displays a formatted summary of all modified configuration sections with visual indicators

## Usage Example

```go
// Initialize configuration wizard
wizard := &ConfigurationWizard{modifier: modifier}

// Load base configuration
config, err := wizard.loadBaseValues()
if err != nil {
    return err
}

// Apply modifications (deployment mode, SaaS config, etc.)
config.DeploymentMode = &types.DeploymentModeStandalone
config.ModifiedSections = append(config.ModifiedSections, "deployment")

// Create temporary file for Helm installation
err = wizard.createTemporaryValuesFile(config)
if err != nil {
    return err
}

// Display summary of changes
wizard.ShowConfigurationSummary(config)
// Output: âœ“ Deployment mode: standalone
```

The helpers work together to provide a complete workflow for configuration management, from loading existing values to preparing files for Helm deployment.
