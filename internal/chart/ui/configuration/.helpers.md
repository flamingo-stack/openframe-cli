A helper module that provides utility functions for the ConfigurationWizard, handling file operations, temporary value creation, and user feedback display.

## Key Components

- **`loadBaseValues()`** - Loads existing Helm values from the current directory or creates default configuration
- **`createTemporaryValuesFile()`** - Generates temporary Helm values file with applied configuration changes
- **`ShowConfigurationSummary()`** - Displays a formatted summary of all configuration modifications made during the wizard

## Usage Example

```go
// Load base configuration
config, err := wizard.loadBaseValues()
if err != nil {
    return fmt.Errorf("failed to load base values: %w", err)
}

// Apply configuration changes and create temporary file
if err := wizard.createTemporaryValuesFile(config); err != nil {
    return fmt.Errorf("failed to create temporary values: %w", err)
}

// Display summary of changes to user
wizard.ShowConfigurationSummary(config)
```

The module integrates with the `modifier` component to handle actual file operations and uses `pterm` for styled console output. The configuration summary provides visual feedback showing which sections were modified (deployment mode, SaaS settings, branch, Docker registry, or ingress configuration) with specific details for each change made during the wizard flow.
