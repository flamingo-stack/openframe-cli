Test suite for the branch configurator component that validates branch configuration functionality in Helm charts. Tests cover constructor initialization, branch modification scenarios, validation logic, and edge cases for deployment configuration.

## Key Components

- **TestNewBranchConfigurator**: Tests constructor and dependency injection
- **TestBranchConfigurator_Configure_KeepExisting**: Validates preservation of existing branch values
- **TestBranchConfigurator_Configure_CustomBranch**: Tests custom branch assignment functionality
- **TestBranchConfigurator_Configure_WithEmptyValues**: Tests branch configuration with empty initial values
- **TestBranchConfigurator_Configure_BranchValidation**: Tests branch name format validation
- **TestBranchConfigurator_Configure_NoChanges**: Verifies no-op behavior when no changes are made

## Usage Example

```go
func TestBranchConfiguration(t *testing.T) {
    modifier := templates.NewHelmValuesModifier()
    configurator := NewBranchConfigurator(modifier)
    
    // Test custom branch configuration
    existingValues := map[string]interface{}{
        "deployment": map[string]interface{}{
            "oss": map[string]interface{}{
                "repository": map[string]interface{}{
                    "branch": "main",
                },
            },
        },
    }
    
    newBranch := "develop"
    deploymentMode := types.DeploymentModeOSS
    config := &types.ChartConfiguration{
        Branch:           &newBranch,
        DeploymentMode:   &deploymentMode,
        ModifiedSections: []string{"branch"},
        ExistingValues:   existingValues,
    }
    
    err := modifier.ApplyConfiguration(existingValues, config)
    assert.NoError(t, err)
}
```
