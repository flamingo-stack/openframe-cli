Test file for the branch configuration component that validates git branch selection functionality in Helm chart configurations.

## Key Components

- **TestNewBranchConfigurator**: Tests constructor initialization with HelmValuesModifier
- **TestBranchConfigurator_Configure_KeepExisting**: Validates existing branch value retrieval
- **TestBranchConfigurator_Configure_CustomBranch**: Tests custom branch configuration updates
- **TestBranchConfigurator_Configure_WithEmptyValues**: Tests behavior with empty/missing configuration
- **TestBranchConfigurator_Configure_BranchValidation**: Validates various branch name formats
- **TestBranchConfigurator_Configure_NoChanges**: Tests no-operation scenarios

## Usage Example

```go
// Run specific test
go test -run TestBranchConfigurator_Configure_CustomBranch

// Run all branch configuration tests
go test -v ./internal/configuration -run "TestBranchConfigurator"

// Test custom branch validation
modifier := templates.NewHelmValuesModifier()
config := &types.ChartConfiguration{
    Branch:           &"feature/new-feature",
    DeploymentMode:   &types.DeploymentModeOSS,
    ModifiedSections: []string{"branch"},
    ExistingValues:   existingValues,
}
err := modifier.ApplyConfiguration(existingValues, config)
```

The tests cover branch configuration scenarios including existing value preservation, custom branch updates, empty configuration handling, branch name validation, and no-change operations for OSS deployment structures.
