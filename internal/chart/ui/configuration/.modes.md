This file provides user interface methods for the configuration wizard, handling deployment and configuration mode selection through interactive prompts.

## Key Components

- **showDeploymentModeSelection()** - Displays interactive prompt for selecting deployment mode (OSS, SaaS, or SaaS Shared)
- **showConfigurationModeSelection()** - Prompts user to choose between default or interactive configuration
- **configureWithDefaults()** - Creates default configuration without user interaction for the selected deployment mode
- **configureInteractive()** - Runs the full interactive configuration wizard with step-by-step prompts

## Usage Example

```go
// Create configuration wizard instance
wizard := &ConfigurationWizard{
    // ... initialize with required dependencies
}

// Show deployment mode selection
deploymentMode, err := wizard.showDeploymentModeSelection()
if err != nil {
    return fmt.Errorf("deployment mode selection failed: %w", err)
}

// Show configuration mode selection
configMode, err := wizard.showConfigurationModeSelection()
if err != nil {
    return fmt.Errorf("configuration mode selection failed: %w", err)
}

// Configure based on selected mode
var config *types.ChartConfiguration
if configMode == "default" {
    config, err = wizard.configureWithDefaults(deploymentMode)
} else {
    config, err = wizard.configureInteractive(deploymentMode)
}
```
