This file provides interactive prompts for selecting and configuring different deployment modes in the OpenFrame CLI configuration wizard.

## Key Components

- **showDeploymentModeSelection()** - Interactive prompt for selecting between OSS Tenant, SaaS Tenant, or SaaS Shared deployment modes
- **showConfigurationModeSelection()** - Interactive prompt for choosing between default or interactive configuration approaches
- **configureWithDefaults()** - Sets up default configuration values without user interaction for the selected deployment mode
- **configureInteractive()** - Runs the full interactive configuration wizard, prompting users through all configuration sections

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/configuration"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/types"
)

func main() {
    wizard := &configuration.ConfigurationWizard{
        // Initialize wizard components
    }
    
    // Get deployment mode selection from user
    deploymentMode, err := wizard.showDeploymentModeSelection()
    if err != nil {
        panic(err)
    }
    
    // Get configuration mode preference
    configMode, err := wizard.showConfigurationModeSelection()
    if err != nil {
        panic(err)
    }
    
    var config *types.ChartConfiguration
    if configMode == "default" {
        config, err = wizard.configureWithDefaults(deploymentMode)
    } else {
        config, err = wizard.configureInteractive(deploymentMode)
    }
    
    if err != nil {
        panic(err)
    }
    
    // Use the generated configuration
    fmt.Printf("Configuration complete for %s mode\n", deploymentMode)
}
```

The file uses the promptui library to create user-friendly terminal interfaces with styled selections and visual feedback, making it easy for users to choose their preferred deployment and configuration options.
