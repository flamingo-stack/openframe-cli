Provides deployment and configuration mode selection functionality for the OpenFrame CLI configuration wizard.

## Key Components

**Deployment Mode Selection:**
- `showDeploymentModeSelection()` - Interactive prompt for choosing between OSS, SaaS, or SaaS Shared deployment modes
- Returns `types.DeploymentMode` enum values based on user selection

**Configuration Mode Selection:**
- `showConfigurationModeSelection()` - Prompts user to choose between default or interactive configuration
- Returns string indicating "default" or "interactive" mode

**Configuration Executors:**
- `configureWithDefaults()` - Creates default configuration without user interaction for specified deployment mode
- `configureInteractive()` - Runs full interactive configuration wizard with step-by-step prompts

## Usage Example

```go
// Create configuration wizard instance
wizard := &ConfigurationWizard{}

// Show deployment mode selection
deploymentMode, err := wizard.showDeploymentModeSelection()
if err != nil {
    return fmt.Errorf("deployment mode selection failed: %w", err)
}

// Show configuration mode selection
configMode, err := wizard.showConfigurationModeSelection()
if err != nil {
    return fmt.Errorf("configuration mode selection failed: %w", err)
}

// Execute appropriate configuration flow
var config *types.ChartConfiguration
if configMode == "default" {
    config, err = wizard.configureWithDefaults(deploymentMode)
} else {
    config, err = wizard.configureInteractive(deploymentMode)
}
```

The file uses `promptui` for interactive command-line prompts and `pterm` for styled terminal output, providing a user-friendly configuration experience.
