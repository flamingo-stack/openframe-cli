This file implements interactive and default configuration modes for a Helm chart deployment wizard, providing users with options to configure their deployment through CLI prompts or use sensible defaults.

## Key Components

- **showDeploymentModeSelection()** - Presents a CLI prompt for selecting deployment mode (OSS, SaaS, or SaaS Shared)
- **showConfigurationModeSelection()** - Prompts user to choose between default or interactive configuration
- **configureWithDefaults()** - Creates a default configuration without user interaction for the specified deployment mode
- **configureInteractive()** - Runs the full interactive configuration wizard with step-by-step prompts

## Usage Example

```go
wizard := &ConfigurationWizard{}

// Get deployment mode from user
deploymentMode, err := wizard.showDeploymentModeSelection()
if err != nil {
    return err
}

// Get configuration preference
configMode, err := wizard.showConfigurationModeSelection()
if err != nil {
    return err
}

// Configure based on user choice
var config *types.ChartConfiguration
if configMode == "default" {
    config, err = wizard.configureWithDefaults(deploymentMode)
} else {
    config, err = wizard.configureInteractive(deploymentMode)
}

if err != nil {
    return fmt.Errorf("configuration failed: %w", err)
}
```

The file uses `promptui` for interactive CLI prompts and `pterm` for styled terminal output, providing a user-friendly configuration experience for Helm chart deployments.
