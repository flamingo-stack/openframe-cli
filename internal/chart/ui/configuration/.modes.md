Handles deployment and configuration mode selection for the OpenFrame CLI chart installation wizard.

## Key Components

**Main Functions:**
- `showDeploymentModeSelection()` - Interactive prompt for selecting deployment mode (OSS, SaaS, SaaS Shared)
- `showConfigurationModeSelection()` - Prompt for choosing default or interactive configuration
- `configureWithDefaults()` - Creates default configuration without user interaction
- `configureInteractive()` - Runs full interactive configuration wizard

**Deployment Modes:**
- OSS Tenant (self-hosted default)
- SaaS Tenant deployment
- SaaS Shared deployment

## Usage Example

```go
wizard := &ConfigurationWizard{}

// Get deployment mode from user
deploymentMode, err := wizard.showDeploymentModeSelection()
if err != nil {
    return err
}

// Get configuration preference
configMode, err := wizard.showConfigurationModeSelection()
if err != nil {
    return err
}

// Configure based on user choice
var config *types.ChartConfiguration
if configMode == "default" {
    config, err = wizard.configureWithDefaults(deploymentMode)
} else {
    config, err = wizard.configureInteractive(deploymentMode)
}
```

The file orchestrates the chart configuration workflow by first determining the deployment target, then applying either automated defaults or guided interactive setup including branch, Docker registry, and ingress configuration.
