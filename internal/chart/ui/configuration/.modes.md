This file handles deployment and configuration mode selection for the OpenFrame CLI configuration wizard, providing both interactive and default setup options.

## Key Components

- **`showDeploymentModeSelection()`** - Interactive prompt for selecting deployment mode (OSS, SaaS, or SaaS Shared)
- **`showConfigurationModeSelection()`** - Interactive prompt for choosing between default or interactive configuration
- **`configureWithDefaults()`** - Creates a default configuration without user interaction
- **`configureInteractive()`** - Runs the full interactive configuration wizard with step-by-step prompts

## Usage Example

```go
// Create configuration wizard instance
wizard := &ConfigurationWizard{}

// Show deployment mode selection
deploymentMode, err := wizard.showDeploymentModeSelection()
if err != nil {
    return fmt.Errorf("deployment mode selection failed: %w", err)
}

// Show configuration mode selection
configMode, err := wizard.showConfigurationModeSelection()
if err != nil {
    return fmt.Errorf("configuration mode selection failed: %w", err)
}

// Configure based on selected mode
var config *types.ChartConfiguration
if configMode == "default" {
    config, err = wizard.configureWithDefaults(deploymentMode)
} else {
    config, err = wizard.configureInteractive(deploymentMode)
}

if err != nil {
    return fmt.Errorf("configuration failed: %w", err)
}
```

The file uses `promptui` for interactive CLI prompts and `pterm` for styled terminal output, supporting three deployment modes and two configuration approaches for flexible OpenFrame setup.
