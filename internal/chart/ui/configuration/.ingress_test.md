Test file for the ingress configurator component that validates Helm chart ingress configuration functionality including localhost and ngrok tunnel setup.

## Key Components

- **TestNewIngressConfigurator**: Tests constructor initialization
- **TestIngressConfigurator_Configure_LocalhostIngress**: Validates localhost ingress configuration
- **TestIngressConfigurator_Configure_NgrokIngress**: Tests ngrok tunnel setup with credentials
- **TestIngressConfigurator_Configure_NgrokWithAllowedIPs**: Validates IP allowlist functionality
- **TestIngressConfigurator_Configure_GetCurrentIngressSettings**: Tests ingress type detection
- **TestIngressConfigurator_configureNgrok_CredentialsValidation**: Validates ngrok credential requirements
- **TestIngressConfigurator_Configure_SwitchIngressTypes**: Tests switching between ingress types
- **TestIngressConfigurator_Configure_NgrokRegistrationURLs**: Validates ngrok documentation URLs
- **TestIngressConfigurator_Configure_IPAllowlistScenarios**: Tests various IP restriction scenarios

## Usage Example

```go
func TestIngressConfiguration(t *testing.T) {
    modifier := templates.NewHelmValuesModifier()
    configurator := NewIngressConfigurator(modifier)
    
    // Test ngrok configuration with IP allowlist
    ngrokConfig := &types.NgrokConfig{
        AuthToken:     "auth_token_123",
        APIKey:        "api_key_456", 
        Domain:        "example.ngrok-free.app",
        UseAllowedIPs: true,
        AllowedIPs:    []string{"192.168.1.1", "10.0.0.1"},
    }
    
    existingValues := map[string]interface{}{}
    err := configurator.applyNgrokConfig(existingValues, ngrokConfig)
    assert.NoError(t, err)
}
```
