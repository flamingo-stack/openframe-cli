Unit test file for the ingress configuration module, providing comprehensive test coverage for ingress setup including localhost and ngrok configurations.

## Key Components

- **TestNewIngressConfigurator** - Tests constructor initialization
- **TestIngressConfigurator_Configure_LocalhostIngress** - Validates localhost ingress configuration
- **TestIngressConfigurator_Configure_NgrokIngress** - Tests ngrok ingress setup with credentials
- **TestIngressConfigurator_Configure_NgrokWithAllowedIPs** - Tests ngrok with IP allowlisting
- **TestIngressConfigurator_Configure_GetCurrentIngressSettings** - Tests detection of current ingress type
- **TestIngressConfigurator_configureNgrok_CredentialsValidation** - Validates ngrok credential requirements
- **TestIngressConfigurator_Configure_SwitchIngressTypes** - Tests switching between ingress types
- **TestIngressConfigurator_Configure_NgrokRegistrationURLs** - Validates ngrok documentation URLs
- **TestIngressConfigurator_Configure_IPAllowlistScenarios** - Tests various IP allowlist configurations

## Usage Example

```go
func TestBasicIngressConfiguration(t *testing.T) {
    // Create ingress configurator
    modifier := templates.NewHelmValuesModifier()
    configurator := NewIngressConfigurator(modifier)
    
    // Test localhost setup
    values := map[string]interface{}{}
    err := configurator.applyLocalhostConfig(values)
    assert.NoError(t, err)
    
    // Test ngrok setup with credentials
    ngrokConfig := &types.NgrokConfig{
        AuthToken: "auth_token_123",
        APIKey:    "api_key_456", 
        Domain:    "example.ngrok-free.app",
    }
    err = configurator.applyNgrokConfig(values, ngrokConfig)
    assert.NoError(t, err)
}
```
