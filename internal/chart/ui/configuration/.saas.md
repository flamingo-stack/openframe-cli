Handles SaaS-specific configuration for the OpenFrame CLI, including repository access tokens, GitHub Container Registry credentials, and branch selection for both SaaS and OSS repositories.

## Key Components

- **configureSaaSDefaults()** - Configures required SaaS settings in default mode, prompting only for essential credentials
- **configureSaaSInteractive()** - Full interactive SaaS configuration with branch selection options
- **configureSaaSBranch()** - Interactive branch configuration for SaaS repository with current branch detection
- **configureOSSBranchForSaaS()** - OSS repository branch configuration in SaaS deployment context
- **getSaaSBranchFromValues()** - Extracts current SaaS branch from existing Helm values

## Usage Example

```go
wizard := &ConfigurationWizard{}
config := &types.ChartConfiguration{}

// Default mode - minimal prompts
err := wizard.configureSaaSDefaults(config)
if err != nil {
    log.Fatal(err)
}

// Interactive mode - full configuration
err = wizard.configureSaaSInteractive(config)
if err != nil {
    log.Fatal(err)
}

// Access configured values
fmt.Printf("SaaS Branch: %s\n", config.SaaSConfig.SaaSBranch)
fmt.Printf("Registry Username: %s\n", config.DockerRegistry.Username)
```

The configuration populates both `SaaSConfig` and `DockerRegistry` fields in the chart configuration and tracks modifications in the `ModifiedSections` slice.
