Handles SaaS-specific configuration for OpenFrame deployments, including repository authentication, Docker registry credentials, and branch management for both SaaS and OSS components.

## Key Components

- **configureSaaSDefaults**: Sets up required SaaS settings in default mode, preserving existing branch values while prompting for credentials
- **configureSaaSInteractive**: Full interactive configuration allowing users to modify all SaaS settings including branch selection
- **configureSaaSBranch**: Interactive branch selection for SaaS repository with current branch preservation
- **configureOSSBranchForSaaS**: OSS repository branch configuration within SaaS deployment context
- **getSaaSBranchFromValues**: Extracts current SaaS branch from existing Helm values with fallback to "main"

## Usage Example

```go
// Configure SaaS in default mode (minimal prompts)
wizard := &ConfigurationWizard{...}
config := &types.ChartConfiguration{...}

err := wizard.configureSaaSDefaults(config)
if err != nil {
    log.Fatal(err)
}

// Configure SaaS interactively (full customization)
err = wizard.configureSaaSInteractive(config)
if err != nil {
    log.Fatal(err)
}

// Access configured SaaS settings
saasConfig := config.SaaSConfig
fmt.Printf("SaaS Branch: %s, OSS Branch: %s", 
    saasConfig.SaaSBranch, saasConfig.OSSBranch)
```

The configuration wizard ensures proper setup of SaaS deployments by collecting necessary authentication tokens, configuring Docker registry access, and managing repository branches for both SaaS and OSS components. It supports both default and interactive modes to accommodate different deployment scenarios.
