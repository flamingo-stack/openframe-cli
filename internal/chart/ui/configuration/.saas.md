This file handles SaaS (Software-as-a-Service) configuration for the OpenFrame CLI tool, providing both interactive and default configuration modes for SaaS deployments.

## Key Components

- **configureSaaSDefaults**: Configures required SaaS settings in default mode, using existing branch values without prompting
- **configureSaaSInteractive**: Full interactive configuration for SaaS settings including branch selection
- **configureSaaSBranch**: Interactive branch configuration for the SaaS repository with current branch detection
- **configureOSSBranchForSaaS**: OSS repository branch configuration in SaaS context
- **getSaaSBranchFromValues**: Extracts current SaaS branch from existing Helm values

## Usage Example

```go
// Configure SaaS in default mode (minimal prompts)
config := &types.ChartConfiguration{
    ExistingValues: existingHelmValues,
}

wizard := &ConfigurationWizard{modifier: modifier}
err := wizard.configureSaaSDefaults(config)
if err != nil {
    return fmt.Errorf("SaaS configuration failed: %w", err)
}

// Configure SaaS interactively (full prompts)
err = wizard.configureSaaSInteractive(config)
if err != nil {
    return fmt.Errorf("interactive SaaS configuration failed: %w", err)
}

// Access configured values
fmt.Printf("SaaS branch: %s\n", config.SaaSConfig.SaaSBranch)
fmt.Printf("Registry user: %s\n", config.DockerRegistry.Username)
```
