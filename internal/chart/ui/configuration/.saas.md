This file implements SaaS (Software as a Service) configuration functionality for the OpenFrame CLI, handling the collection of credentials and repository settings needed for SaaS deployments.

## Key Components

- `configureSaaSDefaults()` - Configures SaaS settings in default mode with minimal prompts
- `configureSaaSInteractive()` - Provides full interactive configuration for SaaS deployments
- `configureSaaSBranch()` - Allows users to configure the SaaS repository branch
- `configureOSSBranchForSaaS()` - Configures OSS repository branch in SaaS context
- `getSaaSBranchFromValues()` - Extracts current SaaS branch from existing Helm values

## Usage Example

```go
wizard := &ConfigurationWizard{}
config := &types.ChartConfiguration{}

// Configure SaaS in interactive mode
err := wizard.configureSaaSInteractive(config)
if err != nil {
    return fmt.Errorf("SaaS configuration failed: %w", err)
}

// Access the configured SaaS settings
saasConfig := config.SaaSConfig
fmt.Printf("SaaS Branch: %s\n", saasConfig.SaaSBranch)
fmt.Printf("OSS Branch: %s\n", saasConfig.OSSBranch)

// Docker registry credentials are also configured
registry := config.DockerRegistry
fmt.Printf("Registry Username: %s\n", registry.Username)
```

The configuration process collects repository access tokens, GitHub Container Registry credentials, and branch settings, storing them in the chart configuration for deployment use.
