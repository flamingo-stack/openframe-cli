Tests for GHCR (GitHub Container Registry) credential extraction and configuration validation functionality.

## Key Components

- **TestConfigurationWizard_ExtractGHCRCredentials**: Comprehensive test suite validating GHCR credential extraction logic from existing Helm values, covering scenarios like missing credentials, existing credentials, default values, and partial configurations
- **TestConfigurationWizard_GHCRCredentialsInConfig**: Test ensuring GHCR credentials are properly stored in the chart configuration structure
- **Test scenarios**: Multiple edge cases including empty values, default credentials, and missing registry sections

## Usage Example

```go
// Run the credential extraction tests
func TestGHCRExtraction(t *testing.T) {
    config := &types.ChartConfiguration{
        ExistingValues: map[string]interface{}{
            "registry": map[string]interface{}{
                "ghcr": map[string]interface{}{
                    "username": "testuser",
                    "email":    "test@example.com",
                },
            },
        },
    }
    
    // Test extracts username and email, checks if credentials exist
    // Logic validates against default values ("default", "default@example.com")
}

// Test credential storage in configuration
config := &types.ChartConfiguration{
    DockerRegistry: &types.DockerRegistryConfig{
        Username: "ghcr-user",
        Password: "ghcr-pass",
        Email:    "ghcr@example.com",
    },
}
```
