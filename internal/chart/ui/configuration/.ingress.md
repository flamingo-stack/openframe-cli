Manages ingress configuration for different deployment types including localhost, Ngrok, and GCP with interactive user prompts and Helm values modification.

## Key Components

- **IngressConfigurator**: Main struct that handles ingress configuration with Helm values modification
- **NewIngressConfigurator()**: Constructor function that creates a new configurator instance
- **Configure()**: Primary method that presents ingress options based on deployment mode and applies selected configuration
- **configureNgrok()**: Handles complete Ngrok setup including credentials collection and IP allowlist configuration
- **applyLocalhostConfig()**: Applies localhost ingress settings to Helm values
- **applyNgrokConfig()**: Applies Ngrok ingress settings with credentials and optional IP restrictions
- **applyGCPConfig()**: Applies GCP ingress configuration with tenant ID setup

## Usage Example

```go
// Create ingress configurator
modifier := templates.NewHelmValuesModifier()
configurator := configuration.NewIngressConfigurator(modifier)

// Configure ingress for a chart
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    ExistingValues: make(map[string]interface{}),
}

// Interactive configuration
err := configurator.Configure(config)
if err != nil {
    log.Fatal(err)
}

// Access configured ingress settings
if config.IngressConfig.Type == types.IngressTypeNgrok {
    fmt.Printf("Ngrok domain: %s", config.IngressConfig.NgrokConfig.Domain)
}
```
