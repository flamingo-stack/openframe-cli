Manages ingress configuration for the OpenFrame CLI, providing interactive setup for localhost, ngrok, and GCP ingress options based on deployment mode.

## Key Components

- `IngressConfigurator` - Main struct that handles ingress configuration with embedded HelmValuesModifier
- `NewIngressConfigurator()` - Factory function that creates a new configurator instance
- `Configure()` - Primary method that guides users through ingress setup based on deployment mode
- `configureNgrok()` - Handles complete Ngrok tunnel setup including credentials and IP allowlists
- `applyLocalhostConfig()` - Applies localhost-only ingress configuration to Helm values
- `applyNgrokConfig()` - Applies Ngrok tunnel configuration with credentials to Helm values  
- `applyGCPConfig()` - Applies Google Cloud Platform ingress configuration to Helm values

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/ui/templates"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/types"
)

// Create configurator with Helm values modifier
modifier := templates.NewHelmValuesModifier()
configurator := NewIngressConfigurator(modifier)

// Configure ingress for a chart
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    ExistingValues: make(map[string]interface{}),
}

err := configurator.Configure(config)
if err != nil {
    log.Fatalf("Ingress configuration failed: %v", err)
}

// Access the configured ingress settings
if config.IngressConfig.Type == types.IngressTypeNgrok {
    fmt.Printf("Ngrok domain: %s\n", config.IngressConfig.NgrokConfig.Domain)
}
```
