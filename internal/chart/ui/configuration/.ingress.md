Manages ingress configuration for Kubernetes deployments, supporting localhost, ngrok tunnel, and GCP ingress options based on deployment mode.

## Key Components

- **IngressConfigurator**: Main struct that handles ingress configuration with Helm values modification
- **Configure()**: Interactive configuration method that prompts users to select ingress type (localhost/ngrok/GCP)
- **configureNgrok()**: Handles complete ngrok setup including credentials and IP allowlist
- **applyLocalhostConfig()**: Applies localhost-only ingress settings to Helm values
- **applyNgrokConfig()**: Applies ngrok tunnel configuration with domain, API key, and auth token
- **applyGCPConfig()**: Configures GCP ingress with tenant ID for SaaS deployments

## Usage Example

```go
// Create ingress configurator
modifier := templates.NewHelmValuesModifier()
configurator := NewIngressConfigurator(modifier)

// Configure ingress interactively
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeSaaS,
    ExistingValues: make(map[string]interface{}),
}

err := configurator.Configure(config)
if err != nil {
    log.Fatal(err)
}

// Access configured ingress settings
if config.IngressConfig.Type == types.IngressTypeNgrok {
    fmt.Printf("Ngrok domain: %s\n", config.IngressConfig.NgrokConfig.Domain)
}
```

The configurator automatically adapts available options based on deployment mode (SaaS vs OSS) and applies the selected configuration to Helm values for deployment.
