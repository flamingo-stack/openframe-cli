Handles ingress configuration for different deployment environments, including localhost, ngrok tunneling for external access, and GCP cloud deployments with interactive credential collection.

## Key Components

- **IngressConfigurator**: Main struct that manages ingress configuration using a Helm values modifier
- **Configure()**: Interactive configuration method that presents different ingress options based on deployment mode (SaaS vs OSS)
- **configureNgrok()**: Complete ngrok setup flow including registration guidance and credential collection
- **applyLocalhostConfig()/applyNgrokConfig()/applyGCPConfig()**: Methods that apply specific ingress configurations to Helm values
- **collectNgrokCredentials()**: Interactive collection of ngrok domain, API key, and auth token with masking
- **configureNgrokIPAllowlist()**: IP restriction configuration for ngrok tunnels

## Usage Example

```go
// Create configurator with Helm values modifier
modifier := templates.NewHelmValuesModifier()
configurator := NewIngressConfigurator(modifier)

// Configure ingress based on deployment mode
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    ExistingValues: make(map[string]interface{}),
}

err := configurator.Configure(config)
if err != nil {
    log.Fatalf("Ingress configuration failed: %v", err)
}

// Result includes ingress type and credentials
fmt.Printf("Configured ingress: %s", config.IngressConfig.Type)
```

The configurator automatically presents appropriate options (localhost/ngrok for OSS, localhost/gcp for SaaS) and guides users through credential setup with helpful registration URLs and secure input masking for sensitive data.
