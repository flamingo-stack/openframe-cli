Manages ingress configuration for OpenFrame deployments, supporting localhost, Ngrok, and GCP ingress types based on deployment mode.

## Key Components

- **IngressConfigurator**: Main configurator that handles ingress setup through interactive prompts
- **NewIngressConfigurator()**: Creates a new ingress configurator with a Helm values modifier
- **Configure()**: Interactive configuration flow that sets up ingress based on deployment mode (SaaS vs OSS)
- **configureNgrok()**: Complete Ngrok setup including credentials and IP allowlist configuration
- **applyLocalhostConfig()**: Applies localhost-only ingress settings to Helm values
- **applyNgrokConfig()**: Applies Ngrok tunnel configuration with domain, API key, and auth token
- **applyGCPConfig()**: Applies Google Cloud Platform ingress settings for SaaS deployments

## Usage Example

```go
// Initialize ingress configurator
modifier := templates.NewHelmValuesModifier()
configurator := NewIngressConfigurator(modifier)

// Configure ingress interactively
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    ExistingValues: make(map[string]interface{}),
}

err := configurator.Configure(config)
if err != nil {
    log.Fatal("Failed to configure ingress:", err)
}

// Access the configured ingress settings
if config.IngressConfig.Type == types.IngressTypeNgrok {
    fmt.Printf("Ngrok domain: %s\n", config.IngressConfig.NgrokConfig.Domain)
}
```

The configurator automatically selects appropriate ingress options based on deployment mode - localhost and Ngrok for OSS deployments, or localhost and GCP for SaaS deployments. For Ngrok configuration, it guides users through domain setup, credential collection, and optional IP allowlist configuration.
