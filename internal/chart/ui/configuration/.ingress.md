Handles ingress configuration for Openframe deployments, supporting localhost, Ngrok, and GCP ingress types with interactive user prompts and Helm values modification.

## Key Components

- **IngressConfigurator**: Main struct that handles ingress configuration with a Helm values modifier
- **Configure()**: Primary method that prompts users to choose ingress type based on deployment mode
- **configureNgrok()**: Complete Ngrok setup flow including credential collection and IP allowlist configuration
- **applyLocalhostConfig()**: Configures localhost-only ingress in Helm values
- **applyNgrokConfig()**: Applies Ngrok configuration with credentials and IP restrictions
- **applyGCPConfig()**: Sets up GCP ingress with tenant ID configuration

## Usage Example

```go
import (
    "github.com/flamingo-stack/openframe-cli/internal/chart/ui/templates"
    "github.com/flamingo-stack/openframe-cli/internal/chart/utils/types"
)

// Create configurator with Helm values modifier
modifier := templates.NewHelmValuesModifier()
configurator := NewIngressConfigurator(modifier)

// Configure ingress for a chart
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    ExistingValues: make(map[string]interface{}),
}

err := configurator.Configure(config)
if err != nil {
    log.Fatal(err)
}

// Access configured ingress settings
ingressType := config.IngressConfig.Type
```

The configurator automatically adjusts available options based on deployment mode (SaaS vs OSS) and handles the complete setup flow including credential collection for external services.
