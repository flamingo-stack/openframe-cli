A utility for reading, modifying, and writing Helm values files in YAML format, providing structured configuration management for OpenFrame deployments across different modes (OSS, SaaS, SaaS Shared).

## Key Components

- **HelmValuesModifier**: Main struct that handles all Helm values operations
- **LoadExistingValues/LoadOrCreateBaseValues**: Methods for reading existing Helm values files
- **ApplyConfiguration**: Applies deployment configuration changes to Helm values
- **CreateTemporaryValuesFile**: Creates temporary values files for deployment operations
- **WriteValues**: Persists modified values back to YAML files
- **Deployment Mode Management**: Handles OSS, SaaS, and SaaS Shared deployment configurations
- **Configuration Extractors**: Methods to read current settings (branch, Docker, ingress, deployment mode)

## Usage Example

```go
// Create a new Helm values modifier
modifier := NewHelmValuesModifier()

// Load existing values or create empty ones
values, err := modifier.LoadOrCreateBaseValues()
if err != nil {
    return fmt.Errorf("failed to load values: %w", err)
}

// Apply configuration changes
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    Branch:         stringPtr("feature-branch"),
    DockerRegistry: &types.DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass",
        Email:    "user@example.com",
    },
}

err = modifier.ApplyConfiguration(values, config)
if err != nil {
    return fmt.Errorf("failed to apply config: %w", err)
}

// Write updated values back to file
err = modifier.WriteValues(values, "helm-values.yaml")
if err != nil {
    return fmt.Errorf("failed to write values: %w", err)
}

// Extract current settings
currentMode := modifier.GetCurrentDeploymentMode(values)
currentBranch := modifier.GetCurrentOSSBranch(values)
dockerSettings := modifier.GetCurrentDockerSettings(values)
```
