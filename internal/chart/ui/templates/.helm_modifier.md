This Go file provides a comprehensive utility for reading, modifying, and writing Helm values files for chart configuration management. It handles different deployment modes (OSS, SaaS, SaaS Shared) and manages various configuration settings like Docker registry credentials, repository branches, and ingress settings.

## Key Components

- **HelmValuesModifier**: Main struct that handles all Helm values operations
- **LoadExistingValues()**: Loads and parses existing YAML values files
- **LoadOrCreateBaseValues()**: Loads values from current directory or creates empty defaults
- **ApplyConfiguration()**: Applies chart configuration changes to values map
- **WriteValues()**: Writes updated values back to YAML files
- **CreateTemporaryValuesFile()**: Creates temporary values files for processing
- **Get methods**: Extract current settings (branch, Docker config, deployment mode, ingress)

## Usage Example

```go
// Create modifier and load existing values
modifier := NewHelmValuesModifier()
values, err := modifier.LoadOrCreateBaseValues()
if err != nil {
    log.Fatal(err)
}

// Apply configuration changes
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    Branch: stringPtr("develop"),
    DockerRegistry: &types.DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass", 
        Email: "user@example.com",
    },
}

err = modifier.ApplyConfiguration(values, config)
if err != nil {
    log.Fatal(err)
}

// Write updated values
err = modifier.WriteValues(values, "helm-values.yaml")
```
