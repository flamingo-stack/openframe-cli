Provides utilities for reading, modifying, and writing Helm values YAML files with support for different deployment modes (OSS, SaaS, SaaS Shared) and configuration management.

## Key Components

- **HelmValuesModifier**: Main struct that handles all Helm values file operations
- **LoadExistingValues**: Reads and parses existing Helm values from file
- **LoadOrCreateBaseValues**: Loads values from current directory or creates empty defaults
- **ApplyConfiguration**: Applies deployment mode, branch, registry, and SaaS configurations
- **CreateTemporaryValuesFile**: Creates temporary values files for Helm operations
- **WriteValues**: Marshals and writes updated values back to YAML files
- **Getter methods**: Extract current settings for deployment mode, branches, Docker registry, and ingress

## Usage Example

```go
// Create a new modifier
modifier := NewHelmValuesModifier()

// Load existing values or create empty ones
values, err := modifier.LoadOrCreateBaseValues()
if err != nil {
    log.Fatal(err)
}

// Apply configuration changes
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    Branch:         stringPtr("develop"),
    DockerRegistry: &types.DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass",
        Email:    "user@example.com",
    },
}

err = modifier.ApplyConfiguration(values, config)
if err != nil {
    log.Fatal(err)
}

// Write updated values back to file
err = modifier.WriteValues(values, "helm-values.yaml")
if err != nil {
    log.Fatal(err)
}

// Get current settings
currentMode := modifier.GetCurrentDeploymentMode(values)
currentBranch := modifier.GetCurrentOSSBranch(values)
dockerSettings := modifier.GetCurrentDockerSettings(values)
```
