This file provides a comprehensive utility for managing Helm values files, enabling reading, modification, and writing of YAML-based Helm configurations for different deployment modes.

## Key Components

- **HelmValuesModifier**: Main struct that handles all Helm values operations
- **LoadExistingValues/LoadOrCreateBaseValues**: Functions to load existing Helm values files or create defaults
- **ApplyConfiguration**: Applies chart configuration changes including deployment mode, Docker registry, and SaaS settings
- **CreateTemporaryValuesFile**: Creates temporary values files for Helm operations
- **WriteValues**: Writes updated values back to YAML files
- **Getter methods**: Extract current settings like deployment mode, Docker configuration, and branch information

## Usage Example

```go
// Create a new Helm values modifier
modifier := NewHelmValuesModifier()

// Load existing values or create base values
values, err := modifier.LoadOrCreateBaseValues()
if err != nil {
    log.Fatal(err)
}

// Apply configuration changes
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    Branch: stringPtr("develop"),
    DockerRegistry: &types.DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass",
        Email: "user@example.com",
    },
}

err = modifier.ApplyConfiguration(values, config)
if err != nil {
    log.Fatal(err)
}

// Write updated values to file
err = modifier.WriteValues(values, "helm-values.yaml")
if err != nil {
    log.Fatal(err)
}

// Extract current settings
currentMode := modifier.GetCurrentDeploymentMode(values)
currentBranch := modifier.GetCurrentOSSBranch(values)
```
