A utility for reading, modifying, and writing Helm values files to manage deployment configurations for the OpenFrame CLI. It provides methods to handle different deployment modes, repository branches, Docker registry settings, and SaaS-specific configurations.

## Key Components

- **HelmValuesModifier**: Main struct that handles all Helm values file operations
- **LoadExistingValues/LoadOrCreateBaseValues**: Load configuration from existing files or create defaults
- **ApplyConfiguration**: Apply deployment mode, branch, registry, and SaaS settings to Helm values
- **WriteValues/CreateTemporaryValuesFile**: Write modified values to files
- **Getter methods**: Extract current settings for deployment mode, branches, Docker config, and ingress

## Usage Example

```go
// Create modifier and load existing values
modifier := NewHelmValuesModifier()
values, err := modifier.LoadOrCreateBaseValues()
if err != nil {
    log.Fatal(err)
}

// Create configuration changes
config := &types.ChartConfiguration{
    DeploymentMode: &types.DeploymentModeOSS,
    Branch:         stringPtr("develop"),
    DockerRegistry: &types.DockerRegistryConfig{
        Username: "myuser",
        Password: "mypassword", 
        Email:    "user@example.com",
    },
}

// Apply changes and write to file
if err := modifier.ApplyConfiguration(values, config); err != nil {
    log.Fatal(err)
}

if err := modifier.WriteValues(values, "helm-values.yaml"); err != nil {
    log.Fatal(err)
}

// Extract current settings
currentMode := modifier.GetCurrentDeploymentMode(values)
currentBranch := modifier.GetCurrentOSSBranch(values)
dockerSettings := modifier.GetCurrentDockerSettings(values)
```
