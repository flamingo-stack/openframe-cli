This file contains comprehensive test cases for a Helm values modifier component that handles YAML configuration files for Kubernetes deployments.

## Key Components

- **TestHelmValuesModifier_New** - Tests modifier instantiation
- **TestHelmValuesModifier_LoadExistingValues** - Tests loading/parsing YAML files with error handling for missing files, invalid YAML, and empty files
- **TestHelmValuesModifier_GetCurrentOSSBranch** - Tests extracting OSS deployment branch from values
- **TestHelmValuesModifier_GetCurrentDockerSettings** - Tests retrieving Docker registry configuration
- **TestHelmValuesModifier_ApplyConfiguration** - Tests applying branch and Docker configuration changes
- **TestHelmValuesModifier_WriteValues** - Tests writing modified values back to YAML files
- **TestHelmValuesModifier_GetCurrentIngressSettings** - Tests ingress configuration extraction (partially shown)

## Usage Example

```go
// Test loading existing Helm values
func TestLoadValues(t *testing.T) {
    modifier := NewHelmValuesModifier()
    
    // Load existing YAML configuration
    values, err := modifier.LoadExistingValues("values.yaml")
    assert.NoError(t, err)
    
    // Apply new configuration
    config := &types.ChartConfiguration{
        Branch: &branch,
        DockerRegistry: &types.DockerRegistryConfig{
            Username: "newuser",
            Password: "newpass",
        },
        ModifiedSections: []string{"branch", "docker"},
    }
    
    err = modifier.ApplyConfiguration(values, config)
    assert.NoError(t, err)
    
    // Write updated values
    err = modifier.WriteValues(values, "updated-values.yaml")
    assert.NoError(t, err)
}
```
