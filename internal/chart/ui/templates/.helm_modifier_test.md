This test file validates the functionality of a Helm values modifier component that handles reading, modifying, and writing Helm chart values files for different deployment configurations.

## Key Components

- **TestHelmValuesModifier_New**: Tests constructor initialization
- **TestHelmValuesModifier_LoadExistingValues**: Tests loading and parsing YAML values files with various scenarios (valid, invalid, empty, not found)
- **TestHelmValuesModifier_GetCurrentOSSBranch**: Tests extraction of OSS deployment branch settings
- **TestHelmValuesModifier_GetCurrentDockerSettings**: Tests retrieval of Docker registry configuration
- **TestHelmValuesModifier_ApplyConfiguration**: Tests applying chart configuration changes for branches and Docker settings
- **TestHelmValuesModifier_WriteValues**: Tests writing modified values back to YAML files

## Usage Example

```go
func TestHelmModification(t *testing.T) {
    // Initialize modifier
    modifier := NewHelmValuesModifier()
    
    // Load existing values
    values, err := modifier.LoadExistingValues("values.yaml")
    require.NoError(t, err)
    
    // Apply configuration changes
    config := &types.ChartConfiguration{
        Branch: &newBranch,
        DeploymentMode: &deploymentMode,
        ModifiedSections: []string{"branch"},
    }
    err = modifier.ApplyConfiguration(values, config)
    assert.NoError(t, err)
    
    // Write updated values
    err = modifier.WriteValues(values, "updated-values.yaml")
    assert.NoError(t, err)
}
```

The tests cover error handling, default value scenarios, and various deployment configurations including OSS and Docker registry settings.
