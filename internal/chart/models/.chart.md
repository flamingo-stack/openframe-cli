A Go package defining data structures and types for managing Helm chart information and categorization within a Kubernetes deployment system.

## Key Components

**Types:**
- `ChartInfo` - Struct containing comprehensive metadata about installed Helm charts
- `ChartType` - String-based type for categorizing different chart deployment patterns

**Constants:**
- `ChartTypeArgoCD` - Represents ArgoCD application charts
- `ChartTypeAppOfApps` - Represents the "app of apps" pattern for managing multiple applications

## Usage Example

```go
package main

import (
    "fmt"
    "your-project/models"
)

func main() {
    // Create chart info for an installed application
    chart := models.ChartInfo{
        Name:       "my-web-app",
        Namespace:  "production",
        Status:     "deployed",
        Version:    "1.2.3",
        AppVersion: "2.1.0",
    }
    
    // Categorize chart types
    var chartType models.ChartType
    if chart.Name == "argocd" {
        chartType = models.ChartTypeArgoCD
    } else {
        chartType = models.ChartTypeAppOfApps
    }
    
    fmt.Printf("Chart %s (%s) is %s in %s namespace\n", 
        chart.Name, chartType, chart.Status, chart.Namespace)
}
```

The `ChartInfo` struct provides a standardized way to track Helm chart deployments with essential metadata like deployment status and version information, while `ChartType` enables categorization of different GitOps and application management patterns.
