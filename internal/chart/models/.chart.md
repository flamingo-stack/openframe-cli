Defines data models for Helm chart management and categorization within the OpenFrame platform.

## Key Components

### Structs
- **`ChartInfo`** - Represents metadata for an installed Helm chart including name, namespace, status, version, and application version
- **`ChartType`** - Type alias for string-based chart categorization

### Constants
- **`ChartTypeArgoCD`** - Constant for ArgoCD chart type classification
- **`ChartTypeAppOfApps`** - Constant for App-of-Apps pattern chart type

## Usage Example

```go
package main

import "github.com/openframe-org/openframe/internal/models"

func main() {
    // Create chart info for an installed chart
    chart := models.ChartInfo{
        Name:       "my-application",
        Namespace:  "default",
        Status:     "deployed",
        Version:    "1.0.0",
        AppVersion: "2.1.3",
    }

    // Use chart type constants for categorization
    var chartType models.ChartType = models.ChartTypeArgoCD
    
    // Check chart type
    if chartType == models.ChartTypeAppOfApps {
        // Handle App-of-Apps pattern
    }
}
```

This model provides a standardized way to represent Helm chart information and supports GitOps workflows through ArgoCD and App-of-Apps pattern recognition.
