This file defines data structures and constants for representing Helm chart information and types within the application.

## Key Components

- **ChartInfo**: Struct containing metadata about an installed Helm chart including name, namespace, status, version, and application version
- **ChartType**: String-based type for categorizing different chart types
- **Chart type constants**: Predefined chart types for ArgoCD and App-of-Apps patterns

## Usage Example

```go
package main

import "your-module/models"

func main() {
    // Create chart information
    chart := models.ChartInfo{
        Name:       "my-app",
        Namespace:  "default",
        Status:     "deployed",
        Version:    "1.0.0",
        AppVersion: "2.1.3",
    }

    // Use chart types
    var chartType models.ChartType = models.ChartTypeArgoCD
    
    // Check chart type
    if chartType == models.ChartTypeAppOfApps {
        // Handle app-of-apps pattern
    }
}
```

This model is typically used for tracking and managing Helm chart deployments, particularly in GitOps workflows with ArgoCD.
