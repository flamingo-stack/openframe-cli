Defines data structures for representing Helm chart information and chart type classifications within the application.

## Key Components

- **`ChartInfo`** - Struct containing metadata about installed Helm charts including name, namespace, status, version, and application version
- **`ChartType`** - String-based type for categorizing different chart types
- **Chart Type Constants**:
  - `ChartTypeArgoCD` - Represents ArgoCD chart installations
  - `ChartTypeAppOfApps` - Represents App-of-Apps pattern charts

## Usage Example

```go
// Create chart information for an installed chart
chart := &ChartInfo{
    Name:       "my-app",
    Namespace:  "production",
    Status:     "deployed",
    Version:    "1.0.0",
    AppVersion: "2.3.1",
}

// Use chart type constants
func processChart(chartType ChartType) {
    switch chartType {
    case ChartTypeArgoCD:
        // Handle ArgoCD specific logic
        fmt.Println("Processing ArgoCD chart")
    case ChartTypeAppOfApps:
        // Handle App-of-Apps pattern
        fmt.Println("Processing App-of-Apps chart")
    }
}

// Example usage
processChart(ChartTypeArgoCD)
```

This model provides a foundation for tracking Helm chart deployments and implementing chart-type-specific business logic throughout the application.
