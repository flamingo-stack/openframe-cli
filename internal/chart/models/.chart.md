This file defines data structures for representing Helm chart information and chart types within the application.

## Key Components

- **ChartInfo struct**: Contains metadata about installed Helm charts including name, namespace, status, version, and application version
- **ChartType**: Custom string type for categorizing different chart types
- **Chart type constants**: Pre-defined chart types for ArgoCD and App-of-Apps patterns

## Usage Example

```go
// Create chart information
chart := models.ChartInfo{
    Name:       "my-application",
    Namespace:  "production",
    Status:     "deployed",
    Version:    "1.2.3",
    AppVersion: "v2.1.0",
}

// Use chart type constants
var chartType models.ChartType = models.ChartTypeArgoCD

// Switch on chart type
switch chartType {
case models.ChartTypeArgoCD:
    // Handle ArgoCD chart
case models.ChartTypeAppOfApps:
    // Handle App-of-Apps pattern
}
```
