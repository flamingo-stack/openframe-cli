Defines data models for Helm chart management and tracking within a Kubernetes environment. This file provides structures for representing chart metadata and classification.

## Key Components

- **`ChartInfo`** - Struct containing complete metadata about an installed Helm chart including name, namespace, deployment status, chart version, and application version
- **`ChartType`** - String-based type for categorizing different chart types
- **Chart Type Constants**:
  - `ChartTypeArgoCD` - Represents ArgoCD application charts
  - `ChartTypeAppOfApps` - Represents App of Apps pattern charts

## Usage Example

```go
// Create chart information for a deployed application
chartInfo := models.ChartInfo{
    Name:       "my-app",
    Namespace:  "production",
    Status:     "deployed",
    Version:    "1.2.3",
    AppVersion: "v2.0.1",
}

// Use chart types for categorization
func processChart(chartType models.ChartType) {
    switch chartType {
    case models.ChartTypeArgoCD:
        // Handle ArgoCD specific logic
    case models.ChartTypeAppOfApps:
        // Handle App of Apps pattern
    }
}

processChart(models.ChartTypeArgoCD)
```
