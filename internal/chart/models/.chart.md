Contains data structures for representing Helm chart information and chart type classifications in the application.

## Key Components

**Types:**
- `ChartInfo` - Struct containing essential metadata about an installed Helm chart
- `ChartType` - String-based enum for categorizing different chart types

**Constants:**
- `ChartTypeArgoCD` - Represents ArgoCD application charts
- `ChartTypeAppOfApps` - Represents App of Apps pattern charts

## Usage Example

```go
// Create chart information
chart := models.ChartInfo{
    Name:       "my-app",
    Namespace:  "production",
    Status:     "deployed",
    Version:    "1.2.3",
    AppVersion: "v2.1.0",
}

// Use chart types for categorization
var chartType models.ChartType = models.ChartTypeArgoCD

// Switch based on chart type
switch chartType {
case models.ChartTypeArgoCD:
    // Handle ArgoCD chart
case models.ChartTypeAppOfApps:
    // Handle App of Apps pattern
}
```

The `ChartInfo` struct provides a standardized way to track chart deployments, while `ChartType` enables type-safe classification of different chart patterns commonly used in GitOps workflows.
