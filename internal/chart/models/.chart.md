Defines data structures for representing Helm chart information and chart type classifications in a Kubernetes deployment management system.

## Key Components

**ChartInfo struct**
- `Name` - Chart name identifier
- `Namespace` - Kubernetes namespace where chart is deployed
- `Status` - Current deployment status
- `Version` - Chart version number
- `AppVersion` - Application version being deployed

**ChartType constants**
- `ChartTypeArgoCD` - Represents ArgoCD chart deployments
- `ChartTypeAppOfApps` - Represents App-of-Apps pattern charts

## Usage Example

```go
package main

import "your-module/models"

func main() {
    // Create chart information
    chart := models.ChartInfo{
        Name:       "my-application",
        Namespace:  "production",
        Status:     "deployed",
        Version:    "1.2.3",
        AppVersion: "2.1.0",
    }
    
    // Use chart types for classification
    var chartType models.ChartType = models.ChartTypeArgoCD
    
    if chartType == models.ChartTypeAppOfApps {
        // Handle app-of-apps pattern
    }
}
```

These models are typically used in Helm chart management systems to track deployed applications and categorize different deployment patterns.
