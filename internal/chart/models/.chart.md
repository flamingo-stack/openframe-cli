Defines data structures for representing Helm chart information and chart type classifications within the models package.

## Key Components

**ChartInfo struct**
- `Name` - Chart name identifier
- `Namespace` - Kubernetes namespace where chart is deployed
- `Status` - Current deployment status
- `Version` - Chart version
- `AppVersion` - Application version contained in the chart

**ChartType constants**
- `ChartTypeArgoCD` - Represents ArgoCD chart type
- `ChartTypeAppOfApps` - Represents App-of-Apps pattern chart type

## Usage Example

```go
// Create chart information
chart := &models.ChartInfo{
    Name:       "my-application",
    Namespace:  "production",
    Status:     "deployed",
    Version:    "1.2.3",
    AppVersion: "2.1.0",
}

// Use chart types
func deployChart(chartType models.ChartType) {
    switch chartType {
    case models.ChartTypeArgoCD:
        // Deploy ArgoCD chart
    case models.ChartTypeAppOfApps:
        // Deploy App-of-Apps pattern
    }
}

deployChart(models.ChartTypeArgoCD)
```
