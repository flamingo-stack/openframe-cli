Defines data structures for representing Helm chart information and chart types within the application.

## Key Components

- **`ChartInfo`** - Struct containing metadata about an installed Helm chart including name, namespace, status, version, and application version
- **`ChartType`** - String-based type for categorizing different chart types
- **Chart Type Constants**:
  - `ChartTypeArgoCD` - Represents ArgoCD charts
  - `ChartTypeAppOfApps` - Represents App-of-Apps pattern charts

## Usage Example

```go
package main

import "your-project/models"

func main() {
    // Create chart info for an installed chart
    chart := models.ChartInfo{
        Name:       "my-app",
        Namespace:  "production",
        Status:     "deployed",
        Version:    "1.2.3",
        AppVersion: "2.1.0",
    }

    // Use chart type constants
    var chartType models.ChartType = models.ChartTypeArgoCD
    
    if chartType == models.ChartTypeAppOfApps {
        // Handle app-of-apps pattern
    }
}
```
