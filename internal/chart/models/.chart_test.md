Test suite that validates the behavior and structure of chart-related models, ensuring proper initialization, value assignment, and type constants for Helm chart information.

## Key Components

- **TestChartInfo_DefaultValues**: Tests that `ChartInfo` struct initializes with empty default values
- **TestChartInfo_WithValues**: Validates that `ChartInfo` fields can be properly set and retrieved
- **TestChartType_Constants**: Verifies chart type constants match expected values
- **TestChartType_StringConversion**: Tests string conversion of chart type constants

## Usage Example

Run these tests to validate your chart models:

```go
// Run all chart model tests
go test ./models -run TestChart

// Run specific test
go test ./models -run TestChartInfo_DefaultValues

// Run with verbose output
go test -v ./models -run TestChart
```

The tests verify that `ChartInfo` supports common Helm chart properties like name, namespace, status, version, and app version, while `ChartType` constants properly represent ArgoCD and App-of-Apps chart types.
