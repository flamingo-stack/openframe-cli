Test suite for the ChartInfo model and ChartType constants, validating chart metadata structure and type definitions used in Kubernetes Helm chart operations.

## Key Components

- **TestChartInfo_DefaultValues**: Validates that ChartInfo struct initializes with empty default values
- **TestChartInfo_WithValues**: Tests ChartInfo struct with populated fields (name, namespace, status, version, app version)
- **TestChartType_Constants**: Verifies ChartType constant definitions for ArgoCD and App-of-Apps patterns
- **TestChartType_StringConversion**: Tests string representation of ChartType constants

## Usage Example

```go
// Run all chart-related tests
go test -run TestChart

// Run specific test for chart info validation
go test -run TestChartInfo_WithValues

// Test ChartInfo struct behavior
info := &ChartInfo{
    Name:       "my-app",
    Namespace:  "production",
    Status:     "deployed",
    Version:    "2.1.0",
    AppVersion: "v1.5.2",
}

// Test ChartType usage
chartType := ChartTypeArgoCD
fmt.Println(string(chartType)) // Output: "argocd"
```

These tests ensure the chart models work correctly for Helm chart metadata tracking and ArgoCD application management workflows.
