Test suite for the AppOfAppsConfig model that validates configuration management for ArgoCD app-of-apps pattern deployments.

## Key Components

- **TestNewAppOfAppsConfig** - Tests the constructor function and default configuration values
- **TestAppOfAppsConfig_GetGitURL** - Validates Git URL generation for Helm chart repositories
- **TestAppOfAppsConfig_GetGitURL_WithGitSuffix** - Tests Git URL handling when repository URLs end with `.git`
- **TestAppOfAppsConfig_Fields** - Verifies all configuration fields are properly accessible
- **TestAppOfAppsConfig_CompleteConfiguration** - Tests a fully configured instance with custom values

## Usage Example

```go
// Run specific test
go test -run TestNewAppOfAppsConfig

// Run all tests in the file
go test ./models -v

// Test with coverage
go test -cover ./models

// Example of what's being tested - creating a new config
config := NewAppOfAppsConfig()
// Validates defaults: GitHub repo, branch "main", ArgoCD namespace, 60m timeout

// Testing Git URL generation
config := &AppOfAppsConfig{
    GitHubRepo:   "https://github.com/test/repo",
    GitHubBranch: "develop", 
    ChartPath:    "charts/app-of-apps",
}
gitURL := config.GetGitURL() // Returns: git+https://github.com/test/repo@charts/app-of-apps?ref=develop
```
