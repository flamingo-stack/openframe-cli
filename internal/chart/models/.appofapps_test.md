Test suite for the AppOfAppsConfig model that validates configuration creation, Git URL generation, and field accessibility for ArgoCD app-of-apps management.

## Key Components

- **TestNewAppOfAppsConfig** - Verifies default configuration initialization
- **TestAppOfAppsConfig_GetGitURL** - Tests Git URL generation for Helm chart repositories
- **TestAppOfAppsConfig_GetGitURL_WithGitSuffix** - Validates URL handling with .git suffix
- **TestAppOfAppsConfig_Fields** - Ensures all configuration fields are properly accessible
- **TestAppOfAppsConfig_CompleteConfiguration** - Tests comprehensive configuration scenarios

## Usage Example

```go
// Run specific test
go test -run TestNewAppOfAppsConfig

// Run all AppOfAppsConfig tests
go test -run TestAppOfAppsConfig

// Run with verbose output
go test -v ./models -run TestAppOfAppsConfig
```

The tests validate that the AppOfAppsConfig properly handles:
- Default values for GitHub repository, branch, and ArgoCD settings
- Git URL construction for Helm charts with proper formatting
- Field assignment and retrieval for all configuration properties
- Edge cases like repositories with .git suffixes
