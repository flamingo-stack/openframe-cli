This file defines configuration structures and utilities for managing app-of-apps installations using Helm and Git repositories. It provides a standardized way to configure GitOps-style deployments with ArgoCD.

## Key Components

- **AppOfAppsConfig**: Main configuration struct containing GitHub repository details, certificate paths, values files, and Helm deployment settings
- **NewAppOfAppsConfig()**: Constructor function that creates a new configuration with sensible defaults for OpenFrame OSS tenant deployment
- **GetGitURL()**: Method that formats repository URLs for compatibility with the helm-git plugin v1.4.0 format

## Usage Example

```go
// Create default configuration
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.Namespace = "custom-namespace"
config.ValuesFile = "values/production.yaml"
config.CertDir = "/etc/ssl/certs"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use in helm command
fmt.Printf("helm install my-app %s --namespace %s --timeout %s", 
    gitURL, config.Namespace, config.Timeout)
```

The configuration supports GitOps workflows by enabling deployment of application manifests directly from Git repositories through Helm charts.
