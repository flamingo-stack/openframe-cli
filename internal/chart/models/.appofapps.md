Provides configuration management for ArgoCD app-of-apps installations using Helm and Git repositories. This model handles the necessary parameters and URL formatting for deploying application hierarchies through the helm-git plugin.

## Key Components

- **AppOfAppsConfig**: Main configuration struct containing GitHub repository settings, certificate directory, values file path, and Helm deployment parameters
- **NewAppOfAppsConfig()**: Constructor function that creates a new config with sensible defaults for the flamingo-stack repository
- **GetGitURL()**: Method that formats the repository URL into the helm-git plugin v1.4.0 format for Helm chart installation

## Usage Example

```go
// Create default configuration
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.CertDir = "/path/to/certs"
config.ValuesFile = "custom-values.yaml"
config.Timeout = "30m"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use with Helm installation command
fmt.Printf("helm install app-of-apps %s --namespace %s --timeout %s", 
    gitURL, config.Namespace, config.Timeout)
```
