This file defines configuration structures and utilities for managing Kubernetes app-of-apps installations using Helm and Git repositories.

## Key Components

- **`AppOfAppsConfig`** - Main configuration struct containing GitHub repository settings, certificate directories, values files, and Helm deployment parameters
- **`NewAppOfAppsConfig()`** - Constructor function that returns a default configuration with sensible defaults for the Flamingo stack
- **`GetGitURL()`** - Method that formats repository information into a helm-git plugin compatible URL

## Usage Example

```go
// Create default configuration
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.Namespace = "my-argocd"
config.ValuesFile = "/path/to/values.yaml"
config.CertDir = "/etc/certs"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use in Helm installation commands
fmt.Printf("Installing from: %s\n", gitURL)
```

The configuration supports the helm-git plugin v1.4.0 format and is designed to work with ArgoCD app-of-apps patterns for GitOps deployments. Authentication is handled externally through environment variables or Git credentials.
