Defines configuration management for ArgoCD app-of-apps pattern installations using Helm with Git repositories. Provides structured configuration and URL formatting for the helm-git plugin.

## Key Components

- **`AppOfAppsConfig`** - Configuration struct containing Git repository details, certificate paths, Helm settings, and deployment parameters
- **`NewAppOfAppsConfig()`** - Factory function that creates a config instance with sensible defaults for OpenFrame OSS tenant deployment
- **`GetGitURL()`** - Formats repository information into helm-git plugin compatible URL syntax

## Usage Example

```go
// Create config with defaults
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.CertDir = "/etc/ssl/certs"
config.ValuesFile = "custom-values.yaml"
config.Timeout = "90m"

// Get formatted Git URL for Helm
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use in Helm commands
fmt.Printf("helm install my-app %s --namespace %s --timeout %s", 
    gitURL, config.Namespace, config.Timeout)
```

The configuration supports GitOps workflows where ArgoCD manages application deployments through Helm charts stored in Git repositories.
