This file defines configuration for managing ArgoCD app-of-apps installations using Helm with Git repositories.

## Key Components

- **`AppOfAppsConfig`** - Main configuration struct containing Git repository details, certificate directories, values files, and Helm settings
- **`NewAppOfAppsConfig()`** - Constructor function that returns a config instance with sensible defaults for the flamingo-stack/openframe-oss-tenant repository
- **`GetGitURL()`** - Method that formats the Git URL for compatibility with the helm-git plugin v1.4.0

## Usage Example

```go
// Create config with defaults
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop" 
config.CertDir = "/etc/ssl/certs"
config.ValuesFile = "custom-values.yaml"
config.Timeout = "30m"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use with different repository
config.GitHubRepo = "https://github.com/myorg/myrepo"
config.ChartPath = "charts/app-of-apps"
gitURL = config.GetGitURL()
// Returns: "git+https://github.com/myorg/myrepo@charts/app-of-apps?ref=develop"
```
