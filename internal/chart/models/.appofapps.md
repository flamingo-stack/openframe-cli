Configuration model for ArgoCD app-of-apps pattern deployment using Helm with git-based chart sources.

## Key Components

**AppOfAppsConfig**
- Configuration struct containing GitHub repository details, certificate directories, values files, and Helm deployment settings
- Manages git URL formatting for helm-git plugin integration
- Supports customizable namespaces, timeouts, and chart paths

**NewAppOfAppsConfig()**
- Factory function that creates default configuration for Flamingo Stack's OpenFrame tenant repository
- Sets sensible defaults: main branch, argocd namespace, 60-minute timeout

**GetGitURL()**
- Formats repository URL for helm-git plugin v1.4.0 compatibility
- Generates git+https URL format with branch reference and chart path
- Handles URL normalization by removing .git suffixes

## Usage Example

```go
// Create default configuration
config := NewAppOfAppsConfig()

// Customize for specific deployment
config.GitHubBranch = "develop"
config.Namespace = "gitops"
config.CertDir = "/etc/ssl/certs"
config.ValuesFile = "values-prod.yaml"

// Get formatted git URL for Helm
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use with custom repository
customConfig := &AppOfAppsConfig{
    GitHubRepo:   "https://github.com/myorg/charts.git",
    GitHubBranch: "release/v1.0",
    ChartPath:    "apps/platform",
    Namespace:    "platform",
    Timeout:      "30m",
}
```
