This file defines configuration structures and utilities for managing app-of-apps installations using Helm and GitOps patterns, specifically designed for ArgoCD deployments with the helm-git plugin.

## Key Components

- **`AppOfAppsConfig`** - Configuration struct containing GitHub repository details, certificate directories, values files, and Helm deployment settings
- **`NewAppOfAppsConfig()`** - Constructor function that returns a pre-configured instance with sensible defaults for Flamingo Stack deployments
- **`GetGitURL()`** - Method that formats repository information into a helm-git plugin compatible URL string

## Usage Example

```go
// Create default configuration
config := NewAppOfAppsConfig()

// Customize for specific deployment
config.GitHubBranch = "develop" 
config.CertDir = "/etc/ssl/certs"
config.ValuesFile = "values-prod.yaml"
config.Timeout = "120m"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use in Helm command
fmt.Printf("helm install app-of-apps %s --namespace %s --timeout %s", 
    gitURL, config.Namespace, config.Timeout)
```

The `GetGitURL()` method specifically formats URLs for helm-git plugin v1.4.0+ compatibility, enabling direct installation of Helm charts from Git repositories without manual cloning.
