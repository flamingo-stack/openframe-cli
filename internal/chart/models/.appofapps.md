This file defines the configuration structure and utilities for app-of-apps pattern installations, typically used with ArgoCD and Helm for managing multiple applications as a single unit.

## Key Components

- **`AppOfAppsConfig`** - Main configuration struct containing GitHub repository settings, certificate directory, values file path, and Helm deployment parameters
- **`NewAppOfAppsConfig()`** - Constructor function that returns a pre-configured instance with sensible defaults for the flamingo-stack/openframe-oss-tenant repository
- **`GetGitURL()`** - Method that formats the repository URL into helm-git plugin compatible format for Git-based Helm chart installations

## Usage Example

```go
// Create config with defaults
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.Namespace = "my-argocd"
config.CertDir = "/path/to/certs"
config.ValuesFile = "custom-values.yaml"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use in Helm command
fmt.Printf("helm install my-app %s --namespace %s", gitURL, config.Namespace)
```
