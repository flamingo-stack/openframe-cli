Defines configuration and utilities for managing app-of-apps Helm chart installations, particularly for ArgoCD-based GitOps workflows.

## Key Components

**Types:**
- `AppOfAppsConfig` - Configuration struct for app-of-apps installation with GitHub repository, certificate, values, and Helm settings

**Functions:**
- `NewAppOfAppsConfig()` - Creates a new configuration instance with sensible defaults
- `GetGitURL()` - Formats Git repository URL for helm-git plugin compatibility

## Usage Example

```go
// Create configuration with defaults
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.ValuesFile = "/path/to/values.yaml"
config.CertDir = "/etc/ssl/certs"
config.Timeout = "30m"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop

// Use with different repository
config.GitHubRepo = "https://github.com/myorg/myrepo"
config.ChartPath = "charts/applications"
gitURL = config.GetGitURL()
// Returns: git+https://github.com/myorg/myrepo@charts/applications?ref=develop
```
