Configuration model for deploying app-of-apps patterns using ArgoCD and Helm with Git repositories. Provides structured configuration and URL formatting for the helm-git plugin.

## Key Components

- **AppOfAppsConfig**: Main configuration struct containing Git repository settings, certificate directory, values file path, and Helm deployment parameters
- **NewAppOfAppsConfig()**: Factory function that creates a new config instance with sensible defaults for the flamingo-stack repository
- **GetGitURL()**: Method that formats the Git URL according to helm-git plugin v1.4.0 specification

## Usage Example

```go
// Create config with defaults
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.CertDir = "/path/to/certs"
config.ValuesFile = "custom-values.yaml"
config.Timeout = "30m"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use in Helm commands or ArgoCD applications
fmt.Printf("Installing from: %s\n", gitURL)
```

The configuration supports customizable repository sources, branches, chart paths, and deployment settings for flexible app-of-apps installations.
