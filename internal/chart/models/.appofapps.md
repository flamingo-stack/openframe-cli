Configuration model for managing app-of-apps deployment using Helm and Git repositories. This struct encapsulates all necessary parameters for installing Helm charts from Git sources with the helm-git plugin.

## Key Components

**AppOfAppsConfig**: Main configuration struct containing Git repository details, certificate paths, values file location, and Helm deployment settings.

**NewAppOfAppsConfig()**: Factory function that creates a new configuration instance with sensible defaults for OpenFrame OSS tenant deployment.

**GetGitURL()**: Method that formats the Git repository URL into the specific format required by the helm-git plugin (v1.4.0+).

## Usage Example

```go
// Create config with defaults
config := NewAppOfAppsConfig()

// Customize configuration
config.GitHubBranch = "develop"
config.CertDir = "/etc/ssl/certs"
config.ValuesFile = "custom-values.yaml"
config.Timeout = "30m"

// Get formatted Git URL for helm-git plugin
gitURL := config.GetGitURL()
// Returns: "git+https://github.com/flamingo-stack/openframe-oss-tenant@manifests/app-of-apps?ref=develop"

// Use in helm command
fmt.Printf("helm install my-app %s --namespace %s --timeout %s", 
    gitURL, config.Namespace, config.Timeout)
```
