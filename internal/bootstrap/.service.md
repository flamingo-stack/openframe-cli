This service orchestrates the complete bootstrap process for OpenFrame, creating a Kubernetes cluster and installing charts in one command.

## Key Components

- **Service struct** - Main bootstrap service with no internal state
- **Execute()** - Command handler that processes CLI flags and executes bootstrap workflow
- **bootstrap()** - Core orchestration method that handles cluster creation and chart installation
- **createClusterSuppressed()** - Creates K3d cluster with suppressed UI output
- **installChartWithMode()** - Installs charts with specified deployment mode configuration
- **initializeWSL()** - Windows-specific WSL2 initialization for Ubuntu environment
- **buildClusterConfig()** - Constructs cluster configuration with sensible defaults

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/bootstrap"
    "github.com/spf13/cobra"
)

func main() {
    service := bootstrap.NewService()
    
    // Create bootstrap command
    cmd := &cobra.Command{
        Use: "bootstrap [cluster-name]",
        RunE: service.Execute,
    }
    
    // Add flags
    cmd.Flags().String("deployment-mode", "", "Deployment mode (oss-tenant, saas-tenant, saas-shared)")
    cmd.Flags().Bool("non-interactive", false, "Run in non-interactive mode")
    cmd.Flags().Bool("verbose", false, "Enable verbose output")
    
    // Execute with cluster name
    cmd.Execute()
}
```

The service validates deployment modes, handles Windows WSL2 initialization, creates a K3d cluster with 4 nodes, and installs OpenFrame charts using the specified deployment configuration. It integrates with the cluster and chart services to provide a seamless setup experience.
