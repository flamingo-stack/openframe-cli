This file provides a bootstrap service that orchestrates cluster creation and chart installation in a single command for the OpenFrame CLI.

## Key Components

- **Service**: Main bootstrap service struct with command execution logic
- **NewService()**: Factory function to create a new bootstrap service instance
- **Execute()**: Main entry point that handles command flags, validation, and error handling
- **bootstrap()**: Core workflow orchestrating cluster creation followed by chart installation
- **createClusterSuppressed()**: Creates a Kubernetes cluster using k3d with non-interactive mode
- **installChartWithMode()**: Installs OpenFrame charts with specified deployment mode configuration

## Usage Example

```go
// Create a new bootstrap service
service := NewService()

// Execute bootstrap command (typically called by cobra command)
cmd := &cobra.Command{}
args := []string{"my-cluster"}
err := service.Execute(cmd, args)
if err != nil {
    log.Fatal(err)
}

// Direct bootstrap execution
err = service.bootstrap(
    "my-cluster", 
    "oss-tenant", 
    true,  // non-interactive
    false, // verbose
)
```

The service supports three deployment modes: `oss-tenant`, `saas-tenant`, and `saas-shared`. It validates flags, creates a k3d cluster with the specified name (defaults to "openframe-dev"), and installs the appropriate OpenFrame charts based on the deployment mode.
