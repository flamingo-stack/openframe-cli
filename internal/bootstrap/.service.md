Provides bootstrap functionality that orchestrates cluster creation and chart installation in a single command, streamlining the OpenFrame development environment setup process.

## Key Components

- **Service**: Main bootstrap service struct that coordinates cluster and chart operations
- **NewService()**: Factory function to create a new bootstrap service instance
- **Execute()**: Command handler that processes CLI flags and orchestrates the bootstrap workflow
- **bootstrap()**: Core orchestration method that executes cluster creation followed by chart installation
- **createClusterSuppressed()**: Creates a Kubernetes cluster with suppressed UI output
- **buildClusterConfig()**: Builds cluster configuration with defaults
- **installChartWithMode()**: Installs charts with specified deployment mode configuration

## Usage Example

```go
// Create and use bootstrap service
service := NewService()

// Execute bootstrap command (typically called by cobra command)
cmd := &cobra.Command{
    Use: "bootstrap [cluster-name]",
    Run: service.Execute,
}

// Manual bootstrap execution
err := service.bootstrap(
    "my-cluster",      // cluster name
    "oss-tenant",      // deployment mode
    false,             // non-interactive
    true,              // verbose
)
if err != nil {
    log.Fatal(err)
}
```

The service validates deployment modes (oss-tenant, saas-tenant, saas-shared) and requires deployment mode when running in non-interactive mode. It creates a K3d cluster with 3 nodes by default and installs OpenFrame charts using the specified configuration.
