This service orchestrates the complete OpenFrame cluster bootstrap process by combining cluster creation and chart installation into a single command.

## Key Components

- **Service**: Main bootstrap service struct providing end-to-end cluster setup
- **NewService()**: Factory function creating a new bootstrap service instance
- **Execute()**: Primary command handler processing CLI flags and coordinating bootstrap operations
- **bootstrap()**: Core orchestration method executing cluster creation followed by chart installation
- **createClusterSuppressed()**: Wrapper for cluster creation with suppressed UI output
- **buildClusterConfig()**: Helper building cluster configuration with sensible defaults
- **installChartWithMode()**: Wrapper for chart installation with deployment mode configuration

## Usage Example

```go
// Create and execute bootstrap service
service := bootstrap.NewService()

// Execute with cobra command (handles flags internally)
err := service.Execute(cmd, []string{"my-cluster"})
if err != nil {
    log.Fatal(err)
}

// The service automatically:
// 1. Validates deployment mode (oss-tenant, saas-tenant, saas-shared)
// 2. Creates k3d cluster with name "my-cluster" (or "openframe-dev" default)
// 3. Installs OpenFrame charts with specified deployment mode
// 4. Handles verbose output and non-interactive modes
```

The service supports deployment modes, non-interactive execution, and verbose logging while providing consistent error handling throughout the bootstrap process.
