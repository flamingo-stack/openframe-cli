Provides a comprehensive bootstrap service that combines cluster creation and chart installation into a single streamlined operation. The service handles deployment mode validation, command-line flag processing, and orchestrates the two-step bootstrap process with proper error handling.

## Key Components

- **Service**: Main bootstrap service struct that orchestrates cluster and chart operations
- **NewService()**: Factory function to create a new bootstrap service instance
- **Execute()**: Command handler that processes CLI arguments and flags, validates deployment modes
- **bootstrap()**: Core orchestration method that executes cluster creation followed by chart installation
- **createClusterSuppressed()**: Wrapper for creating clusters with suppressed UI in non-interactive mode
- **installChartWithMode()**: Installs charts with specified deployment mode configuration

## Usage Example

```go
// Create and use the bootstrap service
service := bootstrap.NewService()

// Execute bootstrap with cobra command
err := service.Execute(cmd, []string{"my-cluster"})
if err != nil {
    log.Fatal(err)
}

// Or call bootstrap directly
err = service.bootstrap("openframe-dev", "oss-tenant", true, false)
if err != nil {
    log.Fatal(err)
}
```

The service supports three deployment modes: `oss-tenant`, `saas-tenant`, and `saas-shared`, with automatic validation and default cluster naming when no name is provided.
