This file provides a bootstrap service that combines cluster creation and chart installation into a single streamlined command for OpenFrame CLI.

## Key Components

- **Service**: Main bootstrap service struct that orchestrates cluster creation and chart installation
- **Execute()**: Command handler that validates flags and executes the bootstrap process
- **bootstrap()**: Core method that creates a cluster then installs charts with deployment mode configuration
- **createClusterSuppressed()**: Creates a k3d cluster with non-interactive mode for automation
- **installChartWithMode()**: Installs OpenFrame charts with specific deployment mode settings

## Usage Example

```go
// Create a new bootstrap service
service := NewService()

// Execute bootstrap command with Cobra
var cmd *cobra.Command
err := service.Execute(cmd, []string{"my-cluster"})
if err != nil {
    log.Fatal(err)
}

// Direct bootstrap execution
err = service.bootstrap("openframe-dev", "oss-tenant", true, false)
if err != nil {
    log.Fatal(err)
}
```

The service supports three deployment modes (`oss-tenant`, `saas-tenant`, `saas-shared`) and can run in non-interactive mode for automation. It validates that deployment mode is specified when running non-interactively and uses consistent error handling across the OpenFrame CLI.
