This file implements a bootstrap service that combines cluster creation and chart installation into a single streamlined operation for the OpenFrame CLI.

## Key Components

- **Service**: Main bootstrap service struct that orchestrates cluster setup and chart deployment
- **NewService()**: Factory function to create a new bootstrap service instance
- **Execute()**: Primary command handler that processes CLI flags and validates deployment modes
- **bootstrap()**: Core orchestration method that sequences cluster creation and chart installation
- **createClusterSuppressed()**: Wrapper for non-interactive cluster creation with prerequisite checks
- **buildClusterConfig()**: Builds default cluster configuration with k3d type and 3 nodes
- **installChartWithMode()**: Installs charts with specified deployment mode configuration

## Usage Example

```go
// Create and use the bootstrap service
service := NewService()

// Execute bootstrap with cobra command
err := service.Execute(cmd, []string{"my-cluster"})
if err != nil {
    log.Fatal(err)
}

// Manual bootstrap call
err = service.bootstrap(
    "openframe-dev",     // cluster name
    "oss-tenant",        // deployment mode
    false,               // non-interactive
    true,                // verbose
)
```

The service validates deployment modes (oss-tenant, saas-tenant, saas-shared) and requires deployment mode when running non-interactively. It creates a k3d cluster with 3 nodes by default and installs charts from the OpenFrame OSS tenant repository.
