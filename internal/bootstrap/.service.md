This service orchestrates the bootstrap process by combining cluster creation and chart installation into a single command operation.

## Key Components

- **Service**: Main bootstrap service struct that coordinates the bootstrap workflow
- **NewService()**: Factory function to create a new bootstrap service instance
- **Execute()**: Command handler that processes CLI flags and executes the bootstrap sequence
- **bootstrap()**: Core orchestration method that creates cluster then installs charts
- **createClusterSuppressed()**: Creates a Kubernetes cluster with minimal UI output
- **installChartWithMode()**: Installs OpenFrame charts with specified deployment configuration
- **buildClusterConfig()**: Constructs cluster configuration with defaults

## Usage Example

```go
// Create and execute bootstrap service
service := NewService()

// Execute bootstrap with cobra command context
err := service.Execute(cmd, []string{"my-cluster"})
if err != nil {
    log.Fatal(err)
}

// Or use the internal bootstrap method directly
err = service.bootstrap("openframe-dev", "oss-tenant", true, false)
if err != nil {
    log.Fatal(err)
}
```

The service validates deployment modes (oss-tenant, saas-tenant, saas-shared), handles both interactive and non-interactive modes, and provides consistent error handling throughout the bootstrap process.
