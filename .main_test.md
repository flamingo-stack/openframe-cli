Integration test file for the main OpenFrame CLI application that validates command-line interface behavior by building and executing the binary with various arguments.

## Key Components

- **TestMainIntegration**: Primary integration test function that builds a test binary and validates CLI behavior
- **Test cases**: Covers help flag (`--help`), version flag (`--version`), and invalid flag handling
- **Binary execution**: Uses `os/exec` to build and run the CLI binary in isolated test environments

## Usage Example

```go
// Run the integration tests
go test -v ./main_test.go

// The test automatically:
// 1. Builds a test binary named "openframe-test-main"
// 2. Executes it with different CLI arguments
// 3. Validates expected output and exit codes
// 4. Cleans up the test binary

// Example test case structure:
tests := []struct {
    name     string
    args     []string
    wantErr  bool
    contains string
}{
    {
        name:     "help",
        args:     []string{"--help"},
        wantErr:  false,
        contains: "OpenFrame CLI",
    },
}
```

The test ensures the CLI properly handles standard flags and provides appropriate help/version information while gracefully handling invalid inputs.
